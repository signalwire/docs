# Using React Icons

We use `react-icons` to include icons in our components. React Icons provides a comprehensive collection of popular icon libraries that can be easily imported and used as React components.

This is meant to be used both in the internal components, and in the `mdx` documentation.


## Finding and Using Icons

1. Visit the [React Icons Search Page](https://react-icons.github.io/react-icons/icons/md/) to browse available icons
2. Search for icons using keywords in the search bar (e.g. "expand", "close", "menu")
3. Click on an icon to copy its import statement and component name

## Example Usage

Here's how to use Material Design icons from the `react-icons/md` collection:

```
import {MdCottage} from "react-icons/md";

<MdCottage /> Nothing like home 
```

import { MdCottage } from "react-icons/md";
import { MdExpandMore, MdExpandLess, MdSearch, MdMenu } from "react-icons/md";
import { FaGithub, FaTwitter, FaSlack, FaReact, FaNode, FaJava } from "react-icons/fa";
import { IoMdCheckmark, IoMdClose } from "react-icons/io";
import { SiPython, SiRuby, SiJavascript, SiTypescript } from "react-icons/si";

- <MdCottage /> Nothing like home :)

## Common Icons Examples

- <MdSearch /> MdSearch - Used for search functionality
- <MdMenu /> MdMenu - Used for mobile menu toggles
- <FaGithub /> FaGithub - GitHub link
- <FaTwitter /> FaTwitter - Twitter/X link
- <FaSlack /> FaSlack - Slack link
- <IoMdCheckmark /> IoMdCheckmark - Success indicators
- <IoMdClose /> IoMdClose - Close buttons or error indicators
- <SiPython /> SiPython - Python
- <FaReact /> FaReact - React
- <SiRuby /> SiRuby - Ruby
- <FaNode /> FaNode - Node.js
- <FaJava /> FaJava - Java
- <SiJavascript /> SiJavascript - JavaScript
- <SiTypescript /> SiTypescript - TypeScript

:::info try to stick to a single provider

The `react-icons` package will only bundle
icons that are used in the page, so size isn't an issue.

But for visual consistency, it might be a good idea to stick to a single provider.

:::
