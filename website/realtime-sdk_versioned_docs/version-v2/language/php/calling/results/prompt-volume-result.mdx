# Relay.Calling.PromptVolumeResult

This object is returned by [`volume`][prompt] method and represents the final result of a volume control operation.

## Properties

| Property | Type | Description |
| :-------- | :-------- | :----------- |
| `successful` | `boolean` | Whether the volume has been changed successfully. |
