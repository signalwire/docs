Events emitted by a CallSession object during call lifecycle, member actions, and room updates.

## Core Call Events

### `call.joined`

Fired when you successfully join a call.

<APIField name="params.room_session" type="[InternalCallSessionEntity](/sdks/signalwire-sdk/types#internalcallsessionentity)">
  Complete room session object.
</APIField>

<APIField name="params.room_id" type="string" required>
  The room ID.
</APIField>

<APIField name="params.room_session_id" type="string" required>
  The room session ID.
</APIField>

<APIField name="params.call_id" type="string" required>
  The call ID.
</APIField>

<APIField name="params.member_id" type="string" required>
  Your member ID.
</APIField>

<APIField name="params.capabilities" type="[CallCapabilitiesContract](/sdks/signalwire-sdk/types#callcapabilitiescontract)" required>
  Your permissions and capabilities for this call session.
</APIField>

<APIField name="params.origin_call_id" type="string" required>
  The original call ID.
</APIField>

<APIField name="params.node_id" type="string">
  Node ID (may be undefined).
</APIField>

```typescript
call.on('call.joined', (params) => {
  console.log('Joined call:', params.call_id)
  console.log('Your member ID:', params.member_id)
  console.log('Capabilities:', params.capabilities)
})
```

## Layout Events

### `layout.changed`

Fired when the video layout changes.

<APIField name="params.layout" type="[InternalVideoLayout](/sdks/signalwire-sdk/types#internalvideolayout)" required>
  Layout object with name and layers.
</APIField>

<APIField name="params.layout.name" type="string" required>
  Layout name (e.g., "grid-responsive", "2x2").
</APIField>

<APIField name="params.layout.layers" type="Layer[]" required>
  Array of layer objects with member positions.
</APIField>

<APIField name="params.room_id" type="string" required>
  The room ID.
</APIField>

```typescript
call.on('layout.changed', (params) => {
  console.log('Layout changed to:', params.layout.name)

  // Find your position
  const myLayer = params.layout.layers.find(
    layer => layer.member_id === call.memberId
  )
  if (myLayer) {
    console.log('Your position:', myLayer.position)
  }
})
```

## Connection State Events

### `active`

Connection is active and media is flowing.

```typescript
  console.log('Call is active')
})
```

### `answering`

Call is being answered.

```typescript

  console.log('Answering call...')

```

### `early`

Early media state (before call is fully answered).


### `hangup`

Call is hanging up.

### `destroy`

### `held`

Call is on hold.

### `new`

New connection being established.


### `ringing`

Call is ringing.


### `trying`

Attempting to establish connection.

### `requesting`

Connection request is being initiated.

### `recovering`

Connection is in recovery mode, attempting to restore state.

## Common Patterns

### Managing Event Listeners

```typescript
// Add listener
const handler = (params) => {
  console.log('Event fired:', params)
}
call.on('member.joined', handler)

// Remove listener
call.off('member.joined', handler)

// One-time listener
call.once('call.joined', (params) => {
  console.log('Joined - fires only once')
})
```

### Track Member Count

```typescript
let memberCount = 0

call.on('member.joined', () => {
  memberCount++
  updateUI({ memberCount })
})

call.on('member.left', () => {
  memberCount--
  updateUI({ memberCount })
})
```

### Monitor Active Speakers

```typescript
const activeSpeakers = new Set()

call.on('member.talking', (params) => {
  if (params.member.talking) {
    activeSpeakers.add(params.member.member_id)
  } else {
    activeSpeakers.delete(params.member.member_id)
  }

  console.log(`Active speakers: ${activeSpeakers.size}`)
})
```

### Handle Connection Issues

```typescript
call.on('media.disconnected', () => {
  showNotification('Connection lost')
})

call.on('media.reconnecting', () => {
  showNotification('Reconnecting...')
})

call.on('media.connected', () => {
  showNotification('Connection restored')
})
```
