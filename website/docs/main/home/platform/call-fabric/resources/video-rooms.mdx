# Video Rooms

<Subtitle>Resources on the Call Fabric platform</Subtitle>

## About

The Video Room Resource type is used to create and manage Video Rooms.

<Frame caption="The Resource selection menu">
  ![Select a Resource in the Signalwire Dashboard.](@image/dashboard/resources/add-new-resource-video-marked.png)
</Frame>



## Details 

A Video Room Resource has the following details:


| Detail | Description |
| :-- | :--
| `Room Name`<span className="optional-arg">Optional</span> | This specifies the uniqie name for the video room. |
| `Experience`<span className="required-arg">Required</span> | You can create a room with a preset embeddable UI or build your own with the [browser-sdk][browser-sdk] or our server [realtime sdk][realtime-sdk] and customize the experience to fit your needs |

</Frame>
