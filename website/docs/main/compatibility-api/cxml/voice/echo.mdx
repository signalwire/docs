The `<Echo>` verb will echo audio back to the call.

## Verb Attributes

|                                                Attribute |                                                                                                                       |
| -------------------------------------------------------: | --------------------------------------------------------------------------------------------------------------------- |
| `timeout` <span className="optional-arg">optional</span> | The number of seconds SignalWire will echo, from 5 up to 120 seconds. Defaults to 60 seconds if no value is provided. |

## Nesting

No other verbs can be nested within `<Echo>`.

## Examples

### A simple echo test

<Tabs groupId="cxml-api">
  <TabItem value="XML">

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Response>
    <Echo timeout="120"/>
    <Hangup/>
</Response>
```

  </TabItem>

</Tabs>

SignalWire answers calls and echoes what it hears for 2 minutes.
