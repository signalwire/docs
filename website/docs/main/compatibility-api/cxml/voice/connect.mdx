The `<Connect>` verb connects an existing call to another resource.

You can use the `<Connect>` verb with the following nouns:

|                    Noun                     |                             |
|:-------------------------------------------:|:---------------------------:|
|         [`<Room>`](./room-noun.mdx)         |        A video room.        |
|         [`<Stream>`](./stream.mdx)          |   A bidirectional stream.   |
| [`<VirtualAgent>`](./virtualagent-noun.mdx) | A Dialogflow virtual agent. |

## Examples

### A Simple Connect

<Tabs groupId="cxml-api">
  <TabItem value="XML">

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Response>
  <Connect>
    <Room>my-room-name</Room>
  </Connect>
</Response>
```

  </TabItem>

</Tabs>

### Bidirectional Media Stream

<Tabs groupId="cxml-api">
  <TabItem value="XML">

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Response>
  <Connect>
    <Stream url="wss://your-application.com/audiostream" />

```
