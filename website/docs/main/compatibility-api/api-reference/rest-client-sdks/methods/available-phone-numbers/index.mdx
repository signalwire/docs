# Available Phone Numbers

The ability to search for local, toll-free, and mobile phone numbers to purchase.

The parameters listed below apply to local, toll-free, and mobile phone numbers.

|                                                Attribute |                                                                                                                                                                               |
| -------------------------------------------------------: | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|         `beta` <span class="optional-arg">boolean</span> | New numbers on SignalWire are marked as `beta`. Possible values are `true` or `false`.                                                                                        |
|  `capabilities` <span class="optional-arg">object</span> | Whether or not a number can receive calls and messages. Possible values are `Voice`, `SMS`, `MMS`, and `Fax`. Each of these values have a boolean value of `true` or `false`. |
| `friendly_name` <span class="optional-arg">string</span> | A formatted version of the number.                                                                                                                                            |
|   `iso_country` <span class="optional-arg">string</span> | The ISO country code of the number.                                                                                                                                           |
|          `lata` <span class="optional-arg">string</span> | The LATA of the number. Only available for numbers in US and Canada.                                                                                                          |
|      `latitude` <span class="optional-arg">string</span> | The latitude of the number. Only available for numbers in US and Canada.                                                                                                      |
|     `longitude` <span class="optional-arg">string</span> | The longitude of the number. Only available for numbers in US and Canada.                                                                                                     |
|  `phone_number` <span class="optional-arg">string</span> | The number in E.164 format.                                                                                                                                                   |
|   `postal_code` <span class="optional-arg">string</span> | The postal/zip code of the number. Only available for numbers in US and Canada.                                                                                               |
|   `rate_center` <span class="optional-arg">string</span> | The rate center of the number. Only available for numbers in US and Canada.                                                                                                   |
|        `region` <span class="optional-arg">string</span> | The state or province abbreviation of the number. Only available for numbers in US and Canada.    

> A sample AvailablePhoneNumber returned from the API.

```json
{
  "friendly_name": "253-218-6751",
  "phone_number": "+12532186751",
  "lata": null,
  "rate_center": "AUBURN",
  "latitude": null,
  "longitude": null,
  "region": "WA",
  "postal_code": null,
  "iso_country": "US",
  "capabilities": {
    "voice": true,
    "SMS": true,
    "MMS": true
  },
  "beta": false
}
```                                                                            |
