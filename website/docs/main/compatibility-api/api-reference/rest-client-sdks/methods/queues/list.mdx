# List All Queues

Use this endpoint for the [Queues](./index.mdx) method to read all of the queues that are associated with your account. 
This will be returned as a list of queues.

## Parameters

| Parameter                                                 | Description                                                   |
| :-------------------------------------------------------- | :------------------------------------------------------------ |
| `FriendlyName` <span className="required-arg">Required</span> | Returns queues that exactly match the provided friendly name. |
| `MaxSize` <span className="optional-arg">Optional</span>      | Returns queues that have the provided maximum size.           |
