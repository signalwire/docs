# Delete a Queue

Use this endpoint for the [Queues](./index.mdx) method to delete a single call queue. 
Only empty queues can be deleted. 
If the delete is successful, a 204 response, with no body, will be returned.

## Parameters

| Parameter                                        |
|:-------------------------------------------------|
| <span class="optional-arg">No Parameters</span>  |

## Request

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Queues/{QueueSid}.json \
  -X DELETE \
  -u "YourProjectID:YourAuthToken"
```
