# Fax Media

The **Fax Media** resource provides detailed information about the media files attached to fax instances.

## Properties

> A sample fax media returned from the API

```json
{
  "account_sid": "b3877c40-da60-4998-90ad-b792e98472ac",
  "content_type": "application/pdf",
  "date_created": "2018-11-27T20:00:00Z",
  "date_updated": "2018-11-28T20:00:00Z",
  "fax_sid": "b3877c40-da60-4998-90ad-b792e98472fs",
  "sid": "b3877c40-da60-4998-90ad-b792e98472fx",
  "url": "/api/laml/2010-04-01/Accounts/b3877c40-da60-4998-90ad-b792e98472ac/Faxes/b3877c40-da60-4998-90ad-b792e98472fs/Media/b3877c40-da60-4998-90ad-b792e98472fx"
}
```

|                                               Attribute |                                                                          |
| ------------------------------------------------------: | ------------------------------------------------------------------------ |
|  `account_sid` <span class="optional-arg">string</span> | The unique identifier for the account this fax media is associated with. |
| `content_type` <span class="optional-arg">string</span> | The content type of the stored media.                                    |
| `date_created` <span class="optional-arg">string</span> | The date and time, in ISO 8601 format, the fax media was created.        |
| `date_updated` <span class="optional-arg">string</span> | The date and time, in ISO 8601 format, the fax media was updated.        |
|      `fax_sid` <span class="optional-arg">string</span> | The unique identifier of the fax that the media is associated with.      |
|          `sid` <span class="optional-arg">string</span> | The unique identifier for the fax media.                                 |
|          `url` <span class="optional-arg">string</span> | The URL of this resource.                                                |
