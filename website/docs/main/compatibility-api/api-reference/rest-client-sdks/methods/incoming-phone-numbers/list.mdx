# List All IncomingPhoneNumbers

Use this endpoint for the [IncomingPhoneNumber](./index.mdx) method to read all of the IncomingPhoneNumbers that are associated with your SignalWire account. 
This will be returned as a list of IncomingPhoneNumbers.

## Parameters

| Parameter                                                 | Description                                                                                                                               |
| :-------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------- |
| `AccountSid` <span className="required-arg">Required</span>   | The unique identifier of the account associated with this phone number.                                                                   |
| `Beta` <span className="optional-arg">Optional</span>         | Shows new SignalWire numbers. Possible values are `true` or `false`. Default is **true**.                                                 |
| `FriendlyName` <span className="optional-arg">Optional</span> | Shows numbers that exactly match the provided friendly name.                                                                              |
| `Origin` <span className="optional-arg">Optional</span>       | Shows numbers that have the provided origin. SignalWire numbers are denoted as `signalwire` while hosted numbers are denoted as `hosted`. |
| `PhoneNumber` <span className="optional-arg">Optional</span>  | Shows the numbers that exactly match the provided pattern.                                                                                |
