# Retrieve an Account

Use this endpoint for the [Accounts](./index.mdx) method to retrieve a single account.

## Parameters

|                                        Parameter |                                                                  |
| -----------------------------------------------: | ---------------------------------------------------------------- |
| `Sid` <span className="required-arg">Required</span> | The Project ID that uniquely identifies the account to retrieve. |

---

## Request

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{Sid}.json \
  -X GET \
  -u "YourProjectID:YourAuthToken"
```

</TabItem>

<TabItem value="node" label="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.api.accounts('YourProjectID')
          .fetch()
          .then(account => console.log(account.friendlyName))
          .done();
```

</TabItem>

<TabItem value="csharp" label="C#">

using System;
