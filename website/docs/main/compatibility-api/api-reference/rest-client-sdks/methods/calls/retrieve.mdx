# Retrieve a Call

Use this endpoint for the [Calls](./index.mdx) method to retrieve a single call.

## Parameters

| Parameter                                        | Description                                                                           |
| :----------------------------------------------- | :------------------------------------------------------------------------------------ |
| `Sid` <span className="required-arg">Required</span> | The Call [SID](/platform/dashboard/guides/what-is-a-sid/) that uniquely identifies the call to retrieve. |

---

## Request

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Calls/{Sid}.json \
  -X GET \
  -u "YourProjectID:YourAuthToken"
```

</TabItem>

<TabItem value="node" label="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.calls('Sid')
      .fetch()
      .then(call => console.log(call.to))
      .done();
```
