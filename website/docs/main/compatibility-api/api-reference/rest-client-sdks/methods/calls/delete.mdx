# Delete a Call

Use this endpoint for the [Calls](./index.mdx) method to delete a call. 
If the delete is successful, a 204 response, with no body, will be returned.

## Parameters

| Parameter                                        | Description                                                                         |
| :----------------------------------------------- | :---------------------------------------------------------------------------------- |
| `Sid` <span className="required-arg">Required</span> | The Call [SID](/platform/dashboard/guides/what-is-a-sid) that uniquely identifies the call to delete. |

---
