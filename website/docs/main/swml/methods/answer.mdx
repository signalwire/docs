Answer incoming call and set an optional maximum duration.

<APIField
  name="answer"
  type="object"
>
  An object that contains the [`answer parameters`](#answer-parameters).
</APIField>


## **answer Parameters**

The `answer` method expects the following parameters.

<APIField
  name="max_duration"
  type="integer"
  default="14400 seconds (4 hours)"
>
  Maximum duration in seconds. Can not be less than 15 seconds.
</APIField>

<APIField
  name="sip_auth_username"
  type="string"
>
  Username to use for SIP authentication.
</APIField>

<APIField
  name="sip_auth_password"
  type="string"
>
  Password to use for SIP authentication.
</APIField>


## **Examples**

### No parameters

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
```

---

### Named parameter

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer:
        max_duration: 60
```
