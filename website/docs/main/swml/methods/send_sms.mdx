Send an outbound message to a PSTN phone number.

<APIField
  name="send_sms"
  type="object"
  required={true}
>
  An object that accepts the [`send_sms parameters`](#send_sms-parameters).

## **send_sms Parameters**

<Tabs>
<TabItem value="sms" label="SMS" default>

<APIField
  name="to_number"
  type="string"
  required={true}
>
  Phone number to send SMS message to in e.164 format
</APIField>

<APIField
  name="from_number"
  type="string"
  required={true}
>
  Phone number SMS message will be sent from
</APIField>

<APIField
  name="body"
  type="string"
  required={true}
>
  Body of the text message
</APIField>

<APIField
  name="region"
  type="string"
  default="Chosen based on account preferences or device location"
>
  Region of the world to originate the message from
</APIField>

<APIField
  name="tags"
  type="string[]"
>
  Array of tags to associate with the message to facilitate log searches
</APIField>

</TabItem>

## **Variables**

Set by the method:

- **send_sms_result:** (out) `success` | `failed`.

## **Examples**

<Tabs>
<TabItem value="sms" label="SMS" default>

Send a text-only message:

```yaml andJson
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        body: "Hi, I hope you're well."
```

</TabItem>
<TabItem value="mms" label="MMS">

Send a message with media attachment:

```yaml andJson
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        media: ["https://example.com/image.jpg"]
        body: "Check out this image!"
```

</TabItem>
</Tabs>
