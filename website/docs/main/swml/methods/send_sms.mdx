Send an outbound message to a PSTN phone number.

| Name                                                     | Type     | Default | Description                                                               |
|:---------------------------------------------------------|:---------|:--------|:--------------------------------------------------------------------------|
| `send_sms`<span className="required-arg">Required</span> | `object` | -       | An object that accepts the [`send_sms parameters`](#send_sms-parameters). |

## **send_sms Parameters**

<Tabs>
<TabItem value="sms" label="SMS" default>

| Name                                                    | Type       | Default                                                | Description                                                            |
|:--------------------------------------------------------|:-----------|:-------------------------------------------------------|:-----------------------------------------------------------------------|
| `to_number`<span className="required-arg">Required</span> | `string`   |                                                        | Phone number to send SMS message to in e.164 format                    |
| `from_number`<span className="required-arg">Required</span> | `string`   |                                                        | Phone number SMS message will be sent from                             |
| `body`<span className="required-arg">Required</span>      | `string`   |                                                        | Body of the text message                                               |
| `region`<span className="optional-arg">Optional</span>    | `string`   | picked based on account preferences or device location | Region of the world to originate the message from                      |
| `tags`<span className="optional-arg">Optional</span>      | `string[]` |                                                        | Array of tags to associate with the message to facilitate log searches |

</TabItem>
<TabItem value="mms" label="MMS">

| Name                                                    | Type       | Default                                                | Description                                                            |
|:--------------------------------------------------------|:-----------|:-------------------------------------------------------|:-----------------------------------------------------------------------|
| `to_number`<span className="required-arg">Required</span> | `string`   |                                                        | Phone number to send SMS message to in e.164 format                    |
| `from_number`<span className="required-arg">Required</span> | `string`   |                                                        | Phone number SMS message will be sent from                             |
| `media`<span className="required-arg">Required</span>     | `string[]` |                                                        | Array of media URLs to include in the message                          |
| `body`<span className="optional-arg">Optional</span>      | `string`   |                                                        | Optional text to accompany the media                                   |
| `region`<span className="optional-arg">Optional</span>    | `string`   | picked based on account preferences or device location | Region of the world to originate the message from                      |
| `tags`<span className="optional-arg">Optional</span>      | `string[]` |                                                        | Array of tags to associate with the message to facilitate log searches |

</TabItem>
</Tabs>

## **Variables**

Set by the method:

- **send_sms_result:** (out) `success` | `failed`.

## **Examples**

<Tabs>
<TabItem value="sms" label="SMS" default>

Send a text-only message:

```yaml andJson
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        body: "Hi, I hope you're well."
```

</TabItem>
<TabItem value="mms" label="MMS">

Send a message with media attachment:

```yaml andJson
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        media: ["https://example.com/image.jpg"]
        body: "Check out this image!"
```

</TabItem>
</Tabs>
