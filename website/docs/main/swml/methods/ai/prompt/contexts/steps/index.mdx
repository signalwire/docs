An array of objects that define the steps in the context.
These steps are used to define the flow of the conversation.

<APIField
  name="steps"
  type="object[]"
  required={true}
>
  An array of objects that accept the [`steps parameters`](#steps-parameters).
</APIField>


## steps Parameters

The `steps` property accepts one of the following step types:

<TabItem value="text-step" label="Text Step">

<APIField

  type="string"
  required={true}
>
  The prompt or instructions given to the AI at this step.
