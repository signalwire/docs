## **Variable Expansion**

Use the following syntax to expand variables into your prompt.

<APIField
  name="${call_direction}"
  type="string"
>
  Inbound or outbound.
</APIField>

<APIField
  name="${caller_id_number}"
  type="string"
>
  The caller ID number.
</APIField>

<APIField
  name="${local_date}"
  type="string"
>
  The local date.
</APIField>

<APIField
  name="${spoken_date}"
  type="string"
>
  The spoken date.
</APIField>

<APIField
  name="${local_time}"
  type="string"
>
  The local time.
</APIField>

<APIField
  name="${time_of_day}"
  type="string"
>
  The time of day.
</APIField>

<APIField
  name="${supported_languages}"
  type="string"
>
  A list of supported languages.
</APIField>

<APIField
  name="${default_language}"
  type="string"
>
  The default language.
</APIField>
