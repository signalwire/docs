Set script variables to the specified values. Variables set using `set` can be removed
using [`unset`](./unset.mdx).

<APIField
  name="set"
  type="object"
  required={true}
>
  An object that accepts user-defined key-value pairs.
</APIField>

## **Variables**

Any variable can be set by this method.

## **Examples**

### Setting variables

```yaml andJson
version: 1.0.0
sections:
  main:
    - set:
        num_var: 1
    - play:
        url: 'say: ${num_var}'
    - set:
        num_var: 2
    - play:
        url: 'say: ${num_var}'
```

### Setting multiple variables

```yaml andJson
version: 1.0.0
sections:
  main:
    - set:
        items:
          - drill bit
          - drill
        person: handyman
        systems:
          ventilation:
            - inlet
            - outlet
            - fans
          hr:
            - lucy
            - liam
            - luke
    - play:
        url: 'say: The items ${items} will be used by ${person} to fix ${systems.ventilation}.'
```
