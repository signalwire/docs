# live_transcribe.action
This page provides an overview of all valid actions for the [`live_transcribe`](../) SWML method.

The `live_transcribe` method requires exactly [`oneOf`](https://json-schema.org/understanding-json-schema/reference/combining#oneOf)
the below `action` items. If more than one (or none) is provided, the `action` will fail.

<APIField
  name="action"
  type="string | object"
  required={true}
>
  The action to be performed.<br/>An object that contains **one** of the [`live_translate actions`](#actions)<br/>**Possible Values:** [[`stop`](./stop.mdx), [`start`](./start.mdx), [`summarize`](./summarize.mdx)]
</APIField>



## **Actions**

<APIField
  name="stop"
  type="string"
  required={true}
>
  [Stops](./stop.mdx) the live transcribing session.
</APIField>

<APIField
  name="start"
  type="object"
  required={true}
>
  [Starts](./start.mdx) a live transcribing session.
</APIField>

<APIField
  name="summarize"
  type="object"
  required={true}
>
  [Summarizes](./summarize.mdx) the conversation.
</APIField>
