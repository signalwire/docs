Stop an active background recording.

<APIField
  name="stop_record_call"
  type="object"
  required={true}
>
  An object that accepts the [`stop_record_call parameters`](#stop_record_call-parameters).
</APIField>

## **stop_record_call Parameters**

<APIField
  name="control_id"
  type="string"
  default="The last started recording will be stopped"
>
  Identifier for the recording to stop
</APIField>

## **Variables**

Read by the method:

- **record_control_id:** (in) control ID of last recording started.

Set by the method:

- **stop_record_call_result:** (out) `success` | `failed`

## **Examples**

### Stop last call recording

```yaml andJson
version: 1.0.0
sections:
  main:
    - stop_record_call: {}
```

### Stop a specific call recording

```yaml andJson
version: 1.0.0
sections:
  main:
    - stop_record_call:
        control_id: my-recording-id
```
