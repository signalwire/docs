Execute different instructions based on a variable's value

<APIField
  name="switch"
  type="object"
  required={true}
>
  An object that accepts the [`switch parameters`](#switch-parameters).
</APIField>

## **switch Parameters**

<APIField
  name="variable"
  type="string"
  required={true}
>
  Name of the variable whose value needs to be compared.
</APIField>

<APIField
  name="case"
  type="object"
  required={true}
>
  [`Case_params`](#case_params) object of key-mapped values to array of [SWML Methods](./index.mdx) to execute.
</APIField>

<APIField
  name="default"
  type="[]"
>
  Array of [SWML Methods](./index.mdx) to execute if no cases match.
</APIField>

### case_params

The `case_params` object serves as a dictionary where each key is a string identifier, and
the associated value is an array of SWML Method objects.

<APIField
  name="[key: string]"
  type={"SWML Methods[]"}
  required={true}
>
  Name of the variable whose value needs to be compared
</APIField>

## **Examples**

```yaml andJson
version: 1.0.0
sections:
  main:
    - switch:
        variable: call.type
        case:
          sip:
            - play:
                url: "say: You're calling from SIP."
          phone:
            - play:
                url: "say: You're calling from phone."
        default:
          - play:
              url: 'say: Unexpected error'
```

```yaml andJson
version: 1.0.0
sections:
  main:
    - set:
        foo: 5
    - execute:
        dest: example_fn
        params:
          foo: '${foo}'
  example_fn:
    - switch:
        variable: params.foo
        default:
          - play:
              url: 'say: nothing matches'
        case:
          '5':
            - play:
                url: 'say: yup, math works!'
```
