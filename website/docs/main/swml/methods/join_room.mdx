Join a RELAY room. If the room doesn't exist, it creates a new room.

<APIField
  name="join_room"
  type="object"
  required={true}
>
  An object that accepts the [`join_room parameters`](#join_room-parameters).
</APIField>

## **join_room Parameters**

<APIField
  name="name"
  type="string"
  required={true}
>
  Name of the room to join. Allowed characters: [`A-Z | a-z | 0-9_-`](/rest/signalwire-rest/endpoints/video/create-room)
</APIField>

## **Variables**

Set by the method:

- **join_room_result:** (out) `joined` | `failed`

## **Examples**

### Joining a room

```yaml andJson
version: 1.0.0
sections:
  main:
    - join_room:
        name: my_room
```
