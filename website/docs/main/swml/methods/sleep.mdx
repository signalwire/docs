Set the amount of time for the current application to sleep for in milliseconds before
continuing to the next action.

<APIField
  name="sleep"
  type="object"
  required={true}
>
  An object that accepts the [`sleep parameters`](#sleep-parameters).
</APIField>

## **sleep Parameters**

<APIField
  name="duration"
  type="integer"
  required={true}
  default="0"
>
  The amount of time to sleep in milliseconds. Must be a positive number. Can also be set to a `-1` integer for the sleep to never end.<br />**Possible Values:** [`-1`, `<Any Positive Number>`]
</APIField>

## **Examples**

### Timed Sleep Example

```yaml andJson
version: 1.0.0
sections:
  main:
    - sleep: 5000
```

### Forever Sleep Example

```yaml andJson
version: 1.0.0
sections:
  main:
    - sleep: -1
```
