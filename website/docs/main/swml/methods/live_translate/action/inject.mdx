# action.inject

Inject a message into the conversation.

<APIField
  name="inject"
  type="object"
  required={true}
>
  An object that contains the [`inject parameters`](#inject-parameters).
</APIField>

## **inject Parameters**

<APIField
  name="message"
  type="string"
  required={true}
>
  The message to be injected.
</APIField>

<APIField
  name="direction"
  type="string"
  required={true}
  default="local-caller"
>
  The direction of the message.<br/>**Possible Values**: `"remote-caller"`, `"local-caller"`
</APIField>

## **Example**

```yaml andJson
live_translate:
  action:
    inject:
      message: This is an injected message.
      direction: remote-caller
```
