Send digit presses as DTMF tones.

<APIField
  name="send_digits"
  type="object"
  required={true}
>
  An object that accepts the [`send_digits parameters`](#send_digits-parameters).
</APIField>

## **send_digits Parameters**

<APIField
  name="digits"
  type="string"
  required={true}
>
  The digits to send. Valid values are `0123456789*#ABCDWw`. Character `W` is a 1 second delay, and `w` is a 500 ms delay.
</APIField>

## **Variables**

Set by the method:

- **send_digits_result:** (out) `success` | `failed`

## **Examples**

### Send digits

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
    - send_digits:
        digits: '012345'
    - play:
        url: 'say: ${send_digits_result}'
```
