Mark any point of the SWML section with a label so that [`goto`](./goto.mdx) can jump to it.


## **label Parameters**

<APIField
  name="label"
  type="string"
>
  an identifier string for the current label
</APIField>

## **Examples**

```yaml andJson
version: 1.0.0
sections:
  main:
    - label: foo
    - play:
        url: 'say: This speech will be repeated 4 times.'
    - goto:
        label: foo
        max: 3
```
