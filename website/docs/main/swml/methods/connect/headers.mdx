Custom SIP headers to add to INVITE. Has no effect on calls to phone numbers.

<APIField
  name="headers"
  type="object[]"
>
  An array of objects that accept the [headers parameters](#headers-parameters).
</APIField>



## headers parameters

<APIField
  name="name"
  type="string"
  required={true}
>
  The name of the header.
</APIField>

<APIField
  name="value"
  type="string"
  required={true}
>
  The value of the header.
</APIField>


## Example

```yaml andJson
version: 1.0.0
sections:
  main:
    - connect:
        to: "sip:user-a@<domain>.com"
        headers:
          - name: "x-FROM_NUMBER"
            value: "+2B123456789"
```
