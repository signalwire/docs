Stop an active tap stream.

<APIField
  name="stop_tap"
  type="object"
  required={true}
>
  An object that accepts the [`stop_tap parameters`](#stop_tap-parameters).
</APIField>

## **stop_tap Parameters**

<APIField
  name="control_id"
  type="string"
  default="The last tap started will be stopped"
>
  ID of the tap to stop
</APIField>

## **Variables**

Read by the method:

- **tap_control_id:** (in) Control ID of last tap stream started.

Set by the method:

- **stop_tap_result:** (out) Success or failed.

## **Examples**

### Stop the last call tap

```yaml andJson
version: 1.0.0
sections:
  main:
    - stop_tap: {}
```

### Stop a specific call tap

```yaml andJson
version: 1.0.0
sections:
  main:
    - stop_tap:
        control_id: my-tap-id
```
