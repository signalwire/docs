# SWML Technical Reference

<Subtitle>Authoritative specifications for SWML language features</Subtitle>

The SWML Technical Reference provides comprehensive documentation for all SWML language features, methods, and specifications. This section contains detailed technical information about how SWML works, including complete syntax references, parameter specifications, and behavior descriptions.

## What's in the Technical Reference

### Variables and Expressions

Complete reference for SWML's variable system, including:
- Variable syntax and substitution (`${}` and `%{}`)
- All variable scopes (call, params, vars, args, meta_data, global_data, etc.)
- The Call Object specification
- JavaScript expressions in variables
- Template functions and transformations
- Variables in conditional logic
- Best practices and common patterns

**[View Variables and Expressions Reference →](/swml/guides/variables-and-expressions)**

### Methods Reference

Complete documentation for all SWML methods, organized by category:

**Control Flow Methods**
- [`cond`](/swml/methods/cond) - Conditional execution
- [`switch`](/swml/methods/switch) - Switch-case logic
- [`goto`](/swml/methods/goto) - Jump to labels
- [`execute`](/swml/methods/execute) - Execute sections/external scripts
- [`transfer`](/swml/methods/transfer) - Transfer calls
- [`return`](/swml/methods/return) - Return values

**Call Handling Methods**
- [`answer`](/swml/methods/answer) - Answer incoming calls
- [`hangup`](/swml/methods/hangup) - End calls
- [`connect`](/swml/methods/connect) - Connect calls
- [`denoise`](/swml/methods/denoise) - Noise reduction
- [`receive`](/swml/methods/receive) - Receive media

**Media Playback Methods**
- [`play`](/swml/methods/play) - Play audio/video
- [`stop`](/swml/methods/stop) - Stop media playback

**User Input Methods**
- [`prompt`](/swml/methods/prompt) - Collect user input
- [`record`](/swml/methods/record) - Record audio/video

**AI Methods**
- [`ai`](/swml/methods/ai) - AI agent interactions
- SWAIG function integration

**Data Methods**
- [`set`](/swml/methods/set) - Set variables
- [`unset`](/swml/methods/unset) - Remove variables
- [`request`](/swml/methods/request) - HTTP requests
