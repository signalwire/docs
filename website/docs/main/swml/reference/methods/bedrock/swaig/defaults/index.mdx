Default settings for all SWAIG functions. If `defaults` is not set, settings may be set in each [SWAIG function](../functions/index.mdx) object.

<APIField
  name="SWAIG.defaults"
  type="object"
>
  An object that contains the [`defaults Parameters`](#parameters).
</APIField>

## defaults Parameters {#parameters}

<APIField
  name="defaults.web_hook_url"
  type="string"
>
  Default URL to send status callbacks and reports to. Authentication can also be set in the url in the format of `username:password@url`.

  See [`web_hook_url`](./defaults.web_hook_url.mdx) for details on the request body.
</APIField>
