Jump to a `label` within the current section, optionally based on a condition.
The `goto` method will only navigate to a label within the same section.

<APIField
  name="goto"
  type="object"
  required={true}
>
  An object that accepts the [`goto parameters`](#goto-parameters).
</APIField>


## **goto Parameters**

<APIField
  name="goto.label"
  type="string"
  required={true}
>
  The label in section to jump to.
</APIField>

<APIField
  name="goto.when"
  type="string"
>
  A JavaScript condition that determines whether to perform the jump. If the condition evaluates to true, the jump is executed. If omitted, the jump is unconditional.
</APIField>

<APIField
  name="goto.max"
  type="integer"
  default="100"
>
  The maximum number of times to jump, from the minimum of `1` to max `100` jumps.
</APIField>

## **Examples**

### Loop with max

```yaml andJson
version: 1.0.0
sections:
  main:
    - label: foo
    - play:
        url: 'say: This speech will be repeated 4 times.'
    - goto:
        label: foo
        max: 3
```

### Loop if a condition is satisfied

```yaml andJson
version: 1.0.0
sections:
  main:
    - label: foo
    - play:
        url: 'say: This is some text that will be repeated 4 times.'
    - set:
        do_loop: true
    - goto:
        label: foo
        when: do_loop === true
        max: 3
```
