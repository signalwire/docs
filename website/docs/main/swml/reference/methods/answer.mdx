Answer incoming call and set an optional maximum duration.

<APIField
  name="answer"
  type="object"
>
  An object that contains the [`answer parameters`](#answer-parameters).
</APIField>


## **answer Parameters**

The `answer` method expects the following parameters.

<APIField
  name="answer.max_duration"
  type="integer"
  default="14400 seconds (4 hours)"
>
  Maximum duration in seconds for the call.
</APIField>

<APIField
  name="answer.codecs"
  type="string"
>
  Comma-separated string of codecs to offer. Valid codecs are: `PCMU`, `PCMA`, `G722`, `G729`, `AMR-WB`, `OPUS`, `VP8`, `H264`.
</APIField>

<APIField
  name="answer.username"
  type="string"
>
  Username to use for SIP authentication.
</APIField>

<APIField
  name="answer.password"
  type="string"
>
  Password to use for SIP authentication.
</APIField>


## **Examples**

### No parameters

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
```

---

### Named parameter

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer:
        max_duration: 60
```
