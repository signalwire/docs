## Relay.Calling.StopResult

This object is returned from one the _synchronous_ `stop` methods on an action when an _asynchronous_ operation is being stopped, which represent the final result of a stop operation.

### Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `successful` | boolean | Whether the stop operation has completed successfully. |
