## Relay.Calling.AnswerResult

This object is returned by the [`answer`][answer] method.

### Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `successful` | boolean | Whether the call has been answered from the remote party. |
| `event` | [`Relay.Event`][relay-event] | Last event that completed the operation. |
