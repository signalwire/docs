<ParamField path="action" type="object" required={true} toc={true}>
  The transcription action to perform. Use one of:

  - `{ start: {...} }` — Start live transcription. Accepts: `lang`, `direction`,
    `webhook`, `live_events`, `ai_summary`, `ai_summary_prompt`, `speech_engine`
    (default `"deepgram"`), `speech_timeout`, `vad_thresh`, `debug_level`.
  - `{ summarize: {...} }` — Request a summary of the current transcription.
  - `"stop"` — Stop live transcription.
</ParamField>
