<ParamField path="detect" type="object" required={true} toc={true}>
  Detection configuration specifying what to detect. Contains a `type` field and
  type-specific `params`:

  - `"machine"` — Answering machine detection (AMD). Optional params: `initial_timeout`, `end_silence_timeout`, `machine_voice_threshold`, `machine_ready_timeout`.
  - `"fax"` — Fax tone detection. Optional params: `tone`.
  - `"digit"` — DTMF digit detection. Optional params: `digits`.
</ParamField>

<ParamField path="timeout" type="number" toc={true}>
  Maximum time in seconds to run the detection.
</ParamField>

<ParamField path="control_id" type="string" toc={true}>
  Unique identifier for this detect operation, used to control it later.
  If not provided, one is generated automatically.
</ParamField>
