<ParamField path="digits" type="object" toc={true}>
  Configuration for collecting DTMF digit input. Supported keys:

  - `max` — Maximum number of digits to collect
  - `terminators` — DTMF digits that signal the end of input (e.g., `"#"`)
  - `digit_timeout` — Time in seconds to wait between digit presses
</ParamField>

<ParamField path="speech" type="object" toc={true}>
  Configuration for collecting speech input. Supported keys:

  - `end_silence_timeout` — Time in seconds of silence after speech to consider input complete
  - `speech_timeout` — Maximum time in seconds to wait for speech input
  - `language` — The language for speech recognition (e.g., `"en-US"`)
  - `hints` — Array of words or phrases to bias the speech recognition
  - `engine` — The speech recognition engine to use
</ParamField>

<ParamField path="initial_timeout" type="number" toc={true}>
  Maximum time in seconds to wait for initial input.
</ParamField>

<ParamField path="continuous" type="boolean" toc={true}>
  Whether to continuously collect input (restart after each result).
</ParamField>

<ParamField path="partial_results" type="boolean" toc={true}>
  Whether to send partial results as speech is recognized.
</ParamField>

<ParamField path="control_id" type="string" toc={true}>
  Unique identifier for this collect operation, used to control it later.
  If not provided, one is generated automatically.
</ParamField>
