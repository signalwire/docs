Handlers for Content-Encoding.

See: https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Content-Encoding

## Module Contents

### Classes

| Name | Description |
|------|-------------|
| [`BrotliDecoder`](#httpx-_decoders-BrotliDecoder) | Handle 'brotli' decoding. |
| [`ByteChunker`](#httpx-_decoders-ByteChunker) | Handles returning byte content in fixed-size chunks. |
| [`ContentDecoder`](#httpx-_decoders-ContentDecoder) | - |
| [`DeflateDecoder`](#httpx-_decoders-DeflateDecoder) | Handle 'deflate' decoding. |
| [`GZipDecoder`](#httpx-_decoders-GZipDecoder) | Handle 'gzip' decoding. |
| [`IdentityDecoder`](#httpx-_decoders-IdentityDecoder) | Handle unencoded data. |
| [`LineDecoder`](#httpx-_decoders-LineDecoder) | Handles incrementally reading lines from text. |
| [`MultiDecoder`](#httpx-_decoders-MultiDecoder) | Handle the case where multiple encodings have been applied. |
| [`TextChunker`](#httpx-_decoders-TextChunker) | Handles returning text content in fixed-size chunks. |
| [`TextDecoder`](#httpx-_decoders-TextDecoder) | Handles incrementally decoding bytes into text |
| [`ZStandardDecoder`](#httpx-_decoders-ZStandardDecoder) | Handle 'zstd' RFC 8878 decoding. |

Handle 'brotli' decoding.

Requires `pip install brotlipy`. See: https://brotlipy.readthedocs.io/
    or   `pip install brotli`. See https://github.com/google/brotli
Supports both 'brotlipy' and 'Brotli' packages since they share an import
name. The top branches are for 'brotlipy' and bottom branches for 'Brotli'
