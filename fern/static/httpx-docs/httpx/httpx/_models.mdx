<CodeBlock showLineNumbers={false} wordWrap={true}>

```python
httpx._models._parse_header_links(
    value: str
) -> list[dict[str, str]]
```

</CodeBlock>
</Anchor>

<Indent>

Returns a list of parsed link headers, for more info see:
https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Link
The generic syntax of those is:
Link: &lt; uri-reference &gt;; param1=value1; param2="value2"
So for instance:
Link; '&lt;http:/.../front.jpeg&gt;; type="image/jpeg",&lt;http://.../back.jpeg&gt;;'
would return
    [
        &#123;"url": "http:/.../front.jpeg", "type": "image/jpeg"&#125;,
        &#123;"url": "http://.../back.jpeg"&#125;,
    ]
:param value: HTTP Link entity-header field
:return: list of parsed link headers


</Indent>
