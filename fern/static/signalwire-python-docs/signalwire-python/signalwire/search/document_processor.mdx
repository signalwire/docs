<Indent>

Create chunks from pre-processed JSON content

This strategy expects content to be a JSON string with the following structure:
&#123;
    "chunks": [
        &#123;
            "chunk_id": "unique_id",
            "type": "content|toc",
            "content": "text content",
            "metadata": &#123;
                "url": "https://...",
                "section_number": 1,
                "related_toc": "toc_id",
                ...
            &#125;
        &#125;,
        ...
    ]
&#125;

**Parameters:**

<ParamField path="content" type="str">
JSON string containing pre-chunked content
</ParamField>

<ParamField path="filename" type="str">
Name of the source file
</ParamField>

<ParamField path="file_type" type="str">
Should be 'json'
</ParamField>

**Returns:** `List[Dict[str, Any]]`

List of chunk dictionaries formatted for the search index


</Indent>
