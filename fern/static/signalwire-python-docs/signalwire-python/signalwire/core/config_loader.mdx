Configuration loader with environment variable substitution.

Supports $&#123;VAR|default&#125; syntax for referencing environment variables
within JSON configuration files. This provides a clean pattern for
configuration across all SignalWire services.

<Anchor id="signalwire-core-config_loader-ConfigLoader-substitute_vars">

<CodeBlock showLineNumbers={false} wordWrap={true}>

```python
signalwire.core.config_loader.ConfigLoader.substitute_vars(
    value: typing.Any,
    max_depth: int = 10
) -> typing.Any
```

</CodeBlock>
</Anchor>

<Indent>

Recursively substitute environment variables in configuration values.

Supports $&#123;VAR|default&#125; syntax where:
- VAR is the environment variable name
- default is the fallback value if VAR is not set

**Parameters:**

<ParamField path="value" type="Any">
The value to process (can be string, dict, list, etc.)
</ParamField>

<ParamField path="max_depth" type="int" default="10">
Maximum recursion depth to prevent infinite loops
</ParamField>

**Returns:** `Any`

The value with all environment variables substituted


</Indent>
