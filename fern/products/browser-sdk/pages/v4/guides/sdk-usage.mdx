For full control over the call lifecycle and UI, use `@signalwire/js` directly instead of the prebuilt web components.

## Install

```bash
npm install @signalwire/js
```

## Connect and dial

```js
import { SignalWire } from "@signalwire/js";

const client = await SignalWire({
  token: "YOUR_SAT",
});

const call = await client.dial({
  to: "/private/support",
  rootElement: document.getElementById("root"),
  audio: true,
  video: true,
});

await call.start();

call.on("destroy", () => {
  console.log("Call ended");
});
```

## Hang up

```js
await call.hangup();
```

For the full API surface — events, devices, mute/unmute, screen sharing — see the [Reference](/js/v4/reference).
