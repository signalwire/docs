The Browser SDK ships a set of custom HTML elements (e.g. `<sw-call-widget>`, `<sw-audio-level>`) that you can drop into any page. They wrap the underlying SDK so you can build a calling UI without writing imperative JavaScript.

## Install

<CardGroup cols={2}>
  <Card title="npm" icon="brands npm" href="https://www.npmjs.com/package/@signalwire/web-components">
    @signalwire/web-components
  </Card>
</CardGroup>

```bash
npm install @signalwire/web-components
```

Or load from a CDN:

```html
<script type="module" src="https://unpkg.com/@signalwire/web-components"></script>
```

## Use

Import once, then use the elements anywhere in your markup.

```html
<script type="module">
  import "@signalwire/web-components";
</script>

<sw-call-provider token="YOUR_SAT">
  <sw-call-widget destination="/private/support"></sw-call-widget>
</sw-call-provider>
```

See the [Web Components reference](/js/v4/web-components) for the full list of elements and their attributes.
