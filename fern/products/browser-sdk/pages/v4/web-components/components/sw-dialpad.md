# `<sw-call-dialpad>`

DTMF bridge. Sends digits via `callState.sendDigits()`. Consumes `callStateContext`.

## Props

| prop | type | default | description |
|------|------|---------|-------------|
| `show-call-button` | `boolean` | `false` | Show dial/hangup button |
| `placeholder` | `string` | `'Enter number'` | Input placeholder text |
