```ts
hangup(): Promise<void>
```
Hangs up the call and releases all resources.

Sends a Verto `bye` to the server, transitions status to `'disconnecting'`,
then destroys the call. After this, the call instance is no longer usable.

## Example

```ts
await call.hangup();
```
