```ts
setLayout(layout, positions): Promise<void>
```
Sets the call layout and participant positions.

## Parameters

<ParamField path="layout" type="string" required>
  Layout name (must be one of [layouts](#layouts)).
</ParamField>

<ParamField path="positions" type="Record<string, VideoPosition>" required>
  Map of member IDs to [`VideoPosition`](../type-aliases/VideoPosition.mdx) values.
</ParamField>

## Returns

`Promise<void>`

## Throws

If the layout is not in the available [layouts](#layouts).

## Example

```ts
await call.setLayout('grid-responsive', {
  [participantId]: 'reserved-0',
});
```
