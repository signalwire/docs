```ts
setLayout(layout, positions): Promise<void>
```
Sets the call layout and participant positions.

## **Parameters**

<ParamField path="layout" type="string" required toc={true}>
  Layout name (must be one of [layouts](#layouts)).
</ParamField>

<ParamField path="positions" type="Record<string, VideoPosition>" required toc={true}>
  Map of member IDs to [`VideoPosition`](/docs/browser-sdk/v4/reference/type-aliases/video-position) values.
</ParamField>

## **Returns**

`Promise<void>`

## **Throws**

If the layout is not in the available [layouts](#layouts).

## **Examples**

```ts
await call.setLayout('grid-responsive', {
  [participantId]: 'reserved-0',
});
```
