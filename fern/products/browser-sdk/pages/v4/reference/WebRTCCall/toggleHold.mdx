```ts
toggleHold(): Promise<void>
```
Toggles the hold state of the call (pauses/resumes local media transmission).

Distinct from [Participant.toggleMute](/docs/browser-sdk/v4/reference/participant#togglemute) which mutes individual tracks.

## **Returns**

`Promise<void>`
