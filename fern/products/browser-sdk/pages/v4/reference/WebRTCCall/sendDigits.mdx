```ts
sendDigits(dtmf): Promise<void>
```
Sends DTMF digits on the call.

## Parameters

<ParamField path="dtmf" type="string" required>
  The digit string to send (e.g. `'1234#'`).
</ParamField>

## Returns

`Promise<void>`

## Example

```ts
await call.sendDigits('1234#');
```
