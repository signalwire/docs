Concrete WebRTC call implementation.

Manages the full lifecycle of a call including signaling, media streams,
participants, layout, and event routing. Created via [SignalWire.dial](../SignalWire/index.mdx#dial)
or received as an inbound call.

## Extends

- `Destroyable`

## Implements

- `CallManager`

## Constructors
