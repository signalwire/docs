```ts
get qualityLevel$(): Observable<QualityLevel>
```
Observable of simplified quality level (excellent/good/fair/poor/critical).

## **Examples**

```ts
call.qualityLevel$.subscribe((qualityLevel) => {
  console.log('qualityLevel:', qualityLevel);
});
```
