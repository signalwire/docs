```ts
answer(options?): void
```
Accepts an inbound call, optionally overriding media options for the answer.

## Parameters

<ParamField path="options" type="MediaOptions">
  Optional media constraints for the answer (audio/video).
</ParamField>

## Returns

`void`

## Example

```ts
// Accept with defaults
call.answer();

// Accept audio-only
call.answer({ audio: true, video: false });
```

 - [reject](#reject) to decline the call instead.
