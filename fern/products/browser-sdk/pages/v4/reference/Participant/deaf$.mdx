Observable indicating whether the participant is deafened.

## deaf

```ts
get deaf(): boolean
```

Whether the participant is deafened (incoming audio muted).

## **Examples**

```ts
participant.deaf$.subscribe((deaf) => {
  console.log('deaf:', deaf);
});
```
