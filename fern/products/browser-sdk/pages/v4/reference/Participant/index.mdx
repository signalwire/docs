Represents a participant in a call.

Provides observable state (audio/video mute, hand raise, volume, etc.)
and control methods for the participant. See [SelfParticipant](/docs/browser-sdk/v4/reference/self-participant) for
the local participant with additional device control.

- `Destroyable`

## **Extended by**

- [`SelfParticipant`](/docs/browser-sdk/v4/reference/self-participant)

## **Implements**
