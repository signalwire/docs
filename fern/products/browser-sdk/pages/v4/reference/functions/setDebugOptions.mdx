```ts
setDebugOptions(options): void
```
Configure debug options (e.g., `{ logWsTraffic: true }`).

## **Parameters**

<ParamField path="options" type="DebugOptions | null" toc={true}>
  Debug options to apply, or `null` to clear all overrides. See [`DebugOptions`](/docs/browser-sdk/v4/reference/interfaces/debug-options).
</ParamField>

## **Returns**

`void`

## **Examples**

### Enable WebSocket-traffic logging

```ts
import { setDebugOptions } from '@signalwire/js';

setDebugOptions({ logWsTraffic: true });
```

### Clear all debug overrides

```ts
setDebugOptions(null);
```
