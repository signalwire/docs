```ts
setLogLevel(level): void
```
Set the log level for the built-in logger.
Has no effect when a custom logger is set via `setLogger()`.

## **Parameters**

<ParamField path="level" type="LogLevel" required toc={true}>
  New minimum log level for the built-in logger.
</ParamField>

## **Returns**

`void`
