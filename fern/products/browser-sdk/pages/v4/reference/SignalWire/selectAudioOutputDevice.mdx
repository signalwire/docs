```ts
selectAudioOutputDevice(device): void
```
Sets the preferred audio output device.

## **Parameters**

<ParamField path="device" type="MediaDeviceInfo | null" required toc={true}>
  Device to select. Pass `null` to clear the current selection.
</ParamField>

## **Returns**

`void`

## **Examples**

```ts
client.selectAudioOutputDevice(device);
```
