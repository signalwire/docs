```ts
setStorageManager(storageManager): void
```
Injects a storage manager into the device controller for persistence.

## **Parameters**

<ParamField path="storageManager" type="StorageManager" required toc={true}>
  Storage manager implementation used to persist device and session state.
</ParamField>

## **Returns**

`void`

## **Examples**

```ts
client.setStorageManager(storageManager);
```
