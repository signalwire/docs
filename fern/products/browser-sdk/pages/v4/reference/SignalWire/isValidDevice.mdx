```ts
isValidDevice(deviceInfo): Promise<boolean>
```

<sub>[Go to Code](https://github.com/signalwire/browser-sdk/blob/29c1f6cf3278bb4128a214178e02793719436b60/packages/main/src/clients/SignalWire.ts#L1259)</sub>

Checks whether a device is still available and usable.

## Parameters

<ParamField path="deviceInfo" type="MediaDeviceInfo | null">
  The device to validate, or `null`.
</ParamField>

## Returns

`Promise<boolean>`

`true` if the device is valid and available. Returns `false` for `null`, audio output devices, or unavailable devices.
