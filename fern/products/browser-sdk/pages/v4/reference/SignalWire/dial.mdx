```ts
dial(destination, options?): Promise<Call>
```

<sub>[Go to Code](https://github.com/signalwire/browser-sdk/blob/29c1f6cf3278bb4128a214178e02793719436b60/packages/main/src/clients/SignalWire.ts#L1053)</sub>

Places an outbound call to the given destination.

Waits for authentication before dialing. Media options are merged from
saved preferences, destination query parameters (e.g. `?channel=video`),
and the provided `options` (highest priority).

Returns a [Call](../interfaces/Call.mdx) in `'ringing'` state. Subscribe to [Call.status$](../interfaces/Call.mdx#status-1)
to track progression through `'connected'` → `'disconnected'`.

## Parameters

<ParamField path="destination" type="string | Address" required>
  Address URI string (e.g. `'/public/my-room'`) or [`Address`](../Address/index.mdx) instance.
</ParamField>

<ParamField path="options" type="DialOptions">
  Media and dial options (audio/video, device constraints). Overrides defaults.
</ParamField>

## Returns

`Promise<Call>`

The created [`Call`](../interfaces/Call.mdx) instance.

## Throws

If authentication is not complete or call creation fails.

## Example

```ts
const call = await client.dial('/public/conference', {
  audio: true,
  video: true,
});
call.status$.subscribe(status => console.log('Call:', status));
```
