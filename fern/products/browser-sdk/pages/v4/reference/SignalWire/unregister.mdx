```ts
unregister(): Promise<void>
```
Unregisters the user, going offline for inbound calls.

The WebSocket connection remains open; use [disconnect](#disconnect) to fully close it.

## **Returns**

`Promise<void>`

## **Examples**

```ts
await client.unregister();
```
