Represents a contact or room in the directory.

Provides identity metadata, conversation history, text messaging,
and activity state for an address entry.

## **Extends**

- `Destroyable`

## **Constructors**

### Constructor

```ts
new Address(addressId, conversationManager, addressProvider): Address
```

[Go to Code](https://github.com/signalwire/browser-sdk/blob/main/packages/main/src/core/entities/Address.ts#L67)

#### Parameters

<ParamField path="addressId" type="string" required toc={true}>
  Server-assigned ID of the address.
</ParamField>

<ParamField path="conversationManager" type="ConversationsProvider" required toc={true}>
  Provider used to load and manage conversations for this address.
</ParamField>

<ParamField path="addressProvider" type="AddressProvider<Address>" required toc={true}>
  Provider used to load directory entries for this address.
</ParamField>

## **Properties**

<ParamField path="_destroyed$" type="Subject<void>" toc={true}>

<ParamField path="history$" type="Observable<EntityCollectionTransformed<GetConversationMessageResponse, AddressHistory<Address>> | undefined>" toc={true}>

<ParamField path="subjects" type="Subject<unknown>[]" toc={true}>
  Internal list of RxJS subjects tracked for cleanup on destroy.

<ParamField path="subscriptions" type="Subscription[]" toc={true}>
  Internal list of active subscriptions tracked for cleanup on destroy.
