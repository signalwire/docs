Whether to filter out UDP-based ICE servers.

## **Parameters**

<ParamField path="value" type="boolean" toc={true}>
  If `true`, ICE servers using UDP transport are excluded.
</ParamField>

## **Examples**

```ts
console.log(client.preferences.disableUdpIceServers);
```
