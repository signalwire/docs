```ts
get keyframeBurstWindow(): number
set keyframeBurstWindow(value): void
```
Keyframe burst window duration in milliseconds — the window over which video
keyframe requests are counted before a
[`keyframeCooldown`](/docs/browser-sdk/v4/reference/client-preferences/keyframe-cooldown)
applies. See [Keyframe recovery](/docs/browser-sdk/v4/guides/client-preferences#keyframe-recovery)
for the full burst-with-cooldown mechanism. Default: `3000` (3 seconds).

## **Parameters**

<ParamField path="value" type="number" toc={true}>
  Keyframe burst window duration (ms).
</ParamField>

## **Examples**

```ts
console.log(client.preferences.keyframeBurstWindow);
```
