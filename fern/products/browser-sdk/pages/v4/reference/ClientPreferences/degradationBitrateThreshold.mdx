## **Parameters**

<ParamField path="value" type="number" required toc={true}>
  Outgoing bitrate (kbps) below which video is automatically disabled.
</ParamField>

## **Examples**

```ts
// Read
console.log(client.preferences.degradationBitrateThreshold);

// Write
client.preferences.degradationBitrateThreshold = 150;
```
