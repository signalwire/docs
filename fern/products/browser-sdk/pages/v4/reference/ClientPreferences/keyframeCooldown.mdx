```ts
get keyframeCooldown(): number
set keyframeCooldown(value): void
```
Cooldown period in milliseconds after the keyframe burst limit is reached —
how long the SDK stops sending video keyframe requests once
[`keyframeMaxBurst`](/docs/browser-sdk/v4/reference/client-preferences/keyframe-max-burst)
is hit within a
[`keyframeBurstWindow`](/docs/browser-sdk/v4/reference/client-preferences/keyframe-burst-window).
See [Keyframe recovery](/docs/browser-sdk/v4/guides/client-preferences#keyframe-recovery)
for the full mechanism. Default: `10000` (10 seconds).

## **Parameters**

<ParamField path="value" type="number" toc={true}>
  Cooldown period (ms) after the keyframe burst limit is reached.
</ParamField>

## **Examples**

```ts
console.log(client.preferences.keyframeCooldown);
```
