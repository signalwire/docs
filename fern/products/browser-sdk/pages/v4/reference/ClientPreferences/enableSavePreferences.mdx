```ts
enableSavePreferences(storage): void
```
Enables persistence of preferences to storage.
Loads any previously saved preferences and syncs future changes.

## **Parameters**

<ParamField path="storage" type="StorageManager" required toc={true}>
  Storage manager used to persist user preferences across sessions.
</ParamField>
