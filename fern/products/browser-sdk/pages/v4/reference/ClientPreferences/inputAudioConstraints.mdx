```ts
get inputAudioConstraints(): MediaTrackConstraints | undefined

```
Default audio input track constraints.

## **Parameters**

<ParamField path="value" type="MediaTrackConstraints | undefined" toc={true}>
  Constraints applied when acquiring audio input. Pass `undefined` to clear. See [`MediaTrackConstraints`](https://developer.mozilla.org/en-US/docs/Web/API/MediaTrackConstraints).
</ParamField>

## **Examples**

```ts
console.log(client.preferences.inputAudioConstraints);
```
