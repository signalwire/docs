```ts
addVideoInputDevice(__namedParameters?): Promise<void>
```
Adds or replaces the primary video input device with optional constraints or stream.

## **Parameters**

<ParamField path="__namedParameters" type="{ constraints?: MediaTrackConstraints; stream?: MediaStream; }" toc={true}>
  Capture options: either explicit `constraints` or a pre-acquired `stream`.
</ParamField>

## **Returns**

`Promise<void>`

## **Examples**

```ts
await selfParticipant.addVideoInputDevice(__namedParameters);
```
