```ts
selectVideoInputDevice(device, options?): void
```
Selects the video input device for future calls. Optionally saves as a preference.

## **Parameters**

<ParamField path="device" type="MediaDeviceInfo" required toc={true}>
  Device to select. Pass `null` to clear the current selection.
</ParamField>

<ParamField path="options" type="SelectDeviceOptions" toc={true}>
  Selection options, including whether to persist the choice.
</ParamField>

## **Returns**

`void`

## **Examples**

```ts
selfParticipant.selectVideoInputDevice(device, options);
```
