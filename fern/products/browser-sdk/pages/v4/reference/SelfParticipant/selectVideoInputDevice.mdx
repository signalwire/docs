```ts
selectVideoInputDevice(device, options?): void
```
Selects the video input device for future calls. Optionally saves as a preference.

## **Parameters**

<ParamField path="device" type="MediaDeviceInfo" required toc={true}>
  Device to select. See [`MediaDeviceInfo`](https://developer.mozilla.org/en-US/docs/Web/API/MediaDeviceInfo).
</ParamField>

<ParamField path="options" type="SelectDeviceOptions" toc={true}>
  Selection options, including whether to persist the choice. See [`SelectDeviceOptions`](/docs/browser-sdk/v4/reference/interfaces/select-device-options).
</ParamField>

## **Returns**

`void`

## **Examples**

```ts
selfParticipant.selectVideoInputDevice(device, options);
```
