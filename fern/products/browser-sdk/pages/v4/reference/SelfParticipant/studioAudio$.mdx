## studioAudio

```ts
get studioAudio(): boolean
```

Whether studio audio (raw/unprocessed audio) mode is currently enabled.

## **Examples**

```ts
selfParticipant.studioAudio$.subscribe((studioAudio) => {
  console.log('studioAudio:', studioAudio);
```
