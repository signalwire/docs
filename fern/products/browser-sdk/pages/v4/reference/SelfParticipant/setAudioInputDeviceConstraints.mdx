```ts
setAudioInputDeviceConstraints(constraints): Promise<void>
```
Updates the audio input track constraints for the active call.

## **Parameters**

<ParamField path="constraints" type="MediaTrackConstraints" required toc={true}>
  Media-track constraints to apply to the current audio input. See [`MediaTrackConstraints`](https://developer.mozilla.org/en-US/docs/Web/API/MediaTrackConstraints).
</ParamField>

## **Returns**

`Promise<void>`

## **Examples**

```ts
await selfParticipant.setAudioInputDeviceConstraints(constraints);
```
