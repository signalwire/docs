Member-level capabilities for self or other members

## **Properties**

<ParamField path="audioFlags" type="boolean" required toc={true}>
  Whether audio-related flags (mute, deaf) can be changed for this member.
</ParamField>

<ParamField path="deaf" type="OnOffCapability" required toc={true}>
  Capability to deafen or un-deafen this member.
</ParamField>

<ParamField path="meta" type="boolean" required toc={true}>
  Whether arbitrary metadata can be set on this member.
</ParamField>

<ParamField path="microphoneSensitivity" type="boolean" required toc={true}>
  Whether microphone sensitivity can be adjusted for this member.
</ParamField>

<ParamField path="microphoneVolume" type="boolean" required toc={true}>
  Whether microphone volume can be adjusted for this member.
</ParamField>

<ParamField path="muteAudio" type="OnOffCapability" required toc={true}>
  Capability to mute or unmute this member's audio.
</ParamField>

<ParamField path="muteVideo" type="OnOffCapability" required toc={true}>
  Capability to mute or unmute this member's video.
</ParamField>

<ParamField path="position" type="boolean" required toc={true}>
  Whether this member's position in the layout can be changed.
</ParamField>

<ParamField path="raisehand" type="OnOffCapability" required toc={true}>
  Capability to raise or lower this member's hand.
</ParamField>

<ParamField path="remove" type="boolean" required toc={true}>
  Whether this member can be removed from the call.
</ParamField>

<ParamField path="speakerVolume" type="boolean" required toc={true}>
  Whether speaker volume can be adjusted for this member.
</ParamField>
