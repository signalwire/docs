SAT claims returned by /api/fabric/subscriber/info.

## **Properties**

<ParamField path="cnf" type="object" toc={true}>
  Confirmation claim binding the token to a key.
</ParamField>

<ParamField path="cnf.jkt" type="string" toc={true}>
  DPoP key thumbprint binding the SAT to a specific key.
</ParamField>

<ParamField path="expires_at" type="number" toc={true}>
  Token expiry timestamp in seconds since epoch.
</ParamField>

<ParamField path="scope" type="string[]" toc={true}>
  Token scopes (e.g., ["sat:refresh"]).
</ParamField>
