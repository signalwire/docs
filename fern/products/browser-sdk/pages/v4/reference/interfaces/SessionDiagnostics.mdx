Structured diagnostic bundle for a session.

## **Properties**

<ParamField path="calls" type="readonly CallDiagnosticSummary[]" required toc={true}>
  Quality summary per call.
</ParamField>

<ParamField path="capabilities" type="PlatformCapabilities" required toc={true}>
  Platform capabilities detected at construction time.
</ParamField>

<ParamField path="deviceChanges" type="readonly DeviceRecoveryEvent[]" required toc={true}>
  Device changes that occurred during the session.
</ParamField>

<ParamField path="devices" type="object" required toc={true}>
  Current device list snapshot.
</ParamField>

<ParamField path="devices.audioInput" type="readonly MediaDeviceInfo[]" required toc={true}>
  Snapshot of audio input devices visible to the session.
</ParamField>

<ParamField path="devices.audioOutput" type="readonly MediaDeviceInfo[]" required toc={true}>
  Snapshot of audio output devices visible to the session.
</ParamField>

<ParamField path="devices.videoInput" type="readonly MediaDeviceInfo[]" required toc={true}>
  Snapshot of video input devices visible to the session.
</ParamField>

<ParamField path="events" type="readonly DiagnosticEvent[]" required toc={true}>
  Timeline of significant events during the session.
</ParamField>

<ParamField path="sdkVersion" type="string" required toc={true}>
  SDK version.
</ParamField>

<ParamField path="userAgent" type="string" required toc={true}>
  Browser/platform user agent string.
</ParamField>
