Options for constructing a [SignalWire](../SignalWire/index.mdx).

## Properties

<ParamField path="debug" type="DebugOptions">
  Debug options for verbose SDK diagnostics (e.g., `{ logWsTraffic: true }`).
</ParamField>

<ParamField path="logger" type="SDKLogger | null">
  Custom logger implementation. Must implement the [SDKLogger](SDKLogger.mdx) interface. Pass `null` to restore the built-in logger. **Note:** Logger configuration is global — setting it on one instance affects all instances.
</ParamField>

<ParamField path="logLevel" type="LogLevel">
  Log level for the built-in logger. Default: `'warn'`. Set to `'debug'` for verbose SDK output. Has no effect when a custom `logger` is provided. **Note:** Logger configuration is global — setting it on one instance affects all instances.
</ParamField>

<ParamField path="persistSession" type="boolean">
  Persist the session across page reloads. When `true`, credential, authorization state, and protocol are stored in `localStorage` (survives reload). The DPoP key pair is persisted in IndexedDB. On reload, the SDK restores the session from cache without calling `credentialProvider.authenticate()`. When `false` (default), session data lives in `sessionStorage` and is lost on reload. Call [destroy()](../SignalWire/index.mdx) to clear all persisted state (explicit logout).
</ParamField>

<ParamField path="reconnectAttachedCalls" type="boolean">
  Whether to reconnect to previously attached calls.
</ParamField>

<ParamField path="savePreferences" type="boolean">
  Whether to save preferences.
</ParamField>

<ParamField path="skipConnection" type="boolean">
  Skip automatic WebSocket connection on construction.
</ParamField>

<ParamField path="skipDeviceMonitoring" type="boolean">
  Skip monitoring media device changes.
</ParamField>

<ParamField path="skipRegister" type="boolean">
  Skip automatic subscriber registration on construction.
</ParamField>

<ParamField path="storageImplementation" type="Storage">
  Custom storage implementation for persistence.
</ParamField>

<ParamField path="webRTCApiProvider" type="WebRTCApiProvider">
  Custom WebRTC API provider
</ParamField>

<ParamField path="webSocketConstructor" type="NodeSocketAdapter | WebSocketAdapter">
  Custom WebSocket constructor
</ParamField>
