Options controlling which media tracks to send and receive.

## Extended by

- [`DialOptions`](DialOptions.mdx)
- [`CallOptions`](CallOptions.mdx)

## Properties

<ParamField path="audio" type="boolean">
  Enable audio input. Defaults to `true` when not specified.
</ParamField>

<ParamField path="inputAudioDeviceConstraints" type="MediaTrackConstraints">
  Custom constraints for the audio input track.
</ParamField>

<ParamField path="inputAudioStream" type="MediaStream">
  Pre-existing audio stream to use instead of `getUserMedia`.
</ParamField>

<ParamField path="inputVideoDeviceConstraints" type="MediaTrackConstraints">
  Custom constraints for the video input track.
</ParamField>

<ParamField path="inputVideoStream" type="MediaStream">
  Pre-existing video stream to use instead of `getUserMedia`.
</ParamField>

<ParamField path="receiveAudio" type="boolean">
  Whether to receive remote audio.
</ParamField>

<ParamField path="receiveVideo" type="boolean">
  Whether to receive remote video.
</ParamField>

<ParamField path="video" type="boolean">
  Enable video input. Defaults to `false` when not specified.
</ParamField>
