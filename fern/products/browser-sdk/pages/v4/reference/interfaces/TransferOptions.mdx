Options accepted by [`WebRTCCall.transfer`](/docs/browser-sdk/v4/reference/webrtc-call/transfer). The required `destination` is any dialable target the server's dial plan understands.

## **Properties**

<ParamField path="destination" type="string" required toc={true}>
  Target destination URI (address or SIP URI) to transfer the call to.
</ParamField>
