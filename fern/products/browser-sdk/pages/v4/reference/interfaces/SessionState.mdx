Extended session interface that adds call management and authentication
state on top of the narrow ClientSession contract.

Accessible via `client.session`. Call and CallFactory continue to depend
only on the narrow ClientSession interface.

## Extends

- `ClientSession`

## Properties

<ParamField path="authenticated" type="boolean" required>
  Current authentication state. Returns `true` if the session is currently authenticated.
</ParamField>

<ParamField path="authenticated$" type="Observable<boolean>" required>
  Observable that emits `true` once the session has been authenticated, and `false` after disconnect.
</ParamField>

<ParamField path="calls" type="Call[]" required>
  Current snapshot of all active calls.
</ParamField>

<ParamField path="calls$" type="Observable<Call[]>" required>
  Observable stream of all currently active calls (both inbound and outbound).
</ParamField>

<ParamField path="incomingCalls" type="Call[]" required>
  Current snapshot of active inbound calls.
</ParamField>

<ParamField path="incomingCalls$" type="Observable<Call[]>" required>
  Observable stream of currently active inbound calls. Filters `calls$` to only include calls with `direction === 'inbound'`.
</ParamField>

<Accordion title="Inherited from ClientSession">
  <ParamField path="iceServers" type="RTCIceServer[] | undefined" required>

  </ParamField>

  <ParamField path="signalingEvent$" type="Observable<Record<string, unknown>>" required>
    Observable stream of incoming signaling events Used by Call to listen for call-related events from the server
  </ParamField>
</Accordion>
