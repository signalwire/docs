Results of a preflight connectivity test.

## **Properties**

<ParamField path="bandwidth" type="{ downloadKbps: number; uploadKbps: number; } | null" toc={true}>
  Bandwidth estimation (null if skipMediaTest).
</ParamField>

<ParamField path="connectivity" type="object" required toc={true}>
  ICE/TURN connectivity.
</ParamField>

<ParamField path="connectivity.rttMs" type="number" required toc={true}>
  RTT to media server in ms.
</ParamField>

<ParamField path="connectivity.stunReachable" type="boolean" required toc={true}>
  Whether STUN servers are reachable.
</ParamField>

<ParamField path="connectivity.turnReachable" type="boolean" required toc={true}>
  Whether TURN servers are reachable.
</ParamField>

<ParamField path='connectivity.type' type='"failed" | "direct" | "relay"' required toc={true}>
  'direct' = host/srflx worked, 'relay' = only TURN relay, 'failed' = nothing.
</ParamField>

<ParamField path="devices" type="object" required toc={true}>
  Device test results.
</ParamField>

<ParamField path="devices.audioInput" type="object" required toc={true}>
  Audio input portion of the preflight result.
</ParamField>

<ParamField path="devices.audioInput.device" type="MediaDeviceInfo | null" toc={true}>
  Audio input device used for the test, or `null` if none was available.
</ParamField>

<ParamField path="devices.audioInput.working" type="boolean" required toc={true}>
  Whether the audio input device produced a usable signal.
</ParamField>

<ParamField path="devices.audioOutput" type="object" required toc={true}>
  Audio output portion of the preflight result.
</ParamField>

<ParamField path="devices.audioOutput.available" type="boolean" required toc={true}>
  Whether an audio output device was available for the test.
</ParamField>

<ParamField path="devices.audioOutput.device" type="MediaDeviceInfo | null" toc={true}>
  Audio output device used for the test, or `null` if none was available.
</ParamField>

<ParamField path="devices.videoInput" type="object" required toc={true}>
  Video input portion of the preflight result.
</ParamField>

<ParamField path="devices.videoInput.device" type="MediaDeviceInfo | null" toc={true}>
  Video input device used for the test, or `null` if none was available.
</ParamField>

<ParamField path="devices.videoInput.working" type="boolean" required toc={true}>
  Whether the video input device produced a usable signal.
</ParamField>

<ParamField path="ok" type="boolean" required toc={true}>
  Overall pass/fail.
</ParamField>

<ParamField path="signaling" type="object" required toc={true}>
  Signaling server reachability.
</ParamField>

<ParamField path="signaling.reachable" type="boolean" required toc={true}>
  Whether the signaling endpoint was reachable.
</ParamField>

<ParamField path="signaling.rttMs" type="number" required toc={true}>
  Measured signaling round-trip time in milliseconds.
</ParamField>

<ParamField path="warnings" type="readonly string[]" required toc={true}>
  Human-readable warnings.
</ParamField>
