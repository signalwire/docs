Represents an on/off capability state
Both `on` and `off` can be true if the parent permission grants both

## **Properties**

<ParamField path="off" type="boolean" required toc={true}>
  Whether the local participant can turn this capability off.
</ParamField>

<ParamField path="on" type="boolean" required toc={true}>
  Whether the local participant can turn this capability on.
</ParamField>
