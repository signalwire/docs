Shape of an outgoing JSON-RPC 2.0 request. Used as the input to [`WebRTCCall.execute`](/docs/browser-sdk/v4/reference/webrtc-call/execute) for direct RPC sends.

## **Type Parameters**

| Type Parameter | Default type |
| ------ | ------ |
| `TParams` | `unknown` |

## **Properties**

<ParamField path="id" type="string" required toc={true}>
  Request ID, echoed back in the matching response.
</ParamField>

<ParamField path='jsonrpc' type='"2.0"' required toc={true}>
  JSON-RPC protocol version. Always `"2.0"`.
</ParamField>

<ParamField path="method" type="string" required toc={true}>
  RPC method name.
</ParamField>

<ParamField path="params" type="TParams" toc={true}>
  Method-specific parameters.
</ParamField>
