Result of a media permissions request (Section 5.10).

## **Properties**

<ParamField path="audio" type="boolean" required toc={true}>
  Whether audio permission was granted.
</ParamField>

<ParamField path="selectedAudioDevice" type="MediaDeviceInfo" toc={true}>
  The audio device the user selected in the browser picker, if any.
</ParamField>

<ParamField path="selectedVideoDevice" type="MediaDeviceInfo" toc={true}>
  The video device the user selected in the browser picker, if any.
</ParamField>

<ParamField path="video" type="boolean" required toc={true}>
  Whether video permission was granted.
</ParamField>
