## **Properties**

<ParamField path="audio" type="object" required toc={true}>
  Audio-stream metrics for the most recent sample window.
</ParamField>

<ParamField path="audio.jitter" type="number" required toc={true}>
  Audio jitter in seconds.
</ParamField>

<ParamField path="audio.packetsLost" type="number" required toc={true}>
  Audio packets lost since the start of the call.
</ParamField>

<ParamField path="audio.packetsReceived" type="number" required toc={true}>
  Audio packets received since the start of the call.
</ParamField>

<ParamField path="availableOutgoingBitrate" type="number" toc={true}>
  Estimated available outgoing bitrate in bits per second.
</ParamField>

<ParamField path="roundTripTime" type="number" required toc={true}>
  Current round-trip time in seconds.
</ParamField>

<ParamField path="timestamp" type="number" required toc={true}>
  Wall-clock timestamp (ms since epoch) of this sample.
</ParamField>

<ParamField path="video" type="object" required toc={true}>
  Video-stream metrics for the most recent sample window.
</ParamField>

<ParamField path="video.packetsLost" type="number" required toc={true}>
  Video packets lost since the start of the call.
</ParamField>

<ParamField path="video.packetsReceived" type="number" required toc={true}>
  Video packets received since the start of the call.
</ParamField>
