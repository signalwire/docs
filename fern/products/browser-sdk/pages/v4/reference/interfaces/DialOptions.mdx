Options for [SignalWire.dial](../SignalWire/index.mdx). Extends [MediaOptions](MediaOptions.mdx) with dial-specific settings.

## **Extends**

- [`MediaOptions`](MediaOptions.mdx)

## **Properties**

<ParamField path="nodeId" type="string" toc={true}>
  Optional node ID for routing the call
</ParamField>

<ParamField path="preferredAudioCodecs" type="string[]" toc={true}>
  Preferred audio codecs for this call (overrides global preferences).
</ParamField>

<ParamField path="preferredVideoCodecs" type="string[]" toc={true}>
  Preferred video codecs for this call (overrides global preferences).
</ParamField>

<ParamField path="stereo" type="boolean" toc={true}>
  Enable stereo Opus for this call (overrides global preferences).
</ParamField>

<ParamField path="userVariables" type="Record<string, unknown>" toc={true}>
  Custom variables sent with the Verto invite. Merged with `client.preferences.userVariables` and any query-string variables on the destination URI; values here take precedence.
</ParamField>
