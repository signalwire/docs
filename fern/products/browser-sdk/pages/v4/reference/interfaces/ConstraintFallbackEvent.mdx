Event emitted when getUserMedia falls back to looser constraints.

## Properties

<ParamField path="actualDevice" type="MediaDeviceInfo | null" required>
  The device that the browser actually provided.
</ParamField>

<ParamField path="fallbackLevel" type="exact | preferred | default" required>
  The constraint level that succeeded.
</ParamField>

<ParamField path="kind" type="audioinput | videoinput" required>
  The kind of input device.
</ParamField>

<ParamField path="requestedDevice" type="MediaDeviceInfo | null" required>
  The device that was originally requested.
</ParamField>
