Event emitted when the SDK auto-switches a device.

## **Properties**

<ParamField path="kind" type="audioinput | audiooutput | videoinput" required toc={true}>
  The kind of device that was switched.
</ParamField>

<ParamField path="newDevice" type="MediaDeviceInfo | null" required toc={true}>
  The device that was selected as a replacement (null if none available).
</ParamField>

<ParamField path="previousDevice" type="MediaDeviceInfo | null" required toc={true}>
  The device that was previously selected (null if none).
</ParamField>

<ParamField path="reason" type="device_disconnected | device_reconnected | session_restored | fallback_to_default | default_changed | ambiguous_match" required toc={true}>
  The reason for the device switch.
</ParamField>
