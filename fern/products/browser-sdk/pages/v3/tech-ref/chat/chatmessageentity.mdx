---
title: "ChatMessageEntity"
---

An object representing a Chat Message with only the state properties of [`ChatMessage`](/browser-sdk/chat/message).

## Properties

### content

- `Readonly` **content**: `any`

The content of this message. This can be any JSON-serializable object or value.

---

### id

- `Readonly` **id**: `string`

The id of this message.

---

### member

- `Readonly` **member**: [`ChatMember`](/browser-sdk/chat/member)

The member which sent this message.

---

### meta

- `Readonly` **meta?**: `any`

Any metadata associated with this message.

---

### publishedAt

- `Readonly` **publishedAt**: `Date`

The date and time at which this message was published.
