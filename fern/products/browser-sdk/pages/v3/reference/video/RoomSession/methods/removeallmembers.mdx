### removeAllMembers

- **removeAllMembers**(): `Promise<void>`

Removes all the members from this room session. The room session will end.

#### Returns

`Promise<void>`

#### Permissions

- `room.member.remove`: to remove a remote member.

You need to specify the permissions when [creating the Video Room Token](/docs/apis/rest/video/room-tokens/create-room-token) on the server side.

#### Example

```typescript
await roomSession.removeAllMembers();
```
