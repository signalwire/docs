Represents a specific stream of a room session. This is an RTMP stream of the
audio/video content of the room, which will be sent to an external party (e.g.,
to YouTube).

You can start a stream with [RoomSession.startStream](/docs/browser-sdk/v3/js/reference/video/room-session/start-stream).

## Properties

<ParamField path="duration" type="number" toc={true}>
  Total seconds of time spent streaming, if available. This is equal to (`endedAt` - `startedAt`).
</ParamField>

<ParamField path="endedAt" type="Date" toc={true}>
  End time, if available.
</ParamField>

<ParamField path="id" type="string" required={true} toc={true}>
  The unique id of this stream.
</ParamField>

<ParamField path="roomSessionId" type="string" required={true} toc={true}>
  The id of the room session associated to this stream.
</ParamField>

<ParamField path="startedAt" type="Date" required={true} toc={true}>
  Start time, if available.
</ParamField>

<ParamField path="state" type='"streaming" | "completed"' required={true} toc={true}>
  Current state of the stream.
</ParamField>

<ParamField path="url" type="string" required={true} toc={true}>
  The RTMP URL of the stream.
</ParamField>

## Methods

### stop

- **stop**(): `Promise<void>`

Stops the stream.

#### Returns

`Promise<void>`

#### Example

```javascript
await stream.stop();
```
