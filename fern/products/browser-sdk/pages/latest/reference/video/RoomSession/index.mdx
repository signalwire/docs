A RoomSession allows you to start and control video sessions.

For example, the following code joins a video session and listens for new members joining:

```javascript
const roomSession = new SignalWire.Video.RoomSession({
  token: "<YourRoomToken>",
  rootElement: document.getElementById("myVideoElement"),
});

roomSession.on("member.joined", (e) => {
  console.log(`${e.member.name} joined`);
});

roomSession.join();
```

### Obtaining a token

The room token is obtained from the REST API from your backend server. 

<EndpointRequestSnippet endpoint="POST /api/video/room_tokens" />

Please refer to the [Simple Video Demo](/browser-sdk/guides/build-a-video-calling-application-with-the-browser-sdk) guide to learn how to obtain Video Room tokens.

## Constructor

Creates a new RoomSession. Note that the room will not be joined until [join](#join) has been called.

```javascript
const roomSession = new SignalWire.Video.RoomSession({
  token: "<YourRoomToken>",
  // ...
});
```

### Parameters

The RoomSession constructor accepts the following parameters:

<Indent>
  <ParamField path="token" type="string" required={true}>
    SignalWire video room token (get one from the [REST APIs](/apis/video/room-tokens/create-room-token))
  </ParamField>
  <ParamField path="rootElement" type="HTMLElement">
    HTML element in which to display the video stream.
  </ParamField>
  <ParamField path="applyLocalVideoOverlay" type="boolean" default="true">
    Whether to apply the local-overlay on top of your video.
  </ParamField>
  <ParamField path="iceServers" type="RTCIceServer[]">
    List of [ICE servers](https://developer.mozilla.org/en-US/docs/Web/API/RTCIceServer).
  </ParamField>
  <ParamField path="localStream" type="MediaStream">
    A custom [media stream](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream) to use in place of a camera.
  </ParamField>
  <ParamField path="logLevel" type="'trace' | 'debug' | 'info' | 'warn' | 'error' | 'silent'">
    Logging level.
  </ParamField>
  <ParamField path="speakerId" type="string">
    Id of the speaker device to use for audio output. If undefined, picks a default speaker.
  </ParamField>
  <ParamField path="stopCameraWhileMuted" type="boolean" default="true">
    Whether to stop the camera when the member is muted.
  </ParamField>
  <ParamField path="stopMicrophoneWhileMuted" type="boolean" default="true">
    Whether to stop the microphone when the member is muted.
  </ParamField>
  <ParamField path="audio" type="boolean | MediaTrackConstraints" default="true" deprecated={true}>
    Audio constraints to use when joining the room. _Deprecated: please use the equivalent parameter in [join](methods/join)._
  </ParamField>
  <ParamField path="video" type="boolean | MediaTrackConstraints" default="true" deprecated={true}>
    Video constraints to use when joining the room. _Deprecated: please use the equivalent parameter in [join](methods/join)._
  </ParamField>
</Indent>

## Properties

The RoomSession class has the following properties:

<Indent>
  <ParamField path="active" type="boolean">
    Whether the connection is currently active. _Read-only._
  </ParamField>

  <ParamField path="cameraId" type="string | null">
    The id of the video device, or null if not available. _Read-only._
  </ParamField>

  <ParamField path="cameraLabel" type="string | null">
    The label of the video device, or null if not available. _Read-only._
  </ParamField>

  <ParamField path="deviceList" type="RoomSessionDevice[]">
    Contains any additional devices added via [addCamera](methods/addcamera), [addMicrophone](methods/addmicrophone), or [addDevice](methods/adddevice). _Read-only._
  </ParamField>

  <ParamField path="interactivityMode" type="'audience' | 'member'">
    The current interactivity mode (_member_ or _audience_) for the local member.

    Member participants are allowed to transmit their own audio and/or video to the
    rest of the room (as in a typical video conference), while audience participants
    can only view and/or listen. See [join](methods/join). _Read-only._
  </ParamField>

  <ParamField path="localAudioTrack" type="MediaStreamTrack | null">
    Provides access to the local audio [MediaStreamTrack](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamTrack). _Read-only._
  </ParamField>

  <ParamField path="localStream" type="MediaStream | undefined">
    Provides access to the local [MediaStream](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream). _Read-only._
  </ParamField>

  <ParamField path="localVideoTrack" type="MediaStreamTrack | null">
    Provides access to the local video [MediaStreamTrack](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamTrack). _Read-only._
  </ParamField>

  <ParamField path="localOverlay" type="LocalOverlay">
    Provides access to the local video overlay. Use this for example to mirror the local video. _Read-only._
  </ParamField>

  <ParamField path="memberId" type="string">
    The id of the current member within the room. _Read-only._
  </ParamField>

  <ParamField path="microphoneId" type="string | null">
    The id of the audio input device, or null if not available. _Read-only._
  </ParamField>

  <ParamField path="microphoneLabel" type="string | null">
    The label of the audio input device, or null if not available. _Read-only._
  </ParamField>

  <ParamField path="permissions" type="string[]">
    The list of permissions currently available to the local member. _Read-only._
  </ParamField>

  <ParamField path="previewUrl" type="string">
    If the Room has been created with the property `enable_room_previews` set to `true`, this field contains the URL to the room preview. _Read-only._
  </ParamField>

  <ParamField path="remoteStream" type="MediaStream | undefined">
    Provides access to the remote [MediaStream](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream). _Read-only._
  </ParamField>

  <ParamField path="roomId" type="string">
    The unique identifier for the room. _Read-only._
  </ParamField>

  <ParamField path="roomSessionId" type="string">
    The unique identifier for the room session. _Read-only._
  </ParamField>

  <ParamField path="screenShareList" type="RoomSessionScreenShare[]">
    Contains any local screen shares added to the room via [startScreenShare](methods/startscreenshare). _Read-only._
  </ParamField>
</Indent>
