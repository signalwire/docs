### layout.changed

- **layout.changed**(`event`)

The layout of the room has changed. This event is not limited to changes
associated to the grid layout of the room: it also includes for example changes
in the position of the participants within the grid of the room.

<Accordion title="Properties">

<ParamField path="event" type="object">
  Event object
</ParamField>
<Indent>
  <ParamField path="event.room_session_id" type="string">
    Id of the room session.
  </ParamField>
  <ParamField path="event.room_id" type="string">
    Id of the room.
  </ParamField>
  <ParamField path="event.layout" type="object">
    Information on the current layout.
  </ParamField>
  <Indent>
    <ParamField path="event.layout.name" type="string">
      Name of the current layout.
    </ParamField>
    <ParamField path="event.layout.room_session_id" type="string">
      Id of the room session.
    </ParamField>
    <ParamField path="event.layout.room_id" type="string">
      Id of the room.
    </ParamField>
    <ParamField path="event.layout.layers" type="VideoLayoutLayer[]">
      Array containing information on all the layers in the current layout.

  </Indent>
</Indent>

</Accordion>

### media.connected

- **media.connected**

New media has been connected. There is no payload, as this event is used to troubleshoot advanced use cases when hooking into the media connection.

### media.disconnected

- **media.disconnected**

Media has been disconnected from the session. There is no payload, as this event is used to troubleshoot advanced use cases when hooking into the media connection.

### media.reconnecting

- **media.reconnecting**

Media is attempting to reconnect to the session. There is no payload, as this event is used to troubleshoot advanced use cases when hooking into the media connection.

### member.updated

- **member.updated**(`event`)

A property of a member of the room has been updated.

<Accordion title="Properties">

<ParamField path="event" type="object">
  Event object
</ParamField>
<Indent>
  <ParamField path="event.room_session_id" type="string">
    Id for the room session that the updated member is in.
  </ParamField>
  <ParamField path="event.room_id" type="string">
    Id for the room that the updated member is in.
  </ParamField>
  <ParamField path="event.member" type="object">
    Information on the member that was updated.
  </ParamField>
  <Indent>
    <ParamField path="event.member.updated" type="string[]">
      Which room properties were updated.
    </ParamField>
    <ParamField path="event.member.id" type="string">
      Id for the member that was updated.
    </ParamField>
    <ParamField path="event.member.room_session_id" type="string">
      Id for the room session that the updated member is in.
    </ParamField>
    <ParamField path="event.member.room_id" type="string">
      Id for the room that the updated member is in.
    </ParamField>
    <ParamField path="event.member.type" type="string">
      The member type, either `audience` or `member`.
    </ParamField>
  </Indent>
</Indent>

</Accordion>

In the field `member.updated` you find an array of all the properties that have been updated. The new values for those properties are available as additional fields of member.

For example, say `visible` and `video_muted` have been updated. The received object will be:

```javascript
{
  "room_session_id": "35e85417-09cf-4b07-8f21-d3c16809e5a8",
  "room_id": "aae25822-892c-4832-b0b3-34aac3a0e8d1",
  "member": {
    "updated": ["visible", "video_muted"],
    "room_session_id": "35e85417-09cf-4b07-8f21-d3c16809e5a8",
    "visible": false,
    "video_muted": true,
    "id": "4a829c9f-812c-49d7-b272-e3077213c55e",
    "room_id": "aae25822-892c-4832-b0b3-34aac3a0e8d1",
    "type": "member"
  }
}
```

### playback.ended

- **playback.ended**

A playback has ended. You only receive this event if your token has the `room.playback` permission. The event handler receives a [RoomSessionPlayback](/browser-sdk/video/room-session-playback) object.

### playback.started

- **playback.started**

A playback has been started. You only receive this event if your token has the `room.playback` permission. The event handler receives a [RoomSessionPlayback](/browser-sdk/video/room-session-playback) object.

---

### playback.updated

- **playback.updated**

A playback has been updated. You only receive this event if your token has the `room.playback` permission. The event handler receives a [RoomSessionPlayback](./video-roomsessionplayback.md) object.
