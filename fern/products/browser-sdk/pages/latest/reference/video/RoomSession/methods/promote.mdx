### promote

- **promote**(`params`): `Promise<void>`

Promotes a participant from "audience" to "member". See [join](#join) and [demote](#demote).

#### Parameters

<ParamField path="params" type="object">
  Object containing the parameters of the method.
</ParamField>

<Indent>
<ParamField path="memberId" type="string">
  Id of the audience participant to promote.
</ParamField>

<ParamField path="joinAudioMuted" type="boolean" required={false}>
  Force the member's audio to be muted right after the promotion.
</ParamField>

<ParamField path="joinVideoMuted" type="boolean" required={false}>
  Force the member's video to be muted right after the promotion.
</ParamField>

<ParamField path="mediaAllowed" type='"all" | "audio-only" | "video-only"' required={false}>
  Specifies the media that the client will be allowed to _send_. A member participant can always receive all media.
</ParamField>

<ParamField path="meta" type="Record<string, unknown>" required={false}>
  Metadata to assign to the member.
</ParamField>

<ParamField path="permissions" type="string[]" required={false}>
  List of [permissions][permissions] to grant when the Audience participant will become a Member.
</ParamField>
</Indent>

#### Returns

`Promise<void>`

#### Permissions

- `room.member.promote`

You need to specify the permissions when [creating the Video Room Token](/rest/signalwire-rest/endpoints/video/create-room-token) on the server side.

#### Example

```javascript
await roomSession.promote({
  memberId: "de550c0c-3fac-4efd-b06f-b5b8614b8966",
  mediaAllowed: "all",
  permissions: [
    "room.self.audio_mute",
    "room.self.audio_unmute",
    "room.self.video_mute",
    "room.self.video_unmute",
    "room.list_available_layouts",
  ],
});
```
