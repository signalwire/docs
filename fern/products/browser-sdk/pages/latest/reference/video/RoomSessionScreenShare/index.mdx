Represents a screen share. This object is returned by the [startScreenShare](/browser-sdk/reference/video/room-session/methods/start-screen-share) method in a RoomSession, and can be used to control the screen share.

## Properties

<Indent>
  <ParamField path="active" type="boolean">
    Whether the connection is currently active.
  </ParamField>

  <ParamField path="cameraId" type="string | null">
    The id of the video device, or null if not available.
  </ParamField>

  <ParamField path="cameraLabel" type="string | null">
    The label of the video device, or null if not available.
  </ParamField>

  <ParamField path="localAudioTrack" type="MediaStreamTrack | null">
    Provides access to the local audio [MediaStreamTrack](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamTrack).
  </ParamField>

  <ParamField path="localStream" type="MediaStream | undefined">
    Provides access to the local [MediaStream](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream).
  </ParamField>

  <ParamField path="localVideoTrack" type="MediaStreamTrack | null">
    Provides access to the local video [MediaStreamTrack](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamTrack).
  </ParamField>

  <ParamField path="memberId" type="string">
    The id of the current member within the room.
  </ParamField>

  <ParamField path="microphoneId" type="string | null">
    The id of the audio input device, or null if not available.
  </ParamField>

  <ParamField path="microphoneLabel" type="string | null">
    The label of the audio input device, or null if not available.
  </ParamField>

  <ParamField path="remoteStream" type="MediaStream | undefined">
    Provides access to the remote [MediaStream](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream).
  </ParamField>

  <ParamField path="roomId" type="string">
    The unique identifier for the room.
  </ParamField>

  <ParamField path="roomSessionId" type="string">
    The unique identifier for the room session.
  </ParamField>
</Indent>
