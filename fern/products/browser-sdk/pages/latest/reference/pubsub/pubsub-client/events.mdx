## Events

### message

- **message**(`message`)

A new message has been received.

<Accordion title="Properties">

<ParamField path="message" type="PubSubMessage<PubSubMessageContract>" required={true}>
  The received message. See [`PubSubMessage`](/browser-sdk/pubsub/message) for more details.
</ParamField>

</Accordion>

### session.expiring

- **session.expiring**()

The session is going to expire. Use the [updateToken](/browser-sdk/reference/pubsub/client/methods/update-token) method to refresh your token.
