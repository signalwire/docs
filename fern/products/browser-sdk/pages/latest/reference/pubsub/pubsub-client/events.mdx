## Events

### message

- **message**(`message`)

A new message has been received.

<Accordion title="Properties">

<ParamField path="message" type="PubSubMessage<PubSubMessageContract>" required={true}>
  The received message. See [`PubSubMessage`](/docs/browser-sdk/v3/js/reference/pubsub) for more details.
</ParamField>

</Accordion>

### session.expiring

- **session.expiring**()

The session is going to expire. Use the [updateToken](/docs/browser-sdk/js/reference/pubsub/client/update-token) method to refresh your token.
