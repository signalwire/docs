### checkPermissions

- `Const` **checkPermissions**(`name?`): `Promise<null | boolean>`

Asynchronously returns whether we have permissions to access the specified resource. Some common parameter values for `name` are `"camera"`, `"microphone"`, and `"speaker"`. In those cases, prefer the dedicated functions [checkCameraPermissions](/browser-sdk/reference/webrtc/functions/check-camera-permissions), [checkMicrophonePermissions](/browser-sdk/reference/webrtc/functions/check-microphone-permissions), and [checkSpeakerPermissions](/browser-sdk/reference/webrtc/functions/check-speaker-permissions).

#### Parameters

<ParamField path="name" type="DevicePermissionName">
  Name of the resource.
</ParamField>

#### Returns

`Promise<null | boolean>`

#### Example

```javascript
await SignalWire.WebRTC.checkPermissions("camera");
// true: we have permission for using the camera
```
