### createCameraDeviceWatcher

- `Const` **createCameraDeviceWatcher**(): `Promise<EventEmitter<DeviceWatcherEvents, any>>`

Asynchronously returns an event emitter that notifies changes in all camera devices. This is equivalent to calling `createDeviceWatcher({ targets: ['camera'] })`, so refer to [createDeviceWatcher](/browser-sdk/reference/webrtc/functions/create-device-watcher) for additional information about the returned event emitter.

#### Returns

`Promise<EventEmitter<DeviceWatcherEvents, any>>`
