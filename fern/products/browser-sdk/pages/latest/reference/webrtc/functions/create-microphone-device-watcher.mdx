### createMicrophoneDeviceWatcher

- `Const` **createMicrophoneDeviceWatcher**(): `Promise<EventEmitter<DeviceWatcherEvents, any>>`

Asynchronously returns an event emitter that notifies changes in all microphone devices. This is equivalent to calling `createDeviceWatcher({ targets: ['microphone'] })`, so refer to [createDeviceWatcher](/browser-sdk/reference/webrtc/functions/create-device-watcher) for additional information about the returned event emitter.

#### Returns

`Promise<EventEmitter<DeviceWatcherEvents, any>>`
