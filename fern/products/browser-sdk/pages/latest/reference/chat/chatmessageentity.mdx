---
title: "ChatMessageEntity"
---

An object representing a Chat Message with only the state properties of [`ChatMessage`](/browser-sdk/chat/message).

## Properties

<ParamField path="content" type="any" required={true}>
  The content of this message. This can be any JSON-serializable object or value.
</ParamField>

<ParamField path="id" type="string" required={true}>
  The id of this message.
</ParamField>

<ParamField path="member" type="ChatMember" required={true}>
  The member which sent this message. See [`ChatMember`](/browser-sdk/chat/member) for more details.
</ParamField>

<ParamField path="meta" type="any">
  Any metadata associated with this message.
</ParamField>

<ParamField path="publishedAt" type="Date" required={true}>
  The date and time at which this message was published.
</ParamField>
