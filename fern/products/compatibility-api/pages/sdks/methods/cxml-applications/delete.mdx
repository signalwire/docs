Use this endpoint for the [cXML Applications](/compatibility-api/index) method to remove a cXML Application from your Project.
Use the unique ID that was returned from your previous request to identify the specific instance.

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/LamlBins/{Sid}" />

## Request Examples

<Tabs groupId="compatibility-sdks">

<Tab title="cURL">

```shell
curl https://{SPACE}.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/LamlBins/5184b831-184f-4209-872d-ccdccc80f2f1 \
  -X DELETE \
  -u 'YourProjectID:YourAuthToken'
```

</Tab>

</Tabs>

## Response

A successful request returns a 204 status code with no body.
