Returns a list of your cXML Applications.
The applications are returned sorted by creation date, with the most recent appearing first.

<EndpointSchemaSnippet endpoint="GET /Accounts/{AccountSid}/LamlBins" />

```shell
curl https://{SPACE}.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/LamlBins?Name=Star \
  -u 'YourProjectID:YourAuthToken'
```
