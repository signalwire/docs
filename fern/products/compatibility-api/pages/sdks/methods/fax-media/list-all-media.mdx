Use this endpoint for the [Fax Media](/compatibility-api/index) method to return a paged list of media belonging to this fax.
The most recent media will appear first.

<EndpointSchemaSnippet endpoint="GET /Accounts/{AccountSid}/Faxes/{FaxSid}/Media" />

<EndpointResponseSnippet endpoint="GET /Accounts/{AccountSid}/Faxes/{FaxSid}/Media" />

## Request examples

<Tabs groupId="compatibility-sdks">

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.fax.faxes('FaxSid')
      .media
      .each(media => console.log(media.sid));
```

</Tab>

<Tab title="C#">

```csharp
using System;
using System.Collections.Generic;
using Twilio;
using Twilio.Rest.Fax.V1.Fax;

class Program
{
    static void Main(string[] args)
    {
        TwilioClient.Init("YourProjectID", "YourAuthToken", new Dictionary<string, object> { ["signalwireSpaceUrl"] = "{SPACE}.signalwire.com" });

        var media = FaxMediaResource.Read(
            pathFaxSid: "FaxSid"
        );

        foreach(var record in media)
        {
           Console.WriteLine(record.Sid);
        }
    }
}
```

</Tab>

<Tab title="Python">

```python
from signalwire.rest import Client as signalwire_client

client = signalwire_client("YourProjectID", "YourAuthToken", signalwire_space_url = 'example.signalwire.com')

media = client.fax.faxes('FaxSid').media.list()

for record in media:
    print(record.sid)
```
