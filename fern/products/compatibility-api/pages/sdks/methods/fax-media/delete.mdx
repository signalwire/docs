Use this endpoint for the Fax Media method to delete a fax media instance.
If the delete is successful, a 204 response, with no body, will be returned.

<Warning title="This action cannot be undone.">
</Warning>

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/Faxes/{FaxSid}/Media/{Sid}" selector="request" />

## Request examples

<CodeBlocks>
```javascript title="Node.js"
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.fax.faxes('FaxSid')
      .media('MediaSid')
      .remove()
      .then(media => console.log(media.sid))
      .done();
```
```csharp
using System;
using System.Collections.Generic;

```

## Response

A successful request returns a 204 status code with no body.
