Use this endpoint for the [IncomingPhoneNumber](/compatibility-api/index) method to delete an IncomingPhoneNumber.
If the delete is successful, a 204 response, with no body, will be returned.

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/IncomingPhoneNumbers/{Sid}" selector="request" />

## Request examples

<Tabs groupId="compatibility-sdks">

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.incomingPhoneNumbers('IncomingPhoneNumberSid')
      .remove()
      .then(incoming_phone_number => console.log(incoming_phone_number.sid))
      .done();
```

</Tab>

A successful request returns a 204 status code with no body.
