Use this endpoint for the [Applications](/compatibility-api/index) method to delete an Application.

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/Applications/{Sid}" selector="request" />

## Request examples

<Tabs groupId="compatibility-sdks">

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.applications('Sid')
      .remove()
      .then(application => console.log(application.sid))
      .done();
```

</Tab>

<Tab title="C#">

```csharp
using System;
using Twilio;
using Twilio.Rest.Api.V2010.Account;

class Program
{
    static void Main(string[] args)
    {
        TwilioClient.Init("YourProjectID", "YourAuthToken", new Dictionary<string, object> { ["signalwireSpaceUrl"] = "{SPACE}.signalwire.com" });

        ApplicationResource.Delete(pathSid: "YourProjectID");
    }
}
```

</Tab>

<Tab title="Python">

```python

client = signalwire_client("YourProjectID", "YourAuthToken", signalwire_space_url = 'example.signalwire.com')

client.applications('Sid').delete()
```

</Tab>


</Tab>

</Tabs>

## Response

A successful request returns a 204 status code with no body.
