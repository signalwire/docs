Use this endpoint for the [Conference Participants](/compatibility-api/index) method to delete a participant, removing them from the conference call.
If the delete is successful, a 204 response, with no body, will be returned.

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/Conferences/{ConferenceSid}/Participants/{CallSid}" selector="request" />

## Request examples

<CodeBlocks>
```javascript title="Node.js"
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.conferences('ConferenceSid')
      .participants('CallSid')
      .remove()
      .then(participant => console.log(participant.callSid))
      .done();
```
```csharp title="C#"
using System;
using System.Collections.Generic;

class Program

    static void Main(string[] args)

```

## Response

A successful request returns a 204 status code with no body.
