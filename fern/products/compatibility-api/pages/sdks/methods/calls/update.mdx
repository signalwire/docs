Use this endpoint for the [Calls](/compatibility-api/index) method to modify an active call.

<EndpointSchemaSnippet endpoint="POST /Accounts/{AccountSid}/Calls/{Sid}" />

## Request Examples

<Tabs groupId="compatibility-sdks">

<Tab title="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Calls/{Sid}.json \
  -X POST \
  -u "YourProjectID:YourAuthToken"
```

</Tab>

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.calls('Sid')
      .update()
      .then(call => console.log(call.to))
      .done();
```

</Tab>

<Tab title="C#">

```csharp
using System;
using System.Collections.Generic;
using Twilio;
using Twilio.Rest.Api.V2010.Account;

class Program
{
    static void Main(string[] args)
    {
        TwilioClient.Init("YourProjectID", "YourAuthToken", new Dictionary<string, object> { ["signalwireSpaceUrl"] = "{SPACE}.signalwire.com" });

        var call = CallResource.Update(
            pathSid: "Sid"
        );

        Console.WriteLine(call.To);
    }
}
```

</Tab>

### Example: Terminate a Call

In this example, we terminate a call in progress by updating the call status to `completed`.

<Tabs groupId="compatibility-sdks">

<Tab title="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Calls/{Sid}.json \
  -X POST \
  --data-urlencode "Status=completed" \
  -u "YourProjectID:YourAuthToken"
```

</Tab>

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.calls('Sid')
      .update({status: 'completed'})
      .then(call => console.log(call.to))
      .done();
```

</Tab>

<Tab title="C#">

```csharp
using System;
using System.Collections.Generic;
using Twilio;
using Twilio.Rest.Api.V2010.Account;

class Program
{
    static void Main(string[] args)
    {
        TwilioClient.Init("YourProjectID", "YourAuthToken", new Dictionary<string, object> { ["signalwireSpaceUrl"] = "{SPACE}.signalwire.com" });

        var call = CallResource.Update(
            status: CallResource.UpdateStatusEnum.Completed,
            pathSid: "Sid"
        );

        Console.WriteLine(call.To);
    }
}
```

</Tab>
