A **Fax** is a fax that has been sent to or received by a SignalWire phone number.

## Properties

> A sample Fax returned from the API.

{
  "account_sid": "b3877c40-da60-4998-90ad-b792e98472ac",

  "date_updated": "2018-11-27T20:00:00Z",
  "direction": "outbound",
  "from": "+13103383454",
  "media_url": "https://example.com/fax.pdf",
  "media_sid": "b3877c40-da60-4998-90ad-b792e98472me",

<ParamField path="account_sid" type="string">
  The unique identifier for the account this fax is associated with.
</ParamField>

<ParamField path="date_created" type="string">
  The date and time, in ISO 8601 format, the fax was created.
</ParamField>

<ParamField path="date_updated" type="string">
  The date and time, in ISO 8601 format, the fax was updated.
</ParamField>

<ParamField path="direction" type="string">
  The direction of the fax. Possible values are `inbound` or `outbound`.
</ParamField>

<ParamField path="duration" type="integer">
  The time, in seconds, it took to deliver a fax.
</ParamField>

<ParamField path="from" type="string">
  The phone number, in E.164 format, the fax was sent from.
</ParamField>

<ParamField path="links" type="object">
  The URL links for resources associated with the fax.
</ParamField>

<ParamField path="media_sid" type="string">
  The unique identifier for the media instance associated with the fax instance.
</ParamField>

<ParamField path="media_url" type="string">
  The URL hosting the received media. Can use this URL to download incoming media.
</ParamField>

<ParamField path="num_pages" type="string">
  The number of pages in the fax document.
</ParamField>

<ParamField path="price" type="string">
  The cost of the fax.
</ParamField>

<ParamField path="price_unit" type="string">
  The currency, in ISO 4217 format, of the price.
</ParamField>

<ParamField path="quality" type="string">
  The quality of the fax. See [below](#quality) for possible values.
</ParamField>

<ParamField path="sid" type="string">
  The unique identifier of the fax.
</ParamField>

<ParamField path="status" type="string">
  The status of the fax. See [below](#status) for possible values.
</ParamField>

<ParamField path="to" type="string">
  The phone number, in E.164 format, the fax was sent to.
</ParamField>

<ParamField path="url" type="string">
  The URL of this resource.
</ParamField>

## `Quality`

The `quality` attribute has the following values:

|                                          Value |                                                                                         |
| ---------------------------------------------: | --------------------------------------------------------------------------------------- |
|  `standard` | A low quality (204x98) fax resolution. This quality should be supported by all devices. |
|      `fine` | A medium quality (204x196) fax resolution.                                              |
| `superfine` | A high quality (204x392) fax resolution.                                                |

## `Status`

The `status` attribute has the following values:

|                                           Value |                                                                            |
| ----------------------------------------------: | -------------------------------------------------------------------------- |
|     `queued` | The fax is queued and waiting for processing.                              |
| `processing` | The fax is being uploaded, downloaded, or converted to a different format. |
|    `sending` | The fax is being sent.                                                     |
|  `delivered` | The fax has been successfully sent.                                        |
|  `receiving` | The fax is being received.                                                 |
|   `received` | The fax has been successfully received.                                    |
|  `no-answer` | The fax failed because the recipient didn't pick up.                       |
|       `busy` | The fax failed because the receiving machine sent back a busy signal.      |
|     `failed` | The fax failed to send or receive.                                         |
|   `canceled` | The fax was canceled.                                                      |
