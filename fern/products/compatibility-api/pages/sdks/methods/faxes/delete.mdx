# Delete a Fax

Use this endpoint for the [Fax](/compatibility-api/index) method to delete a fax. 
If the delete is successful, a 204 response, with no body, will be returned. 

<Warning title="This action cannot be undone.">
</Warning>

## Parameters

| Parameter                                           | Description                       |
| :-------------------------------------------------- | :-------------------------------- |
| `FaxSid` <span className="required-arg">Required</span> | The unique identifier of the fax. |

---

## Request

<Tabs groupId="compatibility-sdks">

<Tab title="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Faxes/{FaxSid}.json \
  -X DELETE \
  -u "YourProjectID:YourAuthToken"
```

</Tab>

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.fax.faxes('FaxSid')
      .remove()
      .then(fax => console.log(fax.sid))
      .done();
```

## Responses 

<h4><span class="badge badge--success">204</span> NO CONTENT</h4>

A `204 No Content` response indicates that the fax was successfully deleted.
