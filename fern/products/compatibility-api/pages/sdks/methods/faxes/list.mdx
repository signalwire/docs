# List All Faxes

Use this endpoint for the [Fax](/compatibility-api/index) method to list all faxes on your SignalWire account.

## Parameters

| Parameter                                                              | Description                                                                                        |
| :--------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| `date_created_after` <span className="optional-arg">Optional</span>        | Filter the returned list of faxes to only those that were created after the specified date.        |
| `date_created_on_or_before` <span className="optional-arg">Optional</span> | Filter the returned list of faxes to only those that were created on or before the specified date. |
| `from` <span className="optional-arg">Optional</span>                      | Filter the returned list of faxes to only those that were sent from the specified phone number.    |
| `to` <span className="optional-arg">Optional</span>                        | Filter the returned list of faxes to only those that were sent to the specified phone number.      |
