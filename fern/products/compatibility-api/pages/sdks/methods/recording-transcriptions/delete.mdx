Use this endpoint for the [Recording Transcriptions](/compatibility-api/index) method to delete a recording transcription from your account.
If the delete is successful, a 204 response, with no body, will be returned.

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/Transcriptions/{Sid}" selector="request" />

## Request examples

<Tabs groupId="compatibility-sdks">

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.transcriptions('Sid')
      .remove()
      .then(transcription => console.log(transcription.sid))
      .done();
```

</Tab>

## Response

A successful request returns a 204 status code with no body.
