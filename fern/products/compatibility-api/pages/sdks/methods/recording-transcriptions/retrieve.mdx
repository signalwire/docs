# Retrieve a Transcription

Use this endpoint for the [Recording Transcriptions](/compatibility-api/index) method to retrieve a single recording transcription.

## Parameters

| Parameter                                        | Description                                                                                              |
| :----------------------------------------------- | :------------------------------------------------------------------------------------------------------- |
| `Sid` <span className="required-arg">Required</span> | The Call [SID](/platform/dashboard/guides/what-is-a-sid/) that uniquely identifies the recording transcription to retrieve. |

---

## Request

Retrieve a single recording transcription.

<Tabs groupId="compatibility-sdks">

<Tab title="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Transcriptions/{Sid}.json \
  -X GET \
  -u "YourProjectID:YourAuthToken"
```

</Tab>

<Tab title="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.transcriptions('Sid')
      .fetch()
      .then(transcription => console.log(transcription.dateCreated))
      .done();
```
