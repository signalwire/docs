A Recording represents a recording of a voice or conference call. 
It allows you to see meta information about the recording as well as access the media file itself.

Recordings can be initiated using the [`<Record>`](/compatibility-api/cxml/voice/record) verb, 
by setting the `record` attribute for [Calls](/compatibility-api/calls/index),
or by using [Conferences](/compatibility-api/conferences/index).

<ParamField path="account_sid" type="string">
  The unique identifier for the account that is associated with this recording.
</ParamField>

<ParamField path="api_version" type="string">
  The version of the SignalWire API.
</ParamField>

<ParamField path="call_sid" type="string">
  The unique identifier for the call that is associated with this recording.
</ParamField>

<ParamField path="channels" type="integer">
  The number of channels in a recording. Possible values are `1` or `2`.
</ParamField>

<ParamField path="conference_sid" type="string">
  The unique identifier for the conference that is associated with this recording.
</ParamField>

<ParamField path="date_created" type="datetime">
  The date, in RFC 2822 format, this recording was created.
</ParamField>

<ParamField path="date_updated" type="datetime">
  The date, in RFC 2822 format, this recording was updated.
</ParamField>

<ParamField path="duration" type="string">
  The length, in seconds, of the recording.
</ParamField>

<ParamField path="error_code" type="string">
  Further details about a failed recording.
</ParamField>

<ParamField path="price" type="integer">
  The cost for the recording.
</ParamField>

<ParamField path="price_unit" type="string">
  The currency of the price of the recording.
</ParamField>

<ParamField path="sid" type="string">
  The unique identifier for the recording.
</ParamField>

<ParamField path="source" type="string">
  How the recording was made. Possible values are `DialVerb`, `Conference`, `OutboundAPI`, `Trunking`, `RecordVerb`, `StartCallRecordingAPI`, or `StartConferenceRecording`.
</ParamField>

<ParamField path="start_time" type="datetime">
  The time, in RFC 2822 format, this recording began.
</ParamField>

<ParamField path="status" type="string">
  The status of the recording.
</ParamField>

<ParamField path="uri" type="string">
  The URI of the recording.
</ParamField>
