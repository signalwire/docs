Use this endpoint for the Recordings method to delete a recording.
If the deletion is successful, a 204 response, with no request body, will be returned.

<EndpointSchemaSnippet endpoint="DELETE /Accounts/{AccountSid}/Recordings/{Sid}" selector="request" />

## Request examples

<CodeBlocks>
```javascript title="Node.js"
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.recordings('Sid')
      .remove()
      .then(recording => console.log(recording.sid))
      .done();
```
```csharp
using System;
using System.Collections.Generic;

```
```python

```

</CodeBlocks>

## Response

A successful request returns a 204 status code with no body.
