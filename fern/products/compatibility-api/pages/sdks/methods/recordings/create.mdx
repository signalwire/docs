Use this endpoint for the [Recordings](/compatibility-api/index) method to initiate a Recording for a [Call](/compatibility-api/calls/index).
You can use this endpoint to start recording a Call that is already in progress.
If you want to record the entire Call, you can specify recording params during [creation of the Call](/compatibility-api/calls/create).

<EndpointSchemaSnippet endpoint="POST /Accounts/{AccountSid}/Calls/{CallSid}/Recordings" />
