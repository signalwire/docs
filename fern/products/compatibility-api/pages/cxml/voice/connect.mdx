The `<Connect>` verb connects an existing call to another resource.

You can use the `<Connect>` verb with the following nouns:

|                    Noun                     |                             |
|:-------------------------------------------:|:---------------------------:|
|         [`<Room>`](/compatibility-api/room-noun)         |        A video room.        |
|         [`<Stream>`](/compatibility-api/stream)          |   A bidirectional stream.   |
| [`<VirtualAgent>`](/compatibility-api/virtualagent-noun) | A Dialogflow virtual agent. |

## Examples

### A Simple Connect

<Tabs groupId="cxml-api">
  <Tab title="XML">

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Response>
  <Connect>
    <Room>my-room-name</Room>
  </Connect>
</Response>
```

  </Tab>

</Tabs>

### Bidirectional Media Stream

<Tabs groupId="cxml-api">
  <Tab title="XML">

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Response>
  <Connect>
    <Stream url="wss://your-application.com/audiostream" />
  </Connect>
</Response>
```

  </Tab>

</Tabs>
