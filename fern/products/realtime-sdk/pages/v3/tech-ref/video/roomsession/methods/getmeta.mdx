### getMeta

- **getMeta**(): `Promise<{ meta: Object }>`

Returns the metadata assigned to this Room Session.

#### Returns

`Promise<{ meta: Object }>`

#### Example

```javascript
const { meta } = await roomSession.getMeta();
console.log(meta);
```
