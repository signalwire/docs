### getMembers

- **getMembers**(): `Promise<{ members: VideoMemberEntity[] }>`

Returns a list of members currently in the room.

#### Returns

`Promise<{ members: VideoMemberEntity[] }>`

an object with type: `Promise<{ members: VideoMemberEntity[] }>`
