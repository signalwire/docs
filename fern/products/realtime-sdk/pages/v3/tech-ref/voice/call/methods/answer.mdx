### answer

- **answer**(): `Promise<Call>` - See [Call][call-19] for more details.

Answers the incoming call.

#### Returns

`Promise<Call>` - See [Call][call-19] for more details.

#### Example

```js
client.on("call.received", async (call) => {
  try {
    await call.answer();
    console.log("Inbound call answered");
  } catch (error) {
    console.error("Error answering inbound call", error);
  }
});
```
