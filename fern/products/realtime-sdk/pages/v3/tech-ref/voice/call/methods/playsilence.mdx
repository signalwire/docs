### playSilence

- **playSilence**(`params`): `Promise<CallPlayback>` - See [CallPlayback][callplayback-9] for more details.

Plays some silence.

#### Parameters

| Name              | Type     | Description                 |
|:------------------|:---------|:----------------------------|
| `params`          | `Object` | -                           |
| `params.duration` | `number` | Seconds of silence to play. |

#### Returns

`Promise<CallPlayback>` - See [CallPlayback][callplayback-9] for more details.

#### Example

```js
const playback = await call.playSilence({ duration: 3 });
await playback.ended();
```
