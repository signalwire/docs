### hangup

- **hangup**(`reason?`): `Promise<void>`

Hangs up the call.

#### Parameters

| Name      | Type                                                                             | Description                     |
|:----------|:---------------------------------------------------------------------------------|:--------------------------------|
| `reason?` | `"error"` \| `"hangup"` \| `"cancel"` \| `"busy"` \| `"noAnswer"` \| `"decline"` | Optional reason for hanging up. |

#### Returns

`Promise<void>`

#### Example

```js
call.hangup();
```
