### removeAllListeners

- **removeAllListeners**(`event?`)

Detaches all event listeners for the specified event.

#### Parameters

| Name     | Type     | Description                                                                                                                                              |
| :------- | :------- | :------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `event?` | `string` | Name of the event (leave this undefined to detach listeners for all events). See [Events](/docs/server-sdk/v3/node/reference/messaging/client/events) for the list of available events. |
