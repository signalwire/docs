# Relay.Calling.SendDigitsResult

This object is returned by the [`sendDigits`][senddigits] method and represents the final result of the action.

## Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `successful` | `boolean` | Whether the action has completed successfully |
| `event` | [`Relay.Event`][relay-event] | Last event that completed the operation |
