# Relay.Calling.SendDigitsAction

This object is returned by the [`sendDigitsAsync`][senddigitsasync] method and represents a _send digits_ action currently active on a call.

## Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `result` | [`Relay.Calling.SendDigitsResult`][relay-calling-senddigitsresult] | Final result of the action |
| `state` | `string` | Current state of the action |
| `completed` | `boolean` | Whether the action has finished |
| `payload` | `object` | Payload sent to Relay to start the action |
| `controlId` | `string` | UUID to identify the action |
