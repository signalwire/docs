# SignalWire.Relay.Calling.PromptResult

This object returned from one the _synchronous_ [`Prompt`][call] methods or part of the action when an _asynchronous_ equivalent completes, which represent the final result of a prompt operation.

## Properties

| Property | Type | Description |
| :-------- | :---- | :----------- |
| `Successful` | bool | Whether the play operation has completed successfully. |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
| `Type` | [`SignalWire.Relay.Calling.CallCollectType`][signalwire-relay-calling-callcollecttype] | The type of the collected result. |
| `Result` | string | Result of prompt operation. |
| `Terminator` | string | User input that terminated a `digit` prompt. |
| `Confidence` | double? | Confidence of the result on a `speech` prompt. |
