## Relay.Event

This object represents the last Relay event that completed an operation on the Call.

### Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `Name` | `string` | The event name. |
| `Payload` | `JObject` | Raw JSON object of the Relay event. |
