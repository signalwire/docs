# Relay.Calling.PlayVolumeResult

This object is returned by [`volume`][play] method and represents the final result of a volume control operation.

## Properties

| Property | Type | Description |
| :-------- | :-------- | :----------- |
| `successful` | `boolean` | Whether the playing volume has been changed successfully. |
