# Relay::Calling::SendDigitsResult

This object is returned by the [`send_digits`][send_digits] methods and represents the final result of an action sending digits.

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether the action has completed successfully. |
| `event` | [Relay::Event][relay-event] | Last event that completed the action. |
