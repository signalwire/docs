# Relay::Calling::HangupResult

This object is returned by the [`hangup`][call] method.

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether the call has been hung up successfully. |
| `event` | [Relay::Event][relay-event] | Last event that completed the operation. |
| `reason` | String | The hangup reason. |
