A [PubSub][pubsub] client is used to listen for events on a channel and publish messages to a channel.

## **PubSub Client**
### Setting up a PubSub Client

To create a `PubSub` client, you will first need to create a [`SignalWire Realtime-Client`][signalwire-realtime-client].
After the `SignalWire Client` is created, you can access the `PubSub` client using the `pubSub` namespace.

```js
import { SignalWire} from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const pubSubClient = client.pubSub;
```

## **Methods**

### listen

- **listen**(`{ event: Callback }`): `Promise<PubSubEvents>`

Listen for events on the specified channel.

#### Parameters

<ParamField path="params" type="object" required>
  Object containing the parameters of the method.
</ParamField>

<Indent>
<ParamField path="channels" type="string[]" required>
  List of channels to listen to.
</ParamField>

<ParamField path="EVENT" type="Callback">
  The event to listen to. List of events can be found [here][events]. Example event: `onMessageReceived`.
</ParamField>
</Indent>

#### Returns

`Promise<PubSubEvents>`

A promise that resolves to a `PubSubEvents` object that you can use to view the current state or results of the event.

#### Example

```js
import { SignalWire} from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const pubSubClient = client.pubSub;

await pubSubClient.listen({
  channels: ["my-channel"],
  onMessageReceived: (message) => {
    console.log(message);
  }
});
```

In this example:
 - Import the SignalWire module and initialize a new SignalWire client using your project ID and token.
 - Access the PubSub client from the initialized SignalWire client. 
 - Subscribe to the `my-channel` channel and set up a listener for incoming messages.
 - Log received messages to the console.

### publish

- **publish**(`params`): `Promise<void>`

Publish a message into the specified channel.

#### Parameters

<ParamField path="params" type="object" required>
  Object containing the parameters of the method.
</ParamField>

<Indent>
<ParamField path="channel" type="string" required>
  Channel in which to send the message.
</ParamField>

<ParamField path="content" type="any" required>
  The message to send. This can be any JSON-serializable object.
</ParamField>

<ParamField path="meta" type="Record<any, any>">
  Metadata associated with the message. There are no requirements on the content of metadata.
</ParamField>
</Indent>

#### Returns

`Promise<void>`

#### Examples

Publishing a message as a string:

```js
import { SignalWire } from '@signalwire/realtime-api'

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const pubSubClient = client.pubSub;

await pubSubClient.publish({
  channel: "my-channel",
  content: "Hello, world."
});
```

Publishing a message as an object:

```js
import { SignalWire } from '@signalwire/realtime-api'

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const pubSubClient = client.pubSub;

await pubSubClient.publish({
  channel: "my-channel",
  content: {
    field_one: "value_one",
    field_two: "value_two"
  }
});
```

## **Events**

### onMessageReceived

- **client.pubSub.listen**(`{ onMessageReceived: Callback }`)

Emitted when a message is received on a channel. Your event handler will be called with an instance of [`PubSubMessage`][pubsubmessage].

#### Parameters

<ParamField path="message" type="PubSubMessage" required>
  The message that has been received. See [`PubSubMessage`][pubsubmessage].
</ParamField>
