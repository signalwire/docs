You can use instances of this class to initiate or receive calls. Please see [Events][events] for the full list of events you can subscribe to.

## **Voice Client**

### Setting up a Voice Client
To create a `Voice` client, you will need to create a [`SignalWire Realtime-Client`][signalwire-realtime-client] first.
After the `SignalWire Client` is created, you can access the `Voice` client using the `voice` namespace.

#### Example
```javascript
import { SignalWire } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const voiceClient = client.voice
```

## **Methods**

### dial

- **dial**(`params`): `Promise`\<[`Call`][call]>

Makes an outbound Call and waits until it has been answered or hung up. This is an advanced method that lets you call multiple devices in parallel or series: for simpler use cases, see [dialPhone][dialphone] and [dialSip][dialsip].

With this method you can specify a configuration of devices to call in series and/or in parallel: as soon as one device answers the call, the returned promise is resolved. You specify a configuration through a [VoiceDeviceBuilder][voicedevicebuilder] object.

#### Parameters

<ParamField path="params" type="VoiceDeviceBuilder" required>
  The device builder specifying the devices to call. See [`VoiceDeviceBuilder`][voicedevicebuilder].
</ParamField>

#### Returns

`Promise`\<[`Call`][call]>

A call object.

#### Example

Calls a phone number. If the number doesn't answer within 30 seconds, calls two different SIP endpoints in parallel.

```js
import { SignalWire, Voice } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const voiceClient = client.voice

const devices = new Voice.DeviceBuilder()
  .add(Voice.DeviceBuilder.Phone({ from: "+XXXXXX", to: "+YYYYYY", timeout: 30 }))
  .add([
    Voice.DeviceBuilder.Sip({ from: "sip:aaa@bbb.cc", to: "sip:xxx@yyy.zz" }),
    Voice.DeviceBuilder.Sip({ from: "sip:aaa@bbb.cc", to: "sip:ppp@qqq.rr" })
  ]);

try {
  const call = await voiceClient.dial(devices);
  console.log("Call answered", call);
} catch (e) {
  console.log("Call not answered", e);
}
```

### dialPhone

- **dialPhone**(`params`): `Promise`\<[`Call`][call-5]>

Makes an outbound call to a PSTN number.

#### Parameters

<ParamField path="params" type="object" required>
  Object containing the parameters for dialing a phone number.
</ParamField>

<Indent>
<ParamField path="params.to" type="string" required>
  The party you are attempting to call.
</ParamField>

<ParamField path="params.from" type="string">
  The party the call is coming from. Must be a SignalWire number or SIP endpoint that you own.
</ParamField>

<ParamField path="params.timeout" type="number">
  The time, in seconds, the call will ring before it is considered unanswered.
</ParamField>

<ParamField path="params.maxPricePerMinute" type="number">
  The maximum price in USD acceptable for the call to be created. If the rate for the call is greater than this value, the call will not be created. If not set, all calls will be created. Price can have a maximum of four decimal places, i.e. `0.0075`.
</ParamField>

<ParamField path="params.listen" type="object">
  Object that contains callbacks to listen for events. List of Call events can be found [here][call-events].
</ParamField>
</Indent>

#### Returns

`Promise`\<[`Call`][call]>

A call object.

#### Example

```js
import { SignalWire, Voice } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const voiceClient = client.voice

try {
  const call = await voiceClient.dialPhone({
    from: "+YYYYYYYYYY",
    to: "+XXXXXXXXXX",
    timeout: 30
  });
  console.log("Call answered.", call);
} catch (e) {
  console.log("Call not answered.", e);
}
```

### dialSip

- **dialSip**(`params`): `Promise`\<[`Call`][call-5]>

Makes an outbound call to a SIP endpoint.

#### Parameters

<ParamField path="params" type="object" required>
  Object containing the parameters for dialing a SIP endpoint.
</ParamField>

<Indent>
<ParamField path="params.from" type="string" required>
  The party the call is coming from. Must be a SignalWire number or SIP endpoint that you own.
</ParamField>

<ParamField path="params.to" type="string" required>
  The party you are attempting to call.
</ParamField>

<ParamField path="params.timeout" type="number">
  The time, in seconds, the call will ring before it is considered unanswered.
</ParamField>

<ParamField path="params.headers" type="SipHeader[]">
  Array of [`SipHeader`][sipheader] objects. Must be X- headers only, see example below.
</ParamField>

<ParamField path="params.codecs" type="SipCodec[]">
  Optional array of desired codecs in order of preference. Supported values are `PCMU`, `PCMA`, `OPUS`, `G729`, `G722`, `VP8`, `H264`. Default is parent leg codec(s).
</ParamField>

<ParamField path="params.webrtcMedia" type="boolean">
  If `true`, WebRTC media is negotiated. Default is parent leg setting.
</ParamField>

<ParamField path="params.sessionTimeout" type="number">
  Non-negative value, in seconds, to use for the SIP `Session-Expires` header. If `0` or unset, SignalWire will pick the default (typically `600`).
</ParamField>

<ParamField path="params.maxPricePerMinute" type="number">
  The maximum price in USD acceptable for the call to be created. If the rate for the call is greater than this value, the call will not be created. If not set, all calls will be created. Price can have a maximum of four decimal places, i.e. `0.0075`.
</ParamField>

<ParamField path="params.listen" type="object">
  Object that contains callbacks to listen for events. List of Call events can be found [here][call-events].
</ParamField>
</Indent>

#### Returns

`Promise`\<[`Call`][call]>

A call object.

#### Example

```js
import { SignalWire, Voice } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const voiceClient = client.voice

try {
  const call = await voiceClient.dialSip({
    from: "sip:xxx@yyy.zz",
    to: "sip:ppp@qqq.rr",
    timeout: 30,
    headers: [
      { name: "X-SomeKeyA", value: "SomeValueA" },
      { name: "X-SomeKeyB", value: "SomeValueB" }
    ]
  });
  console.log("Call answered.", call);
} catch (e) {
  console.log("Call not answered.", e);
}
```

### listen

- **listen**(`{event: Callback }`): `Promise<Call Events>`

Listens for Voice events.

#### Parameters

<ParamField path="listenOptions" type="object" required>
  Object containing the parameters of the method.
</ParamField>

<Indent>
<ParamField path="listenOptions.topics" type="string[]" required>
  Topics to listen on. Previously known as `"contexts"`.
</ParamField>

<ParamField path="listenOptions.EVENT" type="Callback">
  The event to listen to. List of events can be found [here][events]. Example event: `onCallReceived`.
</ParamField>
</Indent>

#### Returns

`Promise<Call Events>` - See [Call Events][link] for the full list of events you can listen to.

#### Example

```js
import { SignalWire } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here"})

const voiceClient = client.voice;

await voiceClient.listen({
  topics: ["office"],
  onCallReceived: (call) => {
    call.answer();
    console.log("Call received", call);
  }
});
```
