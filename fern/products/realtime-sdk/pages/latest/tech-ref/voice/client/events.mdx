## Events

### onCallReceived [#oncallreceived]

- **client.voice.listen**(`{ onCallReceived: Callback }`)

Emitted when a call is received. Your event handler will be called with a [`Call`][call] object.

#### Parameters

<ParamField path="call" type="Call" required>
  The incoming [`Call`][call] object.
</ParamField>
