The Voice Client allows you to make outbound calls and receive inbound calls. Access it via the `voice` property on a [`SignalWire Client`][signalwire-realtime-client].

```javascript
import { SignalWire } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "<project-id>", token: "<api-token>" });
const voiceClient = client.voice;
```

Use [`dialPhone`][dialphone] or [`dialSip`][dialsip] to initiate outbound calls, or [`listen`][listen] to receive inbound calls on specific topics. See [Events][events] for all available events.

## Examples

### Receiving calls

```javascript
await voiceClient.listen({
  topics: ["office"],
  onCallReceived: async (call) => {
    console.log("Got call", call.from, call.to);
    await call.answer();
  }
});
```

### Making calls

```javascript
const call = await voiceClient.dialPhone({
  from: "+15551234567",
  to: "+15559876543",
  timeout: 30,
});
```
