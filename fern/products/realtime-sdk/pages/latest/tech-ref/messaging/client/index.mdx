The Messaging Client enables sending and receiving SMS and MMS messages. Access it via the `messaging` property on a [`SignalWire Client`][signalwire-realtime-client].

```javascript
import { SignalWire } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "<project-id>", token: "<api-token>" });
const messagingClient = client.messaging;
```

Use [`send`][send] to send outbound messages, or [`listen`][listen] to receive inbound messages on specific topics. See [Events][events] for all available events.

## Examples

### Receiving messages

```javascript
await messagingClient.listen({
  topics: ["office"],
  onMessageReceived: (message) => {
    console.log("From:", message.from, "Body:", message.body);
  }
});
```

### Sending messages

```javascript
await messagingClient.send({
  from: "+15551234567",
  to: "+15559876543",
  body: "Hello from SignalWire!",
});
```
