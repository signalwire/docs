### listen

- **listen**(`{ event: Callback }`): `Promise<ChatEvents>`

Listen for events on the specified channels.

#### Parameters

<ParamField path="params" type="object" required>
  Object containing the parameters of the method.
</ParamField>

<Indent>
<ParamField path="channels" type="string[]" required>
  List of channels to listen to.
</ParamField>

<ParamField path="EVENT" type="Callback">
  The event to listen to. List of events can be found [here][events]. Example event: `onMessageReceived`.
</ParamField>
</Indent>

#### Returns

`Promise<ChatEvents>`

A promise that resolves to a `ChatEvents` object that you can use to view the current state or results of the event.

#### Example

In this example, we listen for the `onMessageReceived` event on the channel
`my-channel` and print the message to the console.

```js
import { SignalWire } from "@signalwire/realtime-api";

const client = await SignalWire({ project: "ProjectID Here", token: "Token Here" })

const chatClient = client.chat;

await chatClient.listen({
  channels: ["my-channel"],
  onMessageReceived: (message) => {
    console.log(message);
  }
});
```
