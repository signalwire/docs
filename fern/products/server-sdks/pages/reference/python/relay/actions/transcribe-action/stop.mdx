Stop transcription.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python {16}
from signalwire.relay import RelayClient

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()
    action = await call.transcribe()

    # Stop transcription early
    await action.stop()

client.run()
```
