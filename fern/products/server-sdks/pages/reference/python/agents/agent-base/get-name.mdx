Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`str` -- The agent name.

## **Example**

```python {4}
from signalwire import AgentBase

agent = AgentBase(name="support", route="/support")
print(agent.get_name())  # "support"
```
