Check whether all `REQUIRED_ENV_VARS` are set in the environment.

## **Parameters**

None.

## **Returns**

`bool` -- `True` if all required variables are present, `False` with error
logging otherwise.

## **Example**

```python {4}
def setup(self) -> bool:
    if not self.validate_env_vars():
        return False
    # Continue with initialization...
    return True
```
