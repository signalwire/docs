List registered 10DLC brands.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/brands" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/brands" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

brands = client.registry.brands.list()
for brand in brands.get("data", []):
    print(brand.get("display_name"), brand.get("entity_type"))
```
