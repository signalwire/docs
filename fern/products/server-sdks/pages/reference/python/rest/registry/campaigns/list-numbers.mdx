List phone numbers assigned to a campaign.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/numbers" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/numbers" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

numbers = client.registry.campaigns.list_numbers("campaign-id")
for n in numbers.get("data", []):
    print(n.get("number"), n.get("id"))
```
