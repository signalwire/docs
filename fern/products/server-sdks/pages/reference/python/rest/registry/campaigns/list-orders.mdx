List number assignment orders for a campaign.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

orders = client.registry.campaigns.list_orders("campaign-id")
for order in orders.get("data", []):
    print(order.get("id"), order.get("status"))
```
