Verify an MFA code submitted by the user.

<EndpointSchemaSnippet endpoint="POST /api/relay/rest/mfa/{mfa_request_id}/verify" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/relay/rest/mfa/{mfa_request_id}/verify" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.mfa.verify("your-mfa-request-id", token="123456")
if result.get("success"):
    print("Verification successful")
else:
    print("Invalid code")
```
