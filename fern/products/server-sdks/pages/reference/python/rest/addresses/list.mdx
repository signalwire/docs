List addresses in the project.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/addresses" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.addresses.list()
for addr in result.get("data", []):
    print(addr.get("friendly_name"), addr.get("street"))
```
