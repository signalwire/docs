List addresses associated with an AI agent resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/ai_agents/{ai_agent_id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/ai_agents/{ai_agent_id}/addresses" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addresses = client.fabric.ai_agents.list_addresses("agent-id")
for addr in addresses.get("data", []):
    print(f"{addr['display_name']}: {addr['id']}")
```
