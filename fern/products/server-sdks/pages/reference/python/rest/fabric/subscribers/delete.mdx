Delete a subscriber resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/subscribers/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.subscribers.delete("subscriber-id")
print("Subscriber deleted")
```
