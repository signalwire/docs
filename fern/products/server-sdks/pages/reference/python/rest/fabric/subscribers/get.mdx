Retrieve a single [subscriber](/docs/platform/subscribers) resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/subscribers/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/subscribers/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

subscriber = client.fabric.subscribers.get("subscriber-id")
print(f"Subscriber: {subscriber['display_name']}, ID: {subscriber['id']}")
```
