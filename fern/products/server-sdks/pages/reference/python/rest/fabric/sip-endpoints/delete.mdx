Delete a SIP endpoint resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/sip_endpoints/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.sip_endpoints.delete("endpoint-id")
print("Sip Endpoint deleted")
```
