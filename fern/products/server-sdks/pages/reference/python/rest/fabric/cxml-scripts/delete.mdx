Delete a cXML script resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/cxml_scripts/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.cxml_scripts.delete("resource-id")
print("Cxml Script deleted")
```
