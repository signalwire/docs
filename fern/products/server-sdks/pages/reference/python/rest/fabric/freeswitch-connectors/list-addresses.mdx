List addresses associated with a FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/freeswitch_connectors/{id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/freeswitch_connectors/{id}/addresses" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addresses = client.fabric.freeswitch_connectors.list_addresses("connector-id")
for addr in addresses.get("data", []):
    print(f"{addr['display_name']}: {addr['id']}")
```
