Create a new FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/freeswitch_connectors" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/freeswitch_connectors" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.freeswitch_connectors.create(
    name="my-connector",
)
print(f"Created: {result['id']}")
```
