Retrieve a single FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/freeswitch_connectors/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/freeswitch_connectors/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

connector = client.fabric.freeswitch_connectors.get("connector-id")
print(f"Freeswitch Connector: {connector['display_name']}, ID: {connector['id']}")
```
