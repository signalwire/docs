List addresses associated with a Fabric resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/{id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/{id}/addresses" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addresses = client.fabric.resources.list_addresses("resource-id")
for addr in addresses.get("data", []):
    print(f"{addr['display_name']}: {addr['id']}")
```
