Retrieve a single RELAY application resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/relay_applications/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/relay_applications/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.relay_applications.get("resource-id")
print(f"Relay Application: {item['display_name']}, ID: {item['id']}")
```
