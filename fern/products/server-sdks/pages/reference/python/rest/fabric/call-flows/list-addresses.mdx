List addresses associated with a call flow resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/call_flow/{id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/call_flow/{id}/addresses" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addresses = client.fabric.call_flows.list_addresses("flow-id")
for addr in addresses.get("data", []):
    print(f"{addr['display_name']}: {addr['id']}")
```
