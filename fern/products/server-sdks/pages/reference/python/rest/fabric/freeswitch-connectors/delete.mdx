Delete a FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/freeswitch_connectors/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.freeswitch_connectors.delete("connector-id")
print("Freeswitch Connector deleted")
```
