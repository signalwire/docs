Create a new cXML webhook resource.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/cxml_webhooks" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/cxml_webhooks" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.cxml_webhooks.create(
    name="my-webhook", primary_request_url="https://example.com/cxml",
)
print(f"Created: {result['id']}")
```
