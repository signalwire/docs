List cXML webhook resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/cxml_webhooks" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/cxml_webhooks" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.cxml_webhooks.list()
for webhook in response.get("data", []):
    print(f"{webhook['display_name']}: {webhook['id']}")
```
