Delete a SWML script resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/swml_scripts/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.swml_scripts.delete("resource-id")
print("Swml Script deleted")
```
