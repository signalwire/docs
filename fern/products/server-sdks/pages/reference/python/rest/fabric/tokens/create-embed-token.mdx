Create an embed authentication token.

<EndpointSchemaSnippet endpoint="POST /api/fabric/embeds/tokens" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/embeds/tokens" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.fabric.tokens.create_embed_token(token="click-to-call-token")
print(f"Embed token: {token['token']}")
```
