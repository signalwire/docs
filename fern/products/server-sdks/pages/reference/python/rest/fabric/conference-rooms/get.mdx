Retrieve a single conference room resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/conference_rooms/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/conference_rooms/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

room = client.fabric.conference_rooms.get("room-id")
print(f"Conference Room: {room['display_name']}, ID: {room['id']}")
```
