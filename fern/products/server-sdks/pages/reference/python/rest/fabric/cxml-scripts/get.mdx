Retrieve a single cXML script resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/cxml_scripts/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/cxml_scripts/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.cxml_scripts.get("resource-id")
print(f"Cxml Script: {item['name']}, ID: {item['id']}")
```
