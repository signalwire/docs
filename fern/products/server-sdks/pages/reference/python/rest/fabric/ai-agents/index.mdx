Manage AI agent resources. Standard CRUD with PATCH updates and address listing.

Access via `client.fabric.ai_agents` on a [`RestClient`][restclient] instance.

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

items = client.fabric.ai_agents.list()
```
