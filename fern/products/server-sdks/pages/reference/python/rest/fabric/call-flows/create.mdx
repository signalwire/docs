## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.call_flows.create(
    title="Main IVR Flow",
)
print(f"Created: {result['id']}")
```
