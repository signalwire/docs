Partially update a SIP gateway resource. Uses PATCH for partial.

<EndpointSchemaSnippet endpoint="PATCH /api/fabric/resources/sip_gateways/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PATCH /api/fabric/resources/sip_gateways/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.sip_gateways.update("gateway-id", name="updated-name")
print(f"Updated: {updated['display_name']}")
```
