Update a number group. Uses PUT to replace fields.

<EndpointSchemaSnippet endpoint="PUT /api/relay/rest/number_groups/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/relay/rest/number_groups/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.number_groups.update("group-id", name="Support Numbers")
print(updated["name"])
```
