Create a new queue.

<EndpointSchemaSnippet endpoint="POST /api/relay/rest/queues" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/relay/rest/queues" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

queue = client.queues.create(name="support", max_size=50)
print("Queue ID:", queue["id"])
```
