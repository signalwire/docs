Update a queue. Uses PUT.

<EndpointSchemaSnippet endpoint="PUT /api/relay/rest/queues/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/relay/rest/queues/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.queues.update("queue-id", name="priority-support")
print(f"Updated: {updated['id']}")
```
