List members currently in a queue.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/queues/{queue_id}/members" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/queues/{queue_id}/members" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

members = client.queues.list_members("queue-id")
for m in members.get("data", []):
    print(m.get("call_id"), m.get("position"))
```
