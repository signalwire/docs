Retrieve a specific message log entry.

<EndpointSchemaSnippet endpoint="GET /api/messaging/logs/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/messaging/logs/{id}" />

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

msg = client.logs.messages.get("message-log-id")
print(msg.get("from"), "->", msg.get("to"))
```
