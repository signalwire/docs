List message log entries.

<EndpointSchemaSnippet endpoint="GET /api/messaging/logs" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/messaging/logs" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

messages = client.logs.messages.list(page_size=20)
for msg in messages.get("data", []):
    print(msg.get("from"), "->", msg.get("to"), msg.get("status"))
```
