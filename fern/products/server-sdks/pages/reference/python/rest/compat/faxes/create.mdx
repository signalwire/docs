## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

fax = client.compat.faxes.create(
    To="+15559876543",
    From="+15551234567",
    MediaUrl="https://example.com/document.pdf"
)
```
