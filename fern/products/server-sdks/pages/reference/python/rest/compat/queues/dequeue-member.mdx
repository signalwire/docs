Dequeue a member from a queue, redirecting the call to a new LAML URL.

<EndpointSchemaSnippet endpoint="POST /Accounts/{AccountSid}/Queues/{QueueSid}/Members/{CallSid}" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /Accounts/{AccountSid}/Queues/{QueueSid}/Members/{CallSid}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.queues.dequeue_member(
    "QU...", "CA...",
    Url="https://example.com/dequeue"
)
```
