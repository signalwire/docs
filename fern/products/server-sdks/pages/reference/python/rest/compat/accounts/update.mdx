Update an account. Uses POST (Twilio convention).

<EndpointSchemaSnippet endpoint="POST /Accounts/{Sid}" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /Accounts/{Sid}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.accounts.update("account-sid", FriendlyName="Sales Team")
```
