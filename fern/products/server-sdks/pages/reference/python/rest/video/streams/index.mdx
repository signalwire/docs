Top-level stream management for retrieving, updating, and deleting individual
streams regardless of which room or conference they belong to.

Access via `client.video.streams` on a [`RestClient`][restclient] instance.

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

stream = client.video.streams.get("stream-id")
```
