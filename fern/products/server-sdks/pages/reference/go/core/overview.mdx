## Idioms

- **Functional options.** Constructors accept variadic option functions like
  `agent.WithName(...)`, `agent.WithRoute(...)`. Each option sets one field on
  the receiver. Pass any combination, in any order.
- **Fluent builders.** Setter methods on `*AgentBase`, `*FunctionResult`,
  `*ContextBuilder`, and `*DataMap` return the receiver for chaining.
- **Errors as values.** Methods return `error` instead of panicking. Always
  check the second return value.
- **Context propagation.** Long-running and cancellable operations accept
  `context.Context` as the first argument.
- **Pointer receivers.** All builder types are passed by pointer
  (`*AgentBase`, `*Call`, `*Document`).
