Return the names of every SWAIG tool registered on the agent in the order they were added. Useful when wiring up [contexts][contextbuilder] that restrict which tools are callable per step.

<Note>
This method is Go-specific and satisfies the `contexts.ToolLister` interface. Python's `AgentBase` does not expose an equivalent — context steps that need a tool list pull it from `_swaig_functions` directly.
</Note>

## **Parameters**

None.

## **Returns**

`[]string` — a fresh slice containing every registered tool name in insertion order. Mutating the returned slice does not affect the agent's registry.

## **Example**

```go {12}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("multi-tool"))
	a.DefineTool(agent.ToolDefinition{Name: "lookup_order"})
	a.DefineTool(agent.ToolDefinition{Name: "cancel_order"})
	fmt.Println(a.ListToolNames()) // [lookup_order cancel_order]
}
```
