[ref-agentbase]: /docs/server-sdks/reference/go/agents/agent-base

Set a callback that runs on every incoming SWML request, receiving an ephemeral copy of the agent so you can dynamically configure **any** aspect of it — skills, prompts, parameters, languages, tools, global data, etc. — based on the request's query params, body, or headers.

This is the primary mechanism for multi-tenant or per-caller customization.

<Note>
The `agent` argument passed to the callback is an **ephemeral clone** of the original agent (produced by `(*AgentBase).clone`). Changes made inside the callback apply only to the current request and do not persist across calls; the original agent is untouched. The callback runs **before** rendering — the framework calls [`RenderSWML`](/docs/server-sdks/reference/go/agents/agent-base/render-swml) on the clone and returns its result.
</Note>

## **Parameters**

<ParamField path="cb" type="DynamicConfigCallback" required={true} toc={true}>
  A function with the signature `func(queryParams map[string]string, bodyParams map[string]any, headers map[string]string, agent *AgentBase)`. Each header and query parameter map captures only the **first** value where multiple are present (`r.URL.Query()` and `r.Header` are flattened by the framework). Mutate the `agent` argument to apply configuration to this request only:

    - `agent.AddSkill("...", nil)`
    - `agent.AddLanguage(...)`
    - `agent.PromptAddSection(...)`
    - `agent.SetParams(...)`
    - `agent.SetGlobalData(...)`
    - `agent.DefineTool(...)`
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

### Per-request tier customization

```go {21}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
	a.SetPromptText("You are a helpful assistant.")

	configure := func(query map[string]string, body map[string]any, headers map[string]string, ag *agent.AgentBase) {
		tier := query["tier"]
		if tier == "" {
			tier = "standard"
		}
		if tier == "premium" {
			ag.AddSkill("web_search", nil)
			ag.SetParams(map[string]any{"end_of_speech_timeout": 500})
		}
		ag.SetGlobalData(map[string]any{"tier": tier})
	}

	a.SetDynamicConfigCallback(configure)
	_ = a.Serve()
}
```
