Replace the entire list of remote SWAIG function includes at once. Each entry should contain a `"url"` and a `"functions"` list; optional keys (e.g. `"meta_data"`) are rendered through unchanged.

<Info>
This maps to the SWML [`ai.swaig.includes`][ai-swaig-includes] array.
</Info>

<Tip>
Use [`AddFunctionInclude`][add-function-include] to append a single include without replacing existing ones.
</Tip>

<Note>
Entries are stored as-is and forwarded to the rendered SWML — there is no add-time validation. The platform rejects malformed includes when SWML schema validation is enabled (see `WithSchemaValidation`); ensure each entry follows the SWAIG include schema before passing it in.
</Note>

## **Parameters**

<ParamField path="includes" type="[]map[string]any" required={true} toc={true}>
  Slice of include objects. Each object should have:

    <Indent>
      <ParamField path="url" type="string" required={true}>
        Remote SWAIG server URL.
      </ParamField>

      <ParamField path="functions" type="[]string">
        Function names to include from that server.
      </ParamField>

      <ParamField path="meta_data" type="map[string]any">
        Optional metadata passed through unchanged.
      </ParamField>
    </Indent>

  Pass an empty or `nil` slice to clear all previously registered includes.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {8-17}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
	a.SetPromptText("You are a helpful assistant.")
	a.SetFunctionIncludes([]map[string]any{
		{
			"url":       "https://tools.example.com/swaig",
			"functions": []string{"lookup_order"},
		},
		{
			"url":       "https://billing.example.com/swaig",
			"functions": []string{"get_invoice", "process_refund"},
		},
	})
	_ = a.Serve()
}
```
