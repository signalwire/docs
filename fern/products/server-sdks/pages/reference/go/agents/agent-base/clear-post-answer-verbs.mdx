[ref-agentbase]: /docs/server-sdks/reference/go/agents/agent-base
[add-post-answer-verb]: /docs/server-sdks/reference/go/agents/agent-base/add-post-answer-verb

Remove every verb previously registered with
[`AddPostAnswerVerb`][add-post-answer-verb]. After this call the post-answer
verb list is empty until new verbs are appended.

## **Parameters**

None.

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {13}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")
	a.AddPostAnswerVerb("play", map[string]any{
		"url": "say:Welcome to Acme Corporation.",
	})
	a.ClearPostAnswerVerbs()

	_ = a.Serve()
}
```
