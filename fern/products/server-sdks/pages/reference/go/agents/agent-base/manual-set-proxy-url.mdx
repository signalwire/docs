[ref-agentbase]: /docs/server-sdks/reference/go/agents/agent-base

Manually set the proxy URL base used when constructing webhook callback URLs in the SWML document. Use this when your agent is behind a reverse proxy, tunnel, or load balancer and the auto-detected URL is incorrect.

<Tip>
You can also set the `SWML_PROXY_URL_BASE` environment variable for the same effect — `NewAgentBase` reads it during construction when no value has been supplied.
</Tip>

## **Parameters**

<ParamField path="url" type="string" required={true} toc={true}>
  The base URL to use for webhooks (e.g., `"https://abc123.ngrok.io"`). Trailing slashes are stripped at render time when the SWAIG webhook URL is assembled.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")
	a.ManualSetProxyUrl("https://abc123.ngrok.io")

	_ = a.Serve()
}
```
