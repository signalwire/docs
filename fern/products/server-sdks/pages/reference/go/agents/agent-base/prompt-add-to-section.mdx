Append text and/or bullets to an existing POM prompt section identified by title.

<Note>
Unlike Python's `prompt_add_to_section`, the Go method **does not create** the section if it is missing — the call is a no-op when no section matches `title`. Use `PromptAddSection` to create the section first, or call `PromptHasSection` to check.
</Note>

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Title of the existing section to update.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  Text appended to the section body. When the section already has a body, the new text is joined with a newline. Pass `""` to skip the body update.
</ParamField>

<ParamField path="opts" type="...PromptAddToSectionOption" toc={true}>
  Variadic functional options. Available options:
  <Indent>
    <ParamField path="WithBullet" type="func(string) PromptAddToSectionOption">
      Add a single bullet point to the section's bullet list.
    </ParamField>
    <ParamField path="WithBullets" type="func([]string) PromptAddToSectionOption">
      Add multiple bullet points to the section's bullet list.
    </ParamField>
  </Indent>
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining (always; no error is signaled when the section is missing).

## **Example**

```go {12,13}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)

	a.PromptAddSection("Rules", "", []string{"Be polite"})
	a.PromptAddToSection("Rules", "", agent.WithBullet("Never lie"))
	a.PromptAddToSection("Rules", "", agent.WithBullets([]string{
		"Keep responses concise",
		"Ask clarifying questions when needed",
	}))

	_ = a.Serve()
}
```
