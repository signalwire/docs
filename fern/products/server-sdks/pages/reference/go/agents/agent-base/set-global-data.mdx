Replace the agent's global data map. The data is forwarded into the rendered SWML's `ai.global_data` field and is available to the AI verb throughout the conversation. This method is thread-safe — concurrent calls from goroutines or dynamic-config callbacks are serialized through the agent's internal mutex.

<Warning>
**Replaces, does not merge.** This method overwrites the entire map and drops any keys set previously (by skills, prior calls, etc.). To incrementally add or update keys without dropping the rest, use [`UpdateGlobalData`][update-global-data] instead.
</Warning>

## **Parameters**

<ParamField path="data" type="map[string]any" required={true} toc={true}>
  Map of key/value pairs that becomes the agent's global data. Values can be any JSON-serializable type. Pass an empty or `nil` map to clear all previously set global data.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {8-12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
	a.SetPromptText("You are a helpful assistant.")
	a.SetGlobalData(map[string]any{
		"company_name":  "Acme Corp",
		"support_hours": "9am-5pm EST",
		"user_tier":     "standard",
	})
	_ = a.Serve()
}
```
