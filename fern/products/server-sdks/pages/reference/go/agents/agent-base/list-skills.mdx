List the names of every skill currently loaded on this agent.

## **Parameters**

None.

## **Returns**

`[]string` — slice of loaded skill names.

## **Example**

```go {12}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
	a.AddSkill("datetime", nil)
	a.AddSkill("math", nil)
	fmt.Println(a.ListSkills()) // [datetime math]
}
```
