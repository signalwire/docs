Retrieve the username and password used for HTTP Basic Auth on this agent's
endpoints. The credentials are sourced (in priority order) from the values
explicitly configured on the underlying SWML service, then from the
`SWML_BASIC_AUTH_USER` / `SWML_BASIC_AUTH_PASSWORD` environment variables,
and finally from auto-generated values created at startup.

<Note>
Unlike the Python SDK, Go's `GetBasicAuthCredentials` does not accept an
`include_source` flag — it always returns only the credentials. The
underlying `swml.Service.GetBasicAuthCredentialsWithSource` returns a third
`source` string, but it is not exposed on `AgentBase`.
</Note>

## **Parameters**

None.

## **Returns**

`(string, string)` — the configured `(username, password)` pair.

## **Example**

```go {12}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
	a.SetPromptText("You are a helpful assistant.")
	user, password := a.GetBasicAuthCredentials()
	fmt.Printf("Auth: %s:%s\n", user, password)

	_ = a.Serve()
}
```
