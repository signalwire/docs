Add query parameters that will be included on every SWAIG webhook URL
generated by this agent. Especially useful for preserving dynamic
configuration state across SWAIG callbacks — for example, passing a tenant
identifier or feature tier so the same configuration is applied when
SignalWire invokes tool webhooks.

<Note>
Subsequent calls **merge into** (not replace) the existing set. Pass the same
key again to overwrite its value. To wipe every entry, use
[`ClearSwaigQueryParams`][clear-swaig-query-params].
</Note>

## **Parameters**

<ParamField path="params" type="map[string]string" required={true} toc={true}>
  Query parameters to merge into every SWAIG URL.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Examples**

### Preserve tier in dynamic config

```go {16}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.SetDynamicConfigCallback(func(qp map[string]string, body map[string]any, headers map[string]string, ag *agent.AgentBase) {
		tier := qp["tier"]
		if tier == "premium" {
			ag.AddSkill("advanced_search", nil)
		}
		ag.AddSwaigQueryParams(map[string]string{"tier": tier})
	})

	_ = a.Serve()
}
```

### Multi-tenant routing

```go {13}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.SetDynamicConfigCallback(func(qp map[string]string, body map[string]any, headers map[string]string, ag *agent.AgentBase) {
		ag.AddSwaigQueryParams(map[string]string{"tenant_id": qp["tenant_id"]})
	})

	_ = a.Serve()
}
```

See [`SetDynamicConfigCallback`][set-dynamic-config-callback] for the
callback signature details.
