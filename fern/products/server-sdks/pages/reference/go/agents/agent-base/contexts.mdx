Return the agent's [`ContextBuilder`][contextbuilder] for defining multi-step
conversation workflows, creating one if it does not already exist. `Contexts`
is a thin alias for [`DefineContexts`][define-contexts] — both methods return
the same builder instance.

<Tip>
Use `Contexts` when reading the agent fluently in a context-building chain
(`a.Contexts().AddContext("default")...`). Use [`DefineContexts`][define-contexts]
when the call site is explicitly creating contexts for the first time. The
underlying behavior is identical.
</Tip>

## **Parameters**

None.

## **Returns**

[`*contexts.ContextBuilder`][contextbuilder] — the agent's attached context
builder. The builder is created on first call and reused on subsequent calls.

## **Example**

```go {12-14}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.Contexts().
		AddContext("default").
		SetPrompt("Greet the caller and ask how you can help.")

	_ = a.Serve()
}
```
