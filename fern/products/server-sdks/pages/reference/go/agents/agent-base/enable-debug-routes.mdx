Reserved hook for opting into agent-mounted debug HTTP routes.

<Warning>
In the current Go SDK, `EnableDebugRoutes` is a no-op placeholder. The method
exists so application code can call it now and stay forward-compatible once
the SDK begins registering `/debug` routes on the agent's mux. Until that
ships, debug endpoints must be added to your own `http.ServeMux` if you need
them.
</Warning>

## **Parameters**

None.

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {11}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("dev-agent"),
		agent.WithRoute("/dev"),
	)
	a.SetPromptText("You are a helpful assistant.")
	a.EnableDebugRoutes()

	_ = a.Serve()
}
```
