Replace all language configurations at once with a slice of raw language maps. Each entry is forwarded into the rendered SWML's `ai.languages` array unchanged.

<Tip>
Prefer [`AddLanguage`][add-language] (or [`AddLanguageTyped`][add-language-typed]) for incremental, ParamField-style additions. Reach for `SetLanguages` when you already have a serialized list (for example, loaded from JSON config) or want to bulk-replace the agent's language roster in a dynamic-config callback.
</Tip>

## **Parameters**

<ParamField path="languages" type="[]map[string]any" required={true} toc={true}>
  Slice of language configuration maps. Each map should include `name`, `code`, and `voice`, and optionally `engine`, `model`, `speech_fillers`, and `function_fillers`. Pass an empty or `nil` slice to clear all previously configured languages.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {8-12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
	a.SetPromptText("You are a helpful assistant.")
	a.SetLanguages([]map[string]any{
		{"name": "English", "code": "en-US", "voice": "rime.spore"},
		{"name": "Spanish", "code": "es-MX", "voice": "rime.luna"},
		{"name": "French", "code": "fr-FR", "voice": "rime.soleil"},
	})
	_ = a.Serve()
}
```
