Add multiple speech recognition hints in a single call. Each hint is appended
to the agent's existing hint list (no replacement). For a single hint, use
[`AddHint`][add-hint].

## **Parameters**

<ParamField path="hints" type="[]string" required={true} toc={true}>
  Words or phrases to boost recognition accuracy.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddHints([]string{"SWML", "SWAIG", "FreeSWITCH"})

	_ = a.Serve()
}
```
