Set the list of native SWAIG function names to enable. Native functions are built into the SignalWire platform and execute server-side without requiring a webhook — they cover common utility operations the AI can invoke during a conversation. The slice is forwarded into the rendered SWML's `ai.native_functions` array.

<Note>
The same list can also be supplied at construction time via the `WithNativeFunctions(names []string)` functional option on [`NewAgentBase`][ref-agentbase].
</Note>

## **Parameters**

<ParamField path="names" type="[]string" required={true} toc={true}>
  Slice of native function names to enable. Common native functions include:

    - `"check_time"` — Get the current time in a given timezone.
    - `"wait_for_user"` — Pause the AI and wait for the caller to speak.
    - `"next_step"` — Advance to the next step in a context workflow.
    - `"transfer"` — Transfer the call to another number or agent.

  Pass an empty or `nil` slice to clear all previously enabled native functions.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Examples**

### Enable native functions on an existing agent

```go {8}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
	a.SetPromptText("You are a helpful assistant.")
	a.SetNativeFunctions([]string{"check_time", "wait_for_user"})
	_ = a.Serve()
}
```

### Or set at construction time

```go {9}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
		agent.WithNativeFunctions([]string{"check_time", "wait_for_user"}),
	)
	a.SetPromptText("You are a helpful assistant.")
	_ = a.Serve()
}
```
