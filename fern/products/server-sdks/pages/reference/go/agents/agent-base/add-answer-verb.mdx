Configure the answer verb that connects the call. Use this to customize answer
behavior such as setting a maximum call duration.

Each call merges its keys into the existing answer-verb configuration, so
subsequent calls accumulate (or overwrite individual keys) rather than
replacing the whole map.

<Info>
The merged configuration maps to the SWML [`answer`][ai-answer] verb that runs
at the start of the rendered document.
</Info>

## **Parameters**

<ParamField path="config" type="map[string]any" required={true} toc={true}>
  Answer verb configuration (e.g. `map[string]any{"max_duration": 3600}`).
  Keys are merged into the agent's existing answer config.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddAnswerVerb(map[string]any{"max_duration": 3600}) // 1 hour max call duration

	_ = a.Serve()
}
```
