Check whether a named section already exists in the agent's POM (Prompt Object Model) prompt. Useful for conditionally adding content to avoid duplicate sections.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  The section title to look up. Must match the title passed to [`PromptAddSection`][prompt-add-section].
</ParamField>

## **Returns**

`bool` — `true` if a section with the given title exists, `false` otherwise.

## **Examples**

### Guard against duplicate sections

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
	)

	a.PromptAddSection("Greeting", "Always greet the caller by name.", nil)
	if !a.PromptHasSection("Policies") {
		a.PromptAddSection("Policies", "Follow company guidelines.", nil)
	}

	_ = a.Serve()
}
```

### Conditional prompt assembly in a helper

```go {6}
package support

import "github.com/signalwire/signalwire-go/pkg/agent"

func AddEscalationRules(a *agent.AgentBase) {
	if !a.PromptHasSection("Escalation") {
		a.PromptAddSection("Escalation", "", []string{
			"Transfer to a human if the caller asks three times.",
			"Always confirm before transferring.",
		})
	}
}
```
