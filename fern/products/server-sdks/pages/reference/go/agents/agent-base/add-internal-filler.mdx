Add filler phrases for a specific internal/native SWAIG function and language.
The AI selects randomly from the supplied list each time the function is
invoked, masking the latency of the call with a spoken phrase.

Each call **replaces** the filler slice for the given `(funcName, langCode)`
pair; calls with different language codes coexist on the same function.

<Note>
`funcName` must be a member of `agent.SupportedInternalFillerNames` (e.g.
`"next_step"`, `"check_time"`, `"wait_for_user"`). Unknown names log a
warning and are stored but do not play at runtime. To replace the entire
filler map, use [`SetInternalFillers`][set-internal-fillers].
</Note>

## **Parameters**

<ParamField path="funcName" type="string" required={true} toc={true}>
  Name of the internal SWAIG function (e.g. `"next_step"`, `"check_time"`).
</ParamField>

<ParamField path="langCode" type="string" required={true} toc={true}>
  Language code (e.g. `"en-US"`, `"es"`, `"fr"`).
</ParamField>

<ParamField path="fillers" type="[]string" required={true} toc={true}>
  Filler phrases for this function and language. The AI randomly selects from
  this list each invocation.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12-14,16-18}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddInternalFiller("next_step", "en-US", []string{
		"Moving to the next step...", "Great, let's continue...",
	})
	a.AddInternalFiller("next_step", "es", []string{
		"Pasando al siguiente paso...", "Continuemos...",
	})

	_ = a.Serve()
}
```
