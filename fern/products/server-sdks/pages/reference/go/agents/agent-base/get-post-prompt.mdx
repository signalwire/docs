Retrieve the post-prompt text most recently set with
[`SetPostPrompt`][set-post-prompt]. The post-prompt is the instruction the AI
runs at end-of-call to produce a structured summary.

## **Parameters**

None.

## **Returns**

`string` — the post-prompt text, or an empty string if none has been set.

## **Example**

```go {12}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
	a.SetPostPrompt("Summarize this call as JSON with intent, resolution, and sentiment.")
	if pp := a.GetPostPrompt(); pp != "" {
		fmt.Println(pp)
	}
}
```
