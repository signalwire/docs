[ref-agentbase]: /docs/server-sdks/reference/go/agents/agent-base

Append a new section to the prompt. Sections give the AI structured instructions that are easier to follow than a single block of text. Each section has a title and optional body text, bullet points, and subsections. The agent automatically switches to POM mode the first time `PromptAddSection` is called.

<Warning>
POM sections cannot be mixed with `SetPromptText` on the same agent — `SetPromptText` clears POM mode and `PromptAddSection` re-enables it. Use one approach or the other.
</Warning>

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section heading displayed to the AI.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  Section body text. Pass `""` to omit the body.
</ParamField>

<ParamField path="bullets" type="[]string" required={true} toc={true}>
  List of bullet point strings. Pass `nil` to omit bullets.
</ParamField>

<ParamField path="opts" type="...PromptSectionOption" toc={true}>
  Variadic functional options. Available options:
  <Indent>
    <ParamField path="WithNumbered" type="func(bool) PromptSectionOption">
      Render the section itself with a numeric marker.
    </ParamField>
    <ParamField path="WithNumberedBullets" type="func(bool) PromptSectionOption">
      Render the bullet list with numeric markers instead of bullets.
    </ParamField>
    <ParamField path="WithSubsections" type="func([]map[string]any) PromptSectionOption">
      Attach child sections. Each map should carry `title`, optional `body`, and optional `bullets`.
    </ParamField>
  </Indent>
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {12,15-19}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)

	a.
		PromptAddSection("Role", "You are a customer support agent for Acme Corp.", nil).
		PromptAddSection(
			"Guidelines",
			"",
			[]string{
				"Always greet the caller by name if available",
				"Never share internal pricing details",
				"Escalate to a human if the caller asks",
			},
		)

	_ = a.Serve()
}
```
