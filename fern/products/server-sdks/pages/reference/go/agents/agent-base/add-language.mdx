Add a language configuration as a raw map. The agent uses these configurations
to support multilingual conversations with appropriate TTS voices.

The configuration is appended to the agent's language list verbatim — no
validation is performed on individual keys.

<Info>
Language configurations map to the SWML [`ai.languages`][ai-languages] array.
See the [SWML languages reference][swml-languages-reference] for the full
specification of supported keys.
</Info>

<Tip>
For a typed signature that matches Python's `add_language(name, code, voice,
...)`, use [`AddLanguageTyped`][add-language-typed].
</Tip>

## **Parameters**

<ParamField path="config" type="map[string]any" required={true} toc={true}>
  Language configuration map. Common keys:
  - `"name"` (`string`) — display name (e.g. `"English"`).
  - `"code"` (`string`) — BCP-47 code (e.g. `"en-US"`).
  - `"voice"` (`string`) — TTS voice. Accepts a simple name (`"en-US-Neural2-F"`),
    an `"engine.voice:model"` combined string, or a short name when paired with
    explicit `"engine"` / `"model"` keys.
  - `"engine"` (`string`, optional) — explicit TTS engine.
  - `"model"` (`string`, optional) — explicit TTS model.
  - `"speech_fillers"` (`[]string`, optional) — fillers spoken during natural speech pauses.
  - `"function_fillers"` (`[]string`, optional) — fillers spoken while SWAIG functions execute.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Examples**

### Simple voice

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddLanguage(map[string]any{
		"name":  "English",
		"code":  "en-US",
		"voice": "rime.spore",
	})

	_ = a.Serve()
}
```

### Explicit engine and model

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)

	a.AddLanguage(map[string]any{
		"name":   "English",
		"code":   "en-US",
		"voice":  "josh",
		"engine": "elevenlabs",
		"model":  "eleven_turbo_v2_5",
	})

	_ = a.Serve()
}
```

### With fillers

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddLanguage(map[string]any{
		"name":             "English",
		"code":             "en-US",
		"voice":            "rime.spore",
		"speech_fillers":   []string{"Um...", "Let me think..."},
		"function_fillers": []string{"One moment please...", "Looking that up..."},
	})

	_ = a.Serve()
}
```
