Get the agent's display name as set at construction time via `WithName`.

## **Parameters**

None.

## **Returns**

`string` — the agent name.

## **Example**

```go {11}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
	fmt.Println(a.GetName()) // "support"
}
```
