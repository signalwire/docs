[ref-agentbase]: /docs/server-sdks/reference/go/agents/agent-base
[add-swaig-query-params]: /docs/server-sdks/reference/go/agents/agent-base/add-swaig-query-params

Clear every parameter previously added with
[`AddSwaigQueryParams`][add-swaig-query-params]. After this call SWAIG webhook
URLs no longer include any extra query parameters until new ones are added.

## **Parameters**

None.

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Examples**

### Reset params in dynamic config

```go {15}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.SetDynamicConfigCallback(func(qp map[string]string, body map[string]any, headers map[string]string, ag *agent.AgentBase) {
		// Start fresh each request to avoid stale params.
		ag.ClearSwaigQueryParams()
		tier, ok := qp["tier"]
		if !ok {
			tier = "free"
		}
		ag.AddSwaigQueryParams(map[string]string{"tier": tier})
	})

	_ = a.Serve()
}
```
