Connect the agent to an external [Model Context Protocol](https://modelcontextprotocol.io/)
(MCP) server. Tools are discovered via the MCP protocol at session start and
automatically registered as SWAIG functions. Optionally, the server's resources
can be fetched into the agent's global data.

<Note>
This method connects your agent **to** an MCP server as a client. To expose
your agent's own tools **as** an MCP server, use
[`EnableMcpServer`][enable-mcp-server].
</Note>

## **Parameters**

<ParamField path="cfg" type="agent.MCPServerConfig" required={true} toc={true}>
  Configuration struct for the external MCP server connection. Fields:
</ParamField>

<Indent>
<ParamField path="cfg.URL" type="string" required={true} toc={true}>
  MCP server HTTP endpoint URL.
</ParamField>

<ParamField path="cfg.Headers" type="map[string]string" toc={true}>
  HTTP headers sent with every request to the MCP server (e.g.
  `{"Authorization": "Bearer sk-xxx"}`). Omitted from the rendered config when
  empty.
</ParamField>

<ParamField path="cfg.Resources" type="bool" default="false" toc={true}>
  When `true`, fetch the server's resources into the agent's global data.
</ParamField>

<ParamField path="cfg.ResourceVars" type="map[string]string" toc={true}>
  Variables for URI template substitution when fetching resources. Supports
  SignalWire call variables such as `${caller_id_number}`. Omitted when empty.
</ParamField>
</Indent>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12-17}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddMcpServer(agent.MCPServerConfig{
		URL:          "https://mcp.example.com/sse",
		Headers:      map[string]string{"Authorization": "Bearer sk-xxx"},
		Resources:    true,
		ResourceVars: map[string]string{"caller_id": "${caller_id_number}"},
	})

	_ = a.Serve()
}
```
