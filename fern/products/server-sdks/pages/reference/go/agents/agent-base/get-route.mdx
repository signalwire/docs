Get the agent's HTTP route path as set at construction time via `WithRoute`. This is the path the SWML and SWAIG handlers are mounted on when `Serve` registers the agent's HTTP routes.

<Note>
  This is a Go-specific accessor. Python's `AgentBase` exposes the route as the `_route` attribute and does not provide a public `get_route` method.
</Note>

## **Parameters**

None.

## **Returns**

`string` — the route path (e.g., `/support`).

## **Example**

```go {11}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
	fmt.Println(a.GetRoute()) // "/support"
}
```
