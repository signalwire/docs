[ref-agentbase]: /docs/server-sdks/reference/go/agents/agent-base
[add-pre-answer-verb]: /docs/server-sdks/reference/go/agents/agent-base/add-pre-answer-verb

Remove every verb previously registered with
[`AddPreAnswerVerb`][add-pre-answer-verb]. After this call the pre-answer verb
list is empty until new verbs are appended.

## **Parameters**

None.

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {14}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")
	a.AddPreAnswerVerb("play", map[string]any{
		"urls":        []string{"ring:us"},
		"auto_answer": false,
	})
	a.ClearPreAnswerVerbs()

	_ = a.Serve()
}
```
