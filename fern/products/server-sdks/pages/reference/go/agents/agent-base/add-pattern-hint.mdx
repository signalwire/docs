Add a hint with pattern matching and replacement. This intercepts specific
ASR output and rewrites it before it reaches the AI — useful for correcting
consistent misrecognitions or normalizing variations. For plain hints (no
pattern), use [`AddHint`][add-hint].

## **Parameters**

<ParamField path="hint" type="string" required={true} toc={true}>
  The hint string the model receives after substitution.
</ParamField>

<ParamField path="pattern" type="string" required={true} toc={true}>
  Regular expression pattern to match against the raw ASR output.
</ParamField>

<ParamField path="replace" type="string" required={true} toc={true}>
  Replacement text substituted when the pattern matches.
</ParamField>

<ParamField path="ignoreCase" type="...bool" default="false" toc={true}>
  Variadic flag. Pass `true` for case-insensitive matching. Only the first
  argument is read; further values are ignored. Defaults to `false`.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12-17}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddPatternHint(
		"SignalWire",
		`signal\s*wire|signal\s*wired`,
		"SignalWire",
		true, // ignoreCase
	)

	_ = a.Serve()
}
```
