Append a subsection under an existing parent POM section identified by `parentTitle`.

<Note>
Unlike Python's `prompt_add_subsection`, the Go method **does not create** the parent section if it is missing — the call is a no-op when no section matches `parentTitle`. Call `PromptAddSection` first to create the parent.
</Note>

## **Parameters**

<ParamField path="parentTitle" type="string" required={true} toc={true}>
  Title of the parent section.
</ParamField>

<ParamField path="title" type="string" required={true} toc={true}>
  Subsection heading.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  Subsection body text. Pass `""` to omit.
</ParamField>

<ParamField path="bullets" type="[]string" required={true} toc={true}>
  Subsection bullet points. Pass `nil` to omit.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {12,14}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)

	a.PromptAddSection("Capabilities", "You can help with the following:", nil)
	a.PromptAddSubsection("Capabilities", "Orders",
		"Look up order status, process returns and exchanges.", nil)
	a.PromptAddSubsection("Capabilities", "Billing",
		"Check account balances and explain recent charges.", nil)

	_ = a.Serve()
}
```
