Add a single speech recognition hint. Hints boost the automatic speech
recognition (ASR) engine's accuracy for specific words or phrases that may
otherwise be misrecognized — such as product names, technical jargon, or
uncommon proper nouns.

The hint is appended to the agent's hint list. To add several at once, use
[`AddHints`][add-hints]. For regex pattern hints, use
[`AddPatternHint`][add-pattern-hint].

## **Parameters**

<ParamField path="hint" type="string" required={true} toc={true}>
  A word or phrase to boost recognition accuracy.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddHint("SignalWire")

	_ = a.Serve()
}
```
