`AgentBase` is the central struct in the SignalWire AI Agents SDK. It provides
a complete framework for building AI-powered voice agents, combining prompt
management, tool definitions, skill loading, speech configuration, and HTTP
serving into a single composable type.

`AgentBase` embeds an [`*swml.Service`][swmlservice] internally and composes
functionality from `pkg/contexts`, `pkg/skills`, `pkg/security`, and
`pkg/swaig`. All setter methods return the receiver (`*AgentBase`) for fluent
method chaining.

<Info>
AgentBase generates a SWML document with the [`ai`][ai] verb. See the
[SWML reference][swml-reference] for the full specification of all supported
parameters and behaviors.
</Info>

## **Constructor**

```go {6-9}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	_ = a.Run()
}
```

`NewAgentBase(opts ...AgentOption) *AgentBase` constructs an agent and
applies the supplied functional options in order. Pass any combination of
`agent.With*` options below.

### Constructor Parameters

<ParamField path="WithName(name string)" type="AgentOption" toc={true}>
  Sets the agent's display name. Used in logging, SIP username mapping, and
  the default prompt fallback. Defaults to `"Agent"`.
</ParamField>

<ParamField path="WithRoute(route string)" type="AgentOption" toc={true}>
  HTTP route path where this agent is served. Used by
  [`AgentServer`][agentserver] when hosting multiple agents on one process.
  Defaults to `"/"`.
</ParamField>

<ParamField path="WithHost(host string)" type="AgentOption" toc={true}>
  Network interface the HTTP server binds to. Defaults to `"0.0.0.0"`.
</ParamField>

<ParamField path="WithPort(port int)" type="AgentOption" toc={true}>
  Port the HTTP server listens on. Defaults to `3000`.
</ParamField>

<ParamField path="WithBasicAuth(user, password string)" type="AgentOption" toc={true}>
  Explicit Basic Auth credentials for all HTTP endpoints. If not set, the SDK
  reads `SWML_BASIC_AUTH_USER` / `SWML_BASIC_AUTH_PASSWORD` env vars or
  auto-generates credentials on startup.
</ParamField>

<ParamField path="WithAutoAnswer(autoAnswer bool)" type="AgentOption" toc={true}>
  Automatically emit an `answer` verb before the AI verb in the SWML document.
  Defaults to `true`.
</ParamField>

<ParamField path="WithRecordCall(record bool)" type="AgentOption" toc={true}>
  Enable call recording. When `true`, a `record_call` verb is added to the
  SWML document. Defaults to `false`.
</ParamField>

<ParamField path="WithRecordFormat(format string)" type="AgentOption" toc={true}>
  Recording file format. Common values: `"mp4"`, `"wav"`. Defaults to `"mp4"`.
</ParamField>

<ParamField path="WithRecordStereo(stereo bool)" type="AgentOption" toc={true}>
  Record in stereo (separate channels per party) when `true`. Defaults to
  `true`.
</ParamField>

<ParamField path="WithTokenExpiry(secs int)" type="AgentOption" toc={true}>
  Expiration time in seconds for SWAIG function authentication tokens.
  Defaults to `3600`.
</ParamField>

<ParamField path="WithAIVerbName(name string)" type="AgentOption" toc={true}>
  Override the SWML verb name used for the AI section. Default is `"ai"`. Set
  to `"amazon_bedrock"` for the Bedrock variant.
</ParamField>

<ParamField path="WithUsePom(usePom bool)" type="AgentOption" toc={true}>
  Enable Prompt Object Model for structured prompt building. Set to `false`
  to use plain text prompts only. Defaults to `true`.
</ParamField>

<ParamField path="WithDefaultWebhookURL(url string)" type="AgentOption" toc={true}>
  Default webhook URL for SWAIG function callbacks. Used as the fallback for
  any tool that does not specify its own `WebhookURL`.
</ParamField>

<ParamField path="WithAgentID(id string)" type="AgentOption" toc={true}>
  Fixed agent ID. If not provided, a UUID is generated automatically when the
  agent is constructed.
</ParamField>

<ParamField path="WithNativeFunctions(names []string)" type="AgentOption" toc={true}>
  Initial list of native SWAIG function names to enable at construction time
  (e.g. `[]string{"check_time", "wait_for_user"}`). Can also be set later via
  [`SetNativeFunctions`][setnativefunctions].
</ParamField>

<ParamField path="WithSchemaPath(path string)" type="AgentOption" toc={true}>
  Path to a custom SWML JSON Schema file used for validation. Falls back to
  the built-in schema when omitted.
</ParamField>

<ParamField path="WithSuppressLogs(suppress bool)" type="AgentOption" toc={true}>
  Suppress info-level agent lifecycle logs. Useful in testing or when
  integrating with external logging.
</ParamField>

<ParamField path="WithEnablePostPromptOverride(enable bool)" type="AgentOption" toc={true}>
  Register the `/post_prompt_override` route, allowing external callers to
  replace post-prompt summary handling at runtime.
</ParamField>

<ParamField path="WithCheckForInputOverride(enable bool)" type="AgentOption" toc={true}>
  Register the `/check_for_input` route, allowing external systems to inject
  input into an active AI session at runtime.
</ParamField>

<ParamField path="WithConfigFile(path string)" type="AgentOption" toc={true}>
  Path to a YAML/JSON service configuration file. When provided, its values
  are merged with explicit constructor options; explicit options always win.
</ParamField>

<ParamField path="WithSchemaValidation(validate bool)" type="AgentOption" toc={true}>
  Enable SWML schema validation on rendered documents. Defaults to `true`.
  Can also be disabled via the `SWML_SKIP_SCHEMA_VALIDATION=1` env var.
</ParamField>

## **Properties**

<ParamField path="AgentID" type="string" toc={true}>
  Unique identifier for this agent instance. Auto-generated as a UUID when
  not supplied via [`WithAgentID`](#withagentidid-string).
</ParamField>

<ParamField path="Logger" type="*logging.Logger" toc={true}>
  Structured logger for this agent. Inherited from the embedded
  [`*swml.Service`][swmlservice].
</ParamField>

## **Examples**

### Basic agent with a tool

```go {26-43}
package main

import (
	"github.com/signalwire/signalwire-go/pkg/agent"
	"github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support-agent"),
		agent.WithRoute("/support"),
	)

	a.AddLanguage(map[string]any{
		"name":  "English",
		"code":  "en-US",
		"voice": "rime.spore",
	})
	a.SetPromptText("You are a friendly customer support agent.")
	a.AddHints([]string{"SignalWire", "SWML", "SWAIG"})
	a.SetParams(map[string]any{
		"temperature":           0.7,
		"end_of_speech_timeout": 1000,
	})

	a.DefineTool(agent.ToolDefinition{
		Name:        "check_order",
		Description: "Look up the status of a customer order",
		Parameters: map[string]any{
			"type": "object",
			"properties": map[string]any{
				"order_id": map[string]any{
					"type":        "string",
					"description": "The order ID to look up",
				},
			},
		},
		Required: []string{"order_id"},
		Handler: func(args, raw map[string]any) *swaig.FunctionResult {
			orderID, _ := args["order_id"].(string)
			return swaig.NewFunctionResult("Order " + orderID + " shipped on March 28.")
		},
	})

	if err := a.Run(); err != nil {
		panic(err)
	}
}
```

### Embedding for a custom agent

```go {6}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

type SupportAgent struct {
	*agent.AgentBase
}

func NewSupportAgent() *SupportAgent {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.PromptAddSection("Role", "You are a customer support agent for Acme Corp.", nil)
	a.PromptAddSection("Guidelines", "", []string{
		"Be polite and professional",
		"Escalate billing disputes",
	})
	return &SupportAgent{AgentBase: a}
}

func main() {
	if err := NewSupportAgent().Run(); err != nil {
		panic(err)
	}
}
```
