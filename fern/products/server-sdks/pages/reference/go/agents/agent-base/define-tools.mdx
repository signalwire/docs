Return every SWAIG tool currently registered on this agent. The slice contains
tools added through [`DefineTool`][define-tool],
[`RegisterSwaigFunction`][register-swaig-function], and any DataMap tools
attached via skills, in the order they were inserted.

<Note>
Unlike Python's `define_tools` (which is an override hook on `AgentBase`), the
Go method is a read-only accessor. To filter or reorder the tools that ship in
the rendered SWML, either remove tools before calling `Serve` or wrap the
agent with custom rendering logic.
</Note>

## **Parameters**

None.

## **Returns**

`[]*ToolDefinition` — every registered tool in insertion order. The slice is
a fresh copy; mutating it does not affect the agent's internal registry.

## **Example**

```go {13}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("inspector"), agent.WithRoute("/inspect"))
	a.SetPromptText("You are a helpful assistant.")

	for _, tool := range a.DefineTools() {
		fmt.Printf("registered tool: %s\n", tool.Name)
	}

	_ = a.Serve()
}
```
