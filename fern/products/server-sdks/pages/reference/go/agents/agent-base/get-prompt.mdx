Retrieve the current prompt. Returns a `string` if [`SetPromptText`][set-prompt-text] was used, or a POM structure (`[]map[string]any`, defensively copied) if [`PromptAddSection`][prompt-add-section] was used.

<Note>
Unlike Python, the Go SDK does **not** synthesize a default prompt when neither `SetPromptText` nor `PromptAddSection` has been called — it returns an empty string. Type-assert the return based on which prompt API the agent uses.
</Note>

## **Parameters**

None.

## **Returns**

`any` — `string` in raw-text mode, `[]map[string]any` in POM mode. The POM slice is a fresh copy; mutating it does not affect the agent.

## **Example**

```go {12}
package main

import (
	"fmt"

	"github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
	a := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
	a.SetPromptText("You are a customer support agent for Acme Corp.")
	prompt := a.GetPrompt()
	fmt.Println(prompt)
}
```
