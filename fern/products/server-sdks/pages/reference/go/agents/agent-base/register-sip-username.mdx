Register a specific SIP username that should be routed to this agent. Usernames are stored lowercased and matched case-insensitively. Pair with [`EnableSipRouting`][enable-sip-routing] to expose the SIP-routing endpoint, or use [`AutoMapSipUsernames`][auto-map-sip-usernames] to derive usernames from the agent's name and route automatically.

## **Parameters**

<ParamField path="username" type="string" required={true} toc={true}>
  SIP username to register for this agent. Stored as lowercase; lookups are case-insensitive.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for method chaining.

## **Example**

```go {12-13}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a, _ := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")
	a.EnableSipRouting(false, "")
	a.RegisterSipUsername("help-desk").
		RegisterSipUsername("customer-service")
	_ = a.Serve()
}
```
