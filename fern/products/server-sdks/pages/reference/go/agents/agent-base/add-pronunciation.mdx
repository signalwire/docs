Add a pronunciation rule that tells the TTS engine how to speak a specific
word or phrase. Useful for brand names, acronyms, and technical terms that
are commonly mispronounced by text-to-speech engines.

To replace the entire pronunciation list in one call, use
[`SetPronunciations`][set-pronunciations].

## **Parameters**

<ParamField path="replace" type="string" required={true} toc={true}>
  The expression to match in the AI's output text.
</ParamField>

<ParamField path="withText" type="string" required={true} toc={true}>
  The phonetic spelling or alternative text the TTS engine should speak
  instead. Rendered as the `with` key in the SWML pronunciation entry.
</ParamField>

<ParamField path="ignoreCase" type="...bool" default="false" toc={true}>
  Variadic flag. Pass `true` for case-insensitive matching. Only the first
  argument is read; further values are ignored. Defaults to `false`.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Example**

```go {12-13}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("support"),
		agent.WithRoute("/support"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddPronunciation("SQL", "sequel")
	a.AddPronunciation("SWAIG", "swig", true) // ignoreCase

	_ = a.Serve()
}
```
