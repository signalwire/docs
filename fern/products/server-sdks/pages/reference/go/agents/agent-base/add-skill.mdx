Load and activate a skill on this agent. Skills are pluggable capability
modules that register their own tools, prompts, hints, and global data. The
SDK ships with built-in skills for common tasks like `datetime`, `web_search`,
and `math`.

When loading succeeds, every tool returned by the skill's `RegisterTools()`
is registered via `DefineTool`, hints from `GetHints()` are appended via
`AddHints`, global data from `GetGlobalData()` is merged via
`UpdateGlobalData`, and prompt sections from `GetPromptSections()` are
installed via `PromptAddSection` (unless `params["skip_prompt"]` is `true`).

<Note>
If the skill name is not registered, or the skill fails to load, the error is
logged through `a.Logger.Error` and the call returns the receiver unchanged —
it does **not** return an error or panic. Use
[`ListSkills`][list-skills] to confirm the skill loaded, or consult the
[skills catalog][skills-catalog] for all available built-in skills.
</Note>

## **Parameters**

<ParamField path="skillName" type="string" required={true} toc={true}>
  Registered skill name (e.g. `"datetime"`, `"web_search"`, `"math"`).
</ParamField>

<ParamField path="params" type="map[string]any" toc={true}>
  Skill-specific configuration. Pass `nil` for none. Two keys are recognised
  by `AddSkill` itself across every skill:
  - `"skip_prompt"` (`bool`) — when `true`, the skill's prompt sections are
    not installed on the agent.

  Every other key is forwarded to the skill factory verbatim.
</ParamField>

## **Returns**

[`*AgentBase`][ref-agentbase] — returns the receiver for fluent chaining.

## **Examples**

### Add built-in skills

```go {12-13}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddSkill("datetime", nil)
	a.AddSkill("math", nil)

	_ = a.Serve()
}
```

### Skill with configuration

```go {12}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
	a := agent.NewAgentBase(
		agent.WithName("assistant"),
		agent.WithRoute("/assistant"),
	)
	a.SetPromptText("You are a helpful assistant.")

	a.AddSkill("web_search", map[string]any{
		"api_key":     "your-search-api-key",
		"max_results": 5,
	})

	_ = a.Serve()
}
```
