Verify an MFA code submitted by the user.

<EndpointSchemaSnippet endpoint="POST /api/relay/rest/mfa/{mfa_request_id}/verify" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/relay/rest/mfa/{mfa_request_id}/verify" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.mfa.verify("your-mfa-request-id", { token: "123456" });
if (result.success) {
  console.log("Verification successful");
} else {
  console.log("Invalid code");
}
```
