Retrieve a specific message log entry.

<EndpointSchemaSnippet endpoint="GET /api/messaging/logs/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/messaging/logs/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const msg = await client.logs.messages.get("message-log-id");
console.log(msg.from, "->", msg.to);
```
