List message log entries.

<EndpointSchemaSnippet endpoint="GET /api/messaging/logs" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/messaging/logs" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const messages = await client.logs.messages.list({ pageSize: 20 });
for (const msg of messages.data ?? []) {
    console.log(msg.from, "->", msg.to, msg.status);
}
```
