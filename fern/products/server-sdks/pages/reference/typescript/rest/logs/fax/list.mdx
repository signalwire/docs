List fax log entries.

<EndpointSchemaSnippet endpoint="GET /api/fax/logs" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fax/logs" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const faxes = await client.logs.fax.list();
for (const fax of faxes.data ?? []) {
    console.log(fax.from, "->", fax.to, fax.status);
}
```
