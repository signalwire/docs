List conference log entries.

<EndpointSchemaSnippet endpoint="GET /api/logs/conferences" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/logs/conferences" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const confs = await client.logs.conferences.list();
for (const conf of confs.data ?? []) {
    console.log(conf.name, conf.status);
}
```
