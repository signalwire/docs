List recordings for a particular video room session.

<EndpointSchemaSnippet endpoint="GET /api/video/room_sessions/{id}/recordings" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_sessions/{id}/recordings" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const recordings = await client.video.roomSessions.listRecordings("session-id");
for (const recording of recordings.data ?? []) {
  console.log(`${recording.id}: ${recording.status}`);
}
```
