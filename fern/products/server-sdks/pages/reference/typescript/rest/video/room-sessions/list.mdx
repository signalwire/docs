List past and in-progress video room sessions.

<EndpointSchemaSnippet endpoint="GET /api/video/room_sessions" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_sessions" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.video.roomSessions.list();
for (const session of response.data ?? []) {
  console.log(`${session.name}: ${session.id}`);
}
```
