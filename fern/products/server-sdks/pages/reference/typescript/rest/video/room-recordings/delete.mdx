Delete a video room recording.

<EndpointSchemaSnippet endpoint="DELETE /api/video/room_recordings/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.video.roomRecordings.delete("recording-id");
console.log("Recording deleted");
```
