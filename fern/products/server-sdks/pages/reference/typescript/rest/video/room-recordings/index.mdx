Query and manage video room recordings, including their events.

Access via `client.video.roomRecordings` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const items = await client.video.roomRecordings.list();
```
