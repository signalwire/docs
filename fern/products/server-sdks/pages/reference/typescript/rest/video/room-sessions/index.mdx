Query past and in-progress video room sessions, including their events, members, and recordings.

Access via `client.video.roomSessions` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const items = await client.video.roomSessions.list();
```
