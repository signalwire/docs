Create a new stream for a video room.

<EndpointSchemaSnippet endpoint="POST /api/video/rooms/{id}/streams" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/video/rooms/{id}/streams" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const stream = await client.video.rooms.createStream("room-id", {
  url: "rtmp://broadcaster.example.com/live",
});
console.log("Stream created:", stream.id);
```
