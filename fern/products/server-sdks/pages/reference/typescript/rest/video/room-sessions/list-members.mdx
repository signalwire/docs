List members for a particular video room session.

<EndpointSchemaSnippet endpoint="GET /api/video/room_sessions/{id}/members" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_sessions/{id}/members" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const members = await client.video.roomSessions.listMembers("session-id");
for (const member of members.data ?? []) {
  console.log(`${member.name}: ${member.id}`);
}
```
