Find a video room session by ID.

<EndpointSchemaSnippet endpoint="GET /api/video/room_sessions/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_sessions/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const session = await client.video.roomSessions.get("session-id");
console.log("Session:", session.name, "ID:", session.id);
```
