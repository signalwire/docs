Delete a SIP gateway resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/sip_gateways/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.sipGateways.delete("gateway-id");
console.log("Sip Gateway deleted");
```
