List conference room resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/conference_rooms" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/conference_rooms" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const response = await client.fabric.conferenceRooms.list();
for (const room of response.data ?? []) {
  console.log(`${room.display_name}: ${room.id}`);
}
```
