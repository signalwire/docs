Delete a SIP endpoint resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/sip_endpoints/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.sipEndpoints.delete("endpoint-id");
console.log("Sip Endpoint deleted");
```
