Delete a [subscriber](/docs/platform/subscribers) resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/subscribers/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.fabric.subscribers.delete("subscriber-id");
console.log("Subscriber deleted");
```
