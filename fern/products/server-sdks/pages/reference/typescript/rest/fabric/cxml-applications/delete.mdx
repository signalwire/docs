Delete a cXML application resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/cxml_applications/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.cxmlApplications.delete("app-id");
console.log("cXML application deleted");
```
