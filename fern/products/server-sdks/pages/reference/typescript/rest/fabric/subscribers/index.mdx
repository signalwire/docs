Manage [subscriber](/docs/platform/subscribers) resources. Extends standard CRUD (PUT updates) with address listing
and SIP endpoint management.

Access via `client.fabric.subscribers` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const items = await client.fabric.subscribers.list();
```
