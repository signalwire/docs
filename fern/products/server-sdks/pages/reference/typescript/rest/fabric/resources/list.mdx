List all Fabric resources across all resource types.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const response = await client.fabric.resources.list();
for (const resource of response.data ?? []) {
  console.log(`${resource.type}: ${resource.name} (${resource.id})`);
}
```
