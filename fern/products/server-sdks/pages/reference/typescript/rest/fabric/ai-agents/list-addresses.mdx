List addresses associated with an AI agent resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/ai_agents/{ai_agent_id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/ai_agents/{ai_agent_id}/addresses" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const addresses = await client.fabric.aiAgents.listAddresses("agent-id");
for (const addr of addresses.data ?? []) {
  console.log(`${addr.display_name}: ${addr.id}`);
}
```
