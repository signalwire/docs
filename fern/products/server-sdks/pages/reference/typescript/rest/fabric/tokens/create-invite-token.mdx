Create an invite token for a [subscriber](/docs/platform/subscribers).

<EndpointSchemaSnippet endpoint="POST /api/fabric/subscriber/invites" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/subscriber/invites" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const token = await client.fabric.tokens.createInviteToken({ address_id: "address-uuid" });
console.log(`Invite token: ${token.token}`);
```
