Create an embed authentication token.

<EndpointSchemaSnippet endpoint="POST /api/fabric/embeds/tokens" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/embeds/tokens" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const token = await client.fabric.tokens.createEmbedToken({ token: "click-to-call-token" });
console.log(`Embed token: ${token.token}`);
```
