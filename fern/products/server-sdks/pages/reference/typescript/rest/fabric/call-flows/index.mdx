Manage call flow resources with version management and deployment. Extends standard CRUD
(PUT updates) with methods to list versions and deploy specific versions.

Access via `client.fabric.callFlows` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const items = await client.fabric.callFlows.list();
```
