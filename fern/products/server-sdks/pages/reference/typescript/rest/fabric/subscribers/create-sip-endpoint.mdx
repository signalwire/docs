Create a new SIP endpoint for a [subscriber](/docs/platform/subscribers).

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/subscribers/{fabric_subscriber_id}/sip_endpoints" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/subscribers/{fabric_subscriber_id}/sip_endpoints" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.subscribers.createSipEndpoint("subscriber-id", {
  username: "alice-sip",
  password: "s3cret"
});
console.log("Created SIP endpoint:", result.id);
```
