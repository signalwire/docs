Retrieve a single conference room resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/conference_rooms/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/conference_rooms/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const room = await client.fabric.conferenceRooms.get("room-id");
console.log(`Conference Room: ${room.display_name}, ID: ${room.id}`);
```
