Delete a cXML webhook resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/cxml_webhooks/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.cxmlWebhooks.delete("webhook-id");
console.log("Cxml Webhook deleted");
```
