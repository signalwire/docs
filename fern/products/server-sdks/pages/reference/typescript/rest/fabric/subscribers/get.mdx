Retrieve a single [subscriber](/docs/platform/subscribers) resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/subscribers/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/subscribers/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const subscriber = await client.fabric.subscribers.get("subscriber-id");
console.log("Subscriber:", subscriber.display_name, "ID:", subscriber.id);
```
