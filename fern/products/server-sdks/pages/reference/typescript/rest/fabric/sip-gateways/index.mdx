Manage SIP gateway resources. Standard CRUD with PATCH updates and address listing.

Access via `client.fabric.sipGateways` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const items = await client.fabric.sipGateways.list();
```
