Manage RELAY application resources. Standard CRUD with PUT updates and address listing.

Access via `client.fabric.relayApplications` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const items = await client.fabric.relayApplications.list();
```
