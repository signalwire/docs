Create a new SIP endpoint resource.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/sip_endpoints" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/sip_endpoints" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.sipEndpoints.create({
  username: "my-endpoint",
  caller_id: "+15551234567",
  send_as: "+15551234567",
  encryption: "required",
  ciphers: ["AEAD_AES_256_GCM_8"],
  codecs: ["OPUS"],
  call_handler: "relay_app",
  calling_handler_resource_id: "resource-id",
});
console.log(`Created: ${result.id}`);
```
