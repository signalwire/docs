Update a queue. Uses PUT.

<EndpointSchemaSnippet endpoint="PUT /api/relay/rest/queues/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/relay/rest/queues/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.queues.update("queue-id", { name: "priority-support" });
console.log(`Updated: ${updated.id}`);
```
