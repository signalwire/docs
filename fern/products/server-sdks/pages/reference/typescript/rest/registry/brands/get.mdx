Retrieve a specific 10DLC brand by ID.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/brands/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/brands/{id}" />

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const brand = await client.registry.brands.get("brand-id");
console.log(brand.display_name, brand.entity_type);
```
