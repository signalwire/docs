List phone numbers assigned to a campaign.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/numbers" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/numbers" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const numbers = await client.registry.campaigns.listNumbers("campaign-id");
for (const n of numbers.data ?? []) {
    console.log(n.number, n.id);
}
```
