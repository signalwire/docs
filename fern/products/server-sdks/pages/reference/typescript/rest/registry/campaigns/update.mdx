Update a 10DLC campaign. Uses PUT to replace fields.

<EndpointSchemaSnippet endpoint="PUT /api/relay/rest/registry/beta/campaigns/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/relay/rest/registry/beta/campaigns/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.registry.campaigns.update("campaign-id", {
    description: "Updated campaign description",
});
console.log(updated.description);
```
