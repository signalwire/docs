Create a number assignment order to assign phone numbers to a campaign.

<EndpointSchemaSnippet endpoint="POST /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const order = await client.registry.campaigns.createOrder("campaign-id", {
    phone_numbers: ["+15125551234"],
});
console.log("Order ID:", order.id);
```
