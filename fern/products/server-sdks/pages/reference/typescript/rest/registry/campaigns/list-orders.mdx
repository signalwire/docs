List number assignment orders for a campaign.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const orders = await client.registry.campaigns.listOrders("campaign-id");
for (const order of orders.data ?? []) {
    console.log(order.id, order.status);
}
```
