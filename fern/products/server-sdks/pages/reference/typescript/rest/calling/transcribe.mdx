Start transcribing an active call in the background. The call is recorded and the
recording is transcribed; the response returns the recording's location (`url`).
Status updates are delivered to the optional `status_url`. Supply a `control_id` so you
can stop the transcription later with
[`transcribeStop`](/docs/server-sdks/reference/typescript/rest/calling/transcribe-stop).

<EndpointSchemaSnippet endpoint="POST /api/calling/calls" />

<EndpointResponseSnippet endpoint="POST /api/calling/calls" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.transcribe("call-id-xxx", {
  control_id: "transcribe-1",
  status_url: "https://example.com/transcribe-status",
});
const recordingUrl = result.url;
```
