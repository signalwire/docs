List recordings in the account.

<EndpointSchemaSnippet endpoint="GET /Accounts/{AccountSid}/Recordings" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /Accounts/{AccountSid}/Recordings" />

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const recordings = await client.compat.recordings.list({ PageSize: 20 });
```
