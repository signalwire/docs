Create a new call queue.

<EndpointSchemaSnippet endpoint="POST /Accounts/{AccountSid}/Queues" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /Accounts/{AccountSid}/Queues" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const queue = await client.compat.queues.create({
  FriendlyName: "Support Queue",
  MaxSize: 100
});
```
