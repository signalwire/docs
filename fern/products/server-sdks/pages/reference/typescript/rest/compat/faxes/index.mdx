Manage faxes with CRUD operations and media sub-resources. The media sub-resource
pattern is identical to [`messages`][messages].

Access via `client.compat.faxes` on a [`RestClient`][restclient] instance.

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const faxes = await client.compat.faxes.list();
```
