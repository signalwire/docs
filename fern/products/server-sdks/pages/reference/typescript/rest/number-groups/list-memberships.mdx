List phone numbers that belong to a group.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/number_groups/{NumberGroupId}/number_group_memberships" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/number_groups/{NumberGroupId}/number_group_memberships" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const memberships = await client.numberGroups.listMemberships("group-id");
for (const m of memberships.data ?? []) {
    console.log(m["phone_number_id"], m.id);
}
```
