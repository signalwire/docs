Remove a phone number from a group.

<Note>
Like `getMembership()`, this method takes a membership ID directly.
</Note>

<EndpointSchemaSnippet endpoint="DELETE /api/relay/rest/number_group_memberships/{id}" />

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.numberGroups.deleteMembership("membership-id");
```
