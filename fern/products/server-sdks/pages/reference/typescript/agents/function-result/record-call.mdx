Start recording the call in the background. The conversation continues while
recording is active.

<Note>
For continuous call recording, omit `initial_timeout` and `end_silence_timeout`.
Those parameters are for voicemail-style recordings that automatically stop on
silence. Use [`stopRecordCall()`][stop-record-call] to end continuous recordings.
</Note>

## **Parameters**

<ParamField path="opts" type="object" toc={true}>
  Optional recording configuration.
</ParamField>

<Indent>
<ParamField path="opts.controlId" type="string" toc={true}>
  Identifier for this recording. Pass the same ID to `stopRecordCall()` to
  stop this specific recording.
</ParamField>

<ParamField path="opts.stereo" type="boolean" default="false" toc={true}>
  Record in stereo (`true`) or mono (`false`).
</ParamField>

<ParamField path="opts.format" type="string" default="wav" toc={true}>
  Recording file format.

  - `"wav"` -- uncompressed WAV audio
  - `"mp3"` -- compressed MP3 audio
</ParamField>

<ParamField path="opts.direction" type="string" default="both" toc={true}>
  Audio direction to record.

  - `"speak"` -- what the agent says
  - `"listen"` -- what the caller says
  - `"both"` -- both sides of the conversation
</ParamField>

<ParamField path="opts.terminators" type="string" toc={true}>
  DTMF digits that stop recording when pressed (e.g., `"#"`).
</ParamField>

<ParamField path="opts.beep" type="boolean" default="false" toc={true}>
  Play a beep tone before recording starts.
</ParamField>

<ParamField path="opts.inputSensitivity" type="number" default="44.0" toc={true}>
  Input sensitivity level for the recording.
</ParamField>

<ParamField path="opts.initialTimeout" type="number" toc={true}>
  Seconds to wait for speech to begin before auto-stopping. Used for
  voicemail-style recordings.
</ParamField>

<ParamField path="opts.endSilenceTimeout" type="number" toc={true}>
  Seconds of silence after speech to wait before auto-stopping. Used for
  voicemail-style recordings.
</ParamField>

<ParamField path="opts.maxLength" type="number" toc={true}>
  Maximum recording duration in seconds.
</ParamField>

<ParamField path="opts.statusUrl" type="string" toc={true}>
  URL to receive recording status webhook events.
</ParamField>
</Indent>

## **Returns**

[`FunctionResult`][functionresult] -- `this`, for chaining.

## **Examples**

### Continuous Recording

```typescript {4}
import { FunctionResult } from '@signalwire/sdk';

const result = new FunctionResult('Recording started.')
  .recordCall({ format: 'mp3', direction: 'both' });
```

### Voicemail Recording

```typescript {4}
import { FunctionResult } from '@signalwire/sdk';

const result = new FunctionResult('Please leave your message after the beep.')
  .recordCall({
    controlId: 'voicemail',
    beep: true,
    maxLength: 120,
    endSilenceTimeout: 3,
  });
```
