Stop an active background call recording.

## **Parameters**

<ParamField path="controlId" type="string | undefined" default="undefined" toc={true}>
  Identifier of the recording to stop. If not provided, the most recent
  recording is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] -- `this`, for chaining.

## **Example**

```typescript {4}
import { FunctionResult } from '@signalwire/sdk';

const result = new FunctionResult('Recording stopped.')
  .stopRecordCall();
```
