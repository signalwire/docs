Search the web using the Google Custom Search JSON API. The skill fetches
more results than requested, scrapes each page, scores the extracted
content, and returns only the highest-quality matches to the AI.

**Class:** `WebSearchSkill`

**Tools:** `web_search` (configurable via `tool_name`)

**Env vars:** `GOOGLE_SEARCH_API_KEY`, `GOOGLE_SEARCH_ENGINE_ID` (legacy
`GOOGLE_SEARCH_CX` is still accepted)

**Multi-instance:** yes — instance key combines `search_engine_id` and
`tool_name`.

<ParamField path="api_key" type="string" required={true} toc={true}>
  Google Custom Search API key. Falls back to the `GOOGLE_SEARCH_API_KEY`
  environment variable.
</ParamField>

<ParamField path="search_engine_id" type="string" required={true} toc={true}>
  Google Custom Search Engine ID. Falls back to the
  `GOOGLE_SEARCH_ENGINE_ID` environment variable (or the legacy
  `GOOGLE_SEARCH_CX`).
</ParamField>

<ParamField path="tool_name" type="string" toc={true}>
  Custom tool name for this Web Search instance (useful when registering
  multiple instances at once).
</ParamField>

<ParamField path="num_results" type="integer" default="3" toc={true}>
  Number of high-quality results to return (range `1-10`).
</ParamField>

<ParamField path="delay" type="number" default="0.5" toc={true}>
  Delay between scraping pages in seconds (minimum `0`).
</ParamField>

<ParamField path="max_content_length" type="integer" default="32768" toc={true}>
  Maximum total response size in characters (minimum `1000`).
</ParamField>

<ParamField path="oversample_factor" type="number" default="2.5" toc={true}>
  How many extra results to fetch for quality filtering — e.g. `2.5` fetches
  2.5× the requested `num_results` before scoring. Range `1.0-3.5`.
</ParamField>

<ParamField path="min_quality_score" type="number" default="0.3" toc={true}>
  Minimum quality score (0–1) required to include a result.
</ParamField>

<ParamField path="no_results_message" type="string" toc={true}>
  Message returned when no quality results are found. Use `{query}` as a
  placeholder for the original search term.
</ParamField>

<ParamField path="safe_search" type="string" default="medium" toc={true}>
  Safe-search level. One of `"off"`, `"medium"`, `"high"`.
</ParamField>

<ParamField path="per_page_timeout" type="number" default="2.0" toc={true}>
  Maximum seconds to wait on a single page scrape. Minimum `0.1`.
</ParamField>

<ParamField path="overall_deadline" type="number" default="10.0" toc={true}>
  Wall-clock budget in seconds for the entire tool call. In-flight scrapes are
  abandoned past this point so the response beats the kernel webhook timeout.
  Minimum `1.0`.
</ParamField>

<ParamField path="parallel_scrape" type="boolean" default="true" toc={true}>
  Scrape all candidate pages concurrently (raced against the deadline) instead
  of sequentially.
</ParamField>

<ParamField path="snippets_only" type="boolean" default="false" toc={true}>
  Skip page scraping entirely and return Google CSE snippets only. The fastest
  mode (sub-second).
</ParamField>

<ParamField path="response_prefix" type="string" toc={true}>
  Optional text prepended (separated by a blank line) to every non-empty search
  result.
</ParamField>

<ParamField path="response_postfix" type="string" toc={true}>
  Optional text appended (separated by a blank line) to every non-empty search
  result.
</ParamField>

## Example

```typescript {6-10}
import { AgentBase, WebSearchSkill } from '@signalwire/sdk';

const agent = new AgentBase({ name: 'assistant', route: '/assistant' });
agent.setPromptText('You are a helpful assistant.');

await agent.addSkill(new WebSearchSkill({
  num_results: 3,
  safe_search: 'high',
  min_quality_score: 0.4,
}));

agent.run();
```
