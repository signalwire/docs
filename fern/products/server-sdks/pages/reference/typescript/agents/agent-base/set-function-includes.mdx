Replace the agent's SWAIG function-includes list in one call. Use
[`addFunctionInclude`][ref-add-include] to append a single include instead.

## **Parameters**

<ParamField path="includes" type="FunctionInclude[]" required={true} toc={true}>
  Ordered list of function-include entries.
</ParamField>

## **Returns**

`this` — the agent for method chaining.
