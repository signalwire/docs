[serve]: /docs/server-sdks/reference/typescript/agents/agent-base/serve

Alias for [`serve()`][serve]. Starts the Hono HTTP server and begins listening
for requests.

## **Parameters**

<ParamField path="opts" type="object" toc={true}>
  Optional host and port overrides.
</ParamField>

<Indent>
<ParamField path="opts.host" type="string" toc={true}>
  Host to bind the server to. Falls back to the constructor's `host` option.
</ParamField>

<ParamField path="opts.port" type="number" toc={true}>
  Port to listen on. Falls back to the constructor's `port` option.
</ParamField>
</Indent>

## **Returns**

`Promise<void>` -- Resolves once the server is running.

## **Example**

```typescript {5}
import { AgentBase } from '@signalwire/sdk';

const agent = new AgentBase({ name: 'my-agent', route: '/agent' });
agent.setPromptText('You are a helpful assistant.');
await agent.run();
```
