Replace the agent's prompt with the supplied POM (Prompt Object Model) array.
Requires the agent to be constructed with `usePom: true` — throws otherwise.

## **Parameters**

<ParamField path="pom" type={"Record<string, unknown>[]"} required={true} toc={true}>
  Ordered array of POM section objects.
</ParamField>

## **Returns**

`this` — the agent for method chaining.
