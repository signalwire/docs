[ref-ctxbuilder]: /docs/server-sdks/reference/typescript/agents/context-builder

Remove every context defined on this agent's internal
[`ContextBuilder`][ref-ctxbuilder], returning the builder to its initial
state. Use this inside a dynamic-config callback to rebuild contexts per
request.

## **Returns**

`this` — the agent for method chaining.
