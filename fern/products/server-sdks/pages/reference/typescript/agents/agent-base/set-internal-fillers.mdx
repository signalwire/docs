Set every internal-filler entry on the agent in one call. Unknown filler
names (not in the SWML schema's supported set) are rejected with a warning.

## **Parameters**

<ParamField path="internalFillers" type={"Record<string, Record<string, string[]>>"} required={true} toc={true}>
  Outer key: filler name (e.g. `"pre_speech"`). Inner key: language code
  (e.g. `"en-US"`). Value: array of filler phrases.
</ParamField>

## **Returns**

`this` — the agent for method chaining.

## **Example**

```typescript {1-9}
agent.setInternalFillers({
  pre_speech: {
    'en-US': ['one moment', 'let me check'],
  },
  summary: {
    'en-US': ['so to summarize', 'in short'],
  },
});
```
