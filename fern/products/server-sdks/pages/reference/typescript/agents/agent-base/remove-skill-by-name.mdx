max-toc-depth: 3
---

Remove every skill instance whose `skillName` matches the given name.
`cleanup()` is invoked on each instance before removal.

## **Parameters**

<ParamField path="skillName" type="string" required={true} toc={true}>
  Skill name to match.
</ParamField>

## **Returns**

`Promise<boolean>` — `true` if at least one instance was removed.
