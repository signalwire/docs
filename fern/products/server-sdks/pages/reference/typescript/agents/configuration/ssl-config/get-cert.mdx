Read the PEM certificate file from disk.

## **Parameters**

None.

## **Returns**

`string | null` -- The certificate file contents, or `null` if the file path
is not set or the file does not exist.

## **Example**

```typescript {4}
import { SslConfig } from '@signalwire/sdk';

const ssl = new SslConfig({ certPath: './cert.pem', keyPath: './key.pem' });
const cert = ssl.getCert();
console.log(cert ? 'Certificate loaded' : 'Certificate not found');
```
