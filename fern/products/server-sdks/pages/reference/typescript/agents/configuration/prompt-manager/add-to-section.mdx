Append body text or bullets to an existing POM section, creating it if absent.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  The section heading to append to.
</ParamField>

<ParamField path="opts" type="object" toc={true}>
  Content to add.
</ParamField>

<Indent>
<ParamField path="opts.body" type="string" toc={true}>
  Body text to append.
</ParamField>

<ParamField path="opts.bullet" type="string" toc={true}>
  A single bullet point to add.
</ParamField>

<ParamField path="opts.bullets" type="string[]" toc={true}>
  Multiple bullet points to add.
</ParamField>
</Indent>

## **Returns**

`void`

## **Example**

```typescript {5-6}
import { PromptManager } from '@signalwire/sdk';

const pm = new PromptManager();
pm.addSection('Rules', { bullets: ['Be concise'] });
pm.addToSection('Rules', { bullet: 'Be polite' });
pm.addToSection('Rules', { bullets: ['Never guess', 'Stay on topic'] });
console.log(pm.getPrompt());
```
