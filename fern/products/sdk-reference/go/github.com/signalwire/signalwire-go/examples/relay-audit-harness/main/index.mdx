# `main`

Example: relay_audit_harness

Audit-only harness — runs a RelayClient against a local fixture that speaks JSON-RPC 2.0 + the SignalWire RELAY handshake. Used by porting-sdk/scripts/audit_relay_handshake.py to prove the Go RELAY client opens a real WebSocket, runs the connect handshake, subscribes to a context, and dispatches an inbound event end-to-end.

Contract (from porting-sdk SUBAGENT_PLAYBOOK.md):

Not for production use. The harness's whole purpose is to give the audit a small, fast binary to drive its fixture against.
