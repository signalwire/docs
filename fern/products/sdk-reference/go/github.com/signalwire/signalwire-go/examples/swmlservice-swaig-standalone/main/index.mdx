# `main`

Example: swmlservice_swaig_standalone

Proves that swml.Service — by itself, with NO agent.AgentBase — can host SWAIG functions and serve them on its own /swaig endpoint.

This is the path you take when you want a SWAIG-callable HTTP service that isn't an `<ai>` agent: the SWAIG verb is a generic LLM-tool surface and swml.Service is the host. agent.AgentBase is just a swml.Service composition that *also* layers in prompts, AI config, dynamic config, and token validation.
