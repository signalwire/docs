# `server`

Package server provides AgentServer for hosting multiple AI agents on a single HTTP server with route-based dispatch.

## Signature

```go
package server
```

  <Card title="WithLogLevel" href="/reference/go/github.com/signalwire/signalwire-go/pkg/server/server/with-log-level">

Mirrors Python AgentServer(log\_level=...) behavior: the level is applied globally via logging.SetGlobalLevel so all loggers in the process are affected. The default level is "info".
