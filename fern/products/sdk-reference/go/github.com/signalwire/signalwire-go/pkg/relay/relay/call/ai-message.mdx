# `AIMessage`

AIMessage sends a text message within an active AI session. All parameters are optional, matching Python's ai_message(*, message_text=None, role=None, reset=None, global_data=None). Pass "" for controlID/text/role and nil for reset/globalData to omit them from the wire payload (Python omits the key entirely when the argument is None).
