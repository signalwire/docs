# `ValidateToolToken`

ValidateToolToken verifies that a SWAIG tool security token is authentic, unexpired, and matches the given function name and call ID. Returns false when the function is not registered, the SessionManager rejects the token, or the validation panics for any reason.

Python parity: state_mixin.StateMixin.validate_tool_token. Python rejects unknown function names up-front and swallows exceptions, returning false.
