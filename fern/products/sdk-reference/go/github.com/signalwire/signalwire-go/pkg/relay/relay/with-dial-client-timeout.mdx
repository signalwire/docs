# `WithDialClientTimeout`

WithDialClientTimeout bounds how long Dial() will wait for the calling.call.dial event before raising a timeout error. Mirrors Python's dial(dial\_timeout=\<seconds>). Default is 120s when omitted.

The duration is consumed by the Go Dial() loop; it never goes on the wire — that's why it's stored under an underscore-prefixed key removed before transmit.
