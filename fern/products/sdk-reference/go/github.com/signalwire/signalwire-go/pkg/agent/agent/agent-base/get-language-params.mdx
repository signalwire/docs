# `GetLanguageParams`

GetLanguageParams reads the per-language params dict for a previously-added language.

Python equivalent: ai\_config\_mixin.AIConfigMixin.get\_language\_params Python signature: get\_language\_params(code) -> Optional\[Dict\[str, Any]]

Returns the params map if set, or nil otherwise (including when the code is unknown). Callers can distinguish "no params set" from "empty params set" by the fact that empty maps are never stored (SetLanguageParams with an empty dict removes the key).
