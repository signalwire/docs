# `ListSkillsWithParams`

ListSkillsWithParams returns the complete parameter schema for all registered skills. It instantiates each skill with nil params to obtain its GetParameterSchema output. This mirrors Python's skill_registry.get_all_skills_schema(). The returned map has skill names as keys and their parameter schemas as values.
