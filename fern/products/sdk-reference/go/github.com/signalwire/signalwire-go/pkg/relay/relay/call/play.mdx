# `Play`

Play starts playing media on the call and returns a PlayAction.

Use WithPlayControlID to supply an explicit control_id (mirrors Python's play(control_id=...)). When omitted the SDK auto-generates a UUID — same as Python's `cid = control_id or str(uuid.uuid4())` at relay/call.py:506.
