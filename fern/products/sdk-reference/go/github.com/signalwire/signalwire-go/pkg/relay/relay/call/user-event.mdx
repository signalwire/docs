# `UserEvent`

UserEvent sends a user-defined event on the call. eventName is the custom event identifier; pass "" to omit it. Optional `extra` maps are merged into the top-level wire params (mirroring Python's **kwargs in user_event(*, event: Optional[str] = None, **kwargs)).
