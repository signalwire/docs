# `util`

Package util provides cross-cutting helpers used across the Go SDK.

validate_url is the SSRF-prevention guard applied to user-supplied URLs before they are fetched. It must mirror the Python reference at signalwire.utils.url_validator.validate_url:

The blocked-network list is identical across all SDK ports.
