# `WithEnvDefaults`

WithEnvDefaults is now a no-op pass-through retained for backwards compatibility — env defaults are loaded automatically at the end of NewRelayClient (mirroring Python RelayClient.**init**). New code can rely on the auto-load behavior and omit this option entirely.
