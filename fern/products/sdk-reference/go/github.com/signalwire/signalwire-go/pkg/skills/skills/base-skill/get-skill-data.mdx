# `GetSkillData`

GetSkillData reads this skill instance's namespaced state from rawData. rawData is the raw_data map passed to SWAIG function handlers, expected to contain a "global_data" key. Returns an empty map when not found. Mirrors Python's get_skill_data(raw_data).
