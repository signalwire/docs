# `GetSkillData`

GetSkillData reads this skill instance's namespaced state from rawData. rawData is the raw\_data map passed to SWAIG function handlers, expected to contain a "global\_data" key. Returns an empty map when not found. Mirrors Python's get\_skill\_data(raw\_data).
