# `SetGatherInfo`

SetGatherInfo enables info gathering for this step and returns the Step for fluent chaining. This matches the Python SDK's set_gather_info, which returns self so that step-level setters (SetFunctions, SetValidSteps, etc.) can be chained after configuring gather info.

To add questions to the gather info, use AddGatherQuestion on the same *Step receiver.
