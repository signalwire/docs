# `IsServerlessMode`

IsServerlessMode reports whether the process is running in any serverless environment (i.e. NOT in plain long-running server mode).

Mirrors Python's is\_serverless\_mode():

```
def is_serverless_mode() -> bool:
    return get_execution_mode() != 'server'
```
