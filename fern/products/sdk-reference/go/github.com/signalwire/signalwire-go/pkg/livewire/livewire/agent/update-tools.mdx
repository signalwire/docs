# `UpdateTools`

UpdateTools replaces the agent's tool list. Mirrors Python Agent.update_tools (livewire/__init__.py:394) which accepts List[Any] and stores self._tools = list(tools). In Go, the parameter is []any to keep the unexported toolDef out of the public signature (the original typed []toolDef made the method uncallable from external packages). Elements that aren't recognized tools are silently skipped, matching Python's permissive storage semantics. Returns the Agent for method chaining.
