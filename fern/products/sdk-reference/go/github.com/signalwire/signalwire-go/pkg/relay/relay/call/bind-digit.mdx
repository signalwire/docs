# `BindDigit`

BindDigit binds a DTMF digit sequence to trigger a RELAY method. bindParams, realm, and maxTriggers are optional (nil/zero-value omits them), matching Python's bind_digit(digits, bind_method, *, bind_params, realm, max_triggers).
