# `VerifyBearerToken`

VerifyBearerToken performs a timing-safe comparison of the supplied token against the service's configured bearer token. Returns false when no bearer token is configured (i.e. WithBearerToken was not called).

VerifyBearerToken performs a timing-safe comparison of the supplied token
against the service's configured bearer token.
Returns false when no bearer token is configured (i.e. WithBearerToken was
not called).

Callers typically extract the token from the "Authorization: Bearer &lt;token>"
header before calling this method.
Mirrors Python AuthHandler.verify_bearer_token(credentials) -> bool.
