# `ManualSetProxyUrl`

ManualSetProxyUrl overrides the proxy URL base used for URL generation. Call this at runtime to set or update the proxy URL (e.g. an ngrok URL). Mirrors Python's manual_set_proxy_url(proxy_url: str) on SWMLService.
