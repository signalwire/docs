# `NewSchemaUtils`

NewSchemaUtils constructs a SchemaUtils. Mirrors Python's “SchemaUtils(schema\_path, schema\_validation=True)“.

Pass schemaPath="" to use the embedded schema.json bundled with the SDK. schemaValidation=false disables validation; the env var SWML\_SKIP\_SCHEMA\_VALIDATION=1/true/yes also disables it.
