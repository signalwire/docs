# `SetOnSwmlRequestHook`

SetOnSwmlRequestHook registers a function that customizes the SWML response on a per-request basis. The hook receives the parsed body, the callback path (for routing-callback dispatch), and the raw *http.Request for header / query inspection. Returning a non-nil map applies modifications to the rendered SWML; returning nil falls through to the default rendering.

Python parity: this is the Go-idiomatic way of "overriding" on_swml_request — Go has no method inheritance.
