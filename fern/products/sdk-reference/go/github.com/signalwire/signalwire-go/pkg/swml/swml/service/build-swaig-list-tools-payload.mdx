# `BuildSwaigListToolsPayload`

BuildSwaigListToolsPayload returns the JSON payload that --example mode expects between the sentinel markers: \{"tools": \[\<definitions>]\}. Each element echoes whatever the in-memory ToolDefinition stores (name, description, parameters); we deliberately do not normalize so the CLI can be permissive about field names.

Exposed for tests so the env-var-driven exit branch in Serve() can be asserted without forking a process.
