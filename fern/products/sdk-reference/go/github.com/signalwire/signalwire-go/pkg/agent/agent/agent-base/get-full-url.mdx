# `GetFullURL`

GetFullURL returns the full URL for this agent's endpoint, optionally embedding basic-auth credentials.

Python equivalent: AgentBase.get\_full\_url(include\_auth=False) (agent\_base.py:325)

The Python implementation handles serverless URL construction (CGI / Lambda / Cloud Functions / Azure) inline. In the Go SDK, serverless URL construction lives in pkg/lambda; this method delegates server-mode URL building to the embedded swml.Service and matches Python's server-mode behavior.
