# `RecordCallOptions`

RecordCallOptions holds optional parameters for RecordCall beyond the required fields.

## Signature

```go
type RecordCallOptions struct
```

## Properties

<ParamField path="Beep" type="Beep bool" required>
  Beep plays a beep before recording starts when true.
</ParamField>

<ParamField path="EndSilenceTimeout" type="EndSilenceTimeout float64" required>
  EndSilenceTimeout is seconds of silence before ending (voicemail-style). Negative value is omitted.
</ParamField>

<ParamField path="EndSilenceTimeoutSet" type="EndSilenceTimeoutSet bool" required>
  EndSilenceTimeoutSet must be true for EndSilenceTimeout of 0.0 to be included.
</ParamField>

<ParamField path="InitialTimeout" type="InitialTimeout float64" required>
  InitialTimeout is the time in seconds to wait for speech to start (voicemail-style). Negative value is omitted.
</ParamField>

<ParamField path="InitialTimeoutSet" type="InitialTimeoutSet bool" required>
  InitialTimeoutSet must be true for InitialTimeout of 0.0 to be included.
</ParamField>

<ParamField path="InputSensitivity" type="InputSensitivity float64" required>
  InputSensitivity sets the input sensitivity for recording (default 44.0 in Python). Zero value is omitted from the SWML payload.
</ParamField>

<ParamField path="MaxLength" type="MaxLength float64" required>
  MaxLength is the maximum recording length in seconds. Negative value is omitted.
</ParamField>

<ParamField path="MaxLengthSet" type="MaxLengthSet bool" required>
  MaxLengthSet must be true for MaxLength of 0.0 to be included.
</ParamField>

<ParamField path="StatusURL" type="StatusURL string" required>
  StatusURL is the URL to send recording status events to.
</ParamField>

<ParamField path="Terminators" type="Terminators string" required>
  Terminators specifies digits that stop recording when pressed.
</ParamField>
