# `AIHold`

AIHold places the AI-controlled call on hold. controlID, timeout and prompt are all optional — pass "" to omit any of them, matching Python's ai_hold(*, timeout: Optional[str] = None, prompt: Optional[str] = None) which has no control_id parameter and only writes keys conditionally.
