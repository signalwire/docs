# `GetAuthInfo`

GetAuthInfo returns a map describing every configured authentication method. The map mirrors the Python AuthHandler.get_auth_info() return value:

Only methods that are actively configured appear in the map. Basic auth is always present because the service always has a username/password (either explicit or auto-generated). Mirrors Python AuthHandler.get_auth_info() -> Dict[str, Any].
