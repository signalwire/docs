# `GetPromptSections`

GetPromptSections returns nil (no prompt sections by default). When skip_prompt is set to true in Params, returns nil even for concrete overrides — concrete overrides that inject prompt sections MUST call ShouldSkipPrompt() and return nil (or an empty slice) when it is true.
