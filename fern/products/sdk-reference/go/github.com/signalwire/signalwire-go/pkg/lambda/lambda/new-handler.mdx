# `NewHandler`

NewHandler returns a Handler that dispatches Lambda events to the given http.Handler. The handler is usually agent.AsRouter() but any http.Handler works.

Panics if h is nil to fail loudly at cold-start rather than silently returning 500 on every invocation.
