# `AutoMaterializedWebhookResource`

AutoMaterializedWebhookResource is a Fabric webhook resource that is normally auto-created by the phone_numbers.Set*Webhook helpers. Exposed for backwards compatibility: list/get/update/delete work as usual, but Create now emits a deprecation warning because creating a webhook resource directly produces an orphan that isn't bound to any phone number.
