# `RoutingCallbackPaths`

RoutingCallbackPaths returns the paths that have routing callbacks registered. Callers use this to register corresponding HTTP endpoints (mirrors Python web_mixin.py line 428 which iterates self._routing_callbacks to register callback endpoints on the router). Paths are returned in sorted order for deterministic HTTP registration.
