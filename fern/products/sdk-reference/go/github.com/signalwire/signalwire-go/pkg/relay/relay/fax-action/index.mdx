# `FaxAction`

FaxAction represents a long-running fax send/receive operation. methodPrefix distinguishes "send\_fax" from "receive\_fax" and is used to build the operation-specific stop command (e.g. "calling.send\_fax.stop").

## Methods

### Stop

Stop sends "calling.\{methodPrefix\}.stop" (e.g. "calling.send\_fax.stop" or "calling.receive\_fax.stop") to halt the active fax operation.
