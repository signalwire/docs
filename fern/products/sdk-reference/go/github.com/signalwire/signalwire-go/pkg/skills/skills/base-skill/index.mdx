# `BaseSkill`

BaseSkill provides default implementations for the SkillBase interface. Concrete skills should embed this struct and override methods as needed.

## Signature

```go
type BaseSkill struct
```

## Properties

<ParamField path="Logger" type="Logger *logging.Logger" required>
  Logger is a named logger for this skill instance. It is initialized automatically when Name() is first called via NewBaseSkill, or can be set explicitly. Mirrors Python SkillBase.logger.
