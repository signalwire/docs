# `WithAgentID`

WithAgentID sets a fixed agent ID. If not provided, a UUID is generated automatically in NewAgentBase.

Python equivalent: agent\_id parameter in AgentBase.**init** Python behavior: self.agent\_id = agent\_id or str(uuid.uuid4())
