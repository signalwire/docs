# `ParseEvent`

ParseEvent parses a raw signalwire event payload dict into a typed event object. It reads "event_type" from the top-level payload and "params" as the inner parameter map, then dispatches to the appropriate typed constructor. If the event_type is not recognised, a plain *RelayEvent is returned. Callers can type-assert or type-switch on the result to access the concrete event fields.
