# `ClaudeSkillsSkill`

ClaudeSkillsSkill loads Claude-style SKILL.md files as SignalWire agent tools.

Each directory under skills_path that contains a SKILL.md becomes a SWAIG tool. The SKILL.md frontmatter provides the tool name and description; the body becomes the tool's response content when invoked.
