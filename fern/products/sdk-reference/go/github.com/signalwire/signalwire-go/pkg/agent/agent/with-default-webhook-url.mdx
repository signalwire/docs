# `WithDefaultWebhookURL`

WithDefaultWebhookURL sets the default webhook URL for all SWAIG functions. When set, this URL is used as the fallback for all tools that do not specify their own WebhookURL.

Python equivalent: default\_webhook\_url parameter in AgentBase.**init**
