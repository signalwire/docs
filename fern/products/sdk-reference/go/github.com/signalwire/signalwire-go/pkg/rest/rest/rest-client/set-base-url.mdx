# `SetBaseURL`

SetBaseURL overrides the base URL used by the underlying HttpClient. Useful for pointing the client at a non-default endpoint such as the audit_rest_transport.py harness fixture, a recorded-cassette mock server, or a regional endpoint without re-running the constructor.
