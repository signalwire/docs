# `NewClientOnly`

NewClientOnly returns a fresh *relay.Client connected to the running mock, with no contexts subscribed. Useful for tests that need to drive multiple clients in one test (e.g. reconnect-with-protocol).

The harness is shared with any previously created clients in the same test — t.Cleanup is registered to disconnect this client when the test ends.
