# `PayOptions`

PayOptions holds all optional parameters for the Pay method.

## Signature

```go
type PayOptions struct
```

## Properties

<ParamField path="AIResponse" type="AIResponse string" required>
  AIResponse is the message set via the "set" verb that always precedes the pay verb; empty string uses the Python default status message. The set verb is always emitted (Python has no suppression path), matching pay()'s SWML.
</ParamField>

<ParamField path="ChargeAmount" type="ChargeAmount string" required>
  ChargeAmount is the amount to charge as a decimal string (e.g. "9.99").
</ParamField>

<ParamField path="Currency" type="Currency string" required>
  Currency is the currency code. Defaults to "usd".
</ParamField>

<ParamField path="Description" type="Description string" required>
  Description is a custom payment description.
</ParamField>

<ParamField path="InputMethod" type="InputMethod string" required>
  InputMethod is the method to collect payment details ("dtmf" or "voice"). Defaults to "dtmf".
</ParamField>

<ParamField path="Language" type="Language string" required>
  Language is the language for prompts. Defaults to "en-US".
</ParamField>

<ParamField path="MaxAttempts" type="MaxAttempts int" required>
  MaxAttempts is the number of retry attempts. Defaults to 1.
</ParamField>

<ParamField path="MinPostalCodeLength" type="MinPostalCodeLength int" required>
  MinPostalCodeLength sets the minimum number of postal code digits. Defaults to 0.
</ParamField>

<ParamField path="Parameters" type="Parameters []map[string]string" required>
  Parameters is an array of name/value pairs for the payment connector.
</ParamField>

<ParamField path="PaymentMethod" type="PaymentMethod string" required>
  PaymentMethod is the payment method type. Defaults to "credit-card".
</ParamField>

<ParamField path="PostalCode" type="PostalCode any" required>
  PostalCode controls whether to prompt for postal code, or supplies the actual code. String value is used as-is; bool true/false becomes "true"/"false".
</ParamField>

<ParamField path="Prompts" type="Prompts []map[string]any" required>
  Prompts is an array of custom prompt configurations.
</ParamField>

<ParamField path="SecurityCode" type="SecurityCode bool" required>
  SecurityCode controls whether to prompt for security code. Defaults to true. Use SecurityCodeSet to override; zero value (false) is treated as "not set".
</ParamField>

<ParamField path="SecurityCodeSet" type="SecurityCodeSet bool" required>
  SecurityCodeSet must be true to explicitly set SecurityCode=false.
</ParamField>

<ParamField path="StatusURL" type="StatusURL string" required>
  StatusURL is the URL for payment status change notifications.
</ParamField>

<ParamField path="Timeout" type="Timeout int" required>
  Timeout is the seconds to wait for the next digit. Defaults to 5.
</ParamField>

<ParamField path="TokenType" type="TokenType string" required>
  TokenType is the payment token type: "one-time" or "reusable". Defaults to "reusable".
</ParamField>

<ParamField path="ValidCardTypes" type="ValidCardTypes string" required>
  ValidCardTypes is a space-separated list of card types. Defaults to "visa mastercard amex".
</ParamField>

<ParamField path="Voice" type="Voice string" required>
  Voice is the TTS voice to use. Defaults to "woman".
</ParamField>
