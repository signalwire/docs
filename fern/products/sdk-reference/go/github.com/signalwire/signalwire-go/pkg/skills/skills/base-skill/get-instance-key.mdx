# `GetInstanceKey`

GetInstanceKey returns a unique key for tracking this skill instance. When SupportsMultipleInstances() returns true, the key is composed of the skill name and the "tool_name" parameter (defaulting to the skill name), matching Python's get_instance_key() behavior for multi-instance skills. When SupportsMultipleInstances() returns false, returns the skill name.
