# `HandlerFactory`

HandlerFactory is a zero-arg function that constructs the http.Handler under test. The factory is called AFTER the simulator has activated the platform env vars, so any env-var-driven state the agent captures at construction (notably SWML_PROXY_URL_BASE, which pkg/swml.Service reads in its constructor) reflects the simulated environment rather than the outer shell.

first, then the agent module is imported/loaded, then invocations
