# `RelayError`

RelayError is a typed error returned by the RELAY protocol client. It carries the numeric code and message from the server so callers can programmatically inspect failures via errors.As.

The error format matches Python's RelayError.**str**: "RELAY error \{code\}: \{message\}".

A RelayError may also carry a wrapped sentinel (one of the Err\* values above) so the same value satisfies BOTH errors.As(\&RelayError) and errors.Is(err, ErrDialTimeout): the typed error preserves the server code/message, and the sentinel gives callers a stable, string-free branch.
