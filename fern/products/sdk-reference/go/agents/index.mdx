  </Card>

  <Card title="pom" href="/docs/sdk-reference/reference/go/agents/pom">

The rendered output matches the Python reference at signalwire/signalwire/pom/pom.py byte-for-byte for the canonical scenarios covered by the cross-port parity tests in tests/unit/pom/test\_pom\_render\_parity.py.
  </Card>

  <Card title="prefabs" href="/docs/sdk-reference/reference/go/agents/prefabs">
    Package prefabs provides pre-built agent patterns that extend AgentBase with common conversational workflows such as information gathering, surveys, reception/routing, FAQ answering, and virtual concierge services.
  </Card>

  <Card title="server" href="/docs/sdk-reference/reference/go/agents/server">
    Package server provides AgentServer for hosting multiple AI agents on a single HTTP server with route-based dispatch.
  </Card>

  <Card title="skills" href="/docs/sdk-reference/reference/go/agents/skills">

  </Card>
