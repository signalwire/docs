# `SimulateLambdaOptions`

SimulateLambdaOptions tunes an in-process Lambda simulation.

## Signature

```go
type SimulateLambdaOptions struct
```

## Properties

<ParamField path="FunctionName" type="FunctionName string" required>
  FunctionName overrides AWS\_LAMBDA\_FUNCTION\_NAME. Empty means use the preset default ("test-agent-function").
</ParamField>

<ParamField path="FunctionURLOverride" type="FunctionURLOverride string" required>
  FunctionURLOverride, if non-empty, is assigned to AWS\_LAMBDA\_FUNCTION\_URL during the simulation. The default (empty) lets GetFullURL fall back to constructing the URL from AWS\_LAMBDA\_FUNCTION\_NAME + AWS\_REGION, which is the more interesting code path to exercise in tests.
</ParamField>

<ParamField path="Logger" type="Logger func(format string, args ...any)" required>
  Logger receives warnings about env state — notably if SWML\_PROXY\_URL\_BASE is still set after the clear attempt. A nil Logger sends warnings to os.Stderr.
</ParamField>

<ParamField path="Region" type="Region string" required>
  Region overrides AWS\_REGION. Empty means use the preset default ("us-east-1").
</ParamField>
