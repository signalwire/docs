# `lambda`

Handler wraps an http.Handler so it can service AWS Lambda invocations. Construct one with NewHandler.

## Signature

```go
type Handler struct
```

## Examples

<CodeBlocks>
  <CodeBlock title="Example 1">
    ```go
    package main

    import (
        "github.com/aws/aws-lambda-go/lambda"
        swlambda "github.com/signalwire/signalwire-go/pkg/lambda"
        "github.com/signalwire/signalwire-go/pkg/agent"
    )

    var a = agent.NewAgentBase(agent.WithName("MyAgent"), agent.WithRoute("/my-agent"))
    var handler = swlambda.NewHandler(a.AsRouter())

    func main() {
        lambda.Start(handler.HandleFunctionURL)
    }
    ```

### HandleAPIGatewayV2

### NewHandler

NewHandler returns a Handler that dispatches Lambda events to the given http.Handler. The handler is usually agent.AsRouter() but any http.Handler works.

Panics if h is nil to fail loudly at cold-start rather than silently returning 500 on every invocation.

#### Signature

