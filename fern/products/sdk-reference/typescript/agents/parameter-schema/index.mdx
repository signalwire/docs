# `ParameterSchema`

Fluent builder producing a [ParameterSchemaObject](/docs/sdk-reference/reference/typescript/agents/parameter-schema#parameter-schema-object) byte-identical to
the equivalent hand-written `parameters` blob. Every method returns `this`
for chaining; call `build` to obtain the plain object to hand to
`defineTool`/`defineTypedTool`.

Insertion order is preserved: properties appear in the order their methods
were called (matching a hand-written object literal), and `required` lists
names in the order `required` received them.
