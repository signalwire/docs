# `Section`

A section in the Prompt Object Model.

Each section contains a title, optional body text, optional bullet points,
and can have any number of nested subsections.

## Signature

```typescript
class Section
```

## Properties

<ParamField path="body" type="string" required>

</ParamField>

<ParamField path="bullets" type="string[]" required>

<ParamField path="numbered" type="boolean | null" required>
  Whether this section should be numbered.
</ParamField>

<ParamField path="numberedBullets" type="boolean" required>
