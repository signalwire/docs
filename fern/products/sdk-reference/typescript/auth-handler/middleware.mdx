# `middleware`

Create a Hono-compatible middleware that rejects unauthorized requests with a 401 response.

## Signature

```typescript
middleware(
  optional: boolean = false
): (c: any, next: () => Promise<void>) => Promise<void | Response>
```

## Parameters

<ParamField path="optional" type="boolean" required default="false">
  When true, unauthenticated requests are allowed through instead of being rejected (default: false).
</ParamField>

## Returns

`(c: any, next: () => Promise<void>) => Promise<void | Response>` — A middleware function suitable for use with Hono's `app.use()`.
