# `verifyBearerToken`

Verify a Bearer token against the configured token.

Returns false immediately if Bearer token auth is not configured.
Uses constant-time comparison to prevent timing attacks.

## Signature

```typescript
verifyBearerToken(token: string): boolean
```

## Parameters

<ParamField path="token" type="string" required>
  The Bearer token string to verify (without the "Bearer " prefix).
</ParamField>

## Returns

`boolean` — True if the token matches the configured Bearer token.
