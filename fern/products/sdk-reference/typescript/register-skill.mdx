# `registerSkill`

Register a custom skill class with the global `SkillRegistry`.

Equivalent to Python's `register_skill(skill_class)`. Allows third-party
code to register skills directly, bypassing the built-in directory scan.

## Signature

```typescript
registerSkill(skillClass: typeof SkillBase): void
```

## Parameters

<ParamField path="skillClass" type="typeof SkillBase" required>
  Skill class to register (a subclass of `SkillBase`).
</ParamField>

## Returns

`void`
