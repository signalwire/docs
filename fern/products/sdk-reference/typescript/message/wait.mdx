# `wait`

Wait for the message to reach a terminal state (`delivered`, `failed`,
`undelivered`, etc.).

**Throws:**
- When the optional timeout elapses before a terminal state.

## Signature

```typescript
wait(timeout?: number): Promise<RelayEvent>
```

## Parameters

<ParamField path="timeout" type="number">
  Maximum time to wait in **seconds** (matches the Python
SDK convention — not milliseconds).
</ParamField>

## Returns

`Promise<RelayEvent>` — The final `messaging.state` [RelayEvent](/reference/typescript/relay-event).
