# `start`

Start the HTTP(S) service.

When `SWAIG_CLI_MODE=true` is set in the environment, the call is a
no-op so config can be inspected without binding a port.

## Signature

```typescript
start(
  host?: string,
  port?: number,
  sslCert?: string,
  sslKey?: string
): Promise<void>
```

## Parameters

<ParamField path="host" type="string">
  Bind address. Defaults to `'0.0.0.0'`.
</ParamField>
<ParamField path="port" type="number">
  Port override. Defaults to `this.port`.
</ParamField>
<ParamField path="sslCert" type="string">
  Path to SSL certificate file (overrides `SslConfig`).
</ParamField>
<ParamField path="sslKey" type="string">
  Path to SSL key file (overrides `SslConfig`).
</ParamField>

## Returns

`Promise<void>` — Resolves once the server has begun listening.
