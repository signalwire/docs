# `getInstanceKey`

Instance key for the SkillManager. Includes the configured
`search_engine_id` (or `"default"`) and `tool_name` (or `"web_search"`)
to match Python's `"{SKILL_NAME}_{search_engine_id}_{tool_name}"` scheme.
