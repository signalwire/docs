# `waitFor`

Wait for a specific event, optionally filtered by predicate.

Registers a one-shot listener that is removed after resolving (or after
the timeout elapses). Useful for awaiting a specific state transition.

**Throws:**
- When the optional timeout elapses before a match.

## Signature

```typescript
waitFor(
  eventType: string,
  predicate?: (event: RelayEvent) => boolean,
  timeout?: number
): Promise<RelayEvent>
```

## Parameters

<ParamField path="eventType" type="string" required>
  Event type to wait for.
</ParamField>
<ParamField path="predicate" type="(event: RelayEvent) => boolean">
  Optional filter — only events for which this returns
`true` resolve the wait.
</ParamField>
<ParamField path="timeout" type="number">
  Optional timeout in milliseconds. Omit to wait forever.
</ParamField>

## Returns

`Promise<RelayEvent>` — The first matching [RelayEvent](/reference/typescript/relay-event).
