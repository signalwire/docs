# `HttpClient`

Low-level HTTP client used by every REST namespace resource.

Handles Basic Auth, JSON encoding/decoding, and error normalisation
([RestError](/docs/sdk-reference/reference/typescript/rest/rest-error) on non-2xx). Normally you do not instantiate this
directly — construct a `RestClient` instead.
