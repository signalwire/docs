  PhoneCallHandler — enum of `call_handler` values accepted by [PhoneNumbersResource.update](/docs/sdk-reference/reference/typescript/rest/phone-numbers/phone-numbers-resource#update). Named `PhoneCallHandler` (not `CallHandler`) to avoid colliding with the RELAY client's inbound-call-handler callback type already exported from `src/relay/types.ts`. Setting a phone number's `call_handler` + the handler-specific companion field routes inbound calls and auto-materializes the matching Fabric resource on the server.
