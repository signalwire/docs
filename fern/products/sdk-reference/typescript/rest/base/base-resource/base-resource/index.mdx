# `BaseResource`

Abstract base class for all REST resources.

Every namespace resource (e.g. [PhoneNumbersResource](/reference/typescript/rest/namespaces/phone-numbers/phone-numbers-resource), [VideoRooms](/reference/typescript/rest/namespaces/video/video-rooms))
extends this class to get a typed reference to the shared [HttpClient](/reference/typescript/rest/http-client)
and a small path-building helper. Not exported for direct use — subclass it.

**Modifiers:** `abstract`
