# `constructor`

Create a new REST client.

## Parameters

<ParamField path="options" type="ClientOptions" required default="{}">
  Connection options. `project`, `token`, and `host` are required. If any are omitted they fall back to `SIGNALWIRE_PROJECT_ID`, `SIGNALWIRE_API_TOKEN`, and `SIGNALWIRE_SPACE` environment variables.
</ParamField>

## Returns
