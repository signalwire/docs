# `update`

Update a phone number resource by ID. Body is optional to match Python `**kwargs`.

Setting `call_handler` + the matching companion field (see
`PhoneCallHandler`) on the phone number auto-materializes the
matching Fabric resource on the server; prefer the `set*` helpers
below for the common cases.

## Signature

```typescript
update(resourceId: string, body: any = {}): Promise<any>
```

## Parameters

<ParamField path="resourceId" type="string" required>
  Unique phone-number resource ID.
</ParamField>

<ParamField path="body" type="any" required default="{}">
  Partial update payload. Defaults to `{}`.
</ParamField>

## Returns

`Promise<any>` — The updated phone-number resource.

## Throws

- On any non-2xx HTTP response.
