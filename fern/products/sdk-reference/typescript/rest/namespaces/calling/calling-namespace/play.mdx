# `play`

Start media playback on a call.

## Signature

```typescript
play(callId: string, params: any = {}): Promise<any>
```

## Parameters

<ParamField path="callId" type="string" required>
  Target call's ID.
</ParamField>

<ParamField path="params" type="any" required default="{}">
  Playback parameters — see `Play` action schema. Defaults to `{}`.
</ParamField>

## Returns

`Promise<any>` — The play command response, containing a `control_id` used to pause / resume / stop the playback later.

## Throws

- On any non-2xx HTTP response.
