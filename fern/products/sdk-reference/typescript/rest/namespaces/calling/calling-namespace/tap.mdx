# `tap`

Start a media tap (mirror audio to an external URI).

## Signature

```typescript
tap(callId: string, params: any = {}): Promise<any>
```

## Parameters

<ParamField path="callId" type="string" required>
  Target call's ID.
</ParamField>

<ParamField path="params" type="any" required default="{}">
  Tap configuration (`uri`, `direction`, `codec`, etc.). Defaults to `{}`.
</ParamField>

## Returns

`Promise<any>` — The tap command response containing a `control_id`.

## Throws

- On any non-2xx HTTP response.
