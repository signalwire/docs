# `update`

Update a LaML Bin's stored script. LAML uses `POST` (not PATCH/PUT).

## Signature

```typescript
update(sid: string, body: any = {}): Promise<any>
```

## Parameters

<ParamField path="sid" type="string" required>
  LaML Bin SID (e.g. `"LA..."`).
</ParamField>

<ParamField path="body" type="any" required default="{}">
  Payload containing the new `VoiceMethod`, `Url`, etc. Defaults to `{}`.
</ParamField>

## Returns

`Promise<any>` — The updated LaML Bin record.

## Throws

- On any non-2xx HTTP response.
