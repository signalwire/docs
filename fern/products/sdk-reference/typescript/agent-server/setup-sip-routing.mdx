# `setupSipRouting`

Set up central SIP-based routing for the server.

This configures all agents to handle SIP requests at the specified path,
using a coordinated routing system where each agent checks if it can
handle SIP requests for specific usernames.

## Signature

```typescript
setupSipRouting(route: string = '/sip', autoMap: boolean = true): void
```

## Parameters

<ParamField path="route" type="string" required default="'/sip'">
  The path for SIP routing (default: '/sip').
</ParamField>
<ParamField path="autoMap" type="boolean" required default="true">
  Whether to automatically map SIP usernames to agent routes (default: true).
</ParamField>

## Returns

`void`
