# `register`

Register an agent at the given route prefix.

**Throws:**
- If the route is already occupied by another agent.

## Signature

```typescript
register(agent: AgentBase, route?: string): void
```

<ParamField path="agent" type="AgentBase" required>
  The AgentBase instance to mount.
</ParamField>
<ParamField path="route" type="string">
  Route prefix; defaults to the agent's own route or '/'.
</ParamField>
