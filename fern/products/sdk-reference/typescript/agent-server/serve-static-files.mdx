# `serveStaticFiles`

Serve static files from a local directory under a given route prefix.
Includes path traversal protection (rejects `..`), MIME type detection,
and security headers (Cache-Control, X-Content-Type-Options).

## Signature

```typescript
serveStaticFiles(directory: string, route: string = '/'): void
```

## Parameters

<ParamField path="directory" type="string" required>
  Absolute or relative path to the directory to serve.
</ParamField>
<ParamField path="route" type="string" required default="'/'">
  Route prefix for static files (defaults to '/').
</ParamField>

## Returns

`void`
