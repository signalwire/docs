# `AgentSession`

Mirrors a LiveKit `AgentSession` — binds an [Agent](/sdk-reference/typescript/livewire/agent) to SignalWire.

Call [AgentSession.start](/sdk-reference/typescript/livewire/agent-session/start) with an `Agent` to construct an internal
[AgentBase](/sdk-reference/typescript/agent-base) and begin serving SWML. Pipeline-related options are
accepted for API parity but are no-ops server-side.
