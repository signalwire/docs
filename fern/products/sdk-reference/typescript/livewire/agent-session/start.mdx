# `start`

Start the session by binding the agent to a freshly-constructed
[AgentBase](/sdk-reference/typescript/agent-base), mapping LiveKit-style options onto SignalWire AI params.

Must be called before any other method on this session. The underlying
`AgentBase` is not started here — use [runApp](/sdk-reference/typescript/livewire/run-app) or an `AgentServer`
to serve it.
