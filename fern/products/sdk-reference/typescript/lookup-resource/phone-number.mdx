# `phoneNumber`

Look up carrier and CNAM information for a phone number.

## Signature

```typescript
phoneNumber(e164: string, params?: QueryParams): Promise<any>
```

## Parameters

<ParamField path="e164" type="string" required>

</ParamField>
<ParamField path="params" type="QueryParams">
  Optional query parameters, most commonly
