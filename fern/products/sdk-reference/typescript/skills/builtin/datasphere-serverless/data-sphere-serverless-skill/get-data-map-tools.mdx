# `getDataMapTools`

Get DataMap-based tool definitions for server-side execution.
These are registered directly in the SWML output as data\_map functions
rather than as webhook-backed SWAIG tools.

## Signature

```typescript
getDataMapTools(): Record<string, unknown>[]
```

## Returns

`Record<string, unknown>[]` — Array containing the DataSphere search DataMap function definition.
