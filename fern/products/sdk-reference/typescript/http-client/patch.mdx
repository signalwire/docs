# `patch`

Perform an authenticated HTTP PATCH and return the parsed JSON response.

**Throws:**
- On any non-2xx HTTP response.

## Signature

```typescript
patch<T>(path: string, body?: any): Promise<T>
```

</ParamField>

</ParamField>

## Returns

`Promise<T>` — The parsed JSON body, or `{}` on `204 No Content`.
