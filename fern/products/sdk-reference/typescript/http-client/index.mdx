# `HttpClient`

Low-level HTTP client used by every REST namespace resource.

Handles Basic Auth, JSON encoding/decoding, and error normalisation
([RestError](/reference/typescript/rest-error) on non-2xx). Normally you do not instantiate this
directly — construct a [RestClient](/reference/typescript/rest-client) instead.
