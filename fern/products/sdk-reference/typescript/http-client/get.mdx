# `get`

Perform an authenticated HTTP GET and return the parsed JSON response.

**Throws:**
- On any non-2xx HTTP response.

## Signature

```typescript
get<T>(path: string, params?: QueryParams): Promise<T>
```

## Parameters

<ParamField path="path" type="string" required>
  Absolute URL or path relative to [HttpClient.baseUrl](/sdk-reference/typescript/http-client#base-url).
</ParamField>
<ParamField path="params" type="QueryParams">
  Optional query parameters; `undefined` values are skipped.
</ParamField>

## Returns

`Promise<T>` — The parsed JSON body, or `{}` on `204 No Content`.
