# `lock`

Lock one or more skill names to prevent overwriting.
If called with no arguments, locks all currently registered skills.

## Signature

```typescript
lock(names?: string[]): void
```

## Parameters

<ParamField path="names" type="string[]">
  Skill names to lock; if omitted, all current names are locked.
</ParamField>

## Returns

`void`
