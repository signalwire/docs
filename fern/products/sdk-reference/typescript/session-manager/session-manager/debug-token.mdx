# `debugToken`

Debug a token without validating it.

Requires [debugMode](/reference/typescript/session-manager/session-manager#debug-mode) to be `true`. When disabled, returns
`{ error: "debug mode not enabled" }` matching the Python SDK behaviour.

## Signature

```typescript
debugToken(token: string): DebugTokenResult
```

## Parameters

<ParamField path="token" type="string" required>
  The base64url-encoded token to decode.
</ParamField>

## Returns

`DebugTokenResult` — A nested debug structure matching the Python SDK, or an error object.
