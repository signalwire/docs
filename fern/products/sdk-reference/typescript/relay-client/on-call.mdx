# `onCall`

Register the inbound call handler.

The handler is invoked once per inbound call, with a fully-formed
[Call](/sdk-reference/typescript/call) already in state `"created"`. Answer, reject, or forward
the call from inside the handler.

## Signature

```typescript
onCall(handler: CallHandler): CallHandler
```

## Parameters

<ParamField path="handler" type="CallHandler" required>
  Callback invoked for each inbound call. May return a
promise; errors are logged but do not tear down the client.
</ParamField>

## Returns

`CallHandler` — The same handler, to support decorator-style usage.
