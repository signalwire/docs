# `onMessage`

Register the inbound message handler.

The handler is invoked once per inbound SMS/MMS delivered to a subscribed
context, with a [Message](/reference/typescript/message) already in state `"received"`.

## Signature

```typescript
onMessage(handler: MessageHandler): MessageHandler
```

## Parameters

<ParamField path="handler" type="MessageHandler" required>
  Callback invoked for each inbound message. May return a
promise; errors are logged but do not tear down the client.
</ParamField>

## Returns

`MessageHandler` — The same handler, to support decorator-style usage.
