# `unreceive`

Unsubscribe from contexts previously passed to [receive](/reference/typescript/relay-client/receive) (or the
constructor). A no-op when `contexts` is empty.

**Throws:**
- When the server rejects the unsubscribe request.

## Signature

```typescript
unreceive(contexts: string[]): Promise<void>
```

## Parameters

<ParamField path="contexts" type="string[]" required>
  Context names to unsubscribe from.
</ParamField>

## Returns

`Promise<void>` — Resolves once the server has confirmed the unsubscribe.
