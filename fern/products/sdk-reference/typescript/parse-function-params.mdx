# `parseFunctionParams`

Parse function parameter names and default values from source code.

Handles arrow functions, regular functions, and method shorthand.

## Signature

```typescript
parseFunctionParams(source: string): ParsedParam[]
```

## Parameters

<ParamField path="source" type="string" required>
  The function source text, typically from `fn.toString()`.
</ParamField>

## Returns

`ParsedParam[]` — An array of `{ name, defaultValue? }` records in declaration order.
Returns an empty array if no parameter list is present.
