# `createTypedHandlerWrapper`

Create a wrapper function that adapts a typed handler to the standard
`(args, rawData) => result` SWAIG handler signature.

The wrapper extracts named parameters from the args dict and passes them
as positional arguments to the original function.

## Signature

```typescript
createTypedHandlerWrapper(
  fn: Function,
  paramNames: string[],
  hasRawData: boolean
): SwaigHandler
```

## Parameters

<ParamField path="fn" type="Function" required>
  The typed handler whose parameters match `paramNames` in order.
</ParamField>

<ParamField path="paramNames" type="string[]" required>
  Ordered list of parameter names extracted from `fn`, produced by [inferSchema](/reference/typescript/type-inference/infer-schema).
</ParamField>

<ParamField path="hasRawData" type="boolean" required>
  When `true`, the raw-data record is appended as the final positional argument to mirror the old-style handler shape.
</ParamField>

## Returns

`SwaigHandler` — A [SwaigHandler](/reference/typescript/swaig-function/swaig-handler) suitable for registration with [AgentBase.defineTool](/reference/typescript/agent-base/agent-base/define-tool).
