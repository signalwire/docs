# `search`

Search for a config file in standard locations.

Default search paths: CWD, `./config`, `$HOME/.signalwire`,
`.swml/`, `$HOME/.swml/`, `/etc/swml/`.

## Signature

```typescript
search(
  filename: string,
  additionalPaths?: string[],
  serviceName?: string
): ConfigLoader | null
```

## Parameters

<ParamField path="filename" type="string" required>
  The config file name to search for.
</ParamField>
<ParamField path="additionalPaths" type="string[]">
  Extra directories to search before the defaults.
</ParamField>
<ParamField path="serviceName" type="string">
  Optional service name; prepends service-specific filenames to the search.
</ParamField>

## Returns

`ConfigLoader | null` — A loaded ConfigLoader if found, or null if the file does not exist in any search path.
