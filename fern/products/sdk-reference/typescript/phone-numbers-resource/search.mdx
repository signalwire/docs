# `search`

Search available phone numbers for purchase.

**Throws:**
- On any non-2xx HTTP response.

## Signature

```typescript
search(params?: QueryParams): Promise<any>
```

## Parameters

<ParamField path="params" type="QueryParams">
  Search filters (e.g. `areaCode`, `contains`, `region`,
`number_type`, pagination).
</ParamField>

## Returns

`Promise<any>` — A paginated list of matching available numbers.

```typescript
const results = await client.phoneNumbers.search({ areaCode: '512', contains: '5555' });
```
