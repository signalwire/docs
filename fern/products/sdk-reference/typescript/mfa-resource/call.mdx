# `call`

Initiate MFA by placing a phone call that reads out a one-time code.

**Throws:**
- On any non-2xx HTTP response.

## Signature

```typescript
call(body: any): Promise<any>
```

## Parameters

<ParamField path="body" type="any" required>
  MFA request payload (typically `{ to: "+15551234567" }`).
</ParamField>

## Returns

`Promise<any>` — The MFA request record; its `id` is used by [verify](/reference/typescript/mfa-resource/verify).
