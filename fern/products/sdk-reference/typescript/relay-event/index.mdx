# `RelayEvent`

Base class for all typed RELAY events.

Raw events arrive as `signalwire.event` JSON-RPC notifications; the client
looks up the correct subclass in [EVENT_CLASS_MAP](/sdk-reference/typescript/event-class-map) and invokes
[RelayEvent.fromPayload](/sdk-reference/typescript/relay-event/from-payload) to build a typed wrapper. Handlers receive
this wrapper; they can always read the original dict from `params`.
