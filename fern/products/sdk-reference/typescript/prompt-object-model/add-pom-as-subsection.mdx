# `addPomAsSubsection`

Add another PromptObjectModel as subsections of an existing section.

## Signature

```typescript
addPomAsSubsection(target: string | Section, pomToAdd: PromptObjectModel): void
```

## Parameters

<ParamField path="target" type="string | Section" required>
  Either the title of the target section, or a `Section` reference.
</ParamField>
<ParamField path="pomToAdd" type="PromptObjectModel" required>
  The model whose top-level sections will be appended as subsections.
</ParamField>

## Returns

`void`

## Throws

- Error if `target` is a string and no section with that title is found.
