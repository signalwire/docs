
<ParamField path="contexts" type="string[]">

</ParamField>

  RELAY host. Defaults to relay.signalwire.com.
</ParamField>

<ParamField path="jwtToken" type="string">
  JWT token for authentication. Defaults to env SIGNALWIRE\_JWT\_TOKEN.
</ParamField>

<ParamField path="maxActiveCalls" type="number">
  Maximum number of concurrent active calls. Defaults to env RELAY\_MAX\_ACTIVE\_CALLS or 1000.
</ParamField>

<ParamField path="project" type="string">
  SignalWire project ID. Defaults to env SIGNALWIRE\_PROJECT\_ID.
</ParamField>

<ParamField path="scheme" type="&#x22;ws&#x22; | &#x22;wss&#x22;">
  WebSocket scheme — `'wss'` (production, the default) or `'ws'` (loopback fixtures and local audit harnesses). Reads from `SIGNALWIRE_RELAY_SCHEME` if not explicitly set; defaults to `'wss'`. Production deployments should never pass `'ws'` — the value exists solely so the porting-sdk's `audit_relay_handshake.py` can drive the real client against a plain-WS loopback fixture without standing up TLS termination.
</ParamField>

<ParamField path="token" type="string">
  SignalWire API token. Defaults to env SIGNALWIRE\_API\_TOKEN.
</ParamField>
