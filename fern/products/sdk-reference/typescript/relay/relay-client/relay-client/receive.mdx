# `receive`

Subscribe to additional RELAY contexts on an already-connected client.

Inbound calls and messages on any of the listed contexts will be delivered
to the `onCall` / `onMessage` handlers. A no-op when `contexts` is empty.

## Signature

```typescript
receive(contexts: string[]): Promise<void>
```

## Parameters

<ParamField path="contexts" type="string[]" required>
  Context names to subscribe to.
</ParamField>

## Returns

`Promise<void>` — Resolves once the server has confirmed the subscription.

## Throws

- When the server rejects the subscribe request.
