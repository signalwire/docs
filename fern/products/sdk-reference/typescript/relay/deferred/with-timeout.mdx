# `withTimeout`

Race a promise against a timeout.

## Signature

```typescript
withTimeout<T>(
  promise: Promise<T>,
  ms: number,
  label: string = 'Operation'
): Promise<T>
```

## Type Parameters

<ParamField path="T" required>
  Resolution value type of the input promise.
</ParamField>

## Parameters

<ParamField path="promise" type="Promise<T>" required>
  Promise to race against the timeout.
</ParamField>

<ParamField path="ms" type="number" required>
  Timeout in milliseconds.
</ParamField>

<ParamField path="label" type="string" required default="'Operation'">
  Label used in the timeout error message. Defaults to `"Operation"`.
</ParamField>

## Returns

`Promise<T>` — A new promise that resolves with `promise`'s value if it settles before the timeout, and rejects with a timeout `Error` otherwise.
