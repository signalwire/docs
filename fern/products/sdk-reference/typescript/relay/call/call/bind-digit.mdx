# `bindDigit`

Bind a DTMF digit sequence to trigger a RELAY method automatically when
the caller presses it.

## Signature

```typescript
bindDigit(
  digits: string,
  bindMethod: string,
  options: { ...3 fields } = {}
): Promise<Record<string, unknown>>
```

## Parameters

<ParamField path="digits" type="string" required>
  DTMF sequence that triggers the binding (e.g. `"*9"`).
</ParamField>

<ParamField path="bindMethod" type="string" required>
  RELAY method to invoke when the sequence is detected.
</ParamField>

<ParamField path="options" type="object" required default="{}">
  Binding behaviour.

  <ParamField path="options.bindParams" type="Record<string, unknown>">
    Params forwarded to `bindMethod` on trigger.
  </ParamField>

  <ParamField path="options.maxTriggers" type="number">
    Maximum number of times the binding can fire.
  </ParamField>

  <ParamField path="options.realm" type="string">
    Optional realm label so bindings can be cleared in groups.
  </ParamField>
</ParamField>

## Returns

`Promise<Record<string, unknown>>` — The platform's bind\_digit response.

## Throws

- When the bind\_digit command is rejected.
