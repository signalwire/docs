# `playAndCollect`

Play audio and collect digit / speech input in a single operation.

Convenient for IVR prompts — playback stops as soon as input is received.

## Signature

```typescript

  media: Record<string, unknown>[],
  collect: Record<string, unknown>,
  options: { ...3 fields } = {}
```
