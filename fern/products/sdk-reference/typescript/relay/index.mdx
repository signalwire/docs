## Interfaces

### Deferred

A promise with externalised `resolve`/`reject` and a `settled` flag,
used by the RELAY client to bridge server-push events back to in-flight
JSON-RPC request callers.

#### Signature

```typescript
interface Deferred<T>
```

#### Properties

<ParamField path="promise" type="Promise<T>" required>
  The wrapped promise.
</ParamField>

<ParamField path="reject" type="(reason?: unknown) => void" required>
  Reject the promise idempotently (later calls are ignored).
</ParamField>

<ParamField path="resolve" type="(value: T | PromiseLike<T>) => void" required>
  Resolve the promise idempotently (later calls are ignored).
</ParamField>

<ParamField path="settled" type="boolean" required>
  True once either `resolve` or `reject` has been called.
</ParamField>

#### Functions

##### createDeferred

Create a [Deferred](/docs/sdk-reference/reference/typescript/relay#deferred) with externalised `resolve` / `reject` and a
`settled` flag.

The returned object exposes the promise plus its resolve and reject
functions so callers outside the executor can settle it. Later calls to
`resolve` or `reject` are idempotent.

###### Signature

```typescript
createDeferred<T>(): Deferred<T>
```

<ParamField path="T" required>
  Resolution value type of the wrapped promise.
</ParamField>

###### Returns

`Deferred<T>` — A fresh `Deferred<T>` in pending state.

###### Signature


###### Type Parameters

  Timeout in milliseconds.
</ParamField>
