# `startAgent`

Start an agent's HTTP server.

Equivalent to Python's `start_agent(agent)`. Delegates to `agent.serve(options)`.

## Signature

```typescript
startAgent(agent: AgentBase, options?: { ...2 fields }): Promise<void>
```

  The [AgentBase](/sdk-reference/typescript/agent-base) instance to start.
</ParamField>
<ParamField path="options" type="object">
  Optional host / port overrides. When omitted, values come
from the agent's constructor options or the `PORT` environment variable.
</ParamField>

## Returns

`Promise<void>` — Resolves once the HTTP server has begun listening.
