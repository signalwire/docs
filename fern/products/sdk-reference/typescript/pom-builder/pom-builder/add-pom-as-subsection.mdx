# `addPomAsSubsection`

Appends every top-level section of another PomBuilder as subsections of a target section.

## Signature

```typescript
addPomAsSubsection(target: string | PomSection, pomToAdd: PomBuilder): this
```

## Parameters

<ParamField path="target" type="string | PomSection" required>
  The heading of the target section, or the PomSection to append into.
</ParamField>

<ParamField path="pomToAdd" type="PomBuilder" required>
  The PomBuilder whose sections should be appended as subsections.
</ParamField>

## Returns

`this` — This builder for chaining.

## Throws

- If target is a string and no section with that title is found.
