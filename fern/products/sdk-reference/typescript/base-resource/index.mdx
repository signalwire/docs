# `BaseResource`

Abstract base class for all REST resources.

Every namespace resource (e.g. [PhoneNumbersResource](/reference/typescript/phone-numbers-resource), [VideoRooms](/reference/typescript/video-rooms))
extends this class to get a typed reference to the shared [HttpClient](/reference/typescript/http-client)
and a small path-building helper. Not exported for direct use — subclass it.

**Modifiers:** `abstract`
