# `description`

Alias for [purpose](/reference/typescript/data-map/data-map/purpose); sets the LLM-facing tool description.

This string is read by the model to decide WHEN to call this tool.
See [purpose](/reference/typescript/data-map/data-map/purpose) for bad-vs-good examples.
