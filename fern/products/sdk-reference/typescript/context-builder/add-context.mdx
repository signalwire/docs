# `addContext`

Adds a new named context to the builder.

**Throws:**
- When the max-context limit is exceeded or a context with
the same name already exists.

## Signature

```typescript
addContext(name: string): Context
```

## Parameters

<ParamField path="name" type="string" required>
  The unique context name. A single-context flow must name
its only context `"default"`.
</ParamField>

## Returns

`Context` — The newly created [Context](/sdk-reference/typescript/context) for further configuration.
