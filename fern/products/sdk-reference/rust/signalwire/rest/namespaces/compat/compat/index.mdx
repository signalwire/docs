# `Compat`

Compat (Twilio-compatible LAML) API namespace.

Mirrors `signalwire.rest.namespaces.compat.CompatNamespace` from the
Python SDK. The base path is
`/api/laml/2010-04-01/Accounts/{account_sid}` and every sub-resource
