# `to_value`

Convert the model to a `serde_json::Value` (a JSON array of
section dicts). Mirrors Python's `to_dict`. The Rust name
follows serde idiom (`to_value`) but the cross-port surface
audit treats `to_value` ≡ `to_dict`.
