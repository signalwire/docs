# `list_skills_with_params`

Get complete schema for all available skills.
Mirrors Python's signalwire.list_skills_with_params(). Returns a map keyed by skill name where each value contains parameter metadata. Useful for GUI configuration tools, API documentation, or programmatic skill discovery.
C++ skills don't carry rich Python-style parameter introspection in v1, so each entry contains the skill name and an empty parameter map; built-in skills that expose parameter_schema() via SkillBase get richer detail merged in.
