# `swaig_pre_dispatch`

Extension point: invoked between argument parsing and function dispatch on POST /swaig. Returns a target Service* (defaults to this) and an optional short-circuit JSON. If short_circuit is non-null, it's returned as the SWAIG response without calling on_function_call. AgentBase overrides for token validation.
