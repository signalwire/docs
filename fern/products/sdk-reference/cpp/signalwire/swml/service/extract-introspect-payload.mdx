# `extract_introspect_payload`

Pure-string extractor: slice the JSON payload between __SWAIG_TOOLS_BEGIN__ and __SWAIG_TOOLS_END__ sentinels in a captured stdout. Returns empty string if either marker is missing or the order is wrong. Static so the swaig-test CLI / tests can reuse it.
