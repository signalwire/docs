# `PaginatedIterator`

Iterates items across paginated API responses.
Mirrors signalwire-python's signalwire.rest._pagination.PaginatedIterator: fetches the configured path with the configured params, walks the data_key array, then follows links.next (parsing its query string for the next page's params) until the response carries no links.next.
Iteration is lazy  the constructor records inputs but performs no HTTP. The first has_next() / next() call performs the first fetch. Cursor query params are extracted by parsing links.next like Python's urllib.parse.urlparse + parse_qs.
