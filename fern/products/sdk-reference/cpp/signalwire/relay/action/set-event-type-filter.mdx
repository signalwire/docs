# `set_event_type_filter`

Set the wire-event types the Action should accept state updates from. Empty (the default) means "match any component event whose
control_id matches this Action". Used by play_and_collect to listen on calling.call.collect only — a calling.call.play(finished) must NOT resolve a play_and_collect action.
