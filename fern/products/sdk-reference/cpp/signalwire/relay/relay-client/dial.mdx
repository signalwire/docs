# `dial`

Dial outbound. The devices argument is the nested "device-of-leg-of-leg" array used by the Python SDK ([[\{type:phone,...}]]). Returns a Call once the server emits calling.call.dial(answered) for the dial's tag, or an empty Call on timeout / failure.
tag lets callers pin an explicit dial tag for journal-based assertions; if blank, a UUID is generated. dial_timeout_ms caps how long dial() blocks waiting for the server's terminal dial event. max_duration (seconds) is forwarded into the calling.dial frame when non-zero.
