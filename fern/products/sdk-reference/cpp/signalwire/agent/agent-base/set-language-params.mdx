# `set_language_params`

Set (or replace) the per-language params dict on an already-added language. Useful when language entries are built via add_language() first and engine-specific tuning is added later (e.g. from a config loader). Passing an empty object removes the params key (treated as unset). No-op if code isn't found among previously-added languages.
Python parity: AIConfigMixin.set_language_params (029ca6f).
