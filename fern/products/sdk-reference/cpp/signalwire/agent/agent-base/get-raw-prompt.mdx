# `get_raw_prompt`

Returns the raw prompt text whatever set_prompt_text stored, or std::nullopt when no raw prompt has been set. Distinct from get_prompt which renders the POM array when use_pom is true.
Mirrors Python's PromptManager.get_raw_prompt.
