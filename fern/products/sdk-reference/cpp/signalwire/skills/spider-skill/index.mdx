# `SpiderSkill`

Spider scrape skill — issues a real GET against the URL the LLM passes in. Strips HTML tags from the response and returns the text payload. Matches Python SpiderSkill's scrape_url behavior.
SPIDER_BASE_URL env var overrides the host portion of the URL the caller passes in (used by audit_skills_dispatch.py to redirect scrape requests at a loopback fixture).
