# `LoggingConfig`

Cross-language SDK contract for serverless / deployment-mode detection.

Mirrors signalwire.core.logging\_config.get\_execution\_mode and
signalwire.utils.is\_serverless\_mode in the Python reference. Order
of precedence (FIRST match wins):

1. GATEWAY\_INTERFACE                                          -> 'cgi'
2. AWS\_LAMBDA\_FUNCTION\_NAME or LAMBDA\_TASK\_ROOT               -> 'lambda'
3. FUNCTION\_TARGET, K\_SERVICE, or GOOGLE\_CLOUD\_PROJECT        -> 'google\_cloud\_function'
4. AZURE\_FUNCTIONS\_ENVIRONMENT, FUNCTIONS\_WORKER\_RUNTIME, or
   AzureWebJobsStorage                                        -> 'azure\_function'
5. otherwise                                                  -> 'server'

The static methods getExecutionMode() and isServerlessMode() project
onto the Python free functions

- signalwire.core.logging\_config.get\_execution\_mode
- signalwire.utils.is\_serverless\_mode
  via scripts/enumerate\_signatures.py FREE\_FUNCTION\_PROJECTIONS.

**Modifiers:** `final`
