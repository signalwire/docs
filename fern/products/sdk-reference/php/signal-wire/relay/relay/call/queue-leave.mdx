# `queueLeave`

Remove the call from a named queue.

Mirrors Python's Call.queue\_leave(queue\_name, \*, control\_id=None,
queue\_id=None, status\_url=None, \*\*kwargs). When called with no args
the legacy "leave the current queue" behavior is preserved.
