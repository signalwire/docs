# `logging`

Simple logging system with level control via environment variables.

&lt;p>
Levels: DEBUG, INFO, WARN, ERROR, OFF
&lt;p>
Environment variables:
&lt;ul>
  &lt;li>SIGNALWIRE_LOG_LEVEL - set log level (debug/info/warn/error/off)&lt;/li>
  &lt;li>SIGNALWIRE_LOG_MODE  - set to "off" to suppress all output&lt;/li>
&lt;/ul>

**Modifiers:** `final`
