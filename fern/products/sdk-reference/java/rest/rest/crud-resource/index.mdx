# `CrudResource`

Generic CRUD resource for REST API namespaces.

&lt;p>
Provides standard list, get, create, update, and delete operations
against a base path.

Used by namespace classes to avoid repetitive
HTTP boilerplate.

&lt;pre>numbers = new CrudResource(httpClient, "/phone_numbers");
var all = numbers.list();
var one = numbers.get("pn-abc-123");&lt;/pre>
