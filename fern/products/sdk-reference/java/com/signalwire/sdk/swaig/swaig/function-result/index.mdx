# `FunctionResult`

SwaigFunctionResult - the response builder returned by tool handlers.

&lt;p>
 Every method returns `this` for fluent chaining. Contains 40+ action methods
 covering call control, state management, media, speech/AI config, and advanced features.
 &lt;p>
 Serialization rules:
 &lt;ul>
   &lt;li>response - always included (string)&lt;/li>
   &lt;li>action - only included if at least one action exists (array of objects)&lt;/li>
   &lt;li>post_process - only included if true and actions exist (boolean)&lt;/li>
 &lt;/ul>
