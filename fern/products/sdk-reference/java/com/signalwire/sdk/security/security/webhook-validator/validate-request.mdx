# `validateRequest`

Legacy `@signalwire/compatibility-api` drop-in entry point.

&lt;p>Dispatches on the runtime type of `paramsOrRawBody`:
 &lt;ul>
   &lt;li>`String` \u2192 delegates to
       String, String, String).&lt;/li>
   &lt;li>`Map` or any `Iterable` of `Map.Entry` (or
       2-element arrays / lists) \u2192 treats the value as pre-parsed form
       params and runs Scheme B directly with URL port normalization.&lt;/li>
   &lt;li>`null` \u2192 treated as an empty params map (Scheme B).&lt;/li>
   &lt;li>Anything else \u2192 `IllegalArgumentException`.&lt;/li>
 &lt;/ul>

**Throws:**
- IllegalArgumentException when `signingKey` is empty or
                   `paramsOrRawBody` is of an unsupported type.
