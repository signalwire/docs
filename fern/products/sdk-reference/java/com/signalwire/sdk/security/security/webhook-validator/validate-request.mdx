# `validateRequest`

Legacy `@signalwire/compatibility-api` drop-in entry point.

&lt;p>Dispatches on the runtime type of `paramsOrRawBody`:
&lt;ul>
  &lt;li>`String` → delegates to
      String, String, String).&lt;/li>
  &lt;li>`Map` or any `Iterable` of `Map.Entry` (or
      2-element arrays / lists) → treats the value as pre-parsed form
      params and runs Scheme B directly with URL port normalization.&lt;/li>
  &lt;li>`null` → treated as an empty params map (Scheme B).&lt;/li>
  &lt;li>Anything else → `IllegalArgumentException`.&lt;/li>
&lt;/ul>

**Modifiers:** `static`

## Returns

`boolean` — `true` on match, `false` otherwise.

## Throws
