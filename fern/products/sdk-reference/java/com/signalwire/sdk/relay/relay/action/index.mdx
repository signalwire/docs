# `Action`

Base class for long-running call actions tracked by `control_id`.

&lt;p>
 Actions support three completion patterns:
 &lt;ol>
   &lt;li>&lt;b>Wait inline&lt;/b>: `action.waitForCompletion()` blocks until terminal&lt;/li>
   &lt;li>&lt;b>Fire and forget&lt;/b>: don't wait, check `isDone()` later&lt;/li>
   &lt;li>&lt;b>Callback&lt;/b>: pass an `onCompleted` callback that fires on terminal state&lt;/li>
 &lt;/ol>
 &lt;p>
 Subclasses add action-specific sub-commands (pause, resume, volume, etc.).
