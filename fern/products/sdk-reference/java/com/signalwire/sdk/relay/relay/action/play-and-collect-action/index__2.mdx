# `PlayAndCollectAction`

Play-and-collect action.

&lt;p>
 Shares one `control_id` across both play and collect phases.
 Events arrive as BOTH `calling.call.play` and `calling.call.collect`.
 This action only resolves on collect events, NOT play events.
