# `setRealEnvOverrideForTests`

Inject an alternative "real env" for the simulator's fallback
 layer.

Primarily for tests that need to pretend the process env
 has `SWML_PROXY_URL_BASE` set without actually mutating it.
 Null means "use `EnvProvider#SYSTEM`".

void setRealEnvOverrideForTests(com.signalwire.sdk.runtime.EnvProvider env)
