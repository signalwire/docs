# `withBaseUrl`

Build a `RestClient` pointed at an explicit base URL \u2014 typically
a loopback fixture used by the porting-sdk's REST-transport audit.

The
returned client signs requests with the given `project`/`token`
pair via Basic Auth and routes every namespace's HTTP through the
fixture instead of the live SignalWire space.

**Modifiers:** `static`

## Signature

```java
public static com.signalwire.sdk.rest.RestClient withBaseUrl(
    java.lang.String baseUrl,
    java.lang.String project,
    java.lang.String token
)
```

## Parameters

<ParamField path="baseUrl" type="java.lang.String" required>
  fully qualified base URL (e.g. `"http://127.0.0.1:NNNN/api"`); `"/api"` is appended if not already present
</ParamField>

<ParamField path="project" type="java.lang.String" required>
  project ID used as the Basic Auth username
</ParamField>

<ParamField path="token" type="java.lang.String" required>
  API token used as the Basic Auth password
</ParamField>
