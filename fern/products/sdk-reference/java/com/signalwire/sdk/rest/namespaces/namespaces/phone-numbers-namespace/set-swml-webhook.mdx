# `setSwmlWebhook`

Route inbound calls on this phone number to an SWML webhook URL.

&lt;p>
Your backend returns an SWML document per call.

The server
auto-creates a `swml_webhook` Fabric resource keyed off this URL.

## Signature

```java
public Map<String,Object> setSwmlWebhook(String id, String url)
```

## Parameters

<ParamField path="id" type="String" required>
  phone number SID
</ParamField>

<ParamField path="url" type="String" required>
  SWML endpoint the server should fetch per call
</ParamField>

## Returns

`Map<String,Object>` — the updated phone number representation
