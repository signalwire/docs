# `detect`

Detect the execution mode from the given `EnvProvider`.

&lt;p>Detection order matches the Python reference: CGI, Lambda,
 Google Cloud Functions, Azure Functions, then fall back to
 `#SERVER`. Order matters because some environments set
 overlapping variables (e.g. a container running under API
 Gateway in "Lambda container mode").
