### EnvProvider

Simple read-only view of process environment variables.

&lt;p>Extracted as an interface so tests can inject a deterministic map
instead of relying on actual OS environment variables (which Java
cannot mutate at runtime without native hackery).

&lt;p>The default implementation delegates to `System#getenv(String)`.

## Enums

### ExecutionMode

Runtime deployment environments the SDK knows about.

&lt;p>Detected from well-known environment variables set by the hosting
platform.

Mirrors `get_execution_mode()` in the Python SDK and
`detectPlatform()` in the TypeScript SDK.

&lt;p>Use `#detect()` to read the actual process environment, or
`#detect(EnvProvider)` to pass a test double.

#### Signature

```java
public enum ExecutionMode
```

##### getExecutionMode

Cross-language SDK contract: return the execution mode as the
canonical lower-case-with-underscores string used by every port.
Mirrors `signalwire.core.logging_config.get_execution_mode`
in Python: one of `"cgi"`, `"lambda"`,
`"google_cloud_function"`, `"azure_function"`, or
`"server"`.
