# `stripAgentPath`

Strip the agent's route prefix from an incoming path.

&lt;p>`agentPath` is `""` for root or e.g. `"/my-agent"`.
 Returns the remainder (e.g. `"/swaig"`) or `null` if the
 path doesn't start with the agent's route.
