# `Section`

Represents a section in the Prompt Object Model.

&lt;p>Each section contains a title, optional body text, optional bullet points,
 and can have any number of nested subsections. Mirrors the Python reference
 `signalwire.pom.pom.Section` (see signalwire-python: pom/pom.py).

 &lt;p>Idiom mapping:
 &lt;ul>
   &lt;li>Python `to_dict()` \u2194 Java `#toMap()` (returns
       `Map&lt;String,Object>`)&lt;/li>
   &lt;li>Python section_number) \u2194 Java
       java.util.List)&lt;/li>
   &lt;li>Python section_number) \u2194 Java
       java.util.List)&lt;/li>
 &lt;/ul>

 &lt;p>Output format is byte-identical to Python's renderer so cross-port POMs
 can round-trip through markdown/XML/JSON.
