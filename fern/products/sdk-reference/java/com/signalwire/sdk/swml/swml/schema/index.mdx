# `Schema`

Singleton that loads schema.json from resources and extracts the 38 SWML verb definitions.

&lt;p>
 The extraction algorithm:
 &lt;ol>
   &lt;li>Parse schema.json&lt;/li>
   &lt;li>For each entry in $defs.SWMLMethod.anyOf, get the $ref&lt;/li>
   &lt;li>Look up the definition in $defs&lt;/li>
   &lt;li>The actual verb name is the first key in properties&lt;/li>
 &lt;/ol>

**Modifiers:** `final`
