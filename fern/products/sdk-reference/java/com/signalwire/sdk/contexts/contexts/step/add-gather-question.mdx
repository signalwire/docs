# `addGatherQuestion`

Add a question to this step's gather\_info configuration.
String, String) must be called first.

&lt;p>&lt;b>IMPORTANT — gather mode locks function access:&lt;/b> While the model
is asking gather questions, the runtime forcibly deactivates ALL of the
step's other functions.

The only callable tools during a gather
question are:

&lt;ul>
  &lt;li>`gather_submit` (the native answer-submission tool)&lt;/li>
  &lt;li>Whatever names you pass in this question's `functions`
      argument&lt;/li>
&lt;/ul>

&lt;p>`next_step` and `change_context` are also filtered out —
the model cannot navigate away until the gather completes. This is by
design: it forces a tight ask → submit → next-question loop.

&lt;p>If a question needs to call out to a tool (e.g. validate an email,
geocode a ZIP), list that tool name in this question's `functions`
argument. Functions listed here are active ONLY for this question.
