# `setInitialStep`

Set which step the context starts on when entered.

&lt;p>By default, a context starts on its first step (index 0).

Use this
to skip a preamble step on re-entry via `change_context`.

## Signature

```java
public Context setInitialStep(String stepName)
```

## Parameters

<ParamField path="stepName" type="String" required>
  name of the step to start on (must exist in this context).
</ParamField>

## Returns

`Context` — this context for chaining.
