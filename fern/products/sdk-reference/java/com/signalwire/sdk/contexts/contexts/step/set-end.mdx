# `setEnd`

Mark this step as terminal for the step flow.

&lt;p>&lt;b>IMPORTANT:&lt;/b> `end=true` does NOT end the conversation or
 hang up the call. It exits step mode entirely after this step executes
 \u2014 clearing the steps list, current step index, valid_steps, and
 valid_contexts. The agent keeps running, but operates only under the
 base system prompt and the context-level prompt; no more step
 instructions are injected and no more `next_step` tool is offered.

 &lt;p>To actually end the call, call a hangup tool or define a
 hangup hook.

## Signature

```java
public com.signalwire.sdk.contexts.Step setEnd(boolean end)
```

## Parameters

<ParamField path="end" type="boolean" required>
  true to exit step mode after this step.
</ParamField>

`com.signalwire.sdk.contexts.Step` — this step for chaining.
