# `setIsolated`

Mark this context as isolated \u2014 entering it wipes conversation history.

&lt;p>When `isolated=true` and the context is entered via
 change_context, the runtime wipes the conversation array. The model
 starts fresh with only the new context's system_prompt + step
 instructions, with no memory of prior turns.

 &lt;p>&lt;b>EXCEPTION \u2014 reset overrides the wipe:&lt;/b> If the context also has
 a reset configuration (via `#setConsolidate(boolean)` or
 `#setFullReset(boolean)`), the wipe is skipped in favor of the
 reset behavior. Use reset with consolidate=true to summarize prior
 history into a single message instead of dropping it entirely.

 &lt;p>Use cases: switching to a sensitive billing flow that should not see
 prior small-talk; handing off to a different agent persona; resetting
 after a long off-topic detour.
