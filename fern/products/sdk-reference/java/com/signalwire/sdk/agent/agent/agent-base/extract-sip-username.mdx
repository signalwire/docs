# `extractSipUsername`

Extract the username portion from a SIP URI.

&lt;p>
 Handles formats:
 &lt;ul>
   &lt;li>`sip:user@host` -> `user`&lt;/li>
   &lt;li>`sip:user@host:port` -> `user`&lt;/li>
   &lt;li>`user@host` -> `user`&lt;/li>
   &lt;li>`+15551234567` -> `+15551234567` (returned as-is)&lt;/li>
 &lt;/ul>

## Signature

```java
public static java.lang.String extractSipUsername(java.lang.String sipUri)
```

## Parameters

<ParamField path="sipUri" type="java.lang.String" required>
  The SIP URI or phone number
</ParamField>

## Returns

`java.lang.String` — The extracted username, or the original string if no @ is found
