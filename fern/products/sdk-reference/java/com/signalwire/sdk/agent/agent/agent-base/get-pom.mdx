# `getPom`

Read-only snapshot of the agent's POM as a typed
 `PromptObjectModel`.

Wraps the internal map-based section list so
 callers get the rich Section / render API without mutating internal
 state.

 &lt;p>Python parity: `agent.pom` instance attribute
 (agent_base.py line 209). Returns `null` when `usePom` is
 false (mirroring Python's = None).
