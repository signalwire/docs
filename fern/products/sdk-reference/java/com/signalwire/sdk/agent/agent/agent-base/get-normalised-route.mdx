# `getNormalisedRoute`

Return the agent's route normalised to an empty string for the
root route or `"/<path>"` otherwise.

Exposed so non-HTTP
transports can construct paths correctly.

## Signature

```java
public String getNormalisedRoute()
```

## Returns

`String` — normalised route prefix.
