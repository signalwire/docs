# `addInternalFiller`

Add internal fillers for a single internal function and language.

&lt;p>See `#setInternalFillersMap(java.util.Map)` for the complete
 list of supported function names and an explanation of what fillers
 do. Names outside the supported set log a warning and are stored, but
 the runtime will not play them.

## Signature

```java
public com.signalwire.sdk.agent.AgentBase addInternalFiller(
    java.lang.String functionName,
    java.lang.String languageCode,
    java.util.List<java.lang.String> fillers
)
```

## Parameters

<ParamField path="functionName" type="java.lang.String" required>
  one of `#SUPPORTED_INTERNAL_FILLER_NAMES`.
</ParamField>
<ParamField path="languageCode" type="java.lang.String" required>
  BCP-47 language code (e.g. `"en-US"`).
</ParamField>
<ParamField path="fillers" type="java.util.List<java.lang.String>" required>
  phrases to speak while the function runs.
</ParamField>

## Returns

`com.signalwire.sdk.agent.AgentBase` — this agent for chaining.
