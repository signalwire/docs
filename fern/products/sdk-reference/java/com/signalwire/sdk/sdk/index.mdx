# `sdk`

Top-level convenience entry points — mirror Python's
`signalwire/__init__.py` package-level helpers
(`RestClient`, `register_skill`,
`add_skill_directory`, `list_skills_with_params`).

&lt;p>This is a static-only utility class; it cannot be instantiated.

The
helpers delegate to the underlying classes (RestClient, SkillRegistry)
so they're a strict thin layer.

&lt;p>The audit projects each method onto the canonical Python
`signalwire.&lt;name>` free-function path via a per-port
`FREE_FUNCTION_PROJECTIONS` entry in
`scripts/enumerate_signatures.py`.

**Modifiers:** `final`

### addSkillDirectory

Add a directory to search for skills.

&lt;p>Mirrors Python's `signalwire.add_skill_directory(path)` —
delegates to the singleton `SkillRegistry` instance so
third-party skill collections can be registered by path.
Subsequent calls accumulate (de-duplicated) into a shared
external paths list.

**Modifiers:** `static`

### listSkillsWithParams

Get complete schema for all available skills.

&lt;p>Mirrors Python's `signalwire.list_skills_with_params()`.
Returns a map keyed by skill name where each value contains
parameter metadata.

Useful for GUI configuration tools, API
documentation, or programmatic skill discovery.

&lt;p>Java skills don't carry rich Python-style parameter
introspection in v1, so each entry contains the skill name and an
empty parameter map; built-in skills that expose
`parameterSchema()` get richer detail.

**Modifiers:** `static`

#### Signature

```java
public static Map<String,Map<String,Object>> listSkillsWithParams()
```

#### Returns

`Map<String,Map<String,Object>>` — map of skill name to schema metadata

### registerSkill

Register a custom skill class with the global `SkillRegistry`.

&lt;p>Mirrors Python's `signalwire.register_skill(skill_class)`.
Java skills are constructed via a no-arg constructor (the
registry stores `Supplier&lt;SkillBase>` factories), so we
adapt the supplied class reference into a factory using
reflection.

Throws `IllegalArgumentException` when the
class can't be instantiated reflectively or when its skill name
cannot be derived.

**Modifiers:** `static`

#### Signature

```java
public static void registerSkill(Class<? extends SkillBase> skillClass)
```

#### Parameters

<ParamField path="skillClass" type="Class<? extends SkillBase>" required>
  a `SkillBase` subclass
</ParamField>

### RestClient

Construct a `RestClient` from positional or keyword
credentials.

&lt;p>Mirrors Python's top-level **kwargs) factory.

Supports two call shapes:

&lt;ul>
  &lt;li>token, space) — three positional
      strings, mapping straight onto the Java builder.&lt;/li>
  &lt;li>kwargs) — variadic-shaped form
      matching Python's signature exactly. `args` is an
      array of three strings (project/token/space) or empty;
      `kwargs` carries the same fields by name.&lt;/li>
&lt;/ul>

&lt;p>The audit projects this method onto Python's
**kwargs) via
`FREE_FUNCTION_PROJECTIONS`.

**Modifiers:** `static`
