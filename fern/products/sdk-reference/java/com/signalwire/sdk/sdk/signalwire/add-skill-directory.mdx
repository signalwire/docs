# `addSkillDirectory`

Add a directory to search for skills.

&lt;p>Mirrors Python's `signalwire.add_skill_directory(path)` \u2014
 delegates to the singleton `SkillRegistry` instance so
 third-party skill collections can be registered by path.
 Subsequent calls accumulate (de-duplicated) into a shared
 external paths list.

**Throws:**
- IllegalArgumentException when the path doesn't exist or
                                  isn't a directory

