# `addSkillDirectory`

Add a directory to search for skills.

&lt;p>Mirrors Python's `signalwire.add_skill_directory(path)` \u2014
 delegates to the singleton `SkillRegistry` instance so
 third-party skill collections can be registered by path.
 Subsequent calls accumulate (de-duplicated) into a shared
 external paths list.

**Modifiers:** `static`

## Signature

```java
public static void addSkillDirectory(java.lang.String path)
```

## Parameters

<ParamField path="path" type="java.lang.String" required>
  absolute or relative path to a directory containing skill subdirectories
</ParamField>

## Throws

- `IllegalArgumentException` — when the path doesn't exist or isn't a directory
