### getDialStateEnum

The dial outcome as a typed `DialState`, &lt;em>alongside&lt;/em> the
raw string `#getDialState()` (which stays canonical for parity
and forward-compat).

Returns `Optional.empty()` when the wire
`dial_state` is not one of the three known `DialState`
values — the set mirrors server-emitted values that can grow, so an
unrecognised state is tolerated here rather than crashing dispatch.

#### Signature

```java
public Optional<DialState> getDialStateEnum()
```

#### Returns

`Optional<DialState>` — the typed dial state, or empty if unknown/unset.
