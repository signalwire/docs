### configure\_logging

Configure logging system once, globally, based on environment variables

> \[!NOTE]
> SIGNALWIRE\_LOG\_MODE: off, stderr, default, auto
> SIGNALWIRE\_LOG\_LEVEL: debug, info, warning, error, critical
> SIGNALWIRE\_LOG\_FORMAT: console, json (default: console)

#### Signature

```python
configure_logging()
```

### get\_logger

Get a logger instance for the specified name with structured logging support

This is the single entry point for all logging in the SDK.
All modules should use this instead of direct logging module usage.

#### Signature

```python
get_logger(name)
```

#### Parameters

<ParamField path="name" required>
  Logger name, typically **name**
</ParamField>

#### Returns

A structlog BoundLogger that supports .bind(), .info(), .debug(), etc.
