### search\_and\_scrape\_best

Search and scrape with quality filtering and source diversity.

#### Signature

```python
search_and_scrape_best(
    query: str,
    num_results: int = 3,
    oversample_factor: float = 4.0,
    delay: float = 0.5,
    min_quality_score: float = 0.2,
    per_page_timeout: float = 2.0,
    overall_deadline: float = 10.0,
    parallel_scrape: bool = True,
    snippets_only: bool = False
) -> str
```

#### Parameters

<ParamField path="query" type="str" required>
  Search query
</ParamField>

<ParamField path="num_results" type="int" default="3">
  Number of best results to return
</ParamField>

<ParamField path="oversample_factor" type="float" default="4.0">
  How many extra results to fetch (e.g., 4.0 = fetch 4x)
</ParamField>

<ParamField path="delay" type="float" default="0.5">
  Delay between scrape requests (ignored in parallel mode)
</ParamField>

<ParamField path="min_quality_score" type="float" default="0.2">
  Minimum quality score to include a result
</ParamField>

<ParamField path="per_page_timeout" type="float" default="2.0">
  Max seconds to wait on each page scrape
</ParamField>

<ParamField path="overall_deadline" type="float" default="10.0">
  Wall-clock budget (seconds) for the whole call. Anything not yet returned by this deadline is abandoned.
</ParamField>

<ParamField path="parallel_scrape" type="bool" default="True">
  Run all page scrapes concurrently in a thread pool
</ParamField>

<ParamField path="snippets_only" type="bool" default="False">
  Skip page scraping entirely; format the CSE snippets directly. Use for fast voice answers when full page content is not required.
</ParamField>

#### Returns

`str` — Formatted string with the best N results from diverse sources
