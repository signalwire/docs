# `setup_sip_routing`

Set up central SIP-based routing for the server

This configures all agents to handle SIP requests at the specified path,
using a coordinated routing system where each agent checks if it can
handle SIP requests for specific usernames.

## Signature

```python
setup_sip_routing(route: str = '/sip', auto_map: bool = True) -> None
```

## Parameters

<ParamField path="route" type="str" default="'/sip'">
  The path for SIP routing (default: "/sip")
</ParamField>

<ParamField path="auto_map" type="bool" default="True">
  Whether to automatically map SIP usernames to agent routes
</ParamField>

## Returns

`None`
