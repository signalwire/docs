# `run`

Universal run method that automatically detects environment and handles accordingly

Detects execution mode and routes appropriately:

- Server mode: Starts uvicorn server with FastAPI
- CGI mode: Uses same routing logic but outputs CGI headers
- Lambda mode: Uses same routing logic but returns Lambda response

## Signature

```python
run(
    event = None,
    context = None,
    host: Optional[str] = None,
    port: Optional[int] = None
) -> Any
```

## Parameters

<ParamField path="event" default="None">
  Serverless event object (Lambda, Cloud Functions)
</ParamField>

<ParamField path="context" default="None">
  Serverless context object (Lambda, Cloud Functions)
</ParamField>

<ParamField path="host" type="Optional[str]" default="None">
  Optional host to override the default (server mode only)
</ParamField>

<ParamField path="port" type="Optional[int]" default="None">
  Optional port to override the default (server mode only)
</ParamField>

## Returns

`Any` — Response for serverless modes, None for server mode
