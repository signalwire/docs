# `load_service_from_file`

Load a SWML service from a Python file

## Signature

```python
load_service_from_file(
    service_path: str,
    service_identifier: Optional[str] = None,
    prefer_route: bool = True
) -> SWMLService
```

## Parameters

<ParamField path="service_path" type="str" required>
  Path to the Python file containing the service
</ParamField>

<ParamField path="service_identifier" type="Optional[str]" default="None">
  Optional service identifier - can be class name or route
</ParamField>

<ParamField path="prefer_route" type="bool" default="True">
  If True, interpret identifier as route first, then class name
</ParamField>

## Returns

`SWMLService` — SWMLService instance (could be AgentBase or basic SWMLService)

## Throws

- `ImportError` — If the file cannot be imported
- `ValueError` — If no service is found in the file
