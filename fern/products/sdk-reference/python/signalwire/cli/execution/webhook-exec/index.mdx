## Functions

### execute\_external\_webhook\_function

Execute an external webhook SWAIG function by making an HTTP request to the external service.
This simulates what SignalWire would do when calling an external webhook function.

#### Signature

```python
execute_external_webhook_function(
    func: SWAIGFunction,
    function_name: str,
    function_args: Dict[str, Any],
    post_data: Dict[str, Any],
    verbose: bool = False
) -> Dict[str, Any]
```

#### Parameters

<ParamField path="func" type="SWAIGFunction" required>
  The SWAIGFunction object with webhook\_url
</ParamField>

<ParamField path="function_name" type="str" required>
  Name of the function being called
</ParamField>

<ParamField path="function_args" type="Dict[str, Any]" required>
  Parsed function arguments
</ParamField>
