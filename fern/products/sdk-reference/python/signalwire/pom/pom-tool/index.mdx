# `pom_tool`

POM Tool - Command line utility for working with Prompt Object Model files

> \[!NOTE]
> pom\_tool \<input\_file> \[--output=&lt;format>] \[--outfile=&lt;file>] \[--merge\_pom="&lt;section name>:&lt;filename>"]
> pom\_tool (-h | --help)

> \[!NOTE]
> -h --help           Show this help message
> \--output=&lt;format>   Output format: md, xml, json, yaml \[default: md]
> \--outfile=&lt;file>    Output file (if not specified, prints to stdout)
> \--merge\_pom=&lt;arg>   Merge another POM into a section: "&lt;section name>:&lt;filename>"
