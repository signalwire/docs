# `add_subsection`

Add a subsection to this section.

## Signature

```python
add_subsection(
    title: str,
    *,
    body: str = '',
    bullets: Optional[List[str]] = None,
    numbered: bool = False,
    numberedBullets: bool = False
) -> Section
```

## Parameters

<ParamField path="title" type="str" required>
  The title of the subsection
</ParamField>

<ParamField path="body" type="str" default="''">
  Optional body text for the subsection
</ParamField>

<ParamField path="bullets" type="Optional[List[str]]" default="None">
  Optional list of bullet points
</ParamField>

<ParamField path="numbered" type="bool" default="False">
  Whether this section should be numbered
</ParamField>

<ParamField path="numberedBullets" type="bool" default="False">
  Whether bullets should be numbered
</ParamField>

## Returns

`Section` — The newly created Section object

## Throws

- `ValueError` — If the title is None or if the section has neither a body nor bullets
