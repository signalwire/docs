# `render_markdown`

Render this section and all its subsections as markdown.

## Signature

```python

    level: int = 2,
    section_number: Optional[List[int]] = None
) -> str
```

  The heading level to start with (default: 2, which corresponds to ##)
</ParamField>

<ParamField path="section_number" type="Optional[List[int]]" default="None">
  The current section number for numbered sections
</ParamField>
