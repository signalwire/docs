# `add_pom_as_subsection`

Add another PromptObjectModel as a subsection to a section with the given title or section object.

## Signature

```python
add_pom_as_subsection(
    target: Union[str, Section],
    pom_to_add: PromptObjectModel
)
```

## Parameters

<ParamField path="target" type="Union[str, Section]" required>
  The title of the section or the Section object to which the POM should be added as a subsection.
</ParamField>

<ParamField path="pom_to_add" type="PromptObjectModel" required>
  The PromptObjectModel to add as a subsection.
</ParamField>

## Throws

- `ValueError` — If no section with the target title is found (when target is a string).
