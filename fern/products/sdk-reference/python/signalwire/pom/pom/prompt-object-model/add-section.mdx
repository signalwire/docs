# `add_section`

Add a top-level section to the model.

## Signature

```python
add_section(
    title: Optional[str] = None,
    *,
    body: str = '',
    bullets: Optional[Union[List[str], str]] = None,
    numbered: Optional[bool] = None,
    numberedBullets: bool = False
) -> Section
```

## Parameters

<ParamField path="title" type="Optional[str]" default="None">
  The title of the section
</ParamField>

<ParamField path="body" type="str" default="''">
  Optional body text for the section
</ParamField>

<ParamField path="bullets" type="Optional[Union[List[str], str]]" default="None">
  Optional list of bullet points or a single string
</ParamField>

<ParamField path="numbered" type="Optional[bool]" default="None">
  Whether this section should be numbered
</ParamField>

<ParamField path="numberedBullets" type="bool" default="False">
  Whether bullets should be numbered
</ParamField>

## Returns

`Section` — The newly created Section object

## Throws

- `ValueError` — If a section without a title is added after the first section
