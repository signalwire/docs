# `register_skill`

Register a skill class directly

This allows third-party code to register skill classes without
requiring them to be in a specific directory structure.

> \[!NOTE]
> from my\_custom\_skills import MyWeatherSkill
> skill\_registry.register\_skill(MyWeatherSkill)

## Signature

```python
register_skill(skill_class: Type[SkillBase]) -> None
```

## Parameters

<ParamField path="skill_class" type="Type[SkillBase]" required>
  A class that inherits from SkillBase
</ParamField>

## Returns

`None`
