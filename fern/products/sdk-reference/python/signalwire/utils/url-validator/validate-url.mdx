# `validate_url`

Validate that a URL is safe to fetch (not pointing to private/internal resources).

## Signature

```python
validate_url(url: str, allow_private: bool = False) -> bool
```

## Parameters

<ParamField path="url" type="str" required>
  The URL to validate
</ParamField>

<ParamField path="allow_private" type="bool" default="False">
  If True, allow private IP ranges (default: False)
</ParamField>

## Returns

`bool` — True if the URL is safe to fetch, False otherwise
