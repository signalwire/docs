# `RelayClient`

Manages a WebSocket connection to SignalWire RELAY.

Usage::

```
client = RelayClient(project="...", token="...", contexts=["default"])
```

```
@client.on_call
async def handle(call):
    await call.answer()
    await call.hangup()
```

```
client.run()
```
