# `add_skill_directory`

Add a directory to search for skills

This allows third-party skill collections to be registered by path.
Skills in these directories should follow the same structure as built-in skills.

> \[!NOTE]
>
> > > > add\_skill\_directory('/opt/custom\_skills')
> > > >
> > > > # Now agent.add\_skill('my\_custom\_skill') will search in this directory

## Signature

```python
add_skill_directory(path)
```

## Parameters

<ParamField path="path" required>
  Path to directory containing skill subdirectories
</ParamField>
