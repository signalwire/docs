# `create_typed_handler_wrapper`

Wrap a typed handler function so it can be called with the standard
SWAIG calling convention (args_dict, raw_data).

The wrapper unpacks the args dict into keyword arguments for the
original function.

## Signature

```python
create_typed_handler_wrapper(func, has_raw_data: bool)
```

## Parameters

<ParamField path="func" required>
  The original typed handler function.
</ParamField>
<ParamField path="has_raw_data" type="bool" required>
  If True, pass raw_data as a keyword argument.
</ParamField>

## Returns

A wrapper function with signature (args, raw_data).
