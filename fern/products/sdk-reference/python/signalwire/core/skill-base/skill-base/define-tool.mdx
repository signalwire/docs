# `define_tool`

Wrapper method that automatically includes swaig\_fields when defining tools.

This method delegates to self.agent.define\_tool() but automatically merges
any swaig\_fields configured for this skill. Skills should use this method
instead of calling self.agent.define\_tool() directly.
