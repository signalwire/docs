# `add_language`

Add a language configuration to support multilingual conversations

# Simple voice name

agent.add\_language("English", "en-US", "en-US-Neural2-F")

# Explicit parameters

agent.add\_language("English", "en-US", "josh", engine="elevenlabs", model="eleven\_turbo\_v2\_5")

# Combined format

agent.add\_language("English", "en-US", "elevenlabs.josh:eleven\_turbo\_v2\_5")

# Per-language params (engine-specific knobs)

agent.add\_language("English", "en-US", "josh",
engine="elevenlabs",
params=\{"stability": 0.5, "similarity\_boost": 0.75\})

## Parameters

<ParamField path="name" type="str" required>
  Name of the language (e.g., "English", "French")
</ParamField>

<ParamField path="code" type="str" required>
  Language code (e.g., "en-US", "fr-FR")
</ParamField>

<ParamField path="voice" type="str" required>
  TTS voice to use. Can be a simple name (e.g., "en-US-Neural2-F") or a combined format "engine.voice:model" (e.g., "elevenlabs.josh:eleven\_turbo\_v2\_5")
</ParamField>

<ParamField path="speech_fillers" type="Optional[List[str]]" default="None">
  Optional list of filler phrases for natural speech
</ParamField>

<ParamField path="function_fillers" type="Optional[List[str]]" default="None">
  Optional list of filler phrases during function calls
</ParamField>

<ParamField path="engine" type="Optional[str]" default="None">
  Optional explicit engine name (e.g., "elevenlabs", "rime")
</ParamField>

<ParamField path="model" type="Optional[str]" default="None">
  Optional explicit model name (e.g., "eleven\_turbo\_v2\_5", "arcana")
</ParamField>

<ParamField path="params" type="Optional[Dict[str, Any]]" default="None">
  Optional per-language params dict (engine-specific tuning, voice settings, etc.). Emitted as the language object's `params` key in SWML.
</ParamField>

## Returns

`AgentBase` — Self for method chaining

## Examples

```python
# Explicit parameters
agent.add_language("English", "en-US", "josh", engine="elevenlabs", model="eleven_turbo_v2_5")

# Combined format
agent.add_language("English", "en-US", "elevenlabs.josh:eleven_turbo_v2_5")

# Per-language params (engine-specific knobs)

                   engine="elevenlabs",
```
