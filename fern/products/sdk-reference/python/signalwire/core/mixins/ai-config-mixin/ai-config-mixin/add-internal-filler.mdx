# `add_internal_filler`

Add internal fillers for a single internal function and language.

See set_internal_fillers() for the complete list of supported
function_name values and an explanation of what fillers do.

## Signature

```python
add_internal_filler(
    function_name: str,
    language_code: str,
    fillers: List[str]
) -> AgentBase
```

## Parameters

<ParamField path="function_name" type="str" required>
  One of the supported internal function names
(see SUPPORTED_INTERNAL_FILLER_NAMES). Common values:
'next_step', 'change_context', 'check_time', 'wait_for_user',
'hangup'. Names outside the supported set log a warning and
are ignored by the runtime.
</ParamField>
<ParamField path="language_code" type="str" required>
  Language code (e.g. 'en-US', 'es', 'fr').
</ParamField>
<ParamField path="fillers" type="List[str]" required>
  List of filler phrases for this function and language.
</ParamField>

## Returns

`AgentBase` — Self for method chaining.
