# `add_mcp_server`

Add an external MCP server for tool discovery and invocation.

Tools are discovered via the MCP protocol at session start and
registered as SWAIG functions. Resources are optionally fetched
into global\_data.

## Signature

```python
add_mcp_server(
    url: str,
    headers: Optional[Dict[str, str]] = None,
    resources: bool = False,
    resource_vars: Optional[Dict[str, str]] = None
) -> AgentBase
```

## Parameters

<ParamField path="url" type="str" required>
  MCP server HTTP endpoint URL
</ParamField>

<ParamField path="headers" type="Optional[Dict[str, str]]" default="None">
  Optional HTTP headers (e.g. \{"Authorization": "Bearer sk-xxx"\})
</ParamField>

<ParamField path="resources" type="bool" default="False">
  Whether to fetch resources into global\_data
