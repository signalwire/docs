  Optional SignalWire Signing Key (from Dashboard → API Credentials). When set, webhook signature validation is enforced on POST /, /swaig, /post\_prompt — unsigned or invalidly-signed requests get a 403. Falls back to the SIGNALWIRE\_SIGNING\_KEY env var if not passed. See porting-sdk/webhooks.md for the contract.
