# `expression`

Add an expression pattern for pattern-based responses

## Signature

```python
expression(
    test_value: str,
    pattern: Union[str, Pattern],
    output: FunctionResult,
    nomatch_output: Optional[FunctionResult] = None
) -> DataMap
```

## Parameters

<ParamField path="test_value" type="str" required>
  Template string to test (e.g., "$\{args.command\}")
</ParamField>

<ParamField path="pattern" type="Union[str, Pattern]" required>
  Regex pattern string or compiled Pattern object to match against
</ParamField>

<ParamField path="output" type="FunctionResult" required>
  FunctionResult to return when pattern matches
</ParamField>

<ParamField path="nomatch_output" type="Optional[FunctionResult]" default="None">
  Optional FunctionResult to return when pattern doesn't match
</ParamField>

`DataMap` — Self for method chaining
