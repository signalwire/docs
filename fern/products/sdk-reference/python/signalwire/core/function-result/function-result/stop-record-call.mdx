# `stop_record_call`

Stop an active background call recording using SWML.

This is a virtual helper that generates SWML to stop a recording that 
was started with record_call().

## Signature

```python
stop_record_call(control_id: Optional[str] = None) -> FunctionResult
```

## Parameters

<ParamField path="control_id" type="Optional[str]" default="None">
  Identifier for the recording to stop. If not provided,
       the most recent recording will be stopped.
</ParamField>
