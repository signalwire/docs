# `rpc_dial`

Dial out to a number with a destination SWML URL using execute\_rpc.

This is commonly used in call screening scenarios where you place a caller
on hold and dial out to a human, with the dest\_swml specifying what agent
handles the outbound leg.

> \[!NOTE]
> result = (
> FunctionResult("Please hold while I connect you.")
> .hold(timeout=120)
> .rpc\_dial(
> to\_number="+15551234567",
> from\_number="+15559876543",
> dest\_swml="https://example.com/call-agent?caller=John"
> )
> )

## Signature

```python
rpc_dial(
    to_number: str,
    from_number: str,
    dest_swml: str,
    device_type: str = 'phone'
) -> FunctionResult
```

## Parameters

<ParamField path="to_number" type="str" required>
  Phone number to dial (E.164 format)
</ParamField>

<ParamField path="from_number" type="str" required>
  Caller ID to use (E.164 format)
</ParamField>

<ParamField path="dest_swml" type="str" required>
  URL to the SWML that handles the outbound call
</ParamField>

<ParamField path="device_type" type="str" default="'phone'">
  Device type, typically "phone" (default: "phone")
</ParamField>
