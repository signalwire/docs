# `replace_in_history`

After first send, replace tool_call+result pair in conversation history.

## Signature

```python
replace_in_history(text: Union[str, bool] = True) -> FunctionResult
```

## Parameters

<ParamField path="text" type="Union[str, bool]" default="True">
  String = replace tool_call with an assistant message containing this text.
  True = remove the tool_call+result pair from history entirely.
