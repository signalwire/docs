# `rpc_ai_unhold`

Unhold another call using execute\_rpc.

This releases a call from hold state, typically used after injecting
a message to the held caller's AI agent.

> \[!NOTE]
> result = (
> FunctionResult("Understood, I'll let them know.")
> .rpc\_ai\_message(call\_id, "No one is available. Please take a message.")
> .rpc\_ai\_unhold(call\_id)
> )
