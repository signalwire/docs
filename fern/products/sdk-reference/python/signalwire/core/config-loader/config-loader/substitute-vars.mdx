# `substitute_vars`

Recursively substitute environment variables in configuration values.

Supports $\{VAR|default\} syntax where:

- VAR is the environment variable name
- default is the fallback value if VAR is not set

## Signature

```python
substitute_vars(value: Any, max_depth: int = 10) -> Any
```

## Parameters

<ParamField path="value" type="Any" required>
  The value to process (can be string, dict, list, etc.)
</ParamField>

<ParamField path="max_depth" type="int" default="10">
  Maximum recursion depth to prevent infinite loops
</ParamField>

## Returns

`Any` — The value with all environment variables substituted
