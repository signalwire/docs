# `Section`

[SignalWire::POM::Section](SignalWire::POM::Section) is a Perl port of `signalwire.pom.pom.Section` from the Python SignalWire SDK. Both implementations render byte-for-byte identical Markdown/XML/JSON output so prompts authored in either language can be consumed interchangeably.

**Synopsis**

```perl
use SignalWire::POM::Section;
```

```perl
my $section = SignalWire::POM::Section->new(
    title   => 'Greeting',
    body    => 'You are a helpful assistant.',
    bullets => ['Be polite', 'Be concise'],
);
```

```perl
my $sub = $section->add_subsection(
    title => 'Tone',
    body  => 'Speak warmly.',
);
```

```perl
print $section->render_markdown;
print $section->render_xml;
```
