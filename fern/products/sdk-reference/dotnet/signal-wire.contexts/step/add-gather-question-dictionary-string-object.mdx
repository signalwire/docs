# `AddGatherQuestion(Dictionary<string, object>)`

Add a question to this step's gather_info. Initializes gather_info
if not yet set.

&lt;p>&lt;b>IMPORTANT — gather mode locks function access:&lt;/b> While
the model is asking gather questions, the runtime forcibly
deactivates ALL of the step's other functions. The only callable
tools during a gather question are:&lt;/p>

&lt;ul>&lt;li>&lt;code>gather_submit&lt;/code> (the native answer-submission tool)&lt;/li>&lt;li>Whatever names you pass in this question's &lt;code>"functions"&lt;/code>
option&lt;/li>&lt;/ul>

&lt;p>&lt;code>next_step&lt;/code> and &lt;code>change_context&lt;/code> are also filtered
out — the model cannot navigate away until the gather completes.
This is by design: it forces a tight ask → submit → next-question
loop.&lt;/p>

&lt;p>If a question needs to call out to a tool (e.g. validate an
email, geocode a ZIP), list that tool name in this question's
&lt;code>"functions"&lt;/code> option. Functions listed here are active ONLY
for this question.&lt;/p>
