# `Parameter(string, string, string, bool, List<string>?)`

Add a parameter to this data-map tool — the &lt;code class="paramref">description&lt;/code>
is LLM-FACING.

&lt;p>Each parameter description is rendered into the OpenAI
tool schema under &lt;code>parameters.properties.&lt;name&gt;.description&lt;/code>
and sent to the model. The model uses it to decide HOW to fill
in the argument from user speech. It is prompt engineering, not
developer FYI.&lt;/p>

&lt;p>&lt;b>Bad vs good:&lt;/b>&lt;/p>
&lt;pre>&lt;code class="lang-csharp">BAD : .Parameter("city", "string", "the city")
GOOD: .Parameter("city", "string",
          "The name of the city to get weather for, e.g. "
          + "'San Francisco'. Ask the user if they did not "
          + "provide one. Include the state or country if the "
          + "city name is ambiguous.")&lt;/code>&lt;/pre>
