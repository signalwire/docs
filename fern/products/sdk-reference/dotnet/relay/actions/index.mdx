# `actions`

## Classes

<CardGroup cols={2}>
  <Card title="Action" href="/docs/sdk-reference/reference/dotnet/relay/actions/action">
    Base class for all RELAY call actions (play, record, collect, etc.). An Action is the client-side handle returned when you start an asynchronous operation on a call. It accumulates events, tracks state, and resolves once the operation reaches a terminal state. Uses &lt;xref href="System.Threading.Tasks.TaskCompletionSource" data-throw-if-not-resolved="false">&lt;/xref> for native async/await support.
