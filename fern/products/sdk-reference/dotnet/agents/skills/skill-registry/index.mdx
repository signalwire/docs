# `SkillRegistry`

Thread-safe singleton that maps snake\_case skill names to factory delegates.
All 18 built-in skills are registered lazily on first access.

**Modifiers:** `sealed`

### AddSkillDirectory(string)

Add a directory to the external skill-source path list.
.NET ports loading skills from disk SHOULD consult this list.
Throws when the path does not exist or is not a directory.
(Python parity: `SkillRegistry.add_skill_directory(path)`.)

### DiscoverSkills()

Discover and return all available skills.
Skills resolve on-demand, so there is nothing to eagerly register;
this returns the discoverable inventory (mirrors &lt;xref href="SignalWire.Skills.SkillRegistry.ListSkills" data-throw-if-not-resolved="false">&lt;/xref>).
(Python parity: `SkillRegistry.discover_skills` now returns
`list_skills()` — it was a no-op until the reference stub was fixed.)
