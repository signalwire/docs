# `DefineContexts()`

Return the ContextBuilder, creating it lazily on first access.
The builder is wired to report registered SWAIG tool names back
so its &lt;xref href="SignalWire.Contexts.ContextBuilder.Validate" data-throw-if-not-resolved="false">&lt;/xref> can check for
collisions with reserved native tool names (&lt;code>next\_step&lt;/code>, &lt;code>change\_context&lt;/code>, &lt;code>gather\_submit&lt;/code>).
