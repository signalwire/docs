# `AddLanguage(string, string, string, Dictionary<string, object?>?)`

Add a language configuration with optional per-language engine-specific
params (e.g. voice stability/similarity for ElevenLabs, model knobs).
The &lt;code>params&lt;/code> key is only emitted into SWML when non-empty, so
existing language entries stay byte-identical when no params are passed.
Mirrors signalwire-python's &lt;code>AIConfigMixin.add_language(params=...)&lt;/code>.

## Signature

```dotnet
public AgentBase AddLanguage(string name, string code, string voice, Dictionary<string, object?>? languageParams)
```

  Language name (e.g. "English").

<ParamField path="code" type="string" required>

</ParamField>
<ParamField path="voice" type="string" required>

</ParamField>
<ParamField path="languageParams" type="Dictionary<string,object>" required>
  Optional engine-specific params dict.
    &lt;code>null&lt;/code> or empty omits the SWML &lt;code>params&lt;/code> key.
</ParamField>
