# `AgentBase`

AI agent built on &lt;xref href="SignalWire.SWML.Service" data-throw-if-not-resolved="false">&lt;/xref>. Provides prompt management, SWAIG tool dispatch,
context switching, skill stubs, and a 5-phase SWML rendering pipeline.
All configuration methods return &lt;code>this&lt;/code> for fluent chaining.

## Signature

```dotnet
public class AgentBase : Service
```
