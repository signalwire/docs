# `DatasphereSkill`

SignalWire DataSphere knowledge-base search skill.

Mirrors signalwire-python's &lt;code>signalwire.skills.datasphere.skill&lt;/code>.
POSTs a JSON body to &lt;code>https://\<space\_name>.signalwire.com
/api/datasphere/documents/search&lt;/code> with HTTP Basic auth
(project\_id : token). Real DataSphere returns matches under &lt;code>chunks&lt;/code>; the porting-sdk audit fixture uses &lt;code>results&lt;/code>;
accept either so the skill round-trips against the live API and the
offline audit alike (matching the precedent already in Java/PHP/Perl/
Rust).

Upstream URL override: &lt;code>DATASPHERE\_BASE\_URL&lt;/code>. Path &lt;code>/api/datasphere/documents/search&lt;/code> is preserved.
