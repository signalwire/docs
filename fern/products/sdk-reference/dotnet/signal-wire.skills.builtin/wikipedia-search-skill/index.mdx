# `WikipediaSearchSkill`

Wikipedia search skill backed by the public Wikipedia REST API.

Mirrors signalwire-python's &lt;code>signalwire.skills.wikipedia\_search.skill&lt;/code>.
The Python skill makes two API calls per query — first &lt;code>action=query\&list=search&lt;/code> to find article titles, then &lt;code>action=query\&prop=extracts&lt;/code> to fetch each article's intro
extract. Returns the article(s) as &lt;code>**Title**\n\nExtract&lt;/code> joined
by separators.

Upstream URL override: &lt;code>WIKIPEDIA\_BASE\_URL&lt;/code>. Path &lt;code>/w/api.php&lt;/code> is preserved when the env var is set so the audit
fixture sees the documented Wikipedia API path on the wire.
