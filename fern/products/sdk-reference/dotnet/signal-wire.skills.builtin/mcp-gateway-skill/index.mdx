# `McpGatewaySkill`

Bridge MCP (Model Context Protocol) servers with SWAIG functions.

Mirrors signalwire-python's &lt;code>signalwire.skills.mcp\_gateway.skill&lt;/code>.
Each configured service gets one or more SWAIG tools whose handler
POSTs to the gateway with the tool name, arguments, and per-call
session id (derived from &lt;code>raw\_data.global\_data.mcp\_call\_id&lt;/code> if
present, else &lt;code>raw\_data.call\_id&lt;/code>).

Auth: &lt;code>auth\_token&lt;/code> (Bearer) or &lt;code>auth\_user&lt;/code>/&lt;code>auth\_password&lt;/code>
(Basic). The skill prefers Bearer when both are configured.

Endpoint shape: &lt;code>POST \<gateway\_url>/services/\<service>/call&lt;/code>
per the Python implementation. Audit fixtures point &lt;code>gateway\_url&lt;/code>
at a loopback URL via the per-skill convention.

**Modifiers:** `sealed`
