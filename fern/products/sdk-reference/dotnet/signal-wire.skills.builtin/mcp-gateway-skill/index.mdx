# `McpGatewaySkill`

Bridge MCP (Model Context Protocol) servers with SWAIG functions.

Mirrors signalwire-python's &lt;code>signalwire.skills.mcp_gateway.skill&lt;/code>.
Each configured service gets one or more SWAIG tools whose handler
POSTs to the gateway with the tool name, arguments, and per-call
session id (derived from &lt;code>raw_data.global_data.mcp_call_id&lt;/code> if
present, else &lt;code>raw_data.call_id&lt;/code>).

Auth: &lt;code>auth_token&lt;/code> (Bearer) or &lt;code>auth_user&lt;/code>/&lt;code>auth_password&lt;/code>
(Basic). The skill prefers Bearer when both are configured.

Endpoint shape: &lt;code>POST &lt;gateway_url&gt;/services/&lt;service&gt;/call&lt;/code>
per the Python implementation. Audit fixtures point &lt;code>gateway_url&lt;/code>
at a loopback URL via the per-skill convention.
