# `RunAsync()`

Main event loop -- drains the inbound queue and processes messages
until disconnect. Used by the test path that pushes JSON strings
into <xref href="SignalWire.Relay.Client.InboundQueue" data-throw-if-not-resolved="false"></xref>; production reads come from the
WebSocket reader started in <xref href="SignalWire.Relay.Client.ConnectAsync" data-throw-if-not-resolved="false"></xref>.
