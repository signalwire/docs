# `DialAsync(Dictionary<string, object?>)`

Originate an outbound call, awaiting until the dial resolves.
Honours &lt;code>params\_\["tag"]&lt;/code> when provided; otherwise a UUID is
generated. Honours &lt;code>params\_\["dial\_timeout"]&lt;/code> (seconds) for the
resolve-or-throw deadline.
