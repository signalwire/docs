# `ReconstructUrl(Dictionary<string, string>, string, string?, int)`

Reconstruct the public URL SignalWire POSTed to. Resolution order:

&lt;ol>&lt;li>&lt;code>SWML_PROXY_URL_BASE&lt;/code> env var (joined with path + query).&lt;/li>&lt;li>&lt;code>X-Forwarded-Proto&lt;/code> / &lt;code>X-Forwarded-Host&lt;/code> headers
    when &lt;code>trustProxy=true&lt;/code>.&lt;/li>&lt;li>&lt;code>http://hostFallback:portFallback{path}&lt;/code> as a last-resort
    local construction.&lt;/li>&lt;/ol>
