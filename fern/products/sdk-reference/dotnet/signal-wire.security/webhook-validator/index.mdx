# `WebhookValidator`

Validates SignalWire webhook signatures for both Scheme A (RELAY/JSON,
hex HMAC-SHA1 over <code>url + rawBody</code>) and Scheme B (Compat/cXML form,
base64 HMAC-SHA1 over <code>url + sortedFormParams</code>) per <code>porting-sdk/webhooks.md</code>. The contract is byte-identical across all
SignalWire SDK ports — see the cross-port test vectors in the spec.

**Modifiers:** `static`
