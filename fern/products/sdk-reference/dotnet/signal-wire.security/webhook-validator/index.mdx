# `WebhookValidator`

Validates SignalWire webhook signatures for both Scheme A (RELAY/JSON,
hex HMAC-SHA1 over &lt;code>url + rawBody&lt;/code>) and Scheme B (Compat/cXML form,
base64 HMAC-SHA1 over &lt;code>url + sortedFormParams&lt;/code>) per
&lt;code>porting-sdk/webhooks.md&lt;/code>. The contract is byte-identical across all
SignalWire SDK ports — see the cross-port test vectors in the spec.
