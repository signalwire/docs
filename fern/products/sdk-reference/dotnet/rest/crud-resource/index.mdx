# `CrudResource`

Generic CRUD wrapper around an &lt;xref href="SignalWire.REST.HttpClient" data-throw-if-not-resolved="false">&lt;/xref> and a base API path.

Provides List / Create / Get / Update / Delete for any REST resource that
follows the standard SignalWire collection+item URL pattern.
