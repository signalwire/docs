# `HttpClient`

Low-level HTTP client for SignalWire REST APIs.

Uses &lt;xref href="System.Net.Http.HttpClient" data-throw-if-not-resolved="false">&lt;/xref> with Basic Auth,
and returns parsed JSON responses as dictionaries.

### Dispose

Release the underlying &lt;xref href="System.Net.Http.HttpClient" data-throw-if-not-resolved="false">&lt;/xref> — but
ONLY when this object created it (&lt;code>\_ownsHttp&lt;/code>). A caller-injected
HttpClient is left untouched: its lifetime belongs to whoever passed it
in. Idempotent.
