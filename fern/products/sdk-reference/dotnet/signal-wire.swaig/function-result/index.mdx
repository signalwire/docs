### ExecuteRpc(string, Dictionary\<string, object>?, string?, string?)

Execute an RPC method on a call using SWML.

**Remarks:** Parity with the Python reference &lt;code>execute\_rpc(method, params=None, call\_id=None, node\_id=None)&lt;/code>: the
rpc params dict is keyed &lt;code>\{method, call\_id?, node\_id?, params?\}&lt;/code> — &lt;code>call\_id&lt;/code>/&lt;code>node\_id&lt;/code> are TOP-LEVEL siblings of &lt;code>method&lt;/code>/&lt;code>params&lt;/code>,

no &lt;code>jsonrpc&lt;/code> envelope, and method strings are bare (e.g. &lt;code>"dial"&lt;/code>,
not &lt;code>"calling.dial"&lt;/code>). &lt;code>params&lt;/code> is omitted when empty.

### JoinConference

Join an ad-hoc audio conference (RELAY + CXML calls) using SWML.
Full parity with the Python reference &lt;code>signalwire/core/function\_result.py::join\_conference&lt;/code>: the conference &lt;code class="paramref">name&lt;/code> plus 18 optional parameters, each validated to the
same closed sets / bounds as Python, and emitted under its snake\_case wire
key only when it differs from its default. When every parameter is at its
default the action value is the bare conference-name string (simple form);
otherwise it is a &lt;code>\{ "name": ..., ... \}&lt;/code> object (full form).

**Remarks:** This flat, all-string overload is the parity-bearing signature against the

### Pay(string, string, string?, string, int, int, bool, object?, int, string, string?, string, string, string, string?, string, List\<Dictionary\<string, string>>?, List\<Dictionary\<string, object>>?, string?)

Process a payment using the SWML &lt;code>pay&lt;/code> verb.

**Remarks:** Full parity with the Python reference &lt;code>pay(payment\_connector\_url, input\_method="dtmf", status\_url=None,
payment\_method="credit-card", timeout=5, max\_attempts=1, security\_code=True,
postal\_code=True, min\_postal\_code\_length=0, token\_type="reusable",
charge\_amount=None, currency="usd", language="en-US", voice="woman",
description=None, valid\_card\_types="visa mastercard amex", parameters=None,
prompts=None, ai\_response=…)&lt;/code>. The SWML document is a two-verb main
section — a leading &lt;code>\{set: \{ai\_response: …\}\}&lt;/code> followed by &lt;code>\{pay: …\}&lt;/code> — routed through &lt;xref href="SignalWire.SWAIG.FunctionResult.ExecuteSwml(System.Object%2cSystem.Boolean)" data-throw-if-not-resolved="false">&lt;/xref> under the &lt;code>SWML&lt;/code> action key (NOT a bare top-level &lt;code>pay&lt;/code>).

&lt;p>
Wire-shape details matching Python: the collection-method key is
&lt;code>input&lt;/code> (NOT &lt;code>input_method&lt;/code>); numeric and boolean fields are
emitted as STRINGS (&lt;code>"5"&lt;/code>/&lt;code>"true"&lt;/code>); &lt;code class="paramref">postalCode&lt;/code>
is a lowercased bool-string when boolean, or verbatim when a string. The
optional fields (status_url/charge_amount/description/parameters/prompts)
are emitted only when supplied.
&lt;/p>

### RecordCall

Start background call recording using SWML — canonical, full-arity
overload with the two closed-set arguments (&lt;code class="paramref">format&lt;/code> / &lt;code class="paramref">direction&lt;/code>) as the typed &lt;xref href="SignalWire.SWAIG.RecordFormat" data-throw-if-not-resolved="false">&lt;/xref> / &lt;xref href="SignalWire.SWAIG.RecordDirection" data-throw-if-not-resolved="false">&lt;/xref> enums.

**Remarks:** Full parity with the Python reference &lt;code>record\_call(control\_id, stereo, format, direction, terminators, beep,
input\_sensitivity, initial\_timeout, end\_silence\_timeout, max\_length,
status\_url)&lt;/code>, in the same parameter order. The Python reference
validates the bare-string &lt;code>format&lt;/code>/&lt;code>direction&lt;/code> against the closed
sets \{wav,mp3,mp4\} / \{speak,listen,both\}; this overload surfaces those
knowable sets as enums so a bad value is a compile error rather than a
runtime &lt;code>ValueError&lt;/code> (a same-arity bare-string overload preserves the
Python &lt;code>str&lt;/code> path — see &lt;xref href="SignalWire.SWAIG.FunctionResult.RecordCall(System.String%2cSystem.Boolean%2cSystem.String%2cSystem.String%2cSystem.String%2cSystem.Boolean%2cSystem.Double%2cSystem.Nullable%7bSystem.Double%7d%2cSystem.Nullable%7bSystem.Double%7d%2cSystem.Nullable%7bSystem.Double%7d%2cSystem.String)" data-throw-if-not-resolved="false">&lt;/xref>).
