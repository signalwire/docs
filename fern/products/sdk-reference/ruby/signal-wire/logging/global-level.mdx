# `global_level`

Returns the current global log level, derived from:
  1. SIGNALWIRE_LOG_MODE=off  -> :off  (suppresses everything)
  2. SIGNALWIRE_LOG_LEVEL env  -> the named level
  3. Default                   -> :info
