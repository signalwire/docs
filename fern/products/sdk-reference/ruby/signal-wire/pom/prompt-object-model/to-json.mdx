# `to_json`

Convert the model to a JSON string. Output matches Python's
``json.dumps(..., indent=2)`` byte-for-byte, with one
special case: an empty model serializes to ``"[]"`` (Ruby's
default ``JSON.pretty_generate([])`` emits ``"[\n\n]"``).
