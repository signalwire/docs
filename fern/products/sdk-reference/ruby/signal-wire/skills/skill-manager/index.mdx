# `SkillManager`

Thread-safe lifecycle manager for loaded skill instances.

manager = SkillManager.new
manager.load('datetime', DateTimeSkill.new)
manager.get('datetime')  #=> &lt;DateTimeSkill>
manager.unload('datetime')
