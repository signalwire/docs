# `SkillRegistry`

Global registry mapping skill names to factory lambdas.

SkillRegistry.register('datetime') \{ |params| DateTimeSkill.new(params) }
  factory = SkillRegistry.get_factory('datetime')
  skill   = factory.call(\{ 'timezone' => 'UTC' })
