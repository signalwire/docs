# `get_raw_prompt`

Returns the raw prompt text whatever set\_prompt\_text stored, or
nil when no raw prompt has been set. Distinct from #get\_prompt
which may return the POM array when use\_pom is true.

Mirrors Python's PromptManager#get\_raw\_prompt.
