# `run`

Universal run method — mirrors Python's
``WebMixin.run(event=None, context=None, force_mode=None,
host=None, port=None)``.

Detects execution mode (server / lambda / cgi) and routes
accordingly. ``force_mode`` overrides auto-detection.
