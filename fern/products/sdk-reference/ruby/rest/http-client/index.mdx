### initialize

+base\_url+ overrides the derived +https://\{space\}+ value when set,
which is how the audit fixture and tests point the client at a
loopback server. Pass either +space+ ("acme") / a host
("acme.signalwire.com") OR an explicit +base\_url+ ("`http://127.0.0.1:NNNN`").

+ca\_file+ (optional) names a PEM CA bundle to trust for HTTPS, for
private-CA / pinned-CA deployments. When set, requests verify the peer
(VERIFY\_PEER) against a store seeded from the OpenSSL defaults (which
honor SSL\_CERT\_FILE) plus that bundle. When unset, Net::HTTP's default
verification (system store, VERIFY\_PEER) applies unchanged. HTTPS is
always verified either way — there is no VERIFY\_NONE path.
