# `FabricResource`

Standard fabric resource with CRUD + addresses. Inherits the
+list\_addresses+ helper from +CrudWithAddresses+ (matching Python's
+FabricResource(CrudWithAddresses)+ hierarchy); subclasses that use a
singular sub-resource path override +list\_addresses+ below.
