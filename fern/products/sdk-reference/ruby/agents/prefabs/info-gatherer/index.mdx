# `InfoGatherer`

Prefab agent for collecting answers to a series of questions.

agent = InfoGatherer.new(
questions: \[
\{ 'key\_name' => 'full\_name', 'question\_text' => 'What is your full name?' \},
\{ 'key\_name' => 'email',     'question\_text' => 'Email?', 'confirm' => true \}
]

### on\_swml\_request

(InfoGathererAgent#on\_swml\_request). In dynamic mode, invokes the
registered question callback (or a fallback) and returns a
\{ 'global\_data' => \{...\} \} Hash that AgentBase merges into the SWML
response. In static mode this is a no-op (returns nil).
