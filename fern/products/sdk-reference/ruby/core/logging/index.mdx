## Methods

### global\_level

Returns the current global log level, derived from:

1. SIGNALWIRE\_LOG\_MODE=off  -> :off  (suppresses everything)
2. SIGNALWIRE\_LOG\_LEVEL env  -> the named level
3. Default                   -> :info
