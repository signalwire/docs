Stop transcription.

## **Returns**

`Hash` -- Server acknowledgment.

## **Example**

```ruby {16}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    action = call.transcribe()

    # Stop transcription early
    action.stop()

client.run()
```
