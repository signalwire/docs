Adjust the playback volume.

## **Parameters**

<ParamField path="volume" type="Float" required={true} toc={true}>
  Volume adjustment in dB. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`Hash` -- Server acknowledgment.

## **Example**

```ruby {19,22}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    action = call.play(
        [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
        loop=0,
    )

    # Increase volume
    action.volume(10.0)

    # Decrease volume
    action.volume(-5.0)

client.run()
```
