Pause the recording.

## **Parameters**

<ParamField path="behavior" type="String, nil" toc={true}>
  Optional pause behavior. Controls what happens to the audio stream while paused
  (e.g., silence insertion).
</ParamField>

## **Returns**

`Hash` -- Server acknowledgment.

## **Example**

```ruby {16}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    action = call.record({"direction": "both", "format": "wav"})

    # Pause during sensitive information
    action.pause()
    # ... sensitive exchange ...
    action.resume()

client.run()
```
