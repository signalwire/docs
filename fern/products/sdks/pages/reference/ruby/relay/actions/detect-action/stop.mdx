Stop detection immediately.

## **Returns**

`Hash` -- Server acknowledgment.

## **Example**

```ruby {18}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    action = call.detect(
        {"type": "machine", "params": {"initial_timeout": 5.0}}
    )

    # Stop detection early
    action.stop()

client.run()
```
