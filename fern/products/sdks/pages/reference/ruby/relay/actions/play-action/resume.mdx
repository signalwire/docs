Resume paused playback from where it was paused.

## **Returns**

`Hash` -- Server acknowledgment.

## **Example**

```ruby {20}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    action = call.play(
        [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
        loop=0,
    )

    action.pause()
    # ... do something ...
    action.resume()

client.run()
```
