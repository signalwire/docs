Stop noise reduction on the call, restoring the original unfiltered audio.

## **Parameters**

`nil`

## **Returns**

`Hash` -- Server response confirming noise reduction has stopped.

## **Example**

```ruby {20}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    # Enable noise reduction
    call.denoise()

    action = call.play([{"type": "tts", "text": "Noise reduction is now active."}])
    action.wait()

    result = call.denoise_stop()
    puts("Noise reduction stopped: #{result}")

client.run()
```
