Leave the current room.

## **Parameters**

`nil`

## **Returns**

`Hash` -- Server response confirming the leave.

## **Example**

```ruby {18}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    # Join a room
    call.join_room("meeting-room-1")

    # Later, leave the room
    call.leave_room()
    call.hangup()

client.run()
```
