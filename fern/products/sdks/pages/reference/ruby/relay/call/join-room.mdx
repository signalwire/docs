[leave-room]: /docs/sdks/reference/ruby/relay/call/leave-room

Join a video/audio room. Rooms provide multi-party communication with
additional features beyond conferences, including video support.

<Info>
Use [`leave_room()`][leave-room] to leave the room.
</Info>

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  Room name to join.
</ParamField>

<ParamField path="status_url" type="String, nil" toc={true}>
  URL to receive room status webhooks.
</ParamField>

## **Returns**

`Hash` -- Server response confirming the join.

## **Example**

```ruby {15}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    # Join a room
    call.join_room("meeting-room-1")

    # The call is now in the room until leave_room() or hangup

client.run()
```
