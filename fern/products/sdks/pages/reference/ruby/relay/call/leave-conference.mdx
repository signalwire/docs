Leave an audio conference.

## **Parameters**

<ParamField path="conference_id" type="String" required={true} toc={true}>
  The conference ID to leave.
</ParamField>

## **Returns**

`Hash` -- Server response confirming the leave.

## **Example**

```ruby {18}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    # Join a conference
    result = call.join_conference("team-standup", beep: "onEnter")

    # Later, leave the conference
    call.leave_conference(result["conference_id"])

client.run()
```
