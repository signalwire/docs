Send a custom user-defined event on the call. User events allow you to pass
application-specific data through the RELAY event system. Other listeners on
the same call can receive and react to these events.

## **Parameters**

<ParamField path="event" type="String, nil" toc={true}>
  The event name or identifier.
</ParamField>

Additional keyword arguments are forwarded as event parameters.

## **Returns**

`Hash` -- Server response confirming the event was sent.

## **Example**

```ruby {15}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    # Send a custom event with metadata
    result = call.user_event(
        event="customer_identified",
        customer_id="cust-12345",
        tier="premium",
    )
    puts("User event sent: #{result}")

client.run()
```
