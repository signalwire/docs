Wait for the call to reach the `ended` state. This is a convenience method
equivalent to waiting for a `calling.call.state` event with `call_state == "ended"`.

## **Parameters**

<ParamField path="timeout" type="Float, nil" toc={true}>
  Maximum seconds to wait. Raises `Timeout::Error` if exceeded. `nil`
  waits indefinitely.
</ParamField>

## **Returns**

[`RelayEvent`][relayevent] -- The state-change event indicating the call has ended.

## **Example**

```ruby {18}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    action = call.play([{"type": "tts", "text": "Goodbye!"}])
    action.wait()
    call.hangup()

    event = call.wait_for_ended()
    puts("Call #{call.call_id} has ended: #{event.params}")

client.run()
```
