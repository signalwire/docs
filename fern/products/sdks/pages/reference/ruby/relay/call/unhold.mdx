Release the call from hold, resuming normal audio between the parties.

<Info>
This method emits [`calling.call.hold`][calling-call-hold] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

`nil`

## **Returns**

`Hash` -- Server response confirming the unhold operation.

## **Example**

```ruby {21}
# import asyncio
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    call.play([{"type": "tts", "text": "Please hold while I look that up."}])

    # Put the call on hold
    call.hold()

    # Do some processing...
    asyncio.sleep(5)

    result = call.unhold()
    puts("Call resumed from hold: #{result}")
    call.play([{"type": "tts", "text": "Thanks for holding. I have your answer."}])

client.run()
```
