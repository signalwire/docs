Start a payment collection session on the call. Collects credit card or other
payment information from the caller via DTMF. Returns a
[`PayAction`][payaction] that you can use to stop
the payment flow or wait for it to complete.

<Info>
This method emits [`calling.call.pay`][calling-call-pay] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="payment_connector_url" type="String" required={true} toc={true}>
  URL of the payment connector service that processes the payment.
</ParamField>

<ParamField path="control_id" type="String, nil" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="input_method" type="String, nil" toc={true}>
  How the caller provides payment info.

  - `"dtmf"` -- caller enters digits on the keypad
  - `"speech"` -- caller speaks the payment information
</ParamField>

<ParamField path="status_url" type="String, nil" toc={true}>
  URL to receive payment status webhooks.
</ParamField>

<ParamField path="payment_method" type="String, nil" toc={true}>
  Payment method type. Valid values: `"credit-card"`.
</ParamField>

<ParamField path="timeout" type="String, nil" toc={true}>
  Timeout for the payment session.
</ParamField>

<ParamField path="max_attempts" type="String, nil" toc={true}>
  Maximum number of input attempts before failing.
</ParamField>

<ParamField path="security_code" type="String, nil" toc={true}>
  Whether to collect CVV.

  - `"true"` -- prompt the caller for the security code
  - `"false"` -- skip security code collection
</ParamField>

<ParamField path="postal_code" type="String, nil" toc={true}>
  Whether to collect postal code.

  - `"true"` -- prompt the caller for the postal code
  - `"false"` -- skip postal code collection
</ParamField>

<ParamField path="min_postal_code_length" type="String, nil" toc={true}>
  Minimum length for the postal code.
</ParamField>

<ParamField path="token_type" type="String, nil" toc={true}>
  Tokenization type for the payment data.
</ParamField>

<ParamField path="charge_amount" type="String, nil" toc={true}>
  Amount to charge (e.g., `"29.99"`).
</ParamField>

<ParamField path="currency" type="String, nil" toc={true}>
  Currency code (e.g., `"USD"`, `"EUR"`).
</ParamField>

<ParamField path="language" type="String, nil" toc={true}>
  Language for payment prompts (e.g., `"en"`).
</ParamField>

<ParamField path="voice" type="String, nil" toc={true}>
  Voice for TTS prompts during the payment flow.
</ParamField>

<ParamField path="description" type="String, nil" toc={true}>
  Description of the payment/charge.
</ParamField>

<ParamField path="valid_card_types" type="String, nil" toc={true}>
  Comma-separated list of accepted card types (e.g., `"visa,mastercard,amex"`).
</ParamField>

<ParamField path="parameters" type="Array<dict>, nil" toc={true}>
  Additional parameters to pass to the payment connector.
</ParamField>

<ParamField path="prompts" type="Array<dict>, nil" toc={true}>
  Custom prompts for the payment flow.
</ParamField>

<ParamField path="on_completed" type="Proc, nil" toc={true}>
  Callback invoked when the payment operation completes.
</ParamField>

## **Returns**

[`PayAction`][payaction] -- An action handle with
`stop()` and `wait()` methods.

## **Example**

```ruby {15}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    call.play([{"type": "tts", "text": "We will now collect your payment."}])

    action = call.pay(
        payment_connector_url="https://example.com/payment-connector",
        charge_amount="29.99",
        currency="USD",
        security_code="true",
        postal_code="true",
    )
    event = action.wait()

    state = event.params.fetch("state", "")
    if state == "finished"
        call.play([{"type": "tts", "text": "Payment processed successfully."}])
    else
        call.play([{"type": "tts", "text": "Payment failed. Please try again later."}])

    call.hangup()

client.run()
```
