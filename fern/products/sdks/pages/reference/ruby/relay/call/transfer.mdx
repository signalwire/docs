Transfer control of the call to another RELAY application or SWML script.
The current application loses control of the call after a successful transfer.

## **Parameters**

<ParamField path="dest" type="String" required={true} toc={true}>
  The transfer destination. This can be a RELAY context name or a URL pointing
  to a SWML script.
</ParamField>

Additional keyword arguments are forwarded to the RELAY `calling.transfer` request.

## **Returns**

`Hash` -- Server response confirming the transfer.

## **Example**

```ruby {16}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    action = call.play([{"type": "tts", "text": "Transferring you now..."}])
    action.wait()

    result = call.transfer("support-queue")
    puts("Transfer result: #{result}")

client.run()
```
