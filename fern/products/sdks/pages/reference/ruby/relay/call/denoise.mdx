Start noise reduction on the call. Filters background noise from the audio
to improve clarity.

<Info>
Use [`denoise_stop()`][denoise-stop] to disable noise reduction.
</Info>

<Info>
This method emits [`calling.call.denoise`][calling-call-denoise] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

`nil`

## **Returns**

`Hash` -- Server response confirming noise reduction has started.

## **Example**

```ruby {14}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    result = call.denoise()
    puts("Noise reduction started: #{result}")

    action = call.play([{"type": "tts", "text": "Noise reduction is now active."}])
    action.wait()

client.run()
```
