Put the call on hold. The remote party hears hold music or silence while the
call is held.

<Info>
Use [`unhold()`][unhold] to release the call from hold.
</Info>

<Info>
This method emits [`calling.call.hold`][calling-call-hold] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

`nil`

## **Returns**

`Hash` -- Server response confirming the hold operation.

## **Example**

```ruby {15}
# import asyncio
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()
    call.play([{"type": "tts", "text": "Please hold while I look that up."}])

    result = call.hold()
    puts("Hold started: #{result}")

    # Do some processing...
    asyncio.sleep(5)

    # Take the call off hold
    call.unhold()

client.run()
```
