Connect the call to an Amazon Bedrock AI agent. Similar to
[`ai()`][ai] but uses Amazon Bedrock as
the LLM backend.

## **Parameters**

<ParamField path="prompt" type="Object, nil" toc={true}>
  The prompt configuration for the Bedrock agent.
</ParamField>

<ParamField path="SWAIG" type="dict, nil" toc={true}>
  SWAIG configuration for tool/function definitions.
</ParamField>

<ParamField path="ai_params" type="dict, nil" toc={true}>
  AI parameters for the Bedrock session.
</ParamField>

<ParamField path="global_data" type="dict, nil" toc={true}>
  Data accessible to the AI and SWAIG functions.
</ParamField>

<ParamField path="post_prompt" type="dict, nil" toc={true}>
  Post-prompt configuration.
</ParamField>

<ParamField path="post_prompt_url" type="String, nil" toc={true}>
  URL to receive the post-prompt result.
</ParamField>

## **Returns**

`Hash` -- Server response confirming the Bedrock session.

## **Example**

```ruby {15}
require 'signalwire/relay/client'

client = SignalWire::Relay::Client.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
def handle_call(call)
    call.answer()

    # Start an Amazon Bedrock AI agent
    result = call.amazon_bedrock(
        prompt={"text": "You are a helpful assistant."},
        ai_params={"barge_confidence": 0.02},
    )

client.run()
```
