List members currently in a queue.

## **Parameters**

<ParamField path="queue_id" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of member objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

members = client.queues.list_members("queue-id")
for m in members.fetch("data", [])
    puts(m["call_id"], m["position"])
```
