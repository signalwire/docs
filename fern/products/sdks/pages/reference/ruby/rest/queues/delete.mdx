Delete a queue.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.queues.delete("queue-id")
puts("Deleted successfully")
```
