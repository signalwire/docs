Create a new queue.

## **Parameters**

<ParamField path="name" type="String" toc={true}>
  Queue name.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional queue configuration fields (e.g., `max_size`, `fifo`).
</ParamField>

## **Returns**

`Hash` -- The newly created queue object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

queue = client.queues.create(name: "support", max_size: 50)
puts("Queue ID:", queue["id"])
```
