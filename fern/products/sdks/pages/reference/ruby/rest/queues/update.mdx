Update a queue. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`, `max_size`).
</ParamField>

## **Returns**

`Hash` -- The updated queue object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.queues.update("queue-id", name: "priority-support")
puts("Updated: #{updated['id']}")
```
