Retrieve a single SWML script resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

## **Returns**

`Hash` -- The SWML script resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.swml_scripts["resource-id"]
puts("Swml Script: #{item['name']}, ID: #{item['id']}")
```
