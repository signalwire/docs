Create a new call flow resource.

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  Display name of the call flow.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Hash` -- The created call flow resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.call_flows.create(
    name="inbound-flow",
)
puts("Created: #{result['id']}")
```
