Create a guest authentication token.

## **Parameters**

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Hash` -- The created guest token.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.fabric.tokens.create_guest_token(name: "guest-user")
puts("Guest token: #{token['token']}")
```
