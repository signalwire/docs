Delete a cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the cXML application to delete.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success (HTTP 204).

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.cxml_applications.delete("app-id")
puts("cXML application deleted")
```
