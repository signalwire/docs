List subscriber resources in the project.

## **Parameters**

<ParamField path="page_size" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing subscriber resources.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.subscribers.list()
for subscriber in response.fetch("data", [])
    puts("#{subscriber['name']}: #{subscriber['id']}")
```
