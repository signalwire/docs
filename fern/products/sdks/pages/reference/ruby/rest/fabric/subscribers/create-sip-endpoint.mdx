Create a new SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Hash` -- The created SIP endpoint.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.subscribers.create_sip_endpoint(
    "subscriber-id", username="alice-sip", password="s3cret",
)
puts("Created SIP endpoint: #{result['id']}")
```
