Partially update an AI agent resource. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`Hash` -- The updated AI agent resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.ai_agents.update("agent-id", name: "updated-name")
puts("Updated: #{updated['name']}")
```
