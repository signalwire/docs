Retrieve a single cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

## **Returns**

`Hash` -- The cXML application resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

app = client.fabric.cxml_applications["app-id"]
puts("cXML Application: #{app['name']}, ID: #{app['id']}")
```
