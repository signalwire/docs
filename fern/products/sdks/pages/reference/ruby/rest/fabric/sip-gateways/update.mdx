Partially update a SIP gateway resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`Hash` -- The updated SIP gateway resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.sip_gateways.update("gateway-id", name: "updated-name")
puts("Updated: #{updated['name']}")
```
