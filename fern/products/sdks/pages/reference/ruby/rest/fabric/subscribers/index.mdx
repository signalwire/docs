Manage subscriber resources. Extends standard CRUD (PUT updates) with address listing
and SIP endpoint management.

Access via `client.fabric.subscribers` on a [`RestClient`][restclient] instance.

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

items = client.fabric.subscribers.list()
```
