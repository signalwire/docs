Create a new conference room resource.

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  Display name of the conference room.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Hash` -- The created conference room resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.conference_rooms.create(
    name="my-room",
)
puts("Created: #{result['id']}")
```
