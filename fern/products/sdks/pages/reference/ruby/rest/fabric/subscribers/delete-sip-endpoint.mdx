Delete a SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="String" required={true} toc={true}>
  The unique identifier of the SIP endpoint to delete.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success (HTTP 204).

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.subscribers.delete_sip_endpoint("subscriber-id", "endpoint-id")
puts("SIP endpoint deleted")
```
