Manage cXML application resources. Supports list, get, update (PUT), and delete operations.
The `create` method is not available for cXML applications and raises `NotImplementedError`.

Access via `client.fabric.cxml_applications` on a [`RestClient`][restclient] instance.

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

items = client.fabric.cxml_applications.list()
```
