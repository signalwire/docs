Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`Hash` -- The RELAY application resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.relay_applications["resource-id"]
puts("Relay Application: #{item['name']}, ID: #{item['id']}")
```
