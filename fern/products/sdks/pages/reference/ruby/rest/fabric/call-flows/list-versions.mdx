List all versions of a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<ParamField path="page_size" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing call flow versions.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.call_flows.list_versions("call-flow-id")
for version in response.fetch("data", [])
    puts("Version #{version['version']}: #{version['id']}")
```
