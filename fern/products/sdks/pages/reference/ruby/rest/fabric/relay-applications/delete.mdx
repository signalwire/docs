Delete a RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the RELAY application to delete.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success (HTTP 204).

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.relay_applications.delete("resource-id")
puts("Relay Application deleted")
```
