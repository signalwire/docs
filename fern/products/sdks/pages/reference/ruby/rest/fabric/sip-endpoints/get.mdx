Retrieve a single SIP endpoint resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`Hash` -- The SIP endpoint resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

endpoint = client.fabric.sip_endpoints["endpoint-id"]
puts("Sip Endpoint: #{endpoint['name']}, ID: #{endpoint['id']}")
```
