Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`Hash` -- The call flow resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

flow = client.fabric.call_flows["flow-id"]
puts("Call Flow: #{flow['name']}, ID: #{flow['id']}")
```
