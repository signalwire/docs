Replace a subscriber resource. Uses PUT, so the full resource body must be provided.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are the fields for the replacement resource body.

## **Returns**

`Hash` -- The updated subscriber resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.subscribers.update("subscriber-id", name: "bob")
puts("Updated: #{updated['name']}")
```
