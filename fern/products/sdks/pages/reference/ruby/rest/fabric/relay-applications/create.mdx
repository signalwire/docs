Create a new RELAY application resource.

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  Display name of the RELAY application.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Hash` -- The created RELAY application resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.relay_applications.create(
    name="my-item",
)
puts("Created: #{result['id']}")
```
