Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`Hash` -- The Fabric resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

resource = client.fabric.resources["resource-id"]
puts("#{resource['type']}: #{resource['name']}, ID: #{resource['id']}")
```
