Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`Hash` -- The SIP gateway resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

gateway = client.fabric.sip_gateways["gateway-id"]
puts("Sip Gateway: #{gateway['name']}, ID: #{gateway['id']}")
```
