Retrieve a single cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`Hash` -- The cXML webhook resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

webhook = client.fabric.cxml_webhooks["webhook-id"]
puts("Cxml Webhook: #{webhook['name']}, ID: #{webhook['id']}")
```
