Retrieve a single AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`Hash` -- The AI agent resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

agent = client.fabric.ai_agents["agent-id"]
puts("AI Agent: #{agent['name']}, ID: #{agent['id']}")
```
