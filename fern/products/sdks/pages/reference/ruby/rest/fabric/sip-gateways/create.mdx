Create a new SIP gateway resource.

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  Display name of the SIP gateway.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Hash` -- The created SIP gateway resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.sip_gateways.create(
    name="my-gateway",
)
puts("Created: #{result['id']}")
```
