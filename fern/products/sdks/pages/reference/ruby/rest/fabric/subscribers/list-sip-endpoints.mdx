List SIP endpoints associated with a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="page_size" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing SIP endpoints for the subscriber.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.subscribers.list_sip_endpoints("subscriber-id")
for ep in response.fetch("data", [])
    puts("#{ep['username']}: #{ep['id']}")
```
