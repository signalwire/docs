Retrieve a single cXML script resource.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`Hash` -- The cXML script resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.cxml_scripts["resource-id"]
puts("Cxml Script: #{item['name']}, ID: #{item['id']}")
```
