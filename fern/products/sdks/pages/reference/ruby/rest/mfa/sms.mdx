Send an MFA verification code via SMS.

## **Parameters**

<ParamField path="to" type="String" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="String" required={true} toc={true}>
  The sender phone number in E.164 format. Must be a number in your project.
  Since `from` is a reserved keyword in Ruby, pass it as a string key (see example).
</ParamField>

<ParamField path="message" type="String" required={true} toc={true}>
  The message body. Include `{code}` as a placeholder for the generated code
  (e.g., `"Your verification code is: {code}"`).
</ParamField>

<ParamField path="token_length" type="Integer" toc={true}>
  Length of the generated code.
</ParamField>

<ParamField path="max_attempts" type="Integer" toc={true}>
  Maximum number of verification attempts allowed.
</ParamField>

<ParamField path="valid_for" type="Integer" toc={true}>
  How long the code remains valid, in seconds.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`Hash` -- Response containing the `id` of the MFA request (used in `verify()`).

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.mfa.sms(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your code is: {code}",
})
request_id = result["id"]
puts("MFA request:", request_id)
```
