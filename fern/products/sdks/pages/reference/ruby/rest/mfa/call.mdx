Send an MFA verification code via voice call.

## **Parameters**

<ParamField path="to" type="String" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="String" required={true} toc={true}>
  The caller phone number in E.164 format.
  Since `from` is a reserved keyword in Ruby, pass it as a string key (see example).
</ParamField>

<ParamField path="message" type="String" required={true} toc={true}>
  The message to speak. Include `{code}` as a placeholder.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters (e.g., `token_length`, `max_attempts`, `valid_for`).
</ParamField>

## **Returns**

`Hash` -- Response containing the `id` of the MFA request.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.mfa.call(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your verification code is {code}",
})
request_id = result["id"]
```
