List phone numbers owned by the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `page_size`,
  `page_token`).
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.phone_numbers.list(page_size: 10)
for number in result.fetch("data", [])
    puts(number["number"], number["name"])
```
