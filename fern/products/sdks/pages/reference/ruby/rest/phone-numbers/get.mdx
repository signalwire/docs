Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`Hash` -- The phone number object.

## **Example**

```ruby {9-10}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

number = client.phone_numbers["phone-number-id"]
puts(number["number"], number["name"])
```
