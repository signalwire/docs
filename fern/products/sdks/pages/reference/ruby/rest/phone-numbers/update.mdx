Update configuration for a phone number. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`, `call_handler`, `call_request_url`).
</ParamField>

## **Returns**

`Hash` -- The updated phone number object.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.phone_numbers.update("phone-number-id", name: "Support Line")
```
