Search for available phone numbers that can be purchased.

## **Parameters**

<ParamField path="area_code" type="String" toc={true}>
  Filter by area code (e.g., `"512"`).
</ParamField>

<ParamField path="quantity" type="Integer" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Additional query parameters such as `region`, `city`, `country`, or
  `number_type`.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of available number objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

available = client.phone_numbers.search(area_code: "503", quantity: 5)
for number in available.fetch("data", [])
    puts(number["number"], number["region"])
```
