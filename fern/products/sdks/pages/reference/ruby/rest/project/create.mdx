### tokens.create

Create a new API token for the project.

## **Parameters**

<ParamField path="**kwargs" type="dict" toc={true}>
  Token configuration fields (e.g., `name`, `scopes`).
</ParamField>

## **Returns**

`Hash` -- The newly created token object, including the token value.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.project.tokens.create(name: "ci-token")
puts("Token:", token["token"])
puts("ID:", token["id"])
```
