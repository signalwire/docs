Update an existing document.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`, `content`, `metadata`).
</ParamField>

## **Returns**

`Hash` -- The updated document object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.datasphere.documents.update(
    "document-id",
    name="Updated FAQ",
)
puts(updated["name"])
```
