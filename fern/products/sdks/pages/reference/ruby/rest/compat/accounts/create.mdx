Create a new subproject (sub-account).

## **Parameters**

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable name for the subproject.
</ParamField>

## **Returns**

`Hash` -- The created account resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

sub = client.compat.accounts.create(FriendlyName: "Marketing Team")
```
