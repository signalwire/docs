Search for available toll-free phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="String" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="Contains" type="String" toc={true}>
  Filter by number pattern (e.g., `"800"`).
</ParamField>

<ParamField path="PageSize" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing available toll-free number resources.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

numbers = client.compat.phone_numbers.search_toll_free("US", Contains: "800")
```
