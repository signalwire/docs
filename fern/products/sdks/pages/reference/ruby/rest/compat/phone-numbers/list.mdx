List incoming phone numbers in the account.

## **Parameters**

<ParamField path="PhoneNumber" type="String" toc={true}>
  Filter by exact phone number.
</ParamField>

<ParamField path="FriendlyName" type="String" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="PageSize" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing phone number resources.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

numbers = client.compat.phone_numbers.list(PageSize: 20)
```
