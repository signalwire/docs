Update a conference (e.g., end it, or set the announce URL).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The conference SID.
</ParamField>

<ParamField path="Status" type="String" toc={true}>
  Set to `"completed"` to end the conference.
</ParamField>

<ParamField path="AnnounceUrl" type="String" toc={true}>
  URL returning LAML to play an announcement to all participants.
</ParamField>

## **Returns**

`Hash` -- The updated conference resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.conferences.update("CF...", Status: "completed")
```
