Update a message (e.g., redact the body of a sent message).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The message SID.
</ParamField>

<ParamField path="Body" type="String" toc={true}>
  Set to an empty string to redact the message body.
</ParamField>

## **Returns**

`Hash` -- The updated message resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.messages.update("SM...", Body: "")
```
