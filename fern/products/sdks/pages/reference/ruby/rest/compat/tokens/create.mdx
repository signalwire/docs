Create a new API token.

## **Parameters**

<ParamField path="Ttl" type="Integer" toc={true}>
  Time-to-live in seconds for the token.
</ParamField>

## **Returns**

`Hash` -- The created token resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.compat.tokens.create(Ttl: 3600)
```
