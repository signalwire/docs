Retrieve a specific media item from a fax.

## **Parameters**

<ParamField path="fax_sid" type="String" required={true} toc={true}>
  The SID of the fax.
</ParamField>

<ParamField path="media_sid" type="String" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`Hash` -- The media resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.compat.faxes.get_media("FX...", "ME...")
```
