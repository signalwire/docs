Update an application's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The application SID.
</ParamField>

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="String" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="SmsUrl" type="String" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="StatusCallback" type="String" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`Hash` -- The updated application resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.applications.update(
    "AP...",
    VoiceUrl="https://example.com/new-voice"
)
```
