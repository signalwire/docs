Update a queue's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The queue SID.
</ParamField>

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="Integer" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`Hash` -- The updated queue resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.queues.update("QU...", MaxSize: 200)
```
