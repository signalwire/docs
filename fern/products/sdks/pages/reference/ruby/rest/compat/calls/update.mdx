Update an active call (e.g., redirect to new LAML, end the call).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The call SID.
</ParamField>

<ParamField path="Url" type="String" toc={true}>
  New URL to fetch LAML instructions from.
</ParamField>

<ParamField path="Status" type="String" toc={true}>
  Set to `"completed"` to end the call.
</ParamField>

## **Returns**

`Hash` -- The updated call resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.calls.update("CA...", Status: "completed")
```
