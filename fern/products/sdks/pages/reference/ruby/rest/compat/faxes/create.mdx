Send a new fax.

## **Parameters**

<ParamField path="To" type="String" required={true} toc={true}>
  The destination fax number in E.164 format.
</ParamField>

<ParamField path="From" type="String" required={true} toc={true}>
  The sender number (must be a number you own).
</ParamField>

<ParamField path="MediaUrl" type="String" required={true} toc={true}>
  URL of the document to fax (PDF format).
</ParamField>

<ParamField path="StatusCallback" type="String" toc={true}>
  URL to receive fax status webhook events.
</ParamField>

## **Returns**

`Hash` -- The created fax resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

fax = client.compat.faxes.create(
    To="+15559876543",
    From="+15551234567",
    MediaUrl="https://example.com/document.pdf"
)
```
