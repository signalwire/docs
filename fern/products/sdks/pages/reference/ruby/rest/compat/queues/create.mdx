Create a new call queue.

## **Parameters**

<ParamField path="FriendlyName" type="String" required={true} toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="Integer" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`Hash` -- The created queue resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

queue = client.compat.queues.create(
    FriendlyName="Support Queue",
    MaxSize=100
)
```
