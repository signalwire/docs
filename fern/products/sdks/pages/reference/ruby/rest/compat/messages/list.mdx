List messages in the account.

## **Parameters**

<ParamField path="To" type="String" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="From" type="String" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="PageSize" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing message resources.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

messages = client.compat.messages.list(PageSize: 20)
```
