Create a new LAML bin containing a cXML/LaML script.

## **Parameters**

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="String" required={true} toc={true}>
  The cXML/LaML script content.
</ParamField>

## **Returns**

`Hash` -- The created LAML bin resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

laml_bin = client.compat.laml_bins.create(
    FriendlyName="Greeting",
    Contents=(
        '<?xml version="1.0" encoding="UTF-8"?>'
        "<Response><Say>Hello!</Say></Response>"
    )
)
```
