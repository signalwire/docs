Start a media stream on an active call, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="call_sid" type="String" required={true} toc={true}>
  The SID of the call.
</ParamField>

<ParamField path="Url" type="String" toc={true}>
  The WebSocket URL to stream audio to.
</ParamField>

## **Returns**

`Hash` -- The created stream resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

stream = client.compat.calls.start_stream("CA...", Url: "wss://stream.example.com")
```
