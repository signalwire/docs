Delete an API token.

## **Parameters**

<ParamField path="token_id" type="String" required={true} toc={true}>
  The token identifier to delete.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.tokens.delete("token-id")
```
