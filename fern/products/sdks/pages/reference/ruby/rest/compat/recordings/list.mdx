List recordings in the account.

## **Parameters**

<ParamField path="DateCreated" type="String" toc={true}>
  Filter by creation date.
</ParamField>

<ParamField path="CallSid" type="String" toc={true}>
  Filter by the call SID that generated the recording.
</ParamField>

<ParamField path="PageSize" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Hash` -- Paginated response containing recording resources.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recordings = client.compat.recordings.list(PageSize: 20)
```
