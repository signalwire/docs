Start recording an active call.

## **Parameters**

<ParamField path="call_sid" type="String" required={true} toc={true}>
  The SID of the call to record.
</ParamField>

<ParamField path="RecordingStatusCallback" type="String" toc={true}>
  URL to receive recording status events.
</ParamField>

<ParamField path="RecordingChannels" type="String" toc={true}>
  Recording channel mode.

  - `"mono"` -- both legs of the call are mixed into a single audio channel
  - `"dual"` -- each leg of the call is recorded in a separate audio channel
</ParamField>

## **Returns**

`Hash` -- The created recording resource.
