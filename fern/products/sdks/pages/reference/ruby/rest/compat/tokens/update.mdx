Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="String" required={true} toc={true}>
  The token identifier.
</ParamField>

<ParamField path="Ttl" type="Integer" toc={true}>
  New time-to-live in seconds for the token.
</ParamField>

## **Returns**

`Hash` -- The updated token resource.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.tokens.update("token-id", Ttl: 7200)
```
