List phone numbers assigned to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="String" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of number objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

numbers = client.registry.campaigns.list_numbers("campaign-id")
for n in numbers.fetch("data", [])
    puts(n["number"], n["id"])
```
