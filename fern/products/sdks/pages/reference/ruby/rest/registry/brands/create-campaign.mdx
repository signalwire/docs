Create a messaging campaign under a brand.

## **Parameters**

<ParamField path="brand_id" type="String" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="description" type="String" toc={true}>
  Campaign description.
</ParamField>

<ParamField path="usecase" type="String" toc={true}>
  Campaign use case.

  - `"MIXED"` -- multiple or general-purpose messaging use cases
  - `"MARKETING"` -- promotional or marketing messages
  - `"CUSTOMER_CARE"` -- customer support and service messages
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional campaign fields.
</ParamField>

## **Returns**

`Hash` -- The newly created campaign object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

campaign = client.registry.brands.create_campaign(
    "brand-id",
    description="Customer notifications",
    usecase="MIXED",
)
puts("Campaign ID:", campaign["id"])
```
