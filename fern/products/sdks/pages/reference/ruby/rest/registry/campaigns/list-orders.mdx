List number assignment orders for a campaign.

## **Parameters**

<ParamField path="campaign_id" type="String" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of order objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

orders = client.registry.campaigns.list_orders("campaign-id")
for order in orders.fetch("data", [])
    puts(order["id"], order["status"])
```
