Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brand_id" type="String" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`Hash` -- The brand object.

## **Example**

```ruby {9-10}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

brand = client.registry.brands["brand-id"]
puts(brand["display_name"], brand["entity_type"])
```
