List messaging campaigns under a brand.

## **Parameters**

<ParamField path="brand_id" type="String" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of campaign objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

campaigns = client.registry.brands.list_campaigns("brand-id")
for c in campaigns.fetch("data", [])
    puts(c["description"], c["usecase"])
```
