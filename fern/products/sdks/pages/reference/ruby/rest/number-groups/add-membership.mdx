Add a phone number to a group.

## **Parameters**

<ParamField path="group_id" type="String" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="phone_number_id" type="String" toc={true}>
  The phone number resource ID to add to the group.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional membership fields.
</ParamField>

## **Returns**

`Hash` -- The newly created membership object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

membership = client.number_groups.add_membership(
    "group-id",
    phone_number_id="phone-number-id",
)
puts("Membership ID:", membership["id"])
```
