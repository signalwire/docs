Update a number group. Uses PUT to replace fields.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`Hash` -- The updated group object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.number_groups.update("group-id", name: "Support Numbers")
puts(updated["name"])
```
