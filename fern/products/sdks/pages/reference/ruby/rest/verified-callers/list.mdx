List verified caller IDs in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of verified caller objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.verified_callers.list()
for caller in result.fetch("data", [])
    puts(caller["phone_number"], caller["verification_state"])
```
