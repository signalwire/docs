Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="phone_number" type="String" toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters (e.g., `friendly_name`).
</ParamField>

## **Returns**

`Hash` -- The newly created caller ID object with its verification state.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

caller = client.verified_callers.create(phone_number: "+15551234567")
puts("Verification state:", caller["verification_state"])
```
