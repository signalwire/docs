Retrieve a specific verified caller ID.

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`Hash` -- The caller ID object.

## **Example**

```ruby {9-10}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

caller = client.verified_callers["caller-id"]
puts(caller["phone_number"], caller["verification_state"])
```
