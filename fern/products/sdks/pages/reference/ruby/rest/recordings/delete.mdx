Delete a recording.

## **Parameters**

<ParamField path="recording_id" type="String" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.recordings.delete("recording-id")
```
