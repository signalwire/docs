Retrieve a specific recording.

## **Parameters**

<ParamField path="recording_id" type="String" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`Hash` -- The recording object, including download URL and metadata.

## **Example**

```ruby {9-10}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

rec = client.recordings["recording-id"]
puts(rec["duration"], rec["url"])
```
