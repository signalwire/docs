List recordings in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of recording objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.recordings.list(page_size: 20)
for rec in result.fetch("data", [])
    puts(rec["id"], rec["duration"], rec["state"])
```
