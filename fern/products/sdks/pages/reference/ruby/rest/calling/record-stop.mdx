[record]: /docs/sdks/reference/ruby/rest/calling/record

Stop an active recording. The recording file becomes available after stopping.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="String" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`Hash` — The API response containing the recording URL and metadata.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.record_stop(call_id: "call-id-xxx", control_id: "ctrl-id")
puts(result)  # Contains recording URL
```
