Stop an active detector.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="String" required={true} toc={true}>
  The control ID returned from [`detect()`][detect].
</ParamField>

## **Returns**

`Hash` — The API response confirming the detector was stopped.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.detect_stop(call_id: "call-id-xxx", control_id: "ctrl-id")
```
