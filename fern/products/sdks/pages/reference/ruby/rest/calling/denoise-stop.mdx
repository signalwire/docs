Disable noise reduction on an active call.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

## **Returns**

`Hash` — The API response confirming noise reduction was disabled.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.denoise_stop(call_id: "call-id-xxx")
```
