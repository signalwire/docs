Adjust the volume of an active playback.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="String" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

<ParamField path="volume" type="Float" required={true} toc={true}>
  Volume level adjustment. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`Hash` — The API response confirming the volume was adjusted.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.play_volume(call_id: "call-id-xxx", control_id: "ctrl-id", volume: 5.0)
```
