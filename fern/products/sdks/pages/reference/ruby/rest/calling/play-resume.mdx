Resume a paused playback.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="String" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

## **Returns**

`Hash` — The API response confirming playback was resumed.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.play_resume(call_id: "call-id-xxx", control_id: "ctrl-id")
```
