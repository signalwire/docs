Put an active AI session on hold. The AI agent stops processing speech
while the call remains connected. This is useful for transferring the
caller to a human agent or performing background operations.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

## **Returns**

`Hash` — The API response confirming the AI session was put on hold.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.ai_hold(call_id: "call-id-xxx")
```
