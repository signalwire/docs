Stop an active input collection.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="String" required={true} toc={true}>
  The control ID returned from [`collect()`][collect].
</ParamField>

## **Returns**

`Hash` — The API response confirming collection was stopped.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.collect_stop(call_id: "call-id-xxx", control_id: "ctrl-id")
```
