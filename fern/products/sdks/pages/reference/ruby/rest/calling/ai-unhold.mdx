[ai-hold]: /docs/sdks/reference/ruby/rest/calling/ai-hold

Resume an AI session that was previously put on hold with
[`ai_hold()`][ai-hold].

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call with a held AI session.
</ParamField>

## **Returns**

`Hash` — The API response confirming the AI session was resumed.

## **Example**

```ruby {14}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

# Hold, perform an action, then resume
client.calling.ai_hold(call_id: "call-id-xxx")

# ... do some background processing ...

client.calling.ai_unhold(call_id: "call-id-xxx")
```
