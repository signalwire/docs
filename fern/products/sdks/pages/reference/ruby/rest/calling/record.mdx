Start recording an active call. Returns a `control_id` used to
[pause][pause],
[resume][resume],
or [stop][stop] the recording.

## **Parameters**

<ParamField path="call_id" type="String" required={true} toc={true}>
  The ID of the call to record.
</ParamField>

<ParamField path="record" type="dict" required={true} toc={true}>
  Recording configuration object. Contains:

  - `audio` — Audio recording settings (e.g., `{"format": "mp3", "stereo": true}`)
</ParamField>

<ParamField path="listen_on" type="String" toc={true}>
  Which leg of the call to record.

  - `"self"` -- record only the audio from the call's own leg
  - `"both"` -- record audio from both legs of the call
</ParamField>

## **Returns**

`Hash` — The API response containing the `control_id` for managing the recording.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.record(
    call_id="call-id-xxx",
    record={"audio": {"format": "mp3", "stereo": true}}
)
control_id = result["control_id"]
```
