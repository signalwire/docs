Update a short code's configuration. Uses PUT.

## **Parameters**

<ParamField path="short_code_id" type="String" required={true} toc={true}>
  The short code resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `message_handler`, `message_request_url`).
</ParamField>

## **Returns**

`Hash` -- The updated short code object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.short_codes.update(
    "short-code-id",
    message_request_url="https://example.com/sms",
)
```
