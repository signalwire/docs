List voice call log entries.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of voice log objects.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

calls = client.logs.voice.list(page_size: 20)
for call in calls.fetch("data", [])
    puts(call["from"], "->", call["to"], call["duration"])
```
