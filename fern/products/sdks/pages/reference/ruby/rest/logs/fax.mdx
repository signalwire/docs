Query fax log entries. Fax logs provide a read-only view of fax activity
in your project.

Access via `client.logs.fax` on a [`RestClient`][restclient] instance.

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new()
```

### list

List fax log entries.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Hash` -- JSON response containing a `data` list of fax log objects.

---

### get

Retrieve a specific fax log entry.

## **Parameters**

<ParamField path="log_id" type="String" required={true} toc={true}>
  The fax log ID.
</ParamField>

## **Returns**

`Hash` -- The fax log object.

## **Examples**

### List fax logs

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

faxes = client.logs.fax.list()
for fax in faxes.fetch("data", [])
    puts(fax["from"], "->", fax["to"], fax["status"])
```

### Get a specific fax log

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

fax = client.logs.fax["fax-log-id"]
puts(fax["from"], "->", fax["to"])
```
