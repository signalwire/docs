Import externally-hosted phone numbers into your SignalWire project. This allows
you to route calls through SignalWire for numbers hosted by another carrier. Only
the create operation is supported.

Access via `client.imported_numbers` on a [`RestClient`][restclient] instance.

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new()
```

---

### create

Import a phone number from an external provider.

## **Parameters**

<ParamField path="number" type="String" toc={true}>
  The phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional import parameters (e.g., `name`, `call_handler`,
  `call_request_url`).
</ParamField>

## **Returns**

`Hash` -- The imported phone number object.

## **Example**

```ruby
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

imported = client.imported_numbers.create(
    number="+15551234567",
    name="External Support Line",
)
puts("Imported:", imported["id"])
```
