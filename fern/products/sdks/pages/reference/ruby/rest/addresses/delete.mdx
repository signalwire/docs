Delete an address from the project.

## **Parameters**

<ParamField path="address_id" type="String" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`Hash` -- Empty dict on success.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.addresses.delete("address-id")
```
