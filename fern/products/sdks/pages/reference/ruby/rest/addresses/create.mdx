Create a new regulatory address.

## **Parameters**

<ParamField path="customer_name" type="String" toc={true}>
  The customer or business name for the address.
</ParamField>

<ParamField path="street" type="String" toc={true}>
  Street address.
</ParamField>

<ParamField path="city" type="String" toc={true}>
  City name.
</ParamField>

<ParamField path="state" type="String" toc={true}>
  State or province.
</ParamField>

<ParamField path="postal_code" type="String" toc={true}>
  Postal or ZIP code.
</ParamField>

<ParamField path="country" type="String" toc={true}>
  ISO country code (e.g., `"US"`).
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional address fields as needed by the API.
</ParamField>

## **Returns**

`Hash` -- The newly created address object.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addr = client.addresses.create(
    customer_name="Acme Corp",
    street="123 Main St",
    city="Austin",
    state="TX",
    postal_code="78701",
    country="US",
)
puts("Created address:", addr["id"])
```
