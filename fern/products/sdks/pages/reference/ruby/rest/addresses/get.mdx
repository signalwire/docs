Retrieve a specific address.

## **Parameters**

<ParamField path="address_id" type="String" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`Hash` -- The address object.

## **Example**

```ruby {9-10}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addr = client.addresses["address-id"]
puts(addr["customer_name"], addr["street"])
```
