List video conferences in the project.

## **Parameters**

<ParamField path="page_size" type="Integer" toc={true}>
  Number of results per page.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering and pagination.

## **Returns**

`Hash` -- Paginated response containing video conferences.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

conferences = client.video.conferences.list()
```
