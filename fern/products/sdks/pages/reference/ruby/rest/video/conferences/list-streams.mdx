List active streams for a conference.

## **Parameters**

<ParamField path="conference_id" type="String" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Hash` -- List of streams in the conference.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

streams = client.video.conferences.list_streams("conference-id")
```
