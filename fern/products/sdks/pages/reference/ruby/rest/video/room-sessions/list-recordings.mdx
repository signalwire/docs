List recordings created during a room session.

## **Parameters**

<ParamField path="session_id" type="String" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Hash` -- Paginated list of session recordings.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recordings = client.video.room_sessions.list_recordings("session-id")
```
