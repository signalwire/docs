Replace a video conference. Uses PUT for full replacement, so all fields must be
provided in the request body.

<Note>
This method performs a full replacement (PUT), not a partial update (PATCH).
Include all fields you want to keep in the request body.
</Note>

## **Parameters**

<ParamField path="resource_id" type="String" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Hash` -- The updated conference.

## **Example**

```ruby {9-9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

conf = client.video.conferences.update("conference-id", name: "team-meeting-v2")
```
