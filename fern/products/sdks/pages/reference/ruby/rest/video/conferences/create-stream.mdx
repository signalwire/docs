Create a new stream for a conference, enabling external streaming output
(e.g., RTMP to a streaming platform).

## **Parameters**

<ParamField path="conference_id" type="String" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

<ParamField path="url" type="String" toc={true}>
  The streaming destination URL (e.g., an RTMP endpoint).
</ParamField>

Additional keyword arguments define the stream configuration.

## **Returns**

`Hash` -- The created stream resource.

## **Example**

```ruby {9}
require 'signalwire/rest/rest_client'

client = SignalWire::REST::RestClient.new(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

stream = client.video.conferences.create_stream(
    "conference-id",
    url="rtmp://live.example.com/key"
)
```
