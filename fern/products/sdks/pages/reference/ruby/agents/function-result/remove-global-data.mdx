Remove one or more keys from the global session data.

## **Parameters**

<ParamField path="keys" type="str | Array\<String\>" required={true} toc={true}>
  A single key string or a list of key strings to remove from global data.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```ruby {11}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")
agent.set_prompt_text("You are a helpful assistant.")

def clear_session_data(args, raw_data)
    return (
        SignalWire::Swaig::FunctionResult.new("Session data cleared.")
        .remove_global_data(["customer_id", "verified"])
    )

agent.serve()
```
