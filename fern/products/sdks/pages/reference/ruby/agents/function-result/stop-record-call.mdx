Stop an active background call recording.

## **Parameters**

<ParamField path="control_id" type="String, nil" default="nil" toc={true}>
  Identifier of the recording to stop. If not provided, the most recent
  recording is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```ruby {11}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")
agent.set_prompt_text("You are a helpful assistant.")

def stop_recording(args, raw_data)
    return (
        SignalWire::Swaig::FunctionResult.new("Recording stopped.")
        .stop_record_call(control_id: "main_recording")
    )

agent.serve()
```
