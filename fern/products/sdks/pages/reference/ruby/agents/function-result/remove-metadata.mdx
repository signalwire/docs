Remove one or more keys from the current function's metadata store.

## **Parameters**

<ParamField path="keys" type="str | Array\<String\>" required={true} toc={true}>
  A single key string or a list of key strings to remove from metadata.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```ruby {11}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")
agent.set_prompt_text("You are a helpful assistant.")

def clear_metadata(args, raw_data)
    return (
        SignalWire::Swaig::FunctionResult.new("Metadata cleared.")
        .remove_metadata(["timestamp", "user_id"])
    )

agent.serve()
```
