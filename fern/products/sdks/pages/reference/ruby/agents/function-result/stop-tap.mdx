Stop an active media tap stream.

## **Parameters**

<ParamField path="control_id" type="String, nil" default="nil" toc={true}>
  Identifier of the tap to stop. If not provided, the most recently started
  tap is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```ruby {11}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")
agent.set_prompt_text("You are a helpful assistant.")

def stop_monitoring(args, raw_data)
    return (
        SignalWire::Swaig::FunctionResult.new("Monitoring stopped.")
        .stop_tap(control_id: "supervisor_tap")
    )

agent.serve()
```
