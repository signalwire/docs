Convert all contexts to a dictionary for SWML generation. Calls `validate()`
first and raises `ArgumentError` if the configuration is invalid.

## **Returns**

`dict[str, Any]` -- A mapping of context names to their serialized hashes,
ready for inclusion in the SWML document.

## **Example**

```ruby {9}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")

contexts = agent.define_contexts()
ctx = contexts.add_context("default")
ctx.add_step("greet").set_text("Hello!")

data = contexts.to_dict()
# {"default": {"steps": [{"name": "greet", "text": "Hello!"}]}}

agent.serve()
```
