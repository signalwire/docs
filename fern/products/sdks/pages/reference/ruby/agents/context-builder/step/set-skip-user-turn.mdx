[ref-step]: /docs/sdks/reference/ruby/agents/context-builder/step

Skip waiting for user input after this step completes. The AI proceeds
immediately without waiting for the caller to speak.

## **Parameters**

<ParamField path="skip" type="Boolean" required={true} toc={true}>
  Whether to skip the user turn.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```ruby {9}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")

contexts = agent.define_contexts()
ctx = contexts.add_context("default")
intro = ctx.add_step("intro")
intro.set_text("Introduce yourself and immediately begin the intake process.")
intro.set_skip_user_turn(true)
ctx.add_step("intake").set_text("Collect the caller's information.")

agent.serve()
```
