[ref-step]: /docs/sdks/reference/ruby/agents/context-builder/step

Set the step's prompt text directly. Cannot be used with `add_section()` or
`add_bullets()`.

<Warning>
Mixing `set_text()` with `add_section()` or `add_bullets()` raises `ArgumentError`.
Use one approach or the other.
</Warning>

## **Parameters**

<ParamField path="text" type="String" required={true} toc={true}>
  Plain-text prompt instructions for this step.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```ruby {8}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")

contexts = agent.define_contexts()
ctx = contexts.add_context("default")
step = ctx.add_step("greeting")
step.set_text("Welcome the caller and ask how you can help.")

agent.serve()
```
