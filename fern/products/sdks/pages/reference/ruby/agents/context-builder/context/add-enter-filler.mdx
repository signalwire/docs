Add enter fillers for a specific language.

## **Parameters**

<ParamField path="language_code" type="String" required={true} toc={true}>
  Language code (e.g., `"en-US"`, `"es"`) or `"default"` for a catch-all.
</ParamField>

<ParamField path="fillers" type="Array\<String\>" required={true} toc={true}>
  List of filler phrases.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```ruby {9}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "my-agent", route: "/agent")

contexts = agent.define_contexts()
contexts.add_context("default").add_step("menu").set_text("Ask what the caller needs.")

support = contexts.add_context("support")
support.add_enter_filler("en-US", [
    "Let me connect you with support...",
    "Transferring you now..."
])
support.add_step("diagnose").set_text("Understand the customer's issue.")

agent.serve()
```
