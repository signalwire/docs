Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skill_name" type="String" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `true` if the skill is loaded, `false` otherwise.

## **Example**

```ruby {5-6}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "assistant", route: "/assistant")
agent.add_skill("datetime")
puts(agent.has_skill("datetime"))  # true
puts(agent.has_skill("web_search"))  # false
```
