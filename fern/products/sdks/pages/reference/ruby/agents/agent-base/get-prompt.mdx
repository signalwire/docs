[set-prompt-text]: /docs/sdks/reference/ruby/agents/agent-base/set-prompt-text

Retrieve the current prompt. Returns a string if
[`set_prompt_text()`][set-prompt-text] was used, or a
POM structure (list of hashes) if prompt sections were used. If neither was
explicitly set, returns a default prompt: `"You are {name}, a helpful AI assistant."`.

## **Returns**

`str | Array\<Hash\>` -- The current prompt in whatever format was configured.

## **Example**

```ruby {5}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "support", route: "/support")
agent.set_prompt_text("You are a customer support agent for Acme Corp.")
prompt = agent.get_prompt()
puts(prompt)
```
