Retrieve the username and password used for HTTP Basic Auth on this agent. Optionally
includes a source label indicating where the credentials came from.

## **Parameters**

<ParamField path="include_source" type="Boolean" default="false" toc={true}>
  When `true`, the returned array includes a third element indicating the credential
  source: `"provided"` (set explicitly), `"environment"` (from `SWML_BASIC_AUTH_USER`
  / `SWML_BASIC_AUTH_PASSWORD` environment variables), or `"generated"` (auto-generated
  at startup).
</ParamField>

## **Returns**

`Array\<String\>` -- `(username, password)` when `include_source` is `false`.

`Array\<String\>` -- `(username, password, source)` when `include_source` is
`true`.

## **Examples**

### Log credentials at startup

```ruby {5}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "assistant", route: "/assistant")
agent.set_prompt_text("You are a helpful assistant.")
user, password = agent.get_basic_auth_credentials()
puts("Auth: #{user}:#{password}")
agent.serve()
```

### Check credential source

```ruby {5}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "assistant", route: "/assistant")
agent.set_prompt_text("You are a helpful assistant.")
user, password, source = agent.get_basic_auth_credentials(include_source: true)
if source == "generated"
    puts("Auto-generated credentials: #{user}:#{password}")
    puts("Set SWML_BASIC_AUTH_USER and SWML_BASIC_AUTH_PASSWORD to use your own.")
agent.serve()
```
