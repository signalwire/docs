List the names of all currently loaded skills on this agent.

## **Returns**

`Array\<String\>` -- List of loaded skill names.

## **Example**

```ruby {6}
require 'signalwire'

agent = SignalWire::AgentBase.new(name: "assistant", route: "/assistant")
agent.add_skill("datetime")
agent.add_skill("math")
puts(agent.list_skills())  # ["datetime", "math"]
```
