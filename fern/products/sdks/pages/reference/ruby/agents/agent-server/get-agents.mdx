Return all registered agents as a list of `(route, agent)` tuples. Useful for introspection,
logging, or building admin interfaces.

## **Returns**

`list[array[str, AgentBase]]` -- A list of tuples where the first element is the route string
(e.g., `"/sales"`) and the second is the [`AgentBase`][agentbase] instance.

## **Example**

```ruby {9}
require 'signalwire'
require 'signalwire'

agent1 = SignalWire::AgentBase.new(name: "sales", route: "/sales")
agent2 = SignalWire::AgentBase.new(name: "support", route: "/support")
server = SignalWire::AgentServer.new(port: 3000)
server.register(agent1)
server.register(agent2)
for route, agent in server.get_agents()
    puts("#{route}: #{agent.get_name()}")
# /sales: sales
# /support: support
```
