Purchase a phone number. The exact keyword arguments depend on the number being
purchased (typically obtained from a prior `search()` call).

## **Parameters**

<ParamField path="**kwargs" type="dict" required={true} toc={true}>
  Number details. Common fields include `number` (the E.164 phone number to
  purchase).
</ParamField>

## **Returns**

`dict` -- The newly purchased phone number object.

## **Example**

```php {13}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

// Purchase the first available number from a search
$available = $client->phoneNumbers->search(area_code: "512", quantity: 1);
$numbers = $available["data"] ?? [];
if ($numbers) {
    $purchased = $client->phoneNumbers->create(number: numbers[0]["number"]);
    echo "Purchased:", $purchased["number"];
}


```
