List phone numbers owned by the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `page_size`,
  `page_token`).
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->phoneNumbers->list(page_size: 10);
foreach ($result["data"] ?? [] as $number) {
    echo $number["number"], $number["name"] ?? null;
}


```
