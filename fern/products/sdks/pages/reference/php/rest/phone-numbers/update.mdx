Update configuration for a phone number. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`, `call_handler`, `call_request_url`).
</ParamField>

## **Returns**

`dict` -- The updated phone number object.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->phoneNumbers->update("phone-number-id", name: "Support Line");


```
