List registered 10DLC brands.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of brand objects.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$brands = $client->registry->brands->list();
foreach ($brands["data"] ?? [] as $brand) {
    echo $brand["display_name"] ?? null, $brand["entity_type"] ?? null;
}


```
