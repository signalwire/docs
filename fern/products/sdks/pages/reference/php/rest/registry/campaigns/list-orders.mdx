List number assignment orders for a campaign.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of order objects.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$orders = $client->registry->campaigns->list_orders("campaign-id");
foreach ($orders["data"] ?? [] as $order) {
    echo $order["id"] ?? null, $order["status"] ?? null;
}


```
