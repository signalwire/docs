Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brand_id" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`dict` -- The brand object.

## **Example**

```php {9-10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$brand = $client->registry->brands->get("brand-id");
echo $brand["display_name"] ?? null, $brand["entity_type"] ?? null;


```
