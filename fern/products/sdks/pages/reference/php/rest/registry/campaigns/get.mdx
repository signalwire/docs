Retrieve a specific 10DLC campaign by ID.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

## **Returns**

`dict` -- The campaign object.

## **Example**

```php {9-10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$campaign = $client->registry->campaigns->get("campaign-id");
echo $campaign["description"] ?? null, $campaign["usecase"] ?? null;


```
