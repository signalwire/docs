Retrieve a specific recording.

## **Parameters**

<ParamField path="recording_id" type="string" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`dict` -- The recording object, including download URL and metadata.

## **Example**

```php {9-10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$rec = $client->recordings->get("recording-id");
echo $rec["duration"] ?? null, $rec["url"] ?? null;


```
