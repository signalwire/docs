List recordings in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of recording objects.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->recordings->list(page_size: 20);
foreach ($result["data"] ?? [] as $rec) {
    echo $rec["id"] ?? null, $rec["duration"] ?? null, $rec["state"] ?? null;
}


```
