Delete a recording.

## **Parameters**

<ParamField path="recording_id" type="string" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->recordings->delete("recording-id");


```
