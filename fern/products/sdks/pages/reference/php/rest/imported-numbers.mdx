Import externally-hosted phone numbers into your SignalWire project. This allows
you to route calls through SignalWire for numbers hosted by another carrier. Only
the create operation is supported.

Access via `client.importedNumbers` on a [`RestClient`][restclient] instance.

```php
<?php
use SignalWire\REST\RestClient;

$client = new RestClient();


```

---

### create

Import a phone number from an external provider.

## **Parameters**

<ParamField path="number" type="string" toc={true}>
  The phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional import parameters (e.g., `name`, `call_handler`,
  `call_request_url`).
</ParamField>

## **Returns**

`dict` -- The imported phone number object.

## **Example**

```php
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$imported = $client->importedNumbers->create(
    number: "+15551234567",
    name: "External Support Line",
);
echo "Imported:", $imported["id"] ?? null;


```
