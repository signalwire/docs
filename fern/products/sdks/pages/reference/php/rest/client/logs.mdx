Access the Logs resource for call and message history.

## **Parameters**

This method takes no parameters.

## **Returns**

`CrudResource` -- A CRUD resource instance for performing operations.

## **Example**

```php
<?php
use SignalWire\REST\RestClient;

$restClient = new RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);

$result = $restClient->logs();
```
