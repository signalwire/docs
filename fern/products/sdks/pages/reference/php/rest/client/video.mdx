Access the Video resource for video conferencing.

## **Parameters**

This method takes no parameters.

## **Returns**

`CrudResource` -- A CRUD resource instance for performing operations.

## **Example**

```php
<?php
use SignalWire\REST\RestClient;

$restClient = new RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);

$result = $restClient->video();
```
