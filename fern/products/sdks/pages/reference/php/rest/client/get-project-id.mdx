Get the project id of the REST client.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- The project id value.

## **Example**

```php
<?php
use SignalWire\REST\RestClient;

$restClient = new RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);

$result = $restClient->getProjectId();
```
