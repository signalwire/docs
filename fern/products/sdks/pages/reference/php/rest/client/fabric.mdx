Access the Fabric namespace for resource management.

## **Parameters**

This method takes no parameters.

## **Returns**

`Fabric` -- The Fabric namespace instance.

## **Example**

```php
<?php
use SignalWire\REST\RestClient;

$restClient = new RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);

$result = $restClient->fabric();
```
