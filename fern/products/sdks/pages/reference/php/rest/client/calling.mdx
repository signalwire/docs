Access the Calling namespace for voice call operations.

## **Parameters**

This method takes no parameters.

## **Returns**

`Calling` -- The Calling namespace instance.

## **Example**

```php
<?php
use SignalWire\REST\RestClient;

$restClient = new RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);

$result = $restClient->calling();
```
