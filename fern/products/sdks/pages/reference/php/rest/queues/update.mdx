Update a queue. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`, `max_size`).
</ParamField>

## **Returns**

`dict` -- The updated queue object.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$updated = $client->queues->update("queue-id", name: "priority-support");
echo "Updated: {$updated['id']}";


```
