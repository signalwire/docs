Retrieve a specific address.

## **Parameters**

<ParamField path="address_id" type="string" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`dict` -- The address object.

## **Example**

```php {9-10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$addr = $client->addresses->get("address-id");
echo $addr["customer_name"] ?? null, $addr["street"] ?? null;


```
