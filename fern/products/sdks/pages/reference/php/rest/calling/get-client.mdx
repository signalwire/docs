Get the client of the Calling namespace.

## **Parameters**

This method takes no parameters.

## **Returns**

`HttpClient` -- The underlying HTTP client.

## **Example**

```php
<?php
use SignalWire\REST\Namespaces\Calling;

// Access via RestClient
$client = new \SignalWire\REST\RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);
$calling = $client->calling();

$result = $calling->getClient();
```
