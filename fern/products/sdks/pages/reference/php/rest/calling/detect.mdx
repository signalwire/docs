Start a detector on an active call. Detectors can identify answering machines,
fax tones, or DTMF digits. Returns a `control_id` for managing the detector.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call to run detection on.
</ParamField>

<ParamField path="detect" type="dict" required={true} toc={true}>
  Detection configuration. Contains a `type` field and type-specific parameters:

  - `"machine"` — Answering machine detection (AMD). Optional: `initial_timeout`, `end_silence_timeout`, `machine_voice_threshold`.
  - `"fax"` — Fax tone detection. Optional: `tone`.
  - `"digit"` — DTMF digit detection. Optional: `digits`.
</ParamField>

<ParamField path="timeout" type="float" toc={true}>
  Maximum seconds to run detection before automatically stopping.
</ParamField>

## **Returns**

`dict` — The API response containing the `control_id`.

## **Examples**

### Answering Machine Detection

```php {10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

// Answering machine detection
$result = $client->calling->detect(
    call_id: "call-id-xxx",
    detect: {
        "type": "machine",
        "params": {
            "initial_timeout": 4.5,
            "end_silence_timeout": 1.0,
        }
    },
    timeout: 30.0,
);


```

### Fax Detection

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",

);

```
