End an active call.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call to end.
</ParamField>

<ParamField path="reason" type="string" toc={true}>
  The reason for ending the call.

  - `"hangup"` -- normal call termination
  - `"busy"` -- end the call with a busy signal
  - `"cancel"` -- cancel the call before it is answered
</ParamField>

## **Returns**

`dict` — The API response confirming the call was ended.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->calling->end(call_id: "call-id-xxx", reason: "hangup");


```
