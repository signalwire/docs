Start transcribing speech on an active call. Transcription results are delivered
via events. Returns a `control_id` for stopping the transcription.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call to transcribe.
</ParamField>

<ParamField path="language" type="string" toc={true}>
  Language code for transcription (e.g., `"en-US"`, `"es-ES"`).
</ParamField>

## **Returns**

`dict` — The API response containing the `control_id`.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->calling->transcribe(
    call_id: "call-id-xxx",
    language: "en-US",
);
$control_id = $result["control_id"] ?? null;


```
