Update a call.

## **Parameters**

<ParamField path="params" type="array" required={false} toc={true}>
  Additional parameters.
  Defaults to `[]`.
</ParamField>

## **Returns**

`array` -- The result array.

## **Example**

```php
<?php
use SignalWire\REST\Namespaces\Calling;

// Access via RestClient
$client = new \SignalWire\REST\RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);
$calling = $client->calling();

$result = $calling->updateCall(["key" => "value"]);
```
