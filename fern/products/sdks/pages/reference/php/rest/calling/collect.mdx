Start collecting user input on an active call. Supports DTMF digit collection
and speech recognition. Returns a `control_id` for managing the collection.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call to collect input on.
</ParamField>

<ParamField path="collect" type="dict" required={true} toc={true}>
  Collection configuration. Supports the following keys:

  - `digits` — DTMF digit collection settings (e.g., `{"max": 4, "terminators": "#"}`)
  - `speech` — Speech recognition settings (e.g., `{"end_silence_timeout": 1.0}`)
</ParamField>

<ParamField path="initial_timeout" type="float" toc={true}>
  Seconds to wait for the first input before timing out.
</ParamField>

<ParamField path="partial_results" type="bool" toc={true}>
  Whether to send partial speech recognition results as events.
</ParamField>

## **Returns**

`dict` — The API response containing the `control_id`.

## **Examples**

### Collect DTMF Digits

```php {10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

// Collect DTMF digits
$result = $client->calling->collect(
    call_id: "call-id-xxx",
    collect: {
        "digits": ["max" => 4, "terminators": "#", "digit_timeout": 5.0]
    },
    initial_timeout: 10.0,
);


```

### Collect Speech

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->calling->collect(
    call_id: "call-id-xxx",
    collect: {
        "speech": {
            "end_silence_timeout": 1.0,
            "language": "en-US",
        }
    },
);


```

### Collect Both

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->calling->collect(
    call_id: "call-id-xxx",
    collect: {
        "digits": ["max" => 1, "terminators": "#"],
        "speech": ["end_silence_timeout" => 2.0],
    },
    initial_timeout: 15.0,
);


```
