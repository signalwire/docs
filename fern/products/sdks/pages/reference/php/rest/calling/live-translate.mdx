Start live translation on an active call. Spoken audio is transcribed and
translated in real time to a target language.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="source_language" type="string" toc={true}>
  The language being spoken on the call (e.g., `"en-US"`).
</ParamField>

<ParamField path="target_language" type="string" toc={true}>
  The language to translate into (e.g., `"es-ES"`).
</ParamField>

## **Returns**

`dict` — The API response confirming live translation was started.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->calling->live_translate(
    call_id: "call-id-xxx",
    source_language: "en-US",
    target_language: "es-ES",
);


```
