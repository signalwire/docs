Get the base path of the Calling namespace.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- The base path value.

## **Example**

```php
<?php
use SignalWire\REST\Namespaces\Calling;

// Access via RestClient
$client = new \SignalWire\REST\RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);
$calling = $client->calling();

$result = $calling->getBasePath();
```
