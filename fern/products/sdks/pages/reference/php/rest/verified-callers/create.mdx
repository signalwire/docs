Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="phone_number" type="string" toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters (e.g., `friendly_name`).
</ParamField>

## **Returns**

`dict` -- The newly created caller ID object with its verification state.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$caller = $client->verifiedCallers->create(phone_number: "+15551234567");
echo "Verification state:", $caller["verification_state"] ?? null;


```
