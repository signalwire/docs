Manage verified caller IDs for your SignalWire project. Before using a phone
number as a caller ID for outbound calls, it must be verified. This resource
provides full CRUD (with PUT for updates) plus a two-step verification flow.

Access via `client.verifiedCallers` on a [`RestClient`][restclient] instance.

```php
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$callers = $client->verifiedCallers->list();
foreach ($callers["data"] ?? [] as $caller) {
    echo $caller["id"], $caller["phone_number"] ?? null;
}

```
