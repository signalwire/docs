List verified caller IDs in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of verified caller objects.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->verifiedCallers->list();
foreach ($result["data"] ?? [] as $caller) {
    echo $caller["phone_number"] ?? null, $caller["verification_state"] ?? null;
}


```
