Create a new stream for a video room, enabling external streaming output
(e.g., RTMP to a streaming platform).

## **Parameters**

<ParamField path="room_id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

<ParamField path="url" type="string" toc={true}>
  The streaming destination URL (e.g., an RTMP endpoint).
</ParamField>

Additional keyword arguments define the stream configuration.

## **Returns**

`dict` -- The created stream resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$stream = $client->video->rooms->create_stream(
    "room-id",
    url: "rtmp://live.example.com/stream-key"
);


```
