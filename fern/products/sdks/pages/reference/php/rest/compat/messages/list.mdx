List messages in the account.

## **Parameters**

<ParamField path="To" type="string" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="From" type="string" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing message resources.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$messages = $client->compat->messages->list(PageSize: 20);


```
