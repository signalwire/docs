Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The token identifier.
</ParamField>

<ParamField path="Ttl" type="int" toc={true}>
  New time-to-live in seconds for the token.
</ParamField>

## **Returns**

`dict` -- The updated token resource.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->compat->tokens->update("token-id", Ttl: 7200);


```
