Update a message (e.g., redact the body of a sent message).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The message SID.
</ParamField>

<ParamField path="Body" type="string" toc={true}>
  Set to an empty string to redact the message body.
</ParamField>

## **Returns**

`dict` -- The updated message resource.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->compat->messages->update("SM...", Body: "");


```
