List LAML bins in the account.

## **Parameters**

<ParamField path="FriendlyName" type="string" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing LAML bin resources.

## **Example**

```php {9-9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$bins = $client->compat->laml_bins->list(PageSize: 20);


```
