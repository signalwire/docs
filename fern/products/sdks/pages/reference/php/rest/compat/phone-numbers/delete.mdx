Release a phone number from the account.

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The phone number SID to release.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

```php {9-9}
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->compat->phoneNumbers->delete("PN...");


```
