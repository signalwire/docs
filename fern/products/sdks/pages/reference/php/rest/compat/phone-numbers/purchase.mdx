Purchase a new phone number for the account.

## **Parameters**

<ParamField path="PhoneNumber" type="string" required={true} toc={true}>
  The phone number to purchase in E.164 format.
</ParamField>

<ParamField path="FriendlyName" type="string" toc={true}>
  A human-readable label for the number.
</ParamField>

<ParamField path="VoiceUrl" type="string" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="SmsUrl" type="string" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="StatusCallback" type="string" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`dict` -- The purchased phone number resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$number = $client->compat->phoneNumbers->purchase(
    PhoneNumber: "+15551234567",
    FriendlyName: "Main Line"
);


```
