Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`dict` -- The SIP gateway resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$gateway = $client->fabric->sip_gateways->get("gateway-id");
echo "Sip Gateway: {$gateway['name']}, ID: {$gateway['id']}";


```
