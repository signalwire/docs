Access the Conference Rooms resource.

## **Parameters**

This method takes no parameters.

## **Returns**

`CrudResource` -- A CRUD resource instance for performing operations.

## **Example**

```php
<?php
use SignalWire\REST\Namespaces\Fabric;

// Access via RestClient
$client = new \SignalWire\REST\RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);
$fabric = $client->fabric();

$result = $fabric->conferenceRooms();
```
