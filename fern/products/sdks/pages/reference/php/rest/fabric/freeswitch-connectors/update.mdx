Replace a FreeSWITCH connector resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated FreeSWITCH connector resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$updated = $client->fabric->freeswitch_connectors->update("connector-id", name: "updated-name");
echo "Updated: {$updated['name']}";


```
