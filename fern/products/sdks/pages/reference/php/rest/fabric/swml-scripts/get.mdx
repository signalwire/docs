Retrieve a single SWML script resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

## **Returns**

`dict` -- The SWML script resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$item = $client->fabric->swml_scripts->get("resource-id");
echo "Swml Script: {$item['name']}, ID: {$item['id']}";


```
