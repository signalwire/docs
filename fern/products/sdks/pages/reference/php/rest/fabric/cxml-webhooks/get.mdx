Retrieve a single cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`dict` -- The cXML webhook resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$webhook = $client->fabric->cxml_webhooks->get("webhook-id");
echo "Cxml Webhook: {$webhook['name']}, ID: {$webhook['id']}";


```
