Replace a SIP endpoint resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated SIP endpoint resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$updated = $client->fabric->sip_endpoints->update("endpoint-id", name: "updated-name");
echo "Updated: {$updated['name']}";


```
