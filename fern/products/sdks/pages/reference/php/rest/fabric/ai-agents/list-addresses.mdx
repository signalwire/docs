List addresses associated with an AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$addresses = $client->fabric->ai_agents->list_addresses("agent-id");
foreach ($addresses["data"] ?? [] as $addr) {
    echo "{$addr['display_name']}: {$addr['id']}";
}


```
