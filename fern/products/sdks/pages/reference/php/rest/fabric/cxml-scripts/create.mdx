Create a new cXML script resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the cXML script.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created cXML script resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->fabric->cxml_scripts->create(
    name: "my-item",
);
echo "Created: {$result['id']}";


```
