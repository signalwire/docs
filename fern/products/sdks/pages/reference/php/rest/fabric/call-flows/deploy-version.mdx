Deploy a specific version of a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The deployed version details.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->fabric->call_flows->deploy_version(
    "call-flow-id", version_id: "version-id",
);
echo "Deployed version: {$result['id']}";


```
