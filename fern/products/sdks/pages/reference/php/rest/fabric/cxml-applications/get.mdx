Retrieve a single cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

## **Returns**

`dict` -- The cXML application resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$app = $client->fabric->cxml_applications->get("app-id");
echo "cXML Application: {$app['name']}, ID: {$app['id']}";


```
