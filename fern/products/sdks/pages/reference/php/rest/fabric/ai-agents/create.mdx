Create a new AI agent resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the AI agent.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created AI agent resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->fabric->ai_agents->create(
    name: "support-bot", type: "ai_agent",
);
echo "Created: {$result['id']}";


```
