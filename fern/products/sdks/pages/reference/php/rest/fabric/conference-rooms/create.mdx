Create a new conference room resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the conference room.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created conference room resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$result = $client->fabric->conference_rooms->create(
    name: "my-room",
);
echo "Created: {$result['id']}";


```
