List all Fabric resources across all resource types.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing Fabric resources.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$response = $client->fabric->resources->list();
foreach ($response["data"] ?? [] as $resource) {
    echo "{$resource['type']}: {$resource['name']} ({$resource['id']})";
}


```
