Delete a SWML script resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SWML script to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success (HTTP 204).

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$client->fabric->swml_scripts->delete("resource-id");
echo "Swml Script deleted";


```
