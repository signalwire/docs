Get the client of the Fabric namespace.

## **Parameters**

This method takes no parameters.

## **Returns**

`HttpClient` -- The underlying HTTP client.

## **Example**

```php
<?php
use SignalWire\REST\Namespaces\Fabric;

// Access via RestClient
$client = new \SignalWire\REST\RestClient(
    space: "your-space",
    projectId: "your-project-id",
    token: "your-api-token"
);
$fabric = $client->fabric();

$result = $fabric->getClient();
```
