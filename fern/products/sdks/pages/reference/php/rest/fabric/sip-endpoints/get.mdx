Retrieve a single SIP endpoint resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- The SIP endpoint resource.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$endpoint = $client->fabric->sip_endpoints->get("endpoint-id");
echo "Sip Endpoint: {$endpoint['name']}, ID: {$endpoint['id']}";


```
