Create a new number group.

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Group name.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional group configuration.
</ParamField>

## **Returns**

`dict` -- The newly created group object.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$group = $client->numberGroups->create(name: "Sales Numbers");
echo "Group ID:", $group["id"];


```
