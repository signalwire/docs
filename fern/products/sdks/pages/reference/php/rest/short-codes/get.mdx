Retrieve a specific short code.

## **Parameters**

<ParamField path="short_code_id" type="string" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`dict` -- The short code object.

## **Example**

```php {9-10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$code = $client->shortCodes->get("short-code-id");
echo $code["short_code"] ?? null;


```
