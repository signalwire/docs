Retrieve a specific voice call log entry.

## **Parameters**

<ParamField path="log_id" type="string" required={true} toc={true}>
  The voice log ID.
</ParamField>

## **Returns**

`dict` -- The voice log object.

## **Example**

```php {9-10}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$call = $client->logs->voice->get("voice-log-id");
echo $call["from"] ?? null, "->", $call["to"] ?? null, $call["duration"] ?? null;


```
