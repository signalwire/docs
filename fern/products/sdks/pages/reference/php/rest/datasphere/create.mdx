Upload a new document to the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Document name or title.
</ParamField>

<ParamField path="content" type="string" toc={true}>
  Document content text.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional document fields (e.g., `metadata`).
</ParamField>

## **Returns**

`dict` -- The newly created document object.

## **Example**

```php {9}
<?php
use SignalWire\REST\RestClient;

$client = new RestClient(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
);

$doc = $client->datasphere->documents->create(
    name: "FAQ",
    content: "Frequently asked questions and answers...",
);
echo "Document ID:", $doc["id"] ?? null;


```
