Convert the POM to a JSON string.

## **Returns**

`string` -- JSON representation of the POM.

## **Example**

```php {7}
<?php
use SignalWire\Core\PomBuilder\PomBuilder;

$pom = new PomBuilder();
$pom->addSection("Role", body: "You are a helpful assistant.");
$pom->addSection("Rules", bullets: ["Be concise", "Be accurate"]);

$json_str = $pom->toJson();
echo $json_str;


```
