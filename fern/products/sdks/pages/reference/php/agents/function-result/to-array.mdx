[ref-class]: /docs/sdks/reference/php/agents/function-result

Convert the function result to an array.

## **Parameters**

This method takes no parameters.

## **Returns**

`array` -- Array representation.

## **Example**

```php
<?php
use SignalWire\SWAIG\FunctionResult;

$functionResult = new FunctionResult();

$result = $functionResult->toArray();
```
