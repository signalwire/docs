Remove all dynamically added speech recognition hints. This clears every hint
that was previously added via [`addDynamicHints()`][add-dynamic-hints].
Static hints defined on the agent at configuration time are not affected.

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```php {11}
<?php
use SignalWire\Agent\AgentBase;
use SignalWire\SWAIG\FunctionResult;

$agent = new AgentBase(name: "my-agent", route: "/agent");
$agent->setPromptText("You are a helpful assistant.");

// Tool decorator: @agent.tool(name="reset_context", description="Reset speech recognition context")
function resetContext($args, $raw_data)
{
    return (;
        new FunctionResult("Context reset.")
        .clearDynamicHints()
    );
}

$agent->serve();


```
