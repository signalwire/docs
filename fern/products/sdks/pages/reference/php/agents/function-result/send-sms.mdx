Send an SMS or MMS message to a phone number. At least one of `body` or `media`
must be provided. Generates a SWML `sendSms` verb under the hood.

## **Parameters**

<ParamField path="to_number" type="string" required={true} toc={true}>
  Destination phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="from_number" type="string" required={true} toc={true}>
  Sender phone number in E.164 format. Must be a number in your SignalWire project.
</ParamField>

<ParamField path="body" type="?string" default="None" toc={true}>
  Text body of the message.
</ParamField>

<ParamField path="media" type="?array" default="None" toc={true}>
  List of media URLs to include as MMS attachments.
</ParamField>

<ParamField path="tags" type="?array" default="None" toc={true}>
  Tags to associate with the message for searching and filtering in the
  SignalWire dashboard.
</ParamField>

<ParamField path="region" type="?string" default="None" toc={true}>
  Region to originate the message from.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Text Message

```php {13}
<?php
use SignalWire\Agent\AgentBase;
use SignalWire\SWAIG\FunctionResult;

$agent = new AgentBase(name: "my-agent", route: "/agent");
$agent->setPromptText("You are a helpful assistant.");

// Tool decorator: @agent.tool(name="send_confirmation", description="Send an order confirmation SMS")
function sendConfirmation($args, $raw_data)
{
    $phone = $args["phone_number"] ?? null;
    $order_id = $args["order_id"] ?? null;
    return (;
        new FunctionResult("I've sent you a confirmation text.")
        .sendSms(
            to_number: $phone,
            from_number: "+15559876543",
            body: "Your order {$order_id} has been confirmed!"
        )
    );
}

$agent->serve();


```

### MMS with Media

```php {13}
<?php
use SignalWire\Agent\AgentBase;
use SignalWire\SWAIG\FunctionResult;

$agent = new AgentBase(name: "my-agent", route: "/agent");
$agent->setPromptText("You are a helpful assistant.");

// Tool decorator: @agent.tool(name="send_receipt", description="Send a receipt with media attachment")
function sendReceipt($args, $raw_data)
{
    $phone = $args["phone_number"] ?? null;
    $receipt_url = $args["receipt_url"] ?? null;
    return (;
        new FunctionResult("I've sent your receipt.")
        .sendSms(
            to_number: $phone,
            from_number: "+15559876543",
            body: "Here's your receipt:",
            media: [$receipt_url]
        )
    );
}

$agent->serve();


```
