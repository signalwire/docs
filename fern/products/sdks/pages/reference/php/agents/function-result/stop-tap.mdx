Stop an active media tap stream.

## **Parameters**

<ParamField path="control_id" type="?string" default="None" toc={true}>
  Identifier of the tap to stop. If not provided, the most recently started
  tap is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```php {11}
<?php
use SignalWire\Agent\AgentBase;
use SignalWire\SWAIG\FunctionResult;

$agent = new AgentBase(name: "my-agent", route: "/agent");
$agent->setPromptText("You are a helpful assistant.");

// Tool decorator: @agent.tool(name="stop_monitoring", description="Stop monitoring the call")
function stopMonitoring($args, $raw_data)
{
    return (;
        new FunctionResult("Monitoring stopped.")
        .$stop_tap(control_id: "supervisor_tap")
    );
}

$agent->serve();


```
