Serialize the function result to JSON.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- JSON string representation.

## **Example**

```php
<?php
use SignalWire\SWAIG\FunctionResult;

$functionResult = new FunctionResult();

$result = $functionResult->toJson();
```
