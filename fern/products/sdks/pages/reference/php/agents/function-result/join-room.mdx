Join a SignalWire RELAY room. Rooms enable multi-party communication and
collaboration between participants.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Name of the room to join.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```php {11}
<?php
use SignalWire\Agent\AgentBase;
use SignalWire\SWAIG\FunctionResult;

$agent = new AgentBase(name: "my-agent", route: "/agent");
$agent->setPromptText("You are a helpful assistant.");

// Tool decorator: @agent.tool(name="join_support_room", description="Join the support room")
function joinSupportRoom($args, $raw_data)
{
    return (;
        new FunctionResult("Connecting to the support room.")
        .joinRoom(name: "support-room-1")
    );
}

$agent->serve();


```
