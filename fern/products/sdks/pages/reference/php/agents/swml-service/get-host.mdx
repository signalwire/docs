Get the host of the SWML service.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- The host value.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->getHost();
```
