Get the port of the SWML service.

## **Parameters**

This method takes no parameters.

## **Returns**

`int` -- The integer value.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->getPort();
```
