## **Parameters**

<ParamField path="requestBody" type="?array" required={false} toc={true}>
  The incoming HTTP request body.
</ParamField>

## **Returns**

`?string` -- The value, or `null` if not set.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = Service::extractSipUsername([]);
```
