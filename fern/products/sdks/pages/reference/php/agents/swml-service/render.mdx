Render the SWML document as a JSON string.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- JSON string representation.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->render();
```
