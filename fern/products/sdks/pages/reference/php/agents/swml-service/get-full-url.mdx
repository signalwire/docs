Get the full url of the SWML service.

## **Parameters**

<ParamField path="includeAuth" type="bool" required={false} toc={true}>
  Whether to include authentication credentials in the URL.
  Defaults to `false`.
</ParamField>

## **Returns**

`string` -- The full url value.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->getFullUrl(true);
```
