Handle an incoming request.

## **Parameters**

<ParamField path="method" type="string" required={true} toc={true}>
  The HTTP method.
</ParamField>

<ParamField path="path" type="string" required={true} toc={true}>
  The request path.
</ParamField>

<ParamField path="headers" type="array" required={false} toc={true}>
  HTTP headers to include.
  Defaults to `[]`.
</ParamField>

<ParamField path="body" type="?string" required={false} toc={true}>
  The request body.
  Defaults to `null`.
</ParamField>

## **Returns**

`array` -- The result array.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->handleRequest(
    method: "GET",
    path: "/plants",
    headers: ["Content-Type" => "application/json"],
    body: "Hello"
);
```
