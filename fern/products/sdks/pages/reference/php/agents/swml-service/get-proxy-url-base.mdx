Get the proxy url base of the SWML service.

## **Parameters**

<ParamField path="headers" type="array" required={false} toc={true}>
  HTTP headers to include.
  Defaults to `[]`.
</ParamField>

## **Returns**

`string` -- The proxy url base value.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->getProxyUrlBase(["Content-Type" => "application/json"]);
```
