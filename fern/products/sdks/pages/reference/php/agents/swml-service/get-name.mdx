Get the name of the SWML service.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- The name value.

## **Example**

```php
<?php
use SignalWire\SWML\Service;

$service = new Service(name: "my-service", route: "/swml");

$result = $service->getName();
```
