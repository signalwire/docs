---

Stop the web server. Sets the internal running flag to `False`.

## **Returns**

`null`

## **Example**

```php {6}
<?php
use SignalWire\Core\SwmlService\SWMLService;

$service = new Service(name: "my-service");

// In a signal handler or shutdown hook:
$service->stop();


```
