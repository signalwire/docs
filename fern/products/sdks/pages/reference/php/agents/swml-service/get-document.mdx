Get the current SWML document as a Python dictionary. The returned structure follows
the standard SWML format with `version` and `sections` keys.

## **Returns**

`dict[string, mixed]` — The current SWML document. Structure:

```json
{
    "version": "1.0.0",
    "sections": {
        "main": [
            {"answer": {}},
            {"ai": {...}}
        ]
    }
}
```

## **Example**

```php {7}
<?php
use SignalWire\Core\SwmlService\SWMLService;

$service = new Service(name: "my-service");
$service->add_verb("answer", {});
$service->add_verb("play", ["url" => "https://example.com/audio.mp3"]);

$doc = $service->getDocument();
echo $doc["version"];
echo $len($doc["sections"]["main"]);


```
