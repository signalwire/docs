Override the post-prompt text while this context is active.

## **Parameters**

<ParamField path="post_prompt" type="string" required={true} toc={true}>
  Post-prompt text for this context.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```php {9}
<?php
use SignalWire\Agent\AgentBase;

$agent = new AgentBase(name: "my-agent", route: "/agent");

$contexts = $agent->defineContexts();
$contexts->addContext("default")->addStep("menu")->setText("Ask what the caller needs.");

$billing = $contexts->addContext("billing");
$billing->setPostPrompt("Summarize the billing issue and resolution.");
$billing->addStep("review")->setText("Review the caller's account and recent charges.");

$agent->serve();


```
