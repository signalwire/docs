Add enter fillers for a specific language.

## **Parameters**

<ParamField path="language_code" type="string" required={true} toc={true}>
  Language code (e.g., `"en-US"`, `"es"`) or `"default"` for a catch-all.
</ParamField>

<ParamField path="fillers" type="array" required={true} toc={true}>
  List of filler phrases.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```php {9}
<?php
use SignalWire\Agent\AgentBase;

$agent = new AgentBase(name: "my-agent", route: "/agent");

$contexts = $agent->defineContexts();
$contexts->addContext("default")->addStep("menu")->setText("Ask what the caller needs.");

$support = $contexts->addContext("support");
$support->add_enter_filler("en-US", [;
    "Let me connect you with support...",
    "Transferring you now..."
]);
$support->addStep("diagnose")->setText("Understand the customer's issue.");

$agent->serve();


```
