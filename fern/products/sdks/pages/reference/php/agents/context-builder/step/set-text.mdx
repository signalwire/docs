[ref-step]: /docs/sdks/reference/php/agents/context-builder/step

Set the step's prompt text directly. Cannot be used with `addSection()` or
`addBullets()`.

<Warning>
Mixing `setText()` with `addSection()` or `addBullets()` raises `ValueError`.
Use one approach or the other.
</Warning>

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  Plain-text prompt instructions for this step.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```php {8}
<?php
use SignalWire\Agent\AgentBase;

$agent = new AgentBase(name: "my-agent", route: "/agent");

$contexts = $agent->defineContexts();
$ctx = $contexts->addContext("default");
$step = $ctx->addStep("greeting");
$step->setText("Welcome the caller and ask how you can help.");

$agent->serve();


```
