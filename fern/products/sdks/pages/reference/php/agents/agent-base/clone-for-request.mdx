Clone the agent for processing a single request.

## **Parameters**

This method takes no parameters.

## **Returns**

`static` -- A new AgentBase instance.

## **Example**

```php
<?php
use SignalWire\Agent\AgentBase;

$agentBase = new AgentBase(name: "my-agent", route: "/agent");

$result = $agentBase->cloneForRequest();
```
