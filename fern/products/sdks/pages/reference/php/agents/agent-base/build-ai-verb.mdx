Build the AI verb configuration array.

## **Parameters**

<ParamField path="headers" type="array" required={false} toc={true}>
  HTTP headers to include.
  Defaults to `[]`.
</ParamField>

## **Returns**

`array` -- The result array.

## **Example**

```php
<?php
use SignalWire\Agent\AgentBase;

$agentBase = new AgentBase(name: "my-agent", route: "/agent");

$result = $agentBase->buildAiVerb(["Content-Type" => "application/json"]);
```
