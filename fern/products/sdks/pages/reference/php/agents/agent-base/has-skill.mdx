Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skill_name" type="string" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `True` if the skill is loaded, `False` otherwise.

## **Example**

```php {5-6}
<?php
use SignalWire\Agent\AgentBase;

$agent = new AgentBase(name: "assistant", route: "/assistant");
$agent->addSkill("datetime");
echo $agent->hasSkill("datetime");
echo $agent->hasSkill("web_search");


```
