List the names of all currently loaded skills on this agent.

## **Returns**

`list[string]` -- List of loaded skill names.

## **Example**

```php {6}
<?php
use SignalWire\Agent\AgentBase;

$agent = new AgentBase(name: "assistant", route: "/assistant");
$agent->addSkill("datetime");
$agent->addSkill("math");
echo $agent->listSkills();


```
