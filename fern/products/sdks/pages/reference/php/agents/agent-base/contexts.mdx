Access the context builder for defining conversation contexts.

## **Parameters**

This method takes no parameters.

## **Returns**

`ContextBuilder` -- The context builder instance.

## **Example**

```php
<?php
use SignalWire\Agent\AgentBase;

$agentBase = new AgentBase(name: "my-agent", route: "/agent");

$result = $agentBase->contexts();
```
