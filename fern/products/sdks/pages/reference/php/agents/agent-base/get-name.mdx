Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`string` -- The agent name.

## **Example**

```php {4}
<?php
use SignalWire\Agent\AgentBase;

$agent = new AgentBase(name: "support", route: "/support");
echo $agent->getName();


```
