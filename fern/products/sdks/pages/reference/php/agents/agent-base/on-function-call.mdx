Handle an incoming SWAIG function call.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The name identifier.
</ParamField>

<ParamField path="args" type="array" required={true} toc={true}>
  The function call arguments.
</ParamField>

<ParamField path="rawData" type="array" required={true} toc={true}>
  The raw SWAIG request data.
</ParamField>

## **Returns**

`?FunctionResult` -- The value, or `null` if not set.

## **Example**

```php
<?php
use SignalWire\Agent\AgentBase;

$agentBase = new AgentBase(name: "my-agent", route: "/agent");

$result = $agentBase->onFunctionCall(
    name: "plant-lookup",
    args: [],
    rawData: []
);
```
