Render the complete SWML response for a request.

## **Parameters**

<ParamField path="requestBody" type="?array" required={false} toc={true}>
  The incoming HTTP request body.
  Defaults to `null`.
</ParamField>

<ParamField path="headers" type="array" required={false} toc={true}>
  HTTP headers to include.
  Defaults to `[]`.
</ParamField>

## **Returns**

`array` -- The result array.

## **Example**

```php
<?php
use SignalWire\Agent\AgentBase;

$agentBase = new AgentBase(name: "my-agent", route: "/agent");

$result = $agentBase->renderSwml(
    requestBody: [],
    headers: ["Content-Type" => "application/json"]
);
```
