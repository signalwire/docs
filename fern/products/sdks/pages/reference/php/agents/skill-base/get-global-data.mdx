Return data to merge into the agent's `global_data` dictionary. Override to
provide skill-specific session data that should be available to all SWAIG
function handlers.

## **Returns**

`dict[string, mixed]` -- Key-value pairs (default: empty dict).

## **Example**

```php {7}
<?php
use SignalWire\Core\SkillBase\SkillBase;

class WeatherSkill extends SkillBase
{
    $SKILL_NAME = "weather";
    $SKILL_DESCRIPTION = "Provides weather information";
}

    public function getGlobalData()
    {
        return {;
            "weather_api_base": "https://api.weatherapi.com/v1",
            "default_units": self.$params["units"] ?? "fahrenheit"
        }

    public function setup(): bool
    {
        return true;
}

    public function registerTools()
    {
        // no-op
}


```
