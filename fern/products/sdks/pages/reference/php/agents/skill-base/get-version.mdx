Get the version of the skill.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- The version value.

## **Example**

```php
<?php
use SignalWire\Skills\SkillBase;

// SkillBase is abstract; extend it to create a custom skill
class PlantSkill extends SkillBase {
    // ...
}
$skillBase = new PlantSkill();

$result = $skillBase->getVersion();
```
