Register a sip username.

## **Parameters**

<ParamField path="username" type="string" required={true} toc={true}>
  The SIP username.
</ParamField>

<ParamField path="route" type="string" required={true} toc={true}>
  The route path.
</ParamField>

## **Returns**

`self` -- The the agent server instance for method chaining.

## **Example**

```php
<?php
use SignalWire\Server\AgentServer;

$agentServer = new AgentServer();

$agentServer->registerSipUsername(
    username: "sip-user",
    route: "/plants"
);
```
