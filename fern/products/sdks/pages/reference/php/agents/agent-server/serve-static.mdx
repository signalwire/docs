Serve static files from a directory.

## **Parameters**

<ParamField path="directory" type="string" required={true} toc={true}>
  The directory path.
</ParamField>

<ParamField path="urlPrefix" type="string" required={true} toc={true}>
  URL prefix for serving static files.
</ParamField>

## **Returns**

`self` -- The the agent server instance for method chaining.

## **Example**

```php
<?php
use SignalWire\Server\AgentServer;

$agentServer = new AgentServer();

$agentServer->serveStatic(
    directory: "/var/www/static",
    urlPrefix: "/static"
);
```
