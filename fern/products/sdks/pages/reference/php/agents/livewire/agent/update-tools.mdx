[function-tool]: /docs/sdks/reference/php/agents/livewire/function-tool

Replace the agent's tool list mid-session. The new tools take effect immediately,
replacing all previously registered tools on this agent.

## **Parameters**

<ParamField path="tools" type="array" required={true} toc={true}>
  A new list of [`@function_tool`][function-tool]-decorated
  functions.
</ParamField>

## **Returns**

`null`

## **Example**

```php {16}
<?php
use SignalWire\Livewire\Agent;
use SignalWire\Livewire\AgentSession;
use SignalWire\Livewire\AgentServer;
use SignalWire\Livewire\JobContext;
use SignalWire\Livewire\function_tool;
use SignalWire\Livewire\run_app;

// @function_tool
function greet(string $name): string
{
    """Greet the caller by name."""
    return "Hello, {$name}!";
}

// @function_tool
function farewell(string $name): string
{
    """Say goodbye to the caller."""
    return "Goodbye, {$name}!";
}

$agent = $Agent(instructions: "You are a greeter.", tools: [$greet]);

// Later, swap in a different tool set
$agent->update_tools([$farewell]);


```
