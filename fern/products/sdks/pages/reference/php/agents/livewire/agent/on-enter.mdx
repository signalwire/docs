Lifecycle hook called when the agent enters a session. Override in a subclass to
run custom initialization logic when the agent is first bound to an
[`AgentSession`][agentsession].

## **Parameters**

None.

## **Returns**

`null`

## **Example**

```php {6}
<?php
use SignalWire\Livewire\Agent;
use SignalWire\Livewire\AgentSession;
use SignalWire\Livewire\AgentServer;
use SignalWire\Livewire\JobContext;
use SignalWire\Livewire\run_app;

$server = new AgentServer();

class GreeterAgent extends Agent
{
    public function onEnter()
    {
        if (self.$session) {
            self.$session->say("Welcome! I just started up.");
        }

// @server.rtc_session()
function entrypoint(JobContext $ctx)
{
    $ctx->connect();
    $agent = $GreeterAgent(instructions: "You are a friendly greeter.");
    $session = $AgentSession();
    $session->start($agent, room: ctx.$room);
}

$run_app($server);


```
