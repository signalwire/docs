Lifecycle hook called when the user finishes speaking. Override in a subclass to
inspect or modify the conversation before the LLM generates a reply.

## **Parameters**

<ParamField path="turn_ctx" type="mixed" default="None" toc={true}>
  Turn context object. Accepted for API compatibility.
</ParamField>

<ParamField path="new_message" type="mixed" default="None" toc={true}>
  The new message from the user. Accepted for API compatibility.
</ParamField>

## **Returns**

`null`

## **Example**

```php {6}
<?php
use SignalWire\Livewire\Agent;
use SignalWire\Livewire\AgentSession;
use SignalWire\Livewire\AgentServer;
use SignalWire\Livewire\JobContext;
use SignalWire\Livewire\run_app;

$server = new AgentServer();

class LoggingAgent extends Agent
{
    public function onUserTurnCompleted(turn_ctx: null, new_message: null)
    {
        echo "User finished speaking.";
        echo "Conversation history: {$self->session.history}";
    }

// @server.rtc_session()
function entrypoint(JobContext $ctx)
{
    $ctx->connect();
    $agent = $LoggingAgent(instructions: "You are a helpful assistant.");
    $session = $AgentSession();
    $session->start($agent, room: ctx.$room);
}

$run_app($server);


```
