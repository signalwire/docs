Lifecycle hook called when the agent exits a session. Override in a subclass to
run cleanup logic when the agent's session ends.

## **Parameters**

None.

## **Returns**

`null`

## **Example**

```php {6}
<?php
use SignalWire\Livewire\Agent;
use SignalWire\Livewire\AgentSession;
use SignalWire\Livewire\AgentServer;
use SignalWire\Livewire\JobContext;
use SignalWire\Livewire\run_app;

$server = new AgentServer();

class TrackedAgent extends Agent
{
    public function onExit()
    {
        echo "Agent session has ended, running cleanup.";
    }

// @server.rtc_session()
function entrypoint(JobContext $ctx)
{
    $ctx->connect();
    $agent = $TrackedAgent(instructions: "You are a helpful assistant.");
    $session = $AgentSession();
    $session->start($agent, room: ctx.$room);
}

$run_app($server);


```
