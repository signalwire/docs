Trigger the agent to generate a reply. On SignalWire the prompt handles
generation automatically; when `instructions` is provided, it is queued as
additional text to speak.

## **Parameters**

<ParamField path="instructions" type="?string" default="None" toc={true}>
  Optional instructions to include in the reply.
</ParamField>

## **Returns**

`null`

## **Example**

```php {11}
<?php
use SignalWire\Livewire\Agent;
use SignalWire\Livewire\AgentSession;
use SignalWire\Livewire\AgentServer;
use SignalWire\Livewire\JobContext;
use SignalWire\Livewire\run_app;

$server = new AgentServer();

// @server.rtc_session()
function entrypoint(JobContext $ctx)
{
    $ctx->connect();
    $agent = $Agent(instructions: "You are a helpful assistant.");
    $session = $AgentSession();
    $session->start($agent, room: ctx.$room);
    $session->generate_reply(instructions: "Please introduce yourself to the caller.");
}

$run_app($server);


```
