Get the URL scheme based on SSL configuration.

## **Returns**

`string` -- `"https"` if SSL is enabled, `"http"` otherwise.

## **Example**

```php {4}
<?php
use SignalWire\Core\SecurityConfig\SecurityConfig;

$security = $SecurityConfig();
$scheme = $security->get_url_scheme();

echo "URL scheme: {$scheme}")  # "http" (default;


```
