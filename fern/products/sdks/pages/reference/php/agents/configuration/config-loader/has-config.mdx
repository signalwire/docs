Check whether a configuration file was successfully loaded.

## **Returns**

`bool` -- `True` if a config file was found and loaded.

## **Example**

```php {4}
<?php
use SignalWire\Core\ConfigLoader\ConfigLoader;

$loader = $ConfigLoader();
if ($loader->has_config()) {
    echo "Loaded config from: {$loader->get_config_file()}";
}
} else {
    echo "No config file found, using defaults";
}


```
