Get the path of the loaded config file.

## **Returns**

`?string` -- The file path, or `null` if no config was loaded.

## **Example**

```php {4}
<?php
use SignalWire\Core\ConfigLoader\ConfigLoader;

$loader = $ConfigLoader(["config.json", "/etc/swml/config.json"]);
$path = $loader->get_config_file();

if ($path) {
    echo "Loaded config from: {$path}";
}
} else {
    echo "No config file found";
}


```
