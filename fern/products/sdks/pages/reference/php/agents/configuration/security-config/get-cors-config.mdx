Get CORS configuration suitable for FastAPI's `CORSMiddleware`.

## **Returns**

`dict[string, mixed]` -- Dictionary with `allow_origins`, `allow_credentials`,
`allow_methods`, and `allow_headers` keys.

## **Example**

```php {6}
<?php
$from $fastapi $import $FastAPI;
$from $fastapi->middleware->cors $import $CORSMiddleware;
use SignalWire\Core\SecurityConfig\SecurityConfig;

$security = $SecurityConfig();
$cors_config = $security->get_cors_config();

$app = $FastAPI();
$app->add_middleware($CORSMiddleware, **$cors_config);
echo $cors_config;
// {'allow_origins': ['*'], 'allow_credentials': True, 'allow_methods': ['*'], 'allow_headers': ['*']}


```
