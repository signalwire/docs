Get a configuration value by dot-notation path. Environment variables are
substituted in the returned value.

## **Parameters**

<ParamField path="key_path" type="string" required={true} toc={true}>
  Dot-separated path to the config value (e.g., `"security.ssl_enabled"`,
  `"server.port"`).
</ParamField>

<ParamField path="default" type="mixed" default="None" toc={true}>
  Default value if the path is not found.
</ParamField>

## **Returns**

`mixed` -- The configuration value with environment variables substituted, or the
default if not found.

## **Example**

```php {5-7}
<?php
use SignalWire\Core\ConfigLoader\ConfigLoader;

$loader = $ConfigLoader(["config.json"]);

$port = $loader["server.port"] ?? 3000;
$ssl_enabled = $loader["security.ssl_enabled"] ?? false;
$auth_user = $loader["security.auth.basic.user"] ?? "signalwire";

echo "Port: {$port}, SSL: {$ssl_enabled}, User: {$auth_user}";


```
