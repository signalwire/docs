Set the global error keys for data map processing.

## **Parameters**

<ParamField path="keys" type="array" required={true} toc={true}>
  Array of error key strings.
</ParamField>

## **Returns**

`self` -- The the data map instance for method chaining.

## **Example**

```php
<?php
use SignalWire\DataMap\DataMap;

$dataMap = new DataMap("lookup-plant");

$dataMap->globalErrorKeys(["error_key"]);
```
