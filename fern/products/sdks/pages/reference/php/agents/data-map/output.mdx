Set the output for the most recently added webhook. The
[`FunctionResult`][functionresult] template
is filled with response data after the API call succeeds.

## **Parameters**

<ParamField path="result" type="FunctionResult" required={true} toc={true}>
  A FunctionResult defining the response template. Use `${response.field}` to
  reference API response fields.
</ParamField>

## **Returns**

[`DataMap`][ref-datamap] -- Self for method chaining. Raises `ValueError` if no webhook has been
added yet.

---

Set a top-level fallback output used when all webhooks fail.

## **Parameters**

<ParamField path="result" type="FunctionResult" required={true} toc={true}>
  A FunctionResult defining the fallback response.
</ParamField>

## **Returns**

`DataMap` -- Self for method chaining.

<Tip>
Always set a `fallbackOutput` so the AI has something meaningful to say even when
external APIs are unavailable.
</Tip>

## **Example**

```php {9}
<?php
use SignalWire\AgentBase;
use SignalWire\DataMap;
use SignalWire\SWAIG\FunctionResult;

$weather = (
    new DataMap("get_weather")
    .$purpose("Get current weather for a city")
    .$parameter("city", "string", "City name", required: true)
    .$webhook("GET", "https://api.weatherapi.com/v1/current.json?key: KEY&q: ${enc:args.city}")
    .$output(new FunctionResult(
        "Current weather in ${args.city}: " .
        "${response.current.condition.text}, " .
        "${response.current.temp_f} degrees Fahrenheit"
    ))
    .fallbackOutput(new FunctionResult(
        "Sorry, I couldn't get weather data for ${args.city}"
    ))
);

$agent = new AgentBase(name: "weather-agent");
$agent->setPromptText("You are a helpful assistant.");
$agent->registerSwaigFunction($weather->toSwaigFunction());

    $agent->run();


```
