Create a simple api tool.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The name identifier.
</ParamField>

<ParamField path="purpose" type="string" required={true} toc={true}>
  The purpose description for the tool.
</ParamField>

<ParamField path="parameters" type="array" required={true} toc={true}>
  Parameter definitions for the tool.
</ParamField>

<ParamField path="method" type="string" required={true} toc={true}>
  The HTTP method.
</ParamField>

<ParamField path="url" type="string" required={true} toc={true}>
  The URL endpoint.
</ParamField>

<ParamField path="output" type="mixed" required={true} toc={true}>
  The expected output format.
</ParamField>

<ParamField path="headers" type="array" required={false} toc={true}>
  HTTP headers to include.
  Defaults to `[]`.
</ParamField>

## **Returns**

`array` -- Array representation.

## **Example**

```php
<?php
use SignalWire\DataMap\DataMap;

$dataMap = new DataMap("lookup-plant");

$result = DataMap::createSimpleApiTool(
    name: "plant-lookup",
    purpose: "Look up plant care details",
    parameters: ["key" => "value"],
    method: "GET",
    url: "https://api.example.com/plants",
    output: "value",
    headers: ["Content-Type" => "application/json"]
);
```
