[swmlservice-add-section]: /docs/sdks/reference/php/agents/swml-service/add-section

Add a new named section to the SWML document. Delegates to
[`Service.addSection()`][swmlservice-add-section].

## **Parameters**

<ParamField path="section_name" type="string" required={true} toc={true}>
  Name of the section to create.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```php {10}
<?php
use SignalWire\Core\SwmlService\SWMLService;
use SignalWire\Core\SwmlBuilder\SWMLBuilder;

$service = new Service(name: "multi-section");
$builder = new Document($service);

$doc = (
    $builder
    .$answer()
    .addSection("greeting")
    .$play(url: "say:Hello!")
    .$build()
);
echo $doc;


```
