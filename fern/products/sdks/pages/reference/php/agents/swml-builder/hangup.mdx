Add a `hangup` verb to the main section. Ends the current call.

## **Parameters**

<ParamField path="reason" type="?string" toc={true}>
  An optional reason string for the hangup, included in call logs.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```php {11}
<?php
use SignalWire\Core\SwmlService\SWMLService;
use SignalWire\Core\SwmlBuilder\SWMLBuilder;

$service = new Service(name: "ivr");
$builder = new Document($service);

$doc = (
    $builder
    .$answer()
    .$play(url: "https://example.com/goodbye.mp3")
    .$hangup(reason: "Call completed")
    .$build()
);
echo $doc;


```
