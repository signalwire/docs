Build and render the complete SWML document as a JSON string.

## **Returns**

`string` -- The SWML document serialized as JSON.

## **Example**

```php {7}
<?php
use SignalWire\Core\SwmlService\SWMLService;
use SignalWire\Core\SwmlBuilder\SWMLBuilder;

$service = new Service(name: "demo");
$builder = new Document($service);

$json_str = $builder->answer().$ai(prompt_text: "Hello")->render();
echo $json_str;


```
