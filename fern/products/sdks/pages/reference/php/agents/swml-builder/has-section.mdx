Check whether the section exists.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The name identifier.
</ParamField>

## **Returns**

`bool` -- Whether the condition is true.

## **Example**

```php
<?php
use SignalWire\SWML\Document;

$document = new Document();

$result = $document->hasSection("plant-lookup");
```
