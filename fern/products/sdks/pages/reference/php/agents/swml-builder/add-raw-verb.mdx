## **Parameters**

<ParamField path="section" type="string" required={true} toc={true}>
  The SWML document section name.
</ParamField>

<ParamField path="verbHash" type="array" required={true} toc={true}>
  Raw verb definition as an associative array.
</ParamField>

## **Returns**

`void` -- 

## **Example**

```php
<?php
use SignalWire\SWML\Document;

$document = new Document();

$document->addRawVerb(
    section: "main",
    verbHash: []
```
