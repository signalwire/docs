Add a verb to section.

## **Parameters**

<ParamField path="section" type="string" required={true} toc={true}>
  The SWML document section name.
</ParamField>

<ParamField path="verbName" type="string" required={true} toc={true}>
  The SWML verb name.
</ParamField>

<ParamField path="config" type="mixed" required={true} toc={true}>
  Configuration options.
</ParamField>

## **Returns**

`void` -- 

## **Example**

```php
<?php
use SignalWire\SWML\Document;

$document = new Document();

$document->addVerbToSection(
    section: "main",
    verbName: "play",
    config: "value"
);
```
