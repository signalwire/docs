Render the SWML document as a pretty-printed JSON string.

## **Parameters**

This method takes no parameters.

## **Returns**

`string` -- JSON string representation.

## **Example**

```php
<?php
use SignalWire\SWML\Document;

$document = new Document();

$result = $document->renderPretty();
```
