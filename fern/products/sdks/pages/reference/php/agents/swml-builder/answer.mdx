Add an `answer` verb to the main section. Answers an incoming call.

## **Parameters**

<ParamField path="max_duration" type="?int" toc={true}>
  Maximum call duration in seconds. The call is automatically hung up after this
  duration elapses.
</ParamField>

<ParamField path="codecs" type="?string" toc={true}>
  Comma-separated list of audio codecs to use for the call.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```php {9}
<?php
use SignalWire\Core\SwmlService\SWMLService;
use SignalWire\Core\SwmlBuilder\SWMLBuilder;

$service = new Service(name: "my-service");
$builder = new Document($service);

$doc = (
    $builder
    .$answer(max_duration: 3600)
    .$play(url: "https://example.com/welcome.mp3")
    .$build()
);
echo $doc;


```
