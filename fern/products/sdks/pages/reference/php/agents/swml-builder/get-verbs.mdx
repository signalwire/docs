Get the verbs of the SWML document.

## **Parameters**

<ParamField path="section" type="string" required={false} toc={true}>
  The SWML document section name.
  Defaults to `'main'`.
</ParamField>

## **Returns**

`array` -- The result array.

## **Example**

```php
<?php
use SignalWire\SWML\Document;

$document = new Document();

$result = $document->getVerbs("main");
```
