Retrieve an active session by ID. Returns `null` if the session does not exist,
has expired, or if its underlying process has died. Automatically calls
`touch()` on the session to reset its expiration timer.

## **Parameters**

<ParamField path="session_id" type="string" required={true} toc={true}>
  The session ID to look up.
</ParamField>

## **Returns**

[`Session`][session] or `null`

## **Example**

```php {4}
<?php
use SignalWire\McpGateway\SessionManager;

$manager = $SessionManager($config);
$session = $manager->get_session("call-abc-123");
if ($session) {
    echo "Session active for service: {$session->service_name}";
}
} else {
    echo "Session !found or expired";
}


```
