Manually start the `initial_timeout` timer on the active collect operation. Useful
when `startInputTimers` was set to `false` during the initial call.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```php {22}
<?php
use SignalWire\Relay\Client;

$client = new Client(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
    contexts: ["default"],
);

// Handler: $client->onCall()
function handleCall($call)
{
    $call->answer();
    $action = $await $call->playAndCollect(
        media: [["type" => "tts", "text": "Press 1 for sales, 2 for support."]],
        collect: {
            "digits": ["max" => 1, "digit_timeout": 5.0],
            "startInputTimers": false,
        },
    );
}

    // Start timers when ready
    $action->startInputTimers();

    $event = $await $action->wait();

$client->run();


```
