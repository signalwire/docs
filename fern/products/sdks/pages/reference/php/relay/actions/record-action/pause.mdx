Pause the recording.

## **Parameters**

<ParamField path="behavior" type="?string" toc={true}>
  Optional pause behavior. Controls what happens to the audio stream while paused
  (e.g., silence insertion).
</ParamField>

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```php {16}
<?php
use SignalWire\Relay\Client;

$client = new Client(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
    contexts: ["default"],
);

// Handler: $client->onCall()
function handleCall($call)
{
    $call->answer();
    $action = $await $call->record(["direction" => "both", "format": "wav"]);
}

    // Pause during sensitive information
    $action->pause();
    // ... sensitive exchange ...
    $action->resume();

$client->run();


```
