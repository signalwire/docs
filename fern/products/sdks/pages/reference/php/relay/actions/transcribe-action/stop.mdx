Stop transcription.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```php {16}
<?php
use SignalWire\Relay\Client;

$client = new Client(
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
    contexts: ["default"],
);

// Handler: $client->onCall()
function handleCall($call)
{
    $call->answer();
    $action = $await $call->transcribe();
}

    // Stop transcription early
    $action->stop();

$client->run();


```
