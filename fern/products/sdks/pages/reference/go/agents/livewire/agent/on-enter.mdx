Lifecycle hook called when the agent enters a session. Override in a subclass to
run custom initialization logic when the agent is first bound to an
[`AgentSession`][agentsession].

## **Parameters**

None.

## **Returns**

`nil`

## **Example**

```go {6}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/server"
)

func main() {
    server := AgentServer()

    // GreeterAgent embeds Agent
        async def on_enter(self):
            if session:
                session.say("Welcome! I just started up.")

    async def entrypoint(ctx: JobContext):
        await ctx.connect()
        agent := GreeterAgent(instructions="You are a friendly greeter.")
        session := AgentSession()
        await session.start(agent, room=ctx.room)

    run_app(server)
}
```
