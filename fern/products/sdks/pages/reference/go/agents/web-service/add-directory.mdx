Mount a new directory at a URL route. A leading `/` is added automatically
if missing. Raises `ValueError` if the directory does not exist or the path
is not a directory.

If the HTTP app is already running, the new directory is mounted immediately.

## **Parameters**

<ParamField path="route" type="string" required={true} toc={true}>
  URL path to mount at (e.g., `"/docs"`). A leading `/` is added automatically
  if missing.
</ParamField>

<ParamField path="directory" type="string" required={true} toc={true}>
  Local directory path to serve.
</ParamField>

## **Returns**

`nil`

## **Example**

```go {4-5}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    web := WebService(port=8002)
    web.AddDirectory("/audio", "./audio_files")
    web.AddDirectory("/images", "./public/images")
    web.start()
}
```
