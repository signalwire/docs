Execute a generic RPC method on a call. This is the low-level interface for
cross-call communication. For common operations, prefer the specific helpers
[`RpcDial()`][rpc-dial],
[`RpcAiMessage()`][rpc-ai-message], and
[`RpcAiUnhold()`][rpc-ai-unhold].

## **Parameters**

<ParamField path="method" type="string" required={true} toc={true}>
  RPC method name (e.g., `"dial"`, `"ai_message"`, `"ai_unhold"`).
</ParamField>

<ParamField path="params" type="*map[string]interface{}" default="None" toc={true}>
  Parameters for the RPC method.
</ParamField>

<ParamField path="call_id" type="*string" default="None" toc={true}>
  Target call ID for the RPC command.
</ParamField>

<ParamField path="node_id" type="*string" default="None" toc={true}>
  Target node ID for the RPC command.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "execute_custom_rpc",
        Description: "Execute a custom RPC command",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult("Executing RPC.")
            .execute_rpc(
                method := "ai_message",
                call_id := "target-call-id",
                params := {"role": "system", "message_text": "Hello from another call."}
            )
        )

    agent.Serve()
}
```
