[functionresult]: /docs/sdks/reference/go/agents/function-result

Stop an active media tap stream.

## **Parameters**

<ParamField path="control_id" type="*string" default="None" toc={true}>
  Identifier of the tap to stop. If not provided, the most recently started
  tap is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "stop_monitoring",
        Description: "Stop monitoring the call",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult("Monitoring stopped.")
            .stop_tap(control_id="supervisor_tap")
        )

    agent.Serve()
}
```
