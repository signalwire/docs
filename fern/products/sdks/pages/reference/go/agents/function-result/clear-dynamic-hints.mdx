Remove all dynamically added speech recognition hints. This clears every hint
that was previously added via [`AddDynamicHints()`][add-dynamic-hints].
Static hints defined on the agent at configuration time are not affected.

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "reset_context",
        Description: "Reset speech recognition context",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult("Context reset.")
            .clear_dynamic_hints()
        )

    agent.Serve()
}
```
