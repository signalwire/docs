Join a SignalWire RELAY room. Rooms enable multi-party communication and
collaboration between participants.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Name of the room to join.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "join_support_room",
        Description: "Join the support room",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult("Connecting to the support room.")
            .join_room(name="support-room-1")
        )

    agent.Serve()
}
```
