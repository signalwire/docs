[functionresult]: /docs/sdks/reference/go/agents/function-result

Put the call on hold. The caller hears hold music until the hold is released
or the timeout expires.

## **Parameters**

<ParamField path="timeout" type="int" default="300" toc={true}>
  Maximum hold duration in seconds. Clamped to the range 0--900 (15 minutes max).
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {7,11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "hold_for_agent",
        Description: "Place the caller on hold",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult("Please hold while I find an available agent.")
            .hold(timeout=60)
        )

    agent.Serve()
}
```
