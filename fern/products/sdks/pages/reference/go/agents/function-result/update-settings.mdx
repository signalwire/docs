[functionresult]: /docs/sdks/reference/go/agents/function-result

Update AI runtime settings dynamically during a call. Changes take effect
for subsequent LLM turns.

## **Parameters**

<ParamField path="settings" type="map[string]interface{}" required={true} toc={true}>
  Dictionary of settings to update. Supported keys:

  | Key | Type | Range |
  |-----|------|-------|
  | `frequency-penalty` | `float` | -2.0 to 2.0 |
  | `presence-penalty` | `float` | -2.0 to 2.0 |
  | `max-tokens` | `int` | 0 to 4096 |
  | `top-p` | `float` | 0.0 to 1.0 |
  | `confidence` | `float` | 0.0 to 1.0 |
  | `barge-confidence` | `float` | 0.0 to 1.0 |
  | `temperature` | `float` | 0.0 to 2.0 (clamped to 1.5) |
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "adjust_for_technical_discussion",
        Description: "Adjust AI settings for technical topics",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult("Adjusting response parameters.")
            .update_settings({
                "temperature": 0.3,
                "confidence": 0.9,
                "barge-confidence": 0.8
            })
        )

    agent.Serve()
}
```
