[functionresult]: /docs/sdks/reference/go/agents/function-result

Set or update key-value pairs in the global session data. Global data persists
for the entire agent session and is accessible in prompt variable expansion
(`${global_data.key}`) and by all SWAIG functions.

## **Parameters**

<ParamField path="data" type="map[string]interface{}" required={true} toc={true}>
  Dictionary of key-value pairs to merge into the global data store. Existing
  keys are overwritten; new keys are added.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "save_customer_info",
        Description: "Save customer information",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult(f"I've noted your information, {args.get('name')}.")
            .update_global_data({
                "customer_id": args[]string{"customer_id"},
                "customer_name": args[]string{"name"},
                "verified": true
            })
        )

    agent.Serve()
}
```
