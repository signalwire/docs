Adjust the speech event timeout. This controls how many milliseconds since the
last speech detection event to wait before finalizing recognition. This timeout
works better than [`SetEndOfSpeechTimeout()`][set-end-of-speech-timeout]
in noisy environments.

## **Parameters**

<ParamField path="milliseconds" type="int" required={true} toc={true}>
  Timeout in milliseconds.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {12}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "configure_timeouts",
        Description: "Configure speech recognition timeouts",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        return (
            FunctionResult()
            .set_end_of_speech_timeout(800)
            .set_speech_event_timeout(5000)
        )

    agent.Serve()
}
```
