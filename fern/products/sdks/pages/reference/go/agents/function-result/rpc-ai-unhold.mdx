Release another call from hold. Typically used after injecting a message into
the held caller's AI agent via [`RpcAiMessage()`][rpc-ai-message].

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  Call ID of the call to release from hold.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```go {13}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "release_caller",
        Description: "Release the caller from hold",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        caller_call_id := args[]string{"original_call_id"}
        return (
            FunctionResult("Returning you to the caller.")
            .rpc_ai_message(caller_call_id, "You can take their message now.")
            .rpc_ai_unhold(caller_call_id)
        )

    agent.Serve()
}
```
