## **has_section**

Check if a section with the given title exists.

### **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title to check.
</ParamField>

### **Returns**

`bool` -- `true` if the section exists, `false` otherwise.

---

## **get_section**

Get a section by title for direct manipulation.

### **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title to look up.
</ParamField>

### **Returns**

`*Section` -- The POM Section object, or `nil` if not found.

## **Example**

```go
package main

import "fmt"

func main() {
    pom := PomBuilder()
    pom.AddSection("Role", body="You are a helpful assistant.")

    if pom.has_section("Role"):
        section := pom.get_section("Role")
        fmt.Println(fmt.Sprintf("Found section: %v", section.title))

    fmt.Println(pom.has_section("Missing"))
}
```
