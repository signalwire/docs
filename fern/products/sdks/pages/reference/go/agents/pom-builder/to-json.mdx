Convert the POM to a JSON string.

## **Returns**

`str` -- JSON representation of the POM.

## **Example**

```go {7}
package main

import "fmt"

func main() {
    pom := PomBuilder()
    pom.AddSection("Role", body="You are a helpful assistant.")
    pom.AddSection("Rules", bullets=[]string{"Be concise", "Be accurate"})

    json_str := pom.to_json()
    fmt.Println(json_str)
}
```
