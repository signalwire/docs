Render the entire POM as a Markdown string. Sections become `##` headings,
subsections become `###` headings, and bullets render as Markdown lists.

## **Returns**

`str` -- The rendered Markdown text.

---

Render the entire POM as an XML string. Useful when the AI benefits from
XML-structured prompts.

## **Returns**

`str` -- The rendered XML text.

## **Example**

```go {7,17}
package main

import "fmt"

func main() {
    pom := PomBuilder()
    pom.AddSection("Role", body="You are a helpful assistant.")
    pom.AddSection("Rules", bullets=[]string{"Be concise", "Be accurate"})

    // Render as Markdown
    md := pom.render_markdown()
    fmt.Println(md)
    // ## Role
    // You are a helpful assistant.
    //
    // ## Rules
    // - Be concise
    // - Be accurate

    // Render as XML
    xml := pom.render_xml()
    fmt.Println(xml)
}
```
