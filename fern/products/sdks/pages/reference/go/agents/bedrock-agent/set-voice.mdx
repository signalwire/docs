Set the Bedrock voice ID after construction.

## **Parameters**

<ParamField path="voice_id" type="string" required={true} toc={true}>
  Bedrock voice identifier (e.g., `"matthew"`, `"joanna"`).
</ParamField>

## **Returns**

`nil`

## **Example**

```go {9}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := BedrockAgent(
        name := "bedrock-assistant",
        route := "/assistant",
        system_prompt := "You are a helpful assistant.",
    )

    agent.SetVoice("joanna")
}
```
