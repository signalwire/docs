[play]: /docs/sdks/reference/go/agents/swml-builder/play

Add a text-to-speech `play` verb using the `say:` URL prefix. This is a convenience
wrapper around [`Play()`][play] that
constructs the `say:` URL automatically.

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  The text to speak.
</ParamField>

<ParamField path="voice" type="*string" toc={true}>
  Voice name for text-to-speech.
</ParamField>

<ParamField path="language" type="*string" toc={true}>
  Language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="gender" type="*string" toc={true}>
  Gender for voice selection.
</ParamField>

<ParamField path="volume" type="*float64" toc={true}>
  Volume adjustment level, from `-40` to `40` dB.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```go {10,12}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/swml"
)

func main() {
    service := SWMLService(name="greeting")
    builder := SWMLBuilder(service)

    doc := (
        builder
        .answer()
        .say("Welcome to our service. Please hold.", voice="rime.spore")
        .sleep(duration=500)
        .say("Connecting you now.", voice="rime.spore")
        .build()
    )
    fmt.Println(doc)
}
```
