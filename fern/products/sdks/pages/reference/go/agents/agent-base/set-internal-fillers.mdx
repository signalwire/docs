Set filler phrases for native SWAIG functions. Internal fillers are phrases the AI
speaks while executing built-in functions like `check_time`, `next_step`, or
`wait_for_user`. They prevent silence during processing and make the conversation
feel more natural.

## **Parameters**

<ParamField path="internal_fillers" type="map[str, dict[str][]str]" required={true} toc={true}>
  Nested dictionary mapping function names to language-specific filler phrases.
  Format: `{"function_name": {"language_code": ["phrase1", "phrase2"]}}`.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```go {5}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")
    agent.set_internal_fillers({
        "next_step": {
            "en-US": []string{"Moving on...", "Great, let's continue..."},
            "es": []string{"Pasando al siguiente paso...", "Continuemos..."}
        },
        "check_time": {
            "en-US": []string{"Let me check the time...", "One moment..."}
        }
    })
    agent.Serve()
}
```
