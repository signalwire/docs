Register signal handlers for `SIGTERM` and `SIGINT` so the agent shuts down cleanly.
This is especially useful in Kubernetes deployments where the pod receives `SIGTERM`
before termination.

## **Parameters**

None.

## **Returns**

`nil`

## **Example**

```go {5}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")
    agent.setup_graceful_shutdown()
    agent.Serve()
}
```
