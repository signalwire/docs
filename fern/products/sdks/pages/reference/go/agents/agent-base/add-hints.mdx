Add multiple speech recognition hints at once.

## **Parameters**

<ParamField path="hints" type="[]string" required={true} toc={true}>
  A list of words or phrases.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```go {5}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")
    agent.AddHints([]string{"SWML", "SWAIG", "FreeSWITCH"})
    agent.Serve()
}
```
