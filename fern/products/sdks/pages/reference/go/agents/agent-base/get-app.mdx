Get the fully initialized HTTP application instance. If the app has not been
created yet, this method initializes it with all routes, middleware, health checks,
and security headers -- the same configuration that `Serve()` would create.

This is primarily used with deployment adapters like Mangum (AWS Lambda) or when
embedding the agent in a larger HTTP application.

## **Returns**

`HTTP` -- The configured HTTP application instance with all agent endpoints
registered.

## **Examples**

### AWS Lambda with Mangum

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("lambda-agent"))
    agent.SetPromptText("You are a helpful assistant.")

    app := agent.get_app()
    handler := Mangum(app)
}
```

### Embed in existing HTTP app

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    main_app := HTTP()
    agent := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/agent"))

    main_app.mount("/ai", agent.get_app())
}
```
