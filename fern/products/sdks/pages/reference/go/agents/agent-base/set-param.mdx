Set a single AI parameter by key.

## **Parameters**

<ParamField path="key" type="string" required={true} toc={true}>
  Parameter name (e.g., `"temperature"`, `"end_of_speech_timeout"`).
</ParamField>

<ParamField path="value" type="interface{}" required={true} toc={true}>
  Parameter value. Type depends on the parameter being set.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```go {5}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")
    agent.SetParam("temperature", 0.5)
    agent.Serve()
}
```
