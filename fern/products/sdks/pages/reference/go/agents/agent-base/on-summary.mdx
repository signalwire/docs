[set-post-prompt-url]: /docs/sdks/reference/go/agents/agent-base/set-post-prompt-url

Callback method invoked when a post-prompt summary is received after a conversation
ends. Override this method in a subclass to process summaries -- for example, saving
them to a CRM, triggering follow-up workflows, or logging call outcomes.

A post-prompt must be configured via
[`SetPostPrompt()`][set-post-prompt]
for summaries to be generated.

<Note>
The default implementation does nothing. You must override it in a subclass or set a
[`SetPostPromptUrl()`][set-post-prompt-url]
to receive summaries at an external endpoint.
</Note>

## **Parameters**

<ParamField path="summary" type="*map[string]interface{}" required={true} toc={true}>
  The summary object generated by the AI based on your post-prompt instructions.
  `nil` if no summary could be extracted from the response.
</ParamField>

<ParamField path="raw_data" type="*map[string]interface{}" default="None" toc={true}>
  The complete raw POST data from the post-prompt request, including metadata
  like `call_id` and the full AI response.
</ParamField>

## **Returns**

`nil`

## **Example**

```go {14}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
    // SupportAgent embeds AgentBase
        func __init__() *swaig.FunctionResult {
            // Base config: name="support", route="/support"
            SetPromptText("You are a helpful customer support agent.")
            set_post_prompt("""
                Summarize this call as JSON:
                - intent: caller's primary intent
                - resolved: yes/no/partial
                - sentiment: positive/neutral/negative
            """)

        func OnSummary(summary any, rawData map[string]any) *swaig.FunctionResult {
            if summary:
                call_id := rawData[]string{"call_id"} if rawData else "unknown"
                fmt.Println(fmt.Sprintf("Call %v summary: %v", call_id, summary))
                // save_to_database(call_id, summary)

    agent := SupportAgent()
    agent.Run()
}
```
