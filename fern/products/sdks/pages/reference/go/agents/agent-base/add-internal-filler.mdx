Add filler phrases for a specific native function and language.

## **Parameters**

<ParamField path="function_name" type="string" required={true} toc={true}>
  Name of the native function (e.g., `"next_step"`, `"check_time"`).
</ParamField>

<ParamField path="language_code" type="string" required={true} toc={true}>
  Language code (e.g., `"en-US"`, `"es"`, `"fr"`).
</ParamField>

<ParamField path="fillers" type="[]string" required={true} toc={true}>
  List of filler phrases for this function and language. The AI randomly selects
  from this list each time the function is invoked.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```go {5,9}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")
    agent.add_internal_filler(
        "next_step", "en-US",
        []string{"Moving to the next step...", "Great, let's continue..."}
    )
    agent.add_internal_filler(
        "next_step", "es",
        []string{"Pasando al siguiente paso...", "Continuemos..."}
    )
    agent.Serve()
}
```
