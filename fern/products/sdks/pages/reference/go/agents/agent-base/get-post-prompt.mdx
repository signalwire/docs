Retrieve the current post-prompt text.

## **Returns**

`*string` -- The post-prompt text, or `nil` if not set.

## **Example**

```go {5}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPostPrompt("Summarize this call as JSON with intent, resolution, and sentiment.")
    post_prompt := agent.get_post_prompt()
    if post_prompt:
        fmt.Println(post_prompt)
}
```
