[set-prompt-text]: /docs/sdks/reference/go/agents/agent-base/set-prompt-text

Retrieve the current prompt. Returns a string if
[`SetPromptText()`][set-prompt-text] was used, or a
POM structure (list of dictionaries) if prompt sections were used. If neither was
explicitly set, returns a default prompt: `"You are {name}, a helpful AI assistant."`.

## **Returns**

`str | list[dict[str, Any]]` -- The current prompt in whatever format was configured.

## **Example**

```go {5}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a customer support agent for Acme Corp.")
    prompt := agent.get_prompt()
    fmt.Println(prompt)
}
```
