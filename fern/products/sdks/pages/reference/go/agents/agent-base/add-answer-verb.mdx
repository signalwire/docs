Configure the answer verb that connects the call. Use this to customize answer
behavior such as setting a maximum call duration.

## **Parameters**

<ParamField path="config" type="*map[string]interface{}" toc={true}>
  Answer verb configuration (e.g., `{"max_duration": 3600}`).
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```go {5}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")
    agent.AddAnswerVerb({"max_duration": 3600})
    agent.Serve()
}
```
