Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`str` -- The agent name.

## **Example**

```go {4}
agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
fmt.Println(agent.get_name())
```
