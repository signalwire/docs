Universal entry point that automatically detects the execution environment and starts
the agent in the appropriate mode. In server environments it calls
[`Serve()`][serve]. In serverless
environments (Lambda, Cloud Functions, Azure Functions, CGI) it delegates to
[`HandleServerlessRequest()`][handle-serverless-request].

<Tip>
Use `Run()` as your default entry point. It makes your agent code portable across
development, Docker, and serverless deployments without changes.
</Tip>

## **Parameters**

<ParamField path="event" type="*interface{}" toc={true}>
  Serverless event object. Pass the Lambda event, Cloud Functions request, or
  Azure Functions HttpRequest here.
</ParamField>

<ParamField path="context" type="*interface{}" toc={true}>
  Serverless context object (Lambda context, etc.).
</ParamField>

<ParamField path="force_mode" type="*string" toc={true}>
  Override automatic environment detection. Valid values:
  - `"server"` -- Force web server mode
  - `"lambda"` -- Force AWS Lambda mode
  - `"cgi"` -- Force CGI mode
  - `"google_cloud_function"` -- Force Google Cloud Functions mode
  - `"azure_function"` -- Force Azure Functions mode
</ParamField>

<ParamField path="host" type="*string" toc={true}>
  Host override for server mode.
</ParamField>

<ParamField path="port" type="*int" toc={true}>
  Port override for server mode.
</ParamField>

## **Returns**

`map[string]interface{}` -- In serverless modes, returns the platform-specific response object.
In server mode, blocks until shutdown and returns `nil`.

## **Examples**

### Standard entry point

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))
    agent.SetPromptText("You are a helpful assistant.")

        agent.Run()
}
```

### AWS Lambda handler

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("lambda-agent"))
    agent.SetPromptText("You are a helpful assistant.")

    func Handler(event any, context any) *swaig.FunctionResult {
        return agent.run(event=event, context=context)
}
```

### Google Cloud Function

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("gcf-agent"))
    agent.SetPromptText("You are a helpful assistant.")

    func Main(request any) *swaig.FunctionResult {
        return agent.run(event=request)
}
```
