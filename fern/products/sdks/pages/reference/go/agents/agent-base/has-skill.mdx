Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skill_name" type="string" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `true` if the skill is loaded, `false` otherwise.

## **Example**

```go {5-6}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/skills"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
    agent.AddSkill("datetime")
    fmt.Println(agent.has_skill("datetime"))
    fmt.Println(agent.has_skill("web_search"))
}
```
