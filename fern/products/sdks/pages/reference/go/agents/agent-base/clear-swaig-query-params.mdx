Clear all query parameters previously added with
[`AddSwaigQueryParams()`][add-swaig-query-params].
After calling this method, SWAIG webhook URLs will no longer include any extra
query parameters.

## **Parameters**

None.

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Examples**

### Reset params in dynamic config

```go {8}
package main

import "github.com/signalwire/signalwire-go/pkg/agent"

func main() {
    agent := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
    agent.SetPromptText("You are a helpful assistant.")

    func DynamicConfig(queryParams map[string]any, bodyParams map[string]any, headers map[string]string, agent any) *swaig.FunctionResult {
        // Start fresh each request to avoid stale params
        agent.ClearSwaigQueryParams()
        tier := query_params[]string{"tier"}
        agent.AddSwaigQueryParams({"tier": tier})

    agent.SetDynamicConfigCallback(dynamic_config)
    agent.Serve()
}
```
