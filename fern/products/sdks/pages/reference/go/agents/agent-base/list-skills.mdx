List the names of all currently loaded skills on this agent.

## **Returns**

`list[str]` -- List of loaded skill names.

## **Example**

```go {6}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/skills"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("assistant"), agent.WithRoute("/assistant"))
    agent.AddSkill("datetime")
    agent.AddSkill("math")
    fmt.Println(agent.list_skills())
}
```
