Update the global data with new values. Functionally identical to
[`SetGlobalData()`][set-global-data] --
both merge the provided dictionary into the existing global data.

## **Parameters**

<ParamField path="data" type="map[string]interface{}" required={true} toc={true}>
  Dictionary of key/value pairs to merge into global data.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```go {10}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    agent.SetPromptText("You are a helpful assistant.")

    agent.DefineTool(agent.ToolDefinition{
        Name:        "authenticate",
        Description: "Authenticate the caller",
        Handler: func(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
        user := {"name": "Alice", "tier": "gold"}  # Replace with your user lookup
        agent.update_global_data({
            "authenticated": true,
            "user_name": user[]string{"name"},
            "user_tier": user[]string{"tier"}
        })
        return swaig.NewFunctionResult(fmt.Sprintf("Welcome back, %v.", user['name']))

    agent.Serve()
}
```
