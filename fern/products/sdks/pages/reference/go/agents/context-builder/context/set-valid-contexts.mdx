[ref-context]: /docs/sdks/reference/go/agents/context-builder/context

Set which contexts the agent can navigate to from this context. Acts as a
context-level default; step-level `SetValidContexts()` can further restrict or
expand navigation for individual steps.

## **Parameters**

<ParamField path="contexts" type="[]string" required={true} toc={true}>
  List of context names that are reachable from this context.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```go {9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/contexts"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))

    contexts := agent.define_contexts()
    default := contexts.add_context("default")
    default.AddStep("greet").set_text("Welcome the caller.")
    support := contexts.add_context("support")
    support.SetValidContexts([]string{"default", "billing"})
    support.AddStep("help").set_text("Help the caller with their issue.")
    billing := contexts.add_context("billing")
    billing.AddStep("invoice").set_text("Help with billing inquiries.")

    agent.Serve()
}
```
