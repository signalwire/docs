[ref-step]: /docs/sdks/reference/go/agents/context-builder/step

Skip waiting for user input after this step completes. The AI proceeds
immediately without waiting for the caller to speak.

## **Parameters**

<ParamField path="skip" type="bool" required={true} toc={true}>
  Whether to skip the user turn.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```go {9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/contexts"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))

    contexts := agent.define_contexts()
    ctx := contexts.add_context("default")
    intro := ctx.add_step("intro")
    intro.SetText("Introduce yourself and immediately begin the intake process.")
    intro.SetSkipUserTurn(true)
    ctx.AddStep("intake").set_text("Collect the caller's information.")

    agent.Serve()
}
```
