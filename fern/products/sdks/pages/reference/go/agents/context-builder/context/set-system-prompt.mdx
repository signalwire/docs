Set a new system prompt that takes effect when this context is entered. Triggers
a context reset. Cannot be combined with `AddSystemSection()` / `AddSystemBullets()`.

## **Parameters**

<ParamField path="system_prompt" type="string" required={true} toc={true}>
  New system prompt for this context.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```go {7}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/contexts"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))

    contexts := agent.define_contexts()
    support := contexts.add_context("default")
    support.SetSystemPrompt("You are a patient technical support engineer.")
    support.AddStep("greet").set_text("Greet the caller and ask about their issue.")

    agent.Serve()
}
```
