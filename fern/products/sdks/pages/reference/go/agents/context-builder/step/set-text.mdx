[ref-step]: /docs/sdks/reference/go/agents/context-builder/step

Set the step's prompt text directly. Cannot be used with `AddSection()` or
`AddBullets()`.

<Warning>
Mixing `SetText()` with `AddSection()` or `AddBullets()` raises `ValueError`.
Use one approach or the other.
</Warning>

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  Plain-text prompt instructions for this step.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```go {8}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/contexts"
)

func main() {
    agent := agent.NewAgentBase(agent.WithName("my-agent"), agent.WithRoute("/agent"))

    contexts := agent.define_contexts()
    ctx := contexts.add_context("default")
    step := ctx.add_step("greeting")
    step.SetText("Welcome the caller and ask how you can help.")

    agent.Serve()
}
```
