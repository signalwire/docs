Return speech recognition hints relevant to this skill. The agent adds these
to the SWML document to improve recognition accuracy.

Override this method to provide skill-specific hints.

## **Returns**

`list[str]` -- Words or phrases to boost (default: empty list).

## **Example**

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/skills"

func main() {
    // WeatherSkill embeds SkillBase
        SKILL_NAME := "weather"
        SKILL_DESCRIPTION := "Provides weather information"

        func GetHints() *swaig.FunctionResult {
            return []string{"weather", "temperature", "forecast", "humidity", "wind speed"}

        func Setup() bool {
            return true

        func RegisterTools() *swaig.FunctionResult {
            pass
}
```
