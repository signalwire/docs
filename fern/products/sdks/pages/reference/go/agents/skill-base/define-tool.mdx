Register a tool with the parent agent, automatically merging any `swaig_fields`
configured for this skill. Skills should use this method instead of calling
`Agent.define_tool()` directly.

## **Parameters**

<ParamField path="**kwargs" type="interface{}" required={true} toc={true}>
  All arguments supported by `agent.define_tool()`: `name`, `description`,
  `parameters`, `handler`, `secure`, `fillers`, `wait_file`, etc. Explicit
  values take precedence over `swaig_fields`.
</ParamField>

## **Returns**

`nil`

## **Example**

```go {12}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/skills"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    // SearchSkill embeds SkillBase
        SKILL_NAME := "search"
        SKILL_DESCRIPTION := "Search the web for information"

        func Setup() bool {
            return true

        func RegisterTools() *swaig.FunctionResult {
            define_tool(
                name := "search_web",
                description := "Search the web for information",
                handler := _handle_search,
                parameters := {
                    "type": "object",
                    "properties": {
                        "query": {"type": "string", "description": "Search query"}
                    },
                    "required": []string{"query"}
                },
                secure := true,
                fillers := {"en-US": []string{"Let me search for that..."}}
            )

        func HandleSearch(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
            query := args[]string{"query"}
            return swaig.NewFunctionResult(fmt.Sprintf("Results for: %v", query))
}
```
