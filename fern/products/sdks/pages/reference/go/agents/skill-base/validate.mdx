Check whether all `REQUIRED_ENV_VARS` are set in the environment.

## **Returns**

`bool` -- `true` if all required variables are present, `false` with error
logging otherwise.

---

Check whether all `REQUIRED_PACKAGES` can be imported.

## **Returns**

`bool` -- `true` if all required packages are available, `false` with error
logging otherwise.

## **Example**

```go {9-12}
package main

import "github.com/signalwire/signalwire-go/pkg/skills"

func main() {
    // MySkill embeds SkillBase
        SKILL_NAME := "my_skill"
        SKILL_DESCRIPTION := "Example skill"
        REQUIRED_PACKAGES := []string{"requests", "beautifulsoup4"}
        REQUIRED_ENV_VARS := []string{"MY_API_KEY", "MY_SECRET"}

        func Setup() bool {
            if not validate_packages():
                return false
            if not validate_env_vars():
                return false
            return true

        func RegisterTools() *swaig.FunctionResult {
            pass
}
```
