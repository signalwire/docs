Return data to merge into the agent's `global_data` dictionary. Override to
provide skill-specific session data that should be available to all SWAIG
function handlers.

## **Returns**

`dict[str, Any]` -- Key-value pairs (default: empty dict).

## **Example**

```go {7}
package main

import "github.com/signalwire/signalwire-go/pkg/skills"

func main() {
    // WeatherSkill embeds SkillBase
        SKILL_NAME := "weather"
        SKILL_DESCRIPTION := "Provides weather information"

        func GetGlobalData() *swaig.FunctionResult {
            return {
                "weather_api_base": "https://api.weatherapi.com/v1",
                "default_units": params[]string{"units"}
            }

        func Setup() bool {
            return true

        func RegisterTools() *swaig.FunctionResult {
            pass
}
```
