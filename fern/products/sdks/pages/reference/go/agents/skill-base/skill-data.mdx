Read this skill instance's namespaced state from `raw_data["global_data"]`.
Each skill instance gets an isolated namespace to prevent collisions between
multiple skills.

## **Parameters**

<ParamField path="raw_data" type="map[string]interface{}" required={true} toc={true}>
  The raw_data dict passed to SWAIG function handlers.
</ParamField>

## **Returns**

`dict[str, Any]` -- The skill's state, or empty dict if not found.

---

Write this skill instance's namespaced state into a FunctionResult via
`UpdateGlobalData()`.

## **Parameters**

<ParamField path="result" type="FunctionResult" required={true} toc={true}>
  The FunctionResult to add the global_data update to.
</ParamField>

<ParamField path="data" type="map[string]interface{}" required={true} toc={true}>
  Skill state dictionary to store under the skill's namespace.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] -- The
result, for method chaining.

## **Example**

```go
package main

import (
    "github.com/signalwire/signalwire-go/pkg/skills"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    // CounterSkill embeds SkillBase
        SKILL_NAME := "counter"
        SKILL_DESCRIPTION := "Tracks call count"

        func HandleIncrement(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
            state := get_skill_data(rawData)
            count := state[]string{"count"} + 1

            result := swaig.NewFunctionResult("f"Count is now {count}")
            update_skill_data(result, {"count": count})
            return result

        func Setup() bool {
            return true

        func RegisterTools() *swaig.FunctionResult {
            define_tool(
                name := "increment",
                description := "Increment the counter",
                handler := _handle_increment,
                parameters := {"type": "object", "properties": {}}
            )
}
```
