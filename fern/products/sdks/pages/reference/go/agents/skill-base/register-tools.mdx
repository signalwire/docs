[ref-skillbase]: /docs/sdks/reference/go/agents/skill-base

Register SWAIG tools with the parent agent. Called after `Setup()` returns `true`.
Use `Define_tool()` to register each tool.

This is an abstract method -- you **must** implement it in every [SkillBase][ref-skillbase] subclass.

## **Returns**

`nil`

## **Example**

```go {11}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/skills"
    "github.com/signalwire/signalwire-go/pkg/swaig"
)

func main() {
    // WeatherSkill embeds SkillBase
        SKILL_NAME := "weather"
        SKILL_DESCRIPTION := "Provides weather information"

        func Setup() bool {
            return true

        func RegisterTools() *swaig.FunctionResult {
            define_tool(
                name := "get_weather",
                description := "Get current weather for a location",
                handler := _handle_weather,
                parameters := {
                    "type": "object",
                    "properties": {
                        "location": {
                            "type": "string",
                            "description": "City name"
                        }
                    },
                    "required": []string{"location"}
                }
            )

        func HandleWeather(args map[string]any, rawData map[string]any) *swaig.FunctionResult {
            location := args[]string{"location"}
            return swaig.NewFunctionResult(fmt.Sprintf("Weather in %v: sunny, 72F", location))
}
```
