Validate that SSL configuration is complete and the certificate files exist.

## **Returns**

`(bool, *string)` -- A `(bool, *string)` pair: `(isValid, errorMessage)`.
Returns `(True, None)` when SSL is disabled or when all required files exist.

## **Example**

```go {4}
package main

import "fmt"

func main() {
    security := SecurityConfig()
    is_valid, error = security.ValidateSslConfig()

    if not is_valid:
        fmt.Println(fmt.Sprintf("SSL configuration error: %v", error))
    else:
        fmt.Println("SSL configuration is valid")
}
```
