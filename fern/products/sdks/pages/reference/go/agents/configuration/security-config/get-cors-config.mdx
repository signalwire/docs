Get CORS configuration suitable for HTTP's `CORSMiddleware`.

## **Returns**

`dict[str, Any]` -- Dictionary with `allow_origins`, `allow_credentials`,
`allow_methods`, and `allow_headers` keys.

## **Example**

```go {6}
package main

import "fmt"

func main() {
    security := SecurityConfig()
    cors_config := security.get_cors_config()

    app := HTTP()
    app.AddMiddleware(CORSMiddleware, **cors_config)
    fmt.Println(cors_config)
    // {'allow_origins': ['*'], 'allow_credentials': true, 'allow_methods': ['*'], 'allow_headers': ['*']}
}
```
