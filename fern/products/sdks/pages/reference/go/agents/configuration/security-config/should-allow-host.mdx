Check if a host is in the allowed hosts list.

## **Parameters**

<ParamField path="host" type="string" required={true} toc={true}>
  The hostname to check.
</ParamField>

## **Returns**

`bool` -- `true` if the host is allowed (or if `allowed_hosts` contains `"*"`).

## **Example**

```go {5-6}
package main

import "fmt"

func main() {
    security := SecurityConfig()
    security.allowed_hosts = []string{"agent.example.com", "api.example.com"}
    fmt.Println(security.should_allow_host("agent.example.com"))
    fmt.Println(security.should_allow_host("evil.example.com"))
}
```
