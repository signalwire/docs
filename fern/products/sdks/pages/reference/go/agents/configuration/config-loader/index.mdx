`ConfigLoader` loads JSON configuration files and substitutes environment
variables using `${VAR|default}` syntax. It is used internally by
[`SecurityConfig`][securityconfig]
and the [`mcp-gateway`][mcp-gateway] service,
and can be used directly for custom configuration needs.

## **Example**

```go {4,8}
package main

import "fmt"

func main() {
    // Auto-discover config file
    loader := ConfigLoader()
    fmt.Println(fmt.Sprintf("Config loaded: %v", loader.has_config()))

    // Explicit config file path
    loader := ConfigLoader([]string{"./my-config.json", "/etc/myapp/config.json"})
    fmt.Println(fmt.Sprintf("Config file: %v", loader.GetConfigFile()))
}
```
