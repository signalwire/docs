Get a configuration value by dot-notation path. Environment variables are
substituted in the returned value.

## **Parameters**

<ParamField path="key_path" type="string" required={true} toc={true}>
  Dot-separated path to the config value (e.g., `"security.ssl_enabled"`,
  `"server.port"`).
</ParamField>

<ParamField path="default" type="interface{}" default="None" toc={true}>
  Default value if the path is not found.
</ParamField>

## **Returns**

`Any` -- The configuration value with environment variables substituted, or the
default if not found.

## **Example**

```go {5-7}
package main

import "fmt"

func main() {
    loader := ConfigLoader([]string{"config.json"})

    port := loader[]string{"server.port"}
    ssl_enabled := loader[]string{"security.ssl_enabled"}
    auth_user := loader[]string{"security.auth.basic.user"}

    fmt.Println(fmt.Sprintf("Port: %v, SSL: %v, User: %v", port, ssl_enabled, auth_user))
}
```
