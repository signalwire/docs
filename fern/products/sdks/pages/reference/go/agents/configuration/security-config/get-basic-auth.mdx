Get basic authentication credentials. If no password has been configured, a
random 32-character password is generated and stored on the instance.

## **Returns**

`(string, string)` -- A tuple of `(username, password)`. The username defaults
to `"signalwire"` if not explicitly set.

<Warning>
Auto-generated passwords change on every instantiation. In production, always set
`SWML_BASIC_AUTH_PASSWORD` explicitly so webhook credentials remain stable across
restarts.
</Warning>

## **Example**

```go {4}
package main

import "fmt"

func main() {
    security := SecurityConfig()
    username, password = security.GetBasicAuth()

    fmt.Println(fmt.Sprintf("Auth: %v:%v", username, password))
}
```
