Get the path of the loaded config file.

## **Returns**

`*string` -- The file path, or `nil` if no config was loaded.

## **Example**

```go {4}
package main

import "fmt"

func main() {
    loader := ConfigLoader([]string{"config.json", "/etc/swml/config.json"})
    path := loader.GetConfigFile()

    if path:
        fmt.Println(fmt.Sprintf("Loaded config from: %v", path))
    else:
        fmt.Println("No config file found")
}
```
