Return all registered agents as a list of `(route, agent)` tuples. Useful for introspection,
logging, or building admin interfaces.

## **Returns**

`[]AgentRegistration` -- A list of agent registrations with route strings
(e.g., `"/sales"`) and the second is the [`AgentBase`][agentbase] instance.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/server"
)

func main() {
    agent1 := agent.NewAgentBase(agent.WithName("sales"), agent.WithRoute("/sales"))
    agent2 := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    server := server.NewAgentServer(server.WithServerPort(3000))
    server.register(agent1)
    server.register(agent2)
    for route, agent in server.get_agents():
        fmt.Println(fmt.Sprintf("%v: %v", route, agent.get_name()))
    // /sales: sales
    // /support: support
}
```
