Register a custom routing callback on every registered agent at the specified path.
When a request arrives at that path on any agent, the callback is invoked to determine
whether to redirect the request to a different agent route.

This is the low-level mechanism that powers [`SetupSipRouting()`][setup-sip-routing].
Use it directly when you need custom routing logic beyond SIP username matching --
for example, routing based on HTTP headers, query parameters, or request body content.

## **Parameters**

<ParamField path="callback_fn" type="func(Request, dict) *string" required={true} toc={true}>
  A function that receives a HTTP `Request` object and the parsed request body `dict`.
  Return a route string (e.g., `"/sales"`) to redirect, or `nil` to let the current
  agent handle the request.
</ParamField>

<ParamField path="path" type="string" required={true} toc={true}>
  The URL path where the callback is triggered (e.g., `"/route"`). Leading slashes are
  added and trailing slashes are stripped automatically.
</ParamField>

## **Returns**

`nil`

## **Example**

```go {24}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/server"
)

func main() {
    func RouteByLanguage(request any, body any) *swaig.FunctionResult {
        """Route to different agents based on Accept-Language header."""
        lang := request.headers[]string{"Accept-Language"}
        if lang.startswith("es"):
            return "/spanish-agent"
        elif lang.startswith("fr"):
            return "/french-agent"
        return nil  # Default agent handles it

    english := agent.NewAgentBase(agent.WithName("english"), agent.WithRoute("/english-agent"))
    english.SetPromptText("You are a helpful assistant. Respond in English.")
    spanish := agent.NewAgentBase(agent.WithName("spanish"), agent.WithRoute("/spanish-agent"))
    spanish.SetPromptText("You are a helpful assistant. Respond in Spanish.")
    french := agent.NewAgentBase(agent.WithName("french"), agent.WithRoute("/french-agent"))
    french.SetPromptText("You are a helpful assistant. Respond in French.")

    server := server.NewAgentServer(server.WithServerPort(3000))
    server.register(english)
    server.register(spanish)
    server.register(french)
    server.RegisterGlobalRoutingCallback(route_by_language, "/route")
    server.Run()
}
```
