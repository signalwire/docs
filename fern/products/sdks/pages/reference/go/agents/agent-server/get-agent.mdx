Retrieve a specific agent by its route. Returns `nil` if no agent is registered at
the given path.

## **Parameters**

<ParamField path="route" type="string" required={true} toc={true}>
  The route to look up (e.g., `"/sales"`). Leading slashes are added and trailing slashes
  are stripped automatically for matching.
</ParamField>

## **Returns**

[`AgentBase`][agentbase] | `nil` -- The agent instance
registered at the route, or `nil` if not found.

## **Example**

```go {8}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/agent"
    "github.com/signalwire/signalwire-go/pkg/server"
)

func main() {
    agent1 := agent.NewAgentBase(agent.WithName("sales"), agent.WithRoute("/sales"))
    agent2 := agent.NewAgentBase(agent.WithName("support"), agent.WithRoute("/support"))
    server := server.NewAgentServer(server.WithServerPort(3000))
    server.register(agent1)
    agent := server.get_agent("/sales")
    if agent:
        fmt.Println(fmt.Sprintf("Found: %v", agent.get_name()))
    else:
        fmt.Println("No agent at /sales")
}
```
