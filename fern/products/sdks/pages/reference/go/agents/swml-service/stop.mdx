---

Stop the web server. Sets the internal running flag to `false`.

## **Returns**

`nil`

## **Example**

```go {6}
package main

import "github.com/signalwire/signalwire-go/pkg/swml"

func main() {
    service := SWMLService(name="my-service")

    // In a signal handler or shutdown hook:
    service.stop()
}
```
