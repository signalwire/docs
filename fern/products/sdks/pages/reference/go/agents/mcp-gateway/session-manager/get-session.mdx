Retrieve an active session by ID. Returns `nil` if the session does not exist,
has expired, or if its underlying process has died. Automatically calls
`Touch()` on the session to reset its expiration timer.

## **Parameters**

<ParamField path="session_id" type="string" required={true} toc={true}>
  The session ID to look up.
</ParamField>

## **Returns**

[`Session`][session] or `nil`

## **Example**

```go {4}
package main

import "fmt"

func main() {
    manager := SessionManager(config)
    session := manager.get_session("call-abc-123")
    if session:
        fmt.Println(fmt.Sprintf("Session active for service: %v", session.service_name))
    else:
        fmt.Println("Session not found or expired")
}
```
