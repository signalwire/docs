Resume paused playback from where it was paused.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```go {20}
package main

import "github.com/signalwire/signalwire-go/pkg/relay"

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        await call.answer()
        action := await call.play(
            [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
            loop := 0,
        )

        await action.pause()
        // ... do something ...
        await action.resume()

    client.Run()
}
```
