Connect the call to an Amazon Bedrock AI agent. Similar to
[`Ai()`][ai] but uses Amazon Bedrock as
the LLM backend.

## **Parameters**

<ParamField path="prompt" type="*interface{}" toc={true}>
  The prompt configuration for the Bedrock agent.
</ParamField>

<ParamField path="SWAIG" type="*dict" toc={true}>
  SWAIG configuration for tool/function definitions.
</ParamField>

<ParamField path="ai_params" type="*dict" toc={true}>
  AI parameters for the Bedrock session.
</ParamField>

<ParamField path="global_data" type="*dict" toc={true}>
  Data accessible to the AI and SWAIG functions.
</ParamField>

<ParamField path="post_prompt" type="*dict" toc={true}>
  Post-prompt configuration.
</ParamField>

<ParamField path="post_prompt_url" type="*string" toc={true}>
  URL to receive the post-prompt result.
</ParamField>

## **Returns**

`dict` -- Server response confirming the Bedrock session.

## **Example**

```go {15}
package main

import "github.com/signalwire/signalwire-go/pkg/relay"

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        await call.answer()

        // Start an Amazon Bedrock AI agent
        result := await call.amazon_bedrock(
            prompt := {"text": "You are a helpful assistant."},
            ai_params := {"barge_confidence": 0.02},
        )

    client.Run()
}
```
