Decline control of an inbound call, returning it to the SignalWire routing
engine. The call is not ended -- it continues to ring and may be delivered to
another RELAY client or routing rule.

<Note>
The method is named `Pass()` with a trailing underscore because `pass` is a
reserved keyword in Go.
</Note>

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the pass operation.

## **Example**

```go {14}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/relay"
)

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        // Only handle calls from a specific context
        if call.context != "sales":
            await call.pass_()
            fmt.Println("Call passed back to routing")
            return

        await call.answer()

    client.Run()
}
```
