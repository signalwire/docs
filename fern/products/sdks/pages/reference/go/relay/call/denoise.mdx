Start noise reduction on the call. Filters background noise from the audio
to improve clarity.

<Info>
Use [`DenoiseStop()`][denoise-stop] to disable noise reduction.
</Info>

<Info>
This method emits [`calling.call.denoise`][calling-call-denoise] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming noise reduction has started.

## **Example**

```go {14}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/relay"
)

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        await call.answer()

        result := await call.denoise()
        fmt.Println(fmt.Sprintf("Noise reduction started: %v", result))

        action := await call.play([{"type": "tts", "text": "Noise reduction is now active."}])
        await action.wait()

    client.Run()
}
```
