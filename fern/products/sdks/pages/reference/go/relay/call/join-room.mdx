Join a video/audio room. Rooms provide multi-party communication with
additional features beyond conferences, including video support.

<Info>
Use [`LeaveRoom()`][leave-room] to leave the room.
</Info>

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Room name to join.
</ParamField>

<ParamField path="status_url" type="*string" toc={true}>
  URL to receive room status webhooks.
</ParamField>

## **Returns**

`dict` -- Server response confirming the join.

## **Example**

```go {15}
package main

import "github.com/signalwire/signalwire-go/pkg/relay"

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        await call.answer()

        // Join a room
        await call.join_room("meeting-room-1")

        // The call is now in the room until leave_room() or hangup

    client.Run()
}
```
