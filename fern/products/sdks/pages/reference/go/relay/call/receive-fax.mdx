Start receiving a fax on the call. Returns a
[`FaxAction`][faxaction] that resolves when the
fax is fully received or an error occurs.

<Info>
This method emits [`calling.call.fax`][calling-call-fax] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="control_id" type="*string" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="on_completed" type="*func(RelayEvent) interface{}" toc={true}>
  Callback invoked when the fax reception completes. The event contains
  the received document URL and page count.
</ParamField>

## **Returns**

[`FaxAction`][faxaction] -- An action handle with
`Stop()` and `Wait()` methods.

## **Example**

```go {14}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/relay"
)

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        await call.answer()

        action := await call.receive_fax()
        event := await action.wait()

        fax_result := event.params[]string{"fax"}
        document_url := fax_result[]string{"document"}
        pages := fax_result[]string{"pages"}
        fmt.Println(fmt.Sprintf("Received fax: %v pages, document: %v", pages, document_url))
        await call.hangup()

    client.Run()
}
```
