Stop noise reduction on the call, restoring the original unfiltered audio.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming noise reduction has stopped.

## **Example**

```go {20}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/relay"
)

func main() {
    client := RelayClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
        contexts := []string{"default"},
    )

    async def handle_call(call):
        await call.answer()

        // Enable noise reduction
        await call.denoise()

        action := await call.play([{"type": "tts", "text": "Noise reduction is now active."}])
        await action.wait()

        result := await call.denoise_stop()
        fmt.Println(fmt.Sprintf("Noise reduction stopped: %v", result))

    client.Run()
}
```
