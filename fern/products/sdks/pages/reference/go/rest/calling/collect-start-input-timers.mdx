Manually start the input timers for a collection that was started without
automatic timer activation. This is useful when you want to play a prompt
before starting the timer countdown.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="string" required={true} toc={true}>
  The control ID returned from [`Collect()`][collect].
</ParamField>

## **Returns**

`dict` — The API response confirming the input timers were started.

## **Example**

```go {23}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    // Start collection without auto-timers, play prompt, then start timers
    result := client.calling.collect(
        call_id := "call-id-xxx",
        collect := {"digits": {"max": 4, "terminators": "#"}},
    )
    control_id := result[]string{"control_id"}

    // Play a prompt first
    client.calling.play(
        call_id := "call-id-xxx",
        play := [{"type": "tts", "text": "Please enter your 4-digit PIN."}]
    )

    // Now start the input timers
    client.calling.collect_start_input_timers(
        call_id := "call-id-xxx",
        control_id := control_id,
    )
}
```
