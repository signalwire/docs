Resume a paused recording.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="string" required={true} toc={true}>
  The control ID returned from [`Record()`][record].
</ParamField>

## **Returns**

`dict` — The API response confirming the recording was resumed.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.calling.record_resume(call_id="call-id-xxx", control_id="ctrl-id")
}
```
