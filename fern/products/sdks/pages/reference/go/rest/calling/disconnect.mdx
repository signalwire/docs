Disconnect (unbridge) a connected call. If two call legs are bridged together,
this separates them without ending either leg.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call to disconnect.
</ParamField>

## **Returns**

`dict` — The API response confirming the disconnect.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.calling.disconnect(call_id="call-id-xxx")
}
```
