Start live transcription on an active call. Unlike standard transcription,
live transcription streams partial results in real time as the caller speaks,
enabling display of in-progress speech.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="language" type="string" toc={true}>
  Language code for transcription (e.g., `"en-US"`).
</ParamField>

## **Returns**

`dict` — The API response confirming live transcription was started.

## **Example**

```go {9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.calling.live_transcribe(
        call_id := "call-id-xxx",
        language := "en-US",
    )
}
```
