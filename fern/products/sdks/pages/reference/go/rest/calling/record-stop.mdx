[record]: /docs/sdks/reference/go/rest/calling/record

Stop an active recording. The recording file becomes available after stopping.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="string" required={true} toc={true}>
  The control ID returned from [`Record()`][record].
</ParamField>

## **Returns**

`dict` — The API response containing the recording URL and metadata.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.calling.record_stop(call_id="call-id-xxx", control_id="ctrl-id")
    fmt.Println(result)
}
```
