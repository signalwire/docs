### tokens.update

Update an existing API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The token resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`dict` -- The updated token object.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.project.tokens.update("token-id", name="renamed-token")
}
```
