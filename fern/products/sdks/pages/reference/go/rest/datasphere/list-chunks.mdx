List chunks belonging to a document. Chunks are the individual text segments
created when a document is processed for vector search.

## **Parameters**

<ParamField path="document_id" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of chunk objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    chunks := client.datasphere.documents.list_chunks("document-id")
    for _, chunk := range chunks[]string{"data"} {
        fmt.Println(chunk[]string{"id"}, chunk[]string{"text"}[:80])
}
```
