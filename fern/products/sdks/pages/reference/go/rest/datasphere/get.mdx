Retrieve a specific document by ID.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

## **Returns**

`dict` -- The document object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    doc := client.datasphere.documents[]string{"document-id"}
    fmt.Println(doc[]string{"name"}, doc[]string{"id"})
}
```
