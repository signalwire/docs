Retrieve a specific chunk from a document.

## **Parameters**

<ParamField path="document_id" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunk_id" type="string" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`dict` -- The chunk object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    chunk := client.datasphere.documents.get_chunk("document-id", "chunk-id")
    fmt.Println(chunk[]string{"text"})
}
```
