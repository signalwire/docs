Perform a semantic search across all documents in the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="query_string" type="string" required={true} toc={true}>
  The natural-language search query.
</ParamField>

<ParamField path="count" type="int" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional search parameters (e.g., `tags`, `document_id` to narrow scope).
</ParamField>

## **Returns**

`dict` -- Search results with matching document chunks ranked by relevance.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    results := client.datasphere.documents.search(
        query_string := "How do I reset my password?",
        count := 5,
    )
    for _, chunk := range results[]string{"data"} {
        fmt.Println(chunk[]string{"text"}, "score:", chunk[]string{"score"})
}
```
