Query fax log entries. Fax logs provide a read-only view of fax activity
in your project.

Access via `client.logs.fax` on a [`RestClient`][restclient] instance.

```go
client, err := rest.NewRestClient("", "", "")
if err != nil {
    log.Fatal(err)
}
```

### list

List fax log entries.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of fax log objects.

---

### get

Retrieve a specific fax log entry.

## **Parameters**

<ParamField path="log_id" type="string" required={true} toc={true}>
  The fax log ID.
</ParamField>

## **Returns**

`dict` -- The fax log object.

## **Examples**

### List fax logs

```go
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    faxes := client.logs.fax.list()
    for _, fax := range faxes[]string{"data"} {
        fmt.Println(fax[]string{"from"}, "->", fax[]string{"to"}, fax[]string{"status"})
}
```

### Get a specific fax log

```go
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    fax := client.logs.fax[]string{"fax-log-id"}
    fmt.Println(fax[]string{"from"}, "->", fax[]string{"to"})
}
```
