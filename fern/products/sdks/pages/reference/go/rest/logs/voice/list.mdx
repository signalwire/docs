List voice call log entries.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of voice log objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    calls := client.logs.voice.list(page_size=20)
    for _, call := range calls[]string{"data"} {
        fmt.Println(call[]string{"from"}, "->", call[]string{"to"}, call[]string{"duration"})
}
```
