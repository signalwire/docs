Query conference log entries. Conference logs provide a read-only view of
conference activity in your project.

Access via `client.logs.conferences` on a [`RestClient`][restclient] instance.

```go
client, err := rest.NewRestClient("", "", "")
if err != nil {
    log.Fatal(err)
}
```

<Note>
Conference logs only support `List()`. There is no `Get()` method for
individual conference entries.
</Note>

### list

List conference log entries.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of conference log objects.

## **Example**

```go
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    confs := client.logs.conferences.list()
    for _, conf := range confs[]string{"data"} {
        fmt.Println(conf[]string{"name"}, conf[]string{"status"})
}
```
