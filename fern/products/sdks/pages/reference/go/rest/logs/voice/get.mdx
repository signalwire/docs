Retrieve a specific voice call log entry.

## **Parameters**

<ParamField path="log_id" type="string" required={true} toc={true}>
  The voice log ID.
</ParamField>

## **Returns**

`dict` -- The voice log object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    call := client.logs.voice[]string{"voice-log-id"}
    fmt.Println(call[]string{"from"}, "->", call[]string{"to"}, call[]string{"duration"})
}
```
