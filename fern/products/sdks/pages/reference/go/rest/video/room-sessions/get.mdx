Retrieve a single room session by its unique identifier.

## **Parameters**

<ParamField path="session_id" type="string" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

## **Returns**

`dict` -- The room session resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    session := client.video.room_sessions[]string{"session-id"}
}
```
