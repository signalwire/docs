List tokens associated with a conference.

## **Parameters**

<ParamField path="conference_id" type="string" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- List of conference tokens.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    tokens := client.video.conferences.list_conference_tokens("conference-id")
}
```
