Retrieve a single room recording by its unique identifier.

## **Parameters**

<ParamField path="recording_id" type="string" required={true} toc={true}>
  The unique identifier of the recording.
</ParamField>

## **Returns**

`dict` -- The room recording resource, including download URL and metadata.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    recording := client.video.room_recordings[]string{"recording-id"}
}
```
