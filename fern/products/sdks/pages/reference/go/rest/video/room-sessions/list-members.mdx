List members (participants) who joined the room during a session.

## **Parameters**

<ParamField path="session_id" type="string" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- Paginated list of session members.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    members := client.video.room_sessions.list_members("session-id")
}
```
