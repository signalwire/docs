Retrieve a single video room by its unique identifier.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

## **Returns**

`dict` -- The video room resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    room := client.video.rooms[]string{"room-id"}
}
```
