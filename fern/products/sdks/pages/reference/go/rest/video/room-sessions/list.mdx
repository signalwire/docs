List video room sessions.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering and pagination.

## **Returns**

`dict` -- Paginated response containing room sessions.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    sessions := client.video.room_sessions.list()
}
```
