Retrieve a specific stream by its unique identifier, regardless of which room
or conference it belongs to.

## **Parameters**

<ParamField path="stream_id" type="string" required={true} toc={true}>
  The unique identifier of the stream.
</ParamField>

## **Returns**

`dict` -- The stream resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    stream := client.video.streams[]string{"stream-id"}
}
```
