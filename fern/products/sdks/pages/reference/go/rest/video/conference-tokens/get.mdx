Retrieve a specific conference token.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The unique identifier of the conference token.
</ParamField>

## **Returns**

`dict` -- The conference token resource.

## **Example**

```go {9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    token := client.video.conference_tokens[]string{"token-id"}
}
```
