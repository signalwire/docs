Retrieve a single video conference by its unique identifier.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

## **Returns**

`dict` -- The conference resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    conf := client.video.conferences[]string{"conference-id"}
}
```
