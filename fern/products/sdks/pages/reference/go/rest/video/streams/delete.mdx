Delete a stream.

## **Parameters**

<ParamField path="stream_id" type="string" required={true} toc={true}>
  The unique identifier of the stream to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.video.streams.delete("stream-id")
}
```
