Retrieve a specific short code.

## **Parameters**

<ParamField path="short_code_id" type="string" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`dict` -- The short code object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    code := client.short_codes[]string{"short-code-id"}
    fmt.Println(code[]string{"short_code"})
}
```
