Retrieve a specific address.

## **Parameters**

<ParamField path="address_id" type="string" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`dict` -- The address object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    addr := client.addresses[]string{"address-id"}
    fmt.Println(addr[]string{"customer_name"}, addr[]string{"street"})
}
```
