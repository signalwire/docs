List phone numbers assigned to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of number objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    numbers := client.registry.campaigns.list_numbers("campaign-id")
    for _, n := range numbers[]string{"data"} {
        fmt.Println(n[]string{"number"}, n[]string{"id"})
}
```
