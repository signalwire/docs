Create a number assignment order to assign phone numbers to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="phone_number_ids" type="[]string" toc={true}>
  List of phone number resource IDs to assign.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional order parameters.
</ParamField>

## **Returns**

`dict` -- The newly created order object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    order := client.registry.campaigns.create_order(
        "campaign-id",
        phone_number_ids := []string{"number-id-1", "number-id-2"},
    )
    fmt.Println("Order ID:", order[]string{"id"})
}
```
