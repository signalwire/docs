Retrieve number assignment orders. Orders track the process of assigning phone
numbers to 10DLC campaigns.

Access via `client.registry.orders` on a [`RestClient`][restclient] instance.

```go
client, err := rest.NewRestClient("", "", "")
if err != nil {
    log.Fatal(err)
}
```

## **get**

Retrieve a specific number assignment order.

### **Parameters**

<ParamField path="order_id" type="string" required={true} toc={true}>
  The order resource ID.
</ParamField>

### **Returns**

`dict` -- The order object.

### **Example**

```go
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    order := client.registry.orders[]string{"order-id"}
    fmt.Println(order[]string{"id"}, order[]string{"status"})
}
```
