Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brand_id" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`dict` -- The brand object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    brand := client.registry.brands[]string{"brand-id"}
    fmt.Println(brand[]string{"display_name"}, brand[]string{"entity_type"})
}
```
