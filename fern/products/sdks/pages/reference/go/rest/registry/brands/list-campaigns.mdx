List messaging campaigns under a brand.

## **Parameters**

<ParamField path="brand_id" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of campaign objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    campaigns := client.registry.brands.list_campaigns("brand-id")
    for _, c := range campaigns[]string{"data"} {
        fmt.Println(c[]string{"description"}, c[]string{"usecase"})
}
```
