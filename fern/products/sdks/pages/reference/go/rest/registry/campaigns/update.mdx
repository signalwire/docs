Update a 10DLC campaign. Uses PUT to replace fields.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `description`, `usecase`).
</ParamField>

## **Returns**

`dict` -- The updated campaign object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    updated := client.registry.campaigns.update(
        "campaign-id",
        description := "Updated campaign description",
    )
    fmt.Println(updated[]string{"description"})
}
```
