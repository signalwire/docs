List registered 10DLC brands.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of brand objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    brands := client.registry.brands.list()
    for _, brand := range brands[]string{"data"} {
        fmt.Println(brand[]string{"display_name"}, brand[]string{"entity_type"})
}
```
