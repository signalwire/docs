Retrieve a specific 10DLC campaign by ID.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

## **Returns**

`dict` -- The campaign object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    campaign := client.registry.campaigns[]string{"campaign-id"}
    fmt.Println(campaign[]string{"description"}, campaign[]string{"usecase"})
}
```
