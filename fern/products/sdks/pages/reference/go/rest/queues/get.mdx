Retrieve a specific queue.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- The queue object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    queue := client.queues[]string{"queue-id"}
    fmt.Println(queue[]string{"name"}, queue[]string{"current_size"})
}
```
