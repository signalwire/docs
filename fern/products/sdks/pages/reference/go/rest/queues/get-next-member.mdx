Retrieve the next member in the queue (the member at the front).

## **Parameters**

<ParamField path="queue_id" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- The next member object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    next_member := client.queues.get_next_member("queue-id")
    fmt.Println("Next caller:", next_member[]string{"call_id"})
}
```
