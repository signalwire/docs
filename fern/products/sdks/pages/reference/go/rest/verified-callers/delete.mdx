Delete a verified caller ID.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.verified_callers.delete("caller-id")
    fmt.Println("Deleted successfully")
}
```
