Update a verified caller ID. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `friendly_name`).
</ParamField>

## **Returns**

`dict` -- The updated caller ID object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    updated := client.verified_callers.update("caller-id", friendly_name="Main Office")
    fmt.Println(fmt.Sprintf("Updated: %v", updated['id']))
}
```
