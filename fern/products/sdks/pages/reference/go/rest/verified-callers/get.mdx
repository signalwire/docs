Retrieve a specific verified caller ID.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`dict` -- The caller ID object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    caller := client.verified_callers[]string{"caller-id"}
    fmt.Println(caller[]string{"phone_number"}, caller[]string{"verification_state"})
}
```
