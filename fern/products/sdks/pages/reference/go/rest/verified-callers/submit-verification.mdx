Submit the verification code received during the verification call or SMS.

## **Parameters**

<ParamField path="caller_id" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="verification_code" type="string" toc={true}>
  The code received during the verification call or SMS.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional verification parameters.
</ParamField>

## **Returns**

`dict` -- The updated caller ID object with verified state.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.verified_callers.submit_verification(
        "caller-id",
        verification_code := "123456",
    )
    fmt.Println("State:", result[]string{"verification_state"})
}
```
