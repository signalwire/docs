List verified caller IDs in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of verified caller objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.verified_callers.list()
    for _, caller := range result[]string{"data"} {
        fmt.Println(caller[]string{"phone_number"}, caller[]string{"verification_state"})
}
```
