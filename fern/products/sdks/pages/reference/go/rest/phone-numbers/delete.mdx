Release (delete) a phone number from the project.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.phone_numbers.delete("phone-number-id")
}
```
