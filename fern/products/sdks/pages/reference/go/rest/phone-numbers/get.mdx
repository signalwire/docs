Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`dict` -- The phone number object.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    number := client.phone_numbers[]string{"phone-number-id"}
    fmt.Println(number[]string{"number"}, number[]string{"name"})
}
```
