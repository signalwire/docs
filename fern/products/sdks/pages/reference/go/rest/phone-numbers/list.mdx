List phone numbers owned by the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `page_size`,
  `page_token`).
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.phone_numbers.list(page_size=10)
    for _, number := range result[]string{"data"} {
        fmt.Println(number[]string{"number"}, number[]string{"name"})
}
```
