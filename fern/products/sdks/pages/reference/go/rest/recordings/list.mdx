List recordings in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of recording objects.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.recordings.list(page_size=20)
    for _, rec := range result[]string{"data"} {
        fmt.Println(rec[]string{"id"}, rec[]string{"duration"}, rec[]string{"state"})
}
```
