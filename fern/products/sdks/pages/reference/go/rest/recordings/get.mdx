Retrieve a specific recording.

## **Parameters**

<ParamField path="recording_id" type="string" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`dict` -- The recording object, including download URL and metadata.

## **Example**

```go {9-10}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    rec := client.recordings[]string{"recording-id"}
    fmt.Println(rec[]string{"duration"}, rec[]string{"url"})
}
```
