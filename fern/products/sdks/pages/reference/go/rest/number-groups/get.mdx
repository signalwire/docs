Retrieve a specific number group by ID.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

## **Returns**

`dict` -- The group object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    group := client.number_groups[]string{"group-id"}
    fmt.Println(group[]string{"name"})
}
```
