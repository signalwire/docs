Update a number group. Uses PUT to replace fields.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`dict` -- The updated group object.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    updated := client.number_groups.update("group-id", name="Support Numbers")
    fmt.Println(updated[]string{"name"})
}
```
