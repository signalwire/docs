Partially update a cXML webhook resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated cXML webhook resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    updated := client.fabric.cxml_webhooks.update("webhook-id", name="updated-name")
    fmt.Println(fmt.Sprintf("Updated: %v", updated['name']))
}
```
