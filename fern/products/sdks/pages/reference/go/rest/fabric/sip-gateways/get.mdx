Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`dict` -- The SIP gateway resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    gateway := client.fabric.sip_gateways[]string{"gateway-id"}
    fmt.Println(fmt.Sprintf("Sip Gateway: %v, ID: %v", gateway['name'], gateway['id']))
}
```
