Retrieve a single FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`dict` -- The FreeSWITCH connector resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    connector := client.fabric.freeswitch_connectors[]string{"connector-id"}
    fmt.Println(fmt.Sprintf("Freeswitch Connector: %v, ID: %v", connector['name'], connector['id']))
}
```
