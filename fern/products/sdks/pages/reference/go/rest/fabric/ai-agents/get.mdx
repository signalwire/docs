Retrieve a single AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`dict` -- The AI agent resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    agent := client.fabric.ai_agents[]string{"agent-id"}
    fmt.Println(fmt.Sprintf("AI Agent: %v, ID: %v", agent['name'], agent['id']))
}
```
