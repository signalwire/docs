Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`dict` -- The Fabric resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    resource := client.fabric.resources[]string{"resource-id"}
    fmt.Println(fmt.Sprintf("%v: %v, ID: %v", resource['type'], resource['name'], resource['id']))
}
```
