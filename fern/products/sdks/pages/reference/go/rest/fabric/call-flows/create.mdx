Create a new call flow resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the call flow.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created call flow resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.fabric.call_flows.create(
        name := "inbound-flow",
    )
    fmt.Println(fmt.Sprintf("Created: %v", result['id']))
}
```
