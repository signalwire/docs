List addresses associated with a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this call flow.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    addresses := client.fabric.call_flows.list_addresses("flow-id")
    for _, addr := range addresses[]string{"data"} {
        fmt.Println(fmt.Sprintf("%v: %v", addr['display_name'], addr['id']))
}
```
