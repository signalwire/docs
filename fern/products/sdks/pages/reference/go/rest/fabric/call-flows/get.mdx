Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`dict` -- The call flow resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    flow := client.fabric.call_flows[]string{"flow-id"}
    fmt.Println(fmt.Sprintf("Call Flow: %v, ID: %v", flow['name'], flow['id']))
}
```
