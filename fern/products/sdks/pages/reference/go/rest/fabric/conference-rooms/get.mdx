Retrieve a single conference room resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`dict` -- The conference room resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    room := client.fabric.conference_rooms[]string{"room-id"}
    fmt.Println(fmt.Sprintf("Conference Room: %v, ID: %v", room['name'], room['id']))
}
```
