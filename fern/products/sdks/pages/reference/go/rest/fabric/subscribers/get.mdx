Retrieve a single subscriber resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`dict` -- The subscriber resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    subscriber := client.fabric.subscribers[]string{"subscriber-id"}
    fmt.Println(fmt.Sprintf("Subscriber: %v, ID: %v", subscriber['name'], subscriber['id']))
}
```
