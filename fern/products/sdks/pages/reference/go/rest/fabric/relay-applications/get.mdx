Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`dict` -- The RELAY application resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    item := client.fabric.relay_applications[]string{"resource-id"}
    fmt.Println(fmt.Sprintf("Relay Application: %v, ID: %v", item['name'], item['id']))
}
```
