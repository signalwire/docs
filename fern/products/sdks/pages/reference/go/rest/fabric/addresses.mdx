Read-only access to fabric addresses. Addresses represent the endpoints (phone numbers,
SIP URIs, domains) that route traffic to fabric resources.

Access via `client.fabric.addresses` on a [`RestClient`][restclient] instance.

### list

List all fabric addresses.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing fabric addresses.

### get

Retrieve a single fabric address.

## **Parameters**

<ParamField path="address_id" type="string" required={true} toc={true}>
  The unique identifier of the address.
</ParamField>

## **Returns**

`dict` -- The fabric address resource.

## **Example**

```go
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    // List all addresses
    addresses := client.fabric.addresses.list()
    for _, addr := range addresses[]string{"data"} {
        fmt.Println(fmt.Sprintf("%v: %v", addr['display_name'], addr['type']))

    // Get a specific address
    address := client.fabric.addresses[]string{"address-id"}
    fmt.Println(fmt.Sprintf("Address: %v", address['display_name']))
}
```
