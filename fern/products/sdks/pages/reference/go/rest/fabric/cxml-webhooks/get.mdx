Retrieve a single cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`dict` -- The cXML webhook resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    webhook := client.fabric.cxml_webhooks[]string{"webhook-id"}
    fmt.Println(fmt.Sprintf("Cxml Webhook: %v, ID: %v", webhook['name'], webhook['id']))
}
```
