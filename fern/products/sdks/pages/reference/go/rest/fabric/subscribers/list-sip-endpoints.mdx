List SIP endpoints associated with a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing SIP endpoints for the subscriber.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    response := client.fabric.subscribers.list_sip_endpoints("subscriber-id")
    for _, ep := range response[]string{"data"} {
        fmt.Println(fmt.Sprintf("%v: %v", ep['username'], ep['id']))
}
```
