Retrieve a single SIP endpoint resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- The SIP endpoint resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    endpoint := client.fabric.sip_endpoints[]string{"endpoint-id"}
    fmt.Println(fmt.Sprintf("Sip Endpoint: %v, ID: %v", endpoint['name'], endpoint['id']))
}
```
