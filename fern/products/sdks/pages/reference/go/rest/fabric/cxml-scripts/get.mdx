Retrieve a single cXML script resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`dict` -- The cXML script resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    item := client.fabric.cxml_scripts[]string{"resource-id"}
    fmt.Println(fmt.Sprintf("Cxml Script: %v, ID: %v", item['name'], item['id']))
}
```
