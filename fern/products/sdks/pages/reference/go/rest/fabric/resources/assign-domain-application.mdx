Assign a domain application to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The assigned domain application.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.fabric.resources.assign_domain_application(
        "resource-id", domain="example.signalwire.com",
    )
    fmt.Println(fmt.Sprintf("Domain application assigned: %v", result['id']))
}
```
