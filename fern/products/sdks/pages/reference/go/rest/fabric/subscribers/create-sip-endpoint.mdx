Create a new SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created SIP endpoint.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    result := client.fabric.subscribers.create_sip_endpoint(
        "subscriber-id", username="alice-sip", password="s3cret",
    )
    fmt.Println(fmt.Sprintf("Created SIP endpoint: %v", result['id']))
}
```
