Retrieve a single SWML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SWML webhook.
</ParamField>

## **Returns**

`dict` -- The SWML webhook resource.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    webhook := client.fabric.swml_webhooks[]string{"webhook-id"}
    fmt.Println(fmt.Sprintf("Swml Webhook: %v, ID: %v", webhook['name'], webhook['id']))
}
```
