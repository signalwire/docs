Partially update a SIP endpoint for a subscriber. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="subscriber_id" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated SIP endpoint.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    updated := client.fabric.subscribers.update_sip_endpoint(
        "subscriber-id", "endpoint-id", username="new-username",
    )
    fmt.Println(fmt.Sprintf("Updated: %v", updated['username']))
}
```
