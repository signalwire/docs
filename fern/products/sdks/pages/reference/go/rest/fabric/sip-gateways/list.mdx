List SIP gateway resources in the project.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing SIP gateway resources.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    response := client.fabric.sip_gateways.list()
    for _, gateway := range response[]string{"data"} {
        fmt.Println(fmt.Sprintf("%v: %v", gateway['name'], gateway['id']))
}
```
