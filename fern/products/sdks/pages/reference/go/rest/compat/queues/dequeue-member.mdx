Dequeue a member from a queue, redirecting the call to a new LAML URL.

## **Parameters**

<ParamField path="queue_sid" type="string" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="call_sid" type="string" required={true} toc={true}>
  The call SID of the queue member to dequeue.
</ParamField>

<ParamField path="Url" type="string" toc={true}>
  The URL that returns LAML instructions for the dequeued call.
</ParamField>

<ParamField path="Method" type="string" toc={true}>
  HTTP method for the URL. Defaults to `"POST"`.
</ParamField>

## **Returns**

`dict` -- The dequeued member resource.

## **Example**

```go {9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.compat.queues.dequeue_member(
        "QU...", "CA...",
        Url := "https://example.com/dequeue"
    )
}
```
