Retrieve a single account by its SID.

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The account SID.
</ParamField>

## **Returns**

`dict` -- The account resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    account := client.compat.accounts[]string{"account-sid"}
}
```
