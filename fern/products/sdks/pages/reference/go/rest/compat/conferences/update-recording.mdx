Update a conference recording (e.g., pause, resume, or stop).

## **Parameters**

<ParamField path="conference_sid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="recording_sid" type="string" required={true} toc={true}>
  The SID of the recording.
</ParamField>

<ParamField path="Status" type="string" toc={true}>
  New recording status.

  - `"paused"` -- pause the recording
  - `"in-progress"` -- resume a paused recording
  - `"stopped"` -- stop the recording permanently
</ParamField>

## **Returns**

`dict` -- The updated recording resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.compat.conferences.update_recording("CF...", "RE...", Status="paused")
}
```
