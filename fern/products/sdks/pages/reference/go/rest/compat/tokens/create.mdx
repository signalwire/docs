Create a new API token.

## **Parameters**

<ParamField path="Ttl" type="int" toc={true}>
  Time-to-live in seconds for the token.
</ParamField>

## **Returns**

`dict` -- The created token resource.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    token := client.compat.tokens.create(Ttl=3600)
}
```
