Initiate a new outbound call.

## **Parameters**

<ParamField path="To" type="string" required={true} toc={true}>
  The destination phone number in E.164 format.
</ParamField>

<ParamField path="From" type="string" required={true} toc={true}>
  The caller ID phone number (must be a number you own).
</ParamField>

<ParamField path="Url" type="string" toc={true}>
  The URL that returns LAML instructions when the call connects.
</ParamField>

<ParamField path="StatusCallback" type="string" toc={true}>
  URL to receive call status webhook events.
</ParamField>

<ParamField path="StatusCallbackEvent" type="[]string" toc={true}>
  Events to send to the status callback.

  - `"initiated"` -- the call request has been created
  - `"ringing"` -- the destination is ringing
  - `"answered"` -- the call has been answered
  - `"completed"` -- the call has ended
</ParamField>

## **Returns**

`dict` -- The created call resource including the call SID.

## **Example**

```go {9}
package main

import (
    "fmt"
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    call := client.compat.calls.create(
        To := "+15559876543",
        From := "+15551234567",
        Url := "https://example.com/voice"
    )
    fmt.Println(call[]string{"sid"})
}
```
