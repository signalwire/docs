Update a LAML bin's content or name. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The LAML bin SID.
</ParamField>

<ParamField path="FriendlyName" type="string" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="string" toc={true}>
  The updated cXML/LaML script content.
</ParamField>

## **Returns**

`dict` -- The updated LAML bin resource.

## **Example**

```go {9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.compat.laml_bins.update(
        "LB...",
        Contents := (
            '<?xml version="1.0" encoding="UTF-8"?>'
            "<Response><Say>Goodbye!</Say></Response>"
        )
    )
}
```
