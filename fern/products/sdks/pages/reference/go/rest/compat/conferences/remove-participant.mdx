Remove a participant from a conference, ending their call leg.

## **Parameters**

<ParamField path="conference_sid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="call_sid" type="string" required={true} toc={true}>
  The call SID of the participant to remove.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```go {9-9}
package main

import (
    "github.com/signalwire/signalwire-go/pkg/rest"
    "log"
)

func main() {
    client := RestClient(
        project := "your-project-id",
        token := "your-api-token",
        host := "your-space.signalwire.com",
    )

    client.compat.conferences.remove_participant("CF...", "CA...")
}
```
