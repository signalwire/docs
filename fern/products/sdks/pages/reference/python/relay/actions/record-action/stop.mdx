max-toc-depth: 3
---

Stop recording immediately. The recording file becomes available.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python
action = await call.record({"direction": "both", "format": "wav"})

# Stop recording early
await action.stop()
```
