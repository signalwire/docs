Returned from [`call.send_fax()`](/docs/sdks/reference/python/relay/call/send-fax) or
[`call.receive_fax()`](/docs/sdks/reference/python/relay/call/receive-fax). Tracks
an active fax send or receive operation. Terminal states: `finished`, `error`.

Inherits all properties and methods from the
[base Action interface](/docs/sdks/reference/python/relay/actions) (`control_id`,
`is_done`, `completed`, `result`, `wait()`).

### stop

Stop the fax operation.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python {11-17}
from signalwire.relay import RelayClient

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()
    action = await call.send_fax(
        document="https://example.com/invoice.pdf",
        identity="+15559876543",
    )

    event = await action.wait()
    fax_info = event.params.get("fax", {})
    print(f"Fax pages: {fax_info.get('pages')}")

client.run()
```
