Returned from [`call.tap()`](/docs/sdks/reference/python/relay/call/tap). Tracks
an active media tap (audio interception) operation. Terminal state: `finished`.

Inherits all properties and methods from the
[base Action interface](/docs/sdks/reference/python/relay/actions) (`control_id`,
`is_done`, `completed`, `result`, `wait()`).

### stop

Stop media interception.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python {11-17}
from signalwire.relay import RelayClient

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()
    action = await call.tap(
        tap={"type": "audio", "params": {"direction": "both"}},
        device={"type": "ws", "params": {"uri": "wss://example.com/tap"}},
    )

    # Stop the tap when done
    await action.stop()

client.run()
```
