Resume paused playback from where it was paused.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python
action = await call.play(
    [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
    loop=0,
)

await action.pause()
# ... do something ...
await action.resume()
```
