Returned from [`call.transcribe()`](/docs/sdks/reference/python/relay/call/transcribe).
Tracks an active transcription operation. Terminal state: `finished`.

Inherits all properties and methods from the
[base Action interface](/docs/sdks/reference/python/relay/actions) (`control_id`,
`is_done`, `completed`, `result`, `wait()`).

### stop

Stop transcription.

## **Returns**

`dict` -- Server acknowledgment.
