Returned from [`call.transcribe()`](/docs/sdks/reference/python/relay/call/transcribe).
Tracks an active transcription operation. Terminal state: `finished`.

Inherits all properties and methods from the
[base Action interface](/docs/sdks/reference/python/relay/actions) (`control_id`,
`is_done`, `completed`, `result`, `wait()`).

### stop

Stop transcription.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python {11-14}
from signalwire.relay import RelayClient

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()
    action = await call.transcribe()

    # Stop transcription early
    await action.stop()

client.run()
```
