Returned from [`call.play()`](/docs/sdks/reference/python/relay/call/play). Tracks
an active audio playback operation. Terminal states: `finished`, `error`.

Inherits all properties and methods from the
[base Action interface](/docs/sdks/reference/python/relay/actions) (`control_id`,
`is_done`, `completed`, `result`, `wait()`).

## **Properties**

No additional properties beyond the [base Action interface](/docs/sdks/reference/python/relay/actions).

## **Example**

```python
action = await call.play(
    [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
    volume=-5.0,
    loop=0,
)

# Pause and resume playback
await action.pause()
await action.resume()

# Adjust volume
await action.volume(10.0)

# Stop playback
await action.stop()
```
