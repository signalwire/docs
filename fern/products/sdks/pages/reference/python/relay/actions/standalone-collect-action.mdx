Returned from [`call.collect()`](/docs/sdks/reference/python/relay/call/collect).
Tracks a standalone input collection (without an accompanying play prompt).
Terminal states: `finished`, `error`, `no_input`, `no_match`.

Inherits all properties and methods from the
[base Action interface](/docs/sdks/reference/python/relay/actions) (`control_id`,
`is_done`, `completed`, `result`, `wait()`).

### stop

Stop the collect operation.

## **Returns**

`dict` -- Server acknowledgment.

### start_input_timers

Manually start the `initial_timeout` timer on the active collect operation.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```python
action = await call.collect(
    speech={"end_silence_timeout": 2.0},
    initial_timeout=10.0,
    start_input_timers=False,
)

# Start timers when ready
await action.start_input_timers()

event = await action.wait()
result = event.params.get("result", {})
```
