Put the call on hold. The remote party hears hold music or silence while the
call is held.

<Info>
Use [`unhold()`](/docs/sdks/reference/python/relay/call/unhold) to release the call from hold.
</Info>

<Info>
This method emits [`calling.call.hold`](/docs/sdks/reference/python/relay/call#events) events. See [Call Events](/docs/sdks/reference/python/relay/call#events) for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the hold operation.

## **Example**

```python
@client.on_call
async def handle_call(call):
    await call.answer()
    await call.play([{"type": "tts", "text": "Please hold while I look that up."}])

    # Put the call on hold
    await call.hold()

    # Do some processing...
    await asyncio.sleep(5)

    # Take the call off hold
    await call.unhold()
```
