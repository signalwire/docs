Leave the current room.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the leave.

## **Example**

```python
@client.on_call
async def handle_call(call):
    await call.answer()

    # Join a room
    await call.join_room("meeting-room-1")

    # Later, leave the room
    await call.leave_room()
    await call.hangup()
```
