Start noise reduction on the call. Filters background noise from the audio
to improve clarity.

<Info>
Use [`denoise_stop()`](/docs/sdks/reference/python/relay/call/denoise-stop) to disable noise reduction.
</Info>

<Info>
This method emits [`calling.call.denoise`](/docs/sdks/reference/python/relay/call#events) events. See [Call Events](/docs/sdks/reference/python/relay/call#events) for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming noise reduction has started.

## **Example**

```python {13-14}
from signalwire.relay import RelayClient

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()

    result = await call.denoise()
    print(f"Noise reduction started: {result}")

    action = await call.play([{"type": "tts", "text": "Noise reduction is now active."}])
    await action.wait()

client.run()
```
