title: "disconnect"

description: "Disconnect (unbridge) a connected call."
max-toc-depth: 3
---

Disconnect a previously bridged call, breaking the connection between the two
call legs. Both parties return to their respective RELAY applications for
further processing. Use this after
[`connect()`](/docs/sdks/reference/python/relay/call/methods/connect) to separate
bridged calls.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the disconnect.

## **Example**

```python
@client.on_call
async def handle_call(call):
    await call.answer()

    # Bridge to an agent
    await call.connect([
        [{"type": "phone", "params": {"to_number": "+15551234567", "from_number": "+15559876543"}}],
    ])

    # Wait for a disconnect event or timeout
    await call.wait_for("calling.call.connect", lambda e: e.params.get("connect_state") == "disconnected")

    # The call is now unbridged, continue with IVR
    await call.play([{"type": "tts", "text": "The other party has disconnected."}])
    await call.hangup()
```
