Release the call from hold, resuming normal audio between the parties.

<Info>
This method emits [`calling.call.hold`](/docs/sdks/reference/python/relay/call#events) events. See [Call Events](/docs/sdks/reference/python/relay/call#events) for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the unhold operation.

## **Example**

```python {20-21}
from signalwire.relay import RelayClient

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()
    await call.play([{"type": "tts", "text": "Please hold while I look that up."}])

    # Put the call on hold
    await call.hold()

    # Do some processing...
    await asyncio.sleep(5)

    result = await call.unhold()
    print(f"Call resumed from hold: {result}")
    await call.play([{"type": "tts", "text": "Thanks for holding. I have your answer."}])

client.run()
```
