Answer an inbound call. This must be called before performing any media
operations on the call. For outbound calls created via
[`client.dial()`](/docs/sdks/reference/python/relay/client/methods/dial),
the call is already answered when the `Call` object is returned.

<Note>
Calling `answer()` on an already-answered call is safe and returns the server
response without error.
</Note>

## **Parameters**

Additional keyword arguments are forwarded to the RELAY `calling.answer` request.

## **Returns**

`dict` -- Server response confirming the answer operation.

## **Example**

```python
@client.on_call
async def handle_call(call):
    await call.answer()
    print(f"Answered call {call.call_id} from {call.device}")
    action = await call.play([{"type": "tts", "text": "Welcome!"}])
    await action.wait()
    await call.hangup()
```
