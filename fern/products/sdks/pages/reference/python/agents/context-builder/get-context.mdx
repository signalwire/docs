Retrieve an existing context by name for inspection or modification.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Name of the context to retrieve.
</ParamField>

## **Returns**

[`Context`](/docs/sdks/reference/python/agents/context-builder/context) if found,
`None` otherwise.

## **Example**

```python
contexts = self.define_contexts()
contexts.add_context("default")

# Later, retrieve and modify
ctx = contexts.get_context("default")
if ctx is not None:
    ctx.add_step("new_step").set_text("Added later.")
```
