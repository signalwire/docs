Set all exit fillers at once. Exit fillers are spoken when the AI navigates away
from this context.

## **Parameters**

<ParamField path="exit_fillers" type="dict[str, list[str]]" required={true} toc={true}>
  Dictionary mapping language codes (or `"default"`) to lists of filler phrases.
</ParamField>

## **Returns**

`Context` -- Self for method chaining.

## **Example**

```python {8-11}
from signalwire_agents import AgentBase

agent = AgentBase(name="my-agent", route="/agent")

contexts = agent.define_contexts()
contexts.add_context("default").add_step("menu").set_text("Ask what the caller needs.")

support = contexts.add_context("support")
support.set_exit_fillers({
    "en-US": ["Thank you for contacting support.", "Glad I could help."],
    "default": ["Thank you."]
})
support.add_step("diagnose").set_text("Understand the customer's issue.")

agent.serve()
```
