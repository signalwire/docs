Stop the agent execution immediately. This halts the AI from processing further
and can be used to interrupt the current speech flow.

## **Returns**

[`FunctionResult`](/docs/sdks/reference/python/agents/function-result) — self, for chaining.

## **Example**

```python
def interrupt_and_restart(self, args, raw_data):
    return (
        FunctionResult()
        .stop()
        .say("Let me start over.")
    )
```
