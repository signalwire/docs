Remove one or more keys from the current function's metadata store.

## **Parameters**

<ParamField path="keys" type="str | list[str]" required={true} toc={true}>
  A single key string or a list of key strings to remove from metadata.
</ParamField>

## **Returns**

[`FunctionResult`](/docs/sdks/reference/python/agents/function-result) — self, for chaining.

## **Example**

```python
def clear_metadata(self, args, raw_data):
    return (
        FunctionResult("Metadata cleared.")
        .remove_metadata(["timestamp", "user_id"])
    )
```
