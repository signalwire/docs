Join a SignalWire RELAY room. Rooms enable multi-party communication and
collaboration between participants.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Name of the room to join.
</ParamField>

## **Returns**

[`FunctionResult`](/docs/sdks/reference/python/agents/function-result) — self, for chaining.

## **Example**

```python
def join_support_room(self, args, raw_data):
    return (
        FunctionResult("Connecting to the support room.")
        .join_room(name="support-room-1")
    )
```
