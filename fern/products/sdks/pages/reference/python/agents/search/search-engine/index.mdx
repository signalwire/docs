`SearchEngine` performs hybrid search over indexed documents by combining vector
similarity search with keyword matching, filename search, and metadata search.
Results from all search strategies are merged and ranked using a max-signal-wins
scoring approach, with diversity penalties to prevent single-file dominance.

The engine supports two storage backends: SQLite (`.swsearch` files) for local
use, and pgvector for PostgreSQL-based deployments.

```python {1}
from signalwire.search import SearchEngine
```

<Warning>
Requires search dependencies. Install with `pip install signalwire[search]`.
</Warning>

## **Properties**

<ParamField path="backend" type="str" toc={true}>
  Storage backend in use. Either `"sqlite"` or `"pgvector"`.
</ParamField>

<ParamField path="config" type="dict[str, str]" toc={true}>
  Index configuration loaded from the backend, including model name, embedding
  dimensions, and creation metadata.
</ParamField>

<ParamField path="embedding_dim" type="int" toc={true}>
  Dimensionality of the embedding vectors in the index. Defaults to `768`.
</ParamField>

<ParamField path="model" type="SentenceTransformer | None" toc={true}>
  Optional sentence transformer model instance passed at construction time.
</ParamField>

## **Constructor**

```python {1-4}
engine = SearchEngine(
    backend="sqlite",
    index_path="knowledge.swsearch",
)
```

<ParamField path="backend" type="str" default="sqlite" toc={true}>
  Storage backend to use. Valid values:
  - `"sqlite"` -- read from a local `.swsearch` file
  - `"pgvector"` -- connect to a PostgreSQL database with the pgvector extension
</ParamField>

<ParamField path="index_path" type="Optional[str]" toc={true}>
  Path to a `.swsearch` file. Required when `backend` is `"sqlite"`.
</ParamField>

<ParamField path="connection_string" type="Optional[str]" toc={true}>
  PostgreSQL connection string. Required when `backend` is `"pgvector"`.
</ParamField>

<ParamField path="collection_name" type="Optional[str]" toc={true}>
  Collection name in the pgvector database. Required when `backend` is `"pgvector"`.
</ParamField>

<ParamField path="model" type="Optional[SentenceTransformer]" toc={true}>
  A pre-loaded sentence transformer model instance.
</ParamField>

#### Example

<CodeBlocks>
<CodeBlock title="SQLite Backend">
```python {3}
from signalwire.search import SearchEngine

engine = SearchEngine(backend="sqlite", index_path="./docs.swsearch")
```
</CodeBlock>
<CodeBlock title="pgvector Backend">
```python {3-7}
from signalwire.search import SearchEngine

engine = SearchEngine(
    backend="pgvector",
    connection_string="postgresql://user:pass@localhost/mydb",
    collection_name="docs",
)
```
</CodeBlock>
</CodeBlocks>
