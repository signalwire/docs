Stop the service and perform cleanup. Call this method to gracefully shut down
the HTTP server and release resources.

## **Parameters**

None.

## **Returns**

`None`

## **Example**

```python {9}
from signalwire.search import SearchService

service = SearchService(indexes={"docs": "./docs.swsearch"})

# In a signal handler or shutdown hook:
import signal

def shutdown(sig, frame):
    service.stop()

signal.signal(signal.SIGTERM, shutdown)
```
