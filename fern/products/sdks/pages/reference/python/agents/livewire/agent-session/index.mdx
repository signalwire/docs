`AgentSession` is the orchestrator that binds an
[`Agent`](/docs/sdks/reference/python/agents/livewire/agent) to the SignalWire platform.
When `start()` is called, it translates the LiveWire agent definition into a SignalWire
[`AgentBase`](/docs/sdks/reference/python/agents/agent-base) instance, mapping
instructions to prompts, tools to SWAIG functions, and timing parameters to
SignalWire AI parameters.

```python {4}
from signalwire.livewire import Agent, AgentSession

agent = Agent(instructions="You are a helpful assistant.")
session = AgentSession()
await session.start(agent)
```

## **Properties**

<ParamField path="userdata" type="Any" toc={true}>
  Arbitrary data attached to the session. Accessible from tool handlers via
  [`RunContext.userdata`](/docs/sdks/reference/python/agents/livewire/run-context).
  Defaults to an empty dict.
</ParamField>

<ParamField path="history" type="list[dict[str, str]]" toc={true}>
  Conversation history as a list of `{"role": ..., "content": ...}` dicts.
</ParamField>

## **Constructor**

```python {1-6}
session = AgentSession(
    llm="gpt-4o",
    allow_interruptions=True,
    min_endpointing_delay=0.5,
    max_endpointing_delay=3.0,
)
```

<ParamField path="llm" type="Any" default="None" toc={true}>
  LLM model identifier. If both the session and the
  [`Agent`](/docs/sdks/reference/python/agents/livewire/agent) specify a model, the
  session value takes precedence.
</ParamField>

<ParamField path="tools" type="Optional[list[Any]]" default="None" toc={true}>
  Additional tools to register alongside the Agent's tools. Both lists are merged
  when the underlying SignalWire agent is built.
</ParamField>

<ParamField path="userdata" type="Any" default="None" toc={true}>
  Initial user data for the session. Defaults to an empty dict when `None`.
</ParamField>

<ParamField path="allow_interruptions" type="bool" default="True" toc={true}>
  Whether the caller can barge in while the agent speaks. When `False`, the SignalWire
  `barge_confidence` parameter is set to `1.0`.
</ParamField>

<ParamField path="min_interruption_duration" type="float" default="0.5" toc={true}>
  Minimum duration (in seconds) of speech needed to trigger an interruption.
  Accepted for API compatibility.
</ParamField>

<ParamField path="min_endpointing_delay" type="float" default="0.5" toc={true}>
  Minimum silence (in seconds) before the platform considers the user's turn
  complete. Mapped to the SignalWire `end_of_speech_timeout` parameter (converted to
  milliseconds).
</ParamField>

<ParamField path="max_endpointing_delay" type="float" default="3.0" toc={true}>
  Maximum silence (in seconds) before the platform stops waiting for more speech.
  Mapped to the SignalWire `attention_timeout` parameter (converted to milliseconds).
</ParamField>

<ParamField path="max_tool_steps" type="int" default="3" toc={true}>
  Maximum number of consecutive tool calls the LLM may make before returning a
  response.
  <Note>No-op. SignalWire's control plane handles tool execution depth automatically.</Note>
</ParamField>

<ParamField path="preemptive_generation" type="bool" default="False" toc={true}>
  Whether to start generating a response before the user finishes speaking.
  Accepted for API compatibility.
</ParamField>

<ParamField path="stt" type="Any" default="None" toc={true}>
  Speech-to-text plugin instance.
  <Note>No-op. SignalWire's control plane handles speech recognition automatically.</Note>
</ParamField>

<ParamField path="tts" type="Any" default="None" toc={true}>
  Text-to-speech plugin instance.
  <Note>No-op. SignalWire's control plane handles text-to-speech automatically.</Note>
</ParamField>

<ParamField path="vad" type="Any" default="None" toc={true}>
  Voice activity detection plugin instance.
  <Note>No-op. SignalWire's control plane handles VAD automatically.</Note>
</ParamField>

<ParamField path="turn_detection" type="Any" default="None" toc={true}>
  Turn detection configuration.
  <Note>No-op. SignalWire's control plane handles turn detection automatically.</Note>
</ParamField>

<ParamField path="mcp_servers" type="Any" default="None" toc={true}>
  MCP server list.
  <Note>Not yet supported in LiveWire. Register tools via [`@function_tool`](/docs/sdks/reference/python/agents/livewire/function-tool) instead.</Note>
</ParamField>
