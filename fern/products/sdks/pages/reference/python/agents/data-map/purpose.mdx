Set the function description shown to the AI when deciding which tool to call.

## **Parameters**

<ParamField path="description" type="str" required={true} toc={true}>
  Human-readable description of what this function does.
</ParamField>

## **Returns**

`DataMap` -- Self for method chaining.

---

Alias for `purpose()`.

## **Parameters**

<ParamField path="description" type="str" required={true} toc={true}>
  Human-readable description of what this function does.
</ParamField>

## **Returns**

`DataMap` -- Self for method chaining.

## **Example**

```python {4}
from signalwire import DataMap

weather = DataMap("get_weather")
weather.purpose("Get the current weather for a given city")

print(weather.to_swaig_function())
# {'function': 'get_weather', 'description': 'Get the current weather for a given city', ...}
```

The `description()` method is an alias for `purpose()`:

```python {4}
from signalwire import DataMap

weather = DataMap("get_weather")
weather.description("Get the current weather for a given city")

print(weather.to_swaig_function())
```
