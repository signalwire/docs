Retrieve a specific conference token.

## **Parameters**

<ParamField path="token_id" type="str" required={true} toc={true}>
  The unique identifier of the conference token.
</ParamField>

## **Returns**

`dict` -- The conference token resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.video.conference_tokens.get("token-id")
```
