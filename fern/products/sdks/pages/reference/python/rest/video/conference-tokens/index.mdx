Manage and reset video conference tokens. Resetting a token invalidates the
current value and generates a replacement.

Access via `client.video.conference_tokens` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

### get

Retrieve a specific conference token.

## **Parameters**

<ParamField path="token_id" type="str" required={true} toc={true}>
  The unique identifier of the conference token.
</ParamField>

## **Returns**

`dict` -- The conference token resource.

---

Reset a conference token, invalidating the current token and generating a new one.

## **Parameters**

<ParamField path="token_id" type="str" required={true} toc={true}>
  The unique identifier of the conference token to reset.
</ParamField>

## **Returns**

`dict` -- The reset result, including the new token.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()

# Get a conference token
token = client.video.conference_tokens.get("token-id")

# Reset a conference token (invalidates the old one)
result = client.video.conference_tokens.reset("token-id")
```
