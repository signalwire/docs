List active streams for a conference.

## **Parameters**

<ParamField path="conference_id" type="str" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- List of streams in the conference.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
streams = client.video.conferences.list_streams("conference-id")
```
