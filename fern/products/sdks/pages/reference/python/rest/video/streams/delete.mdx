Delete a stream.

## **Parameters**

<ParamField path="stream_id" type="str" required={true} toc={true}>
  The unique identifier of the stream to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.video.streams.delete("stream-id")
```
