Delete a stream.

## **Parameters**

<ParamField path="stream_id" type="str" required={true} toc={true}>
  The unique identifier of the stream to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.video.streams.delete("stream-id")
```
