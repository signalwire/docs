Create a new stream for a video room, enabling external streaming output
(e.g., RTMP to a streaming platform).

## **Parameters**

<ParamField path="room_id" type="str" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

<Markdown src="/snippets/sdks/rest/video/streams/_create-stream-params.mdx" />

## **Returns**

`dict` -- The created stream resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

stream = client.video.rooms.create_stream(
    "room-id",
    url="rtmp://live.example.com/stream-key"
)
```
