Query video room sessions and their associated events, members, and recordings.
Sessions represent the period during which a room is active with participants.

Access via `client.video.room_sessions` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

sessions = client.video.room_sessions.list()
```

## Methods

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/video/room-sessions/list) | List video room sessions. |
| [`get`](/docs/sdks/reference/python/rest/video/room-sessions/get) | Retrieve a single room session. |
| [`list_events`](/docs/sdks/reference/python/rest/video/room-sessions/list-events) | List events from a session. |
| [`list_members`](/docs/sdks/reference/python/rest/video/room-sessions/list-members) | List members from a session. |
| [`list_recordings`](/docs/sdks/reference/python/rest/video/room-sessions/list-recordings) | List recordings from a session. |
