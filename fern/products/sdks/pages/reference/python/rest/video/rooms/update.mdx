Replace a video room. Uses PUT for full replacement, so all fields must be
provided in the request body.

<Note>
This method performs a full replacement (PUT), not a partial update (PATCH).
Include all fields you want to keep in the request body.
</Note>

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

<Markdown src="/snippets/sdks/rest/video/rooms/_update-params.mdx" />

## **Returns**

`dict` -- The updated video room.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

room = client.video.rooms.update("room-id", display_name="standup-v2", max_members=20)
```
