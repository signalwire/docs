Retrieve a single room recording by its unique identifier.

## **Parameters**

<ParamField path="recording_id" type="str" required={true} toc={true}>
  The unique identifier of the recording.
</ParamField>

## **Returns**

`dict` -- The room recording resource, including download URL and metadata.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recording = client.video.room_recordings.get("recording-id")
```
