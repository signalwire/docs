Manage video room recordings. Supports listing, retrieval, deletion, and
querying recording-level events.

Access via `client.video.room_recordings` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recordings = client.video.room_recordings.list()
```

## Methods

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/video/room-recordings/list) | List video room recordings. |
| [`get`](/docs/sdks/reference/python/rest/video/room-recordings/get) | Retrieve a single recording. |
| [`delete`](/docs/sdks/reference/python/rest/video/room-recordings/delete) | Delete a recording. |
| [`list_events`](/docs/sdks/reference/python/rest/video/room-recordings/list-events) | List events for a recording. |
