List video room recordings.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering and pagination.

## **Returns**

`dict` -- Paginated response containing room recordings.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
recordings = client.video.room_recordings.list()
```
