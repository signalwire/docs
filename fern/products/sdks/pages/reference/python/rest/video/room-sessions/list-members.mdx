List members (participants) who joined the room during a session.

<EndpointSchemaSnippet endpoint="GET /api/video/room_sessions/{id}/members" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_sessions/{id}/members" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

members = client.video.room_sessions.list_members("session-id")
```
