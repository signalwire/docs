Manage video conferences with full CRUD operations, token listing, and stream
management. Uses PUT for updates (full replacement).

Access via `client.video.conferences` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

conferences = client.video.conferences.list()
```

## Methods

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/video/conferences/list) | List video conferences. |
| [`create`](/docs/sdks/reference/python/rest/video/conferences/create) | Create a new conference. |
| [`get`](/docs/sdks/reference/python/rest/video/conferences/get) | Retrieve a single conference. |
| [`update`](/docs/sdks/reference/python/rest/video/conferences/update) | Replace a conference (PUT). |
| [`delete`](/docs/sdks/reference/python/rest/video/conferences/delete) | Delete a conference. |
| [`list_conference_tokens`](/docs/sdks/reference/python/rest/video/conferences/list-conference-tokens) | List tokens for a conference. |
| [`list_streams`](/docs/sdks/reference/python/rest/video/conferences/list-streams) | List active streams for a conference. |
| [`create_stream`](/docs/sdks/reference/python/rest/video/conferences/create-stream) | Create a new stream for a conference. |
