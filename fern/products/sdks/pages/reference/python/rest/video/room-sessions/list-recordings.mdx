List recordings created during a room session.

## **Parameters**

<ParamField path="session_id" type="str" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- Paginated list of session recordings.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recordings = client.video.room_sessions.list_recordings("session-id")
```
