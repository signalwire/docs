Top-level stream management for retrieving, updating, and deleting individual
streams regardless of which room or conference they belong to.

Access via `client.video.streams` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
stream = client.video.streams.get("stream-id")
```

## Methods

| Method | Description |
|---|---|
| [`get`](/docs/sdks/reference/python/rest/video/streams/get) | Retrieve a specific stream. |
| [`update`](/docs/sdks/reference/python/rest/video/streams/update) | Update a stream (PUT). |
| [`delete`](/docs/sdks/reference/python/rest/video/streams/delete) | Delete a stream. |
