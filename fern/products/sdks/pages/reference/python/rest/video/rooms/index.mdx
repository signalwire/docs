Manage video rooms with full CRUD operations and stream management. Uses PUT
for updates (full replacement).

Access via `client.video.rooms` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
rooms = client.video.rooms.list()
```

## Methods

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/video/rooms/list) | List video rooms in the project. |
| [`create`](/docs/sdks/reference/python/rest/video/rooms/create) | Create a new video room. |
| [`get`](/docs/sdks/reference/python/rest/video/rooms/get) | Retrieve a single video room. |
| [`update`](/docs/sdks/reference/python/rest/video/rooms/update) | Replace a video room (PUT). |
| [`delete`](/docs/sdks/reference/python/rest/video/rooms/delete) | Delete a video room. |
| [`list_streams`](/docs/sdks/reference/python/rest/video/rooms/list-streams) | List active streams for a room. |
| [`create_stream`](/docs/sdks/reference/python/rest/video/rooms/create-stream) | Create a new stream for a room. |
