List video conferences in the project.

## **Parameters**

<ParamField path="starts_after" type="str" toc={true}>
  Filter conferences starting after this date (ISO 8601).
</ParamField>

<ParamField path="ends_before" type="str" toc={true}>
  Filter conferences ending before this date (ISO 8601).
</ParamField>

<ParamField path="include_active_session" type="bool" toc={true}>
  Include active session details in the response.
</ParamField>

<ParamField path="page_number" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="page_size" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing video conferences.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

conferences = client.video.conferences.list()
```
