Verify an MFA code submitted by the user.

## **Parameters**

<ParamField path="request_id" type="str" required={true} toc={true}>
  The MFA request ID returned from `sms()` or `call()`.
</ParamField>

<ParamField path="token" type="str" required={true} toc={true}>
  The verification code entered by the user.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`dict` -- Verification result indicating whether the code is valid.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.mfa.verify(request_id, token="123456")
if result.get("success"):
    print("Verification successful")
else:
    print("Invalid code")
```
