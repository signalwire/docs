Send and verify multi-factor authentication (MFA) codes via SMS or voice call.
The typical flow is: request a code with `sms()` or `call()`, then confirm it
with `verify()`.

Access via `client.mfa` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

request = client.mfa.sms(to="+15551234567", from_="+15559876543")
print(request["id"])
```
