Send an MFA verification code via SMS.

## **Parameters**

<ParamField path="to" type="str" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from_" type="str" required={true} toc={true}>
  The sender phone number in E.164 format. Must be a number in your project.
</ParamField>

<ParamField path="message" type="str" required={true} toc={true}>
  The message body. Include `{code}` as a placeholder for the generated code
  (e.g., `"Your verification code is: {code}"`).
</ParamField>

<ParamField path="token_length" type="int" toc={true}>
  Length of the generated code.
</ParamField>

<ParamField path="max_attempts" type="int" toc={true}>
  Maximum number of verification attempts allowed.
</ParamField>

<ParamField path="valid_for" type="int" toc={true}>
  How long the code remains valid, in seconds.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`dict` -- Response containing the `id` of the MFA request (used in `verify()`).

## **Example**

```python {9-15}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.mfa.sms(
    to="+15559876543",
    from_="+15551234567",
    message="Your code is: {code}",
)
request_id = result["id"]
print("MFA request:", request_id)
```
