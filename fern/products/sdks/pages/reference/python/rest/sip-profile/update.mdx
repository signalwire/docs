Update the project SIP profile. Uses PUT for full replacement.

## **Parameters**

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update on the SIP profile (e.g., `username`, `password`,
  `ciphers`, `codecs`).
</ParamField>

## **Returns**

`dict` -- The updated SIP profile object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.sip_profile.update(username="my-sip-user")
print("Updated username:", updated.get("username"))
```
