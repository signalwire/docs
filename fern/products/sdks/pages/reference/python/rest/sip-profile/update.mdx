Update the project SIP profile. Uses PUT for full replacement.

<EndpointSchemaSnippet endpoint="PUT /api/relay/rest/sip_profile" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/relay/rest/sip_profile" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.sip_profile.update(username="my-sip-user")
print("Updated username:", updated.get("username"))
```
