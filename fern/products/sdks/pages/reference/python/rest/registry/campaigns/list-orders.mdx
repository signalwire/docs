List number assignment orders for a campaign.

## **Parameters**

<ParamField path="campaign_id" type="str" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of order objects.

## **Example**

```python {9-11}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

orders = client.registry.campaigns.list_orders("campaign-id")
for order in orders.get("data", []):
    print(order.get("id"), order.get("status"))
```
