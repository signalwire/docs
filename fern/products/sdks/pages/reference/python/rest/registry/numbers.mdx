Manage phone number assignments to 10DLC campaigns. Use this sub-resource to
remove a number from its campaign assignment.

Access via `client.registry.numbers` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
```

### delete

Remove a phone number from its campaign assignment.

## **Parameters**

<ParamField path="number_id" type="str" required={true} toc={true}>
  The number assignment resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python
client.registry.numbers.delete("number-assignment-id")
```
