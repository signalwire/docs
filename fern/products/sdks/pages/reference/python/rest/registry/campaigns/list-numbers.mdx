List phone numbers assigned to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="str" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of number objects.

## **Example**

```python
numbers = client.registry.campaigns.list_numbers("campaign-id")
for n in numbers.get("data", []):
    print(n.get("number"), n.get("id"))
```
