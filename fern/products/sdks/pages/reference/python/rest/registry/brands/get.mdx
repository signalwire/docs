Retrieve a specific 10DLC brand by ID.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/brands/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/brands/{id}" />

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

brand = client.registry.brands.get("brand-id")
print(brand.get("display_name"), brand.get("entity_type"))
```
