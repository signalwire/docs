Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brand_id" type="str" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`dict` -- The brand object.

## **Example**

```python
brand = client.registry.brands.get("brand-id")
print(brand.get("display_name"), brand.get("entity_type"))
```
