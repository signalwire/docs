Manage 10DLC messaging campaigns. Campaigns define the use case and messaging
behavior for a set of phone numbers under a registered brand.

Access via `client.registry.campaigns` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
```
