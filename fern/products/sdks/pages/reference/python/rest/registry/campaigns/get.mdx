Retrieve a specific 10DLC campaign by ID.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}" />

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

campaign = client.registry.campaigns.get("campaign-id")
print(campaign.get("description"), campaign.get("usecase"))
```
