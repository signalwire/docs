Retrieve a specific 10DLC campaign by ID.

## **Parameters**

<ParamField path="campaign_id" type="str" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

## **Returns**

`dict` -- The campaign object.

## **Example**

```python
campaign = client.registry.campaigns.get("campaign-id")
print(campaign.get("description"), campaign.get("usecase"))
```
