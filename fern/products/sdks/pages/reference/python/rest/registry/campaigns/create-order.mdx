Create a number assignment order to assign phone numbers to a campaign.

<EndpointSchemaSnippet endpoint="POST /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/relay/rest/registry/beta/campaigns/{id}/orders" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

order = client.registry.campaigns.create_order(
    "campaign-id",
    phone_numbers=["+15125551234"],
)
print("Order ID:", order.get("id"))
```
