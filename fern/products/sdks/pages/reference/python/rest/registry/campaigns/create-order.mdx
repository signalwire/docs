Create a number assignment order to assign phone numbers to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="str" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="phone_numbers" type="list[str]" toc={true}>
  List of E.164 phone number strings to assign to the campaign.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional order parameters.
</ParamField>

## **Returns**

`dict` -- The newly created order object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

order = client.registry.campaigns.create_order(
    "campaign-id",
    phone_numbers=["+15125551234"],
)
print("Order ID:", order.get("id"))
```
