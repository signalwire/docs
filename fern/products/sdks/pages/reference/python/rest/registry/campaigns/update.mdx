Update a 10DLC campaign. Uses PUT to replace fields.

## **Parameters**

<ParamField path="campaign_id" type="str" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `description`, `usecase`).
</ParamField>

## **Returns**

`dict` -- The updated campaign object.

## **Example**

```python
updated = client.registry.campaigns.update(
    "campaign-id",
    description="Updated campaign description",
)
print(updated.get("description"))
```
