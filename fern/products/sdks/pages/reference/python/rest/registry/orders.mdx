Retrieve number assignment orders. Orders track the process of assigning phone
numbers to 10DLC campaigns.

Access via `client.registry.orders` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
```

### get

Retrieve a specific number assignment order.

## **Parameters**

<ParamField path="order_id" type="str" required={true} toc={true}>
  The order resource ID.
</ParamField>

## **Returns**

`dict` -- The order object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

order = client.registry.orders.get("order-id")
print(order.get("id"), order.get("status"))
```
