List registered 10DLC brands.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of brand objects.

## **Example**

```python {9-11}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

brands = client.registry.brands.list()
for brand in brands.get("data", []):
    print(brand.get("display_name"), brand.get("entity_type"))
```
