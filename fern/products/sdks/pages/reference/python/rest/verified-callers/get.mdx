Retrieve a specific verified caller ID.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`dict` -- The caller ID object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

caller = client.verified_callers.get("caller-id")
print(caller.get("phone_number"), caller.get("verification_state"))
```
