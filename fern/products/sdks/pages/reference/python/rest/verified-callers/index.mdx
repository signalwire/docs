Manage verified caller IDs for your SignalWire project. Before using a phone
number as a caller ID for outbound calls, it must be verified. This resource
provides full CRUD (with PUT for updates) plus a two-step verification flow.

Access via `client.verified_callers` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
```
