Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="number" type="str" required={true} toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="name" type="str" toc={true}>
  A friendly name for the caller ID (max 200 characters).
</ParamField>

<ParamField path="extension" type="str" toc={true}>
  The extension to dial after the verification call connects.
</ParamField>

## **Returns**

`dict` -- The newly created caller ID object with its verification state.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

caller = client.verified_callers.create(number="+15551234567")
print("Verification state:", caller.get("verification_state"))
```
