Retrieve a specific short code.

## **Parameters**

<ParamField path="short_code_id" type="str" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`dict` -- The short code object.

## **Example**

```python
code = client.short_codes.get("short-code-id")
print(code.get("short_code"))
```
