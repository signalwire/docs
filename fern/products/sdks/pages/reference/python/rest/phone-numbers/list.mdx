List phone numbers owned by the project.

## **Parameters**

<ParamField path="filter_name" type="str" toc={true}>
  Filter by phone number name.
</ParamField>

<ParamField path="filter_number" type="str" toc={true}>
  Filter by phone number.
</ParamField>

<ParamField path="page_number" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="page_size" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.phone_numbers.list(page_size=10)
for number in result.get("data", []):
    print(number["number"], number.get("name"))
```
