Search for available phone numbers that can be purchased.

## **Parameters**

<ParamField path="areacode" type="str" toc={true}>
  Filter by area code (e.g., `"512"`).
</ParamField>

<ParamField path="number_type" type="str" toc={true}>
  Filter by number type (e.g., `"local"`, `"toll-free"`).
</ParamField>

<ParamField path="starts_with" type="str" toc={true}>
  Filter numbers starting with this prefix.
</ParamField>

<ParamField path="contains" type="str" toc={true}>
  Filter numbers containing this pattern.
</ParamField>

<ParamField path="ends_with" type="str" toc={true}>
  Filter numbers ending with this suffix.
</ParamField>

<ParamField path="max_results" type="int" default="50" toc={true}>
  Maximum number of results to return (max 100).
</ParamField>

<ParamField path="region" type="str" toc={true}>
  Filter by region or state.
</ParamField>

<ParamField path="city" type="str" toc={true}>
  Filter by city.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of available number objects.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

available = client.phone_numbers.search(areacode="503", max_results=5)
for number in available.get("data", []):
    print(number["number"], number.get("region"))
```
