Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`dict` -- The phone number object.

## **Example**

```python
number = client.phone_numbers.get("phone-number-id")
print(number["number"], number.get("name"))
```
