Create a new regulatory address.

## **Parameters**

<ParamField path="customer_name" type="str" toc={true}>
  The customer or business name for the address.
</ParamField>

<ParamField path="street" type="str" toc={true}>
  Street address.
</ParamField>

<ParamField path="city" type="str" toc={true}>
  City name.
</ParamField>

<ParamField path="state" type="str" toc={true}>
  State or province.
</ParamField>

<ParamField path="postal_code" type="str" toc={true}>
  Postal or ZIP code.
</ParamField>

<ParamField path="country" type="str" toc={true}>
  ISO country code (e.g., `"US"`).
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional address fields as needed by the API.
</ParamField>

## **Returns**

`dict` -- The newly created address object.

## **Example**

```python {9-17}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addr = client.addresses.create(
    customer_name="Acme Corp",
    street="123 Main St",
    city="Austin",
    state="TX",
    postal_code="78701",
    country="US",
)
print("Created address:", addr.get("id"))
```
