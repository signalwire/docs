Create a new regulatory address.

## **Parameters**

<ParamField path="label" type="str" required={true} toc={true}>
  A human-readable label for the address.
</ParamField>

<ParamField path="country" type="str" required={true} toc={true}>
  Country code (e.g., `"US"`).
</ParamField>

<ParamField path="first_name" type="str" required={true} toc={true}>
  First name of the address contact.
</ParamField>

<ParamField path="last_name" type="str" required={true} toc={true}>
  Last name of the address contact.
</ParamField>

<ParamField path="street_number" type="str" required={true} toc={true}>
  Street number.
</ParamField>

<ParamField path="street_name" type="str" required={true} toc={true}>
  Street name.
</ParamField>

<ParamField path="city" type="str" required={true} toc={true}>
  City name.
</ParamField>

<ParamField path="state" type="str" required={true} toc={true}>
  State, province, or region.
</ParamField>

<ParamField path="postal_code" type="str" required={true} toc={true}>
  Postal or ZIP code.
</ParamField>

<ParamField path="address_type" type="str" toc={true}>
  Type of address (e.g., `"regulatory"`).
</ParamField>

<ParamField path="address_number" type="str" toc={true}>
  Apartment, suite, or unit number.
</ParamField>

## **Returns**

`dict` -- The newly created address object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addr = client.addresses.create(
    label="HQ Address",
    country="US",
    first_name="Jane",
    last_name="Doe",
    street_number="123",
    street_name="Main St",
    city="Austin",
    state="TX",
    postal_code="78701",
)
print("Created address:", addr.get("id"))
```
