Retrieve a specific address.

## **Parameters**

<ParamField path="address_id" type="str" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`dict` -- The address object.

## **Example**

```python
addr = client.addresses.get("address-id")
print(addr.get("customer_name"), addr.get("street"))
```
