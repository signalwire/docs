Start collecting user input on an active call. Supports DTMF digit collection
and speech recognition. Returns a `control_id` for managing the collection.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call to collect input on.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_collect-params.mdx" />

## **Returns**

`dict` — The API response containing the `control_id`.

## **Examples**

### Collect DTMF Digits

```python {10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

# Collect DTMF digits
result = client.calling.collect(
    call_id="call-id-xxx",
    digits={"max": 4, "terminators": "#", "digit_timeout": 5.0},
    initial_timeout=10.0,
)
```

### Collect Speech

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.collect(
    call_id="call-id-xxx",
    speech={
        "end_silence_timeout": 1.0,
        "language": "en-US",
    },
)
```

### Collect Both

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.collect(
    call_id="call-id-xxx",
    digits={"max": 1, "terminators": "#"},
    speech={"end_silence_timeout": 2.0},
    initial_timeout=15.0,
)
```
