Start a detector on an active call. Detectors can identify answering machines,
fax tones, or DTMF digits. Returns a `control_id` for managing the detector.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call to run detection on.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_detect-params.mdx" />

## **Returns**

`dict` — The API response containing the `control_id`.

## **Examples**

### Answering Machine Detection

```python {10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

# Answering machine detection
result = client.calling.detect(
    call_id="call-id-xxx",
    detect={
        "type": "machine",
        "params": {
            "initial_timeout": 4.5,
            "end_silence_timeout": 1.0,
        }
    },
    timeout=30.0,
)
```

### Fax Detection

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.detect(
    call_id="call-id-xxx",
    detect={"type": "fax", "params": {"tone": "CED"}}
)
```

### Digit Detection

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.detect(
    call_id="call-id-xxx",
    detect={"type": "digit", "params": {"digits": "0123456789#*"}}

```
