Stop an active input collection.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID returned from [`collect()`](/docs/sdks/reference/python/rest/calling/collect).
</ParamField>

## **Returns**

`dict` — The API response confirming collection was stopped.

## **Example**

```python
client.calling.collect_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
