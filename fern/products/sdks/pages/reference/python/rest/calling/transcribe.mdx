Start transcribing speech on an active call. Transcription results are delivered
via events. Returns a `control_id` for stopping the transcription.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call to transcribe.
</ParamField>

<ParamField path="language" type="str" toc={true}>
  Language code for transcription (e.g., `"en-US"`, `"es-ES"`).
</ParamField>

## **Returns**

`dict` — The API response containing the `control_id`.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()

result = client.calling.transcribe(
    call_id="call-id-xxx",
    language="en-US",
)
control_id = result.get("control_id")
```
