Start streaming audio from an active call to a WebSocket endpoint. This is
commonly used for real-time speech processing, analytics, or archival.
Returns a `control_id` for stopping the stream.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call to stream audio from.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_stream-params.mdx" />

## **Returns**

`dict` — The API response containing the `control_id`.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.stream(
    call_id="call-id-xxx",
    url="wss://example.com/audio-stream",
)
control_id = result.get("control_id")
```
