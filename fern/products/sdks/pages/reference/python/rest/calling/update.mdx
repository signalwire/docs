Update parameters on an active call, such as metadata or call settings.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call to update. Passed as a keyword argument.
</ParamField>

<ParamField path="metadata" type="dict" toc={true}>
  Key-value metadata to attach to the call.
</ParamField>

## **Returns**

`dict` — The API response confirming the update.

## **Example**

```python
client.calling.update(call_id="call-id-xxx", metadata={"department": "sales"})
```
