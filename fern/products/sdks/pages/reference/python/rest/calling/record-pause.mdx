Pause an active recording. The recording can be resumed later with
[`record_resume()`](/docs/sdks/reference/python/rest/calling/record-resume).

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID returned from [`record()`](/docs/sdks/reference/python/rest/calling/record).
</ParamField>

## **Returns**

`dict` — The API response confirming the recording was paused.

## **Example**

```python
client.calling.record_pause(call_id="call-id-xxx", control_id="ctrl-id")
```
