Resume a paused playback.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID returned from [`play()`](/docs/sdks/reference/python/rest/calling/play).
</ParamField>

## **Returns**

`dict` — The API response confirming playback was resumed.

## **Example**

```python
client.calling.play_resume(call_id="call-id-xxx", control_id="ctrl-id")
```
