Stop an active audio tap.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID returned from [`tap()`](/docs/sdks/reference/python/rest/calling/tap).
</ParamField>

## **Returns**

`dict` — The API response confirming the tap was stopped.

## **Example**

```python
client.calling.tap_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
