Stop an active audio stream.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID returned from [`stream()`](/docs/sdks/reference/python/rest/calling/stream).
</ParamField>

## **Returns**

`dict` — The API response confirming the stream was stopped.

## **Example**

```python
client.calling.stream_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
