Stop an in-progress fax send operation on an active call.

<Note>
This method stops a fax send that is already in progress. To initiate fax sending,
use the RELAY client's [`send_fax()`](/docs/sdks/reference/python/relay/call/methods/send-fax) method.
</Note>

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call with an active fax send operation.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID of the fax send operation to stop.
</ParamField>

## **Returns**

`dict` — The API response confirming the fax send was stopped.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()

client.calling.send_fax_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
