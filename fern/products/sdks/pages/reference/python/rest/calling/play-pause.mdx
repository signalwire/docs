Pause an active playback.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

## **Returns**

`dict` — The API response confirming playback was paused.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.play_pause(call_id="call-id-xxx", control_id="ctrl-id")
```
