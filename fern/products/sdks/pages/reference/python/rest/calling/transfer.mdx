Transfer an active call to a new destination. The current call leg is replaced
by a new connection to the specified target.

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call to transfer.
</ParamField>

<ParamField path="dest" type="dict" required={true} toc={true}>
  The transfer destination. Contains `type` (e.g., `"phone"`, `"sip"`) and
  type-specific parameters such as `to_number`.
</ParamField>

## **Returns**

`dict` — The API response confirming the transfer was initiated.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.transfer(
    call_id="call-id-xxx",
    dest={"type": "phone", "params": {"to_number": "+15559876543"}}
)
```
