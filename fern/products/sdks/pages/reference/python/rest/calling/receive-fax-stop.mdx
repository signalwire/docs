Stop an in-progress fax receive operation on an active call.

<Note>
This method stops a fax receive that is already in progress. To initiate fax receiving,
use the RELAY client's [`receive_fax()`][receive-fax] method.
</Note>

## **Parameters**

<ParamField path="call_id" type="str" required={true} toc={true}>
  The ID of the call with an active fax receive operation.
</ParamField>

<ParamField path="control_id" type="str" required={true} toc={true}>
  The control ID of the fax receive operation to stop.
</ParamField>

## **Returns**

`dict` — The API response confirming the fax receive was stopped.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.receive_fax_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
