Retrieve a specific voice call log entry.

## **Parameters**

<ParamField path="log_id" type="str" required={true} toc={true}>
  The voice log ID.
</ParamField>

## **Returns**

`dict` -- The voice log object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

call = client.logs.voice.get("voice-log-id")
print(call.get("from"), "->", call.get("to"), call.get("duration"))
```
