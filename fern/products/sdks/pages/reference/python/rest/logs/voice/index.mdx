Query voice call log entries. Voice logs provide a read-only view of call
activity in your project, including per-call event timelines.

Access via `client.logs.voice` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {8-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

logs = client.logs.voice.list(page_size=5)
for log in logs.get("data", []):
    print(log["id"], log.get("from"), "->", log.get("to"))
```
