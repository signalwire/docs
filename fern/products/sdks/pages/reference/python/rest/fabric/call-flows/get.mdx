Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`dict` -- The call flow resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

flow = client.fabric.call_flows.get("flow-id")
print(f"Call Flow: {flow['display_name']}, ID: {flow['id']}")
```
