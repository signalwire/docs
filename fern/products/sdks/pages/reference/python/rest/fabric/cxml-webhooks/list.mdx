List cXML webhook resources in the project.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing cXML webhook resources.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
items = client.fabric.cxml_webhooks.list()
```
