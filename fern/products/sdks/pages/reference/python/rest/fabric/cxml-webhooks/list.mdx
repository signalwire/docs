List cXML webhook resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing cXML webhook resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.cxml_webhooks.list()
for webhook in response.get("data", []):
    print(f"{webhook['display_name']}: {webhook['id']}")
```
