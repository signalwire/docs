Replace a SIP endpoint resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated SIP endpoint resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.fabric.sip_endpoints.update("resource-id", name="updated-name")
```
