Create a new AI agent resource.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Display name of the AI agent.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created AI agent resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
agent = client.fabric.ai_agents.create(name="support-bot", type="ai_agent")
```
