Create a new SWML webhook resource.

## **Parameters**

<ParamField path="primary_request_url" type="str" required={true} toc={true}>
  The primary URL for the SWML webhook.
</ParamField>

<ParamField path="name" type="str" toc={true}>
  Display name of the SWML webhook.
</ParamField>

## **Returns**

`dict` -- The created SWML webhook resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.swml_webhooks.create(
    name="my-webhook", primary_request_url="https://example.com/swml",
)
print(f"Created: {result['id']}")
```
