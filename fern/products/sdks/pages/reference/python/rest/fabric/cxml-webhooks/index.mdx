Manage cXML webhook resources. Standard CRUD with PATCH updates and address listing.

Access via `client.fabric.cxml_webhooks` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
items = client.fabric.cxml_webhooks.list()
```
