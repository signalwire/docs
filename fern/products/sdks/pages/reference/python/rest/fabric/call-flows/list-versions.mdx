List all versions of a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`dict` -- Paginated response containing call flow versions.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.call_flows.list_versions("call-flow-id")
for version in response.get("data", []):
    print(f"Version {version['version']}: {version['id']}")
```
