Delete an AI agent resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/ai_agents/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.ai_agents.delete("agent-id")
print("AI Agent deleted")
```
