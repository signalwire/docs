Create an invite token for a subscriber.

## **Parameters**

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created invite token.

## **Example**

```python {8-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.fabric.tokens.create_invite_token(subscriber_id="subscriber-id")
print(f"Invite token: {token['token']}")
```
