Create an invite token for a subscriber.

## **Parameters**

<ParamField path="address_id" type="str" required={true} toc={true}>
  The UUID of the address to create the invite for.
</ParamField>

## **Returns**

`dict` -- The created invite token.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.fabric.tokens.create_invite_token(address_id="address-uuid")
print(f"Invite token: {token['token']}")
```
