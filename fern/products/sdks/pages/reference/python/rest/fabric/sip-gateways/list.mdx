List SIP gateway resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing SIP gateway resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.sip_gateways.list()
for gateway in response.get("data", []):
    print(f"{gateway['display_name']}: {gateway['id']}")
```
