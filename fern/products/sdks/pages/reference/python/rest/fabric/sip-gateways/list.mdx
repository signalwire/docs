List SIP gateway resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/sip_gateways" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/sip_gateways" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.sip_gateways.list()
for gateway in response.get("data", []):
    print(f"{gateway['display_name']}: {gateway['id']}")
```
