Manage cXML script resources. Standard CRUD with PUT updates and address listing.

Access via `client.fabric.cxml_scripts` on a [`RestClient`][restclient] instance.

```python {8}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

items = client.fabric.cxml_scripts.list()
```
