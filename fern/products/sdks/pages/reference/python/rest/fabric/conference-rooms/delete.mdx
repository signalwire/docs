Delete a conference room resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/conference_rooms/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.conference_rooms.delete("room-id")
print("Conference Room deleted")
```
