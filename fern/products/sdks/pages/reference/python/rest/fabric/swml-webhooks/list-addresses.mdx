List addresses associated with a SWML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SWML webhook.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
addresses = client.fabric.swml_webhooks.list_addresses("resource-id")
```
