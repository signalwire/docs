Partially update a SIP gateway resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated SIP gateway resource.

## **Example**

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.sip_gateways.update("gateway-id", name="updated-name")
print(f"Updated: {updated['name']}")
```
