Create a new SIP endpoint resource.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Display name of the SIP endpoint.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created SIP endpoint resource.

## **Example**

```python {8-12}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.sip_endpoints.create(
    name="my-endpoint",
)
print(f"Created: {result['id']}")
```
