Create a new RELAY application resource.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Display name of the RELAY application.
</ParamField>

<ParamField path="topic" type="str" required={true} toc={true}>
  The RELAY topic this application listens on.
</ParamField>

## **Returns**

`dict` -- The created RELAY application resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.relay_applications.create(
    name="my-item",
)
print(f"Created: {result['id']}")
```
