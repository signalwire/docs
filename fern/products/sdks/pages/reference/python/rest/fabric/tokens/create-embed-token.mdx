Create an embed authentication token.

## **Parameters**

<ParamField path="token" type="str" required={true} toc={true}>
  The Click-to-Call token.
</ParamField>

## **Returns**

`dict` -- The created embed token.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.fabric.tokens.create_embed_token(token="click-to-call-token")
print(f"Embed token: {token['token']}")
```
