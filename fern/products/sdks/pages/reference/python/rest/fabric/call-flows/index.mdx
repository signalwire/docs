Manage call flow resources with version management and deployment. Extends standard CRUD
(PUT updates) with methods to list versions and deploy specific versions.

Access via `client.fabric.call_flows` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
flows = client.fabric.call_flows.list()
```
