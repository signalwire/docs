Create a new SIP gateway resource.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Display name of the SIP gateway.
</ParamField>

<ParamField path="uri" type="str" required={true} toc={true}>
  The SIP URI of the gateway.
</ParamField>

<ParamField path="encryption" type="str" required={true} toc={true}>
  Encryption mode for the gateway.
</ParamField>

<ParamField path="ciphers" type="list[str]" required={true} toc={true}>
  Supported cipher suites.
</ParamField>

<ParamField path="codecs" type="list[str]" required={true} toc={true}>
  Supported audio codecs.
</ParamField>

## **Returns**

`dict` -- The created SIP gateway resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.sip_gateways.create(
    name="my-gateway", uri="sip:gw.example.com", encryption="required",
    ciphers=["AEAD_AES_256_GCM_8"], codecs=["OPUS", "PCMU"],
)
print(f"Created: {result['id']}")
```
