Create a new SIP gateway resource.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/sip_gateways" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/sip_gateways" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.sip_gateways.create(
    name="my-gateway", uri="sip:gw.example.com", encryption="required",
    ciphers=["AEAD_AES_256_GCM_8"], codecs=["OPUS", "PCMU"],
)
print(f"Created: {result['id']}")
```
