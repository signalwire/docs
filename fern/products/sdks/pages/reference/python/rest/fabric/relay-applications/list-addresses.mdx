List addresses associated with a RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
addresses = client.fabric.relay_applications.list_addresses("resource-id")
```
