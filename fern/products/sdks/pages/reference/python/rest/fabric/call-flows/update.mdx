Replace a call flow resource. Uses PUT for full replacement.

<EndpointSchemaSnippet endpoint="PUT /api/fabric/resources/call_flows/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/fabric/resources/call_flows/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.call_flows.update("flow-id", title="Updated IVR Flow")
print(f"Updated: {updated['display_name']}")
```
