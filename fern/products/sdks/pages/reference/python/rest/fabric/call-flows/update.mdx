Replace a call flow resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<Markdown src="/snippets/sdks/rest/fabric/call-flows/_update-params.mdx" />

## **Returns**

`dict` -- The updated call flow resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.call_flows.update("flow-id", title="Updated IVR Flow")
print(f"Updated: {updated['display_name']}")
```
