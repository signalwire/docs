List addresses associated with a cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addresses = client.fabric.cxml_applications.list_addresses("app-id")
for addr in addresses.get("data", []):
    print(f"{addr['display_name']}: {addr['id']}")
```
