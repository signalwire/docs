List addresses associated with a FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
addresses = client.fabric.freeswitch_connectors.list_addresses("resource-id")
```
