Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`dict` -- The RELAY application resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
item = client.fabric.relay_applications.get("resource-id")
```
