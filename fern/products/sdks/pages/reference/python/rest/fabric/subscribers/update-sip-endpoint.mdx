Partially update a SIP endpoint for a subscriber. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="subscriber_id" type="str" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="str" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

<ParamField path="username" type="str" toc={true}>
  SIP username for the endpoint.
</ParamField>

<ParamField path="password" type="str" toc={true}>
  SIP password for the endpoint.
</ParamField>

<ParamField path="caller_id" type="str" toc={true}>
  Caller ID for outbound calls from this endpoint.
</ParamField>

<ParamField path="send_as" type="str" toc={true}>
  The identity to present when making outbound calls.
</ParamField>

<ParamField path="ciphers" type="list[str]" toc={true}>
  List of allowed cipher suites for the endpoint.
</ParamField>

<ParamField path="codecs" type="list[str]" toc={true}>
  List of allowed audio codecs for the endpoint.
</ParamField>

<ParamField path="encryption" type="str" toc={true}>
  Encryption mode for the SIP endpoint. `"required"`, `"optional"`, or `"default"`.
</ParamField>

## **Returns**

`dict` -- The updated SIP endpoint.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.subscribers.update_sip_endpoint(
    "subscriber-id", "endpoint-id", username="new-username",
)
print(f"Updated: {updated['username']}")
```
