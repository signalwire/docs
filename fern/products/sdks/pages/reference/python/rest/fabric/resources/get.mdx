Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`dict` -- The Fabric resource.

## **Example**

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

resource = client.fabric.resources.get("resource-id")
print(f"{resource['type']}: {resource['name']}, ID: {resource['id']}")
```
