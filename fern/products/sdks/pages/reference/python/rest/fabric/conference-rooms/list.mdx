List conference room resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/conference_rooms" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/conference_rooms" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.conference_rooms.list()
for room in response.get("data", []):
    print(f"{room['display_name']}: {room['id']}")
```
