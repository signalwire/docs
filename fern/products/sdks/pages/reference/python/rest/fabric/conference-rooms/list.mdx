List conference room resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing conference room resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.conference_rooms.list()
for room in response.get("data", []):
    print(f"{room['display_name']}: {room['id']}")
```
