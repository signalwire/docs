List conference room resources in the project.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing conference room resources.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
items = client.fabric.conference_rooms.list()
```
