Retrieve a single cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`dict` -- The cXML webhook resource.

## **Example**

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

webhook = client.fabric.cxml_webhooks.get("webhook-id")
print(f"Cxml Webhook: {webhook['name']}, ID: {webhook['id']}")
```
