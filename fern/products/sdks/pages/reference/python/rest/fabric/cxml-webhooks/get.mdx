Retrieve a single cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`dict` -- The cXML webhook resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
item = client.fabric.cxml_webhooks.get("resource-id")
```
