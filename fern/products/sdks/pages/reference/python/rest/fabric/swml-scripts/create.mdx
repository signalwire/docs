Create a new SWML script resource.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Display name of the SWML script.
</ParamField>

<ParamField path="contents" type="str" required={true} toc={true}>
  The SWML script contents.
</ParamField>

## **Returns**

`dict` -- The created SWML script resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.swml_scripts.create(
    name="my-item",
)
print(f"Created: {result['id']}")
```
