List SIP endpoint resources in the project.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing SIP endpoint resources.

## **Example**

```python {8-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.sip_endpoints.list()
for endpoint in response.get("data", []):
    print(f"{endpoint['name']}: {endpoint['id']}")
```
