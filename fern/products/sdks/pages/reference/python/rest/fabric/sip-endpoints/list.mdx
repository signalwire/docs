List SIP endpoint resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/sip_endpoints" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/sip_endpoints" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.sip_endpoints.list()
for endpoint in response.get("data", []):
    print(f"{endpoint['name']}: {endpoint['id']}")
```
