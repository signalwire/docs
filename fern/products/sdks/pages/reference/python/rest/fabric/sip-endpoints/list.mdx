List SIP endpoint resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing SIP endpoint resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.sip_endpoints.list()
for endpoint in response.get("data", []):
    print(f"{endpoint['name']}: {endpoint['id']}")
```
