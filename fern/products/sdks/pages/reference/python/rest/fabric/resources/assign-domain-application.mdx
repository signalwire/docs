Assign a domain application to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

<ParamField path="domain_application_id" type="str" required={true} toc={true}>
  The UUID of the domain application to assign.
</ParamField>

## **Returns**

`dict` -- The assigned domain application.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.resources.assign_domain_application(
    "resource-id", domain_application_id="app-uuid",
)
print(f"Domain application assigned: {result['id']}")
```
