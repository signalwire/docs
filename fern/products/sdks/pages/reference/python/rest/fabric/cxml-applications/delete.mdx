Delete a cXML application resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/cxml_applications/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.cxml_applications.delete("app-id")
print("cXML application deleted")
```
