List SIP endpoints associated with a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="str" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing SIP endpoints for the subscriber.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.subscribers.list_sip_endpoints("subscriber-id")
for ep in response.get("data", []):
    print(f"{ep['username']}: {ep['id']}")
```
