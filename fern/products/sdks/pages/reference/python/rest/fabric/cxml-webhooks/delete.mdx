Delete a cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the cXML webhook to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success (HTTP 204).

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.fabric.cxml_webhooks.delete("resource-id")
```
