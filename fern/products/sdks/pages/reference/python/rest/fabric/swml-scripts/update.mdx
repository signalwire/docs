Replace a SWML script resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated SWML script resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.fabric.swml_scripts.update("resource-id", name="updated-name")
```
