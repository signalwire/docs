Assign a phone route to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

<ParamField path="phone_route_id" type="str" required={true} toc={true}>
  The UUID of the phone route to assign.
</ParamField>

<ParamField path="handler" type="str" required={true} toc={true}>
  The handler type: `"calling"` or `"messaging"`.
</ParamField>

## **Returns**

`dict` -- The assigned phone route.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.resources.assign_phone_route(
    "resource-id", phone_route_id="route-uuid", handler="calling",
)
print(f"Phone route assigned: {result['id']}")
```
