Retrieve a single conference room resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`dict` -- The conference room resource.

## **Example**

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

room = client.fabric.conference_rooms.get("room-id")
print(f"Conference Room: {room['name']}, ID: {room['id']}")
```
