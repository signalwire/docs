Replace a RELAY application resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated RELAY application resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.fabric.relay_applications.update("resource-id", name="updated-name")
```
