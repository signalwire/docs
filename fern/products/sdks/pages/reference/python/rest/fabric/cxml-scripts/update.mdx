Replace a cXML script resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated cXML script resource.

## **Example**

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.cxml_scripts.update("resource-id", name="updated-name")
print(f"Updated: {updated['name']}")
```
