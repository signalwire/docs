Retrieve a single FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`dict` -- The FreeSWITCH connector resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

connector = client.fabric.freeswitch_connectors.get("connector-id")
print(f"Freeswitch Connector: {connector['name']}, ID: {connector['id']}")
```
