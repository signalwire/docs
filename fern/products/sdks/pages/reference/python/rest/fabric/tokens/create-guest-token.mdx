Create a guest authentication token.

## **Parameters**

<ParamField path="allowed_addresses" type="list[str]" required={true} toc={true}>
  List of address UUIDs the guest is allowed to access (max 10).
</ParamField>

## **Returns**

`dict` -- The created guest token.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.fabric.tokens.create_guest_token(allowed_addresses=["address-uuid"])
print(f"Guest token: {token['token']}")
```
