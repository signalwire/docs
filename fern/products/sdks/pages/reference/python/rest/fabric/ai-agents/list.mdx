List AI agent resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing AI agent resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.ai_agents.list()
for agent in response.get("data", []):
    print(f"{agent['display_name']}: {agent['id']}")
```
