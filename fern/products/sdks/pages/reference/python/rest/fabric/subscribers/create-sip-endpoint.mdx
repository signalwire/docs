Create a new SIP endpoint for a subscriber.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/subscribers/{fabric_subscriber_id}/sip_endpoints" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/subscribers/{fabric_subscriber_id}/sip_endpoints" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.subscribers.create_sip_endpoint(
    "subscriber-id", username="alice-sip", password="s3cret",
)
print(f"Created SIP endpoint: {result['id']}")
```
