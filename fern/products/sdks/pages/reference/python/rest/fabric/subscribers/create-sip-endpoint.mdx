Create a new SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="str" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="username" type="str" required={true} toc={true}>
  SIP username for the endpoint.
</ParamField>

<ParamField path="password" type="str" required={true} toc={true}>
  SIP password for the endpoint.
</ParamField>

<ParamField path="encryption" type="str" default="default" toc={true}>
  Encryption mode for the SIP endpoint. `"required"`, `"optional"`, or `"default"`.
</ParamField>

<ParamField path="caller_id" type="str" toc={true}>
  Caller ID for outbound calls from this endpoint.
</ParamField>

<ParamField path="send_as" type="str" toc={true}>
  The identity to present when making outbound calls.
</ParamField>

<ParamField path="ciphers" type="list[str]" toc={true}>
  List of allowed cipher suites for the endpoint.
</ParamField>

<ParamField path="codecs" type="list[str]" toc={true}>
  List of allowed audio codecs for the endpoint.
</ParamField>

## **Returns**

`dict` -- The created SIP endpoint.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.subscribers.create_sip_endpoint(
    "subscriber-id", username="alice-sip", password="s3cret",
)
print(f"Created SIP endpoint: {result['id']}")
```
