Create a new SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="str" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created SIP endpoint.

## **Example**

```python {8-12}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.subscribers.create_sip_endpoint(
    "subscriber-id", username="alice-sip", password="s3cret",
)
print(f"Created SIP endpoint: {result['id']}")
```
