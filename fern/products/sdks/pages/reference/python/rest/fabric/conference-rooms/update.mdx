Replace a conference room resource. Uses PUT for full replacement.

<EndpointSchemaSnippet endpoint="PUT /api/fabric/resources/conference_rooms/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/fabric/resources/conference_rooms/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.conference_rooms.update(
    "room-id", name="updated-name",
    enable_room_previews=True, sync_audio_video=True,
)
print(f"Updated: {updated['display_name']}")
```
