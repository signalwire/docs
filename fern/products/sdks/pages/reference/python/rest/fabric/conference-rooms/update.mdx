Replace a conference room resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

<Markdown src="/snippets/sdks/rest/fabric/conference-rooms/_update-params.mdx" />

## **Returns**

`dict` -- The updated conference room resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.conference_rooms.update(
    "room-id", name="updated-name",
    enable_room_previews=True, sync_audio_video=True,
)
print(f"Updated: {updated['display_name']}")
```
