Retrieve a single SWML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SWML webhook.
</ParamField>

## **Returns**

`dict` -- The SWML webhook resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

webhook = client.fabric.swml_webhooks.get("webhook-id")
print(f"Swml Webhook: {webhook['display_name']}, ID: {webhook['id']}")
```
