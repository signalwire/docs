Retrieve a single SWML script resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/swml_scripts/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/swml_scripts/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.swml_scripts.get("resource-id")
print(f"Swml Script: {item['display_name']}, ID: {item['id']}")
```
