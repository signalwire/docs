Retrieve a single SWML script resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

## **Returns**

`dict` -- The SWML script resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.fabric.swml_scripts.get("resource-id")
print(f"Swml Script: {item['display_name']}, ID: {item['id']}")
```
