Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`dict` -- The SIP gateway resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

gateway = client.fabric.sip_gateways.get("gateway-id")
print(f"Sip Gateway: {gateway['display_name']}, ID: {gateway['id']}")
```
