Delete a SWML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SWML webhook to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success (HTTP 204).

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.swml_webhooks.delete("webhook-id")
print("Swml Webhook deleted")
```
