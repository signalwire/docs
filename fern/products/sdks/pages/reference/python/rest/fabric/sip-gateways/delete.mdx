Delete a SIP gateway resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/sip_gateways/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.fabric.sip_gateways.delete("gateway-id")
print("Sip Gateway deleted")
```
