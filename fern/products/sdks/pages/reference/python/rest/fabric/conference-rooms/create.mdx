Create a new conference room resource.

## **Parameters**

<Markdown src="/snippets/sdks/rest/fabric/conference-rooms/_create-params.mdx" />

## **Returns**

`dict` -- The created conference room resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.conference_rooms.create(
    name="my-room", enable_room_previews=True,
)
print(f"Created: {result['display_name']}")
```
