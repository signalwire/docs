List FreeSWITCH connector resources in the project.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing FreeSWITCH connector resources.

## **Example**

```python {8-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.freeswitch_connectors.list()
for connector in response.get("data", []):
    print(f"{connector['name']}: {connector['id']}")
```
