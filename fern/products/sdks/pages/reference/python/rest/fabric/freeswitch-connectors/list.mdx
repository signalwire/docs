List FreeSWITCH connector resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing FreeSWITCH connector resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.freeswitch_connectors.list()
for connector in response.get("data", []):
    print(f"{connector['display_name']}: {connector['id']}")
```
