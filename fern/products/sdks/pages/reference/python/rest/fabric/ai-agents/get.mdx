Retrieve a single AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`dict` -- The AI agent resource.

## **Example**

```python {8-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

agent = client.fabric.ai_agents.get("agent-id")
print(f"AI Agent: {agent['name']}, ID: {agent['id']}")
```
