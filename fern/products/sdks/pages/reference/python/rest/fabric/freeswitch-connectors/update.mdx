Replace a FreeSWITCH connector resource. Uses PUT for full replacement.

<EndpointSchemaSnippet endpoint="PUT /api/fabric/resources/freeswitch_connectors/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/fabric/resources/freeswitch_connectors/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.fabric.freeswitch_connectors.update("connector-id", name="updated-name")
print(f"Updated: {updated['display_name']}")
```
