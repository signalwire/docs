List all Fabric resources across all resource types.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`dict` -- Paginated response containing Fabric resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.resources.list()
for resource in response.get("data", []):
    print(f"{resource['type']}: {resource['name']} ({resource['id']})")
```
