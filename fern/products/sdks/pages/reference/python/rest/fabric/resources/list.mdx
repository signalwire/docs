List all Fabric resources across all resource types.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="str" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing Fabric resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

response = client.fabric.resources.list()
for resource in response.get("data", []):
    print(f"{resource['type']}: {resource['name']} ({resource['id']})")
```
