Manage conference room resources. Standard CRUD with PUT updates and address listing.

Access via `client.fabric.conference_rooms` on a [`RestClient`][restclient] instance.

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

items = client.fabric.conference_rooms.list()
```
