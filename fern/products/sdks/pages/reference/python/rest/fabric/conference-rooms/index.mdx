Manage conference room resources. Standard CRUD with PUT updates and address listing.

Access via `client.fabric.conference_rooms` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {8}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

items = client.fabric.conference_rooms.list()
```
