Deploy a specific version of a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<ParamField path="call_flow_version_id" type="str" toc={true}>
  The version ID to deploy. Provide either this or `document_version`.
</ParamField>

<ParamField path="document_version" type="int" toc={true}>
  The document version number to deploy. Provide either this or `call_flow_version_id`.
</ParamField>

## **Returns**

`dict` -- The deployed version details.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.call_flows.deploy_version(
    "call-flow-id", call_flow_version_id="version-id",
)
print(f"Deployed version: {result['id']}")
```
