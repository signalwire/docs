Retrieve a single SIP endpoint resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- The SIP endpoint resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
item = client.fabric.sip_endpoints.get("resource-id")
```
