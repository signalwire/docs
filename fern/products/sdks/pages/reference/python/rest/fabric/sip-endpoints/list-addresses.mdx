List addresses associated with a SIP endpoint resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
addresses = client.fabric.sip_endpoints.list_addresses("resource-id")
```
