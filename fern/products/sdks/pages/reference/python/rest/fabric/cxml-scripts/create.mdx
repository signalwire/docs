Create a new cXML script resource.

## **Parameters**

<ParamField path="name" type="str" required={true} toc={true}>
  Display name of the cXML script.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created cXML script resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.cxml_scripts.create(
    name="my-item",
)
print(f"Created: {result['id']}")
```
