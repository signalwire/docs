Create a new cXML script resource.

## **Parameters**

<ParamField path="display_name" type="str" required={true} toc={true}>
  Display name of the cXML script.
</ParamField>

<ParamField path="contents" type="str" required={true} toc={true}>
  The cXML script contents.
</ParamField>

## **Returns**

`dict` -- The created cXML script resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.fabric.cxml_scripts.create(
    display_name="my-script", contents="<Response><Say>Hello</Say></Response>",
)
print(f"Created: {result['id']}")
```
