Retrieve a single subscriber resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`dict` -- The subscriber resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

subscriber = client.fabric.subscribers.get("subscriber-id")
print(f"Subscriber: {subscriber['name']}, ID: {subscriber['id']}")
```
