List addresses associated with a conference room resource.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
addresses = client.fabric.conference_rooms.list_addresses("resource-id")
```
