Partially update a cXML webhook resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated cXML webhook resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.fabric.cxml_webhooks.update("resource-id", name="updated-name")
```
