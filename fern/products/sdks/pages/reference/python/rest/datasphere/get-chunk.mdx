Retrieve a specific chunk from a document.

## **Parameters**

<ParamField path="document_id" type="str" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunk_id" type="str" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`dict` -- The chunk object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

chunk = client.datasphere.documents.get_chunk("document-id", "chunk-id")
print(chunk.get("text"))
```
