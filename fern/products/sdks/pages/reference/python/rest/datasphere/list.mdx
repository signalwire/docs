List documents in the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of document objects.

## **Example**

```python
result = client.datasphere.documents.list()
for doc in result.get("data", []):
    print(doc.get("name"), doc.get("id"))
```
