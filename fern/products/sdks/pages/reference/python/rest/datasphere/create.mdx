Upload a new document to the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="url" type="str" toc={true}>
  URL of the document to ingest.
</ParamField>

<ParamField path="tags" type="list[str]" toc={true}>
  Tags to associate with the document for filtering and retrieval.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional document fields (e.g., `metadata`).
</ParamField>

## **Returns**

`dict` -- The newly created document object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

doc = client.datasphere.documents.create(
    url="https://filesamples.com/samples/document/txt/sample3.txt",
    tags=["support", "demo"],
)
print("Document ID:", doc.get("id"))
```
