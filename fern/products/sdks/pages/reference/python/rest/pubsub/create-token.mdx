Create a PubSub authentication token.

## **Parameters**

<ParamField path="channels" type="list[str]" toc={true}>
  List of channel names the token grants access to.
</ParamField>

<ParamField path="member_id" type="str" toc={true}>
  Identifier for the connecting member.
</ParamField>

<ParamField path="ttl" type="int" toc={true}>
  Token time-to-live in seconds.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional token parameters (e.g., `state`, `permissions`).
</ParamField>

## **Returns**

`dict` -- The token object containing the JWT token string.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.pubsub.create_token(
    channels=["notifications", "updates"],
    member_id="user-123",
    ttl=3600,
)
print("PubSub token:", token.get("token"))
```
