Add a phone number to a group.

## **Parameters**

<ParamField path="group_id" type="str" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="phone_number_id" type="str" required={true} toc={true}>
  The phone number resource ID to add to the group.
</ParamField>

## **Returns**

`dict` -- The newly created membership object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

membership = client.number_groups.add_membership(
    "group-id",
    phone_number_id="phone-number-id",
)
print("Membership ID:", membership.get("id"))
```
