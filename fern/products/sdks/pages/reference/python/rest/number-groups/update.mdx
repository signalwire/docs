Update a number group. Uses PUT to replace fields.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="name" type="str" required={true} toc={true}>
  Group name. Required for PUT replacement.
</ParamField>

<ParamField path="sticky_sender" type="bool" toc={true}>
  Whether to use the same sender number for replies within this group.
</ParamField>

## **Returns**

`dict` -- The updated group object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.number_groups.update("group-id", name="Support Numbers")
print(updated["name"])
```
