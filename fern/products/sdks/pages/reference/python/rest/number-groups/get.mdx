Retrieve a specific number group by ID.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The group resource ID.
</ParamField>

## **Returns**

`dict` -- The group object.

## **Example**

```python
group = client.number_groups.get("group-id")
print(group["name"])
```
