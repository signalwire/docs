Remove a phone number from a group.

<Note>
Like `get_membership()`, this method takes a membership ID directly.
</Note>

<EndpointSchemaSnippet endpoint="DELETE /api/relay/rest/number_group_memberships/{id}" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.number_groups.delete_membership("membership-id")
```
