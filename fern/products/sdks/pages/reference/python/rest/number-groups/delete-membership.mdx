Remove a phone number from a group.

<Note>
Like `get_membership()`, this method takes a membership ID directly.
</Note>

## **Parameters**

<ParamField path="membership_id" type="str" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.number_groups.delete_membership("membership-id")
```
