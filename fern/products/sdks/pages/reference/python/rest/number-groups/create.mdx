Create a new number group.

<EndpointSchemaSnippet endpoint="POST /api/relay/rest/number_groups" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/relay/rest/number_groups" />

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

group = client.number_groups.create(name="Sales Numbers")
print("Group ID:", group["id"])
```
