Create a new number group.

## **Parameters**

<ParamField path="name" type="str" toc={true}>
  Group name.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional group configuration.
</ParamField>

## **Returns**

`dict` -- The newly created group object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

group = client.number_groups.create(name="Sales Numbers")
print("Group ID:", group["id"])
```
