Manage project-level API tokens. Tokens control access to your SignalWire project
and can be created, updated, and revoked.

Access via `client.project.tokens` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
```
