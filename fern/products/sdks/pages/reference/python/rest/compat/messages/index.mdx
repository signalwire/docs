Manage SMS and MMS messages with CRUD operations and media sub-resources.

Access via `client.compat.messages` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

messages = client.compat.messages.list()
```

## Methods

### Message CRUD

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/compat/messages/list) | List messages in the account. |
| [`create`](/docs/sdks/reference/python/rest/compat/messages/create) | Send a new SMS or MMS message. |
| [`get`](/docs/sdks/reference/python/rest/compat/messages/get) | Retrieve a single message by SID. |
| [`update`](/docs/sdks/reference/python/rest/compat/messages/update) | Update a message (e.g., redact the body). |
| [`delete`](/docs/sdks/reference/python/rest/compat/messages/delete) | Delete a message record. |

### Media

| Method | Description |
|---|---|
| [`list_media`](/docs/sdks/reference/python/rest/compat/messages/list-media) | List media items attached to a message. |
| [`get_media`](/docs/sdks/reference/python/rest/compat/messages/get-media) | Retrieve a specific media item. |
| [`delete_media`](/docs/sdks/reference/python/rest/compat/messages/delete-media) | Delete a media item from a message. |
