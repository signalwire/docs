Retrieve a specific media item from a message.

## **Parameters**

<ParamField path="message_sid" type="str" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="media_sid" type="str" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`dict` -- The media resource including the content URL.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
item = client.compat.messages.get_media("SM...", "ME...")
```
