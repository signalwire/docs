Update an active call (e.g., redirect to new LAML, end the call). Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The call SID.
</ParamField>

<ParamField path="Url" type="str" toc={true}>
  New URL to fetch LAML instructions from.
</ParamField>

<ParamField path="Status" type="str" toc={true}>
  Set to `"completed"` to end the call.
</ParamField>

## **Returns**

`dict` -- The updated call resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.calls.update("CA...", Status="completed")
```
