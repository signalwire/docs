List recordings in the account.

## **Parameters**

<ParamField path="DateCreated" type="str" toc={true}>
  Filter by creation date.
</ParamField>

<ParamField path="CallSid" type="str" toc={true}>
  Filter by the call SID that generated the recording.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing recording resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recordings = client.compat.recordings.list(PageSize=20)
```
