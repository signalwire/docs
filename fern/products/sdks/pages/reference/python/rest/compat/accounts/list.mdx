List accounts.

## **Returns**

`dict` -- Response containing accounts.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

accounts = client.compat.accounts.list()
```
