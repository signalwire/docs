Manage calls with CRUD operations, plus sub-resources for in-call recording
and streaming. Uses POST for updates (Twilio convention).

Access via `client.compat.calls` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
calls = client.compat.calls.list()
```

## Methods

### Call CRUD

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/compat/calls/list) | List calls in the account. |
| [`create`](/docs/sdks/reference/python/rest/compat/calls/create) | Initiate a new outbound call. |
| [`get`](/docs/sdks/reference/python/rest/compat/calls/get) | Retrieve a single call by SID. |
| [`update`](/docs/sdks/reference/python/rest/compat/calls/update) | Update an active call. |
| [`delete`](/docs/sdks/reference/python/rest/compat/calls/delete) | Delete a call record. |

### Recording

| Method | Description |
|---|---|
| [`start_recording`](/docs/sdks/reference/python/rest/compat/calls/start-recording) | Start recording an active call. |
| [`update_recording`](/docs/sdks/reference/python/rest/compat/calls/update-recording) | Update a call recording (pause, resume, stop). |

### Streaming

| Method | Description |
|---|---|
| [`start_stream`](/docs/sdks/reference/python/rest/compat/calls/start-stream) | Start a media stream on an active call. |
| [`stop_stream`](/docs/sdks/reference/python/rest/compat/calls/stop-stream) | Stop a media stream on a call. |
