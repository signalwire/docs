Retrieve a single account by its SID.

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The account SID.
</ParamField>

## **Returns**

`dict` -- The account resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
account = client.compat.accounts.get("account-sid")
```
