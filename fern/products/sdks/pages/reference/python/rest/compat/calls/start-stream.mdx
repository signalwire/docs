Start a media stream on an active call, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="call_sid" type="str" required={true} toc={true}>
  The SID of the call.
</ParamField>

<ParamField path="Url" type="str" toc={true}>
  The WebSocket URL to stream audio to.
</ParamField>

## **Returns**

`dict` -- The created stream resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
stream = client.compat.calls.start_stream("CA...", Url="wss://stream.example.com")
```
