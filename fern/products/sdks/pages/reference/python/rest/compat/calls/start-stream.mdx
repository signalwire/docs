Start a media stream on an active call, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="call_sid" type="str" required={true} toc={true}>
  The SID of the call.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/calls/_start-stream-params.mdx" />

## **Returns**

`dict` -- The created stream resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

stream = client.compat.calls.start_stream("CA...", Url="wss://stream.example.com")
```
