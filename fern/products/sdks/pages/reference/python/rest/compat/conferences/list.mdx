List conferences.

## **Parameters**

<ParamField path="Status" type="str" toc={true}>
  Filter by conference status.

  - `"init"` -- conference has been created but no participants have joined yet
  - `"in-progress"` -- conference is currently active with participants
  - `"completed"` -- conference has ended
</ParamField>

<ParamField path="FriendlyName" type="str" toc={true}>
  Filter by the conference's friendly name.
</ParamField>

<ParamField path="DateCreated" type="str" toc={true}>
  Filter by creation date (ISO 8601 format).
</ParamField>

<ParamField path="DateUpdated" type="str" toc={true}>
  Filter by last updated date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated<" type="str" toc={true}>
  Filter by conferences created before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated>" type="str" toc={true}>
  Filter by conferences created after this date (ISO 8601 format).
</ParamField>

<ParamField path="DateUpdated<" type="str" toc={true}>
  Filter by conferences updated before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateUpdated>" type="str" toc={true}>
  Filter by conferences updated after this date (ISO 8601 format).
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing conference resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

conferences = client.compat.conferences.list(Status="in-progress")
```
