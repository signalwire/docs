Search for available toll-free phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="str" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="Contains" type="str" toc={true}>
  Filter by number pattern (e.g., `"800"`).
</ParamField>

<ParamField path="AreaCode" type="str" toc={true}>
  Filter by area code.
</ParamField>

<ParamField path="SmsEnabled" type="bool" toc={true}>
  Filter for SMS-capable numbers.
</ParamField>

<ParamField path="VoiceEnabled" type="bool" toc={true}>
  Filter for voice-capable numbers.
</ParamField>

<ParamField path="FaxEnabled" type="bool" toc={true}>
  Filter for fax-capable numbers.
</ParamField>

<ParamField path="MmsEnabled" type="bool" toc={true}>
  Filter for MMS-capable numbers.
</ParamField>

<ParamField path="Beta" type="bool" toc={true}>
  Include beta phone numbers in results.
</ParamField>

<ParamField path="InRegion" type="str" toc={true}>
  Filter by region or state abbreviation.
</ParamField>

<ParamField path="InLocality" type="str" toc={true}>
  Filter by locality or city name.
</ParamField>

<ParamField path="ExcludeAllAddressRequired" type="bool" toc={true}>
  Exclude numbers that require any type of address.
</ParamField>

<ParamField path="ExcludeLocalAddressRequired" type="bool" toc={true}>
  Exclude numbers that require a local address.
</ParamField>

<ParamField path="ExcludeForeignAddressRequired" type="bool" toc={true}>
  Exclude numbers that require a foreign address.
</ParamField>

## **Returns**

`dict` -- Paginated response containing available toll-free number resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

numbers = client.compat.phone_numbers.search_toll_free("US", Contains="800")
```
