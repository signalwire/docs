Update a conference (e.g., end it, or set the announce URL). Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The conference SID.
</ParamField>

<ParamField path="Status" type="str" toc={true}>
  Set to `"completed"` to end the conference.
</ParamField>

<ParamField path="AnnounceUrl" type="str" toc={true}>
  URL returning LAML to play an announcement to all participants.
</ParamField>

## **Returns**

`dict` -- The updated conference resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.conferences.update("CF...", Status="completed")
```
