List media items (images, files) attached to a message.

## **Parameters**

<ParamField path="message_sid" type="str" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="DateCreated" type="str" toc={true}>
  Filter by creation date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated<" type="str" toc={true}>
  Filter by media created before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated>" type="str" toc={true}>
  Filter by media created after this date (ISO 8601 format).
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- List of media resources attached to the message.

from signalwire.rest import RestClient

    token="your-api-token",
    host="your-space.signalwire.com",
)
