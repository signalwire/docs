Update a LAML bin's content or name. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The LAML bin SID.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/laml-bins/_create-update-params.mdx" />

## **Returns**

`dict` -- The updated LAML bin resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.laml_bins.update(
    "LB...",
    Name="Greeting",
    Contents=(
        '<?xml version="1.0" encoding="UTF-8"?>'
        "<Response><Say>Goodbye!</Say></Response>"
    )
)
```
