List messages in the account.

## **Parameters**

<ParamField path="To" type="str" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="From" type="str" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing message resources.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

messages = client.compat.messages.list(PageSize=20)
```
