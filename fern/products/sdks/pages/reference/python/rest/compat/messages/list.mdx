List messages in the account.

## **Parameters**

<ParamField path="To" type="str" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="From" type="str" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="DateSent" type="str" toc={true}>
  Filter by date sent (ISO 8601 format).
</ParamField>

<ParamField path="DateSent<" type="str" toc={true}>
  Filter by messages sent before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateSent>" type="str" toc={true}>
  Filter by messages sent after this date (ISO 8601 format).
</ParamField>

<ParamField path="Status" type="str" toc={true}>
  Filter by message status.

  - `"queued"` -- message is queued for delivery
  - `"sending"` -- message is being sent
  - `"sent"` -- message has been sent
  - `"delivered"` -- message has been delivered
  - `"undelivered"` -- message could not be delivered
  - `"failed"` -- message delivery failed
  - `"received"` -- inbound message received
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing message resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

messages = client.compat.messages.list(PageSize=20)
```
