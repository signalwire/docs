Update a phone number's configuration (e.g., change voice/SMS URLs). Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The phone number SID.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/phone-numbers/_update-params.mdx" />

## **Returns**

`dict` -- The updated phone number resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.phone_numbers.update(
    "PN...",
    VoiceUrl="https://example.com/voice"
)
```
