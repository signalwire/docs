  Filter by name.
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>

</ParamField>

</ParamField>

## **Returns**

## **Example**

```python {9}

    project="your-project-id",
    token="your-api-token",

)

bins = client.compat.laml_bins.list(PageSize=20)
```
