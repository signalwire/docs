Delete a media item from a fax.

## **Parameters**

<ParamField path="fax_sid" type="str" required={true} toc={true}>
  The SID of the fax.
</ParamField>

<ParamField path="media_sid" type="str" required={true} toc={true}>
  The SID of the media item to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.compat.faxes.delete_media("FX...", "ME...")
```
