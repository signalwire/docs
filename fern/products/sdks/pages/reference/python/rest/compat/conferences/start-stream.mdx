Start a media stream on a conference, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="conference_sid" type="str" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="Url" type="str" toc={true}>
  The WebSocket URL to stream audio to.
</ParamField>

## **Returns**

`dict` -- The created stream resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

stream = client.compat.conferences.start_stream(
    "CF...",
    Url="wss://stream.example.com"
)
```
