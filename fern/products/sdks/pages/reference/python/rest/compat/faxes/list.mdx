List faxes in the account.

## **Parameters**

<ParamField path="DateCreatedAfter" type="str" toc={true}>
  Filter by faxes created after this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreatedOnOrBefore" type="str" toc={true}>
  Filter by faxes created on or before this date (ISO 8601 format).
</ParamField>

<ParamField path="From" type="str" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="To" type="str" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing fax resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

faxes = client.compat.faxes.list()
```
