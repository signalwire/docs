Update an application's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The application SID.
</ParamField>

<ParamField path="FriendlyName" type="str" toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="str" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="SmsUrl" type="str" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="StatusCallback" type="str" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`dict` -- The updated application resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.applications.update(
    "AP...",
    VoiceUrl="https://example.com/new-voice"
)
```
