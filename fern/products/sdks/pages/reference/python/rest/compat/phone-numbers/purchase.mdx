Purchase a new phone number for the account.

## **Parameters**

<ParamField path="PhoneNumber" type="str" required={true} toc={true}>
  The phone number to purchase in E.164 format.
</ParamField>

<ParamField path="FriendlyName" type="str" toc={true}>
  A human-readable label for the number.
</ParamField>

<ParamField path="VoiceUrl" type="str" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="SmsUrl" type="str" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="StatusCallback" type="str" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`dict` -- The purchased phone number resource.

## **Example**

```python {9-12}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

number = client.compat.phone_numbers.purchase(
    PhoneNumber="+15551234567",
    FriendlyName="Main Line"
)
```
