Manage call queues with CRUD operations and member sub-resources for dequeuing
callers.

Access via `client.compat.queues` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

queues = client.compat.queues.list()
```
