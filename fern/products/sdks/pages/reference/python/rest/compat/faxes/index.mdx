Manage faxes with CRUD operations and media sub-resources. The media sub-resource
pattern is identical to [`messages`](/docs/sdks/reference/python/rest/compat/messages).

Access via `client.compat.faxes` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python
from signalwire.rest import RestClient

client = RestClient()
faxes = client.compat.faxes.list()
```

## Methods

### Fax CRUD

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/compat/faxes/list) | List faxes in the account. |
| [`create`](/docs/sdks/reference/python/rest/compat/faxes/create) | Send a new fax. |
| [`get`](/docs/sdks/reference/python/rest/compat/faxes/get) | Retrieve a single fax by SID. |
| [`update`](/docs/sdks/reference/python/rest/compat/faxes/update) | Update a fax resource. |
| [`delete`](/docs/sdks/reference/python/rest/compat/faxes/delete) | Delete a fax record. |

### Media

| Method | Description |
|---|---|
| [`list_media`](/docs/sdks/reference/python/rest/compat/faxes/list-media) | List media items attached to a fax. |
| [`get_media`](/docs/sdks/reference/python/rest/compat/faxes/get-media) | Retrieve a specific media item. |
| [`delete_media`](/docs/sdks/reference/python/rest/compat/faxes/delete-media) | Delete a media item from a fax. |
