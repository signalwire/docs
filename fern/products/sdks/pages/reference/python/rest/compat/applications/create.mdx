Create a new application.

## **Parameters**

<ParamField path="FriendlyName" type="str" required={true} toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="str" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="VoiceMethod" type="str" toc={true}>
  HTTP method for the voice URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="SmsUrl" type="str" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="SmsMethod" type="str" toc={true}>
  HTTP method for the SMS URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="StatusCallback" type="str" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`dict` -- The created application resource.

## **Example**

```python {9-13}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

app = client.compat.applications.create(
    FriendlyName="My App",
    VoiceUrl="https://example.com/voice",
    SmsUrl="https://example.com/sms"
)
```
