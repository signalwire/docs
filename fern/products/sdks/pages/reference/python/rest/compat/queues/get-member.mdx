Retrieve a specific member from a queue by call SID.

## **Parameters**

<ParamField path="queue_sid" type="str" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="call_sid" type="str" required={true} toc={true}>
  The call SID of the queue member.
</ParamField>

## **Returns**

`dict` -- The queue member resource.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

member = client.compat.queues.get_member("QU...", "CA...")
```
