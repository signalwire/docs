Update a queue's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The queue SID.
</ParamField>

<ParamField path="FriendlyName" type="str" toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="int" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`dict` -- The updated queue resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.queues.update("QU...", MaxSize=200)
```
