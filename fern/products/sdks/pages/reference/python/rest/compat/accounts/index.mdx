Manage accounts and subprojects. The base path is
`/api/laml/2010-04-01/Accounts` (not scoped to a specific account SID).

Access via `client.compat.accounts` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

accounts = client.compat.accounts.list()
```

## Methods

| Method | Description |
|---|---|
| [`list`](/docs/sdks/reference/python/rest/compat/accounts/list) | List accounts. |
| [`create`](/docs/sdks/reference/python/rest/compat/accounts/create) | Create a new subproject. |
| [`get`](/docs/sdks/reference/python/rest/compat/accounts/get) | Retrieve an account by SID. |
| [`update`](/docs/sdks/reference/python/rest/compat/accounts/update) | Update an account. |
