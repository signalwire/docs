Delete an API token.

## **Parameters**

<ParamField path="token_id" type="str" required={true} toc={true}>
  The token identifier to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.tokens.delete("token-id")
```
