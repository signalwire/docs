Search for available local phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="str" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="AreaCode" type="str" toc={true}>
  Filter by area code.
</ParamField>

<ParamField path="Contains" type="str" toc={true}>
  Filter by number pattern (e.g., `"555"`).
</ParamField>

<ParamField path="InRegion" type="str" toc={true}>
  Filter by region or state abbreviation.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing available local number resources.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

numbers = client.compat.phone_numbers.search_local("US", AreaCode="512")
```
