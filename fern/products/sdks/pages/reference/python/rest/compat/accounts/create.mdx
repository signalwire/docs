Create a new subproject (sub-account).

## **Parameters**

<ParamField path="FriendlyName" type="str" toc={true}>
  A human-readable name for the subproject.
</ParamField>

## **Returns**

`dict` -- The created account resource.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

sub = client.compat.accounts.create(FriendlyName="Marketing Team")
```
