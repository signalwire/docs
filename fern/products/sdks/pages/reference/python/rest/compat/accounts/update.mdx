Update an account. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The account SID to update.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/accounts/_create-update-params.mdx" />

## **Returns**

`dict` -- The updated account resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.accounts.update("account-sid", FriendlyName="Sales Team")
```
