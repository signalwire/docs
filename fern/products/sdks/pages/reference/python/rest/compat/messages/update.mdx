Update a message (e.g., redact the body of a sent message).

## **Parameters**

<ParamField path="sid" type="str" required={true} toc={true}>
  The message SID.
</ParamField>

<ParamField path="Body" type="str" toc={true}>
  Set to an empty string to redact the message body.
</ParamField>

## **Returns**

`dict` -- The updated message resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.compat.messages.update("SM...", Body="")
```
