List calls in the account.

## **Parameters**

<ParamField path="Status" type="str" toc={true}>
  Filter by call status.

  - `"queued"` -- call is waiting to be processed
  - `"ringing"` -- call is ringing at the destination
  - `"in-progress"` -- call is currently connected and active
  - `"completed"` -- call ended normally
  - `"busy"` -- destination returned a busy signal
  - `"failed"` -- call could not be completed due to an error
  - `"no-answer"` -- destination did not answer
  - `"canceled"` -- call was canceled before being answered
</ParamField>

<ParamField path="To" type="str" toc={true}>
  Filter by the destination number.
</ParamField>

<ParamField path="From" type="str" toc={true}>
  Filter by the originating number.
</ParamField>

<ParamField path="StartTime" type="str" toc={true}>
  Filter by call start time (ISO 8601 format).
</ParamField>

<ParamField path="StartTime<" type="str" toc={true}>
  Filter by calls started before this time (ISO 8601 format).
</ParamField>

<ParamField path="StartTime>" type="str" toc={true}>
  Filter by calls started after this time (ISO 8601 format).
</ParamField>

<ParamField path="EndTime" type="str" toc={true}>
  Filter by call end time (ISO 8601 format).
</ParamField>

<ParamField path="EndTime<" type="str" toc={true}>
  Filter by calls ended before this time (ISO 8601 format).
</ParamField>

<ParamField path="EndTime>" type="str" toc={true}>
  Filter by calls ended after this time (ISO 8601 format).
</ParamField>

<ParamField path="ParentCallSid" type="str" toc={true}>
  Filter by the parent call SID (for calls created as a result of a `<Dial>` verb).
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing call resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

calls = client.compat.calls.list(Status="completed", PageSize=20)
```
