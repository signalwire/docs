Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="str" required={true} toc={true}>
  The token identifier.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/tokens/_update-params.mdx" />

## **Returns**

`dict` -- The updated token resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.tokens.update("token-id", name="updated-token")
```
