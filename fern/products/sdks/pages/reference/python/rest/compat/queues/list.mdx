List queues in the account.

## **Parameters**

<ParamField path="FriendlyName" type="str" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing queue resources.

## **Example**

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

queues = client.compat.queues.list(PageSize=20)
```
