List applications in the account.

## **Parameters**

<ParamField path="FriendlyName" type="str" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing application resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

apps = client.compat.applications.list(PageSize=20)
```
