Retrieve a specific media item from a fax.

## **Parameters**

<ParamField path="fax_sid" type="str" required={true} toc={true}>
  The SID of the fax.
</ParamField>

<ParamField path="media_sid" type="str" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`dict` -- The media resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

item = client.compat.faxes.get_media("FX...", "ME...")
```
