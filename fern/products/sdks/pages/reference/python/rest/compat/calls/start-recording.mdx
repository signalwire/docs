Start recording an active call.

## **Parameters**

<ParamField path="call_sid" type="str" required={true} toc={true}>
  The SID of the call to record.
</ParamField>

<ParamField path="RecordingStatusCallback" type="str" toc={true}>
  URL to receive recording status events.
</ParamField>

<ParamField path="RecordingChannels" type="str" toc={true}>
  Recording channel mode.

  - `"mono"` -- both legs of the call are mixed into a single audio channel
  - `"dual"` -- each leg of the call is recorded in a separate audio channel
</ParamField>

## **Returns**

`dict` -- The created recording resource.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
recording = client.compat.calls.start_recording("CA...")
```
