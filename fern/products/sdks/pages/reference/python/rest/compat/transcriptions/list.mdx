List transcriptions in the account.

## **Parameters**

<ParamField path="DateCreated" type="str" toc={true}>
  Filter by creation date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated<" type="str" toc={true}>
  Filter by transcriptions created before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated>" type="str" toc={true}>
  Filter by transcriptions created after this date (ISO 8601 format).
</ParamField>

<ParamField path="Page" type="int" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="int" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="str" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`dict` -- Paginated response containing transcription resources.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

transcriptions = client.compat.transcriptions.list(PageSize=20)
```
