Update a call recording (e.g., pause, resume, or stop).

## **Parameters**

<ParamField path="call_sid" type="str" required={true} toc={true}>
  The SID of the call.
</ParamField>

<ParamField path="recording_sid" type="str" required={true} toc={true}>
  The SID of the recording.
</ParamField>

<ParamField path="Status" type="str" toc={true}>
  New recording status.

  - `"paused"` -- pause the recording
  - `"in-progress"` -- resume a paused recording
  - `"stopped"` -- stop the recording permanently
</ParamField>

## **Returns**

`dict` -- The updated recording resource.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.compat.calls.update_recording("CA...", "RE...", Status="paused")
```
