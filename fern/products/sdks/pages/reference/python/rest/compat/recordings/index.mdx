Manage call recordings with list, get, and delete operations.

Access via `client.compat.recordings` on a [`RestClient`](/docs/sdks/reference/python/rest/client) instance.

```python {9-9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

recordings = client.compat.recordings.list()
```
