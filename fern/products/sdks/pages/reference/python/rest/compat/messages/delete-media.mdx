Delete a media item from a message.

## **Parameters**

<ParamField path="message_sid" type="str" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="media_sid" type="str" required={true} toc={true}>
  The SID of the media item to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```python
from signalwire.rest import RestClient

client = RestClient()
client.compat.messages.delete_media("SM...", "ME...")
```
