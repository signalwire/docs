Retrieve the next member in the queue (the member at the front).

## **Parameters**

<ParamField path="queue_id" type="str" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- The next member object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

next_member = client.queues.get_next_member("queue-id")
print("Next caller:", next_member.get("call_id"))
```
