Update a queue. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="str" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Fields to update (e.g., `name`, `max_size`).
</ParamField>

## **Returns**

`dict` -- The updated queue object.

## **Example**

```python {9-10}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

updated = client.queues.update("queue-id", name="priority-support")
print(f"Updated: {updated['id']}")
```
