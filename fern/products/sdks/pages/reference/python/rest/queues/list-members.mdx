List members currently in a queue.

## **Parameters**

<ParamField path="queue_id" type="str" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of member objects.

## **Example**

```python {9-11}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

members = client.queues.list_members("queue-id")
for m in members.get("data", []):
    print(m.get("call_id"), m.get("position"))
```
