Import a phone number from an external provider.

## **Parameters**

<ParamField path="number" type="str" toc={true}>
  The phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional import parameters (e.g., `name`, `call_handler`,
  `call_request_url`).
</ParamField>

## **Returns**

`dict` -- The imported phone number object.

## **Example**

```python {9}
from signalwire.rest import RestClient

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

imported = client.imported_numbers.create(
    number="+15551234567",
    name="External Support Line",
)
print("Imported:", imported.get("id"))
```
