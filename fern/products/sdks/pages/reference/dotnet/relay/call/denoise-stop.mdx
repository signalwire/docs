Stop noise reduction on the call, restoring the original unfiltered audio.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming noise reduction has stopped.

## **Example**

```csharp
using SignalWire.Relay;

var client = new RelayClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
    contexts: ["default"],;
)

async Task HandleCall(call)
{
    await call.answer();

    // Enable noise reduction
    await call.denoise();

    var action = await call.play([{"type": "tts", "text": "Noise reduction is now active."}]);
    await action.wait();

    var result = await call.denoise_stop();
    Console.WriteLine($"Noise reduction stopped: {result}");

await client.RunAsync();


```
