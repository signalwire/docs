Pause the recording.

## **Parameters**

<ParamField path="behavior" type="string?" toc={true}>
  Optional pause behavior. Controls what happens to the audio stream while paused
  (e.g., silence insertion).
</ParamField>

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```csharp
using SignalWire.Relay;

var client = new RelayClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
    contexts: ["default"],;
)

async Task HandleCall(call)
{
    await call.answer();
    var action = await call.record({"direction": "both", "format": "wav"});

    // Pause during sensitive information
    await action.pause();
    // ... sensitive exchange ...
    await action.resume();

await client.RunAsync();


```
