Retrieve a specific short code.

## **Parameters**

<ParamField path="short_code_id" type="string" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`dict` -- The short code object.

## **Example**

```csharp {9-10}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

code = client.short_codes.get("short-code-id");
Console.WriteLine(code["short_code"]);

```
