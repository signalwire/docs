### tokens.delete

Revoke and delete an API token.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The token resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.project.tokens.delete("token-id");

```
