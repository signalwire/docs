Retrieve a specific membership.

<Note>
This method uses the top-level membership endpoint and takes a membership ID
directly, not a group ID.
</Note>

## **Parameters**

<ParamField path="membership_id" type="string" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`dict` -- The membership object.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

membership = client.number_groups.get_membership("membership-id");
Console.WriteLine(membership["phone_number_id"]);

```
