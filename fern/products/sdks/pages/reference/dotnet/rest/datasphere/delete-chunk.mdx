Delete a specific chunk from a document.

## **Parameters**

<ParamField path="document_id" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunk_id" type="string" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.datasphere.documents.delete_chunk("document-id", "chunk-id");

```
