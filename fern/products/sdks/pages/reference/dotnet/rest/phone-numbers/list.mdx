List phone numbers owned by the project.

## **Parameters**

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `page_size`,
  `page_token`).
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

result = client.phone_numbers.list(page_size=10);
for number in result.get("data", []):
    Console.WriteLine(number["number"], number["name"]);

```
