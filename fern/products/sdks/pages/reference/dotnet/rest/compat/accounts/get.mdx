Retrieve a single account by its SID.

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The account SID.
</ParamField>

## **Returns**

`dict` -- The account resource.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

account = client.compat.accounts.get("account-sid");

```
