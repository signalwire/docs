List accounts.

## **Returns**

`dict` -- Response containing accounts.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

accounts = client.compat.accounts.list();

```
