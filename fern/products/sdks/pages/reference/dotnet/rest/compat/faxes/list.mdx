List faxes in the account.

## **Returns**

`dict` -- Paginated response containing fax resources.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

faxes = client.compat.faxes.list();

```
