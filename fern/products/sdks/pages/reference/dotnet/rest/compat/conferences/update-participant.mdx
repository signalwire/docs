Update a participant in a conference (e.g., mute, hold, or remove).

## **Parameters**

<ParamField path="conference_sid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="call_sid" type="string" required={true} toc={true}>
  The call SID of the participant.
</ParamField>

<ParamField path="Muted" type="bool" toc={true}>
  Set to `true` to mute or `false` to unmute.
</ParamField>

<ParamField path="Hold" type="bool" toc={true}>
  Set to `true` to hold or `false` to unhold.
</ParamField>

## **Returns**

`dict` -- The updated participant resource.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.compat.conferences.update_participant("CF...", "CA...", Muted=true);

```
