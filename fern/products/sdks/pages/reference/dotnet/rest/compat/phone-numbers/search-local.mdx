Search for available local phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="string" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="AreaCode" type="string" toc={true}>
  Filter by area code.
</ParamField>

<ParamField path="Contains" type="string" toc={true}>
  Filter by number pattern (e.g., `"555"`).
</ParamField>

<ParamField path="InRegion" type="string" toc={true}>
  Filter by region or state abbreviation.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing available local number resources.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

numbers = client.compat.phone_numbers.search_local("US", AreaCode: "512");

```
