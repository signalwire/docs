Update a queue's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The queue SID.
</ParamField>

<ParamField path="FriendlyName" type="string" toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="int" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`dict` -- The updated queue resource.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.compat.queues.update("QU...", MaxSize=200);

```
