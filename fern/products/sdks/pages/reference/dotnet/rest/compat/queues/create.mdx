Create a new call queue.

## **Parameters**

<ParamField path="FriendlyName" type="string" required={true} toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="int" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`dict` -- The created queue resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

queue = client.compat.queues.create(;
    FriendlyName: "Support Queue",
    MaxSize=100;
)

```
