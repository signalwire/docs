List incoming phone numbers in the account.

## **Parameters**

<ParamField path="PhoneNumber" type="string" toc={true}>
  Filter by exact phone number.
</ParamField>

<ParamField path="FriendlyName" type="string" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing phone number resources.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

numbers = client.compat.phone_numbers.list(PageSize=20);

```
