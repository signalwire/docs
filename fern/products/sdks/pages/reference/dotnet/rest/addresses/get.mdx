Retrieve a specific address.

## **Parameters**

<ParamField path="address_id" type="string" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`dict` -- The address object.

## **Example**

```csharp {9-10}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

addr = client.addresses.get("address-id");
Console.WriteLine(addr["customer_name"], addr["street"]);

```
