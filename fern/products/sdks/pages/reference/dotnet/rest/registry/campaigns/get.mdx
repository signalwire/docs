Retrieve a specific 10DLC campaign by ID.

## **Parameters**

<ParamField path="campaign_id" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

## **Returns**

`dict` -- The campaign object.

## **Example**

```csharp {9-10}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

campaign = client.registry.campaigns.get("campaign-id");
Console.WriteLine(campaign["description"], campaign["usecase"]);

```
