List messaging campaigns under a brand.

## **Parameters**

<ParamField path="brand_id" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of campaign objects.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

campaigns = client.registry.brands.list_campaigns("brand-id");
for c in campaigns.get("data", []):
    Console.WriteLine(c["description"], c["usecase"]);

```
