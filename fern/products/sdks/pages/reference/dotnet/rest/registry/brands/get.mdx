Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brand_id" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`dict` -- The brand object.

## **Example**

```csharp {9-10}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

brand = client.registry.brands.get("brand-id");
Console.WriteLine(brand["display_name"], brand["entity_type"]);

```
