Generate tokens for the SignalWire Chat service. Chat tokens authenticate
client-side connections for real-time chat messaging.

Access via `client.chat` on a [`RestClient`][restclient] instance.

```csharp
using SignalWire.REST;

client = RestClient();

```

---

### create_token

Create a Chat authentication token.

## **Parameters**

<ParamField path="channels" type="List<string>" toc={true}>
  List of channel names the token grants access to.
</ParamField>

<ParamField path="member_id" type="string" toc={true}>
  Identifier for the connecting member.
</ParamField>

<ParamField path="ttl" type="int" toc={true}>
  Token time-to-live in seconds.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional token parameters (e.g., `state`, `permissions`).
</ParamField>

## **Returns**

`dict` -- The token object containing the JWT token string.

## **Example**

```csharp
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

token = client.chat.create_token(;
    channels: ["support-room"],;
    member_id: "user-456",
    ttl: 3600,
)
Console.WriteLine("Chat token:", token["token"]);

```
