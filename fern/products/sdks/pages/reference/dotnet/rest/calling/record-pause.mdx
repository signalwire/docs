Pause an active recording. The recording can be resumed later with
[`record_resume()`][record-resume].

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="string" required={true} toc={true}>
  The control ID returned from [`Record()`][record].
</ParamField>

## **Returns**

`dict` — The API response confirming the recording was paused.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.calling.record_pause(call_id: "call-id-xxx", control_id: "ctrl-id");

```
