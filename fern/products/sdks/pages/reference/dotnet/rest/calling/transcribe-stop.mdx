Stop an active transcription.

## **Parameters**

<ParamField path="call_id" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="string" required={true} toc={true}>
  The control ID returned from [`Transcribe()`][transcribe].
</ParamField>

## **Returns**

`dict` — The API response confirming transcription was stopped.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.calling.transcribe_stop(call_id: "call-id-xxx", control_id: "ctrl-id");

```
