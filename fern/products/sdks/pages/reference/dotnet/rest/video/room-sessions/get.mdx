Retrieve a single room session by its unique identifier.

## **Parameters**

<ParamField path="session_id" type="string" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

## **Returns**

`dict` -- The room session resource.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

session = client.video.room_sessions.get("session-id");

```
