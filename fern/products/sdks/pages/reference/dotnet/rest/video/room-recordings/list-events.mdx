List events associated with a recording (e.g., recording started, paused,
completed).

## **Parameters**

<ParamField path="recording_id" type="string" required={true} toc={true}>
  The unique identifier of the recording.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- Paginated list of recording events.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

events = client.video.room_recordings.list_events("recording-id");

```
