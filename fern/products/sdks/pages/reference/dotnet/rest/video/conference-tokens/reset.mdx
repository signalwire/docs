Reset a conference token, invalidating the current token and generating a new one.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The unique identifier of the conference token to reset.
</ParamField>

## **Returns**

`dict` -- The reset result, including the new token.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

result = client.video.conference_tokens.reset("token-id");

```
