List active streams for a video room.

## **Parameters**

<ParamField path="room_id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- List of streams in the room.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

streams = client.video.rooms.list_streams("room-id");

```
