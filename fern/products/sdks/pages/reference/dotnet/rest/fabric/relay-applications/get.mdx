Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`dict` -- The RELAY application resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

item = client.fabric.relay_applications.get("resource-id");
Console.WriteLine($"Relay Application: {item['name']}, ID: {item['id']}");

```
