Create a new RELAY application resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the RELAY application.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created RELAY application resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

result = client.fabric.relay_applications.create(;
    name: "my-item",
)
Console.WriteLine($"Created: {result['id']}");

```
