Replace a call flow resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated call flow resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

updated = client.fabric.call_flows.update("flow-id", name: "updated-name");
Console.WriteLine($"Updated: {updated['name']}");

```
