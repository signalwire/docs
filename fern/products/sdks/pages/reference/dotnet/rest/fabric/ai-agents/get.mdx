Retrieve a single AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`dict` -- The AI agent resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

agent = client.fabric.ai_agents.get("agent-id");
Console.WriteLine($"AI Agent: {agent['name']}, ID: {agent['id']}");

```
