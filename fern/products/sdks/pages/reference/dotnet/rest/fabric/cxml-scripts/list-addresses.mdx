List addresses associated with a cXML script resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

addresses = client.fabric.cxml_scripts.list_addresses("resource-id");
for addr in addresses.get("data", []):
    Console.WriteLine($"{addr['display_name']}: {addr['id']}");

```
