Replace a conference room resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated conference room resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

updated = client.fabric.conference_rooms.update("room-id", name: "updated-name");
Console.WriteLine($"Updated: {updated['name']}");

```
