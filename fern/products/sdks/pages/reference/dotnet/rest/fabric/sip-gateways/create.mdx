Create a new SIP gateway resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the SIP gateway.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created SIP gateway resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

result = client.fabric.sip_gateways.create(;
    name: "my-gateway",
)
Console.WriteLine($"Created: {result['id']}");

```
