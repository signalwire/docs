Replace a RELAY application resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated RELAY application resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

updated = client.fabric.relay_applications.update("resource-id", name: "updated-name");
Console.WriteLine($"Updated: {updated['name']}");

```
