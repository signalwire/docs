Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`dict` -- The Fabric resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

resource = client.fabric.resources.get("resource-id");
Console.WriteLine($"{resource['type']}: {resource['name']}, ID: {resource['id']}");

```
