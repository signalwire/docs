Assign a phone route to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The assigned phone route.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

result = client.fabric.resources.assign_phone_route(;
    "resource-id", phone_number: "+15551234567",
)
Console.WriteLine($"Phone route assigned: {result['id']}");

```
