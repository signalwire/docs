Retrieve a single subscriber resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`dict` -- The subscriber resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

subscriber = client.fabric.subscribers.get("subscriber-id");
Console.WriteLine($"Subscriber: {subscriber['name']}, ID: {subscriber['id']}");

```
