Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`dict` -- The call flow resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

flow = client.fabric.call_flows.get("flow-id");
Console.WriteLine($"Call Flow: {flow['name']}, ID: {flow['id']}");

```
