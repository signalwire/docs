Delete a SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP gateway to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success (HTTP 204).

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.fabric.sip_gateways.delete("gateway-id");
Console.WriteLine("Sip Gateway deleted");

```
