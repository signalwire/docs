Partially update a SWML webhook resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SWML webhook.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated SWML webhook resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

updated = client.fabric.swml_webhooks.update("webhook-id", name: "updated-name");
Console.WriteLine($"Updated: {updated['name']}");

```
