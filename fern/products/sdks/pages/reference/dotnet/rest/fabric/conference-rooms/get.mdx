Retrieve a single conference room resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`dict` -- The conference room resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

room = client.fabric.conference_rooms.get("room-id");
Console.WriteLine($"Conference Room: {room['name']}, ID: {room['id']}");

```
