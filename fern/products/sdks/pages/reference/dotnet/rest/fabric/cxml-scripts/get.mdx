Retrieve a single cXML script resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`dict` -- The cXML script resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

item = client.fabric.cxml_scripts.get("resource-id");
Console.WriteLine($"Cxml Script: {item['name']}, ID: {item['id']}");

```
