Retrieve a single FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`dict` -- The FreeSWITCH connector resource.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

connector = client.fabric.freeswitch_connectors.get("connector-id");
Console.WriteLine($"Freeswitch Connector: {connector['name']}, ID: {connector['id']}");

```
