List SIP endpoint resources in the project.

## **Parameters**

<ParamField path="page_size" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing SIP endpoint resources.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

response = client.fabric.sip_endpoints.list();
for endpoint in response.get("data", []):
    Console.WriteLine($"{endpoint['name']}: {endpoint['id']}");

```
