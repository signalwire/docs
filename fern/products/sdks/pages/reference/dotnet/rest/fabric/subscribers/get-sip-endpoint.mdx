Retrieve a single SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- The SIP endpoint.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

endpoint = client.fabric.subscribers.get_sip_endpoint("subscriber-id", "endpoint-id");
Console.WriteLine($"SIP Endpoint: {endpoint['username']}, ID: {endpoint['id']}");

```
