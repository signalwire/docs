Create a new queue.

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Queue name.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional queue configuration fields (e.g., `max_size`, `fifo`).
</ParamField>

## **Returns**

`dict` -- The newly created queue object.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

queue = client.queues.create(name: "support", max_size=50);
Console.WriteLine("Queue ID:", queue["id"]);

```
