Delete a queue.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.queues.delete("queue-id");
Console.WriteLine("Deleted successfully");

```
