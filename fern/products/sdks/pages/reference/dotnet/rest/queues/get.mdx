Retrieve a specific queue.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- The queue object.

## **Example**

```csharp {9-10}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

queue = client.queues.get("queue-id");
Console.WriteLine(queue["name"], queue["current_size"]);

```
