Verify an MFA code submitted by the user.

## **Parameters**

<ParamField path="request_id" type="string" required={true} toc={true}>
  The MFA request ID returned from `sms()` or `call()`.
</ParamField>

<ParamField path="token" type="string" required={true} toc={true}>
  The verification code entered by the user.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`dict` -- Verification result indicating whether the code is valid.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

result = client.mfa.verify("your-mfa-request-id", token: "123456");
if result["success"]:
    Console.WriteLine("Verification successful");
else:
    Console.WriteLine("Invalid code");

```
