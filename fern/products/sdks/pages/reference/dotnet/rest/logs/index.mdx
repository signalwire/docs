Query read-only logs for messages, voice calls, faxes, and conferences. Each
log type is accessed as a sub-resource with `ListAsync()` and `GetAsync()` methods. Voice
logs also support listing individual call events.

Access via `client.logs` on a [`RestClient`][restclient] instance.

```csharp
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

voice_logs = client.logs.voice.list(page_size=5);
for log in voice_logs.get("data", []):
    Console.WriteLine(log["id"], log["from"], "->", log["to"]);

```
