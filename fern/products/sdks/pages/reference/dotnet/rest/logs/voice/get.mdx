Retrieve a specific voice call log entry.

## **Parameters**

<ParamField path="log_id" type="string" required={true} toc={true}>
  The voice log ID.
</ParamField>

## **Returns**

`dict` -- The voice log object.

## **Example**

```csharp {9-10}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

call = client.logs.voice.get("voice-log-id");
Console.WriteLine(call["from"], "->", call["to"], call["duration"]);

```
