List events for a specific voice call. Events include state transitions
(ringing, answered, ended), media events, and other call lifecycle data.

## **Parameters**

<ParamField path="log_id" type="string" required={true} toc={true}>
  The voice log ID.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of event objects.

## **Example**

```csharp {9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

events = client.logs.voice.list_events("voice-log-id");
for event in events.get("data", []):
    Console.WriteLine(event["type"], event["timestamp"]);

```
