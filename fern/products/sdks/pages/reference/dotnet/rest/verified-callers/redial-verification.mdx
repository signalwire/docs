Redial the verification call for a caller ID that has not yet been verified.
Use this when the original verification call was missed.

## **Parameters**

<ParamField path="caller_id" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`dict` -- Confirmation that the verification call has been re-initiated.

## **Example**

```csharp {9-9}
using SignalWire.REST;

client = RestClient(;
    project: "your-project-id",
    token: "your-api-token",
    host: "your-space.signalwire.com",
)

client.verified_callers.redial_verification("caller-id");

```
