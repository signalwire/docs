Add a question to a step's gather-info configuration.

## **Parameters**

<ParamField path="opts" type="Dictionary&lt;string, object&gt;" required={true} toc={true}>
  Question config with keys: `key` (required), `question` (required), `type`, `confirm`, `prompt`, `functions`.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.AddGatherQuestion(new Dictionary<string, object>
{
    ["key"] = "plant_name",
    ["question"] = "What type of plant do you have?",
});
step.AddGatherQuestion(new Dictionary<string, object>
{
    ["key"] = "issue",
    ["question"] = "What issue are you experiencing?",
    ["confirm"] = true
});
```
