Add a POM section to the context prompt.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  Section body text.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.AddSection("Expertise", "You specialize in tropical and desert plants.");
```
