Mark this step as a terminal step.

## **Parameters**

<ParamField path="end" type="bool" required={true} toc={true}>
  `true` to mark this step as the end of the workflow.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
ctx.AddStep("farewell")
   .SetText("Thank you for calling the Plant Care Hotline. Happy gardening!")
   .SetEnd(true);
```
