Automatically advance to the next step.

## **Parameters**

<ParamField path="skip" type="bool" required={true} toc={true}>
  `true` to auto-advance.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetSkipToNextStep(true);
```
