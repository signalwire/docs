Enable full conversation reset for a context.

## **Parameters**

<ParamField path="fullReset" type="bool" required={true} toc={true}>
  `true` to enable full reset.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetFullReset(true);
```
