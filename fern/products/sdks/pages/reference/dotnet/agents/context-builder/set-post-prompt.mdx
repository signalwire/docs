Set the post-prompt for a context.

## **Parameters**

<ParamField path="postPrompt" type="string" required={true} toc={true}>
  Post-prompt text for summarization.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetPostPrompt("Summarize the plant care advice given and any follow-up actions.");
```
