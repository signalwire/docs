Add a POM section with bullet points to a step.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  The section title.
</ParamField>

<ParamField path="bullets" type="List&lt;string&gt;" required={true} toc={true}>
  The bullet point items.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
var step = ctx.AddStep("services");
step.AddBullets("Available plant services", new List<string>
{
    "Watering schedule consultation",
    "Soil and fertilizer recommendations",
    "Pest identification and treatment"
});
```
