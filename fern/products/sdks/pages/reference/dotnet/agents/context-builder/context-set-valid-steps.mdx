Set valid step transitions for the context.

## **Parameters**

<ParamField path="steps" type="List&lt;string&gt;" required={true} toc={true}>
  Allowed step names.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetValidSteps(new List<string> { "greeting", "diagnose" });
```
