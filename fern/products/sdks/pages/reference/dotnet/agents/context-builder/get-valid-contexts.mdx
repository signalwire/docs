Get the list of valid context transitions.

## **Parameters**

None.

## **Returns**

`List<string>?` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

var contexts = ctx.GetValidContexts();
// ["watering", "fertilizing", "pest-control"]
```
