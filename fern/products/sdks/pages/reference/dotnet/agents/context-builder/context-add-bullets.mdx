Add a POM bullet section to the context prompt.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title.
</ParamField>

<ParamField path="bullets" type="List&lt;string&gt;" required={true} toc={true}>
  Bullet point items.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.AddBullets("Specialties", new List<string> { "Tropical plants", "Succulents", "Herb gardens" });
```
