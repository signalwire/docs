Add a new step to a context.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Unique name for the step.
</ParamField>

<ParamField path="opts" type="Dictionary&lt;string, object&gt;?" toc={true}>
  Optional initial configuration.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

var ctx = builder.AddContext("default");
var greeting = ctx.AddStep("greeting");
greeting.SetText("Welcome to the Plant Care Hotline!");
```
