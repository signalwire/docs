Set which steps the AI can transition to from this step.

## **Parameters**

<ParamField path="steps" type="List&lt;string&gt;" required={true} toc={true}>
  Step names the AI may navigate to.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetValidSteps(new List<string> { "next", "farewell" });
```
