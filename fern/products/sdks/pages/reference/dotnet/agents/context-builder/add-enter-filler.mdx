Add a single enter-filler phrase for a language.

## **Parameters**

<ParamField path="lang" type="string" required={true} toc={true}>
  Language code.
</ParamField>

<ParamField path="text" type="string" required={true} toc={true}>
  The filler phrase.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.AddEnterFiller("en-US", "One moment while I look up that plant...");
```
