Static factory that creates a new `ContextBuilder` pre-populated with a single empty context.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Name for the initial context.
</ParamField>

## **Returns**

`ContextBuilder` -- A new builder instance with one context ready for step definitions.

## **Example**

```csharp
using SignalWire.Contexts;

var builder = ContextBuilder.CreateSimpleContext("default");
var ctx = builder.GetContext("default");
ctx.AddStep("greeting").SetText("Welcome to the Plant Care Hotline.");
```
