Add a question to a GatherInfo instance.

## **Parameters**

<ParamField path="opts" type="Dictionary&lt;string, object&gt;" required={true} toc={true}>
  Question definition with keys: `key` (required), `question` (required), `type`, `confirm`, `prompt`, `functions`.
</ParamField>

## **Returns**

`GatherInfo` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

var gather = new GatherInfo(outputKey: "plant_info", completionAction: "next_step");
gather.AddQuestion(new Dictionary<string, object>
{
    ["key"] = "species",
    ["question"] = "What species of plant do you have?",
});
gather.AddQuestion(new Dictionary<string, object>
{
    ["key"] = "sunlight",
    ["question"] = "How much sunlight does it receive?",
    ["confirm"] = true
});
```
