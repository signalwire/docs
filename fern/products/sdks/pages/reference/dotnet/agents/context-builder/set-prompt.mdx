Set the plain-text prompt for a context.

## **Parameters**

<ParamField path="prompt" type="string" required={true} toc={true}>
  The context-level prompt text.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetPrompt("You are a plant care specialist. Help callers with their houseplants.");
```
