Configure structured information gathering for a step.

## **Parameters**

<ParamField path="opts" type="Dictionary&lt;string, object&gt;" required={true} toc={true}>
  Configuration with optional keys: `output_key`, `completion_action`, `prompt`.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetGatherInfo(new Dictionary<string, object>
{
    ["output_key"] = "plant_details",
    ["completion_action"] = "next_step",
    ["prompt"] = "I need a few details about your plant."
});
```
