Add a POM section with a title and body to a step.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  The section title.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  The section body text.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
var step = ctx.AddStep("plant-info");
step.AddSection("Watering schedule", "Most houseplants need watering once per week.");
```
