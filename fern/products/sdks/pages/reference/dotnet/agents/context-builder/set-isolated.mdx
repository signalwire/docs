Mark a context as isolated.

## **Parameters**

<ParamField path="isolated" type="bool" required={true} toc={true}>
  `true` to isolate this context.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetIsolated(true);
```
