Remove a step from a context by name.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The step name to remove.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.RemoveStep("old_step");
```
