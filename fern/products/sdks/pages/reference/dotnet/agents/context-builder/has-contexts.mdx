Check whether the builder contains at least one context definition.

## **Parameters**

None.

## **Returns**

`bool` -- `true` if one or more contexts exist, `false` otherwise.

## **Example**

```csharp
using SignalWire.Contexts;

var builder = new ContextBuilder();
Console.WriteLine(builder.HasContexts()); // False

builder.AddContext("default");
Console.WriteLine(builder.HasContexts()); // True
```
