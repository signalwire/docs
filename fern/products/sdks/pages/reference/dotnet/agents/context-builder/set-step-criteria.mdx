Set criteria that must be met before executing this step.

## **Parameters**

<ParamField path="criteria" type="string" required={true} toc={true}>
  A natural-language description of the criteria.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetStepCriteria("Only proceed if the caller has identified which plant species they need help with.");
```
