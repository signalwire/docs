Get all steps in a context.

## **Parameters**

None.

## **Returns**

`Dictionary<string, Step>` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

var steps = ctx.GetSteps();
foreach (var (name, step) in steps)
{
    Console.WriteLine($"Step: {name}");
}
```
