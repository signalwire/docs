Set the system prompt for a context.

## **Parameters**

<ParamField path="systemPrompt" type="string" required={true} toc={true}>
  The system prompt text.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetSystemPrompt("You are a certified horticulturist.");
```
