Set the plain-text prompt for a step.

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  The prompt text for the step.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
var step = ctx.AddStep("greeting");
step.SetText("Welcome to the Plant Care Hotline. How can I help with your plants today?");
```
