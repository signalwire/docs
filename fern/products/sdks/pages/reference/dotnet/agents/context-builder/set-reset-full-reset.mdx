Perform a full conversation reset when entering this step.

## **Parameters**

<ParamField path="f" type="bool" required={true} toc={true}>
  `true` to fully reset.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetResetFullReset(true);
```
