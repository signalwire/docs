Enable conversation consolidation when entering this step.

## **Parameters**

<ParamField path="c" type="bool" required={true} toc={true}>
  `true` to consolidate.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetResetConsolidate(true);
```
