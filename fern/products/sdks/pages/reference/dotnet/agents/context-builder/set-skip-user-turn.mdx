Skip waiting for user input after this step.

## **Parameters**

<ParamField path="skip" type="bool" required={true} toc={true}>
  `true` to skip the user turn.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetSkipUserTurn(true);
```
