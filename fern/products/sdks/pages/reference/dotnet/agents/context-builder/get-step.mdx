Retrieve a step by name.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The step name to look up.
</ParamField>

## **Returns**

`Step?` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

var step = ctx.GetStep("diagnose");
if (step != null)
{
    step.SetText("Updated diagnosis instructions for tropical plants.");
}
```
