Get the ordered list of step names.

## **Parameters**

None.

## **Returns**

`List<string>` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

var order = ctx.GetStepOrder();
// ["greeting", "diagnose", "recommend", "farewell"]
```
