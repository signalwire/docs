Add a single exit-filler phrase for a language.

## **Parameters**

<ParamField path="lang" type="string" required={true} toc={true}>
  Language code.
</ParamField>

<ParamField path="text" type="string" required={true} toc={true}>
  The filler phrase.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.AddExitFiller("en-US", "Alright, let me switch topics...");
```
