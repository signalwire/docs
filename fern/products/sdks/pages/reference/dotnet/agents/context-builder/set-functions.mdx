Set the SWAIG functions available during this step.

## **Parameters**

<ParamField path="functions" type="object" required={true} toc={true}>
  Function names or definitions available to this step.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetFunctions(new List<string> { "lookup_plant", "check_soil_moisture" });
```
