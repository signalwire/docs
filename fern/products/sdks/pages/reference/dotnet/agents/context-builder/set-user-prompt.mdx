Set the user prompt for a context.

## **Parameters**

<ParamField path="userPrompt" type="string" required={true} toc={true}>
  The user prompt text.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetUserPrompt("Describe the plant issue you are experiencing.");
```
