Set filler phrases played when exiting a context.

## **Parameters**

<ParamField path="fillers" type="Dictionary&lt;string, List&lt;string&gt;&gt;" required={true} toc={true}>
  Language-keyed filler phrases.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetExitFillers(new Dictionary<string, List<string>>
{
    ["en-US"] = new() { "Moving on to the next topic..." }
});
```
