Move a step to a different position in the step order.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The step name to move.
</ParamField>

<ParamField path="position" type="int" required={true} toc={true}>
  Zero-based target index.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.MoveStep("farewell", 2);
```
