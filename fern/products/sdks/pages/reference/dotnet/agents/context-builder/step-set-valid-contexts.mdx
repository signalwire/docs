Set which contexts the AI can switch to from this step.

## **Parameters**

<ParamField path="contexts" type="List&lt;string&gt;" required={true} toc={true}>
  Context names the AI may switch to.
</ParamField>

## **Returns**

`Step` -- Returns self for method chaining.

## **Example**

```csharp
step.SetValidContexts(new List<string> { "watering", "fertilizing" });
```
