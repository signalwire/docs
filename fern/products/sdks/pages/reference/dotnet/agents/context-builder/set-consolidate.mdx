Enable conversation consolidation for a context.

## **Parameters**

<ParamField path="consolidate" type="bool" required={true} toc={true}>
  `true` to enable consolidation.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetConsolidate(true);
```
