Add a POM section to the system prompt.

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  Section body text.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.AddSystemSection("Credentials", "Master Gardener certification, 10 years experience.");
```
