Set which contexts the AI can switch to.

## **Parameters**

<ParamField path="contexts" type="List&lt;string&gt;" required={true} toc={true}>
  Allowed context names.
</ParamField>

## **Returns**

`Context` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.Contexts;

ctx.SetValidContexts(new List<string> { "watering", "fertilizing", "pest-control" });
```
