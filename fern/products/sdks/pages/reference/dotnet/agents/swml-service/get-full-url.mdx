Build the full URL for this service.

## **Parameters**

<ParamField path="includeAuth" type="bool" default="false" toc={true}>
  Include Basic Auth credentials in the URL.
</ParamField>

## **Returns**

`string`

## **Example**

```csharp
using SignalWire.SWML;

var url = service.GetFullUrl();
// "http://0.0.0.0:3000/garden-line"
```
