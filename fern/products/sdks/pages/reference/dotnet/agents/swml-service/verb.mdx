Add a validated SWML verb to the service document.

## **Parameters**

<ParamField path="verbName" type="string" required={true} toc={true}>
  The SWML verb name.
</ParamField>

<ParamField path="config" type="object?" toc={true}>
  Verb configuration.
</ParamField>

<ParamField path="section" type="string" default="main" toc={true}>
  Target section.
</ParamField>

## **Returns**

`Service`

## **Example**

```csharp
using SignalWire.SWML;

var service = new Service(new ServiceOptions { Name = "garden-line" });
service.Verb("answer", new Dictionary<string, object>())
       .Verb("play", new Dictionary<string, object> { ["url"] = "https://example.com/garden-tips.mp3" })
       .Verb("hangup", new Dictionary<string, object>());
```
