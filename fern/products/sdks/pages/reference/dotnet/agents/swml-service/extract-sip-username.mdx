Extract a SIP username from a request body.

## **Parameters**

<ParamField path="body" type="Dictionary&lt;string, object?&gt;?" toc={true}>
  The parsed request body.
</ParamField>

## **Returns**

`string?` -- The extracted username, or `null` if not found.

## **Example**

```csharp
using SignalWire.SWML;

var body = new Dictionary<string, object?> { ["to"] = "sip:greenhouse@example.com" };
var username = Service.ExtractSipUsername(body);
// "greenhouse"
```
