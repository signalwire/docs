Add a sleep verb to pause execution.

## **Parameters**

<ParamField path="milliseconds" type="int" required={true} toc={true}>
  Duration to sleep in milliseconds.
</ParamField>

<ParamField path="section" type="string" default="main" toc={true}>
  Target section.
</ParamField>

## **Returns**

`Service`

## **Example**

```csharp
using SignalWire.SWML;

service.Verb("answer", new Dictionary<string, object>())
       .Sleep(2000)
       .Verb("play", config);
```
