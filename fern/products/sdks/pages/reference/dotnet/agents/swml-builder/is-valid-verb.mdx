Check whether a verb name is valid per the SWML schema.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The verb name to check.
</ParamField>

## **Returns**

`bool` -- `true` if the verb is defined in the schema.

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
Console.WriteLine(Schema.Instance.IsValidVerb("answer")); // True
Console.WriteLine(Schema.Instance.IsValidVerb("plant"));  // False
```
