Append a verb to a named section.

## **Parameters**

<ParamField path="section" type="string" required={true} toc={true}>
  Target section.
</ParamField>

<ParamField path="verbName" type="string" required={true} toc={true}>
  The SWML verb name.
</ParamField>

<ParamField path="config" type="object?" toc={true}>
  Verb configuration.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
doc.AddSection("error_handler");
doc.AddVerbToSection("error_handler", "say", new Dictionary<string, object>
{
    ["text"] = "The greenhouse system encountered an error."
});
```
