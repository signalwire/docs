Get metadata for a specific SWML verb.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The verb name to look up.
</ParamField>

## **Returns**

`VerbInfo?` -- Verb metadata with `Name`, `SchemaName`, and `Definition`.

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
var info = Schema.Instance.GetVerb("answer");
if (info != null)
{
    Console.WriteLine($"Verb: {info.Name}, Schema key: {info.SchemaName}");
}
```
