Check whether a named section exists.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The section name to check.
</ParamField>

## **Returns**

`bool` -- `true` if the section exists.

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
Console.WriteLine(doc.HasSection("main"));   // True
Console.WriteLine(doc.HasSection("custom")); // False
```
