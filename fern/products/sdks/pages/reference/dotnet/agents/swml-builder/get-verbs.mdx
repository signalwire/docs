Get a copy of all verbs in a section.

## **Parameters**

<ParamField path="section" type="string" default="main" toc={true}>
  Section name.
</ParamField>

## **Returns**

`List<Dictionary<string, object?>>` -- A copy of the verbs in the section.

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
var verbs = doc.GetVerbs("main");
Console.WriteLine($"Main section has {verbs.Count} verbs");
```
