Append a verb to the main section.

## **Parameters**

<ParamField path="verbName" type="string" required={true} toc={true}>
  The SWML verb name.
</ParamField>

<ParamField path="config" type="object?" toc={true}>
  Verb configuration.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
doc.AddVerb("answer", new Dictionary<string, object>());
doc.AddVerb("play", new Dictionary<string, object> { ["url"] = "https://example.com/garden-hold.mp3" });
```
