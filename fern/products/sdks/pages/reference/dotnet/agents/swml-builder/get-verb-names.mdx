Get a sorted list of all valid SWML verb names.

## **Parameters**

None.

## **Returns**

`List<string>`

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
var verbs = Schema.Instance.GetVerbNames();
// ["ai", "answer", "hangup", "play", "say", ...]
```
