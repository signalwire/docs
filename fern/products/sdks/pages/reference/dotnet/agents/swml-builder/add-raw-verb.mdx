Append a pre-formatted verb dictionary to a section.

## **Parameters**

<ParamField path="section" type="string" required={true} toc={true}>
  Target section.
</ParamField>

<ParamField path="verbHash" type="Dictionary&lt;string, object?&gt;" required={true} toc={true}>
  The pre-formatted verb dictionary.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.SWML;

var doc = new Document();
doc.AddRawVerb("main", new Dictionary<string, object?>
{
    ["play"] = new Dictionary<string, object> { ["url"] = "https://example.com/plant-tips.mp3" }
});
```
