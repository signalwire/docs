Set global error keys for all webhooks.

## **Parameters**

<ParamField path="keys" type="List&lt;string&gt;" required={true} toc={true}>
  JSON keys that signal an error.
</ParamField>

## **Returns**

`DataMap` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.DataMap;

dm.GlobalErrorKeys(new List<string> { "error", "error_message" });
```
