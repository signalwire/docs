Set query parameters for the most recent webhook.

## **Parameters**

<ParamField path="data" type="Dictionary&lt;string, object&gt;" required={true} toc={true}>
  Key-value pairs for query parameters.
</ParamField>

## **Returns**

`DataMap` -- Returns self for method chaining.

## **Example**

```csharp
using SignalWire.DataMap;

dm.Webhook("GET", "https://api.plants.example.com/lookup")
  .Params(new Dictionary<string, object> { ["format"] = "json" });
```
