Static factory that creates a complete SWAIG function definition with a single webhook.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Function name.
</ParamField>

<ParamField path="purpose" type="string" required={true} toc={true}>
  Description of the function.
</ParamField>

<ParamField path="parameters" type="List&lt;Dictionary&lt;string, object&gt;&gt;" required={true} toc={true}>
  Parameter definitions with `name`, `type`, `description`, and optionally `required` and `enum`.
</ParamField>

<ParamField path="method" type="string" required={true} toc={true}>
  HTTP method.
</ParamField>

<ParamField path="url" type="string" required={true} toc={true}>
  The webhook URL.
</ParamField>

<ParamField path="output" type="object" required={true} toc={true}>
  Output configuration.
</ParamField>

<ParamField path="headers" type="Dictionary&lt;string, string&gt;?" toc={true}>
  Optional HTTP headers.
</ParamField>

## **Returns**

`Dictionary<string, object>` -- A complete SWAIG function definition.

## **Example**

```csharp
using SignalWire.DataMap;

var tool = DataMap.CreateSimpleApiTool(
    name: "get_plant_info",
    purpose: "Look up care instructions for a plant species",
    parameters: new List<Dictionary<string, object>>
    {
        new() { ["name"] = "species", ["type"] = "string", ["description"] = "Plant species name", ["required"] = true }
    },
    method: "GET",
    url: "https://api.plants.example.com/species",
    output: new Dictionary<string, object>
    {
        ["response"] = "Care for %{input.species}: water %{watering}, sunlight %{sunlight}"
    }
);
agent.RegisterSwaigFunction(tool);
```
