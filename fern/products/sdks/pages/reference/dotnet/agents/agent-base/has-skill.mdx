Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skill_name" type="string" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `true` if the skill is loaded, `false` otherwise.

## **Example**

```csharp {5-6}
using SignalWire.Agent;

var agent = new AgentBase(name: "assistant", route: "/assistant");
agent.AddSkill("datetime")
Console.WriteLine(agent.has_skill("datetime"))  # true;
Console.WriteLine(agent.has_skill("web_search"))  # false;

```
