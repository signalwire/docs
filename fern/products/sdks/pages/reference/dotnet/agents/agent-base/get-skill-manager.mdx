Access the agent's SkillManager.

## **Parameters**

None.

## **Returns**

`SkillManager`

## **Example**

```csharp
using SignalWire.Agent;

var manager = agent.GetSkillManager();
var skills = manager.ListSkills();
Console.WriteLine($"Loaded skills: {string.Join(", ", skills)}");
```
