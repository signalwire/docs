Dispatch a SWAIG function call to its registered handler.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The function name to dispatch.
</ParamField>

<ParamField path="args" type="Dictionary&lt;string, object&gt;" required={true} toc={true}>
  The parsed arguments.
</ParamField>

<ParamField path="rawData" type="Dictionary&lt;string, object?&gt;" required={true} toc={true}>
  The raw request data.
</ParamField>

## **Returns**

`FunctionResult?`

## **Example**

```csharp
using SignalWire.Agent;

var result = agent.OnFunctionCall(
    "identify_plant",
    new Dictionary<string, object> { ["description"] = "tall with broad green leaves" },
    rawData
);
```
