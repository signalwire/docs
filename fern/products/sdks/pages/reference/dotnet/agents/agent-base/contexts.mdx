Access the agent's ContextBuilder for multi-step workflows.

## **Parameters**

None.

## **Returns**

`ContextBuilder`

## **Example**

```csharp
using SignalWire.Agent;

var agent = new AgentBase(new AgentOptions { Name = "botanist", Route = "/" });

var builder = agent.Contexts();
var ctx = builder.AddContext("default");
ctx.AddStep("greeting").SetText("Welcome to the Plant Nursery Hotline!");
ctx.AddStep("farewell").SetText("Happy gardening!").SetEnd(true);
```
