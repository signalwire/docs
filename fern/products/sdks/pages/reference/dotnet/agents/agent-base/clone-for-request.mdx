Create a shallow copy of the agent for request-isolated modifications.

## **Parameters**

None.

## **Returns**

`AgentBase`

## **Example**

```csharp
using SignalWire.Agent;

var requestAgent = agent.CloneForRequest();
requestAgent.SetPromptText("Modified prompt for this specific request.");
var swml = requestAgent.RenderSwml();
```
