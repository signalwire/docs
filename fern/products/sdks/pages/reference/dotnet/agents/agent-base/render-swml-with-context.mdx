Render a SWML document with request-specific context data.

## **Parameters**

<ParamField path="requestBody" type="Dictionary&lt;string, object?&gt;?" toc={true}>
  The parsed request body.
</ParamField>

<ParamField path="queryParams" type="Dictionary&lt;string, object?&gt;?" toc={true}>
  Query string parameters.
</ParamField>

<ParamField path="headers" type="Dictionary&lt;string, string&gt;" required={true} toc={true}>
  HTTP headers from the request.
</ParamField>

## **Returns**

`Dictionary<string, object>`

## **Example**

```csharp
using SignalWire.Agent;

var swml = agent.RenderSwmlWithContext(
    requestBody: parsedBody,
    queryParams: null,
    headers: requestHeaders
);
```
