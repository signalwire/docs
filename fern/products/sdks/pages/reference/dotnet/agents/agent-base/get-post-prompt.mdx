Retrieve the current post-prompt text.

## **Returns**

`Optional[str]` -- The post-prompt text, or `None` if not set.

## **Example**

```csharp {5}
using SignalWire.Agent;

var agent = new AgentBase(name: "support", route: "/support");
agent.SetPostPrompt("Summarize this call as JSON with intent, resolution, and sentiment.")
post_prompt = agent.get_post_prompt();
if post_prompt:
    Console.WriteLine(post_prompt);

```
