Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`str` -- The agent name.

## **Example**

```csharp {4}
using SignalWire.Agent;

var agent = new AgentBase(name: "support", route: "/support");
Console.WriteLine(agent.get_name())  # "support";

```
