Register multiple SWAIG tool definitions at once.

## **Parameters**

<ParamField path="toolDefs" type="List&lt;Dictionary&lt;string, object&gt;&gt;" required={true} toc={true}>
  List of raw SWAIG function definitions.
</ParamField>

## **Returns**

`AgentBase`

## **Example**

```csharp
using SignalWire.Agent;

var agent = new AgentBase(new AgentOptions { Name = "botanist", Route = "/botanist" });
agent.DefineTools(new List<Dictionary<string, object>>
{
    new() { ["function"] = "identify_plant", ["purpose"] = "Identify a plant by description" },
    new() { ["function"] = "watering_schedule", ["purpose"] = "Get watering schedule" }
});
```
