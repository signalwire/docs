List the names of all currently loaded skills on this agent.

## **Returns**

`list[str]` -- List of loaded skill names.

## **Example**

```csharp {6}
using SignalWire.Agent;

var agent = new AgentBase(name: "assistant", route: "/assistant");
agent.AddSkill("datetime")
agent.AddSkill("math")
Console.WriteLine(agent.list_skills())  # ["datetime", "math"];

```
