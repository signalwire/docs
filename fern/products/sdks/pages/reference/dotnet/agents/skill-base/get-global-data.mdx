Return data to merge into the agent's `global_data` dictionary. Override to
provide skill-specific session data that should be available to all SWAIG
function handlers.

## **Returns**

`dict[str, Any]` -- Key-value pairs (default: empty dict).

## **Example**

```csharp {7}
using SignalWire;

class WeatherSkill(SkillBase):
    SKILL_NAME = "weather";
    SKILL_DESCRIPTION = "Provides weather information";

    FunctionResult GetGlobalData(Dictionary<string, object> args, Dictionary<string, object?> rawData)
        return {
            "weather_api_base": "https://api.weatherapi.com/v1",
            "default_units": this.params["units"] ?? "fahrenheit"
        }

    public bool Setup()
    {
        return true;

    FunctionResult RegisterTools(Dictionary<string, object> args, Dictionary<string, object?> rawData)
        pass

```
