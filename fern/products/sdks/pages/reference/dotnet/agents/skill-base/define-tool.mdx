Register a tool with the parent agent, automatically merging any `swaig_fields`
configured for this skill. Skills should use this method instead of calling
`this.Agent.DefineTool()` directly.

## **Parameters**

<ParamField path="**kwargs" type="object" required={true} toc={true}>
  All arguments supported by `agent.define_tool()`: `name`, `Description`,
  `parameters`, `handler`, `secure`, `fillers`, `wait_file`, etc. Explicit
  values take precedence over `swaig_fields`.
</ParamField>

## **Returns**

`None`

## **Example**

```csharp {12}
using SignalWire;
using SignalWire.SWAIG;

class SearchSkill(SkillBase):
    SKILL_NAME = "search";
    SKILL_DESCRIPTION = "Search the web for information";

    public bool Setup()
    {
        return true;

    FunctionResult RegisterTools(Dictionary<string, object> args, Dictionary<string, object?> rawData)
        this.DefineTool(
            name: "search_web",
            description: "Search the web for information",
            handler: this._handle_search,
            parameters={
                "type": "object",
                "properties": {
                    "query": {"type": "string", "description": "Search query"}
                },
                "required": ["query"]
            },
            secure: true,
            fillers: {"en-US": ["Let me search for that..."]}
        )

    FunctionResult HandleSearch(Dictionary<string, object> args, Dictionary<string, object?> rawData)
        query = args["query"] ?? "";
        return new FunctionResult($"Results for: {query}");

```
