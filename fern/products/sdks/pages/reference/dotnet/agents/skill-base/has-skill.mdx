Check if a skill is loaded.

## **Parameters**

<ParamField path="key" type="string" required={true} toc={true}>
  The skill instance key.
</ParamField>

## **Returns**

`bool` -- `true` if loaded.

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
if (manager.HasSkill("web_search")) Console.WriteLine("Available");
```
