Get the factory function for a skill by name.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The skill name.
</ParamField>

## **Returns**

`Func<SkillBase>?` -- The factory, or `null`.

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
var factory = SkillRegistry.Instance.GetFactory("web_search");
```
