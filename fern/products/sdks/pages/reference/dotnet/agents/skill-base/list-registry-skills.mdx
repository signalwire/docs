List all registered skill names.

## **Parameters**

None.

## **Returns**

`List<string>` -- Sorted list of all skill names.

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
var all = SkillRegistry.Instance.ListSkills();
// ["api_ninjas_trivia", "claude_skills", ...]
```
