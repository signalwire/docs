Register a custom skill factory in the global registry.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The skill name.
</ParamField>

<ParamField path="factory" type="Func&lt;SkillBase&gt;" required={true} toc={true}>
  Factory function.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
SkillRegistry.Instance.RegisterSkill("plant_identifier", () => new PlantIdentifierSkill());
agent.AddSkill("plant_identifier");
```
