Validate required environment variables for a skill.

## **Parameters**

None.

## **Returns**

`List<string>` -- Names of missing variables.

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
var missing = skill.ValidateEnvVars();
if (missing.Count > 0) Console.WriteLine($"Missing: {string.Join(", ", missing)}");
```
