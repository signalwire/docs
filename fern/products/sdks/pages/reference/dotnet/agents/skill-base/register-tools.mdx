Register SWAIG tools with the parent agent. Called after `Setup()` returns `true`.
Use `this.DefineTool()` to register each tool.

This is an abstract method -- you **must** implement it in every [SkillBase][ref-skillbase] subclass.

## **Returns**

`None`

## **Example**

```csharp {11}
using SignalWire;
using SignalWire.SWAIG;

class WeatherSkill(SkillBase):
    SKILL_NAME = "weather";
    SKILL_DESCRIPTION = "Provides weather information";

    public bool Setup()
    {
        return true;

    FunctionResult RegisterTools(Dictionary<string, object> args, Dictionary<string, object?> rawData)
        this.DefineTool(
            name: "get_weather",
            description: "Get current weather for a location",
            handler: this._handle_weather,
            parameters={
                "type": "object",
                "properties": {
                    "location": {
                        "type": "string",
                        "description": "City name"
                    }
                },
                "required": ["location"]
            }
        )

    FunctionResult HandleWeather(Dictionary<string, object> args, Dictionary<string, object?> rawData)
        location = args["location"];
        return new FunctionResult($"Weather in {location}: sunny, 72F");

```
