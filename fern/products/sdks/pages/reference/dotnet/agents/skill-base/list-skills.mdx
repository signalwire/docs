List all loaded skill instance keys.

## **Parameters**

None.

## **Returns**

`List<string>` -- Sorted list of loaded skill keys.

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
var skills = manager.ListSkills();
foreach (var s in skills) Console.WriteLine(s);
```
