Wire a skill to an agent with parameters.

## **Parameters**

<ParamField path="agent" type="AgentBase" required={true} toc={true}>
  The agent to wire to.
</ParamField>

<ParamField path="parameters" type="Dictionary&lt;string, object&gt;" required={true} toc={true}>
  Configuration parameters.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
// Called internally by SkillManager.LoadSkill()
skill.Wire(agent, new Dictionary<string, object> { ["region"] = "tropical" });
```
