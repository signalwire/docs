Get a loaded skill instance by key.

## **Parameters**

<ParamField path="key" type="string" required={true} toc={true}>
  The skill instance key.
</ParamField>

## **Returns**

`SkillBase?` -- The skill instance, or `null`.

## **Example**

```csharp
using SignalWire.Skills;

var manager = agent.GetSkillManager();
var skill = manager.GetSkill("web_search");
```
