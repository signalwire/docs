[functionresult]: /docs/sdks/reference/dotnet/agents/function-result

Set or update key-value pairs in the global session data. Global data persists
for the entire agent session and is accessible in prompt variable expansion
(`${global_data.key}`) and by all SWAIG functions.

## **Parameters**

<ParamField path="data" type="Dictionary<string, object>" required={true} toc={true}>
  Dictionary of key-value pairs to merge into the global data store. Existing
  keys are overwritten; new keys are added.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```csharp {11}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult SaveCustomerInfo(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    return (;
        FunctionResult($"I've noted your information, {args.get('name')}.");
        .update_global_data({
            "customer_id": args["customer_id"],
            "customer_name": args["name"],
            "verified": true
        })
    )

agent.Serve();

```
