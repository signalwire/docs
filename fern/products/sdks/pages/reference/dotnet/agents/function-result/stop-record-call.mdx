Stop an active background call recording.

## **Parameters**

<ParamField path="control_id" type="string?" default="None" toc={true}>
  Identifier of the recording to stop. If not provided, the most recent
  recording is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```csharp {11}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult StopRecording(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    return (;
        FunctionResult("Recording stopped.");
        .stop_record_call(control_id: "main_recording");
    )

agent.Serve();

```
