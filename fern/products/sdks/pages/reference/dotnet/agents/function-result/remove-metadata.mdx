Remove one or more keys from the current function's metadata store.

## **Parameters**

<ParamField path="keys" type="string | List<string>" required={true} toc={true}>
  A single key string or a list of key strings to remove from metadata.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```csharp {11}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult ClearMetadata(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    return (;
        FunctionResult("Metadata cleared.");
        .remove_metadata(["timestamp", "user_id"]);
    )

agent.Serve();

```
