Send an SMS or MMS message to a phone number. At least one of `body` or `media`
must be provided. Generates a SWML `send_sms` verb under the hood.

## **Parameters**

<ParamField path="to_number" type="string" required={true} toc={true}>
  Destination phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="from_number" type="string" required={true} toc={true}>
  Sender phone number in E.164 format. Must be a number in your SignalWire project.
</ParamField>

<ParamField path="body" type="string?" default="None" toc={true}>
  Text body of the message.
</ParamField>

<ParamField path="media" type="List<string>?" default="None" toc={true}>
  List of media URLs to include as MMS attachments.
</ParamField>

<ParamField path="tags" type="List<string>?" default="None" toc={true}>
  Tags to associate with the message for searching and filtering in the
  SignalWire dashboard.
</ParamField>

<ParamField path="region" type="string?" default="None" toc={true}>
  Region to originate the message from.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Text Message

```csharp {13}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult SendConfirmation(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    phone = args["phone_number"];
    order_id = args["order_id"];
    return (;
        FunctionResult("I've sent you a confirmation text.");
        .send_sms(;
            to_number: phone,
            from_number: "+15559876543",
            body=$"Your order {order_id} has been confirmed!";
        )
    )

agent.Serve();

```

### MMS with Media

```csharp {13}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult SendReceipt(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    phone = args["phone_number"];
    receipt_url = args["receipt_url"];
    return (;
        FunctionResult("I've sent your receipt.");
        .send_sms(;
            to_number: phone,
            from_number: "+15559876543",
            body: "Here's your receipt:",
            media: [receipt_url];
        )
    )

agent.Serve();

```
