[functionresult]: /docs/sdks/reference/dotnet/agents/function-result

Make the AI agent speak specific text immediately, bypassing the normal LLM
response flow. The text is spoken in the agent's configured voice.

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  Text for the agent to speak.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```csharp {12}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult AnnounceStatus(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    status = args["status"];
    return (;
        FunctionResult();
        .say($"Your order status is: {status}");
    )

agent.Serve();

```
