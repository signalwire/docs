Stop the currently playing background audio file.

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```csharp {11}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult StopHoldMusic(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    return (;
        FunctionResult("I'm back.");
        .stop_background_file();
    )

agent.Serve();

```
