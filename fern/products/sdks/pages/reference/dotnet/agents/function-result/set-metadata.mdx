[functionresult]: /docs/sdks/reference/dotnet/agents/function-result

Set function-scoped metadata. Metadata is scoped to the current function's
`meta_data_token`, so each function maintains its own independent metadata
store.

## **Parameters**

<ParamField path="data" type="Dictionary<string, object>" required={true} toc={true}>
  Dictionary of key-value pairs to store as metadata.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```csharp {11}
using SignalWire.Agent;
using SignalWire.SWAIG;

var agent = new AgentBase(name: "my-agent", route: "/agent");
agent.SetPromptText("You are a helpful assistant.")

FunctionResult TrackUsage(Dictionary<string, object> args, Dictionary<string, object?> rawData)
    return (;
        FunctionResult("Usage tracked.");
        .set_metadata({
            "last_action": "search",
            "timestamp": "2024-01-15T10:30:00Z",
            "user_id": args["user_id"]
        })
    )

agent.Serve();

```
