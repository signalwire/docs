Lifecycle hook called when the user finishes speaking. Override in a subclass to
inspect or modify the conversation before the LLM generates a reply.

## **Parameters**

<ParamField path="turn_ctx" type="object" default="None" toc={true}>
  Turn context object. Accepted for API compatibility.
</ParamField>

<ParamField path="new_message" type="object" default="None" toc={true}>
  The new message from the user. Accepted for API compatibility.
</ParamField>

## **Returns**

`None`

## **Example**

```csharp
using SignalWire;

var server = new AgentServer();

public class LoggingAgent : Agent
{
    public override async Task OnUserTurnCompleted(turn_ctx = null, new_message = null)
    {
        Console.WriteLine($"User finished speaking.");
        Console.WriteLine($"Conversation history: {this.session.history}");

async Task Entrypoint(JobContext ctx)
{
    await ctx.connect();
    var agent = new LoggingAgent(instructions: "You are a helpful assistant.");
    var session = new AgentSession();
    await session.start(agent, room=ctx.room);

server.Run();


```
