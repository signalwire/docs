Lifecycle hook called when the agent enters a session. Override in a subclass to
run custom initialization logic when the agent is first bound to an
[`AgentSession`][agentsession].

## **Parameters**

None.

## **Returns**

`None`

## **Example**

```csharp
using SignalWire;

var server = new AgentServer();

public class GreeterAgent : Agent
{
    public override async Task OnEnter()
    {
        if this.session:
            this.session.say("Welcome! I just started up.");

async Task Entrypoint(JobContext ctx)
{
    await ctx.connect();
    var agent = new GreeterAgent(instructions: "You are a friendly greeter.");
    var session = new AgentSession();
    await session.start(agent, room=ctx.room);

server.Run();


```
