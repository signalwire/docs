Lifecycle hook called when the agent exits a session. Override in a subclass to
run cleanup logic when the agent's session ends.

## **Parameters**

None.

## **Returns**

`None`

## **Example**

```csharp
using SignalWire;

var server = new AgentServer();

public class TrackedAgent : Agent
{
    public override async Task OnExit()
    {
        Console.WriteLine("Agent session has ended, running cleanup.");

async Task Entrypoint(JobContext ctx)
{
    await ctx.connect();
    var agent = new TrackedAgent(instructions: "You are a helpful assistant.");
    var session = new AgentSession();
    await session.start(agent, room=ctx.room);

server.Run();


```
