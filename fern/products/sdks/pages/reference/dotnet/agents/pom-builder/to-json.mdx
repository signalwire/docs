Convert the POM to a JSON string.

## **Returns**

`str` -- JSON representation of the POM.

## **Example**

```csharp {7}
using SignalWire;

pom = PomBuilder();
pom.AddSection("Role", body: "You are a helpful assistant.")
pom.AddSection("Rules", bullets: ["Be concise", "Be accurate"])

json_str = pom.to_json();
Console.WriteLine(json_str);

```
