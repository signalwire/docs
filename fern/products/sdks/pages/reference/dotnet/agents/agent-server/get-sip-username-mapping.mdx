Get the current SIP username-to-route mapping.

## **Parameters**

None.

## **Returns**

`Dictionary<string, string>`

## **Example**

```csharp
using SignalWire.Agent;

var mapping = server.GetSipUsernameMapping();
foreach (var (username, route) in mapping)
{
    Console.WriteLine($"  {username} -> {route}");
}
```
