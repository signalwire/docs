Serve static files alongside agent routes.

## **Parameters**

<ParamField path="directory" type="string" required={true} toc={true}>
  Path to the directory containing static files.
</ParamField>

<ParamField path="urlPrefix" type="string" default="/" toc={true}>
  URL path prefix for static files.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.Agent;

var server = new AgentServer(port: 3000);
server.Register(agent);
server.ServeStatic("./public", "/static");
await server.RunAsync();
```
