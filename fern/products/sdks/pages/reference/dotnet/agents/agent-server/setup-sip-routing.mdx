Enable SIP-based call routing across registered agents.

## **Parameters**

<ParamField path="route" type="string" default="/sip" toc={true}>
  URL path where SIP routing requests are handled.
</ParamField>

<ParamField path="autoMap" type="bool" default="true" toc={true}>
  Automatically generate SIP username mappings from agent names.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.Agent;

var server = new AgentServer(port: 3000);
server.Register(tropicalAgent);
server.Register(succulentAgent);
server.SetupSipRouting("/sip", autoMap: true);
await server.RunAsync();
```
