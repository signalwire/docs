Map a SIP username to an agent route.

## **Parameters**

<ParamField path="username" type="string" required={true} toc={true}>
  The SIP username to map.
</ParamField>

<ParamField path="route" type="string" required={true} toc={true}>
  The target agent route.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
using SignalWire.Agent;

server.SetupSipRouting();
server.RegisterSipUsername("tropical-plants", "/tropical");
server.RegisterSipUsername("succulents", "/succulent");
```
