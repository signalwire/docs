Validate a previously issued token by verifying its HMAC signature,
function name, call ID, and expiry timestamp.

## **Parameters**

<ParamField path="functionName" type="string" required={true} toc={true}>
  The expected function name.
</ParamField>

<ParamField path="callId" type="string" required={true} toc={true}>
  The expected call ID.
</ParamField>

<ParamField path="token" type="string" required={true} toc={true}>
  The base64url-encoded token to validate.
</ParamField>

## **Returns**

`bool` -- `true` if the token is valid and not expired.

## **Example**

```csharp
using SignalWire.Security;

var session = new SessionManager();
var callId = session.CreateSession();
var token = session.CreateToken("identify_plant", callId);
var isValid = session.ValidateToken("identify_plant", callId, token);
Console.WriteLine($"Token valid: {isValid}"); // True
```
