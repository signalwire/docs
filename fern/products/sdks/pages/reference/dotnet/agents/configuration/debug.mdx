Log a message at `Debug` level. Suppressed if the logger's current level
is higher or if logging is disabled via `SIGNALWIRE_LOG_MODE=off`.

## **Parameters**

<ParamField path="message" type="string" required={true} toc={true}>
  The message to log.
</ParamField>

## **Returns**

`void`

## **Example**

```csharp
var logger = Logger.GetLogger();
logger.Debug("Soil moisture sensor reading: nominal");
```
