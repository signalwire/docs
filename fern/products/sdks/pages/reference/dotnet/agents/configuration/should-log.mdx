Check if a message at the specified level would be emitted, based on the
current level threshold and suppression state.

## **Parameters**

<ParamField path="level" type="LogLevel" required={true} toc={true}>
  The log level to check.
</ParamField>

## **Returns**

`bool` -- `true` if the message would be logged.

## **Example**

```csharp
if (logger.ShouldLog(LogLevel.Debug))
{
    logger.Debug($"Sensor data: {ReadSensorData()}");
}
```
