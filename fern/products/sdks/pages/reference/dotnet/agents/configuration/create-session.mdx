Create a new session ID or confirm an existing one.

## **Parameters**

<ParamField path="callId" type="string?" toc={true}>
  An existing call ID to confirm. If `null`, a new one is generated.
</ParamField>

## **Returns**

`string` -- The session ID.

## **Example**

```csharp
using SignalWire.Security;

var session = new SessionManager();
var callId = session.CreateSession();
Console.WriteLine($"New session: {callId}");
```
