Generate a base64url-encoded token binding a function name and call ID
with an expiry timestamp and random nonce.

## **Parameters**

<ParamField path="functionName" type="string" required={true} toc={true}>
  The SWAIG function name to bind.
</ParamField>

<ParamField path="callId" type="string" required={true} toc={true}>
  The call ID to bind.
</ParamField>

## **Returns**

`string` -- A base64url-encoded signed token.

## **Example**

```csharp
using SignalWire.Security;

var session = new SessionManager(tokenExpirySecs: 3600);
var callId = session.CreateSession();
var token = session.CreateToken("identify_plant", callId);
```
