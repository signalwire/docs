Get the URL scheme based on SSL configuration.

## **Returns**

`str` -- `"https"` if SSL is enabled, `"http"` otherwise.

## **Example**

```csharp {4}
using SignalWire;

security = SecurityConfig();
scheme = security.get_url_scheme();

Console.WriteLine($"URL scheme: {scheme}")  # "http" (default) or "https";

```
