Thread-safe factory that returns an existing logger or creates a new one.
Level is read from `SIGNALWIRE_LOG_LEVEL` (debug, info, warn, error). Defaults to `info`.

## **Parameters**

<ParamField path="name" type="string" default="signalwire" toc={true}>
  The logger name.
</ParamField>

## **Returns**

`Logger` -- The logger instance.

## **Example**

```csharp
using SignalWire.Logging;

var logger = Logger.GetLogger("greenhouse");
logger.Info("Greenhouse monitoring initialized");
```
