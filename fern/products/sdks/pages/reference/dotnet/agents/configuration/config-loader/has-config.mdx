Check whether a configuration file was successfully loaded.

## **Returns**

`bool` -- `true` if a config file was found and loaded.

## **Example**

```csharp {4}
using SignalWire;

loader = ConfigLoader();
if loader.has_config():
    Console.WriteLine($"Loaded config from: {loader.get_config_file()}");
else:
    Console.WriteLine("No config file found, using defaults");

```
