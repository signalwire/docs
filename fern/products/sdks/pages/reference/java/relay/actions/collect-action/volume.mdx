Adjust the prompt playback volume.

## **Parameters**

<ParamField path="volume" type="double" required={true} toc={true}>
  Volume adjustment in dB. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`Map<String, Object>` -- Server acknowledgment.

## **Example**

```java {19}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()
    var action = await call.play_and_collect( [{"type": "tts", "text": "Press 1 for sales, 2 for support."}], {"digits": {"max": 1, "digit_timeout": 5.0}}, );

    // Increase prompt volume
    await action.volume(10.0)

client.run();


```
