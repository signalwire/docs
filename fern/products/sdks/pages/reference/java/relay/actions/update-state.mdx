Update the action's internal state based on an incoming relay event.
This is called internally when events are received from the server.

## Parameters

<ParamField path="newState" type="String" required={true} toc={true}>
  The new state string.
</ParamField>

<ParamField path="event" type="RelayEvent" required={true} toc={true}>
  The relay event that triggered the state change.
</ParamField>

## Returns

`void`
