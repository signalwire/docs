Stop the currently running action. The action terminates and fires a
completion event.

## Returns

`void`

## Example

```java
import com.signalwire.sdk.relay.Call;

call.onAnswer(c -> {
    var action = c.playTTS("This is a long message about plant care...");
    // Stop playback after 3 seconds
    Thread.sleep(3000);
    action.stop();
});
```
