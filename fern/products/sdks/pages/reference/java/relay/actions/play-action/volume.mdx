Adjust the playback volume.

## **Parameters**

<ParamField path="volume" type="double" required={true} toc={true}>
  Volume adjustment in dB. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`Map<String, Object>` -- Server acknowledgment.

## **Example**

```java {19,22}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()
    var action = await call.play( [{"type": "audio", "url": "https://example.com/hold-music.mp3"}], 0, );

    // Increase volume
    await action.volume(10.0)

    // Decrease volume
    await action.volume(-5.0)

client.run();


```
