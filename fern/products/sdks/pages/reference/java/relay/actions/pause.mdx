Pause the currently running action (e.g., pause audio playback or recording).

## Returns

`void`

## Example

```java
call.onAnswer(c -> {
    var action = c.playTTS("A detailed guide to growing ferns...");
    Thread.sleep(2000);
    action.pause();
    // Resume later
    action.resume();
});
```
