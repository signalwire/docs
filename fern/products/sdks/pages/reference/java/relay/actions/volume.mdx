Adjust the playback volume of an active play or TTS action.

## Parameters

<ParamField path="volumeDb" type="double" required={true} toc={true}>
  Volume adjustment in decibels. Positive values increase volume, negative
  values decrease it.
</ParamField>

## Returns

`void`

## Example

```java
call.onAnswer(c -> {
    var action = c.playTTS("Welcome to the plant nursery hotline.");
    action.volume(-3.0); // Reduce volume by 3dB
});
```
