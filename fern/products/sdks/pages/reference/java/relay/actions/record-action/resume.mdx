Resume a paused recording.

## **Returns**

`Map<String, Object>` -- Server acknowledgment.

## **Example**

```java {17}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()
    var action = await call.record({"direction": "both", "format": "wav"});

    await action.pause()
    // ... sensitive exchange ...
    await action.resume()

    var event = await action.wait();
    System.out.println("Recording URL: " + event.params.get('record', { + ").get('url')}");

client.run();


```
