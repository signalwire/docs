Pause the currently running action with a specified behavior mode that
controls what happens during the pause.

## Parameters

<ParamField path="behavior" type="String" required={true} toc={true}>
  The pause behavior (e.g., `"silence"`, `"hold_music"`).
</ParamField>

## Returns

`void`

## Example

```java
call.onAnswer(c -> {
    var action = c.playTTS("Please hold while we look up your plant order...");
    action.pauseWithBehavior("silence");
});
```
