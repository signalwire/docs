Join a video/audio room. Rooms provide multi-party communication with
additional features beyond conferences, including video support.

<Info>
Use [`leaveRoom()`][leave-room] to leave the room.
</Info>

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  Room name to join.
</ParamField>

<ParamField path="statusUrl" type="@Nullable String" toc={true}>
  URL to receive room status webhooks.
</ParamField>

## **Returns**

`Map<String, Object>` -- Server response confirming the join.

## **Example**

```java {15}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    // Join a room
    await call.join_room("meeting-room-1")

    // The call is now in the room until leave_room() or hangup

client.run();


```
