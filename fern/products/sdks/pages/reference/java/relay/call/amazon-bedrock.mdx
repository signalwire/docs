Connect the call to an Amazon Bedrock AI agent. Similar to
[`ai()`][ai] but uses Amazon Bedrock as
the LLM backend.

## **Parameters**

<ParamField path="prompt" type="@Nullable Object" toc={true}>
  The prompt configuration for the Bedrock agent.
</ParamField>

<ParamField path="SWAIG" type="@Nullable Map<String, Object>" toc={true}>
  SWAIG configuration for tool/function definitions.
</ParamField>

<ParamField path="aiParams" type="@Nullable Map<String, Object>" toc={true}>
  AI parameters for the Bedrock session.
</ParamField>

<ParamField path="globalData" type="@Nullable Map<String, Object>" toc={true}>
  Data accessible to the AI and SWAIG functions.
</ParamField>

<ParamField path="postPrompt" type="@Nullable Map<String, Object>" toc={true}>
  Post-prompt configuration.
</ParamField>

<ParamField path="postPromptUrl" type="@Nullable String" toc={true}>
  URL to receive the post-prompt result.
</ParamField>

## **Returns**

`Map<String, Object>` -- Server response confirming the Bedrock session.

## **Example**

```java {15}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    // Start an Amazon Bedrock AI agent
    var result = await call.amazon_bedrock( {"text": "You are a helpful assistant."}, {"barge_confidence": 0.02}, );

client.run();


```
