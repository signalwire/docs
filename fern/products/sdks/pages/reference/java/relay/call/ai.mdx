Start an AI agent session on the call. The AI agent handles the conversation
using the provided prompt, tools, and configuration. Returns an
[`AIAction`][aiaction] that you can use to stop
the AI session or wait for it to complete.

<Tip>
For building AI agents with the full framework (prompts, tools, skills, contexts),
use [`AgentBase`][agentbase]. The `ai()` method
is for lower-level RELAY control where you configure the AI inline.
</Tip>

<Info>
See also [`amazonBedrock()`][amazon-bedrock] for using Amazon Bedrock as the LLM backend.
</Info>

<Info>
This method executes the SWML [`ai`][ai] verb on the call. See the
[SWML AI reference][swml-ai-reference] for the full specification of all supported
parameters and behaviors.
</Info>

## **Parameters**

<ParamField path="controlId" type="@Nullable String" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="agent" type="@Nullable String" toc={true}>
  Fabric agent resource ID. When set, the AI uses a pre-configured agent
  from SignalWire Fabric instead of inline configuration.
</ParamField>

<ParamField path="prompt" type="@Nullable Map<String, Object>" toc={true}>
  The main prompt configuration.
</ParamField>

<Indent>
<ParamField path="prompt.text" type="String" toc={true}>
  The system prompt text that defines the AI agent's behavior.
</ParamField>

<ParamField path="prompt.temperature" type="double" toc={true}>
  LLM temperature for the main prompt.
</ParamField>

<ParamField path="prompt.top_p" type="double" toc={true}>
  LLM top_p sampling parameter.
</ParamField>
</Indent>

<ParamField path="postPrompt" type="@Nullable Map<String, Object>" toc={true}>
  Post-prompt configuration for summarization or analysis after the
  conversation ends.
</ParamField>

<Indent>
<ParamField path="post_prompt.text" type="String" toc={true}>
  The post-prompt text.
</ParamField>
</Indent>

<ParamField path="postPromptUrl" type="@Nullable String" toc={true}>
  URL to receive the post-prompt result via webhook.
</ParamField>

<ParamField path="postPromptAuthUser" type="@Nullable String" toc={true}>
  Username for basic auth on the post-prompt webhook.
</ParamField>

<ParamField path="postPromptAuthPassword" type="@Nullable String" toc={true}>
  Password for basic auth on the post-prompt webhook.
</ParamField>

<ParamField path="globalData" type="@Nullable Map<String, Object>" toc={true}>
  Data accessible to the AI agent and SWAIG functions throughout the session.
</ParamField>

<ParamField path="pronounce" type="@Nullable List<Map<String, Object>>" toc={true}>
  Pronunciation rules for words or phrases the TTS engine should handle
  specially.
</ParamField>

<ParamField path="hints" type="@Nullable List<String>" toc={true}>
  Speech recognition hints to improve accuracy for domain-specific terms.
</ParamField>

<ParamField path="languages" type="@Nullable List<Map<String, Object>>" toc={true}>
  Language configurations for multilingual support.
</ParamField>

<ParamField path="SWAIG" type="@Nullable Map<String, Object>" toc={true}>
  SWAIG (SignalWire AI Gateway) configuration for tool/function definitions.
</ParamField>

<ParamField path="aiParams" type="@Nullable Map<String, Object>" toc={true}>
  Additional AI parameters such as `bargeConfidence`, `endOfSpeechTimeout`,
  `attentionTimeout`, and other LLM tuning settings.
</ParamField>

<ParamField path="onCompleted" type="@Nullable Function[[RelayEvent, Object]]" toc={true}>
  Callback invoked when the AI session ends.
</ParamField>

## **Returns**

[`AIAction`][aiaction] -- An action handle with `stop()` and `wait()` methods.

## **Example**

```java {15}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    // Start an AI agent on the call
    var action = await call.ai( {"text": "You are a helpful customer support agent for Acme Corp."}, ["Acme", "support", "billing"], {"barge_confidence": 0.02}, );

    // Wait for the AI session to end (caller hangs up or AI stops)
    await action.wait()
    System.out.println("AI session ended");

client.run();


```
