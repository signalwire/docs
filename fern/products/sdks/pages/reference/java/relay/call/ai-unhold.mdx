Resume an AI agent session from hold. The AI resumes processing the
conversation.

## **Parameters**

<ParamField path="prompt" type="@Nullable String" toc={true}>
  A prompt for the AI to speak upon resuming (e.g., "Thank you for holding.
  I have your information now.").
</ParamField>

## **Returns**

`Map<String, Object>` -- Server response confirming the AI unhold.

## **Example**

```java {27}
// import asyncio
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    // Start an AI agent
    var action = await call.ai( {"text": "You are a support agent."}, );

    // Put the AI on hold
    await call.ai_hold("One moment please.")

    // Do some processing...
    await asyncio.sleep(5)

    // Resume the AI
    await call.ai_unhold("Thanks for waiting. I found your information.")

client.run();


```
