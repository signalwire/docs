Start noise reduction on the call. Filters background noise from the audio
to improve clarity.

<Info>
Use [`denoiseStop()`][denoise-stop] to disable noise reduction.
</Info>

<Info>
This method emits [`calling.call.denoise`][calling-call-denoise] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`Map<String, Object>` -- Server response confirming noise reduction has started.

## **Example**

```java {14}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    var result = await call.denoise();
    System.out.println("Noise reduction started: " + result);

    var action = await call.play([{"type": "tts", "text": "Noise reduction is now active."}]);
    await action.wait()

client.run();


```
