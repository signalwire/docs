Leave an audio conference.

## **Parameters**

<ParamField path="conferenceId" type="String" required={true} toc={true}>
  The conference ID to leave.
</ParamField>

## **Returns**

`Map<String, Object>` -- Server response confirming the leave.

## **Example**

```java {18}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    // Join a conference
    var result = await call.join_conference("team-standup", "onEnter");

    // Later, leave the conference
    await call.leave_conference(result.get("conference_id"))

client.run();


```
