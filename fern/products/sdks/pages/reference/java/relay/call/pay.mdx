Start a payment collection session on the call. Collects credit card or other
payment information from the caller via DTMF. Returns a
[`PayAction`][payaction] that you can use to stop
the payment flow or wait for it to complete.

<Info>
This method emits [`calling.call.pay`][calling-call-pay] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="paymentConnectorUrl" type="String" required={true} toc={true}>
  URL of the payment connector service that processes the payment.
</ParamField>

<ParamField path="controlId" type="@Nullable String" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="inputMethod" type="@Nullable String" toc={true}>
  How the caller provides payment info.

  - `"dtmf"` -- caller enters digits on the keypad
  - `"speech"` -- caller speaks the payment information
</ParamField>

<ParamField path="statusUrl" type="@Nullable String" toc={true}>
  URL to receive payment status webhooks.
</ParamField>

<ParamField path="paymentMethod" type="@Nullable String" toc={true}>
  Payment method type. Valid values: `"credit-card"`.
</ParamField>

<ParamField path="timeout" type="@Nullable String" toc={true}>
  Timeout for the payment session.
</ParamField>

<ParamField path="maxAttempts" type="@Nullable String" toc={true}>
  Maximum number of input attempts before failing.
</ParamField>

<ParamField path="securityCode" type="@Nullable String" toc={true}>
  Whether to collect CVV.

  - `"true"` -- prompt the caller for the security code
  - `"false"` -- skip security code collection
</ParamField>

<ParamField path="postalCode" type="@Nullable String" toc={true}>
  Whether to collect postal code.

  - `"true"` -- prompt the caller for the postal code
  - `"false"` -- skip postal code collection
</ParamField>

<ParamField path="minPostalCodeLength" type="@Nullable String" toc={true}>
  Minimum length for the postal code.
</ParamField>

<ParamField path="tokenType" type="@Nullable String" toc={true}>
  Tokenization type for the payment data.
</ParamField>

<ParamField path="chargeAmount" type="@Nullable String" toc={true}>
  Amount to charge (e.g., `"29.99"`).
</ParamField>

<ParamField path="currency" type="@Nullable String" toc={true}>
  Currency code (e.g., `"USD"`, `"EUR"`).
</ParamField>

<ParamField path="language" type="@Nullable String" toc={true}>
  Language for payment prompts (e.g., `"en"`).
</ParamField>

<ParamField path="voice" type="@Nullable String" toc={true}>
  Voice for TTS prompts during the payment flow.
</ParamField>

<ParamField path="description" type="@Nullable String" toc={true}>
  Description of the payment/charge.
</ParamField>

<ParamField path="validCardTypes" type="@Nullable String" toc={true}>
  Comma-separated list of accepted card types (e.g., `"visa,mastercard,amex"`).
</ParamField>

<ParamField path="parameters" type="@Nullable List<Map<String, Object>>" toc={true}>
  Additional parameters to pass to the payment connector.
</ParamField>

<ParamField path="prompts" type="@Nullable List<Map<String, Object>>" toc={true}>
  Custom prompts for the payment flow.
</ParamField>

<ParamField path="onCompleted" type="@Nullable Function[[RelayEvent, Object]]" toc={true}>
  Callback invoked when the payment operation completes.
</ParamField>

## **Returns**

[`PayAction`][payaction] -- An action handle with
`stop()` and `wait()` methods.

## **Example**

```java {15}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()
    await call.play([{"type": "tts", "text": "We will now collect your payment."}])

    var action = await call.pay( "https://example.com/payment-connector", "29.99", "USD", "true", "true", );
    var event = await action.wait();

    var state = event.params.get("state", "");
    if state == "finished":
        await call.play([{"type": "tts", "text": "Payment processed successfully."}])
    else:
        await call.play([{"type": "tts", "text": "Payment failed. Please try again later."}])

    await call.hangup()

client.run();


```
