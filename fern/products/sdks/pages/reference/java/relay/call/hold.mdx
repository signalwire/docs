Put the call on hold. The remote party hears hold music or silence while the
call is held.

<Info>
Use [`unhold()`][unhold] to release the call from hold.
</Info>

<Info>
This method emits [`calling.call.hold`][calling-call-hold] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`Map<String, Object>` -- Server response confirming the hold operation.

## **Example**

```java {15}
// import asyncio
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()
    await call.play([{"type": "tts", "text": "Please hold while I look that up."}])

    var result = await call.hold();
    System.out.println("Hold started: " + result);

    // Do some processing...
    await asyncio.sleep(5)

    // Take the call off hold
    await call.unhold()

client.run();


```
