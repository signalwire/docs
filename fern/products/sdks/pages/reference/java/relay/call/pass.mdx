Decline control of an inbound call, returning it to the SignalWire routing
engine. The call is not ended -- it continues to ring and may be delivered to
another RELAY client or routing rule.

<Note>
The method is named `pass_()` because `pass` is a reserved keyword in some
reserved keyword in Python.
</Note>

## **Parameters**

None.

## **Returns**

`Map<String, Object>` -- Server response confirming the pass operation.

## **Example**

```java {14}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    // Only handle calls from a specific context
    if call.context != "sales":
        await call.pass_()
        System.out.println("Call passed back to routing");
        return

    await call.answer()

client.run();


```
