Start recording audio from the call. Returns a
[`RecordAction`][recordaction] that you can use to
pause, resume, stop, or wait for the recording to complete.

<Info>
This method emits [`calling.call.record`][calling-call-record] events. See [Call Events][call-events] for payload details.
</Info>

<Info>
This method corresponds to the SWML [`record`][record] verb. See the
[SWML record reference][swml-record-reference] for the full specification.
</Info>

## **Parameters**

<ParamField path="audio" type="@Nullable Map<String, Object>" toc={true}>
  Audio recording configuration object.
</ParamField>

<Indent>
<ParamField path="audio.direction" type="String" toc={true}>
  Which audio to record.

  - `"listen"` -- record audio heard by the caller
  - `"speak"` -- record audio spoken by the caller
  - `"both"` -- record audio in both directions
</ParamField>

<ParamField path="audio.format" type="String" toc={true}>
  Recording file format.

  - `"mp3"` -- compressed MP3 format
  - `"wav"` -- uncompressed WAV format
</ParamField>

<ParamField path="audio.stereo" type="boolean" toc={true}>
  Record in stereo (each side on a separate channel).
</ParamField>

<ParamField path="audio.initial_timeout" type="double" toc={true}>
  Seconds to wait for audio before ending with `noInput`.
</ParamField>

<ParamField path="audio.end_silence_timeout" type="double" toc={true}>
  Seconds of silence before stopping automatically.
</ParamField>

<ParamField path="audio.terminators" type="String" toc={true}>
  DTMF digits that stop recording (e.g., `"#"`).
</ParamField>

<ParamField path="audio.beep" type="boolean" toc={true}>
  Play a beep before recording starts.
</ParamField>

<ParamField path="audio.input_sensitivity" type="double" toc={true}>
  Sensitivity threshold for detecting audio input.
</ParamField>
</Indent>

<ParamField path="controlId" type="@Nullable String" toc={true}>
  Custom control ID for this operation. Auto-generated if not provided.
</ParamField>

<ParamField path="onCompleted" type="@Nullable Function[[RelayEvent, Object]]" toc={true}>
  Callback invoked when recording reaches a terminal state (`finished` or
  `noInput`). The event contains the recording `url`, `duration`, and `size`.
</ParamField>

## **Returns**

[`RecordAction`][recordaction] -- An action handle with
`stop()`, `pause()`, `resume()`, and `wait()` methods.

## **Example**

```java {15}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()
    await call.play([{"type": "tts", "text": "Leave a message after the beep."}])

    var action = await call.record( {"beep": true, "end_silence_timeout": 3, "terminators": "#"}, );
    var event = await action.wait();

    var url = event.params.get("record", {}).get("url", "");
    System.out.println("Recording saved: " + url);
    await call.hangup()

client.run();


```
