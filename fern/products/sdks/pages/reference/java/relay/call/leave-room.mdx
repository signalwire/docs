Leave the current room.

## **Parameters**

None.

## **Returns**

`Map<String, Object>` -- Server response confirming the leave.

## **Example**

```java {18}
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .contexts(List.of("default"))
        .build();

async def handle_call(call):
    await call.answer()

    // Join a room
    await call.join_room("meeting-room-1")

    // Later, leave the room
    await call.leave_room()
    await call.hangup()

client.run();


```
