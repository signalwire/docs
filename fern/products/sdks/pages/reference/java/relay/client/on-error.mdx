Register a handler that is invoked when a WebSocket connection error occurs.

## Parameters

<ParamField path="handler" type="Consumer<Exception>" required={true} toc={true}>
  A callback that receives the `Exception` describing the error.
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .build();

client.onError(ex -> {
    System.err.println("Connection error: " + ex.getMessage());
});
client.connect();
```
