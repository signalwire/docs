Register a handler that is invoked when the WebSocket connection to SignalWire
is successfully established.

## Parameters

<ParamField path="handler" type="Runnable" required={true} toc={true}>
  A callback invoked when the connection opens.
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .build();

client.onOpen(() -> {
    System.out.println("Connected to SignalWire");
});
client.connect();
```
