Register a handler that is invoked for every relay event received by the client.

## Parameters

<ParamField path="handler" type="Consumer<RelayEvent>" required={true} toc={true}>
  A callback that receives the [`RelayEvent`][ref-relayevent].
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .build();

client.onEvent(event -> {
    System.out.println("Event: " + event.getName());
});
client.connect();
```
