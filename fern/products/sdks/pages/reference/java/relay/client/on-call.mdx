Register a handler that is invoked when an incoming call is received.

## Parameters

<ParamField path="handler" type="Consumer<Call>" required={true} toc={true}>
  A callback that receives the incoming [`Call`][ref-call] object.
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .build();

client.onCall(call -> {
    System.out.println("Incoming call from: " + call.getFrom());
    call.answer();
    call.playTTS("Welcome to the plant nursery.");
});
client.connect();
```
