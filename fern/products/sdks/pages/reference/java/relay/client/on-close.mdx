Register a handler that is invoked when the WebSocket connection to SignalWire
is closed.

## Parameters

<ParamField path="handler" type="Runnable" required={true} toc={true}>
  A callback invoked when the connection closes.
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.relay.RelayClient;

var client = RelayClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .build();

client.onClose(() -> {
    System.out.println("Disconnected from SignalWire");
});
client.connect();
```
