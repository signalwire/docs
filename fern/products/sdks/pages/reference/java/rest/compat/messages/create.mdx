Send a new SMS or MMS message.

## **Parameters**

<ParamField path="To" type="String" required={true} toc={true}>
  The destination phone number in E.164 format.
</ParamField>

<ParamField path="From" type="String" required={true} toc={true}>
  The sender phone number (must be a number you own).
</ParamField>

<ParamField path="Body" type="String" toc={true}>
  The text body of the message. Required for SMS; optional for MMS if `MediaUrl` is provided.
</ParamField>

<ParamField path="MediaUrl" type="List<String>" toc={true}>
  URLs of media to include (for MMS). Up to 10 media URLs.
</ParamField>

<ParamField path="StatusCallback" type="String" toc={true}>
  URL to receive message status webhook events.
</ParamField>

## **Returns**

`Map<String, Object>` -- The created message resource including the message SID.

## **Example**

```java {10,17}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Send an SMS
var msg = client.compat.messages.create( "+15559876543", "+15551234567", "Hello from SignalWire!" );

// Send an MMS with an image
var mms = client.compat.messages.create( "+15559876543", "+15551234567", "Check this out", ["https://example.com/image.jpg"] );


```
