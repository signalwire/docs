List messages in the account.

## **Parameters**

<ParamField path="To" type="String" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="From" type="String" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing message resources.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var messages = client.compat.messages.list(20);


```
