Create a new LAML bin containing a cXML/LaML script.

## **Parameters**

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="String" required={true} toc={true}>
  The cXML/LaML script content.
</ParamField>

## **Returns**

`Map<String, Object>` -- The created LAML bin resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var laml_bin = client.compat.laml_bins.create( "Greeting", ( '<?xml "1.0" "UTF-8"?>' "<Response><Say>Hello!</Say></Response>" ) );


```
