Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="tokenId" type="String" required={true} toc={true}>
  The token identifier.
</ParamField>

<ParamField path="Ttl" type="int" toc={true}>
  New time-to-live in seconds for the token.
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated token resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.tokens.update("token-id", 7200)


```
