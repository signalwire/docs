Retrieve a specific member from a queue by call SID.

## **Parameters**

<ParamField path="queueSid" type="String" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="callSid" type="String" required={true} toc={true}>
  The call SID of the queue member.
</ParamField>

## **Returns**

`Map<String, Object>` -- The queue member resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var member = client.compat.queues.get_member("QU...", "CA...");


```
