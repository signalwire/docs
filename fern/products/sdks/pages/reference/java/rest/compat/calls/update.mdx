Update an active call (e.g., redirect to new LAML, end the call).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The call SID.
</ParamField>

<ParamField path="Url" type="String" toc={true}>
  New URL to fetch LAML instructions from.
</ParamField>

<ParamField path="Status" type="String" toc={true}>
  Set to `"completed"` to end the call.
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated call resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.calls.update("CA...", "completed")


```
