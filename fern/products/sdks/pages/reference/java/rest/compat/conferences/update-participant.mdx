Update a participant in a conference (e.g., mute, hold, or remove).

## **Parameters**

<ParamField path="conferenceSid" type="String" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="callSid" type="String" required={true} toc={true}>
  The call SID of the participant.
</ParamField>

<ParamField path="Muted" type="boolean" toc={true}>
  Set to `true` to mute or `false` to unmute.
</ParamField>

<ParamField path="Hold" type="boolean" toc={true}>
  Set to `true` to hold or `false` to unhold.
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated participant resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.conferences.update_participant("CF...", "CA...", true)


```
