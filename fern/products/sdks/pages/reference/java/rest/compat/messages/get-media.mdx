Retrieve a specific media item from a message.

## **Parameters**

<ParamField path="messageSid" type="String" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="mediaSid" type="String" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`Map<String, Object>` -- The media resource including the content URL.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var item = client.compat.messages.get_media("SM...", "ME...");


```
