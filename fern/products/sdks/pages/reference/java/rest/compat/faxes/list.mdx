List faxes in the account.

## **Returns**

`Map<String, Object>` -- Paginated response containing fax resources.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var faxes = client.compat.faxes.list();


```
