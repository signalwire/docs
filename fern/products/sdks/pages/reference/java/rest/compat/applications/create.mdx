Create a new application.

## **Parameters**

<ParamField path="FriendlyName" type="String" required={true} toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="String" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="VoiceMethod" type="String" toc={true}>
  HTTP method for the voice URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="SmsUrl" type="String" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="SmsMethod" type="String" toc={true}>
  HTTP method for the SMS URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="StatusCallback" type="String" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`Map<String, Object>` -- The created application resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var app = client.compat.applications.create( "My App", "https://example.com/voice", "https://example.com/sms" );


```
