Search for available toll-free phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="String" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="Contains" type="String" toc={true}>
  Filter by number pattern (e.g., `"800"`).
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing available toll-free number resources.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var numbers = client.compat.phone_numbers.search_toll_free("US", "800");


```
