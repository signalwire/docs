Retrieve a single account by its SID.

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The account SID.
</ParamField>

## **Returns**

`Map<String, Object>` -- The account resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var account = client.compat.accounts.get("account-sid");


```
