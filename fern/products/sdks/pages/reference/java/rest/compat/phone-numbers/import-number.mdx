Import an external phone number into the account.

## **Parameters**

<ParamField path="PhoneNumber" type="String" required={true} toc={true}>
  The phone number to import in E.164 format.
</ParamField>

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable label for the number.
</ParamField>

## **Returns**

`Map<String, Object>` -- The imported phone number resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var number = client.compat.phone_numbers.import_number( "+15559876543" );


```
