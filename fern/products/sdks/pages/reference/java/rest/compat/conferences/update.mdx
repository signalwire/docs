Update a conference (e.g., end it, or set the announce URL).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The conference SID.
</ParamField>

<ParamField path="Status" type="String" toc={true}>
  Set to `"completed"` to end the conference.
</ParamField>

<ParamField path="AnnounceUrl" type="String" toc={true}>
  URL returning LAML to play an announcement to all participants.
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated conference resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.conferences.update("CF...", "completed")


```
