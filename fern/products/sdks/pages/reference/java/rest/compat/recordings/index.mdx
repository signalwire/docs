Manage call recordings with list, get, and delete operations.

Access via `client.compat.recordings` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var recordings = client.compat.recordings.list();


```
