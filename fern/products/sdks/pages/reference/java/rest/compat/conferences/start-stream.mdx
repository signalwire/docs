Start a media stream on a conference, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="conferenceSid" type="String" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="Url" type="String" toc={true}>
  The WebSocket URL to stream audio to.
</ParamField>

## **Returns**

`Map<String, Object>` -- The created stream resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var stream = client.compat.conferences.start_stream( "CF...", "wss://stream.example.com" );


```
