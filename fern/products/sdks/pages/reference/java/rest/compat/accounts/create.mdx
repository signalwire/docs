Create a new subproject (sub-account).

## **Parameters**

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable name for the subproject.
</ParamField>

## **Returns**

`Map<String, Object>` -- The created account resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var sub = client.compat.accounts.create("Marketing Team");


```
