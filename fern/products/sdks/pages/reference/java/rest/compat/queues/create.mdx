Create a new call queue.

## **Parameters**

<ParamField path="FriendlyName" type="String" required={true} toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="int" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`Map<String, Object>` -- The created queue resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var queue = client.compat.queues.create( "Support Queue", 100 );


```
