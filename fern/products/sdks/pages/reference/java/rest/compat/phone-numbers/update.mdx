Update a phone number's configuration (e.g., change voice/SMS URLs).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The phone number SID.
</ParamField>

<ParamField path="FriendlyName" type="String" toc={true}>
  A human-readable label for the number.
</ParamField>

<ParamField path="VoiceUrl" type="String" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="SmsUrl" type="String" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="StatusCallback" type="String" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated phone number resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.phone_numbers.update( "PN...", "https://example.com/voice" )


```
