Dequeue a member from a queue, redirecting the call to a new LAML URL.

## **Parameters**

<ParamField path="queueSid" type="String" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="callSid" type="String" required={true} toc={true}>
  The call SID of the queue member to dequeue.
</ParamField>

<ParamField path="Url" type="String" toc={true}>
  The URL that returns LAML instructions for the dequeued call.
</ParamField>

<ParamField path="Method" type="String" toc={true}>
  HTTP method for the URL. Defaults to `"POST"`.
</ParamField>

## **Returns**

`Map<String, Object>` -- The dequeued member resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.queues.dequeue_member( "QU...", "CA...", "https://example.com/dequeue" )


```
