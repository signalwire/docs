Retrieve a specific participant in a conference.

## **Parameters**

<ParamField path="conferenceSid" type="String" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="callSid" type="String" required={true} toc={true}>
  The call SID of the participant.
</ParamField>

## **Returns**

`Map<String, Object>` -- The participant resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var participant = client.compat.conferences.get_participant("CF...", "CA...");


```
