Manage call queues with CRUD operations and member sub-resources for dequeuing
callers.

Access via `client.compat.queues` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var queues = client.compat.queues.list();


```
