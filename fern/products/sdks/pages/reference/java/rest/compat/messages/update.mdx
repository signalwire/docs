Update a message (e.g., redact the body of a sent message).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The message SID.
</ParamField>

<ParamField path="Body" type="String" toc={true}>
  Set to an empty string to redact the message body.
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated message resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.messages.update("SM...", "")


```
