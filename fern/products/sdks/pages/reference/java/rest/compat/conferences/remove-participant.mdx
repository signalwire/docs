Remove a participant from a conference, ending their call leg.

## **Parameters**

<ParamField path="conferenceSid" type="String" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="callSid" type="String" required={true} toc={true}>
  The call SID of the participant to remove.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.conferences.remove_participant("CF...", "CA...")


```
