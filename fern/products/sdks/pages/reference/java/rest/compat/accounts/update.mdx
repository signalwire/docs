Update an account. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="String" required={true} toc={true}>
  The account SID to update.
</ParamField>

<ParamField path="FriendlyName" type="String" toc={true}>
  Updated friendly name.
</ParamField>

<ParamField path="Status" type="String" toc={true}>
  Account status.

  - `"active"` -- account is fully operational
  - `"suspended"` -- account is temporarily disabled
  - `"closed"` -- account is permanently deactivated
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated account resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.accounts.update("account-sid", "Sales Team")


```
