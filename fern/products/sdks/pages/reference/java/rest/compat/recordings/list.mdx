List recordings in the account.

## **Parameters**

<ParamField path="DateCreated" type="String" toc={true}>
  Filter by creation date.
</ParamField>

<ParamField path="CallSid" type="String" toc={true}>
  Filter by the call SID that generated the recording.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing recording resources.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var recordings = client.compat.recordings.list(20);


```
