Delete a media item from a message.

## **Parameters**

<ParamField path="messageSid" type="String" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="mediaSid" type="String" required={true} toc={true}>
  The SID of the media item to delete.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.compat.messages.delete_media("SM...", "ME...")
```
