List members currently in a queue.

## **Parameters**

<ParamField path="queueId" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Map<String, Object>` -- JSON response containing a `data` list of member objects.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var members = client.queues.list_members("queue-id");
for m in members.get("data", []):
    System.out.println(m.get("call_id"), m.get("position"));


```
