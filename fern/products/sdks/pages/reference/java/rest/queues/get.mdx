Retrieve a specific queue.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- The queue object.

## **Example**

```java {9-10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var queue = client.queues.get("queue-id");
System.out.println(queue["name"], queue.get("current_size"));


```
