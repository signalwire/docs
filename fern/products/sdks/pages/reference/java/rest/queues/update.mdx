Update a queue. Uses PUT.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Fields to update (e.g., `name`, `maxSize`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated queue object.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.queues.update("queue-id", "priority-support");
System.out.println("Updated: " + updated['id']);


```
