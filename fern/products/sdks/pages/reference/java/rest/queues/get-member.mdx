Retrieve a specific member from a queue.

## **Parameters**

<ParamField path="queueId" type="String" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="memberId" type="String" required={true} toc={true}>
  The member resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- The member object.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var member = client.queues.get_member("queue-id", "member-id");
System.out.println(member.get("call_id"), member.get("position"));


```
