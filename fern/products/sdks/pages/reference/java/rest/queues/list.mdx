List queues in the project.

## **Parameters**

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Map<String, Object>` -- JSON response containing a `data` list of queue objects.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.queues.list();
for q in result.get("data", []):
    System.out.println(q["name"], q.get("current_size"));


```
