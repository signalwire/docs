Create a new queue.

## **Parameters**

<ParamField path="name" type="String" toc={true}>
  Queue name.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Additional queue configuration fields (e.g., `maxSize`, `fifo`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The newly created queue object.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var queue = client.queues.create("support", 50);
System.out.println("Queue ID:", queue["id"]);


```
