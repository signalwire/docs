Generate tokens for the SignalWire PubSub (publish-subscribe) real-time messaging
service. PubSub tokens authenticate client-side connections to subscribe to
channels and publish messages.

Access via `client.pubsub` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .build();


```

---

### create_token

Create a PubSub authentication token.

## **Parameters**

<ParamField path="channels" type="List<String>" toc={true}>
  List of channel names the token grants access to.
</ParamField>

<ParamField path="memberId" type="String" toc={true}>
  Identifier for the connecting member.
</ParamField>

<ParamField path="ttl" type="int" toc={true}>
  Token time-to-live in seconds.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Additional token parameters (e.g., `state`, `permissions`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The token object containing the JWT token string.

## **Example**

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var token = client.pubsub.create_token( ["notifications", "updates"], "user-123", 3600, );
System.out.println("PubSub token:", token.get("token"));


```
