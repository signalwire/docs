Update a stream. Uses PUT for full replacement, so all fields must be provided
in the request body.

<Note>
This method performs a full replacement (PUT), not a partial update (PATCH).
Include all fields you want to keep in the request body.
</Note>

## **Parameters**

<ParamField path="streamId" type="String" required={true} toc={true}>
  The unique identifier of the stream.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Map<String, Object>` -- The updated stream.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var stream = client.video.streams.update( "stream-id", "rtmp://new-destination.example.com/key" );


```
