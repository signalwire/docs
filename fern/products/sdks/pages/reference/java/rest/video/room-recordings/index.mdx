Manage video room recordings. Supports listing, retrieval, deletion, and
querying recording-level events.

Access via `client.video.room_recordings` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var recordings = client.video.room_recordings.list();


```
