List active streams for a conference.

## **Parameters**

<ParamField path="conferenceId" type="String" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Map<String, Object>` -- List of streams in the conference.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var streams = client.video.conferences.list_streams("conference-id");


```
