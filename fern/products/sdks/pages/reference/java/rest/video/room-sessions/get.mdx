Retrieve a single room session by its unique identifier.

## **Parameters**

<ParamField path="sessionId" type="String" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

## **Returns**

`Map<String, Object>` -- The room session resource.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var session = client.video.room_sessions.get("session-id");


```
