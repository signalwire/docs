Create a new video room.

## **Parameters**

<ParamField path="name" type="String" required={true} toc={true}>
  The name of the video room.
</ParamField>

<ParamField path="maxParticipants" type="int" toc={true}>
  Maximum number of participants allowed in the room.
</ParamField>

Additional keyword arguments are passed to the API as the request body.

## **Returns**

`Map<String, Object>` -- The created video room.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var room = client.video.rooms.create("standup", 10);


```
