List events associated with a recording (e.g., recording started, paused,
completed).

## **Parameters**

<ParamField path="recordingId" type="String" required={true} toc={true}>
  The unique identifier of the recording.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Map<String, Object>` -- Paginated list of recording events.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var events = client.video.room_recordings.list_events("recording-id");


```
