Reset a conference token, invalidating the current token and generating a new one.

## **Parameters**

<ParamField path="tokenId" type="String" required={true} toc={true}>
  The unique identifier of the conference token to reset.
</ParamField>

## **Returns**

`Map<String, Object>` -- The reset result, including the new token.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.video.conference_tokens.reset("token-id");


```
