Delete a room recording.

## **Parameters**

<ParamField path="recordingId" type="String" required={true} toc={true}>
  The unique identifier of the recording to delete.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.video.room_recordings.delete("recording-id")


```
