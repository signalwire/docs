Replace a video conference. Uses PUT for full replacement, so all fields must be
provided in the request body.

<Note>
This method performs a full replacement (PUT), not a partial update (PATCH).
Include all fields you want to keep in the request body.
</Note>

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Map<String, Object>` -- The updated conference.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var conf = client.video.conferences.update("conference-id", "team-meeting-v2");


```
