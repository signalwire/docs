List active streams for a video room.

## **Parameters**

<ParamField path="roomId" type="String" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Map<String, Object>` -- List of streams in the room.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var streams = client.video.rooms.list_streams("room-id");


```
