List recordings created during a room session.

## **Parameters**

<ParamField path="sessionId" type="String" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Map<String, Object>` -- Paginated list of session recordings.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var recordings = client.video.room_sessions.list_recordings("session-id");


```
