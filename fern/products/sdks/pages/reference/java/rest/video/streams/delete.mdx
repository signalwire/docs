Delete a stream.

## **Parameters**

<ParamField path="streamId" type="String" required={true} toc={true}>
  The unique identifier of the stream to delete.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.video.streams.delete("stream-id")


```
