List events that occurred during a room session (e.g., participant joins,
leaves, audio/video mute changes).

## **Parameters**

<ParamField path="sessionId" type="String" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Map<String, Object>` -- Paginated list of session events.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var events = client.video.room_sessions.list_events("session-id");


```
