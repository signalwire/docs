Create a new stream for a conference, enabling external streaming output
(e.g., RTMP to a streaming platform).

## **Parameters**

<ParamField path="conferenceId" type="String" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

<ParamField path="url" type="String" toc={true}>
  The streaming destination URL (e.g., an RTMP endpoint).
</ParamField>

Additional keyword arguments define the stream configuration.

## **Returns**

`Map<String, Object>` -- The created stream resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var stream = client.video.conferences.create_stream( "conference-id", "rtmp://live.example.com/key" );


```
