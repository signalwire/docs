Retrieve a specific conference token.

## **Parameters**

<ParamField path="tokenId" type="String" required={true} toc={true}>
  The unique identifier of the conference token.
</ParamField>

## **Returns**

`Map<String, Object>` -- The conference token resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var token = client.video.conference_tokens.get("token-id");


```
