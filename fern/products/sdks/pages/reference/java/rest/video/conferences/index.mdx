Manage video conferences with full CRUD operations, token listing, and stream
management. Uses PUT for updates (full replacement).

Access via `client.video.conferences` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var conferences = client.video.conferences.list();


```
