Update configuration for a phone number. Uses PUT.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Fields to update (e.g., `name`, `callHandler`, `callRequestUrl`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated phone number object.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.phone_numbers.update("phone-number-id", "Support Line")


```
