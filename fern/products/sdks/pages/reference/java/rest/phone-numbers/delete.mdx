Release (delete) a phone number from the project.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.phone_numbers.delete("phone-number-id")


```
