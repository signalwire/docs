List phone numbers owned by the project.

## **Parameters**

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `pageSize`,
  `pageToken`).
</ParamField>

## **Returns**

`Map<String, Object>` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.phone_numbers.list(10);
for number in result.get("data", []):
    System.out.println(number["number"], number.get("name"));


```
