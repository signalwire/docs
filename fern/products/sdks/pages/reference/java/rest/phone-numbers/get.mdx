Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- The phone number object.

## **Example**

```java {9-10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var number = client.phone_numbers.get("phone-number-id");
System.out.println(number["number"], number.get("name"));


```
