Search for available phone numbers that can be purchased.

## **Parameters**

<ParamField path="areaCode" type="String" toc={true}>
  Filter by area code (e.g., `"512"`).
</ParamField>

<ParamField path="quantity" type="int" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Additional query parameters such as `region`, `city`, `country`, or
  `numberType`.
</ParamField>

## **Returns**

`Map<String, Object>` -- JSON response containing a `data` list of available number objects.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var available = client.phone_numbers.search("503", 5);
for number in available.get("data", []):
    System.out.println(number["number"], number.get("region"));


```
