### tokens.create

Create a new API token for the project.

## **Parameters**

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Token configuration fields (e.g., `name`, `scopes`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The newly created token object, including the token value.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var token = client.project.tokens.create("ci-token");
System.out.println("Token:", token.get("token"));
System.out.println("ID:", token.get("id"));


```
