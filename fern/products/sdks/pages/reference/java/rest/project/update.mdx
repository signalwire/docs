### tokens.update

Update an existing API token. Uses PATCH.

## **Parameters**

<ParamField path="tokenId" type="String" required={true} toc={true}>
  The token resource ID.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated token object.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.project.tokens.update("token-id", "renamed-token")


```
