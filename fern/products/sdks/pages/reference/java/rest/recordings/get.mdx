Retrieve a specific recording.

## **Parameters**

<ParamField path="recordingId" type="String" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- The recording object, including download URL and metadata.

## **Example**

```java {9-10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var rec = client.recordings.get("recording-id");
System.out.println(rec.get("duration"), rec.get("url"));


```
