Generate tokens for the SignalWire Chat service. Chat tokens authenticate
client-side connections for real-time chat messaging.

Access via `client.chat` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .build();


```

---

### create_token

Create a Chat authentication token.

## **Parameters**

<ParamField path="channels" type="List<String>" toc={true}>
  List of channel names the token grants access to.
</ParamField>

<ParamField path="memberId" type="String" toc={true}>
  Identifier for the connecting member.
</ParamField>

<ParamField path="ttl" type="int" toc={true}>
  Token time-to-live in seconds.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Additional token parameters (e.g., `state`, `permissions`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The token object containing the JWT token string.

## **Example**

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var token = client.chat.create_token( ["support-room"], "user-456", 3600, );
System.out.println("Chat token:", token.get("token"));


```
