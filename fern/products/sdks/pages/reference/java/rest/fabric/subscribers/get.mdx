Retrieve a single subscriber resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`Map<String, Object>` -- The subscriber resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var subscriber = client.fabric.subscribers.get("subscriber-id");
System.out.println("Subscriber: " + subscriber['name'] + ", ID: " + subscriber['id']);


```
