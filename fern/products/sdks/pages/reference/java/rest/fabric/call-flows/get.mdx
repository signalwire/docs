Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`Map<String, Object>` -- The call flow resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var flow = client.fabric.call_flows.get("flow-id");
System.out.println("Call Flow: " + flow['name'] + ", ID: " + flow['id']);


```
