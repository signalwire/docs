Retrieve a single SIP endpoint resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`Map<String, Object>` -- The SIP endpoint resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var endpoint = client.fabric.sip_endpoints.get("endpoint-id");
System.out.println("Sip Endpoint: " + endpoint['name'] + ", ID: " + endpoint['id']);


```
