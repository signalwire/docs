Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`Map<String, Object>` -- The RELAY application resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var item = client.fabric.relay_applications.get("resource-id");
System.out.println("Relay Application: " + item['name'] + ", ID: " + item['id']);


```
