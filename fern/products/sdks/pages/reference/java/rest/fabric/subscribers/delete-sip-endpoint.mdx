Delete a SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriberId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpointId" type="String" required={true} toc={true}>
  The unique identifier of the SIP endpoint to delete.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success (HTTP 204).

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.fabric.subscribers.delete_sip_endpoint("subscriber-id", "endpoint-id")
System.out.println("SIP endpoint deleted");


```
