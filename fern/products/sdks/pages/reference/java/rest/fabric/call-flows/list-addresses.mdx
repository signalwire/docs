List addresses associated with a call flow resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`Map<String, Object>` -- List of addresses assigned to this call flow.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var addresses = client.fabric.call_flows.list_addresses("flow-id");
for addr in addresses.get("data", []):
    System.out.println(addr['display_name'] + ": " + addr['id']);


```
