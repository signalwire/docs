Read-only access to fabric addresses. Addresses represent the endpoints (phone numbers,
SIP URIs, domains) that route traffic to fabric resources.

Access via `client.fabric.addresses` on a [`RestClient`][restclient] instance.

### list

List all fabric addresses.

## **Parameters**

<ParamField path="pageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="pageToken" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing fabric addresses.

### get

Retrieve a single fabric address.

## **Parameters**

<ParamField path="addressId" type="String" required={true} toc={true}>
  The unique identifier of the address.
</ParamField>

## **Returns**

`Map<String, Object>` -- The fabric address resource.

## **Example**

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// List all addresses
var addresses = client.fabric.addresses.list();
for addr in addresses.get("data", []):
    System.out.println(addr['display_name'] + ": " + addr['type']);

// Get a specific address
var address = client.fabric.addresses.get("address-id");
System.out.println("Address: " + address['display_name']);


```
