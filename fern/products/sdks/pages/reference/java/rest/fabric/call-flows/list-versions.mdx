List all versions of a call flow resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<ParamField path="pageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="pageToken" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing call flow versions.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var response = client.fabric.call_flows.list_versions("call-flow-id");
for version in response.get("data", []):
    System.out.println("Version " + version['version'] + ": " + version['id']);


```
