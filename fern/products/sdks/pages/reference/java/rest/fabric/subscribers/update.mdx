Replace a subscriber resource. Uses PUT, so the full resource body must be provided.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are the fields for the replacement resource body.

## **Returns**

`Map<String, Object>` -- The updated subscriber resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.subscribers.update("subscriber-id", "bob");
System.out.println("Updated: " + updated['name']);


```
