Partially update an AI agent resource. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`Map<String, Object>` -- The updated AI agent resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.ai_agents.update("agent-id", "updated-name");
System.out.println("Updated: " + updated['name']);


```
