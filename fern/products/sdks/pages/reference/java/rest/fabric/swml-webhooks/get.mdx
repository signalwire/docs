Retrieve a single SWML webhook resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the SWML webhook.
</ParamField>

## **Returns**

`Map<String, Object>` -- The SWML webhook resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var webhook = client.fabric.swml_webhooks.get("webhook-id");
System.out.println("Swml Webhook: " + webhook['name'] + ", ID: " + webhook['id']);


```
