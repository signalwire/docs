Replace a cXML application resource. Uses PUT, so the full resource body must be provided.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

Additional keyword arguments are the fields for the replacement resource body.

## **Returns**

`Map<String, Object>` -- The updated cXML application resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.cxml_applications.update( "app-id", "updated-cxml-app", );
System.out.println("Updated: " + updated['name']);


```
