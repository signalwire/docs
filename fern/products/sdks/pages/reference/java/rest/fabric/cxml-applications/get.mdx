Retrieve a single cXML application resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

## **Returns**

`Map<String, Object>` -- The cXML application resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var app = client.fabric.cxml_applications.get("app-id");
System.out.println("cXML Application: " + app['name'] + ", ID: " + app['id']);


```
