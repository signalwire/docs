Create subscriber, guest, invite, and embed tokens for Fabric authentication.

Access via `client.fabric.tokens` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var token = client.fabric.tokens.create_subscriber_token("sub-id");


```
