Replace a FreeSWITCH connector resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Map<String, Object>` -- The updated FreeSWITCH connector resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.freeswitch_connectors.update("connector-id", "updated-name");
System.out.println("Updated: " + updated['name']);


```
