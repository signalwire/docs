Generic resource operations that work across all Fabric resource types. Provides
read-only access (list, get, delete) plus address listing and route assignment.
Does not support create or update methods.

Access via `client.fabric.resources` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var items = client.fabric.resources.list();


```
