Partially update a SIP endpoint for a subscriber. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="subscriberId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpointId" type="String" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`Map<String, Object>` -- The updated SIP endpoint.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.subscribers.update_sip_endpoint( "subscriber-id", "endpoint-id", "new-username", );
System.out.println("Updated: " + updated['username']);


```
