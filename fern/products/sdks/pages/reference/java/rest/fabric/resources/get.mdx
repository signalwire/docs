Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`Map<String, Object>` -- The Fabric resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var resource = client.fabric.resources.get("resource-id");
System.out.println(resource['type'] + ": " + resource['name'] + ", ID: " + resource['id']);


```
