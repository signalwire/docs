Retrieve a single SWML script resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

## **Returns**

`Map<String, Object>` -- The SWML script resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var item = client.fabric.swml_scripts.get("resource-id");
System.out.println("Swml Script: " + item['name'] + ", ID: " + item['id']);


```
