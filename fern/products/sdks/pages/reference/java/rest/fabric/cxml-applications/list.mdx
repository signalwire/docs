List cXML application resources in the project.

## **Parameters**

<ParamField path="pageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="pageToken" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing cXML application resources.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var response = client.fabric.cxml_applications.list();
for app in response.get("data", []):
    System.out.println(app['name'] + ": " + app['id']);


```
