List all Fabric resources across all resource types.

## **Parameters**

<ParamField path="pageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="pageToken" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing Fabric resources.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var response = client.fabric.resources.list();
for resource in response.get("data", []):
    System.out.println(resource['type'] + ": " + resource['name'] + " (" + resource['id'] + ")");


```
