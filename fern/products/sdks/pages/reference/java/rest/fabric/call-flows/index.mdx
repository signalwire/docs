Manage call flow resources with version management and deployment. Extends standard CRUD
(PUT updates) with methods to list versions and deploy specific versions.

Access via `client.fabric.call_flows` on a [`RestClient`][restclient] instance.

```java
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var items = client.fabric.call_flows.list();


```
