Retrieve a single FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`Map<String, Object>` -- The FreeSWITCH connector resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var connector = client.fabric.freeswitch_connectors.get("connector-id");
System.out.println("Freeswitch Connector: " + connector['name'] + ", ID: " + connector['id']);


```
