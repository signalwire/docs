Delete a call flow resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the call flow to delete.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success (HTTP 204).

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.fabric.call_flows.delete("flow-id")
System.out.println("Call Flow deleted");


```
