Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`Map<String, Object>` -- The SIP gateway resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var gateway = client.fabric.sip_gateways.get("gateway-id");
System.out.println("Sip Gateway: " + gateway['name'] + ", ID: " + gateway['id']);


```
