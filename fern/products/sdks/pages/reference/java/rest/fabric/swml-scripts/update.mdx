Replace a SWML script resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Map<String, Object>` -- The updated SWML script resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.swml_scripts.update("resource-id", "updated-name");
System.out.println("Updated: " + updated['name']);


```
