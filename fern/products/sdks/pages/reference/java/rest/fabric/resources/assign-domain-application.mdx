Assign a domain application to a Fabric resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Map<String, Object>` -- The assigned domain application.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.fabric.resources.assign_domain_application( "resource-id", "example.signalwire.com", );
System.out.println("Domain application assigned: " + result['id']);


```
