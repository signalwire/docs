Create a subscriber authentication token.

## **Parameters**

<ParamField path="subscriberId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Map<String, Object>` -- The created subscriber token.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var token = client.fabric.tokens.create_subscriber_token("subscriber-id");
System.out.println("Token: " + token['token']);


```
