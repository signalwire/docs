Partially update a SIP gateway resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`Map<String, Object>` -- The updated SIP gateway resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.fabric.sip_gateways.update("gateway-id", "updated-name");
System.out.println("Updated: " + updated['name']);


```
