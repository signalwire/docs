List SIP endpoints associated with a subscriber.

## **Parameters**

<ParamField path="subscriberId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="pageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="pageToken" type="String" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`Map<String, Object>` -- Paginated response containing SIP endpoints for the subscriber.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var response = client.fabric.subscribers.list_sip_endpoints("subscriber-id");
for ep in response.get("data", []):
    System.out.println(ep['username'] + ": " + ep['id']);


```
