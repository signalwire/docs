Retrieve a single SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriberId" type="String" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpointId" type="String" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`Map<String, Object>` -- The SIP endpoint.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var endpoint = client.fabric.subscribers.get_sip_endpoint("subscriber-id", "endpoint-id");
System.out.println("SIP Endpoint: " + endpoint['username'] + ", ID: " + endpoint['id']);


```
