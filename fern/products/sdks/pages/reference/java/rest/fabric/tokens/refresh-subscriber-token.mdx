Refresh an existing subscriber authentication token.

## **Parameters**

<ParamField path="token" type="String" required={true} toc={true}>
  The existing subscriber token to refresh.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`Map<String, Object>` -- The refreshed subscriber token.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var refreshed = client.fabric.tokens.refresh_subscriber_token("existing-token");
System.out.println("Refreshed token: " + refreshed['token']);


```
