Retrieve a single conference room resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`Map<String, Object>` -- The conference room resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var room = client.fabric.conference_rooms.get("room-id");
System.out.println("Conference Room: " + room['name'] + ", ID: " + room['id']);


```
