Retrieve a single cXML script resource.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`Map<String, Object>` -- The cXML script resource.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var item = client.fabric.cxml_scripts.get("resource-id");
System.out.println("Cxml Script: " + item['name'] + ", ID: " + item['id']);


```
