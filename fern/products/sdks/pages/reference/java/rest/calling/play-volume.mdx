Adjust the volume of an active playback.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="String" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

<ParamField path="volume" type="double" required={true} toc={true}>
  Volume level adjustment. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the volume was adjusted.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.play_volume("call-id-xxx", "ctrl-id", 5.0)


```
