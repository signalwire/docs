Start collecting user input on an active call. Supports DTMF digit collection
and speech recognition. Returns a `controlId` for managing the collection.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to collect input on.
</ParamField>

<ParamField path="collect" type="Map<String, Object>" required={true} toc={true}>
  Collection configuration. Supports the following keys:

  - `digits` — DTMF digit collection settings (e.g., `{"max": 4, "terminators": "#"}`)
  - `speech` — Speech recognition settings (e.g., `{"end_silence_timeout": 1.0}`)
</ParamField>

<ParamField path="initialTimeout" type="double" toc={true}>
  Seconds to wait for the first input before timing out.
</ParamField>

<ParamField path="partialResults" type="boolean" toc={true}>
  Whether to send partial speech recognition results as events.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response containing the `controlId`.

## **Examples**

### Collect DTMF Digits

```java {10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Collect DTMF digits
var result = client.calling.collect( "call-id-xxx", { "digits": {"max": 4, "terminators": "#", "digit_timeout": 5.0} }, 10.0, );


```

### Collect Speech

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.calling.collect( "call-id-xxx", { "speech": { "end_silence_timeout": 1.0, "language": "en-US", } }, );


```
