Stop an active AI session on a call. The AI agent is disconnected but
the call itself remains active, allowing further call control operations.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the AI session was stopped.

## **Example**

```java {10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Stop AI and then play a goodbye message
client.calling.ai_stop("call-id-xxx")
client.calling.play( "call-id-xxx", [{"type": "tts", "text": "Thank you for calling. Goodbye!"}] )
client.calling.end("call-id-xxx", "hangup")


```
