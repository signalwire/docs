Pause an active playback.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="String" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming playback was paused.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.play_pause("call-id-xxx", "ctrl-id")


```
