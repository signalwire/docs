Manually start the input timers for a collection that was started without
automatic timer activation. This is useful when you want to play a prompt
before starting the timer countdown.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="String" required={true} toc={true}>
  The control ID returned from [`collect()`][collect].
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the input timers were started.

## **Example**

```java {23}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Start collection without auto-timers, play prompt, then start timers
var result = client.calling.collect( "call-id-xxx", {"digits": {"max": 4, "terminators": "#"}}, );
var control_id = result.get("control_id");

// Play a prompt first
client.calling.play( "call-id-xxx", [{"type": "tts", "text": "Please enter your 4-digit PIN."}] )

// Now start the input timers
client.calling.collect_start_input_timers( "call-id-xxx", control_id, )


```
