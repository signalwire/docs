Start a detector on an active call. Detectors can identify answering machines,
fax tones, or DTMF digits. Returns a `controlId` for managing the detector.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to run detection on.
</ParamField>

<ParamField path="detect" type="Map<String, Object>" required={true} toc={true}>
  Detection configuration. Contains a `type` field and type-specific parameters:

  - `"machine"` — Answering machine detection (AMD). Optional: `initialTimeout`, `endSilenceTimeout`, `machineVoiceThreshold`.
  - `"fax"` — Fax tone detection. Optional: `tone`.
  - `"digit"` — DTMF digit detection. Optional: `digits`.
</ParamField>

<ParamField path="timeout" type="double" toc={true}>
  Maximum seconds to run detection before automatically stopping.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response containing the `controlId`.

## **Examples**

### Answering Machine Detection

```java {10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Answering machine detection
var result = client.calling.detect( "call-id-xxx", { "type": "machine", "params": { "initial_timeout": 4.5, "end_silence_timeout": 1.0, } }, 30.0, );


```
