Start transcribing speech on an active call. Transcription results are delivered
via events. Returns a `controlId` for stopping the transcription.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to transcribe.
</ParamField>

<ParamField path="language" type="String" toc={true}>
  Language code for transcription (e.g., `"en-US"`, `"es-ES"`).
</ParamField>

## **Returns**

`Map<String, Object>` — The API response containing the `controlId`.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.calling.transcribe( "call-id-xxx", "en-US", );
var control_id = result.get("control_id");


```
