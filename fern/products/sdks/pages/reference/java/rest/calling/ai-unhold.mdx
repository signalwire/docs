[ai-hold]: /docs/sdks/reference/java/rest/calling/ai-hold

Resume an AI session that was previously put on hold with
[`aiHold()`][ai-hold].

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call with a held AI session.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the AI session was resumed.

## **Example**

```java {14}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Hold, perform an action, then resume
client.calling.ai_hold("call-id-xxx")

// ... do some background processing ...

client.calling.ai_unhold("call-id-xxx")


```
