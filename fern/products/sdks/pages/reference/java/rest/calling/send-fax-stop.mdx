Stop an in-progress fax send operation on an active call.

<Note>
This method stops a fax send that is already in progress. To initiate fax sending,
use the RELAY client's [`sendFax()`][send-fax] method.
</Note>

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call with an active fax send operation.
</ParamField>

<ParamField path="controlId" type="String" required={true} toc={true}>
  The control ID of the fax send operation to stop.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the fax send was stopped.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.send_fax_stop("call-id-xxx", "ctrl-id")


```
