Start recording an active call. Returns a `controlId` used to
[pause][pause],
[resume][resume],
or [stop][stop] the recording.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to record.
</ParamField>

<ParamField path="record" type="Map<String, Object>" required={true} toc={true}>
  Recording configuration object. Contains:

  - `audio` — Audio recording settings (e.g., `{"format": "mp3", "stereo": true}`)
</ParamField>

<ParamField path="listenOn" type="String" toc={true}>
  Which leg of the call to record.

  - `"self"` -- record only the audio from the call's own leg
  - `"both"` -- record audio from both legs of the call
</ParamField>

## **Returns**

`Map<String, Object>` — The API response containing the `controlId` for managing the recording.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.calling.record( "call-id-xxx", {"audio": {"format": "mp3", "stereo": true}} );
var control_id = result.get("control_id");


```
