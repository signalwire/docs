Put an active AI session on hold. The AI agent stops processing speech
while the call remains connected. This is useful for transferring the
caller to a human agent or performing background operations.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the AI session was put on hold.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.ai_hold("call-id-xxx")


```
