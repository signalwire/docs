Disconnect (unbridge) a connected call. If two call legs are bridged together,
this separates them without ending either leg.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to disconnect.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the disconnect.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.disconnect("call-id-xxx")


```
