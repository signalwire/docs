Start live translation on an active call. Spoken audio is transcribed and
translated in real time to a target language.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="sourceLanguage" type="String" toc={true}>
  The language being spoken on the call (e.g., `"en-US"`).
</ParamField>

<ParamField path="targetLanguage" type="String" toc={true}>
  The language to translate into (e.g., `"es-ES"`).
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming live translation was started.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.live_translate( "call-id-xxx", "en-US", "es-ES", )


```
