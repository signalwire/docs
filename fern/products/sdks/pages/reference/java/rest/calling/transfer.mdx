Transfer an active call to a new destination. The current call leg is replaced
by a new connection to the specified target.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to transfer.
</ParamField>

<ParamField path="dest" type="Map<String, Object>" required={true} toc={true}>
  The transfer destination. Contains `type` (e.g., `"phone"`, `"sip"`) and
  type-specific parameters such as `toNumber`.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the transfer was initiated.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.transfer( "call-id-xxx", {"type": "phone", "params": {"to_number": "+15559876543"}} )


```
