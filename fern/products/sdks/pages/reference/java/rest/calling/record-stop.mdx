[record]: /docs/sdks/reference/java/rest/calling/record

Stop an active recording. The recording file becomes available after stopping.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="String" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`Map<String, Object>` — The API response containing the recording URL and metadata.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.calling.record_stop("call-id-xxx", "ctrl-id");
System.out.println(result);


```
