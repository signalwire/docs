Send a message to an active AI session on a call. This injects instructions
or context into the AI agent's conversation without the caller hearing the
message directly.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

<ParamField path="messageText" type="String" required={true} toc={true}>
  The message text to send to the AI agent. The agent processes this as
  additional context or instruction.
</ParamField>

<ParamField path="role" type="String" toc={true}>
  The role of the message sender. Typically `"system"` or `"user"`.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the message was delivered.

## **Example**

```java {10}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

// Inject a system instruction into the AI session
client.calling.ai_message( "call-id-xxx", "The caller's account has been verified. You can proceed with the transfer.", "system", )


```
