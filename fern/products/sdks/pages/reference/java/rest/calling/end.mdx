End an active call.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call to end.
</ParamField>

<ParamField path="reason" type="String" toc={true}>
  The reason for ending the call.

  - `"hangup"` -- normal call termination
  - `"busy"` -- end the call with a busy signal
  - `"cancel"` -- cancel the call before it is answered
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the call was ended.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.end("call-id-xxx", "hangup")


```
