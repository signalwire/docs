Update parameters on an active call, such as metadata or call settings.

<Note>
Unlike most calling methods, `update()` does **not** take a positional `callId`
parameter. Pass all fields (including call identification) as keyword arguments.
</Note>

## **Parameters**

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Keyword arguments forwarded to the SignalWire API. Common fields include
  `callId`, `metadata`, and other call settings.
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the update.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.update("call-id-xxx", {"department": "sales"})


```
