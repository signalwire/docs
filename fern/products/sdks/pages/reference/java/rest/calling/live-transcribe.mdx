Start live transcription on an active call. Unlike standard transcription,
live transcription streams partial results in real time as the caller speaks,
enabling display of in-progress speech.

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="language" type="String" toc={true}>
  Language code for transcription (e.g., `"en-US"`).
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming live transcription was started.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.live_transcribe( "call-id-xxx", "en-US", )


```
