Pause an active recording. The recording can be resumed later with
[`recordResume()`][record-resume].

## **Parameters**

<ParamField path="callId" type="String" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="String" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`Map<String, Object>` — The API response confirming the recording was paused.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.calling.record_pause("call-id-xxx", "ctrl-id")


```
