List chunks belonging to a document. Chunks are the individual text segments
created when a document is processed for vector search.

## **Parameters**

<ParamField path="documentId" type="String" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Optional query parameters for pagination.
</ParamField>

## **Returns**

`Map<String, Object>` -- JSON response containing a `data` list of chunk objects.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var chunks = client.datasphere.documents.list_chunks("document-id");
for chunk in chunks.get("data", []):
    System.out.println(chunk.get("id"), chunk.get("text")[:80]);


```
