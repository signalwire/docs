Perform a semantic search across all documents in the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="queryString" type="String" required={true} toc={true}>
  The natural-language search query.
</ParamField>

<ParamField path="count" type="int" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Additional search parameters (e.g., `tags`, `documentId` to narrow scope).
</ParamField>

## **Returns**

`Map<String, Object>` -- Search results with matching document chunks ranked by relevance.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var results = client.datasphere.documents.search( "How do I reset my password?", 5, );
for chunk in results.get("data", []):
    System.out.println(chunk.get("text"), "score:", chunk.get("score"));


```
