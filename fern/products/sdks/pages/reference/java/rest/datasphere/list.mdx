List documents in the project.

## **Parameters**

<ParamField path="**params" type="Map<String, Object>" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Map<String, Object>` -- JSON response containing a `data` list of document objects.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var result = client.datasphere.documents.list();
for doc in result.get("data", []):
    System.out.println(doc.get("name"), doc.get("id"));


```
