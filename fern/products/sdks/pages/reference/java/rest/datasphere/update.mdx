Update an existing document.

## **Parameters**

<ParamField path="resourceId" type="String" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Fields to update (e.g., `name`, `content`, `metadata`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The updated document object.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var updated = client.datasphere.documents.update( "document-id", "Updated FAQ", );
System.out.println(updated.get("name"));


```
