Upload a new document to the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="name" type="String" toc={true}>
  Document name or title.
</ParamField>

<ParamField path="content" type="String" toc={true}>
  Document content text.
</ParamField>

<ParamField path="**kwargs" type="Map<String, Object>" toc={true}>
  Additional document fields (e.g., `metadata`).
</ParamField>

## **Returns**

`Map<String, Object>` -- The newly created document object.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var doc = client.datasphere.documents.create( "FAQ", "Frequently asked questions and answers...", );
System.out.println("Document ID:", doc.get("id"));


```
