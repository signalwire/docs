Delete a specific chunk from a document.

## **Parameters**

<ParamField path="documentId" type="String" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunkId" type="String" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- Empty response on success.

## **Example**

```java {9-9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

client.datasphere.documents.delete_chunk("document-id", "chunk-id")


```
