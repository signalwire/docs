Retrieve a specific chunk from a document.

## **Parameters**

<ParamField path="documentId" type="String" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunkId" type="String" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`Map<String, Object>` -- The chunk object.

## **Example**

```java {9}
import com.signalwire.sdk.rest.RestClient;

var client = RestClient.builder()
        .project("your-project-id")
        .token("your-api-token")
        .host("your-space.signalwire.com")
        .build();

var chunk = client.datasphere.documents.get_chunk("document-id", "chunk-id");
System.out.println(chunk.get("text"));


```
