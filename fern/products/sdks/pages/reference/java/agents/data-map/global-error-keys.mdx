[ref-datamap]: /docs/sdks/reference/java/agents/data-map

Set global error response keys. When any of these keys appear in the response,
the data map treats the result as an error regardless of expression matches.

## Parameters

<ParamField path="keys" type="List<String>" required={true} toc={true}>
  List of response keys that indicate an error condition.
</ParamField>

## Returns

[`DataMap`][ref-datamap] -- Returns `this` for method chaining.

## Example

```java
import com.signalwire.sdk.datamap.DataMap;

var dataMap = new DataMap("get_plant_info")
        .description("Get plant information")
        .webhook("GET", "https://api.example.com/plants/${args.plantId}")
        .globalErrorKeys(List.of("error", "fault", "exception"));
```
