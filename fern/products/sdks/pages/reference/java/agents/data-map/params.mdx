[ref-datamap]: /docs/sdks/reference/java/agents/data-map

Set query parameters for the data map's webhook request. These parameters are
appended to the webhook URL.

## Parameters

<ParamField path="data" type="Map<String, Object>" required={true} toc={true}>
  Map of query parameter names to values or template expressions.
</ParamField>

## Returns

[`DataMap`][ref-datamap] -- Returns `this` for method chaining.

## Example

```java
import com.signalwire.sdk.datamap.DataMap;

var dataMap = new DataMap("get_plant_price")
        .description("Look up plant pricing")
        .webhook("GET", "https://api.example.com/plants/price")
        .params(Map.of("name", "${args.plantName}", "format", "json"));
```
