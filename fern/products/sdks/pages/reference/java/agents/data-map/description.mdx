Set the description for this data map function. The AI reads this description
to determine when to invoke the function.

## Parameters

<ParamField path="description" type="String" required={true} toc={true}>
  Human-readable description of what the function does.
</ParamField>

## Returns

[`DataMap`][ref-datamap] -- Returns `this` for method chaining.

## Example

```java
import com.signalwire.sdk.datamap.DataMap;

var dataMap = new DataMap("get_plant_price")
        .description("Look up the price of a plant by name")
        .parameter("plantName", "string", "Name of the plant", true);
```
