List all active sessions with their metadata. Expired and dead sessions are
automatically cleaned up during this call. Returns a dictionary keyed by
session ID, where each value contains `serviceName`, `createdAt`,
`lastAccessed`, `timeout`, `metadata`, and `timeRemaining`.

## **Parameters**

None.

## **Returns**

`Map<String, Map<String, Object>>` -- A dictionary mapping session IDs to session info dictionaries.

## **Example**

```java {4}

var manager = SessionManager(config);
var sessions = manager.listSessions();
for session_id, info in sessions.items():
    System.out.println(session_id + ": " + info['service_name'] + " (" + info['time_remaining']:.0f + "s remaining)");


```
