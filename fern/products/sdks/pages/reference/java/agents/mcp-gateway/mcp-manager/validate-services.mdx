Validate that all configured services can be started successfully. Starts and
immediately stops a client for each service. Returns a dictionary mapping
service names to their validation result (`true` = success).

This method is called automatically during [MCPGateway][ref-mcpgateway] initialization.

## **Parameters**

None.

## **Returns**

`Map<String, Boolean>` -- A dictionary mapping each service name to `true` (valid) or `false` (failed).

## **Example**

```java {11}

var config = Map.of("services", Map.of("todo", Map.of("command", List.of("python3", "todo_mcp.py"), "description", "Todo list", "enabled", true), "search", Map.of("command", List.of("node", "search_mcp.js"), "description", "Search", "enabled", true)));

var manager = MCPManager(config);
var results = manager.validateServices();
for service, valid in results.items():
    var status = "OK" if valid else "FAILED";
    System.out.println("  " + service + ": " + status);


```
