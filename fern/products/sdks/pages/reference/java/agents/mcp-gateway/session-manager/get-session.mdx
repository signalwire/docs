Retrieve an active session by ID. Returns `null` if the session does not exist,
has expired, or if its underlying process has died. Automatically calls
`touch()` on the session to reset its expiration timer.

## **Parameters**

<ParamField path="sessionId" type="String" required={true} toc={true}>
  The session ID to look up.
</ParamField>

## **Returns**

[`Session`][session] or `null`

## **Example**

```java {4}

var manager = SessionManager(config);
var session = manager.getSession("call-abc-123");
if session:
    System.out.println("Session active for service: " + session.service_name);
else:
    System.out.println("Session not found or expired");


```
