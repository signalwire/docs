[mcpgateway-shutdown]: /docs/sdks/reference/java/agents/mcp-gateway/mcp-gateway/shutdown

Stop all active MCP clients that were created via
[`createClient()`][create-client].
Each client's subprocess is terminated and its sandbox directory cleaned up.
Called automatically during
[`MCPGateway.shutdown()`][mcpgateway-shutdown].

## **Parameters**

None.

## **Returns**

`null`

## **Example**

```java {11}

var config = Map.of("services", Map.of("todo", Map.of("command", List.of("python3", "todo_mcp.py"), "description", "Todo list", "enabled", true)));

var manager = MCPManager(config);
var client = manager.createClient("todo");
manager.shutdown();


```
