Return the names of all skills registered in the global registry.

## Returns

`Set<String>` -- Set of registered skill names.

## Example

```java
import com.signalwire.sdk.skills.SkillRegistry;

Set<String> available = SkillRegistry.list();
for (String name : available) {
    System.out.println("Available: " + name);
}
```
