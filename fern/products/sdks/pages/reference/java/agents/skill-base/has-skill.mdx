Check whether a skill with the given name is currently active on this agent.

## Parameters

<ParamField path="skillName" type="String" required={true} toc={true}>
  Name of the skill to check.
</ParamField>

## Returns

`boolean` -- `true` if the skill is active, `false` otherwise.

## Example

```java
if (agent.hasSkill("native_vector_search")) {
    System.out.println("Search skill is active");
}
```
