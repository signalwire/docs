Check whether all `REQUIRED_ENV_VARS` are set in the environment.

## **Returns**

`boolean` -- `true` if all required variables are present, `false` with error
logging otherwise.

---

Check whether all `REQUIRED_PACKAGES` can be imported.

## **Returns**

`boolean` -- `true` if all required packages are available, `false` with error
logging otherwise.

## **Example**

```java {9-12}
public class MySkill extends SkillBase {
    public static final String SKILL_NAME = "my_skill";
    public static final String SKILL_DESCRIPTION = "Example skill";
    public static final List<String> REQUIRED_PACKAGES = List.of("requests", "beautifulsoup4");
    public static final List<String> REQUIRED_ENV_VARS = List.of("MY_API_KEY", "MY_SECRET");

    @Override
    public boolean setup() {
        if (!validatePackages()) {
            return false;
        }
        if (!validateEnvVars()) {
            return false;
        }
        return true;
    }

    @Override
    public void registerTools() {
        // ...
    }
}
```
