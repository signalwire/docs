Register a custom skill factory in the global skill registry. Once registered,
the skill can be added to any agent by name.

## Parameters

<ParamField path="name" type="String" required={true} toc={true}>
  Unique name for the skill.
</ParamField>

<ParamField path="factory" type="Supplier<SkillBase>" required={true} toc={true}>
  Factory function that creates new instances of the skill.
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.skills.SkillRegistry;

SkillRegistry.register("plant_lookup", PlantLookupSkill::new);
```
