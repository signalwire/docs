---

Remove a previously added skill from the agent. The skill's tools are
unregistered and its resources are cleaned up.

## Parameters

<ParamField path="skillName" type="String" required={true} toc={true}>
  Name of the skill to remove.
</ParamField>

## Returns

`void`

## Example

```java
agent.removeSkill("native_vector_search");
```
