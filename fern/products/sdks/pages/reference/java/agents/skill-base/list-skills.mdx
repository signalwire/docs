Return the names of all skills currently active on this agent.

## Returns

`List<String>` -- List of active skill names.

## Example

```java
List<String> activeSkills = agent.listSkills();
for (String skill : activeSkills) {
    System.out.println("Active skill: " + skill);
}
```
