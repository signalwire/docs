Remove a skill from the global registry. After unregistering, the skill
can no longer be added to agents by name.

## Parameters

<ParamField path="name" type="String" required={true} toc={true}>
  Name of the skill to unregister.
</ParamField>

## Returns

`void`

## Example

```java
import com.signalwire.sdk.skills.SkillRegistry;

SkillRegistry.unregister("plant_lookup");
```
