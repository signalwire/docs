Create and return a new instance of a registered skill.

## Parameters

<ParamField path="name" type="String" required={true} toc={true}>
  Name of the registered skill.
</ParamField>

## Returns

[`SkillBase`][ref-skillbase] -- A new instance of the skill, or `null` if not found.

## Example

```java
import com.signalwire.sdk.skills.SkillRegistry;

var skill = SkillRegistry.get("plant_lookup");
```
