Configure the server to serve static files from a local directory at the
specified route. Useful for hosting a web UI alongside agent endpoints.

## Parameters

<ParamField path="directory" type="String" required={true} toc={true}>
  Path to the directory containing static files.
</ParamField>

<ParamField path="route" type="String" required={true} toc={true}>
  URL path prefix for serving the files (e.g., `"/web"`).
</ParamField>

## Returns

[`AgentServer`][ref-agentserver] -- Returns `this` for method chaining.

## Example

```java
import com.signalwire.sdk.server.AgentServer;

var server = new AgentServer(8080);
server.serveStaticFiles("./public", "/web");
server.run();
```
