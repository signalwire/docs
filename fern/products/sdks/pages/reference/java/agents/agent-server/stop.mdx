Stop the agent server gracefully. This shuts down the HTTP server and releases
all resources.

## Returns

`void`

## Example

```java
import com.signalwire.sdk.server.AgentServer;

var server = new AgentServer(8080);
server.run();
// Later, stop the server
server.stop();
```
