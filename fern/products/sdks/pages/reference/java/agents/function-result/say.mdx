[functionresult]: /docs/sdks/reference/java/agents/function-result

Make the AI agent speak specific text immediately, bypassing the normal LLM
response flow. The text is spoken in the agent's configured voice.

## **Parameters**

<ParamField path="text" type="String" required={true} toc={true}>
  Text for the agent to speak.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```java {12}
import com.signalwire.sdk.agent.AgentBase;
import com.signalwire.sdk.swaig.FunctionResult;

var agent = AgentBase.builder()
        .name("my-agent")
        .route("/agent")
        .build();
agent.setPromptText("You are a helpful assistant.");

agent.defineTool("announceStatus",
    "Announce the order status",
    (args, rawData) -> {
        var status = args.get("status");
        return ( FunctionResult() .say(f"Your order status is: {status}") );
    });
agent.serve();


```
