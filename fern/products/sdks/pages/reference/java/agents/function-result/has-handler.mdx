Check whether this tool definition has a local handler function attached.
Tools without handlers forward calls to an external webhook URL instead.

## Returns

`boolean` -- `true` if a local handler is set, `false` otherwise.

## Example

```java
import com.signalwire.sdk.swaig.ToolDefinition;

var tool = new ToolDefinition("get_plant_info", "Get plant information",
        Map.of("type", "object"), handler);

if (tool.hasHandler()) {
    System.out.println("Tool has a local handler");
}
```
