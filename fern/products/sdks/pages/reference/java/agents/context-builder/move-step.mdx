Move a step to a different position in the step sequence within this context.

## Parameters

<ParamField path="stepName" type="String" required={true} toc={true}>
  Name of the step to move.
</ParamField>

<ParamField path="position" type="int" required={true} toc={true}>
  Zero-based target position for the step.
</ParamField>

## Returns

`Context` -- Returns `this` for method chaining.

## Example

```java
agent.contexts()
        .addContext("ordering")
            .moveStep("confirm_order", 0)
        .done();
```
