[ref-step]: /docs/sdks/reference/java/agents/context-builder/step

Skip waiting for user input after this step completes. The AI proceeds
immediately without waiting for the caller to speak.

## **Parameters**

<ParamField path="skip" type="boolean" required={true} toc={true}>
  Whether to skip the user turn.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```java {9}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("my-agent")
        .route("/agent")
        .build();

var contexts = agent.defineContexts();
var ctx = contexts.addContext("default");
var intro = ctx.addStep("intro");
intro.setText("Introduce yourself and immediately begin the intake process.");
intro.setSkipUserTurn(true);
ctx.addStep("intake").setText("Collect the caller's information.");

agent.serve();


```
