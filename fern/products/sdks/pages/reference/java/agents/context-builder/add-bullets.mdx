Add a titled section with bullet points to this context's prompt.

## Parameters

<ParamField path="title" type="String" required={true} toc={true}>
  Section heading.
</ParamField>

<ParamField path="bullets" type="List<String>" required={true} toc={true}>
  List of bullet point strings.
</ParamField>

## Returns

`Context` -- Returns `this` for method chaining.

## Example

```java
agent.contexts()
        .addContext("plant_care")
            .addBullets("Care tips", List.of(
                    "Water when topsoil is dry",
                    "Provide indirect sunlight",
                    "Fertilize monthly during growing season"
            ))
        .done();
```
