Add language-specific filler phrases spoken when the conversation transitions
into this context.

## Parameters

<ParamField path="languageCode" type="String" required={true} toc={true}>
  Language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="fillers" type="List<String>" required={true} toc={true}>
  Filler phrases for the specified language.
</ParamField>

## Returns

`Context` -- Returns `this` for method chaining.

## Example

```java
agent.contexts()
        .addContext("plant_search")
            .addEnterFiller("en-US", List.of(
                    "Let me look that up...",
                    "Searching our plant catalog..."
            ))
        .done();
```
