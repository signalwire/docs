Remove a step from this context by name.

## Parameters

<ParamField path="stepName" type="String" required={true} toc={true}>
  Name of the step to remove.
</ParamField>

## Returns

`Context` -- Returns `this` for method chaining.

## Example

```java
agent.contexts()
        .addContext("ordering")
            .removeStep("optional_step")
        .done();
```
