List the names of all currently loaded skills on this agent.

## **Returns**

`List<String>` -- List of loaded skill names.

## **Example**

```java {6}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("assistant")
        .route("/assistant")
        .build();
agent.addSkill("datetime");
agent.addSkill("math");
System.out.println(agent.list_skills());


```
