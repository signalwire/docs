Add multiple speech recognition hints at once.

## **Parameters**

<ParamField path="hints" type="List<String>" required={true} toc={true}>
  A list of words or phrases.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `this` for method chaining.

## **Example**

```java {5}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("support")
        .route("/support")
        .build();
agent.setPromptText("You are a helpful assistant.");
agent.addHints(List.of("SWML", "SWAIG", "FreeSWITCH"));
agent.serve();


```
