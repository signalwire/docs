[set-prompt-text]: /docs/sdks/reference/java/agents/agent-base/set-prompt-text

Retrieve the current prompt. Returns a string if
[`setPromptText()`][set-prompt-text] was used, or a
POM structure (list of dictionaries) if prompt sections were used. If neither was
explicitly set, returns a default prompt: `"You are {name}, a helpful AI assistant."`.

## **Returns**

`String | List<Map<String, Object>>` -- The current prompt in whatever format was configured.

## **Example**

```java {5}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("support")
        .route("/support")
        .build();
agent.setPromptText("You are a customer support agent for Acme Corp.");
var prompt = agent.getPrompt();
System.out.println(prompt);


```
