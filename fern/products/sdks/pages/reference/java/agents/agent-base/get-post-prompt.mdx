Retrieve the current post-prompt text.

## **Returns**

`Optional[str]` -- The post-prompt text, or `null` if not set.

## **Example**

```java {5}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("support")
        .route("/support")
        .build();
agent.setPostPrompt("Summarize this call as JSON with intent, resolution, and sentiment.");
var post_prompt = agent.getPostPrompt();
if post_prompt:
    System.out.println(post_prompt);


```
