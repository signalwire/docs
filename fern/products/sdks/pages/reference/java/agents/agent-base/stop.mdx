Stop the agent and shut down its HTTP server gracefully. This releases all
resources and stops accepting new requests.

## Returns

`void`

## Example

```java
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("plant-assistant")
        .route("/plants")
        .build();

agent.serve();
// Later, shut down the agent
agent.stop();
```
