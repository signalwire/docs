[ref-agentbase]: /docs/sdks/reference/java/agents/agent-base

Unload a skill from this agent. The skill's tools and configuration are removed.

## **Parameters**

<ParamField path="skillName" type="String" required={true} toc={true}>
  Name of the skill to remove.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `this` for method chaining.

## **Example**

```java {6}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("assistant")
        .route("/assistant")
        .build();
agent.addSkill("datetime");
agent.addSkill("math");
agent.removeSkill("math");
System.out.println(agent.list_skills());


```
