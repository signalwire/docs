Replace all pronunciation rules at once with a list of raw rule dictionaries.

## **Parameters**

<ParamField path="pronunciations" type="List<Map<String, Object>>" required={true} toc={true}>
  List of pronunciation rule dictionaries. Each dictionary must contain `replace` and
  `with` keys, and optionally `ignoreCase`.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `this` for method chaining.

## **Example**

```java {5}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("support")
        .route("/support")
        .build();
agent.setPromptText("You are a helpful assistant.");
agent.setPronunciations(List.of({"replace": "SQL", "with": "sequel"}, {"replace": "SWML", "with": "swimmel", "ignore_case": True}, {"replace": "IEEE", "with": "I triple E"}));
agent.serve();


```
