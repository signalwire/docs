Add a hint with pattern matching and replacement. This allows you to intercept
specific ASR output and rewrite it before it reaches the AI -- useful for correcting
consistent misrecognitions or normalizing variations.

## **Parameters**

<ParamField path="hint" type="String" required={true} toc={true}>
  The hint string to match against ASR output.
</ParamField>

<ParamField path="pattern" type="String" required={true} toc={true}>
  Regular expression pattern to match.
</ParamField>

<ParamField path="replace" type="String" required={true} toc={true}>
  Replacement text to substitute when the pattern matches.
</ParamField>

<ParamField path="ignoreCase" type="boolean" default="false" toc={true}>
  Whether to ignore case when matching the pattern.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `this` for method chaining.

## **Example**

```java {6}
import com.signalwire.sdk.agent.AgentBase;

var agent = AgentBase.builder()
        .name("support")
        .route("/support")
        .build();
agent.setPromptText("You are a helpful assistant.");
// Correct common ASR misrecognition
agent.addPatternHint("SignalWire", r"signal\s*wire|signal\s*wired", "SignalWire", true);
agent.serve();


```
