List countries that have phone numbers available for purchase.

<EndpointSchemaSnippet endpoint="GET /Accounts/{AccountSid}/AvailablePhoneNumbers" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /Accounts/{AccountSid}/AvailablePhoneNumbers" />

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const countries = await client.compat.phoneNumbers.listAvailableCountries();
```
