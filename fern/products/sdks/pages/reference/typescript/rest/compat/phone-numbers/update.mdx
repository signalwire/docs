Update a phone number's configuration (e.g., change voice/SMS URLs). Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The phone number SID.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/phone-numbers/_update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated phone number resource.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
await client.compat.phoneNumbers.update("PN...", {
  VoiceUrl: "https://example.com/voice"
});
```
