## **Parameters**

<ParamField path="conferenceSid" type="string">
  The SID of the conference.
</ParamField>

<ParamField path="recordingSid" type="string">
  The SID of the recording.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.compat.conferences.deleteRecording(conferenceSid, recordingSid);
console.log(result);
```
