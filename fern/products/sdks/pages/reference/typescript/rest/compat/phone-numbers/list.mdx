List incoming phone numbers in the account.

## **Parameters**

<ParamField path="PhoneNumber" type="string" toc={true}>
  Filter by exact phone number.
</ParamField>

<ParamField path="FriendlyName" type="string" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="PageSize" type="number" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="Page" type="number" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing phone number resources.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const numbers = await client.compat.phoneNumbers.list({ PageSize: 20 });
```
