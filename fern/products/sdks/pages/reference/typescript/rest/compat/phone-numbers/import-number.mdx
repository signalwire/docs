Import an external phone number into the account.

## **Parameters**

<Markdown src="/snippets/sdks/rest/compat/phone-numbers/_import-number-params.mdx" />

## **Returns**

`Promise<any>` -- The imported phone number resource.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const number = await client.compat.phoneNumbers.importNumber({
  number: "+15559876543",
  number_type: "longcode",
});
```
