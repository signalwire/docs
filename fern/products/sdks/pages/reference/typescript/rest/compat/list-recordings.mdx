## **Parameters**

<ParamField path="conferenceSid" type="string">
  The SID of the conference.
</ParamField>

<ParamField path="params" type="QueryParams">
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.compat.conferences.listRecordings(conferenceSid);
console.log(result);
```
