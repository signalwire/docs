List recordings for a conference.

## **Parameters**

<ParamField path="conferenceSid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="DateCreated" type="string" toc={true}>
  Filter recordings by creation date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated<" type="string" toc={true}>
  Filter recordings created before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated>" type="string" toc={true}>
  Filter recordings created after this date (ISO 8601 format).
</ParamField>

<ParamField path="Page" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- List of recording resources.
