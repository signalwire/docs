Remove a participant from a conference, ending their call leg.

## **Parameters**

<ParamField path="conferenceSid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="callSid" type="string" required={true} toc={true}>
  The call SID of the participant to remove.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.compat.conferences.removeParticipant("CF...", "CA...");
```
