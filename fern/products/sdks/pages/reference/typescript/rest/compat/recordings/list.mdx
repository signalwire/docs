List recordings in the account.

## **Parameters**

<ParamField path="DateCreated" type="string" toc={true}>
  Filter by creation date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated<" type="string" toc={true}>
  Filter by recordings created before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated>" type="string" toc={true}>
  Filter by recordings created after this date (ISO 8601 format).
</ParamField>

<ParamField path="CallSid" type="string" toc={true}>
  Filter by the call SID that generated the recording.
</ParamField>

<ParamField path="ConferenceSid" type="string" toc={true}>
  Filter by the conference SID that generated the recording.
</ParamField>

<ParamField path="Page" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing recording resources.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const recordings = await client.compat.recordings.list({ PageSize: 20 });
```
