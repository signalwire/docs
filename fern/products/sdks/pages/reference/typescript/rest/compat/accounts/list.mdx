List accounts.

## **Parameters**

<ParamField path="FriendlyName" type="string" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="Page" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- Response containing accounts.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const accounts = await client.compat.accounts.list();
```
