<Markdown src="/snippets/sdks/rest/compat/queues/_create-update-params.mdx" />

## **Returns**

`Promise<any>` -- The created queue resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const queue = await client.compat.queues.create({
  FriendlyName: "Support Queue",
  MaxSize: 100
});
```
