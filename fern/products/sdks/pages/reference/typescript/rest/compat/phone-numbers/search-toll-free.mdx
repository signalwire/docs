Search for available toll-free phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="string" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="Contains" type="string" toc={true}>
  Filter by number pattern (e.g., `"800"`).
</ParamField>

<ParamField path="AreaCode" type="string" toc={true}>
  Filter by area code.
</ParamField>

<ParamField path="SmsEnabled" type="boolean" toc={true}>
  Filter for SMS-capable numbers.
</ParamField>

<ParamField path="VoiceEnabled" type="boolean" toc={true}>
  Filter for voice-capable numbers.
</ParamField>

<ParamField path="FaxEnabled" type="boolean" toc={true}>
  Filter for fax-capable numbers.
</ParamField>

<ParamField path="MmsEnabled" type="boolean" toc={true}>
  Filter for MMS-capable numbers.
</ParamField>

<ParamField path="Beta" type="boolean" toc={true}>
  Include beta phone numbers in results.
</ParamField>

<ParamField path="InRegion" type="string" toc={true}>
  Filter by region or state abbreviation.
</ParamField>

<ParamField path="InLocality" type="string" toc={true}>
  Filter by locality or city name.
</ParamField>

<ParamField path="ExcludeAllAddressRequired" type="boolean" toc={true}>
  Exclude numbers that require any type of address.
</ParamField>

<ParamField path="ExcludeLocalAddressRequired" type="boolean" toc={true}>
  Exclude numbers that require a local address.
</ParamField>

<ParamField path="ExcludeForeignAddressRequired" type="boolean" toc={true}>
  Exclude numbers that require a foreign address.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing available toll-free number resources.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const numbers = await client.compat.phoneNumbers.searchTollFree("US", { Contains: "800" });
```
