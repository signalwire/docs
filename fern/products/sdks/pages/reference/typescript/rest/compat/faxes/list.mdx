List faxes in the account.

## **Parameters**

<ParamField path="DateCreatedAfter" type="string" toc={true}>
  Filter by faxes created after this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreatedOnOrBefore" type="string" toc={true}>
  Filter by faxes created on or before this date (ISO 8601 format).
</ParamField>

<ParamField path="From" type="string" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="To" type="string" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="Page" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing fax resources.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const faxes = await client.compat.faxes.list();
```
