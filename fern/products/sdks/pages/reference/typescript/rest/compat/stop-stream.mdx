## **Parameters**

<ParamField path="callSid" type="string">
  The SID of the call.
</ParamField>

<ParamField path="streamSid" type="string">
  The SID of the stream.
</ParamField>

<ParamField path="body" type="any = {}">
  The request body containing the resource data.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.compat.calls.stopStream(callSid, streamSid, body);
console.log(result);
```
