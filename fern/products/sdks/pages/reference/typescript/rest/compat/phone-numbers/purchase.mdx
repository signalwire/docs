Purchase a new phone number for the account.

## **Parameters**

<Markdown src="/snippets/sdks/rest/compat/phone-numbers/_purchase-params.mdx" />

## **Returns**

`Promise<any>` -- The purchased phone number resource.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const number = await client.compat.phoneNumbers.purchase({
  PhoneNumber: "+15551234567",
  FriendlyName: "Main Line"
});
```
