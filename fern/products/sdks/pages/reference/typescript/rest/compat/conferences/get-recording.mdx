Retrieve a specific conference recording.

## **Parameters**

<ParamField path="conferenceSid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="recordingSid" type="string" required={true} toc={true}>
  The SID of the recording.
</ParamField>

## **Returns**

`Promise<any>` -- The recording resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const recording = await client.compat.conferences.getRecording("CF...", "RE...");
```
