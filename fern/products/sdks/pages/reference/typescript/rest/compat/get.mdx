Retrieve a single resource by its SID. Available on most compatibility sub-resources
(`accounts`, `calls`, `messages`, `faxes`, `conferences`, `phoneNumbers`,
`applications`, `lamlBins`, `queues`, `recordings`, `transcriptions`).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The unique SID of the resource.
</ParamField>

## **Returns**

`Promise<any>` -- The resource object.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const call = await client.compat.calls.get("CA1234567890abcdef");
console.log(call);
```
