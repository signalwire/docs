Delete a resource by its SID. Available on sub-resources that support deletion
(`calls`, `messages`, `faxes`, `phoneNumbers`, `applications`, `lamlBins`,
`queues`, `recordings`, `transcriptions`, `tokens`).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The unique SID (or token ID) of the resource to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Server acknowledgment.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
await client.compat.recordings.delete("RE1234567890abcdef");
```
