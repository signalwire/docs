Retrieve a specific media item from a fax.

## **Parameters**

<ParamField path="fax_sid" type="string" required={true} toc={true}>
  The SID of the fax.
</ParamField>

<ParamField path="media_sid" type="string" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`Promise<any>` -- The media resource.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const item = await client.compat.faxes.getMedia("FX...", "ME...");
```
