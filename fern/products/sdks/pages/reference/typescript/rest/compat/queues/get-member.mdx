Retrieve a specific member from a queue by call SID.

## **Parameters**

<ParamField path="queueSid" type="string" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="callSid" type="string" required={true} toc={true}>
  The call SID of the queue member.
</ParamField>

## **Returns**

`Promise<any>` -- The queue member resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const member = await client.compat.queues.getMember("QU...", "CA...");
```
