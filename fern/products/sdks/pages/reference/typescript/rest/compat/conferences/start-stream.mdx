Start a media stream on a conference, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="conferenceSid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/conferences/_start-stream-params.mdx" />

## **Returns**

`Promise<any>` -- The created stream resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const stream = await client.compat.conferences.startStream(
  "CF...",
  { Url: "wss://stream.example.com" }
);
```
