List LAML bins in the account.

## **Parameters**

<ParamField path="Name" type="string" toc={true}>
  Filter by name.
</ParamField>

<ParamField path="Page" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing LAML bin resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const bins = await client.compat.lamlBins.list({ PageSize: 20 });
```
