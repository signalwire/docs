Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The token identifier.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/tokens/_update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated token resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.compat.tokens.update("token-id", { name: "updated-token" });
```
