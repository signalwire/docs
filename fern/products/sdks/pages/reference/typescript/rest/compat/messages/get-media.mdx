Retrieve a specific media item from a message.

## **Parameters**

<ParamField path="message_sid" type="string" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="media_sid" type="string" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`Promise<any>` -- The media resource including the content URL.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const item = await client.compat.messages.getMedia("SM...", "ME...");
```
