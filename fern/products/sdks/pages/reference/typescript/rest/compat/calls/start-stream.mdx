Start a media stream on an active call, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="call_sid" type="string" required={true} toc={true}>
  The SID of the call.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/calls/_start-stream-params.mdx" />

## **Returns**

`Promise<any>` -- The created stream resource.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const stream = await client.compat.calls.startStream("CA...", {
  Url: "wss://stream.example.com",
});
```
