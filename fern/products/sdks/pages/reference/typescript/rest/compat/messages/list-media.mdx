List media items (images, files) attached to a message.

## **Parameters**

<ParamField path="message_sid" type="string" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="DateCreated" type="string" toc={true}>
  Filter by creation date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated<" type="string" toc={true}>
  Filter by media created before this date (ISO 8601 format).
</ParamField>

<ParamField path="DateCreated>" type="string" toc={true}>
  Filter by media created after this date (ISO 8601 format).
</ParamField>

<ParamField path="Page" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="PageSize" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="PageToken" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- List of media resources attached to the message.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const media = await client.compat.messages.listMedia("SM...");
```
