Delete an API token.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The token identifier to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.compat.tokens.delete("token-id");
```
