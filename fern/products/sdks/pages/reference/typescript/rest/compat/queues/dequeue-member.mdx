Dequeue a member from a queue, redirecting the call to a new LAML URL.

## **Parameters**

<ParamField path="queueSid" type="string" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="callSid" type="string" required={true} toc={true}>
  The call SID of the queue member to dequeue.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/queues/_dequeue-member-params.mdx" />

## **Returns**

`Promise<any>` -- The dequeued member resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.compat.queues.dequeueMember(
  "QU...", "CA...",
  { Url: "https://example.com/dequeue" }
);
```
