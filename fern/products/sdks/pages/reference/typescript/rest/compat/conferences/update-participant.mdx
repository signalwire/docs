Update a participant in a conference (e.g., mute or hold).

## **Parameters**

<ParamField path="conferenceSid" type="string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="callSid" type="string" required={true} toc={true}>
  The call SID of the participant.
</ParamField>

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.compat.conferences.updateParticipant("CF...", "CA...", { Muted: true });
```
