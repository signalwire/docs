Create a new LAML bin containing a cXML/LaML script.

## **Parameters**

<Markdown src="/snippets/sdks/rest/compat/laml-bins/_create-update-params.mdx" />

## **Returns**

`Promise<any>` -- The created LAML bin resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const lamlBin = await client.compat.lamlBins.create({
  Name: "Greeting",
  Contents:
    '<?xml version="1.0" encoding="UTF-8"?>' +
    "<Response><Say>Hello!</Say></Response>",
});
```
