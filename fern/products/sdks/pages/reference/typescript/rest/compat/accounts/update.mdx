Update an account. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="string" required={true} toc={true}>
  The account SID to update.
</ParamField>

<Markdown src="/snippets/sdks/rest/compat/accounts/_create-update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated account resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.compat.accounts.update("account-sid", { FriendlyName: "Sales Team" });
```
