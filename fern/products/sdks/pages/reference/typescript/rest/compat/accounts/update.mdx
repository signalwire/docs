Update an account. Uses POST (Twilio convention).

<EndpointSchemaSnippet endpoint="POST /Accounts/{Sid}" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /Accounts/{Sid}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.compat.accounts.update("account-sid", { FriendlyName: "Sales Team" });
```
