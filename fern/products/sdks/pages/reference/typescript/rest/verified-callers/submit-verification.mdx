Submit the verification code received during the verification call or SMS.

## **Parameters**

<ParamField path="callerId" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="verification_code" type="string" required={true} toc={true}>
  The code received during the verification call or SMS.
</ParamField>

## **Returns**

`Promise<any>` -- The updated caller ID object with verified state.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.verifiedCallers.submitVerification("caller-id", {
    verification_code: "123456",
});
console.log("State:", result.verificationState);
```
