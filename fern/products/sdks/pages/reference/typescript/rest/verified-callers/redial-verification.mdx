Redial the verification call for a caller ID that has not yet been verified.
Use this when the original verification call was missed.

## **Parameters**

<ParamField path="callerId" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Confirmation that the verification call has been re-initiated.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.verifiedCallers.redialVerification("caller-id");
```
