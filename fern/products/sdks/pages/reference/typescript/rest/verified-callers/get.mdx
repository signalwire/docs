Retrieve a specific verified caller ID.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The caller ID object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const caller = await client.verifiedCallers.get("caller-id");
console.log(caller.phoneNumber, caller.verificationState);
```
