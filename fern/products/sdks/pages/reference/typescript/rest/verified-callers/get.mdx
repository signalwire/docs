Retrieve a specific verified caller ID.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`dict` -- The caller ID object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

caller = client.verified_callers.get("caller-id")
console.log(caller["phone_number"], caller["verification_state"])
```
