Delete a verified caller ID.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty dict on success.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.verifiedCallers.delete("caller-id");
console.log("Deleted successfully");
```
