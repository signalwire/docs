Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="phoneNumber" type="string" toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional parameters (e.g., `friendly_name`).
</ParamField>

## **Returns**

`dict` -- The newly created caller ID object with its verification state.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

caller = client.verified_callers.create(phone_number="+15551234567")
console.log("Verification state:", caller["verification_state"])
```
