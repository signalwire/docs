Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="number" type="string" required={true} toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="name" type="string" toc={true}>
  A friendly name for the caller ID (max 200 characters).
</ParamField>

<ParamField path="extension" type="string" toc={true}>
  The extension to dial after the verification call connects.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created caller ID object with its verification state.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const caller = await client.verifiedCallers.create({ number: "+15551234567" });
console.log("Verification state:", caller.verificationState);
```
