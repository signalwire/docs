List verified caller IDs in the project.

## **Parameters**

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of verified caller objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.verifiedCallers.list();
for (const caller of result.data ?? []) {
    console.log(caller.phoneNumber, caller.verificationState);
}
```
