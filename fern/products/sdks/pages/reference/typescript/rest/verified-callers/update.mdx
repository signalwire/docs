Update a verified caller ID. Uses PUT.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="name" type="string" required={true} toc={true}>
  The friendly name for the caller ID. Required for PUT replacement.
</ParamField>

## **Returns**

`Promise<any>` -- The updated caller ID object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.verifiedCallers.update("caller-id", { name: "Main Office" });
console.log(`Updated: ${updated.id}`);
```
