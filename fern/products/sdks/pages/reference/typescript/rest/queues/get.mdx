Retrieve a specific queue.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/queues/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/queues/{id}" />

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const queue = await client.queues.get("queue-id");
console.log(queue.name, queue["current_size"]);
```
