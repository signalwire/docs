Update a queue. Uses PUT.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="body" type="any" required={true} toc={true}>
  Fields to update (e.g., `name`, `max_size`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated queue object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.queues.update("queue-id", { name: "priority-support" });
console.log(`Updated: ${updated.id}`);
```
