Create a new queue.

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Queue name.
</ParamField>

<ParamField path="max_size" type="number" toc={true}>
  Maximum number of members allowed in the queue.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created queue object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const queue = await client.queues.create({ name: "support", max_size: 50 });
console.log("Queue ID:", queue.id);
```
