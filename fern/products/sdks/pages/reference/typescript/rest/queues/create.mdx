Create a new queue.

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Queue name.
</ParamField>

<ParamField path="body" type="any" required={true} toc={true}>
  Additional queue configuration fields (e.g., `max_size`, `fifo`).
</ParamField>

## **Returns**

`Promise<any>` -- The newly created queue object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const queue = await client.queues.create({ name: "support", maxSize: 50 });
console.log("Queue ID:", queue.id);
```
