Retrieve a specific member from a queue.

## **Parameters**

<ParamField path="queueId" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="memberId" type="string" required={true} toc={true}>
  The member resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The member object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const member = await client.queues.getMember("queue-id", "member-id");
console.log(member["call_id"], member["position"]);
```
