List members currently in a queue.

## **Parameters**

<ParamField path="queueId" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of member objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const members = await client.queues.listMembers("queue-id");
for (const m of members.data ?? []) {
    console.log(m["call_id"], m["position"]);
}
```
