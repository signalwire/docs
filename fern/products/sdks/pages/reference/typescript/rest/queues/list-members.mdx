List members currently in a queue.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/queues/{queue_id}/members" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/queues/{queue_id}/members" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const members = await client.queues.listMembers("queue-id");
for (const m of members.data ?? []) {
    console.log(m["call_id"], m["position"]);
}
```
