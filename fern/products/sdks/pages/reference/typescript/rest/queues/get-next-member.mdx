Retrieve the next member in the queue (the member at the front).

## **Parameters**

<ParamField path="queueId" type="string" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- The next member object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

next_member = client.queues.get_next_member("queue-id")
console.log("Next caller:", next_member["call_id"])
```
