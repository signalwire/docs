List short codes in the project.

## **Parameters**

<ParamField path="params" type="Record&lt;string, string | number | boolean | undefined&gt;" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of short code objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.shortCodes.list();
for (const code of result.data ?? []) {
    console.log(code.number, code.id);
}
```
