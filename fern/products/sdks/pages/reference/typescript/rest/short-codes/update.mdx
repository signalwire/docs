Update a short code's configuration. Uses PUT.

## **Parameters**

<ParamField path="shortCodeId" type="string" required={true} toc={true}>
  The short code resource ID.
</ParamField>

<ParamField path="body" type="any" required={true} toc={true}>
  Fields to update (e.g., `message_handler`, `message_request_url`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated short code object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.shortCodes.update("short-code-id", {
    messageRequestUrl: "https://example.com/sms",
});
```
