Retrieve a specific short code.

## **Parameters**

<ParamField path="shortCodeId" type="string" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`dict` -- The short code object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

code = client.short_codes.get("short-code-id")
console.log(code["short_code"])
```
