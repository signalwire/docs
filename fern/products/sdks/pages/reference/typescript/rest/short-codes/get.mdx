Retrieve a specific short code.

## **Parameters**

<ParamField path="shortCodeId" type="string" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The short code object.

## **Example**

```typescript {9-10}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const code = await client.shortCodes.get("short-code-id");
console.log(code.shortCode);
```
