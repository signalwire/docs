## **Parameters**

<ParamField path="subscriberId" type="string" required={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpointId" type="string" required={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.fabric.subscribers.getSipEndpoint("subscriber-id", "endpoint-id");
console.log(result);
```
