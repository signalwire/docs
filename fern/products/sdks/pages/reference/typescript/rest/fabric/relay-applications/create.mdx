Create a new RELAY application resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the RELAY application.
</ParamField>

<ParamField path="topic" type="string" required={true} toc={true}>
  The RELAY topic this application listens on.
</ParamField>

## **Returns**

`Promise<any>` -- The created RELAY application resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.relayApplications.create({
  name: "my-item",
});
console.log(`Created: ${result.id}`);
```
