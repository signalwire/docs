Create a new AI agent resource.

## **Parameters**

<Markdown src="/snippets/sdks/rest/fabric/ai-agents/_create-params.mdx" />

## **Returns**

`Promise<any>` -- The created AI agent resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.aiAgents.create({
  name: "support-bot",
  agent_id: "support-bot-001",
  prompt: { text: "You are a helpful support agent." },
});
console.log(`Created: ${result.id}`);
```
