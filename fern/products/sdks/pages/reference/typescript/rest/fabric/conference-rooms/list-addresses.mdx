List addresses associated with a conference room resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/conference_room/{id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/conference_room/{id}/addresses" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const addresses = await client.fabric.conferenceRooms.listAddresses("room-id");
for (const addr of addresses.data ?? []) {
  console.log(`${addr.display_name}: ${addr.id}`);
}
```
