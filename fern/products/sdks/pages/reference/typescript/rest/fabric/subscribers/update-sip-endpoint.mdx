Partially update a SIP endpoint for a subscriber. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="subscriberId" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpointId" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

<ParamField path="username" type="string" toc={true}>
  SIP username for the endpoint.
</ParamField>

<ParamField path="password" type="string" toc={true}>
  SIP password for the endpoint.
</ParamField>

<ParamField path="caller_id" type="string" toc={true}>
  Caller ID for outbound calls from this endpoint.
</ParamField>

<ParamField path="send_as" type="string" toc={true}>
  The identity to present when making outbound calls.
</ParamField>

<ParamField path="ciphers" type="string[]" toc={true}>
  List of allowed cipher suites for the endpoint.
</ParamField>

<ParamField path="codecs" type="string[]" toc={true}>
  List of allowed audio codecs for the endpoint.
</ParamField>

<ParamField path="encryption" type="string" toc={true}>
  Encryption mode for the SIP endpoint. `"required"`, `"optional"`, or `"default"`.
</ParamField>

## **Returns**

`Promise<any>` -- The updated SIP endpoint.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.fabric.subscribers.updateSipEndpoint(
  "subscriber-id",
  "endpoint-id",
  { username: "new-username" }
);
console.log("Updated:", updated.username);
```
