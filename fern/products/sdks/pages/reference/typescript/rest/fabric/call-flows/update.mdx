Replace a call flow resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<Markdown src="/snippets/sdks/rest/fabric/call-flows/_update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated call flow resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.fabric.callFlows.update("flow-id", {
  title: "Updated IVR Flow",
});
console.log(`Updated: ${updated.display_name}`);
```
