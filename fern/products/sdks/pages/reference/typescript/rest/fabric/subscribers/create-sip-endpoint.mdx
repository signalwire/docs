Create a new SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriberId" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="username" type="string" required={true} toc={true}>
  SIP username for the endpoint.
</ParamField>

<ParamField path="password" type="string" required={true} toc={true}>
  SIP password for the endpoint.
</ParamField>

<ParamField path="encryption" type="string" default="default" toc={true}>
  Encryption mode for the SIP endpoint. `"required"`, `"optional"`, or `"default"`.
</ParamField>

<ParamField path="caller_id" type="string" toc={true}>
  Caller ID for outbound calls from this endpoint.
</ParamField>

<ParamField path="send_as" type="string" toc={true}>
  The identity to present when making outbound calls.
</ParamField>

<ParamField path="ciphers" type="string[]" toc={true}>
  List of allowed cipher suites for the endpoint.
</ParamField>

<ParamField path="codecs" type="string[]" toc={true}>
  List of allowed audio codecs for the endpoint.
</ParamField>

## **Returns**

`Promise<any>` -- The created SIP endpoint.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.subscribers.createSipEndpoint("subscriber-id", {
  username: "alice-sip",
  password: "s3cret"
});
console.log("Created SIP endpoint:", result.id);
```
