Replace a cXML application resource. Uses PUT, so the full resource body must be provided.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

Additional properties are the fields for the replacement resource body.

## **Returns**

`Promise<any>` -- The updated cXML application resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const updated = await client.fabric.cxmlApplications.update("app-id", {
  display_name: "updated-cxml-app",
});
console.log("Updated:", updated.display_name);
```
