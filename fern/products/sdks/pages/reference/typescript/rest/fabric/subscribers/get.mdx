Retrieve a single subscriber resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`Promise<any>` -- The subscriber resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const subscriber = await client.fabric.subscribers.get("subscriber-id");
console.log("Subscriber:", subscriber.display_name, "ID:", subscriber.id);
```
