Replace a FreeSWITCH connector resource. Uses PUT for full replacement.

<EndpointSchemaSnippet endpoint="PUT /api/fabric/resources/freeswitch_connectors/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/fabric/resources/freeswitch_connectors/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const updated = await client.fabric.freeswitchConnectors.update("connector-id", { name: "updated-name" });
console.log(`Updated: ${updated.display_name}`);
```
