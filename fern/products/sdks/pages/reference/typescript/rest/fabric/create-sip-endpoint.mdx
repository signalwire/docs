## **Parameters**

<ParamField path="subscriberId" type="string" required={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="body" type="any" required={true}>
  The request body containing the resource data.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.fabric.subscribers.createSipEndpoint("subscriber-id", {
  username: "sip-user",
  password: "sip-password"
});
console.log(result);
```
