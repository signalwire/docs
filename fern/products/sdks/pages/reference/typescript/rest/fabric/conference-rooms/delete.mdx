Delete a conference room resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/conference_rooms/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.fabric.conferenceRooms.delete("room-id");
console.log("Conference Room deleted");
```
