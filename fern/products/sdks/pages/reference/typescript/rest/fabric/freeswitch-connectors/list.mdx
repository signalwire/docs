List FreeSWITCH connector resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing FreeSWITCH connector resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.fabric.freeswitchConnectors.list();
for (const connector of response.data ?? []) {
  console.log(`${connector.display_name}: ${connector.id}`);
}
```
