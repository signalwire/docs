Replace a SIP endpoint resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Promise<any>` -- The updated SIP endpoint resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const updated = await client.fabric.sipEndpoints.update("endpoint-id", { username: "updated-user" });
console.log(`Updated: ${updated.name}`);
```
