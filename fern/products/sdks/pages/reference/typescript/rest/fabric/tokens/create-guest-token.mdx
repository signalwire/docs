Create a guest authentication token.

## **Parameters**

<ParamField path="allowed_addresses" type="string[]" required={true} toc={true}>
  List of address UUIDs the guest is allowed to access (max 10).
</ParamField>

## **Returns**

`Promise<any>` -- The created guest token.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const token = await client.fabric.tokens.createGuestToken({ allowed_addresses: ["address-uuid"] });
console.log(`Guest token: ${token.token}`);
```
