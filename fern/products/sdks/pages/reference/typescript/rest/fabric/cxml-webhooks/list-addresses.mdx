List addresses associated with a cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`Promise<any>` -- List of addresses assigned to this resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const addresses = await client.fabric.cxmlWebhooks.listAddresses("webhook-id");
for (const addr of addresses.data ?? []) {
  console.log(`${addr.display_name}: ${addr.id}`);
}
```
