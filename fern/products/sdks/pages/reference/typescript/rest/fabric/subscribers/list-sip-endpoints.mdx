List SIP endpoints associated with a subscriber.

## **Parameters**

<ParamField path="subscriberId" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing SIP endpoints for the subscriber.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const response = await client.fabric.subscribers.listSipEndpoints("subscriber-id");
for (const ep of response.data ?? []) {
  console.log(`${ep.username}: ${ep.id}`);
}
```
