Delete a FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/freeswitch_connectors/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.freeswitchConnectors.delete("connector-id");
console.log("Freeswitch Connector deleted");
```
