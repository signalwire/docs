Delete a SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriberId" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpointId" type="string" required={true} toc={true}>
  The unique identifier of the SIP endpoint to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success (HTTP 204).

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.fabric.subscribers.deleteSipEndpoint("subscriber-id", "endpoint-id");
console.log("SIP endpoint deleted");
```
