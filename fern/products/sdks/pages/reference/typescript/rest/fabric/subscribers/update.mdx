Replace a subscriber resource. Uses PUT, so the full resource body must be provided.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<Markdown src="/snippets/sdks/rest/fabric/subscribers/_create-update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated subscriber resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.fabric.subscribers.update("subscriber-id", {
  name: "bob"
});
console.log("Updated:", updated.display_name);
```
