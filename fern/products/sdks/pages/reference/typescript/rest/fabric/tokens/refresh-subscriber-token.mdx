Refresh an existing subscriber authentication token.

## **Parameters**

<ParamField path="refresh_token" type="string" required={true} toc={true}>
  The refresh token from the original subscriber token response.
</ParamField>

## **Returns**

`Promise<any>` -- The refreshed subscriber token.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const refreshed = await client.fabric.tokens.refreshSubscriberToken({ refresh_token: "existing-refresh-token" });
console.log(`Refreshed token: ${refreshed.token}`);
```
