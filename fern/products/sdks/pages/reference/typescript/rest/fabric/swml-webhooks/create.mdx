Create a new SWML webhook resource.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/swml_webhooks" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/swml_webhooks" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.swmlWebhooks.create({
  name: "my-webhook",
  primary_request_url: "https://example.com/swml",
});
console.log("Created:", result.id);
```
