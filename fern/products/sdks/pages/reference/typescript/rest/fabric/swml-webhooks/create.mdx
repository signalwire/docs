Create a new SWML webhook resource.

## **Parameters**

<ParamField path="primary_request_url" type="string" required={true} toc={true}>
  The primary URL for the SWML webhook.
</ParamField>

<ParamField path="name" type="string" toc={true}>
  Display name of the SWML webhook.
</ParamField>

## **Returns**

`Promise<any>` -- The created SWML webhook resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.swmlWebhooks.create({
  name: "my-webhook",
  primary_request_url: "https://example.com/swml",
});
console.log("Created:", result.id);
```
