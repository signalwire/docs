Create an invite token for a subscriber.

## **Parameters**

<ParamField path="address_id" type="string" required={true} toc={true}>
  The UUID of the address to create the invite for.
</ParamField>

## **Returns**

`Promise<any>` -- The created invite token.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const token = await client.fabric.tokens.createInviteToken({ address_id: "address-uuid" });
console.log(`Invite token: ${token.token}`);
```
