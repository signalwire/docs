Replace a RELAY application resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`Promise<any>` -- The updated RELAY application resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const updated = await client.fabric.relayApplications.update("resource-id", { name: "updated-name" });
console.log(`Updated: ${updated.display_name}`);
```
