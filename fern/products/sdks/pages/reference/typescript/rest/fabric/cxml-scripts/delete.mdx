Delete a cXML script resource.

<EndpointSchemaSnippet endpoint="DELETE /api/fabric/resources/cxml_scripts/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.cxmlScripts.delete("resource-id");
console.log("Cxml Script deleted");
```
