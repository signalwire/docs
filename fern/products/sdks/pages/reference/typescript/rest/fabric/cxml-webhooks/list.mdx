List cXML webhook resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/cxml_webhooks" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/cxml_webhooks" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.fabric.cxmlWebhooks.list();
for (const webhook of response.data ?? []) {
  console.log(`${webhook.display_name}: ${webhook.id}`);
}
```
