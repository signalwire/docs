Partially update a SIP gateway resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

Additional properties are the fields to update.

## **Returns**

`Promise<any>` -- The updated SIP gateway resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const updated = await client.fabric.sipGateways.update("gateway-id", {
  name: "updated-name",
});
console.log("Updated:", updated.display_name);
```
