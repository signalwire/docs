List SIP endpoint resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing SIP endpoint resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.fabric.sipEndpoints.list();
for (const endpoint of response.data ?? []) {
  console.log(`${endpoint.name}: ${endpoint.id}`);
}
```
