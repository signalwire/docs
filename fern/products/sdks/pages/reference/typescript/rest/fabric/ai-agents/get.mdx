Retrieve a single AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`Promise<any>` -- The AI agent resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const agent = await client.fabric.aiAgents.get("agent-id");
console.log(`AI Agent: ${agent.display_name}, ID: ${agent.id}`);
```
