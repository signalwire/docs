Create an embed authentication token.

## **Parameters**

<ParamField path="token" type="string" required={true} toc={true}>
  The Click-to-Call token.
</ParamField>

## **Returns**

`Promise<any>` -- The created embed token.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const token = await client.fabric.tokens.createEmbedToken({ token: "click-to-call-token" });
console.log(`Embed token: ${token.token}`);
```
