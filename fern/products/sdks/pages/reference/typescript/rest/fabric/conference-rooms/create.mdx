Create a new conference room resource.

## **Parameters**

<Markdown src="/snippets/sdks/rest/fabric/conference-rooms/_create-params.mdx" />

## **Returns**

`Promise<any>` -- The created conference room resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.conferenceRooms.create({
  name: "my-room",
  enable_room_previews: true,
});
console.log(`Created: ${result.display_name}`);
```
