Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`Promise<any>` -- The SIP gateway resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const gateway = await client.fabric.sipGateways.get("gateway-id");
console.log("Sip Gateway:", gateway.display_name, "ID:", gateway.id);
```
