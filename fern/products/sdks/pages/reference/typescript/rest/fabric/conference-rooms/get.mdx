Retrieve a single conference room resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`Promise<any>` -- The conference room resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const room = await client.fabric.conferenceRooms.get("room-id");
console.log(`Conference Room: ${room.display_name}, ID: ${room.id}`);
```
