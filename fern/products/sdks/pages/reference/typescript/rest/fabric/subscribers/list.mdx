List subscriber resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing subscriber resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const response = await client.fabric.subscribers.list();
for (const subscriber of response.data ?? []) {
  console.log(`${subscriber.display_name}: ${subscriber.id}`);
}
```
