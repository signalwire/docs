Create a new FreeSWITCH connector resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the FreeSWITCH connector.
</ParamField>

<ParamField path="token" type="string" required={true} toc={true}>
  The connector authentication token (UUID format).
</ParamField>

## **Returns**

`Promise<any>` -- The created FreeSWITCH connector resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.freeswitchConnectors.create({
  name: "my-connector",
});
console.log(`Created: ${result.id}`);
```
