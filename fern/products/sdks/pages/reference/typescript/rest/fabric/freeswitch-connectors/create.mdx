Create a new FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="POST /api/fabric/resources/freeswitch_connectors" />

## **Response Example**

<EndpointResponseSnippet endpoint="POST /api/fabric/resources/freeswitch_connectors" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.freeswitchConnectors.create({
  name: "my-connector",
});
console.log(`Created: ${result.id}`);
```
