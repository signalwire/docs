Create a new call flow resource.

## **Parameters**

<Markdown src="/snippets/sdks/rest/fabric/call-flows/_create-params.mdx" />

## **Returns**

`Promise<any>` -- The created call flow resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.callFlows.create({
  title: "Main IVR Flow",
});
console.log(`Created: ${result.id}`);
```
