Create a new SIP endpoint resource.

## **Parameters**

<ParamField path="username" type="string" required={true} toc={true}>
  SIP username for the endpoint.
</ParamField>

<ParamField path="caller_id" type="string" required={true} toc={true}>
  Caller ID for the endpoint.
</ParamField>

<ParamField path="send_as" type="string" required={true} toc={true}>
  The send-as identity for the endpoint.
</ParamField>

<ParamField path="encryption" type="string" required={true} toc={true}>
  Encryption mode for the SIP endpoint.
</ParamField>

<ParamField path="ciphers" type="string[]" required={true} toc={true}>
  Supported cipher suites.
</ParamField>

<ParamField path="codecs" type="string[]" required={true} toc={true}>
  Supported audio codecs.
</ParamField>

<ParamField path="call_handler" type="string" required={true} toc={true}>
  The call handler type.
</ParamField>

<ParamField path="calling_handler_resource_id" type="string" required={true} toc={true}>
  The resource ID of the calling handler.
</ParamField>

## **Returns**

`Promise<any>` -- The created SIP endpoint resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.sipEndpoints.create({
  username: "my-endpoint",
  caller_id: "+15551234567",
  send_as: "+15551234567",
  encryption: "required",
  ciphers: ["AEAD_AES_256_GCM_8"],
  codecs: ["OPUS"],
  call_handler: "relay_app",
  calling_handler_resource_id: "resource-id",
});
console.log(`Created: ${result.id}`);
```
