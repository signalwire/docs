Read-only access to fabric addresses. Addresses represent the endpoints (phone numbers,
SIP URIs, domains) that route traffic to fabric resources.

Access via `client.fabric.addresses` on a [`RestClient`][restclient] instance.

### list

List all fabric addresses.

## **Parameters**

<ParamField path="pageSize" type="number" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="pageToken" type="string" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing fabric addresses.

### get

Retrieve a single fabric address.

## **Parameters**

<ParamField path="addressId" type="string" required={true} toc={true}>
  The unique identifier of the address.
</ParamField>

## **Returns**

`dict` -- The fabric address resource.

## **Example**

```typescript
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

# List all addresses
addresses = client.fabric.addresses.list()
for addr in addresses.get("data", []):
    console.log(`${addr['display_name']}: ${addr['type']}`)

# Get a specific address
address = client.fabric.addresses.get("address-id")
console.log(`Address: ${address['display_name']}`)
```
