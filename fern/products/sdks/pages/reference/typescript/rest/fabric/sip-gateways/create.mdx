Create a new SIP gateway resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the SIP gateway.
</ParamField>

<ParamField path="uri" type="string" required={true} toc={true}>
  The SIP URI of the gateway.
</ParamField>

<ParamField path="encryption" type="string" required={true} toc={true}>
  Encryption mode for the gateway.
</ParamField>

<ParamField path="ciphers" type="string[]" required={true} toc={true}>
  Supported cipher suites.
</ParamField>

<ParamField path="codecs" type="string[]" required={true} toc={true}>
  Supported audio codecs.
</ParamField>

## **Returns**

`Promise<any>` -- The created SIP gateway resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.sipGateways.create({
  name: "my-gateway",
  uri: "sip:gw.example.com",
  encryption: "required",
  ciphers: ["AEAD_AES_256_GCM_8"],
  codecs: ["OPUS", "PCMU"],
});
console.log("Created:", result.id);
```
