Create a new cXML script resource.

## **Parameters**

<ParamField path="display_name" type="string" required={true} toc={true}>
  Display name of the cXML script.
</ParamField>

<ParamField path="contents" type="string" required={true} toc={true}>
  The cXML script contents.
</ParamField>

## **Returns**

`Promise<any>` -- The created cXML script resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.cxmlScripts.create({
  display_name: "my-script",
  contents: "<Response><Say>Hello</Say></Response>",
});
console.log(`Created: ${result.id}`);
```
