Create a new SWML script resource.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Display name of the SWML script.
</ParamField>

<ParamField path="contents" type="string" required={true} toc={true}>
  The SWML script contents.
</ParamField>

## **Returns**

`Promise<any>` -- The created SWML script resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.fabric.swmlScripts.create({
  name: "my-item",
});
console.log(`Created: ${result.id}`);
```
