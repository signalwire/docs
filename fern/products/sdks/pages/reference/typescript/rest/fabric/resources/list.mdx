List all Fabric resources across all resource types.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing Fabric resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const response = await client.fabric.resources.list();
for (const resource of response.data ?? []) {
  console.log(`${resource.type}: ${resource.name} (${resource.id})`);
}
```
