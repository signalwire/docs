Deploy a specific version of a call flow resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<ParamField path="call_flow_version_id" type="string" toc={true}>
  The version ID to deploy. Provide either this or `document_version`.
</ParamField>

<ParamField path="document_version" type="number" toc={true}>
  The document version number to deploy. Provide either this or `call_flow_version_id`.
</ParamField>

## **Returns**

`Promise<any>` -- The deployed version details.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.callFlows.deployVersion("call-flow-id", {
  call_flow_version_id: "version-id",
});
console.log(`Deployed version: ${result.id}`);
```
