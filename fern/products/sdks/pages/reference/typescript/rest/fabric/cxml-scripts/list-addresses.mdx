List addresses associated with a cXML script resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/cxml_scripts/{id}/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/cxml_scripts/{id}/addresses" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const addresses = await client.fabric.cxmlScripts.listAddresses("resource-id");
for (const addr of addresses.data ?? []) {
  console.log(`${addr.display_name}: ${addr.id}`);
}
```
