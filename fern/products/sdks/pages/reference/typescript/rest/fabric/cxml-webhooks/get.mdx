Retrieve a single cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`Promise<any>` -- The cXML webhook resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const webhook = await client.fabric.cxmlWebhooks.get("webhook-id");
console.log("Cxml Webhook:", webhook.display_name, "ID:", webhook.id);
```
