List addresses associated with a call flow resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`Promise<any>` -- List of addresses assigned to this call flow.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const addresses = await client.fabric.callFlows.listAddresses("flow-id");
for (const addr of addresses.data ?? []) {
  console.log(`${addr.display_name}: ${addr.id}`);
}
```
