Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`Promise<any>` -- The Fabric resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const resource = await client.fabric.resources.get("resource-id");
console.log(`${resource.type}: ${resource.name}, ID: ${resource.id}`);
```
