List SIP gateway resources in the project.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/sip_gateways" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/sip_gateways" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.fabric.sipGateways.list();
for (const gateway of response.data ?? []) {
  console.log(`${gateway.display_name}: ${gateway.id}`);
}
```
