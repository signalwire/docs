List SIP gateway resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All options are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing SIP gateway resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.fabric.sipGateways.list();
for (const gateway of response.data ?? []) {
  console.log(`${gateway.display_name}: ${gateway.id}`);
}
```
