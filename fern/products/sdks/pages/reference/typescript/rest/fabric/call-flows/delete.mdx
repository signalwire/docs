Delete a call flow resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the call flow to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success (HTTP 204).

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.fabric.callFlows.delete("flow-id");
console.log("Call Flow deleted");
```
