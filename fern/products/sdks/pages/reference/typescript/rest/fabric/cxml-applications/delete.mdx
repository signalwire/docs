Delete a cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML application to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success (HTTP 204).

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.fabric.cxmlApplications.delete("app-id");
console.log("cXML application deleted");
```
