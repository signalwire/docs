## **Parameters**

<ParamField path="resourceId" type="string" required={true}>
  The unique identifier of the resource.
</ParamField>

<ParamField path="params" type="QueryParams">
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.fabric.resources.listAddresses("resource-id");
console.log(result);
```
