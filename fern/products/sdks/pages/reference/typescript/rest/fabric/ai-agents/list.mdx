List AI agent resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All additional parameters are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing AI agent resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.fabric.aiAgents.list();
for (const agent of response.data ?? []) {
  console.log(`${agent.display_name}: ${agent.id}`);
}
```
