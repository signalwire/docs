Replace a conference room resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

<Markdown src="/snippets/sdks/rest/fabric/conference-rooms/_update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated conference room resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.fabric.conferenceRooms.update("room-id", {
  name: "updated-name",
  enable_room_previews: true,
  sync_audio_video: true,
});
console.log(`Updated: ${updated.display_name}`);
```
