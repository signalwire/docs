Retrieve a single FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`Promise<any>` -- The FreeSWITCH connector resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const connector = await client.fabric.freeswitchConnectors.get("connector-id");
console.log(`Freeswitch Connector: ${connector.display_name}, ID: ${connector.id}`);
```
