Retrieve a single FreeSWITCH connector resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/freeswitch_connectors/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/freeswitch_connectors/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const connector = await client.fabric.freeswitchConnectors.get("connector-id");
console.log(`Freeswitch Connector: ${connector.display_name}, ID: ${connector.id}`);
```
