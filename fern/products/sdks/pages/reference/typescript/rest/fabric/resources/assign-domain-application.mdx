Assign a domain application to a Fabric resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

<ParamField path="domain_application_id" type="string" required={true} toc={true}>
  The UUID of the domain application to assign.
</ParamField>

## **Returns**

`Promise<any>` -- The assigned domain application.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.resources.assignDomainApplication("resource-id", {
  domain_application_id: "app-uuid"
});
console.log(`Domain application assigned: ${result.id}`);
```
