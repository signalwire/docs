Retrieve a single SWML script resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/swml_scripts/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/swml_scripts/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const item = await client.fabric.swmlScripts.get("resource-id");
console.log(`Swml Script: ${item.display_name}, ID: ${item.id}`);
```
