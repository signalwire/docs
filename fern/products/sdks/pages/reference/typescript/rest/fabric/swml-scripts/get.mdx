Retrieve a single SWML script resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

## **Returns**

`Promise<any>` -- The SWML script resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const item = await client.fabric.swmlScripts.get("resource-id");
console.log(`Swml Script: ${item.display_name}, ID: ${item.id}`);
```
