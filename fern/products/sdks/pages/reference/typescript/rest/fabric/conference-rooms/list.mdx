List conference room resources in the project.

## **Parameters**

No query parameters are defined for this endpoint. All keyword arguments are
passed through to the API.

## **Returns**

`Promise<any>` -- Paginated response containing conference room resources.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const response = await client.fabric.conferenceRooms.list();
for (const room of response.data ?? []) {
  console.log(`${room.display_name}: ${room.id}`);
}
```
