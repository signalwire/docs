Retrieve a single RELAY application resource.

<EndpointSchemaSnippet endpoint="GET /api/fabric/resources/relay_applications/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/fabric/resources/relay_applications/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const item = await client.fabric.relayApplications.get("resource-id");
console.log(`Relay Application: ${item.display_name}, ID: ${item.id}`);
```
