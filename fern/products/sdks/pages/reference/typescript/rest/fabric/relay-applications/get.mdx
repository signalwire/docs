Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`Promise<any>` -- The RELAY application resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const item = await client.fabric.relayApplications.get("resource-id");
console.log(`Relay Application: ${item.display_name}, ID: ${item.id}`);
```
