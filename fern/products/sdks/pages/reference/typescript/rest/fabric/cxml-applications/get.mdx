Retrieve a single cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

## **Returns**

`Promise<any>` -- The cXML application resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const app = await client.fabric.cxmlApplications.get("app-id");
console.log("cXML Application:", app.display_name, "ID:", app.id);
```
