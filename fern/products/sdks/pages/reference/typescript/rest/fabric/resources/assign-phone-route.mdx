Assign a phone route to a Fabric resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

<ParamField path="phone_route_id" type="string" required={true} toc={true}>
  The UUID of the phone route to assign.
</ParamField>

<ParamField path="handler" type="string" required={true} toc={true}>
  The handler type: `"calling"` or `"messaging"`.
</ParamField>

## **Returns**

`Promise<any>` -- The assigned phone route.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.fabric.resources.assignPhoneRoute("resource-id", {
  phone_route_id: "route-uuid",
  handler: "calling"
});
console.log(`Phone route assigned: ${result.id}`);
```
