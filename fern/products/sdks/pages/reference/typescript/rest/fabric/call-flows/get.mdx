Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`Promise<any>` -- The call flow resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const flow = await client.fabric.callFlows.get("flow-id");
console.log(`Call Flow: ${flow.display_name}, ID: ${flow.id}`);
```
