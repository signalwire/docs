Verify an MFA code submitted by the user.

## **Parameters**

<ParamField path="requestId" type="string" required={true} toc={true}>
  The MFA request ID returned from `sms()` or `call()`.
</ParamField>

<ParamField path="token" type="string" required={true} toc={true}>
  The verification code entered by the user.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`Promise<any>` -- Verification result indicating whether the code is valid.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.mfa.verify("your-mfa-request-id", { token: "123456" });
if (result.success) {
  console.log("Verification successful");
} else {
  console.log("Invalid code");
}
```
