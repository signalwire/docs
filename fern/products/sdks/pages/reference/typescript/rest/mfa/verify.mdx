Verify an MFA code submitted by the user.

## **Parameters**

<ParamField path="requestId" type="string" required={true} toc={true}>
  The MFA request ID returned from `sms()` or `call()`.
</ParamField>

<ParamField path="body" type="any" required={true} toc={true}>
  Request body containing the verification code. Must include `token` (the code entered by the user).
</ParamField>

## **Returns**

`Promise<any>` -- Verification result indicating whether the code is valid.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.mfa.verify("your-mfa-request-id", { token: "123456" });
if (result.success) {
  console.log("Verification successful");
} else {
  console.log("Invalid code");
}
```
