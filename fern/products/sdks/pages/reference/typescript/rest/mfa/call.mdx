Send an MFA verification code via voice call.

## **Parameters**

<ParamField path="to" type="string" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="string" required={true} toc={true}>
  The caller phone number in E.164 format.
  Since `from` is a Python reserved keyword, pass it via dict unpacking (see example).
</ParamField>

<ParamField path="message" type="string" required={true} toc={true}>
  The message to speak. Include `{code}` as a placeholder.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional parameters (e.g., `token_length`, `max_attempts`, `valid_for`).
</ParamField>

## **Returns**

`Promise<any>` -- Response containing the `id` of the MFA request.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.mfa.call(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your verification code is {code}",
})
request_id = result.id
```
