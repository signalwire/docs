Send an MFA verification code via SMS.

## **Parameters**

<ParamField path="body" type="any" required={true} toc={true}>
  Request body. Common fields include `to` (E.164 recipient), `from` (E.164 sender),
  `message` (body with `{code}` placeholder), `token_length`, `max_attempts`, `valid_for`.
</ParamField>

## **Returns**

`Promise<any>` -- Response containing the `id` of the MFA request (used in `verify()`).

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.mfa.sms({
    to: "+15559876543",
    from: "+15551234567",
    message: "Your code is: {code}",
});
const requestId = result.id;
console.log("MFA request:", requestId);
```
