Send an MFA verification code via SMS.

## **Parameters**

<ParamField path="to" type="string" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="string" required={true} toc={true}>
  The sender phone number in E.164 format. Must be a number in your project.
  Since `from` is a Python reserved keyword, pass it via dict unpacking (see example).
</ParamField>

<ParamField path="message" type="string" required={true} toc={true}>
  The message body. Include `{code}` as a placeholder for the generated code
  (e.g., `"Your verification code is: {code}"`).
</ParamField>

<ParamField path="tokenLength" type="number" toc={true}>
  Length of the generated code.
</ParamField>

<ParamField path="maxAttempts" type="number" toc={true}>
  Maximum number of verification attempts allowed.
</ParamField>

<ParamField path="validFor" type="number" toc={true}>
  How long the code remains valid, in seconds.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`Promise<any>` -- Response containing the `id` of the MFA request (used in `verify()`).

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.mfa.sms(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your code is: {code}",
})
request_id = result.id
console.log("MFA request:", request_id);
```
