Retrieve a specific recording.

## **Parameters**

<ParamField path="recordingId" type="string" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The recording object, including download URL and metadata.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const rec = await client.recordings.get("recording-id");
console.log(rec["duration"], rec["url"]);
```
