Delete a recording.

## **Parameters**

<ParamField path="recordingId" type="string" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.recordings.delete("recording-id")
```
