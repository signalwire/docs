Delete a recording.

## **Parameters**

<ParamField path="recordingId" type="string" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.recordings.delete("recording-id");
```
