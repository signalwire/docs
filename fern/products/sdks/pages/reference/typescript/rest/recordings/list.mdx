List recordings in the project.

## **Parameters**

<ParamField path="queryParams" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of recording objects.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.recordings.list({ pageSize: 20 });
for (const rec of result.data ?? []) {
    console.log(rec.id, rec["duration"], rec["state"]);
}
```
