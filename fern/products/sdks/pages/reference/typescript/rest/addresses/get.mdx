Retrieve a specific address.

## **Parameters**

<ParamField path="addressId" type="string" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`dict` -- The address object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

addr = client.addresses.get("address-id")
console.log(addr["customer_name"], addr["street"])
```
