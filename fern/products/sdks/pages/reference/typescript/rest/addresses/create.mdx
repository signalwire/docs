Create a new regulatory address.

## **Parameters**

<ParamField path="customerName" type="string" toc={true}>
  The customer or business name for the address.
</ParamField>

<ParamField path="street" type="string" toc={true}>
  Street address.
</ParamField>

<ParamField path="city" type="string" toc={true}>
  City name.
</ParamField>

<ParamField path="state" type="string" toc={true}>
  State or province.
</ParamField>

<ParamField path="postalCode" type="string" toc={true}>
  Postal or ZIP code.
</ParamField>

<ParamField path="country" type="string" toc={true}>
  ISO country code (e.g., `"US"`).
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional address fields as needed by the API.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created address object.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const addr = await client.addresses.create(;
    customerName: "Acme Corp",
    street: "123 Main St",
    city: "Austin",
    state: "TX",
    postalCode: "78701",
    country: "US",
)
console.log("Created address:", addr.id);
```
