List addresses in the project.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/addresses" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/addresses" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.addresses.list();
for (const addr of result.data ?? []) {
    console.log(addr.friendlyName, addr.street);
}
```
