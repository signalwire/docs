List addresses in the project.

## **Parameters**

<ParamField path="queryParams" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of address objects.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.addresses.list();
for (const addr of result.data ?? []) {
    console.log(addr.friendlyName, addr.street);
}
```
