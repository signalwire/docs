Delete an address from the project.

## **Parameters**

<ParamField path="addressId" type="string" required={true} toc={true}>
  The address resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.addresses.delete("address-id");
```
