Retrieve a specific membership.

<Note>
This method uses the top-level membership endpoint and takes a membership ID
directly, not a group ID.
</Note>

## **Parameters**

<ParamField path="membershipId" type="string" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The membership object.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const membership = await client.numberGroups.getMembership("membership-id");
console.log(membership["phone_number_id"]);
```
