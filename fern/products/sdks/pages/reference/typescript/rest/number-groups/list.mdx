List number groups in the project.

## **Parameters**

<ParamField path="queryParams" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of group objects.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.numberGroups.list();
for (const group of result.data ?? []) {
    console.log(group.name, group.id);
}
```
