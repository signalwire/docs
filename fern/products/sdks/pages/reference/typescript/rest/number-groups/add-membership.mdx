Add a phone number to a group.

## **Parameters**

<ParamField path="groupId" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="phoneNumberId" type="string" toc={true}>
  The phone number resource ID to add to the group.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional membership fields.
</ParamField>

## **Returns**

`dict` -- The newly created membership object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

membership = client.number_groups.add_membership(
    "group-id",
    phone_number_id="phone-number-id",
)
console.log("Membership ID:", membership["id"])
```
