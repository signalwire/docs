Add a phone number to a group.

## **Parameters**

<ParamField path="groupId" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="phoneNumberId" type="string" toc={true}>
  The phone number resource ID to add to the group.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional membership fields.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created membership object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const membership = await client.numberGroups.addMembership("group-id", {
    phoneNumberId: "phone-number-id",
});
console.log("Membership ID:", membership.id);
```
