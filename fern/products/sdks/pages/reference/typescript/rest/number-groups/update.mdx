Update a number group. Uses PUT to replace fields.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="name" type="string" required={true} toc={true}>
  Group name. Required for PUT replacement.
</ParamField>

<ParamField path="sticky_sender" type="boolean" toc={true}>
  Whether to use the same sender number for replies within this group.
</ParamField>

## **Returns**

`Promise<any>` -- The updated group object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.numberGroups.update("group-id", { name: "Support Numbers" });
console.log(updated.name);
```
