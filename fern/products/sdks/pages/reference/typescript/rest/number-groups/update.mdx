Update a number group. Uses PUT to replace fields.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated group object.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.numberGroups.update("group-id", name: "Support Numbers" });
console.log(updated.name);
```
