Update a number group. Uses PUT to replace fields.

<EndpointSchemaSnippet endpoint="PUT /api/relay/rest/number_groups/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="PUT /api/relay/rest/number_groups/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.numberGroups.update("group-id", { name: "Support Numbers" });
console.log(updated.name);
```
