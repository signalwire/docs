List phone numbers that belong to a group.

## **Parameters**

<ParamField path="groupId" type="string" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of membership objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const memberships = await client.numberGroups.listMemberships("group-id");
for (const m of memberships.data ?? []) {
    console.log(m["phone_number_id"], m.id);
}
```
