Remove a phone number from a group.

<Note>
Like `getMembership()`, this method takes a membership ID directly.
</Note>

## **Parameters**

<ParamField path="membershipId" type="string" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.number_groups.delete_membership("membership-id")
```
