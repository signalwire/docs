Remove a phone number from a group.

<Note>
Like `getMembership()`, this method takes a membership ID directly.
</Note>

## **Parameters**

<ParamField path="membershipId" type="string" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.numberGroups.deleteMembership("membership-id");
```
