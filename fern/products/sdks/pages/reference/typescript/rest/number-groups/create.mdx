Create a new number group.

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Group name.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional group configuration.
</ParamField>

## **Returns**

`dict` -- The newly created group object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

group = client.number_groups.create(name="Sales Numbers")
console.log("Group ID:", group["id"])
```
