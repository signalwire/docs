Create a new number group.

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Group name.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional group configuration.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created group object.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const group = await client.numberGroups.create({ name: "Sales Numbers" });
console.log("Group ID:", group.id);
```
