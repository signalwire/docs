Create a new number group.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Group name.
</ParamField>

<ParamField path="sticky_sender" type="boolean" default="false" toc={true}>
  Whether to use the same sender number for replies within this group.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created group object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const group = await client.numberGroups.create({ name: "Sales Numbers" });
console.log("Group ID:", group.id);
```
