List documents in the project.

## **Parameters**

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of document objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.datasphere.documents.list();
for (const doc of result.data ?? []) {
    console.log(doc.name, doc.id);
}
```
