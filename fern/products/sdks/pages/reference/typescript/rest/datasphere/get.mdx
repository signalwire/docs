Retrieve a specific document by ID.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The document object.

## **Example**

```typescript {9-10}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const doc = await client.datasphere.documents.get("document-id");
console.log(doc.name, doc.id);
```
