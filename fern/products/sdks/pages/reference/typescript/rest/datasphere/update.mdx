Update an existing document.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Fields to update (e.g., `name`, `content`, `metadata`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated document object.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.datasphere.documents.update(;
    "document-id",
    name: "Updated FAQ",
)
console.log(updated.name);
```
