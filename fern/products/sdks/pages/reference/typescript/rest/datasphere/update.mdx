Update an existing document.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<Markdown src="/snippets/sdks/rest/datasphere/documents/_update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated document object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.datasphere.documents.update("document-id", {
    tags: ["support", "updated"],
});
console.log(updated.tags);
```
