Delete a specific chunk from a document.

## **Parameters**

<ParamField path="documentId" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunkId" type="string" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty dict on success.

## **Example**

```typescript {9-9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.datasphere.documents.deleteChunk("document-id", "chunk-id");
```
