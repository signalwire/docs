List chunks belonging to a document. Chunks are the individual text segments
created when a document is processed for vector search.

## **Parameters**

<ParamField path="documentId" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="queryParams" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters for pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of chunk objects.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const chunks = await client.datasphere.documents.listChunks("document-id");
for (const chunk of chunks.data ?? []) {
    console.log(chunk.id, chunk.text.substring(0, 80));
}
```
