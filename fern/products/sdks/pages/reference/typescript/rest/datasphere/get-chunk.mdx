Retrieve a specific chunk from a document.

## **Parameters**

<ParamField path="documentId" type="string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunkId" type="string" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The chunk object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const chunk = await client.datasphere.documents.getChunk("document-id", "chunk-id");
console.log(chunk.text);
```
