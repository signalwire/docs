Upload a new document to the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="name" type="string" toc={true}>
  Document name or title.
</ParamField>

<ParamField path="content" type="string" toc={true}>
  Document content text.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional document fields (e.g., `metadata`).
</ParamField>

## **Returns**

`Promise<any>` -- The newly created document object.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const doc = await client.datasphere.documents.create(;
    name: "FAQ",
    content: "Frequently asked questions and answers...",
)
console.log("Document ID:", doc.id);
```
