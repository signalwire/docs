Upload a new document to the [Datasphere][ref-datasphere].

## **Parameters**

<Markdown src="/snippets/sdks/rest/datasphere/documents/_create-params.mdx" />

## **Returns**

`Promise<any>` -- The newly created document object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const doc = await client.datasphere.documents.create({
    url: "https://filesamples.com/samples/document/txt/sample3.txt",
    tags: ["support", "demo"],
});
console.log("Document ID:", doc.id);
```
