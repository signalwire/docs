Perform a semantic search across all documents in the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="queryString" type="string" required={true} toc={true}>
  The natural-language search query.
</ParamField>

<ParamField path="count" type="number" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional search parameters (e.g., `tags`, `document_id` to narrow scope).
</ParamField>

## **Returns**

`Promise<any>` -- Search results with matching document chunks ranked by relevance.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const results = await client.datasphere.documents.search(;
    queryString: "How do I reset my password?",
    count=5,
)
for (const chunk of results.data ?? []) {
    console.log(chunk.text, "score:", chunk["score"]);
}
```
