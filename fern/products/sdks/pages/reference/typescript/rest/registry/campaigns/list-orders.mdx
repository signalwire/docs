List number assignment orders for a campaign.

## **Parameters**

<ParamField path="campaignId" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of order objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const orders = await client.registry.campaigns.listOrders("campaign-id");
for (const order of orders.data ?? []) {
    console.log(order.id, order.status);
}
```
