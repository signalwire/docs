Update a 10DLC campaign. Uses PUT to replace fields.

## **Parameters**

<ParamField path="campaignId" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Fields to update (e.g., `description`, `usecase`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated campaign object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const updated = await client.registry.campaigns.update("campaign-id", {
    description: "Updated campaign description",
});
console.log(updated.description);
```
