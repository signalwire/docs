Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brandId" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The brand object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const brand = await client.registry.brands.get("brand-id");
console.log(brand.display_name, brand.entity_type);
```
