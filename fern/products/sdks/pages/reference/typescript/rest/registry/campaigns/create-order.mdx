Create a number assignment order to assign phone numbers to a campaign.

## **Parameters**

<ParamField path="campaignId" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="phone_numbers" type="string[]" toc={true}>
  List of E.164 phone number strings to assign to the campaign.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Additional order parameters.
</ParamField>

## **Returns**

`Promise<any>` -- The newly created order object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const order = await client.registry.campaigns.createOrder("campaign-id", {
    phone_numbers: ["+15125551234"],
});
console.log("Order ID:", order.id);
```
