List messaging campaigns under a brand.

## **Parameters**

<ParamField path="brandId" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of campaign objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const campaigns = await client.registry.brands.listCampaigns("brand-id");
for (const c of campaigns.data ?? []) {
    console.log(c.description, c.usecase);
}
```
