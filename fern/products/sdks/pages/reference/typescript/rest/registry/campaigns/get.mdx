Retrieve a specific 10DLC campaign by ID.

<EndpointSchemaSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/relay/rest/registry/beta/campaigns/{id}" />

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const campaign = await client.registry.campaigns.get("campaign-id");
console.log(campaign.description, campaign.usecase);
```
