Retrieve a specific 10DLC campaign by ID.

## **Parameters**

<ParamField path="campaignId" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The campaign object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const campaign = await client.registry.campaigns.get("campaign-id");
console.log(campaign.description, campaign.usecase);
```
