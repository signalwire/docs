List phone numbers assigned to a campaign.

## **Parameters**

<ParamField path="campaignId" type="string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of number objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const numbers = await client.registry.campaigns.listNumbers("campaign-id");
for (const n of numbers.data ?? []) {
    console.log(n.number, n.id);
}
```
