Create a messaging campaign under a brand.

## **Parameters**

<ParamField path="brandId" type="string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="description" type="string" toc={true}>
  Campaign description.
</ParamField>

<ParamField path="sms_use_case" type="string" toc={true}>
  Campaign SMS use case.

  - `"MIXED"` -- multiple or general-purpose messaging use cases
  - `"MARKETING"` -- promotional or marketing messages
  - `"CUSTOMER_CARE"` -- customer support and service messages
</ParamField>

## **Returns**

`Promise<any>` -- The newly created campaign object.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const campaign = await client.registry.brands.createCampaign("brand-id", {
    description: "Customer notifications",
    sms_use_case: "MIXED",
});
console.log("Campaign ID:", campaign.id);
```
