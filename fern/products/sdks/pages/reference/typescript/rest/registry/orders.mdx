Retrieve number assignment orders. Orders track the process of assigning phone
numbers to 10DLC campaigns.

Access via `client.registry.orders` on a [`RestClient`][restclient] instance.

```typescript
import { RestClient } from "@signalwire/sdk";

client = RestClient()
```

## **get**

Retrieve a specific number assignment order.

### **Parameters**

<ParamField path="orderId" type="string" required={true} toc={true}>
  The order resource ID.
</ParamField>

### **Returns**

`dict` -- The order object.

### **Example**

```typescript
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

order = client.registry.orders.get("order-id")
console.log(order["id"], order["status"])
```
