Retrieve a specific voice call log entry.

## **Parameters**

<ParamField path="logId" type="string" required={true} toc={true}>
  The voice log ID.
</ParamField>

## **Returns**

`Promise<any>` -- The voice log object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const call = await client.logs.voice.get("voice-log-id");
console.log(call.from, "->", call.to, call.duration);
```
