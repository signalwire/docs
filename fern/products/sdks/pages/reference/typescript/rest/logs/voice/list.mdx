List voice call log entries.

## **Parameters**

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of voice log objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const calls = await client.logs.voice.list({ pageSize: 20 });
for (const call of calls.data ?? []) {
    console.log(call.from, "->", call.to, call.duration);
}
```
