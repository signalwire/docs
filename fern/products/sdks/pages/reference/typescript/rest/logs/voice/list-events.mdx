List events for a specific voice call. Events include state transitions
(ringing, answered, ended), media events, and other call lifecycle data.

## **Parameters**

<ParamField path="logId" type="string" required={true} toc={true}>
  The voice log ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of event objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const events = await client.logs.voice.listEvents("voice-log-id");
for (const event of events.data ?? []) {
    console.log(event.type, event.timestamp);
}
```
