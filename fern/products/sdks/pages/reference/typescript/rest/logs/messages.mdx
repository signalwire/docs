Query message log entries. Message logs provide a read-only view of SMS/MMS
activity in your project.

Access via `client.logs.messages` on a [`RestClient`][restclient] instance.

```typescript {3}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient();
```

### list

List message log entries.

## **Parameters**

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters for filtering and pagination (e.g., `pageSize`,
  `from`, `to`, `date_created_after`).
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of message log objects.

---

### get

Retrieve a specific message log entry.

## **Parameters**

<ParamField path="logId" type="string" required={true} toc={true}>
  The message log ID.
</ParamField>

## **Returns**

`Promise<any>` -- The message log object.

## **Examples**

### List message logs

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const messages = await client.logs.messages.list({ pageSize: 20 });
for (const msg of messages.data ?? []) {
    console.log(msg.from, "->", msg.to, msg.status);
}
```

### Get a specific message log

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const msg = await client.logs.messages.get("message-log-id");
console.log(msg.from, "->", msg.to);
```
