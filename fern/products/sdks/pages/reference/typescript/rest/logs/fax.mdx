Query fax log entries. Fax logs provide a read-only view of fax activity
in your project.

Access via `client.logs.fax` on a [`RestClient`][restclient] instance.

```typescript
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient();
```

### list

List fax log entries.

## **Parameters**

<ParamField path="queryParams" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of fax log objects.

---

### get

Retrieve a specific fax log entry.

## **Parameters**

<ParamField path="logId" type="string" required={true} toc={true}>
  The fax log ID.
</ParamField>

## **Returns**

`Promise<any>` -- The fax log object.

## **Examples**

### List fax logs

```typescript
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const faxes = await client.logs.fax.list();
for (const fax of faxes.data ?? []) {
    console.log(fax.from, "->", fax.to, fax.status);
}
```

### Get a specific fax log

```typescript
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const fax = await client.logs.fax.get("fax-log-id");
console.log(fax.from, "->", fax.to);
```
