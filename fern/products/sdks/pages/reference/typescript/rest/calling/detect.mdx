Start a detector on an active call. Detectors can identify answering machines,
fax tones, or DTMF digits. Returns a `control_id` for managing the detector.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to run detection on.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_detect-params.mdx" />

## **Returns**

`Promise<any>` -- The API response containing the `control_id`.

## **Examples**

### Answering Machine Detection

```typescript {10}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

// Answering machine detection
const result = await client.calling.detect("call-id-xxx", {
  detect: {
    type: "machine",
    params: {
      initial_timeout: 4.5,
      end_silence_timeout: 1.0,
    }
  },
  timeout: 30.0,
});
```

### Fax Detection

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.detect("call-id-xxx", {
  detect: { type: "fax", params: { tone: "CED" } }
});
```
