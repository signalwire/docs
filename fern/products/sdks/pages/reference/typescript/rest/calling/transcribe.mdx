Start transcribing speech on an active call. Transcription results are delivered
via events. Returns a `control_id` for stopping the transcription.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to transcribe.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_transcribe-params.mdx" />

## **Returns**

`Promise<any>` -- The API response containing the `control_id`.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.transcribe("call-id-xxx");
const controlId = result.control_id;
```
