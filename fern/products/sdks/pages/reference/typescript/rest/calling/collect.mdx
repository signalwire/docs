Start collecting user input on an active call. Supports DTMF digit collection
and speech recognition. Returns a `control_id` for managing the collection.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to collect input on.
</ParamField>

<ParamField path="collect" type="Record&lt;string, unknown&gt;" required={true} toc={true}>
  Collection configuration. Supports the following keys:

  - `digits` — DTMF digit collection settings (e.g., `{"max": 4, "terminators": "#"}`)
  - `speech` — Speech recognition settings (e.g., `{"end_silence_timeout": 1.0}`)
</ParamField>

<ParamField path="initialTimeout" type="number" toc={true}>
  Seconds to wait for the first input before timing out.
</ParamField>

<ParamField path="partialResults" type="boolean" toc={true}>
  Whether to send partial speech recognition results as events.
</ParamField>

## **Returns**

`Promise<any>` -- The API response containing the `control_id`.

## **Examples**

### Collect DTMF Digits

```typescript {10}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

// Collect DTMF digits
const result = await client.calling.collect("call-id-xxx", {
  collect: {
    digits: { max: 4, terminators: "#", digit_timeout: 5.0 }
  },
  initial_timeout: 10.0,
});
```

### Collect Speech

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.collect("call-id-xxx", {
  collect: {
    speech: {
      end_silence_timeout: 1.0,
      language: "en-US",
    }
  },
});
```

import { RestClient } from "@signalwire/sdk/rest";
