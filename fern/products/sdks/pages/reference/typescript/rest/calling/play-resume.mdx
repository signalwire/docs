Resume a paused playback.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming playback was resumed.

## **Example**

```typescript {9-9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.playResume("call-id-xxx", { controlId: "ctrl-id" });
```
