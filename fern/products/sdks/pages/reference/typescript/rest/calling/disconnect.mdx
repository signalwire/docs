Disconnect (unbridge) a connected call. If two call legs are bridged together,
this separates them without ending either leg.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to disconnect.
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming the disconnect.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.disconnect("call-id-xxx");
```
