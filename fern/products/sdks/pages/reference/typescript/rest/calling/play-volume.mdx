Adjust the volume of an active playback.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_play-volume-params.mdx" />

## **Returns**

`Promise<any>` -- The API response confirming the volume was adjusted.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.playVolume("call-id-xxx", { control_id: "ctrl-id", volume: 5.0 });
```
