Adjust the volume of an active playback.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

<ParamField path="volume" type="number" required={true} toc={true}>
  Volume level adjustment. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`dict` — The API response confirming the volume was adjusted.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.play_volume(call_id="call-id-xxx", control_id="ctrl-id", volume=5.0)
```
