Update parameters on an active call, such as metadata or call settings.

<Note>
Unlike most calling methods, `update()` does **not** take a positional `call_id`
parameter. Pass all fields (including call identification) in a single object.
</Note>

## **Parameters**

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Fields forwarded to the SignalWire API. Common fields include
  `call_id`, `metadata`, and other call settings.
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming the update.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.update({ callId: "call-id-xxx", metadata: { department: "sales" } });
```
