Start recording an active call. Returns a `control_id` used to
[pause][pause],
[resume][resume],
or [stop][stop] the recording.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to record.
</ParamField>

<ParamField path="record" type="Record&lt;string, unknown&gt;" required={true} toc={true}>
  Recording configuration object. Contains:

  - `audio` — Audio recording settings (e.g., `{"format": "mp3", "stereo": true}`)
</ParamField>

<ParamField path="listenOn" type="string" toc={true}>
  Which leg of the call to record.

  - `"self"` -- record only the audio from the call's own leg
  - `"both"` -- record audio from both legs of the call
</ParamField>

## **Returns**

`Promise<any>` -- The API response containing the `control_id` for managing the recording.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.record("call-id-xxx", {
  record: { audio: { format: "mp3", stereo: true } }
});
const controlId = result.control_id;
```
