Start recording an active call. Returns a `control_id` used to
[pause][pause],
[resume][resume],
or [stop][stop] the recording.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to record.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_record-params.mdx" />

## **Returns**

`Promise<any>` -- The API response containing the `control_id` for managing the recording.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.record("call-id-xxx", {
  audio: { format: "mp3", stereo: true }
});
const controlId = result.control_id;
```
