Resume a paused recording.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming the recording was resumed.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.recordResume("call-id-xxx", { controlId: "ctrl-id" });
```
