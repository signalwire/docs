Stop an active AI session on a call. The AI agent is disconnected but
the call itself remains active, allowing further call control operations.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

## **Returns**

`dict` — The API response confirming the AI session was stopped.

## **Example**

```typescript {10}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

# Stop AI and then play a goodbye message
client.calling.ai_stop(call_id="call-id-xxx")
client.calling.play(
    call_id="call-id-xxx",
    play=[{"type": "tts", "text": "Thank you for calling. Goodbye!"}]
)
client.calling.end(call_id="call-id-xxx", reason="hangup")
```
