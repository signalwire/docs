End an active call.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to end.
</ParamField>

<ParamField path="reason" type="string" toc={true}>
  The reason for ending the call.

  - `"hangup"` -- normal call termination
  - `"busy"` -- end the call with a busy signal
  - `"cancel"` -- cancel the call before it is answered
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming the call was ended.

## **Example**

```typescript {9-9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.end(callId: "call-id-xxx", reason: "hangup" });
```
