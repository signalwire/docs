Stop an active recording. The recording file becomes available after stopping.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_control-id-params.mdx" />

## **Returns**

`Promise<any>` -- The API response containing the recording URL and metadata.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.calling.recordStop("call-id-xxx", { control_id: "ctrl-id" });
console.log(result); // Contains recording URL
```
