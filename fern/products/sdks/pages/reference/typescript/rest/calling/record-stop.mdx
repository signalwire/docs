[record]: /docs/sdks/reference/typescript/rest/calling/record

Stop an active recording. The recording file becomes available after stopping.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`dict` — The API response containing the recording URL and metadata.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

result = client.calling.record_stop(call_id="call-id-xxx", control_id="ctrl-id")
console.log(result)  # Contains recording URL
```
