Disable noise reduction on an active call.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming noise reduction was disabled.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.denoiseStop("call-id-xxx");
```
