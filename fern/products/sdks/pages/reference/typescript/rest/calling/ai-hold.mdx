Put an active AI session on hold. The AI agent stops processing speech
while the call remains connected. This is useful for transferring the
caller to a human agent or performing background operations.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_ai-hold-params.mdx" />

## **Returns**

`Promise<any>` -- The API response confirming the AI session was put on hold.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.aiHold("call-id-xxx");
```
