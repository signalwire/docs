Play audio or text-to-speech on an active call. Returns a `control_id` that can be
used with [`playPause()`][play-pause],
[`playResume()`][play-resume],
[`playStop()`][play-stop], and
[`playVolume()`][play-volume] to manage
the playback.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to play audio on.
</ParamField>

<ParamField path="play" type={"Record<string, unknown>[]"} required={true} toc={true}>
  A list of media items to play. Each item is an object with a `type` field and
  type-specific parameters. Supported types:

  - `"audio"` — Play an audio file. Requires `url`.
  - `"tts"` — Play text-to-speech. Requires `text`. Optional: `language`, `gender`, `voice`.
  - `"silence"` — Play silence. Requires `duration` (in seconds).
  - `"ring"` — Play a ring tone. Optional: `duration`.
</ParamField>

<ParamField path="volume" type="number" toc={true}>
  Initial volume level for playback. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`Promise<any>` -- The API response containing the `control_id` for managing the playback.

## **Examples**

### Play TTS

```typescript {10}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

// Play TTS
const result = await client.calling.play("call-id-xxx", {
  play: [{ type: "tts", text: "Hello from the REST API!" }]
});
const controlId = result.control_id;
```

### Play Audio File

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.play("call-id-xxx", {
  play: [{ type: "audio", url: "https://example.com/greeting.mp3" }]
});
```

### Play Multiple Items

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.play("call-id-xxx", {
  play: [
    { type: "tts", text: "Please hold while we connect you." },
    { type: "silence", duration: 1 },
    { type: "audio", url: "https://example.com/hold-music.mp3" },
  ]
});
```
