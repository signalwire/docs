Stop an in-progress fax receive operation on an active call.

<Note>
This method stops a fax receive that is already in progress. To initiate fax receiving,
use the RELAY client's [`receiveFax()`][receive-fax] method.
</Note>

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call with an active fax receive operation.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_control-id-params.mdx" />

## **Returns**

`Promise<any>` -- The API response confirming the fax receive was stopped.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.receiveFaxStop("call-id-xxx", { control_id: "ctrl-id" });
```
