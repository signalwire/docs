[play]: /docs/sdks/reference/typescript/rest/calling/play

Stop an active playback immediately.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

## **Returns**

`dict` — The API response confirming playback was stopped.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.play_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
