Transfer an active call to a new destination. The current call leg is replaced
by a new connection to the specified target.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call to transfer.
</ParamField>

<ParamField path="dest" type="Record&lt;string, unknown&gt;" required={true} toc={true}>
  The transfer destination. Contains `type` (e.g., `"phone"`, `"sip"`) and
  type-specific parameters such as `to_number`.
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming the transfer was initiated.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.transfer("call-id-xxx", {
  dest: { type: "phone", params: { to_number: "+15559876543" } }
});
```
