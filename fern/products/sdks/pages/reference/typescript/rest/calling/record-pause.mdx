Pause an active recording. The recording can be resumed later with
[`recordResume()`][record-resume].

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_control-id-params.mdx" />

## **Returns**

`Promise<any>` -- The API response confirming the recording was paused.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.recordPause("call-id-xxx", { control_id: "ctrl-id" });
```
