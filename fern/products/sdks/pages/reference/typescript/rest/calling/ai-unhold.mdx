Resume an AI session that was previously put on hold with
[`aiHold()`][ai-hold].

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call with a held AI session.
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming the AI session was resumed.

## **Example**

```typescript {14}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

// Hold, perform an action, then resume
await client.calling.aiHold("call-id-xxx");

// ... do some background processing ...

await client.calling.aiUnhold("call-id-xxx");
```
