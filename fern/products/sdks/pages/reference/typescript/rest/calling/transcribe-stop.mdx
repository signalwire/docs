Stop an active transcription.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`transcribe()`][transcribe].
</ParamField>

## **Returns**

`dict` — The API response confirming transcription was stopped.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.calling.transcribe_stop(call_id="call-id-xxx", control_id="ctrl-id")
```
