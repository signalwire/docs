Stop an active transcription.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="controlId" type="string" required={true} toc={true}>
  The control ID returned from [`transcribe()`][transcribe].
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming transcription was stopped.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.transcribeStop("call-id-xxx", { controlId: "ctrl-id" });
```
