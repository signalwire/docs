Start live translation on an active call. Spoken audio is transcribed and
translated in real time to a target language.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="sourceLanguage" type="string" toc={true}>
  The language being spoken on the call (e.g., `"en-US"`).
</ParamField>

<ParamField path="targetLanguage" type="string" toc={true}>
  The language to translate into (e.g., `"es-ES"`).
</ParamField>

## **Returns**

`Promise<any>` -- The API response confirming live translation was started.

## **Example**

```typescript {9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.liveTranslate("call-id-xxx", {
  sourceLanguage: "en-US",
  targetLanguage: "es-ES",
});
```
