Start live translation on an active call. Spoken audio is transcribed and
translated in real time to a target language.

## **Parameters**

<ParamField path="callId" type="string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<Markdown src="/snippets/sdks/rest/calling/_live-translate-params.mdx" />

## **Returns**

`Promise<any>` -- The API response confirming the action was executed.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.calling.liveTranslate("call-id-xxx", {
  action: {
    start: {
      from_lang: "en-US",
      to_lang: "es-ES",
      direction: ["local-caller", "remote-caller"],
      from_voice: "elevenlabs.josh",
      to_voice: "elevenlabs.josh",
    }
  },
});
```
