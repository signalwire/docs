List events for a particular video room recording.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the room recording.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to return. Requires `page_token` for values greater than 0.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page. Default is `50`, maximum is `1000`.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination. Required when `page_number` is greater than 0.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing room recording events.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const events = await client.video.roomRecordings.listEvents("recording-id");
for (const event of events.data ?? []) {
  console.log(`${event.type}: ${event.id}`);
}
```
