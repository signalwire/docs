Retrieve a detailed summary of a particular video room recording.

<EndpointSchemaSnippet endpoint="GET /api/video/room_recordings/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_recordings/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const recording = await client.video.roomRecordings.get("recording-id");
console.log("Recording:", recording.id, "Status:", recording.status);
```
