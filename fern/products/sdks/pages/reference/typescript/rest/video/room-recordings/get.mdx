Retrieve a detailed summary of a particular video room recording.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the room recording.
</ParamField>

<ParamField path="media_ttl" type="number" toc={true}>
  Generated media links will be valid for this many seconds. Default is 900 (15 minutes).
</ParamField>

## **Returns**

`Promise<any>` -- The room recording.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const recording = await client.video.roomRecordings.get("recording-id");
console.log("Recording:", recording.id, "Status:", recording.status);
```
