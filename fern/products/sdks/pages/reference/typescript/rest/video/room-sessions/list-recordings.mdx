List recordings for a particular video room session.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

<ParamField path="media_ttl" type="number" toc={true}>
  Generated media links will be valid for this many seconds. Default is 900 (15 minutes).
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to return. Requires `page_token` for values greater than 0.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page. Default is `50`, maximum is `1000`.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination. Required when `page_number` is greater than 0.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing room session recordings.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const recordings = await client.video.roomSessions.listRecordings("session-id");
for (const recording of recordings.data ?? []) {
  console.log(`${recording.id}: ${recording.status}`);
}
```
