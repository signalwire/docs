## **Parameters**

<ParamField path="roomId" type="string" required={true}>
  The unique identifier of the room.
</ParamField>

<ParamField path="body" type="any" required={true}>
  The request body containing the resource data.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {8}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});
const result = await client.video.rooms.createStream("room-id", {
  url: "rtmp://example.com/live"
});
console.log(result);
```
