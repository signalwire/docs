The `VideoNamespace` provides access to SignalWire's video infrastructure through the
[`RestClient`][restclient]. It organizes 7 sub-resources
for managing video rooms, room tokens, room sessions, room recordings, conferences,
conference tokens, and streams.

Access via `client.video` on a [`RestClient`][restclient] instance.

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const rooms = await client.video.rooms.list();
```

## **Sub-resource Properties**

| Property | Description |
|----------|-------------|
| `rooms` | Video room CRUD with stream management |
| `roomTokens` | Generate tokens to authorize participants |
| `roomSessions` | Query room session history with events and members |
| `roomRecordings` | Manage room recordings with event history |
| `conferences` | Conference CRUD with token and stream management |
| `conferenceTokens` | Retrieve and reset conference tokens |
| `streams` | Top-level stream get, update, and delete |
