List all video room recordings.

<EndpointSchemaSnippet endpoint="GET /api/video/room_recordings" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/room_recordings" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.video.roomRecordings.list();
for (const recording of response.data ?? []) {
  console.log(`${recording.id}: ${recording.status}`);
}
```
