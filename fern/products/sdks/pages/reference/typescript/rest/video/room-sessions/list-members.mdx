List members for a particular video room session.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to return. Requires `page_token` for values greater than 0.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page. Default is `50`, maximum is `1000`.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination. Required when `page_number` is greater than 0.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing room session members.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const members = await client.video.roomSessions.listMembers("session-id");
for (const member of members.data ?? []) {
  console.log(`${member.name}: ${member.id}`);
}
```
