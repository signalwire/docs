Create a new stream for a video room.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

<Markdown src="/snippets/sdks/rest/video/streams/_create-stream-params.mdx" />

## **Returns**

`Promise<any>` -- The created stream.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const stream = await client.video.rooms.createStream("room-id", {
  url: "rtmp://broadcaster.example.com/live",
});
console.log("Stream created:", stream.id);
```
