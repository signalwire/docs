List tokens associated with a conference.

## **Parameters**

<ParamField path="conference_id" type="string" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- List of conference tokens.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const tokens = await client.video.conferences.listConferenceTokens("conference-id");
```
