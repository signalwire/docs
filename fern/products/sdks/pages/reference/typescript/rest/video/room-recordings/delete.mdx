Delete a video room recording.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the room recording to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success (HTTP 204).

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.video.roomRecordings.delete("recording-id");
console.log("Recording deleted");
```
