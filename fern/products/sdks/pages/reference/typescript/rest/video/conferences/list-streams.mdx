List active streams for a conference.

## **Parameters**

<ParamField path="conference_id" type="string" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`Promise<any>` -- List of streams in the conference.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const streams = await client.video.conferences.listStreams("conference-id");
```
