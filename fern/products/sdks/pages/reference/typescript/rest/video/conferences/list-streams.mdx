List active streams for a conference.

<EndpointSchemaSnippet endpoint="GET /api/video/conferences/{id}/streams" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/conferences/{id}/streams" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const streams = await client.video.conferences.listStreams("conference-id");
```
