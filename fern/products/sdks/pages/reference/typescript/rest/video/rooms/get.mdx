Retrieve a single video room by ID.

<EndpointSchemaSnippet endpoint="GET /api/video/rooms/{id}" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/rooms/{id}" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const room = await client.video.rooms.get("room-id");
console.log("Room:", room.name, "ID:", room.id);
```
