Retrieve a single video room by ID.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

<ParamField path="include_active_session" type="boolean" toc={true}>
  Whether to include information about the room's active session (if any).
</ParamField>

## **Returns**

`Promise<any>` -- The video room.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const room = await client.video.rooms.get("room-id");
console.log("Room:", room.name, "ID:", room.id);
```
