Find a video room session by ID.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

## **Returns**

`Promise<any>` -- The room session.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const session = await client.video.roomSessions.get("session-id");
console.log("Session:", session.name, "ID:", session.id);
```
