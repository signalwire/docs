Retrieve a specific conference token.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The unique identifier of the conference token.
</ParamField>

## **Returns**

`Promise<any>` -- The conference token resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const token = await client.video.conferenceTokens.get("token-id");
```
