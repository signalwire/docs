Delete a stream.

## **Parameters**

<ParamField path="stream_id" type="string" required={true} toc={true}>
  The unique identifier of the stream to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.video.streams.delete("stream-id");
```
