List video rooms in the project.

## **Parameters**

<ParamField path="include_active_session" type="boolean" toc={true}>
  Whether to include information about the room's active session (if any).
</ParamField>

<ParamField path="starts_after" type="string" toc={true}>
  Return rooms with a `join_from` date on or after this date. Expects RFC 3339 datetime or date string.
</ParamField>

<ParamField path="ends_before" type="string" toc={true}>
  Return rooms with a `join_until` date on or before this date. Expects RFC 3339 datetime or date string.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to return. Requires `page_token` for values greater than 0.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page. Default is `50`, maximum is `1000`.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination. Required when `page_number` is greater than 0.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing video rooms.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.video.rooms.list();
for (const room of response.data ?? []) {
  console.log(`${room.name}: ${room.id}`);
}
```
