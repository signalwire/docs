List video conferences in the project.

## **Parameters**

<ParamField path="starts_after" type="string" toc={true}>
  Filter conferences starting after this date (ISO 8601).
</ParamField>

<ParamField path="ends_before" type="string" toc={true}>
  Filter conferences ending before this date (ISO 8601).
</ParamField>

<ParamField path="include_active_session" type="boolean" toc={true}>
  Include active session details in the response.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing video conferences.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const conferences = await client.video.conferences.list();
```
