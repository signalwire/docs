List streams associated with a video room.

<EndpointSchemaSnippet endpoint="GET /api/video/rooms/{id}/streams" />

## **Response Example**

<EndpointResponseSnippet endpoint="GET /api/video/rooms/{id}/streams" />

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const streams = await client.video.rooms.listStreams("room-id");
for (const stream of streams.data ?? []) {
  console.log(`${stream.url}: ${stream.id}`);
}
```
