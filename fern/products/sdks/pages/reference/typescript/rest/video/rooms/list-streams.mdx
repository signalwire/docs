List streams associated with a video room.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

## **Returns**

`Promise<any>` -- List of streams for the room.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const streams = await client.video.rooms.listStreams("room-id");
for (const stream of streams.data ?? []) {
  console.log(`${stream.url}: ${stream.id}`);
}
```
