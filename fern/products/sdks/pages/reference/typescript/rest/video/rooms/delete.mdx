Delete a video room.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the video room to delete.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success (HTTP 204).

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

await client.video.rooms.delete("room-id");
console.log("Room deleted");
```
