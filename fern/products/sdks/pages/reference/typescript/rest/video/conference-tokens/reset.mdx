Reset a conference token, invalidating the current token and generating a new one.

## **Parameters**

<ParamField path="token_id" type="string" required={true} toc={true}>
  The unique identifier of the conference token to reset.
</ParamField>

## **Returns**

`Promise<any>` -- The reset result, including the new token.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const result = await client.video.conferenceTokens.reset("token-id");
```
