List past and in-progress video room sessions.

## **Parameters**

<ParamField path="room_id" type="string" toc={true}>
  Filter by room ID.
</ParamField>

<ParamField path="room_name" type="string" toc={true}>
  Filter by room name (exact match).
</ParamField>

<ParamField path="room_name_matches" type="string" toc={true}>
  Filter by room name (pattern match).
</ParamField>

<ParamField path="status" type="string" toc={true}>
  Filter by session status.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to return. Requires `page_token` for values greater than 0.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page. Default is `50`, maximum is `1000`.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination. Required when `page_number` is greater than 0.
</ParamField>

## **Returns**

`Promise<any>` -- Paginated response containing room sessions.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const response = await client.video.roomSessions.list();
for (const session of response.data ?? []) {
  console.log(`${session.name}: ${session.id}`);
}
```
