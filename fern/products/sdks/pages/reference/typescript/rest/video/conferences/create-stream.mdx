Create a new stream for a conference, enabling external streaming output
(e.g., RTMP to a streaming platform).

## **Parameters**

<ParamField path="conference_id" type="string" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

<Markdown src="/snippets/sdks/rest/video/streams/_create-stream-params.mdx" />

## **Returns**

`Promise<any>` -- The created stream resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const stream = await client.video.conferences.createStream("conference-id", {
  url: "rtmp://live.example.com/key",
});
```
