Retrieve a single video conference by its unique identifier.

## **Parameters**

<ParamField path="resource_id" type="string" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

## **Returns**

`Promise<any>` -- The conference resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const conf = await client.video.conferences.get("conference-id");
```
