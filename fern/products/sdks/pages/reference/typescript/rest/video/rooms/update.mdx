Update a video room. Uses PUT for full replacement.

## **Parameters**

<ParamField path="id" type="string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

<Markdown src="/snippets/sdks/rest/video/rooms/_update-params.mdx" />

## **Returns**

`Promise<any>` -- The updated video room.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const updated = await client.video.rooms.update("room-id", {
  display_name: "Updated Room Name",
  max_members: 50,
});
console.log("Updated:", updated.name);
```
