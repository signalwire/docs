Generate a room token allowing a client to join a video room.

## **Parameters**

<Markdown src="/snippets/sdks/rest/video/room-tokens/_create-params.mdx" />

## **Returns**

`Promise<any>` -- The created room token.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const tokenResult = await client.video.roomTokens.create({
  room_name: "my_room",
  user_name: "John Smith",
  permissions: [
    "room.self.audio_mute",
    "room.self.audio_unmute",
    "room.self.video_mute",
    "room.self.video_unmute",
  ],
  auto_create_room: true,
});
console.log("Token:", tokenResult.token);
```
