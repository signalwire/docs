Create a new video conference.

## **Parameters**

<Markdown src="/snippets/sdks/rest/video/conferences/_create-params.mdx" />

## **Returns**

`Promise<any>` -- The created conference resource.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com",
});

const conf = await client.video.conferences.create({
  display_name: "Team Meeting",
  name: "team-meeting",
});
```
