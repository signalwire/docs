Release (delete) a phone number from the project.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

client.phone_numbers.delete("phone-number-id")
```
