Release (delete) a phone number from the project.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty object on success.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.phoneNumbers.delete("phone-number-id");
```
