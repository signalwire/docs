Search for available phone numbers that can be purchased.

## **Parameters**

<ParamField path="areaCode" type="string" toc={true}>
  Filter by area code (e.g., `"512"`).
</ParamField>

<ParamField path="quantity" type="number" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**params" type="dict" toc={true}>
  Additional query parameters such as `region`, `city`, `country`, or
  `number_type`.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of available number objects.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

available = client.phone_numbers.search(area_code="503", quantity=5)
for number in available.get("data", []):
    console.log(number["number"], number["region"])
```
