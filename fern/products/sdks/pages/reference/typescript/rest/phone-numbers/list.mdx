List phone numbers owned by the project.

## **Parameters**

<ParamField path="filter_name" type="string" toc={true}>
  Filter by phone number name.
</ParamField>

<ParamField path="filter_number" type="string" toc={true}>
  Filter by phone number.
</ParamField>

<ParamField path="page_number" type="number" default="0" toc={true}>
  Page number to retrieve.
</ParamField>

<ParamField path="page_size" type="number" default="50" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="string" toc={true}>
  Token for cursor-based pagination.
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.phoneNumbers.list({ page_size: 10 });
for (const number of result.data ?? []) {
    console.log(number.number, number.name);
}
```
