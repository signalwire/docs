List phone numbers owned by the project.

## **Parameters**

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `pageSize`,
  `pageToken`).
</ParamField>

## **Returns**

`Promise<any>` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const result = await client.phoneNumbers.list({ pageSize: 10 });
for (const number of result.data ?? []) {
    console.log(number.number, number.name);
}
```
