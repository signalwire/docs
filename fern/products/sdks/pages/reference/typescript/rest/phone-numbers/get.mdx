Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`dict` -- The phone number object.

## **Example**

```typescript {9-10}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

number = client.phone_numbers.get("phone-number-id")
console.log(number["number"], number["name"])
```
