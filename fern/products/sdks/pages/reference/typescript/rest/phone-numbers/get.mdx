Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- The phone number object.

## **Example**

```typescript {9-10}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const number = await client.phoneNumbers.get("phone-number-id");
console.log(number.number, number.name);
```
