Purchase a phone number. The fields in the request body depend on the number being
purchased (typically obtained from a prior `search()` call).

## **Parameters**

<ParamField path="body" type="any" required={true} toc={true}>
  Number details. Common fields include `number` (the E.164 phone number to
  purchase).
</ParamField>

## **Returns**

`Promise<any>` -- The newly purchased phone number object.

## **Example**

```typescript {13}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

// Purchase the first available number from a search
const available = await client.phoneNumbers.search({ areaCode: "512", quantity: 1 });
const numbers = available.data ?? [];
if (numbers.length > 0) {
  const purchased = await client.phoneNumbers.create({ number: numbers[0].number });
  console.log("Purchased:", purchased.number);
}
```
