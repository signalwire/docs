Update configuration for a phone number. Uses PUT.

## **Parameters**

<ParamField path="resourceId" type="string" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

<ParamField path="params" type="Record&lt;string, unknown&gt;" toc={true}>
  Fields to update (e.g., `name`, `call_handler`, `call_request_url`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated phone number object.

## **Example**

```typescript {9-9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.phoneNumbers.update("phone-number-id", name: "Support Line" });
```
