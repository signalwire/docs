Update the project SIP profile. Uses PUT for full replacement.

## **Parameters**

<ParamField path="body" type="any" required={true} toc={true}>
  The request body containing the resource data.
</ParamField>

## **Returns**

`Promise<any>` -- The result of the operation.

## **Example**

```typescript {4}
import { RestClient } from "@signalwire/sdk";

const client = new RestClient();
const result = await client.sipProfile.update({ default_codecs: ['PCMU', 'PCMA'] });
console.log(result);
```
