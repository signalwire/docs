Retrieve the current project SIP profile.

## **Parameters**

None.

## **Returns**

`Promise<any>` -- The SIP profile object.

## **Example**

```typescript {4}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient();
const profile = await client.sipProfile.get();
console.log(profile);
```
