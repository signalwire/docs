Generate tokens for the SignalWire PubSub (publish-subscribe) real-time messaging
service. PubSub tokens authenticate client-side connections to subscribe to
channels and publish messages.

Access via `client.pubsub` on a [`RestClient`][restclient] instance.

```typescript
import { RestClient } from "@signalwire/sdk";

client = RestClient()
```

---

### create_token

Create a PubSub authentication token.

## **Parameters**

<ParamField path="channels" type="string[]" toc={true}>
  List of channel names the token grants access to.
</ParamField>

<ParamField path="memberId" type="string" toc={true}>
  Identifier for the connecting member.
</ParamField>

<ParamField path="ttl" type="number" toc={true}>
  Token time-to-live in seconds.
</ParamField>

<ParamField path="**kwargs" type="dict" toc={true}>
  Additional token parameters (e.g., `state`, `permissions`).
</ParamField>

## **Returns**

`dict` -- The token object containing the JWT token string.

## **Example**

```typescript
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.pubsub.create_token(
    channels=["notifications", "updates"],
    member_id="user-123",
    ttl=3600,
)
console.log("PubSub token:", token["token"])
```
