### tokens.delete

Revoke and delete an API token.

## **Parameters**

<ParamField path="tokenId" type="string" required={true} toc={true}>
  The token resource ID.
</ParamField>

## **Returns**

`Promise<any>` -- Empty dict on success.

## **Example**

```typescript {9-9}
import { RestClient } from "signalwire-agents/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.project.tokens.delete("token-id");
```
