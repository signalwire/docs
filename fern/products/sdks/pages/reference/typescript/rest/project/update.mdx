### tokens.update

Update an existing API token. Uses PATCH.

## **Parameters**

<ParamField path="tokenId" type="string" required={true} toc={true}>
  The token resource ID.
</ParamField>

<ParamField path="body" type="any" required={true} toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`Promise<any>` -- The updated token object.

## **Example**

```typescript {9-9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

await client.project.tokens.update("token-id", { name: "renamed-token" });
```
