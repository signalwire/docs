### tokens.create

Create a new API token for the project.

## **Parameters**

<ParamField path="body" type="any" required={true} toc={true}>
  Token configuration fields (e.g., `name`, `scopes`).
</ParamField>

## **Returns**

`Promise<any>` -- The newly created token object, including the token value.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk/rest";

const client = new RestClient({
  project: "your-project-id",
  token: "your-api-token",
  host: "your-space.signalwire.com"
});

const token = await client.project.tokens.create({ name: "ci-token" });
console.log("Token:", token["token"]);
console.log("ID:", token.id);
```
