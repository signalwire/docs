### tokens.create

Create a new API token for the project.

## **Parameters**

<ParamField path="**kwargs" type="dict" toc={true}>
  Token configuration fields (e.g., `name`, `scopes`).
</ParamField>

## **Returns**

`dict` -- The newly created token object, including the token value.

## **Example**

```typescript {9}
import { RestClient } from "@signalwire/sdk";

client = RestClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
)

token = client.project.tokens.create(name="ci-token")
console.log("Token:", token["token"])
console.log("ID:", token["id"])
```
