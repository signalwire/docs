Stop noise reduction on the call, restoring the original unfiltered audio.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming noise reduction has stopped.

## **Example**

```typescript {20}
import { RelayClient } from "@signalwire/sdk";

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()

    # Enable noise reduction
    await call.denoise()

    action = await call.play([{"type": "tts", "text": "Noise reduction is now active."}])
    await action.wait()

    result = await call.denoiseStop()
    console.log(`Noise reduction stopped: ${result}`)

client.run()
```
