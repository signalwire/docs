Pause the recording.

## **Parameters**

<ParamField path="behavior" type="string | undefined" toc={true}>
  Optional pause behavior. Controls what happens to the audio stream while paused
  (e.g., silence insertion).
</ParamField>

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```typescript {16}
import { RelayClient } from "@signalwire/sdk";

client = RelayClient(
    project="your-project-id",
    token="your-api-token",
    host="your-space.signalwire.com",
    contexts=["default"],
)

@client.on_call
async def handle_call(call):
    await call.answer()
    action = await call.record({"direction": "both", "format": "wav"})

    # Pause during sensitive information
    await action.pause()
    # ... sensitive exchange ...
    await action.resume()

client.run()
```
