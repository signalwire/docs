[play]: /docs/sdks/reference/typescript/agents/swml-builder/play

Add a text-to-speech `play` verb using the `say:` URL prefix. This is a convenience
wrapper around [`play()`][play] that
constructs the `say:` URL automatically.

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  The text to speak.
</ParamField>

<ParamField path="voice" type="string | undefined" toc={true}>
  Voice name for text-to-speech.
</ParamField>

<ParamField path="language" type="string | undefined" toc={true}>
  Language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="gender" type="string | undefined" toc={true}>
  Gender for voice selection.
</ParamField>

<ParamField path="volume" type="number | undefined" toc={true}>
  Volume adjustment level, from `-40` to `40` dB.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript
import { SwmlBuilder } from 'signalwire-agents';

const builder = new SwmlBuilder();
builder.addVerb('answer', {});
builder.addVerb('say', { text: 'Hello, welcome to our service.' });
```
