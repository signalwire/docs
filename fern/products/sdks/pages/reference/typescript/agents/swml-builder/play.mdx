Add a `play` verb to the main section. Plays audio from a URL or a list of URLs.

<Warning>
Exactly one of `url` or `urls` must be provided. Calling `play()` without either
raises a `ValueError`.
</Warning>

## **Parameters**

<ParamField path="url" type="string | undefined" toc={true}>
  Single URL to play. Supports audio file URLs and the `say:` prefix for
  text-to-speech (e.g., `"say:Hello world"`). Mutually exclusive with `urls`.
</ParamField>

<ParamField path="urls" type="string[] | undefined" toc={true}>
  List of URLs to play in sequence. Mutually exclusive with `url`.
</ParamField>

<ParamField path="volume" type="number | undefined" toc={true}>
  Volume adjustment level, from `-40` to `40` dB.
</ParamField>

<ParamField path="sayVoice" type="string | undefined" toc={true}>
  Voice name for text-to-speech playback.
</ParamField>

<ParamField path="sayLanguage" type="string | undefined" toc={true}>
  Language code for text-to-speech (e.g., `"en-US"`).
</ParamField>

<ParamField path="sayGender" type="string | undefined" toc={true}>
  Gender for text-to-speech voice selection.
</ParamField>

<ParamField path="autoAnswer" type="boolean | undefined" toc={true}>
  Whether to automatically answer the call before playing audio.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript
import { SwmlBuilder } from 'signalwire-agents';

const builder = new SwmlBuilder();
builder.addVerb('answer', {});
builder.addVerb('play', { url: 'https://example.com/audio.mp3' });
```
