Add a new named section to the SWML document. Delegates to
[`AgentBase.add_section()`][swmlservice-add-section].

## **Parameters**

<ParamField path="sectionName" type="string" required={true} toc={true}>
  Name of the section to create.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript {10}
import { AgentBase } from "@signalwire/sdk";
import { SwmlBuilder } from "@signalwire/sdk";

service = SWMLService(name="multi-section")
builder = SWMLBuilder(service)

doc = (
    builder
    .answer()
    .add_section("greeting")
    .play(url="say:Hello!")
    .build()
)
console.log(doc)
```
