Add a `hangup` verb to the main section. Ends the current call.

## **Parameters**

<ParamField path="reason" type="string | undefined" toc={true}>
  An optional reason string for the hangup, included in call logs.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript
import { SwmlBuilder } from 'signalwire-agents';

const builder = new SwmlBuilder();
builder.addVerb('answer', {});
builder.addVerb('hangup', {});
```
