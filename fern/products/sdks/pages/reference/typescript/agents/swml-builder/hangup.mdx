Add a `hangup` verb to the main section. Ends the current call.

## **Parameters**

<ParamField path="reason" type="string | undefined" toc={true}>
  An optional reason string for the hangup, included in call logs.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript {11}
import { AgentBase } from "@signalwire/sdk";
import { SwmlBuilder } from "@signalwire/sdk";

service = SWMLService(name="ivr")
builder = SWMLBuilder(service)

doc = (
    builder
    .answer()
    .play(url="https://example.com/goodbye.mp3")
    .hangup(reason="Call completed")
    .build()
)
console.log(doc)
```
