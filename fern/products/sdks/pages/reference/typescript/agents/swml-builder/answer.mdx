Add an `answer` verb to the main section. Answers an incoming call.

## **Parameters**

<ParamField path="maxDuration" type="number | undefined" toc={true}>
  Maximum call duration in seconds. The call is automatically hung up after this
  duration elapses.
</ParamField>

<ParamField path="codecs" type="string | undefined" toc={true}>
  Comma-separated list of audio codecs to use for the call.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript {9}
import { AgentBase } from "@signalwire/sdk";
import { SwmlBuilder } from "@signalwire/sdk";

service = SWMLService(name="my-service")
builder = SWMLBuilder(service)

doc = (
    builder
    .answer(max_duration=3600)
    .play(url="https://example.com/welcome.mp3")
    .build()
)
console.log(doc)
```
