[swmlservice-reset-document]: /docs/sdks/reference/typescript/agents/swml-service/reset-document

Reset the document to an empty state and return the builder for continued chaining.
Delegates to
[`AgentBase.reset_document()`][swmlservice-reset-document].

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```typescript
import { SwmlBuilder } from 'signalwire-agents';

const builder = new SwmlBuilder();
builder.addVerb('answer', {});
builder.addVerb('play', { url: 'https://example.com/audio.mp3' });

builder.reset();
// Document is now empty: { version: '1.0.0', sections: { main: [] } }
```
