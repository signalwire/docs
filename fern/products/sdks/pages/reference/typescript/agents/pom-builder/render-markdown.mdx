PomBuilder - Prompt Object Model for structured prompt sections.

## **Parameters**

<ParamField path="opts" type="unknown">
  Optional section configuration.
</ParamField>

<ParamField path="opts" type="unknown">
  Subsection configuration including title and optional body/bullets.
</ParamField>

<ParamField path="level" type="unknown">
  The heading level to start at (default 2 for ##).
</ParamField>

<ParamField path="sectionNumber" type="number[] = []">
  Hierarchical numbering prefix for numbered sections.
</ParamField>

## **Returns**

`string` -- The result of the operation.

## **Example**

```typescript
import { PomBuilder } from 'signalwire-agents';

const pomBuilder = new PomBuilder();
const result = pomBuilder.renderMarkdown();
console.log(result);
```
