Convert the POM to a list of section dictionaries. Useful for serializing the POM
for storage or passing to other components that expect dictionary input.

## **Returns**

`Record<string, unknown>[]` -- Serialized section list. Each dictionary contains keys
like `"title"`, `"body"`, and `"bullets"`.

## **Example**

```typescript
import { PomBuilder } from 'signalwire-agents';

const pom = new PomBuilder();
pom.addSection('Role', { body: 'You are a helpful assistant.' });

const data = pom.toDict();
console.log(JSON.stringify(data, null, 2));
```
