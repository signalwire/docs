## **has_section**

Check if a section with the given title exists.

### **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title to check.
</ParamField>

### **Returns**

`bool` -- `True` if the section exists, `False` otherwise.

---

## **get_section**

Get a section by title for direct manipulation.

### **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  Section title to look up.
</ParamField>

### **Returns**

`Section | undefined` -- The POM Section object, or `undefined` if not found.

## **Example**

```typescript
import { PomBuilder } from "@signalwire/sdk";

pom = PomBuilder()
pom.addSection("Role", body="You are a helpful assistant.")

if pom.hasSection("Role"):
    section = pom.getSection("Role")
    console.log(`Found section: ${section.title}`)

console.log(pom.hasSection("Missing"))  # false
```
