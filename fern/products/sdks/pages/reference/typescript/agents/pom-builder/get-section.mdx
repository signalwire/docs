PomBuilder - Prompt Object Model for structured prompt sections.

## **Parameters**

<ParamField path="opts" type="unknown">
  Optional section configuration.
</ParamField>

<ParamField path="opts" type="unknown">
  Subsection configuration including title and optional body/bullets.
</ParamField>

<ParamField path="level" type="unknown">
  The heading level to start at (default 2 for ##).
</ParamField>

<ParamField path="sectionNumber" type="unknown">
  Hierarchical numbering prefix for numbered sections.
</ParamField>

<ParamField path="title" type="string">
  The section heading.
</ParamField>

<ParamField path="opts" type="unknown">
  Optional body, bullets, numbering, and subsection configuration.
</ParamField>

<ParamField path="title" type="string">
  The section heading to append to.
</ParamField>

<ParamField path="opts" type="unknown">
  Body text and/or bullets to add.
</ParamField>

<ParamField path="parentTitle" type="unknown">
  The heading of the parent section.
</ParamField>

<ParamField path="title" type="string">
  The subsection heading.
</ParamField>

<ParamField path="opts" type="unknown">
  Optional body text and bullets for the subsection.
</ParamField>

<ParamField path="title" type="string">
  The section heading to look for.
</ParamField>

<ParamField path="title" type="string">
  The section heading to retrieve.
</ParamField>

## **Returns**

`PomSection | undefined` -- The result of the operation.

## **Example**

```typescript
import { PomBuilder } from 'signalwire-agents';

const pomBuilder = new PomBuilder();
const result = pomBuilder.getSection();
console.log(result);
```
