AgentServer - Hosts multiple AgentBase instances under a single HTTP server.

## **Parameters**

<ParamField path="opts" type="unknown">
  Optional host and port overrides; defaults to 0.0.0.0:3000.
</ParamField>

<ParamField path="agent" type="unknown">
  The AgentBase instance to mount.
</ParamField>

<ParamField path="route" type="unknown">
  Route prefix; defaults to the agent's own route or '/'.
</ParamField>

<ParamField path="route" type="unknown">
  The route prefix to unregister.
</ParamField>

<ParamField path="route" type="unknown">
  The route prefix to look up.
</ParamField>

<ParamField path="directory" type="unknown">
  Absolute or relative path to the directory to serve.
</ParamField>

<ParamField path="route" type="unknown">
  Route prefix for static files (defaults to '/static').
</ParamField>

## **Returns**

`Hono` -- The result of the operation.

## **Example**

```typescript
import { AgentServer } from "@signalwire/sdk";

const agentServer = new AgentServer();
const result = agentServer.getApp();
console.log(result);
```
