Retrieve a specific agent by its route. Returns `undefined` if no agent is registered at
the given path.

## **Parameters**

<ParamField path="route" type="string" required={true} toc={true}>
  The route to look up (e.g., `"/sales"`). Leading slashes are added and trailing slashes
  are stripped automatically for matching.
</ParamField>

## **Returns**

[`AgentBase`][agentbase] | `undefined` -- The agent instance
registered at the route, or `undefined` if not found.

## **Example**

```typescript {8}
import { AgentServer } from "@signalwire/sdk";
import { AgentBase } from "@signalwire/sdk";

agent1 = AgentBase(name="sales", route="/sales")
agent2 = AgentBase(name="support", route="/support")
server = AgentServer(port=3000)
server.register(agent1)
agent = server.getAgent("/sales")
if agent:
    console.log(`Found: ${agent.getName()}`)
else:
    console.log("No agent at /sales")
```
