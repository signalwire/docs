Get the required properties for a verb's inner configuration.

## **Parameters**

<ParamField path="verbName" type="string" required={true} toc={true}>
  The verb name.
</ParamField>

## **Returns**

`string[]` -- Array of required property names.

## **Example**

```typescript {4}
import { SchemaUtils } from 'signalwire-agents';

const schema = new SchemaUtils();
const required = schema.getVerbRequiredProperties('connect');
console.log(required);
```
