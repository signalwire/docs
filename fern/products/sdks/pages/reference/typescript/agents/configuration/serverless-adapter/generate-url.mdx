ServerlessAdapter - Adapts AgentBase for serverless platforms.

## **Parameters**

<ParamField path="platform" type="unknown">
  Target platform; defaults to 'auto' which detects from environment variables.
</ParamField>

<ParamField path="app" type="unknown">
  A Hono-compatible application with a `fetch` method.
</ParamField>

<ParamField path="event" type="unknown">
  The incoming serverless event to process.
</ParamField>

<ParamField path="opts" type="{">
  Optional overrides for region, project, function name, stage, or API ID.
</ParamField>

## **Returns**

`string` -- The result of the operation.

## **Example**

```typescript
import { ServerlessAdapter } from "@signalwire/sdk";

const serverlessAdapter = new ServerlessAdapter();
const result = serverlessAdapter.generateUrl();
console.log(result);
```
