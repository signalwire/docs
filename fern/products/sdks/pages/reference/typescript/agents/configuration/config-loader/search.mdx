ConfigLoader - JSON config file loader with env var interpolation.

## **Parameters**

<ParamField path="filePath" type="unknown">
  Path to a JSON config file to load on construction.
</ParamField>

<ParamField path="filePath" type="unknown">
  Path to the JSON config file.
</ParamField>

<ParamField path="filename" type="string">
  The config file name to search for.
</ParamField>

## **Returns**

`ConfigLoader | null` -- The result of the operation.

## **Example**

```typescript
import { ConfigLoader } from "@signalwire/sdk";

const result = ConfigLoader.search();
console.log(result);
```
