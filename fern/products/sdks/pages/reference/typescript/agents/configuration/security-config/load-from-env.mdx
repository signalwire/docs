Reload all settings from environment variables. This is called automatically
during construction but can be called again to pick up runtime changes.

## **Returns**

`undefined`

## **Example**

```typescript {7}
import os
import { AgentBase } from "@signalwire/sdk";

security = SecurityConfig()

os.environ["SWML_SSL_ENABLED"] = "true"
security.load_from_env()

console.log(`SSL enabled: ${security.ssl_enabled}`)  # true
```
