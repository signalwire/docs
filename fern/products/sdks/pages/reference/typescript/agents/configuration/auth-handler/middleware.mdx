AuthHandler - Multi-method authentication handler.

## **Parameters**

<ParamField path="config" type="unknown">
  Authentication configuration specifying one or more auth methods.
</ParamField>

<ParamField path="headers" type="unknown">
  The request headers as a string-keyed record.
</ParamField>

## **Returns**

`(c: any, next: () => Promise<void>) => Promise<Response | void>` -- The result of the operation.

## **Example**

```typescript
import { AuthHandler } from "@signalwire/sdk";

const authHandler = new AuthHandler();
const result = await authHandler.middleware();
console.log(result);
```
