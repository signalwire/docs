SessionManager - Stateless HMAC-SHA256 token generation and validation.

## **Parameters**

<ParamField path="tokenExpirySecs" type="unknown">
  Token validity duration in seconds (default 900).
</ParamField>

<ParamField path="secretKey" type="unknown">
  HMAC signing secret; a random key is generated if omitted.
</ParamField>

<ParamField path="callId" type="string">
  Existing call ID to reuse.
</ParamField>

## **Returns**

`string` -- The result of the operation.

## **Example**

```typescript
import { SessionManager } from "@signalwire/sdk";

const sessionManager = new SessionManager();
const result = sessionManager.createSession();
console.log(result);
```
