Get the number of cached validation results.

## **Parameters**

None.

## **Returns**

`number` -- The current cache entry count.

## **Example**

```typescript {6}
import { SchemaUtils } from 'signalwire-agents';

const schema = new SchemaUtils();
schema.validate(doc1);
schema.validate(doc2);
console.log(schema.getCacheSize()); // 2
```
