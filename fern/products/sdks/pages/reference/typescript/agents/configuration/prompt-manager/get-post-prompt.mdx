PromptManager - Manages agent prompts with POM support.

## **Parameters**

<ParamField path="usePom" type="unknown">
  Whether to use structured POM sections (default true).
</ParamField>

<ParamField path="text" type="unknown">
  The raw prompt string.
</ParamField>

<ParamField path="text" type="unknown">
  The post-prompt string.
</ParamField>

<ParamField path="title" type="unknown">
  The section heading.
</ParamField>

<ParamField path="opts" type="unknown">
  Optional body, bullets, numbering, and subsection configuration.
</ParamField>

<ParamField path="title" type="unknown">
  The section heading to append to.
</ParamField>

<ParamField path="opts" type="unknown">
  Body text and/or bullets to add.
</ParamField>

<ParamField path="parentTitle" type="unknown">
  The heading of the parent section.
</ParamField>

<ParamField path="title" type="unknown">
  The subsection heading.
</ParamField>

<ParamField path="opts" type="unknown">
  Optional body text and bullets for the subsection.
</ParamField>

<ParamField path="title" type="unknown">
  The section heading to look for.
</ParamField>

## **Returns**

`string | null` -- The result of the operation.

## **Example**

```typescript
import { PromptManager } from "@signalwire/sdk";

const promptManager = new PromptManager();
const result = promptManager.getPostPrompt();
console.log(result);
```
