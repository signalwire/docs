ServerlessAdapter - Adapts AgentBase for serverless platforms.

## **Parameters**

<ParamField path="platform" type="unknown">
  Target platform; defaults to 'auto' which detects from environment variables.
</ParamField>

## **Returns**

`platform;
  }

  /**
   * Detect the serverless platform by inspecting well-known environment variables.
   * @returns The detected platform identifier; defaults to 'lambda' if no match is found.
   */
  detectPlatform(): ServerlessPlatform` -- The result of the operation.

## **Example**

```typescript
import { ServerlessAdapter } from "@signalwire/sdk";

const serverlessAdapter = new ServerlessAdapter();
const result = serverlessAdapter.detectPlatform();
console.log(result);
```
