ConfigLoader - JSON config file loader with env var interpolation.

## **Parameters**

<ParamField path="filePath" type="unknown">
  Path to a JSON config file to load on construction.
</ParamField>

<ParamField path="filePath" type="unknown">
  Path to the JSON config file.
</ParamField>

<ParamField path="filename" type="unknown">
  The config file name to search for.
</ParamField>

<ParamField path="path" type="unknown">
  Dot-separated key path into the config object.
</ParamField>

<ParamField path="defaultValue" type="unknown">
  Value returned when the path does not exist.
</ParamField>

<ParamField path="path" type="unknown">
  Dot-separated key path into the config object.
</ParamField>

<ParamField path="value" type="unknown">
  The value to store.
</ParamField>

<ParamField path="path" type="unknown">
  Dot-separated key path to check.
</ParamField>

<ParamField path="obj" type={"Record<string, unknown>"}>
  The configuration object to use.
</ParamField>

## **Returns**

`this` -- The result of the operation.

## **Example**

```typescript
import { ConfigLoader } from "@signalwire/sdk";

const configLoader = new ConfigLoader();
const result = configLoader.loadFromObject();
console.log(result);
```
