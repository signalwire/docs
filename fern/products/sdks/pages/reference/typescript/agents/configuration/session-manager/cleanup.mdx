SessionManager - Stateless HMAC-SHA256 token generation and validation.

<ParamField path="tokenExpirySecs" type="unknown">

<ParamField path="callId" type="unknown">

  The expected function name.
</ParamField>

<ParamField path="token" type="unknown">
  The base64url-encoded token to validate.
</ParamField>

<ParamField path="functionName" type="unknown">
  The expected function name.
</ParamField>

<ParamField path="token" type="unknown">

</ParamField>

`void` -- The result of the operation.
