Get CORS configuration suitable for Hono's `CORSMiddleware`.

## **Returns**

`Record<string, unknown>` -- Dictionary with `allow_origins`, `allow_credentials`,
`allow_methods`, and `allow_headers` keys.

## **Example**

```typescript {6}
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
import { AgentBase } from "@signalwire/sdk";

security = SecurityConfig()
cors_config = security.get_cors_config()

app = FastAPI()
app.add_middleware(CORSMiddleware, **cors_config)
console.log(cors_config)
# {'allow_origins': ['*'], 'allow_credentials': true, 'allow_methods': ['*'], 'allow_headers': ['*']}
```
