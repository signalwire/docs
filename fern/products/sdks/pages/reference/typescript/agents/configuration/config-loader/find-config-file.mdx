### findConfigFile (static)

**[ConfigLoader][ref-configloader].findConfigFile**(`serviceName?: string`, `additionalPaths?: string[]`) -> `string | undefined`

Static method to locate a config file without loading it.

## **Parameters**

<ParamField path="serviceName" type="string" toc={true}>
  Optional service name. When provided, the method also checks for
  `{service_name}_config.json` and `.swml/{service_name}_config.json`.
</ParamField>

<ParamField path="additionalPaths" type="string[]" toc={true}>
  Additional file paths to check before the default search paths.
</ParamField>

## **Returns**

`string | undefined` -- Path to the first config file found, or `undefined`.

## **Example**

```typescript {4,9}
import { AgentBase } from "@signalwire/sdk";

# Find config for the MCP service
path = ConfigLoader.findConfigFile(service_name="mcp")
# Checks: mcp_config.json, .swml/mcp_config.json, config.json, ...
console.log(`Found: ${path}`)

# Find with extra paths
path = ConfigLoader.findConfigFile(
    additional_paths=["/opt/myapp/config.json"]
)
console.log(`Found: ${path}`)
```
