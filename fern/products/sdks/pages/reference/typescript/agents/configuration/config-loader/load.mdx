ConfigLoader - JSON config file loader with env var interpolation.

## **Parameters**

<ParamField path="filePath" type="string">
  Path to a JSON config file to load on construction.
</ParamField>

<ParamField path="filePath" type="string">
  Path to the JSON config file.
</ParamField>

## **Returns**

`this` -- The result of the operation.

## **Example**

```typescript
import { ConfigLoader } from "@signalwire/sdk";

const configLoader = new ConfigLoader();
const result = configLoader.load();
console.log(result);
```
