SessionManager - Stateless HMAC-SHA256 token generation and validation.

## **Parameters**

<ParamField path="tokenExpirySecs" type="unknown">
  Token validity duration in seconds (default 900).
</ParamField>

<ParamField path="secretKey" type="unknown">
  HMAC signing secret; a random key is generated if omitted.
</ParamField>

<ParamField path="callId" type="string">
  Existing call ID to reuse.
</ParamField>

<ParamField path="functionName" type="string">
  The SWAIG function name to bind.
</ParamField>

<ParamField path="callId" type="string">
  The call ID to bind.
</ParamField>

<ParamField path="functionName" type="string">
  The SWAIG function name to bind.
</ParamField>

<ParamField path="callId" type="string">
  The call ID to bind.
</ParamField>

<ParamField path="callId" type="string">
  The expected call ID.
</ParamField>

<ParamField path="functionName" type="string">
  The expected function name.
</ParamField>

<ParamField path="token" type="string">
  The base64url-encoded token to validate.
</ParamField>

## **Returns**

`boolean` -- The result of the operation.

## **Example**

```typescript
import { SessionManager } from "@signalwire/sdk";

const sessionManager = new SessionManager();
const result = sessionManager.validateToken();
console.log(result);
```
