ServerlessAdapter - Adapts AgentBase for serverless platforms.

## **Parameters**

<ParamField path="platform" type="unknown">
  Target platform; defaults to 'auto' which detects from environment variables.
</ParamField>

## **Returns**

`ServerlessPlatform` -- The result of the operation.

## **Example**

```typescript
import { ServerlessAdapter } from "@signalwire/sdk";

const serverlessAdapter = new ServerlessAdapter();
const result = serverlessAdapter.getPlatform();
console.log(result);
```
