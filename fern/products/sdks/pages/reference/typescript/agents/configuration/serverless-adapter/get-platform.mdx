Get the resolved platform identifier this adapter is configured for.

## **Parameters**

None.

## **Returns**

`ServerlessPlatform` -- The platform: `"lambda"`, `"gcf"`, `"azure"`, or `"cgi"`.

## **Example**

```typescript {4}
import { ServerlessAdapter } from '@signalwire/sdk';

const adapter = new ServerlessAdapter('gcf');
console.log(adapter.getPlatform()); // "gcf"
```
