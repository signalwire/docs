Join a SignalWire RELAY room. Rooms enable multi-party communication and
collaboration between participants.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Name of the room to join.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] -- `this`, for chaining.

## **Example**

```typescript
import { FunctionResult } from 'signalwire-agents';

const result = new FunctionResult()
  .joinRoom('support-room');
```
