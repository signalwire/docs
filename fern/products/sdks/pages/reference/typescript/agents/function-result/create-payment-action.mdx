**[FunctionResult][ref-functionresult].create_payment_action**(`action_type`, `phrase`) -> `Record<string, string>`

Static helper that builds a single action entry for a payment prompt.

## **Parameters**

<ParamField path="actionType" type="string" required={true} toc={true}>
  Action type.

  - `"Say"` -- use text-to-speech
  - `"Play"` -- play an audio file URL
</ParamField>

<ParamField path="phrase" type="string" required={true} toc={true}>
  Text to speak (when `action_type` is `"Say"`) or URL to play (when `action_type`
  is `"Play"`).
</ParamField>

## **Returns**

`Record<string, string>` — action dictionary for use in
[`createPaymentPrompt()`][create-payment-prompt].

## **Example**

```typescript
import { FunctionResult } from 'signalwire-agents';

const action = FunctionResult.createPaymentAction(
  'say',
  'Please enter your card number.',
);
```
