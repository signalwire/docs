Send an SMS or MMS message to a phone number. At least one of `body` or `media`
must be provided. Generates a SWML `send_sms` verb under the hood.

## **Parameters**

<ParamField path="toNumber" type="string" required={true} toc={true}>
  Destination phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="fromNumber" type="string" required={true} toc={true}>
  Sender phone number in E.164 format. Must be a number in your SignalWire project.
</ParamField>

<ParamField path="body" type="string | undefined" default="None" toc={true}>
  Text body of the message.
</ParamField>

<ParamField path="media" type="string[] | undefined" default="None" toc={true}>
  List of media URLs to include as MMS attachments.
</ParamField>

<ParamField path="tags" type="string[] | undefined" default="None" toc={true}>
  Tags to associate with the message for searching and filtering in the
  SignalWire dashboard.
</ParamField>

<ParamField path="region" type="string | undefined" default="None" toc={true}>
  Region to originate the message from.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Text Message

```typescript {13}
import { AgentBase } from "@signalwire/sdk";
import { FunctionResult } from "@signalwire/sdk";

agent = AgentBase(name="my-agent", route="/agent")
agent.setPromptText("You are a helpful assistant.")

    phone = args["phone_number"]
    order_id = args["order_id"]
    return (
        FunctionResult("I've sent you a confirmation text.")
        .send_sms(
            to_number=phone,
            from_number="+15559876543",
            body=`Your order ${order_id} has been confirmed!`
        )
    )

agent.run()
```

### MMS with Media

```typescript {13}
import { AgentBase } from "@signalwire/sdk";
import { FunctionResult } from "@signalwire/sdk";

agent = AgentBase(name="my-agent", route="/agent")
agent.setPromptText("You are a helpful assistant.")

    phone = args["phone_number"]
    receipt_url = args["receipt_url"]
    return (
        FunctionResult("I've sent your receipt.")
        .send_sms(
            to_number=phone,
            from_number="+15559876543",
            body="Here's your receipt:",
            media=[receipt_url]
        )
    )

agent.run()
```
