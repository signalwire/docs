Send an SMS or MMS message to a phone number. At least one of `body` or `media`
must be provided. Generates a SWML `send_sms` verb under the hood.

## **Parameters**

<ParamField path="toNumber" type="string" required={true} toc={true}>
  Destination phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="fromNumber" type="string" required={true} toc={true}>
  Sender phone number in E.164 format. Must be a number in your SignalWire project.
</ParamField>

<ParamField path="body" type="string | undefined" default="undefined" toc={true}>
  Text body of the message.
</ParamField>

<ParamField path="media" type="string[] | undefined" default="undefined" toc={true}>
  List of media URLs to include as MMS attachments.
</ParamField>

<ParamField path="tags" type="string[] | undefined" default="undefined" toc={true}>
  Tags to associate with the message for searching and filtering in the
  SignalWire dashboard.
</ParamField>

<ParamField path="region" type="string | undefined" default="undefined" toc={true}>
  Region to originate the message from.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] -- `this`, for chaining.

## **Examples**

### Text Message

```typescript
import { FunctionResult } from 'signalwire-agents';

const result = new FunctionResult('SMS sent.')
  .sendSms({
    toNumber: '+15551234567',
    fromNumber: '+15559876543',
    body: 'Your appointment is confirmed for tomorrow at 3 PM.',
  });
```

### MMS with Media

```typescript
import { AgentBase, FunctionResult } from 'signalwire-agents';

const agent = new AgentBase({ name: 'my-agent', route: '/agent' });
agent.setPromptText("You are a helpful assistant.")

    phone = args["phone_number"]
    receipt_url = args["receipt_url"]
    return (
        new FunctionResult('I've sent your receipt.')
        .sendSms(
            to_number=phone,
            from_number="+15559876543",
            body="Here's your receipt:",
            media=[receipt_url]
        )
    )

agent.run()
```
