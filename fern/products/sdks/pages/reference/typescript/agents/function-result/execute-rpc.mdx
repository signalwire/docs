Execute a generic RPC method on a call. This is the low-level interface for
cross-call communication. For common operations, prefer the specific helpers
[`rpcDial()`][rpc-dial],
[`rpcAiMessage()`][rpc-ai-message], and
[`rpcAiUnhold()`][rpc-ai-unhold].

## **Parameters**

<ParamField path="method" type="string" required={true} toc={true}>
  RPC method name (e.g., `"dial"`, `"ai_message"`, `"ai_unhold"`).
</ParamField>

<ParamField path="params" type={"Record<string, unknown> | undefined"} default="None" toc={true}>
  Parameters for the RPC method.
</ParamField>

<ParamField path="callId" type="string | undefined" default="None" toc={true}>
  Target call ID for the RPC command.
</ParamField>

<ParamField path="nodeId" type="string | undefined" default="None" toc={true}>
  Target node ID for the RPC command.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```typescript {11}
import { AgentBase } from "@signalwire/sdk";
import { FunctionResult } from "@signalwire/sdk";

agent = AgentBase(name="my-agent", route="/agent")
agent.setPromptText("You are a helpful assistant.")

    return (
        FunctionResult("Executing RPC.")
        .execute_rpc(
            method="ai_message",
            call_id="target-call-id",
            params={"role": "system", "message_text": "Hello from another call."}
        )
    )

agent.run()
```
