SkillBase - Abstract base class for agent skills.

## **Parameters**

<ParamField path="skillName" type="unknown">
  The registered name for this skill type.
</ParamField>

<ParamField path="config" type="unknown">
  Optional configuration key-value pairs.
</ParamField>

<ParamField path="rawData" type="unknown">
  The raw request data containing global_data.
</ParamField>

<ParamField path="result" type="FunctionResult">
  The FunctionResult to update.
</ParamField>

<ParamField path="data" type={"Record<string, unknown>"}>
  The data to store under this skill's namespace.
</ParamField>

## **Returns**

`FunctionResult` -- The result of the operation.

## **Example**

```typescript
import { SkillBase } from 'signalwire-agents';

const skillBase = new SkillBase();
const result = skillBase.updateSkillData();
console.log(result);
```
