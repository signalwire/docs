[ref-skillbase]: /docs/sdks/reference/typescript/agents/skill-base

Initialize the skill: validate environment variables, check packages, initialize
API clients, and prepare resources. Called once when the skill is loaded.

This is an abstract method -- you **must** implement it in every [SkillBase][ref-skillbase] subclass.

## **Returns**

`boolean` --- `true` if setup succeeded, `false` to indicate the skill should not
be loaded.

## **Example**

```typescript
class MySkill extends SkillBase {
  async setup(): Promise<void> {
    // Initialize API connections, validate config
    console.log('Skill setup complete');
  }
}
```
