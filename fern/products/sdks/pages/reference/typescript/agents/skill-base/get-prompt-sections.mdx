Return POM prompt sections for the agent. Returns an empty list when the
`skip_prompt` parameter is `True`, otherwise delegates to `GetPromptSections()`.

## **Returns**

`Record<string, unknown>[]` -- List of section dictionaries with `"title"` and
`"body"` or `"bullets"` keys.

<Tip>
Override `GetPromptSections()` instead of this method so your skill
automatically respects the `skip_prompt` configuration parameter.
</Tip>

---

Override this in subclasses to provide prompt sections. This is the recommended
override target rather than `getPromptSections()`.

## **Returns**

`Record<string, unknown>[]` -- List of section dictionaries (default: empty list).

## **Example**

```typescript {7-8}
import { SkillBase } from "@signalwire/sdk";

class WeatherSkill extends SkillBase {
    SKILL_NAME = "weather"
    SKILL_DESCRIPTION = "Provides weather information"

    GetPromptSections() {
        return [
            {
                "title": "Weather Information",
                "body": "You can check weather for any location using the get_weather tool."
            },
            {
                "title": "Weather Guidelines",
                "bullets": [
                    "Always confirm the location before checking weather",
                    "Report temperature in the user's preferred units"
                ]
            }
        ]

    setup(): boolean {
        return true

    registerTools() {
        pass
```
