Return data to merge into the agent's `global_data` dictionary. Override to
provide skill-specific session data that should be available to all SWAIG
function handlers.

## **Returns**

`Record<string, unknown>` -- Key-value pairs (default: empty dict).

## **Example**

```typescript {7}
import { SkillBase } from "@signalwire/sdk";

class WeatherSkill extends SkillBase {
    SKILL_NAME = "weather"
    SKILL_DESCRIPTION = "Provides weather information"

    getGlobalData() {
        return {
            "weather_api_base": "https://api.weatherapi.com/v1",
            "default_units": self.params.get("units", "fahrenheit")
        }

    setup(): boolean {
        return true

    registerTools() {
        pass
```
