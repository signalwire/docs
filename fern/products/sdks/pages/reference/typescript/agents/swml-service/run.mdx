SWMLService - Lightweight HTTP service for non-AI SWML documents.

## **Parameters**

<ParamField path="name" type="unknown">
  Verb name (e.g., 'answer', 'play', 'hangup').
</ParamField>

<ParamField path="config" type="unknown">
  Verb configuration.
</ParamField>

<ParamField path="cb" type="unknown">
  Callback receiving query params, body params, and headers.
</ParamField>

<ParamField path="host" type="string">
  Hostname (default '0.0.0.0').
</ParamField>

<ParamField path="port" type="number">
  Port (default 3000 or PORT env var).
</ParamField>

## **Returns**

`Promise<void>` -- The result of the operation.

## **Example**

```typescript
import { SWMLService } from 'signalwire-agents';

const sWMLService = new SWMLService();
const result = await sWMLService.run();
console.log(result);
```
