SWMLService - Lightweight HTTP service for non-AI SWML documents.

## **Parameters**

This method takes no parameters.

## **Returns**

`SwmlBuilder` -- The result of the operation.

## **Example**

```typescript
import { SWMLService } from "@signalwire/sdk";

const sWMLService = new SWMLService();
const result = sWMLService.getBuilder();
console.log(result);
```
