DataMap - Fluent builder for SWAIG data_map configurations.

## **Parameters**

<ParamField path="prefixes" type="unknown">
  Array of prefix strings to allow.
</ParamField>

<ParamField path="functionName" type="unknown">
  The unique name for this data map tool.
</ParamField>

<ParamField path="enabled" type="unknown">
  Whether to enable expansion (defaults to true).
</ParamField>

## **Returns**

`this` -- The result of the operation.

## **Example**

```typescript
import { DataMap } from "@signalwire/sdk";

const dataMap = new DataMap();
const result = dataMap.enableEnvExpansion();
console.log(result);
```
