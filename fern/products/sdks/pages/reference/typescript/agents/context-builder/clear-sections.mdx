[step]: /docs/sdks/reference/typescript/agents/context-builder/step

Remove all POM sections and raw text from this step. This is a method on
[`Step`][step].

## **Parameters**

None.

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {8}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

const step = ctx.addStep('greet');
step.setText('Old greeting text.');
step.clearSections();
step.setText('New greeting text.');
```
