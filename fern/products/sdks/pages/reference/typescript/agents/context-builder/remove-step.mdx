Remove a step from this context by name. This is a method on [`Context`][context].

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The step name to remove.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {8}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');
ctx.addStep('greet').setText('Greet the caller.');
ctx.addStep('old_step').setText('Unused step.');

ctx.removeStep('old_step');
```
