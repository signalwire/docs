Set the user prompt text for this context. This is a method on [`Context`][context].

## **Parameters**

<ParamField path="userPrompt" type="string" required={true} toc={true}>
  The user prompt string.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setUserPrompt('I need help with my account.');
ctx.addStep('greet').setText('Greet the caller.');
```
