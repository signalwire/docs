Set the user prompt to use when resetting context at this step. This is a
method on [`Step`][step].

## **Parameters**

<ParamField path="userPrompt" type="string" required={true} toc={true}>
  The replacement user prompt text.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('reset_point')
  .setText('Reset the conversation context.')
  .setResetUserPrompt('Please help me with my billing question.');
```
