[context]: /docs/sdks/reference/typescript/agents/context-builder/context

Set raw system prompt text for this context, mutually exclusive with system POM
sections. This is a method on [`Context`][context].

## **Parameters**

<ParamField path="systemPrompt" type="string" required={true} toc={true}>
  The system prompt string.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setSystemPrompt('You are a helpful support agent for Acme Corp.');
ctx.addStep('greet').setText('Greet the caller.');
```
