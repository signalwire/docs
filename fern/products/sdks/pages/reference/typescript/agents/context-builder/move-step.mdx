Move a step to a new position in the step order. This is a method on
[`Context`][context].

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The step name to move.
</ParamField>

<ParamField path="position" type="number" required={true} toc={true}>
  The zero-based index to insert at.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {10}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');
ctx.addStep('greet').setText('Greet the caller.');
ctx.addStep('ask_name').setText('Ask for their name.');
ctx.addStep('confirm').setText('Confirm the information.');

// Move 'confirm' to be the second step (index 1)
ctx.moveStep('confirm', 1);
```
