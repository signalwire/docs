Retrieve an existing context by name for inspection or modification.

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  Name of the context to retrieve.
</ParamField>

## **Returns**

[`Context`][context] if found,
`undefined` otherwise.

## **Example**

```typescript {9}
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="my-agent", route="/agent")

contexts = agent.defineContexts()
default = contexts.add_context("default")
default.add_step("greet").set_text("Greet the caller.")

ctx = contexts.get_context("default")
if ctx is not undefined:
    ctx.addStep("followup").set_text("Ask if there is anything else.")

agent.run()
```
