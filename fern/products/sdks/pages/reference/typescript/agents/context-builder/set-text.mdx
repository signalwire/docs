Set raw text content for this step, mutually exclusive with POM sections.
This is a method on [`Step`][step], not on ContextBuilder.

## **Parameters**

<ParamField path="text" type="string" required={true} toc={true}>
  The raw prompt text for this step.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('greet').setText('Greet the caller and ask how you can help.');
```
