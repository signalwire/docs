Initialize a gather info operation on this step for collecting structured data.
Must be called before `addGatherQuestion()`. This is a method on [`Step`][step].

## **Parameters**

<ParamField path="opts.outputKey" type="string" toc={true}>
  Key in `global_data` under which to nest all answers.
</ParamField>

<ParamField path="opts.completionAction" type="string" toc={true}>
  Where to navigate after all questions are answered: `"next_step"` or a step name.
</ParamField>

<ParamField path="opts.prompt" type="string" toc={true}>
  Preamble text injected when entering the gather step.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7-11}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('intake')
  .setText('Collect the caller\'s information.')
  .setGatherInfo({
    outputKey: 'caller_info',
    completionAction: 'next_step',
    prompt: 'You are a friendly receptionist.',
  })
  .addGatherQuestion({ key: 'name', question: 'What is your full name?' })
  .addGatherQuestion({ key: 'phone', question: 'What is your callback number?' });
```
