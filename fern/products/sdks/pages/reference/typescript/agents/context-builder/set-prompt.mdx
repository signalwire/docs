Set raw prompt text for this context, mutually exclusive with POM prompt sections.
This is a method on [`Context`][context].

## **Parameters**

<ParamField path="prompt" type="string" required={true} toc={true}>
  The prompt string.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setPrompt('Handle customer support inquiries about billing.');
ctx.addStep('greet').setText('Greet the caller.');
```
