Set whether to perform a full conversation reset at this step.
This is a method on [`Step`][step].

## **Parameters**

<ParamField path="fullReset" type="boolean" required={true} toc={true}>
  Whether to fully reset.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('fresh_start')
  .setText('Start completely fresh from here.')
  .setResetFullReset(true);
```
