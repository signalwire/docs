Set whether to automatically advance to the next step after this one completes.
This is a method on [`Step`][step].

## **Parameters**

<ParamField path="skip" type="boolean" required={true} toc={true}>
  Whether to skip to the next step.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('collect_info')
  .setText('Collect the caller\'s information.')
  .setSkipToNextStep(true);
```
