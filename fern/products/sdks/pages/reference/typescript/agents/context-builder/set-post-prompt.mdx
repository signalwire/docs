Set the post-prompt text for this context. This is a method on [`Context`][context].

## **Parameters**

<ParamField path="postPrompt" type="string" required={true} toc={true}>
  The post-prompt string.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setPostPrompt('Summarize the conversation in JSON format.');
ctx.addStep('greet').setText('Greet the caller.');
```
