Set whether to fully reset conversation history when entering this context.
This is a method on [`Context`][context].

## **Parameters**

<ParamField path="fullReset" type="boolean" required={true} toc={true}>
  Whether to fully reset.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setFullReset(true);
ctx.addStep('greet').setText('Greet the caller.');
```
