Return a step by name. This is a method on [`Context`][context].

## **Parameters**

<ParamField path="name" type="string" required={true} toc={true}>
  The step name to retrieve.
</ParamField>

## **Returns**

[`Step`][step] ` | undefined` -- The matching Step, or `undefined` if not found.

## **Example**

```typescript {8}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');
ctx.addStep('greet').setText('Greet the caller.');

// Retrieve and modify an existing step
const step = ctx.getStep('greet');
if (step) {
  step.setStepCriteria('Caller has been greeted');
}
```
