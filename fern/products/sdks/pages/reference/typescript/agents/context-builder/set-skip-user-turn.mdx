[step]: /docs/sdks/reference/typescript/agents/context-builder/step

Set whether to skip waiting for user input when entering this step. This is a
method on [`Step`][step].

## **Parameters**

<ParamField path="skip" type="boolean" required={true} toc={true}>
  Whether to skip the user turn.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('auto_greet')
  .setText('Immediately greet the caller without waiting for input.')
  .setSkipUserTurn(true);
```
