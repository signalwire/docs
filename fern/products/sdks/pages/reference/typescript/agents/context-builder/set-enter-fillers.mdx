Set filler phrases spoken when entering this context, keyed by language code.
This is a method on [`Context`][context].

## **Parameters**

<ParamField path="fillers" type="Record<string, string[]>" required={true} toc={true}>
  Map of language codes to arrays of filler phrases.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6-8}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setEnterFillers({
  'en-US': ['One moment please...', 'Let me look into that...'],
});
ctx.addStep('greet').setText('Greet the caller.');
```
