Mark this step as an end step, terminating the conversation when reached.
This is a method on [`Step`][step].

## **Parameters**

<ParamField path="end" type="boolean" required={true} toc={true}>
  Whether this is an end step.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('goodbye')
  .setText('Thank the caller and say goodbye.')
  .setEnd(true);
```
