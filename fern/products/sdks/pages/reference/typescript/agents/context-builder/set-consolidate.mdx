Set whether to consolidate conversation history when entering this context.
This is a method on [`Context`][context].

## **Parameters**

<ParamField path="consolidate" type="boolean" required={true} toc={true}>
  Whether to consolidate.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.setConsolidate(true);
ctx.addStep('greet').setText('Greet the caller.');
```
