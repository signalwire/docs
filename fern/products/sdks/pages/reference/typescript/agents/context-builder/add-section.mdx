Add a POM section with a body, mutually exclusive with raw text. This method
exists on both [`Step`][step] and [`Context`][context].

## **Parameters**

<ParamField path="title" type="string" required={true} toc={true}>
  The section heading.
</ParamField>

<ParamField path="body" type="string" required={true} toc={true}>
  The section body text.
</ParamField>

## **Returns**

`this` -- The Step or Context instance for chaining.

## **Examples**

### On a Step

```typescript {6}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('greet')
  .addSection('Task', 'Greet the caller warmly.')
  .addSection('Personality', 'Be friendly and professional.');
```

### On a Context

```typescript {6}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addSection('Context Info', 'This is the main support context.');
ctx.addStep('greet').setText('Greet the caller.');
```
