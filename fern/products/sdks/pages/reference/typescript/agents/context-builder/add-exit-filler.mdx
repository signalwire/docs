Add exit filler phrases for a specific language. This is a method on
[`Context`][context].

## **Parameters**

<ParamField path="languageCode" type="string" required={true} toc={true}>
  The language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="fillers" type="string[]" required={true} toc={true}>
  Array of filler phrases.
</ParamField>

## **Returns**

`this` -- The Context instance for chaining.

## **Example**

```typescript {6}
import { ContextBuilder } from '@signalwire/sdk';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addExitFiller('en-US', ['Transferring you now...']);
ctx.addStep('greet').setText('Greet the caller.');
```
