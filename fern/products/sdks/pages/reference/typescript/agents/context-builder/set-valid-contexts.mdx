Set which contexts the AI may navigate to. This method exists on both
[`Step`][step] and [`Context`][context].

## **Parameters**

<ParamField path="contexts" type="string[]" required={true} toc={true}>
  List of allowed context names.
</ParamField>

## **Returns**

`this` -- The Step or Context instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('menu')
  .setText('Ask whether the caller needs sales or support.')
  .setValidContexts(['sales', 'support']);
```
