Set whether to consolidate conversation history when resetting at this step.
This is a method on [`Step`][step].

## **Parameters**

<ParamField path="consolidate" type="boolean" required={true} toc={true}>
  Whether to consolidate.
</ParamField>

## **Returns**

`this` -- The Step instance for chaining.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');

ctx.addStep('consolidate_point')
  .setText('Consolidate conversation history here.')
  .setResetConsolidate(true);
```
