Return the list of valid context names the AI may navigate to from this context.
This is an internal method on [`Context`][context].

## **Parameters**

None.

## **Returns**

`string[] | null` -- The allowed context names, or `null` if not set.

## **Example**

```typescript {7}
import { ContextBuilder } from 'signalwire-agents';

const builder = new ContextBuilder();
const ctx = builder.addContext('default');
ctx.setValidContexts(['sales', 'support']);

console.log(ctx.getValidContexts()); // ["sales", "support"]
```
