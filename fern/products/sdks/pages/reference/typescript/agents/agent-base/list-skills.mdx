List the names of all currently loaded skills on this agent.

## **Returns**

`string[]` -- List of loaded skill names.

## **Example**

```typescript {6}
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="assistant", route="/assistant")
agent.addSkill("datetime")
agent.addSkill("math")
console.log(agent.listSkills())  # ["datetime", "math"]
```
