[set-prompt-text]: /docs/sdks/reference/typescript/agents/agent-base/set-prompt-text

Retrieve the current prompt. Returns a string if
[`setPromptText()`][set-prompt-text] was used, or a
POM structure (list of dictionaries) if prompt sections were used. If neither was
explicitly set, returns a default prompt: `"You are {name}, a helpful AI assistant."`.

## **Returns**

`str | Record<string, unknown>[]` -- The current prompt in whatever format was configured.

## **Example**

```typescript {5}
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="support", route="/support")
agent.setPromptText("You are a customer support agent for Acme Corp.")
prompt = agent.getPrompt()
console.log(prompt)
```
