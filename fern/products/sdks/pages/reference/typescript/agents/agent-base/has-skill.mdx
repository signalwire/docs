Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skillName" type="string" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `True` if the skill is loaded, `False` otherwise.

## **Example**

```typescript {5-6}
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="assistant", route="/assistant")
agent.addSkill("datetime")
console.log(agent.hasSkill("datetime"))  # true
console.log(agent.hasSkill("web_search"))  # false
```
