Get the fully initialized Hono application instance. If the app has not been
created yet, this method initializes it with all routes, middleware, health checks,
and security headers -- the same configuration that `serve()` would create.

This is primarily used with deployment adapters like Mangum (AWS Lambda) or when
embedding the agent in a larger Hono application.

## **Returns**

`Hono` -- The configured Hono application instance with all agent endpoints
registered.

## **Examples**

### AWS Lambda with Mangum

```typescript {7}
from mangum import Mangum
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="lambda-agent")
agent.setPromptText("You are a helpful assistant.")

app = agent.getApp()
handler = Mangum(app)
```

### Embed in existing Hono app

```typescript {7}
from fastapi import FastAPI
import { AgentBase } from "@signalwire/sdk";

main_app = FastAPI()
agent = AgentBase(name="assistant", route="/agent")

main_app.mount("/ai", agent.getApp())
```
