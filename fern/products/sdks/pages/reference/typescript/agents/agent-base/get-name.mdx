Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`str` -- The agent name.

## **Example**

```typescript {4}
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="support", route="/support")
console.log(agent.getName())  # "support"
```
