Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`string` -- The agent name.

## **Example**

```typescript
import { AgentBase } from 'signalwire-agents';

const agent = new AgentBase({ name: 'support', route: '/support' });
console.log(agent.getName()); // "support"
```
