Retrieve the current post-prompt text.

## **Returns**

`string | undefined` -- The post-prompt text, or `undefined` if not set.

## **Example**

```typescript {5}
import { AgentBase } from "@signalwire/sdk";

agent = AgentBase(name="support", route="/support")
agent.setPostPrompt("Summarize this call as JSON with intent, resolution, and sentiment.")
post_prompt = agent.getPostPrompt()
if post_prompt:
    console.log(post_prompt)
```
