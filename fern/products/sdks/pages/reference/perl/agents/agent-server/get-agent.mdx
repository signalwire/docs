Retrieve a specific agent by its route. Returns `None` if no agent is registered at
the given path.

## **Parameters**

<ParamField path="route" type="$scalar" required={true} toc={true}>
  The route to look up (e.g., `"/sales"`). Leading slashes are added and trailing slashes
  are stripped automatically for matching.
</ParamField>

## **Returns**

[`SignalWire::Agent::AgentBase`][agentbase] | `None` -- The agent instance
registered at the route, or `None` if not found.

## **Example**

```perl {8}
use SignalWire::Server::AgentServer;
use SignalWire::Agent::AgentBase;

my $agent1 = SignalWire::Agent::AgentBase->new(name => "sales", route => "/sales");
my $agent2 = SignalWire::Agent::AgentBase->new(name => "support", route => "/support");
my $server = SignalWire::Server::AgentServer->new(port => 3000);
$server->register(agent1);
my $agent = $server->get_agent("/sales");
if (agent) {
    print "Found: ${$agent->get_name()}";
} else {
    print "No agent at /sales";
```
