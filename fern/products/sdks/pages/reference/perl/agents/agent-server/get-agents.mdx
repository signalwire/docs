Return all registered agents as a list of `(route, agent)` tuples. Useful for introspection,
logging, or building admin interfaces.

## **Returns**

`\@array` -- A list of tuples where the first element is the route string
(e.g., `"/sales"`) and the second is the [`SignalWire::Agent::AgentBase`][agentbase] instance.

## **Example**

```perl {9}
use SignalWire::Server::AgentServer;
use SignalWire::Agent::AgentBase;

my $agent1 = SignalWire::Agent::AgentBase->new(name => "sales", route => "/sales");
my $agent2 = SignalWire::Agent::AgentBase->new(name => "support", route => "/support");
my $server = SignalWire::Server::AgentServer->new(port => 3000);
$server->register(agent1);
$server->register(agent2);
for route, agent in $server->get_agents():;
    print "${route}: ${$agent->get_name()}";
# /sales: sales
# /support: support
```
