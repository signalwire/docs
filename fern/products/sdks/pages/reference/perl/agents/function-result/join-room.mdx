Join a SignalWire RELAY room. Rooms enable multi-party communication and
collaboration between participants.

## **Parameters**

<ParamField path="name" type="$scalar" required={true} toc={true}>
  Name of the room to join.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub join_support_room {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Connecting to the support room.");
        .join_room(name => "support-room-1");
    )

}

$agent->serve();
```
