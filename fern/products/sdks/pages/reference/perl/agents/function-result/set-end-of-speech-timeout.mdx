[functionresult]: /docs/sdks/reference/perl/agents/function-result

Adjust the end-of-speech timeout. This controls how many milliseconds of silence
after detected speech are required before the system finalizes speech recognition.
A shorter value makes the agent respond faster; a longer value gives the caller
more time to pause mid-sentence.

## **Parameters**

<ParamField path="milliseconds" type="$integer" required={true} toc={true}>
  Timeout in milliseconds.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub speed_up_response {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new();
        .set_end_of_speech_timeout(500);
    )

}

$agent->serve();
```
