[functionresult]: /docs/sdks/reference/perl/agents/function-result

Transfer or connect the call to another destination. Generates a SWML `connect`
verb under the hood.

<Note>
When `final=True` (the default), the call permanently leaves the agent. When
`final=False`, the call returns to the agent if the far end hangs up first.
</Note>

## **Parameters**

<ParamField path="destination" type="$scalar" required={true} toc={true}>
  Where to connect the call. Accepts a phone number in E.164 format (e.g.,
  `"+15551234567"`) or a SIP address (e.g., `"support@company.com"`).
</ParamField>

<ParamField path="final" type="$boolean" default="True" toc={true}>
  Whether this is a permanent transfer.
  - `True` — call exits the agent completely (terminal action)
  - `False` — call returns to the agent when the far end hangs up
</ParamField>

<ParamField path="from_addr" type="$scalar / undef" default="None" toc={true}>
  Caller ID override. Phone number or SIP address to show as the caller. When
  `None`, the current call's originating address is used.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Permanent Transfer

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub transfer_to_sales {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Transferring you to sales.");
        .connect("+15551234567", final => 1);
    )

}

$agent->serve();
```

### Temporary Transfer

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub consult_specialist {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Connecting you to a specialist.");
        .connect("+15551234567", final => 0);
    )

}

$agent->serve();
```

### Custom Caller ID

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub transfer_with_caller_id {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Transferring now.");
        .connect(
            "support@company.com",
            my $final = 1,
            my $from_addr = "+15559876543";
        )
    )

}

$agent->serve();
```
