[functionresult]: /docs/sdks/reference/perl/agents/function-result

Set or replace the response text after construction. The response is the text
the AI speaks back to the caller after the function executes.

## **Parameters**

<ParamField path="response" type="$scalar" required={true} toc={true}>
  Text for the AI to speak to the caller.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {12,14}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub check_order {
    my ($args, $raw_data) = @_;
    my $order_id = args.get("order_id");
    my $result = SignalWire::SWAIG::FunctionResult->new();
    if (order_id) {
        $result->set_response("Your order ${order_id} shipped yesterday.");
    } else {
        $result->set_response("I couldn't find that order $number->");
    return result;

}

$agent->serve();
```
