Remove all dynamically added speech recognition hints. This clears every hint
that was previously added via [`add_dynamic_hints()`][add-dynamic-hints].
Static hints defined on the agent at configuration time are not affected.

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub reset_context {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Context reset.");
        .clear_dynamic_hints();
    )

}

$agent->serve();
```
