**[FunctionResult][ref-functionresult].create_payment_prompt**(`for_situation`, `actions`, `card_type=None`, `error_type=None`) -> `\%hash`

Static helper that builds a payment prompt object for use with the `prompts`
parameter of [`pay()`][pay].
Payment prompts customize the TTS messages played during different stages of
payment collection.

## **Parameters**

<ParamField path="for_situation" type="$scalar" required={true} toc={true}>
  The payment stage this prompt applies to (e.g., `"payment-card-number"`,
  `"expiration-date"`, `"security-code"`, `"postal-code"`).
</ParamField>

<ParamField path="actions" type="\@arrayref]" required={true} toc={true}>
  List of prompt actions. Use [`create_payment_action()`][create-payment-action]
  to build each entry.
</ParamField>

<ParamField path="card_type" type="$scalar / undef" default="None" toc={true}>
  Space-separated card types this prompt applies to (e.g., `"visa mastercard"`).
</ParamField>

<ParamField path="error_type" type="$scalar / undef" default="None" toc={true}>
  Space-separated error types this prompt handles.
</ParamField>

## **Returns**

`\%hash` — prompt dictionary to pass in the `prompts` list of [`pay()`][pay].

## **Example**

```perl {9}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub collect_card {
    my ($args, $raw_data) = @_;
    my $prompt = FunctionResult.create_payment_prompt(
        my $for_situation = "payment-card-number",
        my $actions = [;
            FunctionResult.create_payment_action("Say", "Please enter your credit card $number->");
        ]
    )
    return (;
        SignalWire::SWAIG::FunctionResult->new("I'll collect your payment now.");
        .pay(
            my $payment_connector_url = "https://api.example.com/pay",
            my $charge_amount = "49.99",
            my $prompts = [prompt];
        )
    )

}

$agent->serve();
```
