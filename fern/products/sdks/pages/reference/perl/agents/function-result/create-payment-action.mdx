**[FunctionResult][ref-functionresult].create_payment_action**(`action_type`, `phrase`) -> `\%hash`

Static helper that builds a single action entry for a payment prompt.

## **Parameters**

<ParamField path="action_type" type="$scalar" required={true} toc={true}>
  Action type.

  - `"Say"` -- use text-to-speech
  - `"Play"` -- play an audio file URL
</ParamField>

<ParamField path="phrase" type="$scalar" required={true} toc={true}>
  Text to speak (when `action_type` is `"Say"`) or URL to play (when `action_type`
  is `"Play"`).
</ParamField>

## **Returns**

`\%hash` — action dictionary for use in
[`create_payment_prompt()`][create-payment-prompt].

## **Example**

```perl {9-10}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub collect_card {
    my ($args, $raw_data) = @_;
    my $say_action = FunctionResult.create_payment_action("Say", "Please enter your card $number->");
    my $play_action = FunctionResult.create_payment_action("Play", "https://example.com/card-prompt.mp3");
    my $prompt = FunctionResult.create_payment_prompt(
        my $for_situation = "payment-card-number",
        my $actions = [say_action];
    )
    return (;
        SignalWire::SWAIG::FunctionResult->new("I'll collect your payment now.");
        .pay(
            my $payment_connector_url = "https://api.example.com/pay",
            my $charge_amount = "19.99",
            my $prompts = [prompt];
        )
    )

}

$agent->serve();
```
