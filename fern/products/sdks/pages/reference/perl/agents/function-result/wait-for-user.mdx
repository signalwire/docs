[functionresult]: /docs/sdks/reference/perl/agents/function-result

Control how the agent pauses and waits for the user to speak. The arguments
are evaluated in priority order: `answer_first` takes precedence, then
`timeout`, then `enabled`. If none are provided, waiting is enabled with no
timeout.

## **Parameters**

<ParamField path="enabled" type="$scalar / undef" default="None" toc={true}>
  Explicitly enable (`True`) or disable (`False`) waiting for user input.
</ParamField>

<ParamField path="timeout" type="$scalar / undef" default="None" toc={true}>
  Number of seconds to wait for the user to speak before the agent continues.
</ParamField>

<ParamField path="answer_first" type="$boolean" default="False" toc={true}>
  When `True`, activates the special `"answer_first"` mode. This answers the
  call and waits for the user to speak before the agent begins its greeting.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Wait with Timeout

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub wait_for_confirmation {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Please respond when you're ready.");
        .wait_for_user(timeout => 30);
    )

}

$agent->serve();
```

### Disable Waiting

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub disable_wait {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Continuing without waiting.");
        .wait_for_user(enabled => 0);
    )

}

$agent->serve();
```
