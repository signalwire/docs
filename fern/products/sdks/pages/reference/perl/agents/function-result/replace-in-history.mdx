[functionresult]: /docs/sdks/reference/perl/agents/function-result

Control how this function call appears in the AI's conversation history. Use
this to remove sensitive information (credit card numbers, SSNs) or replace
verbose tool results with concise summaries.

<Note>
When `True` is passed, the entire tool-call/result pair is removed from history.
When a string is passed, the pair is replaced with that text as an assistant message.
</Note>

## **Parameters**

<ParamField path="text" type="$scalar | $boolean" default="True" toc={true}>
  - `True` — remove the tool-call and result pair from history entirely
  - A `str` — replace the pair with an assistant message containing this text
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Remove Sensitive Exchange

```perl {13}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub verify_identity {
    my ($args, $raw_data) = @_;
    my $ssn = args.get("ssn");
    # ... verify SSN ...
    return (;
        SignalWire::SWAIG::FunctionResult->new("I've verified your identity.");
        .replace_in_history(1);
    )

}

$agent->serve();
```

### Replace with Summary

```perl {13}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub process_payment {
    my ($args, $raw_data) = @_;
    my $card_number = args.get("card_number");
    # ... process payment ...
    return (;
        SignalWire::SWAIG::FunctionResult->new("Payment processed successfully.");
        .replace_in_history("Payment information was collected and processed.");
    )

}

$agent->serve();
```
