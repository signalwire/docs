Release another call from hold. Typically used after injecting a message into
the held caller's AI agent via [`rpc_ai_message()`][rpc-ai-message].

## **Parameters**

<ParamField path="call_id" type="$scalar" required={true} toc={true}>
  Call ID of the call to release from hold.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {13}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub release_caller {
    my ($args, $raw_data) = @_;
    my $caller_call_id = args.get("original_call_id");
    return (;
        SignalWire::SWAIG::FunctionResult->new("Returning you to the caller.");
        .rpc_ai_message(caller_call_id, "You can take their message now.");
        .rpc_ai_unhold(caller_call_id);
    )

}

$agent->serve();
```
