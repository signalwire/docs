End the call immediately.

<Warning>
This is a terminal action. Any actions chained **after** `hangup()` may not execute.
Always place `hangup()` last in the chain.
</Warning>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {12}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub end_call {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Thank you for calling. Goodbye!");
        .update_global_data({"call_ended": 1});
        .hangup();
    )

}

$agent->serve();
```
