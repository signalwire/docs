Adjust the speech event timeout. This controls how many milliseconds since the
last speech detection event to wait before finalizing recognition. This timeout
works better than [`set_end_of_speech_timeout()`][set-end-of-speech-timeout]
in noisy environments.

## **Parameters**

<ParamField path="milliseconds" type="$integer" required={true} toc={true}>
  Timeout in milliseconds.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {12}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub configure_timeouts {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new();
        .set_end_of_speech_timeout(800);
        .set_speech_event_timeout(5000);
    )

}

$agent->serve();
```
