Stop an active background call recording.

## **Parameters**

<ParamField path="control_id" type="$scalar / undef" default="None" toc={true}>
  Identifier of the recording to stop. If not provided, the most recent
  recording is stopped.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub stop_recording {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Recording stopped.");
        .stop_record_call(control_id => "main_recording");
    )

}

$agent->serve();
```
