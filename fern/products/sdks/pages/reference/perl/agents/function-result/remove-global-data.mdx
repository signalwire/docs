Remove one or more keys from the global session data.

## **Parameters**

<ParamField path="keys" type="$scalar | \@arrayref" required={true} toc={true}>
  A single key string or a list of key strings to remove from global data.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub clear_session_data {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Session data cleared.");
        .remove_global_data(["customer_id", "verified"]);
    )

}

$agent->serve();
```
