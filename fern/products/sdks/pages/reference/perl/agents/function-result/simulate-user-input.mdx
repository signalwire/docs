[functionresult]: /docs/sdks/reference/perl/agents/function-result

Inject text as if the user spoke it. The AI agent processes the injected text
exactly as it would process real speech input. Useful for driving automated
workflows or confirmations.

## **Parameters**

<ParamField path="text" type="$scalar" required={true} toc={true}>
  Text to inject as simulated user speech.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub auto_confirm {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new();
        .simulate_user_input("yes, I confirm");
    )

}

$agent->serve();
```
