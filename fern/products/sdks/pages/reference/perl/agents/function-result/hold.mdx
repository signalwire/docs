[functionresult]: /docs/sdks/reference/perl/agents/function-result

Put the call on hold. The caller hears hold music until the hold is released
or the timeout expires.

## **Parameters**

<ParamField path="timeout" type="$integer" default="300" toc={true}>
  Maximum hold duration in seconds. Clamped to the range 0--900 (15 minutes max).
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {7,11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub hold_for_agent {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Please hold while I find an available $agent->");
        .hold(timeout => 60);
    )

}

$agent->serve();
```
