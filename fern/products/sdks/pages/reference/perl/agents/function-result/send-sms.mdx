Send an SMS or MMS message to a phone number. At least one of `body` or `media`
must be provided. Generates a SWML `send_sms` verb under the hood.

## **Parameters**

<ParamField path="to_number" type="$scalar" required={true} toc={true}>
  Destination phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="from_number" type="$scalar" required={true} toc={true}>
  Sender phone number in E.164 format. Must be a number in your SignalWire project.
</ParamField>

<ParamField path="body" type="$scalar / undef" default="None" toc={true}>
  Text body of the message.
</ParamField>

<ParamField path="media" type="\@arrayref / undef" default="None" toc={true}>
  List of media URLs to include as MMS attachments.
</ParamField>

<ParamField path="tags" type="\@arrayref / undef" default="None" toc={true}>
  Tags to associate with the message for searching and filtering in the
  SignalWire dashboard.
</ParamField>

<ParamField path="region" type="$scalar / undef" default="None" toc={true}>
  Region to originate the message from.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Text Message

```perl {13}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub send_confirmation {
    my ($args, $raw_data) = @_;
    my $phone = args.get("phone_number");
    my $order_id = args.get("order_id");
    return (;
        SignalWire::SWAIG::FunctionResult->new("I've sent you a confirmation text.");
        .send_sms(
            my $to_number = phone,
            my $from_number = "+15559876543",
            my $body = "Your order ${order_id} has been confirmed!";
        )
    )

}

$agent->serve();
```

### MMS with Media

```perl {13}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub send_receipt {
    my ($args, $raw_data) = @_;
    my $phone = args.get("phone_number");
    my $receipt_url = args.get("receipt_url");
    return (;
        SignalWire::SWAIG::FunctionResult->new("I've sent your receipt.");
        .send_sms(
            my $to_number = phone,
            my $from_number = "+15559876543",
            my $body = "Here's your receipt:",
            my $media = [receipt_url];
        )
    )

}

$agent->serve();
```
