[functionresult]: /docs/sdks/reference/perl/agents/function-result

Append multiple raw actions at once. Each action dictionary is added to the
action list in order.

## **Parameters**

<ParamField path="actions" type="\@arrayref]" required={true} toc={true}>
  List of action dictionaries to append. Each dictionary should map a single
  action name to its payload.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub batch_actions {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Processing your request.");
        .add_actions([;
            {"set_global_data": {"key": "value"}},
            {"hold": 60},
            {"say": "Please wait."}
        ])
    )

}

$agent->serve();
```
