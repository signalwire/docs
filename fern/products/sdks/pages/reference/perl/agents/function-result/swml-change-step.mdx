Transition to a different step within the current conversation context. Steps
are defined using
[`SignalWire::Contexts::ContextBuilder`][contextbuilder] and
represent discrete stages in a workflow (e.g., "greeting", "verification",
"checkout").

## **Parameters**

<ParamField path="step_name" type="$scalar" required={true} toc={true}>
  Name of the step to transition to.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub move_to_checkout {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Moving to checkout.");
        .swml_change_step("checkout");
    )

}

$agent->serve();
```
