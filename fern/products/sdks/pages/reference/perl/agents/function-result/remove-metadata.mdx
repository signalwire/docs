Remove one or more keys from the current function's metadata store.

## **Parameters**

<ParamField path="keys" type="$scalar | \@arrayref" required={true} toc={true}>
  A single key string or a list of key strings to remove from metadata.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub clear_metadata {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Metadata cleared.");
        .remove_metadata(["timestamp", "user_id"]);
    )

}

$agent->serve();
```
