Play an audio or video file in the background while the conversation continues.

## **Parameters**

<ParamField path="filename" type="$scalar" required={true} toc={true}>
  URL of the audio or video file to play (e.g., `"https://example.com/hold-music.mp3"`).
</ParamField>

<ParamField path="wait" type="$boolean" default="False" toc={true}>
  When `True`, suppresses the agent's attention-getting behavior during playback.
  The agent will not try to re-engage the user while the file is playing.
</ParamField>

## **Returns**

[`SignalWire::SWAIG::FunctionResult`][functionresult] — self, for chaining.

## **Example**

```perl {11}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");
$agent->set_prompt_text("You are a helpful assistant.");

sub play_hold_music {
    my ($args, $raw_data) = @_;
    return (;
        SignalWire::SWAIG::FunctionResult->new("Please hold.");
        .play_background_file(
            "https://example.com/hold-music.mp3",
            my $wait = 1;
        )
    )

}

$agent->serve();
```
