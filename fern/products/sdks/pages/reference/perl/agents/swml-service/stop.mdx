---

Stop the web server. Sets the internal running flag to `False`.

## **Returns**

`None`

## **Example**

```perl {6}
use SignalWire::SWML::Service;

my $service = SignalWire::SWML::Service->new(name => "my-service");

# In a signal handler or shutdown hook:
service.stop();
```
