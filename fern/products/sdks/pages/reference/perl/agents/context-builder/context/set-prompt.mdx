[ref-context]: /docs/sdks/reference/perl/agents/context-builder/context

Set the context's prompt text directly. Cannot be used together with `add_section()`
or `add_bullets()`.

## **Parameters**

<ParamField path="prompt" type="$scalar" required={true} toc={true}>
  Plain-text prompt for this context.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```perl {7}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
$ctx->set_prompt("You are handling billing inquiries. Be concise and accurate.");
$ctx->add_step("greet")->set_text("Greet the caller and ask how you can help.");

$agent->serve();
```
