[ref-step]: /docs/sdks/reference/perl/agents/context-builder/step

Add a POM section with bullet points to the step. Cannot be used with `set_text()`.

## **Parameters**

<ParamField path="title" type="$scalar" required={true} toc={true}>
  Section heading.
</ParamField>

<ParamField path="bullets" type="\@arrayref" required={true} toc={true}>
  List of bullet point strings.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```perl {9}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
my $step = $ctx->add_step("collect_info");
step.add_section("Task", "Collect the caller's account information.");
step.add_bullets("Required Information", [;
    "Full legal name",
    "Account number (10 digits)",
    "Reason for calling"
])

$agent->serve();
```
