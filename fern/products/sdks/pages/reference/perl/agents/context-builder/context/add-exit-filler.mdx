Add exit fillers for a specific language.

## **Parameters**

<ParamField path="language_code" type="$scalar" required={true} toc={true}>
  Language code (e.g., `"en-US"`, `"es"`) or `"default"` for a catch-all.
</ParamField>

<ParamField path="fillers" type="\@arrayref" required={true} toc={true}>
  List of filler phrases.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```perl {9}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
contexts.add_context("default").add_step("menu").set_text("Ask what the caller needs.");

my $support = contexts.add_context("support");
support.add_exit_filler("en-US", [;
    "Thank you for contacting support.",
    "Glad I could help."
])
support.add_step("diagnose").set_text("Understand the customer's issue.");

$agent->serve();
```
