Set whether the conversation should end after this step completes.

## **Parameters**

<ParamField path="end" type="$boolean" required={true} toc={true}>
  Whether to end the conversation after this step.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```perl {10}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
$ctx->add_step("greet")->set_text("Welcome the caller.");
my $farewell = $ctx->add_step("farewell");
farewell.set_text("Thank you for calling. Goodbye!");
farewell.set_end(1);

$agent->serve();
```
