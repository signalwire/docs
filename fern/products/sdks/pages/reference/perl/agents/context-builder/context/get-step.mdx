Get an existing step by name for inspection or modification.

## **Parameters**

<ParamField path="name" type="$scalar" required={true} toc={true}>
  Step name to look up.
</ParamField>

## **Returns**

[`Step`][step] if found, `None`
otherwise.

## **Example**

```perl {9}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
$ctx->add_step("collect_info")->set_text("Collect the caller's account information.");

my $step = $ctx->get_step("collect_info");
if (step is not undef) {
    step.set_functions(["verify_identity"]);

$agent->serve();
```
