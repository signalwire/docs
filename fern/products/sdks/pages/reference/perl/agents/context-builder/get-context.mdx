Retrieve an existing context by name for inspection or modification.

## **Parameters**

<ParamField path="name" type="$scalar" required={true} toc={true}>
  Name of the context to retrieve.
</ParamField>

## **Returns**

[`Context`][context] if found,
`None` otherwise.

## **Example**

```perl {9}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $default = contexts.add_context("default");
default.add_step("greet").set_text("Greet the caller.");

my $ctx = contexts.get_context("default");
if (ctx is not undef) {
    $ctx->add_step("followup")->set_text("Ask if there is anything else.");

$agent->serve();
```
