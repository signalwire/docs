Convert all contexts to a dictionary for SWML generation. Calls `validate()`
first and raises `croak` if the configuration is invalid.

## **Returns**

`\%hash` -- A mapping of context names to their serialized dictionaries,
ready for inclusion in the SWML document.

## **Example**

```perl {9}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
$ctx->add_step("greet")->set_text("Hello!");

my $data = contexts.to_dict();
# {"default": {"steps": [{"name": "greet", "text": "Hello!"}]}}

$agent->serve();
```
