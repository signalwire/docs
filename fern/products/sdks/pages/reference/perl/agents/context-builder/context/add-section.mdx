[ref-context]: /docs/sdks/reference/perl/agents/context-builder/context

Add a POM section to the context prompt. Cannot be used together with `set_prompt()`.

## **Parameters**

<ParamField path="title" type="$scalar" required={true} toc={true}>
  Section heading.
</ParamField>

<ParamField path="body" type="$scalar" required={true} toc={true}>
  Section body text.
</ParamField>

## **Returns**

[`Context`][ref-context] -- Self for method chaining.

## **Example**

```perl {7-8}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
$ctx->add_section("Role", "You are a billing specialist.");
$ctx->add_section("Guidelines", "Always verify account ownership first.");
$ctx->add_step("greet")->set_text("Greet the caller and ask how you can help.");

$agent->serve();
```
