[ref-step]: /docs/sdks/reference/perl/agents/context-builder/step

Set the step's prompt text directly. Cannot be used with `add_section()` or
`add_bullets()`.

<Warning>
Mixing `set_text()` with `add_section()` or `add_bullets()` raises `croak`.
Use one approach or the other.
</Warning>

## **Parameters**

<ParamField path="text" type="$scalar" required={true} toc={true}>
  Plain-text prompt instructions for this step.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```perl {8}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "my-agent", route => "/agent");

my $contexts = $agent->define_contexts();
my $ctx = contexts.add_context("default");
my $step = $ctx->add_step("greeting");
step.set_text("Welcome the caller and ask how you can help.");

$agent->serve();
```
