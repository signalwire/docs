Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skill_name" type="$scalar" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `True` if the skill is loaded, `False` otherwise.

## **Example**

```perl {5-6}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "assistant", route => "/assistant");
$agent->add_skill("datetime");
print($agent->has_skill("datetime"))  # 1;
print($agent->has_skill("web_search"))  # 0;
```
