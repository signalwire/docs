[ref-agentbase]: /docs/sdks/reference/perl/agents/agent-base

Add a subsection to an existing section. If the parent section does not exist, it is
created automatically.

## **Parameters**

<ParamField path="parent_title" type="$scalar" required={true} toc={true}>
  Title of the parent section.
</ParamField>

<ParamField path="title" type="$scalar" required={true} toc={true}>
  Subsection heading.
</ParamField>

<ParamField path="body" type="$scalar" default="" toc={true}>
  Subsection body text.
</ParamField>

<ParamField path="bullets" type="\@arrayref / undef" toc={true}>
  Subsection bullet points.
</ParamField>

## **Returns**

[`SignalWire::Agent::AgentBase`][ref-agentbase] -- Returns self for method chaining.

## **Example**

```perl {6,8}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "support", route => "/support");
$agent->prompt_add_section("Capabilities",
    body => "You can help with the following:");
$agent->prompt_add_subsection("Capabilities", "Orders",
    body => "Look up order status, process returns and exchanges.");
$agent->prompt_add_subsection("Capabilities", "Billing",
    body => "Check account balances and explain recent charges.");
$agent->serve();
```
