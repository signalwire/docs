Retrieve the current post-prompt text.

## **Returns**

`$scalar / undef` -- The post-prompt text, or `None` if not set.

## **Example**

```perl {5}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "support", route => "/support");
$agent->set_post_prompt("Summarize this call as JSON with intent, resolution, and sentiment.");
my $post_prompt = $agent->get_post_prompt();
if (post_prompt) {
    print post_prompt;
```
