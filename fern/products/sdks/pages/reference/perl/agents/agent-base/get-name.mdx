Get the agent's display name as set at construction time.

## **Parameters**

None.

## **Returns**

`str` -- The agent name.

## **Example**

```perl {4}
use SignalWire::Agent::AgentBase;

my $agent = SignalWire::Agent::AgentBase->new(name => "support", route => "/support");
print($agent->get_name())  # "support";
```
