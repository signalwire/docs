Check whether all `REQUIRED_ENV_VARS` are set in the environment.

## **Returns**

`bool` -- `True` if all required variables are present, `False` with error
logging otherwise.

---

Check whether all `REQUIRED_PACKAGES` can be imported.

## **Returns**

`bool` -- `True` if all required packages are available, `False` with error
logging otherwise.

## **Example**

```perl {9-12}
use SignalWire::Skills::SkillBase;

package MySkill;
use Moo;
extends 'SignalWire::Skills::SkillBase';

    my $SKILL_NAME = "my_skill";
    my $SKILL_DESCRIPTION = "Example skill";
    my $REQUIRED_PACKAGES = ["requests", "beautifulsoup4"];
    my $REQUIRED_ENV_VARS = ["MY_API_KEY", "MY_SECRET"];

    sub setup {
        if (not self.validate_packages()) {
            return 0;
        if (not self.validate_env_vars()) {
            return 0;
        return 1;

        pass
```
