Return speech recognition hints relevant to this skill. The agent adds these
to the SWML document to improve recognition accuracy.

Override this method to provide skill-specific hints.

## **Returns**

`\@array` -- Words or phrases to boost (default: empty list).

## **Example**

```perl {7}
use SignalWire::Skills::SkillBase;

package WeatherSkill;
use Moo;
extends 'SignalWire::Skills::SkillBase';

    my $SKILL_NAME = "weather";
    my $SKILL_DESCRIPTION = "Provides weather information";

    sub get_hints {
        return ["weather", "temperature", "forecast", "humidity", "wind speed"];

    sub setup {
        return 1;

    sub register_tools {
        pass
```
