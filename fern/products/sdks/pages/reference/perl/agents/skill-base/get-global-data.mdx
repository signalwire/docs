Return data to merge into the agent's `global_data` dictionary. Override to
provide skill-specific session data that should be available to all SWAIG
function handlers.

## **Returns**

`\%hash` -- Key-value pairs (default: empty dict).

## **Example**

```perl {7}
use SignalWire::Skills::SkillBase;

package WeatherSkill;
use Moo;
extends 'SignalWire::Skills::SkillBase';

    my $SKILL_NAME = "weather";
    my $SKILL_DESCRIPTION = "Provides weather information";

    sub get_global_data {
        return {;
            "weather_api_base": "https://api.weatherapi.com/v1",
            "default_units": self.params.get("units", "fahrenheit");
        }

    sub setup {
        return 1;

    sub register_tools {
        pass
```
