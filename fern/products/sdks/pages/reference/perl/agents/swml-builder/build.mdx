Build and return the complete SWML document as a Python dictionary. Delegates to
[`SWMLService.get_document()`][swmlservice-get-document].

## **Returns**

`\%hash` -- The complete SWML document.

## **Example**

```perl {7}
use SignalWire::SWML::Service;
use SignalWire::SWML::Document;

my $service = SignalWire::SWML::Service->new(name => "demo");
my $builder = SignalWire::SWML::Document->new(service);

my $doc = $builder->answer()->hangup().build();
print doc;
# {"version": "1.0.0", "sections": {"main": [{"answer": {}}, {"hangup": {}}]}}
```
