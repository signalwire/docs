Add an `answer` verb to the main section. Answers an incoming call.

## **Parameters**

<ParamField path="max_duration" type="$scalar / undef" toc={true}>
  Maximum call duration in seconds. The call is automatically hung up after this
  duration elapses.
</ParamField>

<ParamField path="codecs" type="$scalar / undef" toc={true}>
  Comma-separated list of audio codecs to use for the call.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```perl {9}
use SignalWire::SWML::Service;
use SignalWire::SWML::Document;

my $service = SignalWire::SWML::Service->new(name => "my-service");
my $builder = SignalWire::SWML::Document->new(service);

my $doc = (
    builder
    .answer(max_duration => 3600);
    .play(url => "https://example.com/welcome.mp3");
    .build();
)
print doc;
```
