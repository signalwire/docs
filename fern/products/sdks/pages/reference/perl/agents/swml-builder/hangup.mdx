Add a `hangup` verb to the main section. Ends the current call.

## **Parameters**

<ParamField path="reason" type="$scalar / undef" toc={true}>
  An optional reason string for the hangup, included in call logs.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```perl {11}
use SignalWire::SWML::Service;
use SignalWire::SWML::Document;

my $service = SignalWire::SWML::Service->new(name => "ivr");
my $builder = SignalWire::SWML::Document->new(service);

my $doc = (
    builder
    .answer();
    .play(url => "https://example.com/goodbye.mp3");
    .hangup(reason => "Call completed");
    .build();
)
print doc;
```
