Build and render the complete SWML document as a JSON string. Delegates to
[`SWMLService.render_document()`][swmlservice-render-document].

## **Returns**

`str` -- The SWML document serialized as JSON.

## **Example**

```perl {7}
use SignalWire::SWML::Service;
use SignalWire::SWML::Document;

my $service = SignalWire::SWML::Service->new(name => "demo");
my $builder = SignalWire::SWML::Document->new(service);

my $json_str = $builder->answer()->ai(prompt_text => "Hello").render();
print json_str;
```
