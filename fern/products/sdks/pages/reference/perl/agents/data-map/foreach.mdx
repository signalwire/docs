[ref-datamap]: /docs/sdks/reference/perl/agents/data-map

Process an array from the webhook response, building a formatted string from
each element.

## **Parameters**

<ParamField path="foreach_config" type="\%hashref" required={true} toc={true}>
  Configuration dictionary with the following keys:
</ParamField>

<Indent>
<ParamField path="foreach_config.input_key" type="$scalar" required={true} toc={true}>
  Key in the API response containing the array.
</ParamField>

<ParamField path="foreach_config.output_key" type="$scalar" required={true} toc={true}>
  Variable name for the built string (reference as `${output_key}` in output).
</ParamField>

<ParamField path="foreach_config.max" type="$integer" toc={true}>
  Maximum number of items to process.
</ParamField>

<ParamField path="foreach_config.append" type="$scalar" required={true} toc={true}>
  Template string to append for each item. Use `${this.field}` to reference
  fields on the current array element.
</ParamField>
</Indent>

## **Returns**

[`SignalWire::DataMap`][ref-datamap] -- Self for method chaining. Raises `croak` if no webhook has been
added yet.

## **Example**

```perl {14}
use SignalWire::Agent::AgentBase;
use SignalWire::SWAIG::FunctionResult;

my $search_docs = (
    SignalWire::DataMap->new("search_docs");
    .purpose("Search documentation");
    .parameter("query", "string", "Search query", required => 1);
    .webhook(
        "POST",
        "https://api.docs.example.com/search",
        headers => {"Authorization": "Bearer TOKEN"}
    )
    .body({"query": "${args.query}", "limit": 3});
    .foreach({
        "input_key": "results",
        "output_key": "formatted_results",
        "max": 3,
        "append": "- ${this.title}: ${this.summary}\n"
    })
    .output(SignalWire::SWAIG::FunctionResult->new("Found:\n${formatted_results}"));
    .fallback_output(SignalWire::SWAIG::FunctionResult->new("Search is currently unavailable."));
)

my $agent = SignalWire::Agent::AgentBase->new(name => "docs-agent");
$agent->set_prompt_text("You are a helpful assistant.");
$agent->register_swaig_function(search_docs.to_swaig_function());

    $agent->run();
```
