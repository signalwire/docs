List number groups in the project.

## **Parameters**

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of group objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->number_groups->list();
for my $group (@{$result->{"data"} // []}) {
    print group["name"], group.get("id");
```
