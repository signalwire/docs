Update a number group. Uses PUT to replace fields.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Fields to update (e.g., `name`).
</ParamField>

## **Returns**

`dict` -- The updated group object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->number_groups->update("group-id", name => "Support Numbers");
print updated["name"];
```
