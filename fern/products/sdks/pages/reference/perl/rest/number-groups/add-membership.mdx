Add a phone number to a group.

## **Parameters**

<ParamField path="group_id" type="$scalar" required={true} toc={true}>
  The group resource ID.
</ParamField>

<ParamField path="phone_number_id" type="$scalar" toc={true}>
  The phone number resource ID to add to the group.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional membership fields.
</ParamField>

## **Returns**

`dict` -- The newly created membership object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $membership = $client->number_groups->add_membership(
    "group-id",
    phone_number_id => "phone-number-id",
)
print "Membership ID:", membership.get("id");
```
