Retrieve a specific membership.

<Note>
This method uses the top-level membership endpoint and takes a membership ID
directly, not a group ID.
</Note>

## **Parameters**

<ParamField path="membership_id" type="$scalar" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`dict` -- The membership object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $membership = $client->number_groups->get_membership("membership-id");
print membership.get("phone_number_id");
```
