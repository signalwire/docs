Create a new number group.

## **Parameters**

<ParamField path="name" type="$scalar" toc={true}>
  Group name.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional group configuration.
</ParamField>

## **Returns**

`dict` -- The newly created group object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $group = $client->number_groups->create(name => "Sales Numbers");
print "Group ID:", group["id"];
```
