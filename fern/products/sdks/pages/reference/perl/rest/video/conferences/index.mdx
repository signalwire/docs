Manage video conferences with full CRUD operations, token listing, and stream
management. Uses PUT for updates (full replacement).

Access via `client.video.conferences` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $conferences = $client->video->conferences->list();
```
