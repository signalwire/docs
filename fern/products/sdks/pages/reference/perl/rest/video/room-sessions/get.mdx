Retrieve a single room session by its unique identifier.

## **Parameters**

<ParamField path="session_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

## **Returns**

`dict` -- The room session resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $session = $client->video->room_sessions->{"session-id"}
```
