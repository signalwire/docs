Update a stream. Uses PUT for full replacement, so all fields must be provided
in the request body.

<Note>
This method performs a full replacement (PUT), not a partial update (PATCH).
Include all fields you want to keep in the request body.
</Note>

## **Parameters**

<ParamField path="stream_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the stream.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated stream.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $stream = $client->video->streams->update(
    "stream-id",
    url => "rtmp://new-destination.example.com/key";
)
```
