List events associated with a recording (e.g., recording started, paused,
completed).

## **Parameters**

<ParamField path="recording_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the recording.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- Paginated list of recording events.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $events = $client->video->room_recordings->list_events("recording-id");
```
