List video rooms in the project.

## **Parameters**

<ParamField path="page_size" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering and pagination.

## **Returns**

`dict` -- Paginated response containing video rooms.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $rooms = $client->video->rooms->list();
```
