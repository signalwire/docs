Top-level stream management for retrieving, updating, and deleting individual
streams regardless of which room or conference they belong to.

Access via `client.video.streams` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $stream = $client->video->streams->{"stream-id"}
```
