List active streams for a video room.

## **Parameters**

<ParamField path="room_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- List of streams in the room.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $streams = $client->video->rooms->list_streams("room-id");
```
