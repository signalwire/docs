List recordings created during a room session.

## **Parameters**

<ParamField path="session_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the room session.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- Paginated list of session recordings.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $recordings = $client->video->room_sessions->list_recordings("session-id");
```
