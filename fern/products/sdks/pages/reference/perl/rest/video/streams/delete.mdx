Delete a stream.

## **Parameters**

<ParamField path="stream_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the stream to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->video->streams->delete("stream-id");
```
