List tokens associated with a conference.

## **Parameters**

<ParamField path="conference_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the conference.
</ParamField>

Additional keyword arguments are passed as query parameters for filtering.

## **Returns**

`dict` -- List of conference tokens.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $tokens = $client->video->conferences->list_conference_tokens("conference-id");
```
