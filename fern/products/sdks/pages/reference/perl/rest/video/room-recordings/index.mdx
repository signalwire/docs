Manage video room recordings. Supports listing, retrieval, deletion, and
querying recording-level events.

Access via `client.video.room_recordings` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $recordings = $client->video->room_recordings->list();
```
