Retrieve a specific conference token.

## **Parameters**

<ParamField path="token_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the conference token.
</ParamField>

## **Returns**

`dict` -- The conference token resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $token = $client->video->conference_tokens->{"token-id"}
```
