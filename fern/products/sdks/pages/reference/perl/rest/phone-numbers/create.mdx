Purchase a phone number. The exact keyword arguments depend on the number being
purchased (typically obtained from a prior `search()` call).

## **Parameters**

<ParamField path="**kwargs" type="\%hashref" required={true} toc={true}>
  Number details. Common fields include `number` (the E.164 phone number to
  purchase).
</ParamField>

## **Returns**

`dict` -- The newly purchased phone number object.

## **Example**

```perl {13}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

# Purchase the first available number from a search
my $available = $client->phone_numbers->search(area_code => "512", quantity => 1);
my $numbers = $available->{"data"} // [];
if (numbers) {
    my $purchased = $client->phone_numbers->create(number => numbers[0]["number"]);
    print "Purchased:", purchased["number"];
```
