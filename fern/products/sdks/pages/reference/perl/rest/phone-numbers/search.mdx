Search for available phone numbers that can be purchased.

## **Parameters**

<ParamField path="area_code" type="$scalar" toc={true}>
  Filter by area code (e.g., `"512"`).
</ParamField>

<ParamField path="quantity" type="$integer" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**params" type="\%hashref" toc={true}>
  Additional query parameters such as `region`, `city`, `country`, or
  `number_type`.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of available number objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $available = $client->phone_numbers->search(area_code => "503", quantity => 5);
for my $number (@{$available->{"data"} // []}) {
    print number["number"], $number->{"region"};
```
