List phone numbers owned by the project.

## **Parameters**

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters to filter and paginate results (e.g., `page_size`,
  `page_token`).
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of phone number objects and
pagination links.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->phone_numbers->list(page_size => 10);
for my $number (@{$result->{"data"} // []}) {
    print number["number"], $number->{"name"};
```
