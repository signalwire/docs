Retrieve details for a specific phone number.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

## **Returns**

`dict` -- The phone number object.

## **Example**

```perl {9-10}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $number = $client->phone_numbers->{"phone-number-id"}
print number["number"], $number->{"name"};
```
