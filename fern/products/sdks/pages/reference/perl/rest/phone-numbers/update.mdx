Update configuration for a phone number. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The phone number resource ID.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Fields to update (e.g., `name`, `call_handler`, `call_request_url`).
</ParamField>

## **Returns**

`dict` -- The updated phone number object.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->phone_numbers->update("phone-number-id", name => "Support Line");
```
