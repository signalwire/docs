Retrieve a specific short code.

## **Parameters**

<ParamField path="short_code_id" type="$scalar" required={true} toc={true}>
  The short code resource ID.
</ParamField>

## **Returns**

`dict` -- The short code object.

## **Example**

```perl {9-10}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $code = $client->short_codes->{"short-code-id"}
print code.get("short_code");
```
