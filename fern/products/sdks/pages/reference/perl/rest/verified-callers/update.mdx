Update a verified caller ID. Uses PUT.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Fields to update (e.g., `friendly_name`).
</ParamField>

## **Returns**

`dict` -- The updated caller ID object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->verified_callers->update("caller-id", friendly_name => "Main Office");
print "Updated: ${updated['id']}";
```
