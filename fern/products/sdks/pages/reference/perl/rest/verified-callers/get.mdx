Retrieve a specific verified caller ID.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`dict` -- The caller ID object.

## **Example**

```perl {9-10}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $caller = $client->verified_callers->{"caller-id"}
print caller.get("phone_number"), caller.get("verification_state");
```
