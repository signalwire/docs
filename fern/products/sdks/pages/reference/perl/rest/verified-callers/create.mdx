Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="phone_number" type="$scalar" toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional parameters (e.g., `friendly_name`).
</ParamField>

## **Returns**

`dict` -- The newly created caller ID object with its verification state.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $caller = $client->verified_callers->create(phone_number => "+15551234567");
print "Verification state:", caller.get("verification_state");
```
