Submit the verification code received during the verification call or SMS.

## **Parameters**

<ParamField path="caller_id" type="$scalar" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="verification_code" type="$scalar" toc={true}>
  The code received during the verification call or SMS.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional verification parameters.
</ParamField>

## **Returns**

`dict` -- The updated caller ID object with verified state.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->verified_callers->submit_verification(
    "caller-id",
    verification_code => "123456",
)
print "State:", $result->{"verification_state"};
```
