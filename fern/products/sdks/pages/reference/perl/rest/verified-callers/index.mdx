Manage verified caller IDs for your SignalWire project. Before using a phone
number as a caller ID for outbound calls, it must be verified. This resource
provides full CRUD (with PUT for updates) plus a two-step verification flow.

Access via `client.verified_callers` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $callers = $client->verified_callers->list();
for my $caller (@{callers.get("data", [])}) {
    print caller["id"], caller.get("phone_number");
```
