List chunks belonging to a document. Chunks are the individual text segments
created when a document is processed for vector search.

## **Parameters**

<ParamField path="document_id" type="$scalar" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters for pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of chunk objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $chunks = $client->datasphere->documents->list_chunks("document-id");
for my $chunk (@{chunks.get("data", [])}) {
    print chunk.get("id"), chunk.get("text")[:80];
```
