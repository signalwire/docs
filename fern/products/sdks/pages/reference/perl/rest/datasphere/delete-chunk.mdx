Delete a specific chunk from a document.

## **Parameters**

<ParamField path="document_id" type="$scalar" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunk_id" type="$scalar" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->datasphere->documents->delete_chunk("document-id", "chunk-id");
```
