Update an existing document.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Fields to update (e.g., `name`, `content`, `metadata`).
</ParamField>

## **Returns**

`dict` -- The updated document object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->datasphere->documents->update(
    "document-id",
    name => "Updated FAQ",
)
print updated.get("name");
```
