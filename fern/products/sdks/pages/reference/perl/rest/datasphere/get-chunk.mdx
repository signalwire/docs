Retrieve a specific chunk from a document.

## **Parameters**

<ParamField path="document_id" type="$scalar" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="chunk_id" type="$scalar" required={true} toc={true}>
  The chunk resource ID.
</ParamField>

## **Returns**

`dict` -- The chunk object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $chunk = $client->datasphere->documents->get_chunk("document-id", "chunk-id");
print chunk.get("text");
```
