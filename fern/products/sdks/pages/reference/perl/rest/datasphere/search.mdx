Perform a semantic search across all documents in the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="query_string" type="$scalar" required={true} toc={true}>
  The natural-language search query.
</ParamField>

<ParamField path="count" type="$integer" toc={true}>
  Maximum number of results to return.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional search parameters (e.g., `tags`, `document_id` to narrow scope).
</ParamField>

## **Returns**

`dict` -- Search results with matching document chunks ranked by relevance.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $results = $client->datasphere->documents->search(
    query_string => "How do I reset my password?",
    count => 5,
)
for my $chunk (@{results.get("data", [])}) {
    print chunk.get("text"), "score:", chunk.get("score");
```
