List SIP endpoints associated with a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="page_size" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="$scalar" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing SIP endpoints for the subscriber.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $response = $client->fabric->subscribers->list_sip_endpoints("subscriber-id");
for my $ep (@{$response->{"data"} // []}) {
    print "${ep['username']}: ${ep['id']}";
```
