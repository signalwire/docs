Manage SWML webhook resources. Standard CRUD with PATCH updates and address listing.

Access via `client.fabric.swml_webhooks` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $items = $client->fabric->swml_webhooks->list();
```
