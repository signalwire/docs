List conference room resources in the project.

## **Parameters**

<ParamField path="page_size" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="$scalar" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing conference room resources.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $response = $client->fabric->conference_rooms->list();
for my $room (@{$response->{"data"} // []}) {
    print "${room['name']}: ${room['id']}";
```
