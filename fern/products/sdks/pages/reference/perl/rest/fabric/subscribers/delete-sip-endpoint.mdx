Delete a SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SIP endpoint to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success (HTTP 204).

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->fabric->subscribers->delete_sip_endpoint("subscriber-id", "endpoint-id");
print "SIP endpoint deleted";
```
