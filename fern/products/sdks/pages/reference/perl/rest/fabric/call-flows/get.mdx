Retrieve a single call flow resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

## **Returns**

`dict` -- The call flow resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $flow = $client->fabric->call_flows->{"flow-id"}
print "Call Flow: ${flow['name']}, ID: ${flow['id']}";
```
