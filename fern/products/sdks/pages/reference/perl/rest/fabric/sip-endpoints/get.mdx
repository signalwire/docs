Retrieve a single SIP endpoint resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- The SIP endpoint resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $endpoint = $client->fabric->sip_endpoints->{"endpoint-id"}
print "Sip Endpoint: ${endpoint['name']}, ID: ${endpoint['id']}";
```
