Retrieve a single SWML script resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SWML script.
</ParamField>

## **Returns**

`dict` -- The SWML script resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $item = $client->fabric->swml_scripts->{"resource-id"}
print "Swml Script: ${item['name']}, ID: ${item['id']}";
```
