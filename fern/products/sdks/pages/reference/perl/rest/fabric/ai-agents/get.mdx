Retrieve a single AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

## **Returns**

`dict` -- The AI agent resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $agent = $client->fabric->ai_agents->{"agent-id"}
print "AI Agent: ${agent['name']}, ID: ${agent['id']}";
```
