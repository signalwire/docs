Partially update a SIP endpoint for a subscriber. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="subscriber_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated SIP endpoint.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->subscribers->update_sip_endpoint(
    "subscriber-id", "endpoint-id", username => "new-username",
)
print "Updated: ${updated['username']}";
```
