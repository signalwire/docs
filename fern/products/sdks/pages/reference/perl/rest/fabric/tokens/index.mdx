Create subscriber, guest, invite, and embed tokens for Fabric authentication.

Access via `client.fabric.tokens` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $token = $client->fabric->tokens->create_subscriber_token(subscriber_id => "sub-id");
```
