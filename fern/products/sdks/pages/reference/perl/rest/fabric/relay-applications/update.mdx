Replace a RELAY application resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated RELAY application resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->relay_applications->update("resource-id", name => "updated-name");
print "Updated: ${updated['name']}";
```
