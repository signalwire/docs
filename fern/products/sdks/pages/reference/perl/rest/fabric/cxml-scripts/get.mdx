Retrieve a single cXML script resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`dict` -- The cXML script resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $item = $client->fabric->cxml_scripts->{"resource-id"}
print "Cxml Script: ${item['name']}, ID: ${item['id']}";
```
