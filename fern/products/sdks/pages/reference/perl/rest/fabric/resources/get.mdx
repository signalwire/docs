Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`dict` -- The Fabric resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $resource = $client->fabric->resources->{"resource-id"}
print "${resource['type']}: ${resource['name']}, ID: ${resource['id']}";
```
