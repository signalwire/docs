Replace a cXML script resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated cXML script resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->cxml_scripts->update("resource-id", name => "updated-name");
print "Updated: ${updated['name']}";
```
