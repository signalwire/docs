Partially update a cXML webhook resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated cXML webhook resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->cxml_webhooks->update("webhook-id", name => "updated-name");
print "Updated: ${updated['name']}";
```
