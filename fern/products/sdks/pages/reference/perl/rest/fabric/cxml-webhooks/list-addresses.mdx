List addresses associated with a cXML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

## **Returns**

`dict` -- List of addresses assigned to this resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $addresses = $client->fabric->cxml_webhooks->list_addresses("webhook-id");
for my $addr (@{addresses.get("data", [])}) {
    print "${addr['display_name']}: ${addr['id']}";
```
