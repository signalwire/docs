Read-only access to fabric addresses. Addresses represent the endpoints (phone numbers,
SIP URIs, domains) that route traffic to fabric resources.

Access via `client.fabric.addresses` on a [`SignalWire::REST::RestClient`][restclient] instance.

### list

List all fabric addresses.

## **Parameters**

<ParamField path="page_size" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="$scalar" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing fabric addresses.

### get

Retrieve a single fabric address.

## **Parameters**

<ParamField path="address_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the address.
</ParamField>

## **Returns**

`dict` -- The fabric address resource.

## **Example**

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

# List all addresses
my $addresses = $client->fabric->addresses->list();
for my $addr (@{addresses.get("data", [])}) {
    print "${addr['display_name']}: ${addr['type']}";

# Get a specific address
my $address = $client->fabric->addresses->{"address-id"}
print "Address: ${address['display_name']}";
```
