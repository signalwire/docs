Retrieve a single conference room resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

## **Returns**

`dict` -- The conference room resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $room = $client->fabric->conference_rooms->{"room-id"}
print "Conference Room: ${room['name']}, ID: ${room['id']}";
```
