Retrieve a single SIP gateway resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

## **Returns**

`dict` -- The SIP gateway resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $gateway = $client->fabric->sip_gateways->{"gateway-id"}
print "Sip Gateway: ${gateway['name']}, ID: ${gateway['id']}";
```
