Partially update an AI agent resource. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated AI agent resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->ai_agents->update("agent-id", name => "updated-name");
print "Updated: ${updated['name']}";
```
