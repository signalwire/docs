Create a new subscriber resource.

## **Parameters**

<ParamField path="name" type="$scalar" required={true} toc={true}>
  Display name of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created subscriber resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->fabric->subscribers->create(
    name => "alice", type => "subscriber",
)
print "Created: ${result['id']}";
```
