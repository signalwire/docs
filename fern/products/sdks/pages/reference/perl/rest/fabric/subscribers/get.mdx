Retrieve a single subscriber resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

## **Returns**

`dict` -- The subscriber resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $subscriber = $client->fabric->subscribers->{"subscriber-id"}
print "Subscriber: ${subscriber['name']}, ID: ${subscriber['id']}";
```
