List all versions of a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

<ParamField path="page_size" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

<ParamField path="page_token" type="$scalar" toc={true}>
  Pagination token for the next page of results.
</ParamField>

## **Returns**

`dict` -- Paginated response containing call flow versions.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $response = $client->fabric->call_flows->list_versions("call-flow-id");
for my $version (@{$response->{"data"} // []}) {
    print "Version ${version['version']}: ${version['id']}";
```
