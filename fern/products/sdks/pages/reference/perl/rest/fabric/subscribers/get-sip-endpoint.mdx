Retrieve a single SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`dict` -- The SIP endpoint.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $endpoint = $client->fabric->subscribers->get_sip_endpoint("subscriber-id", "endpoint-id");
print "SIP Endpoint: ${endpoint['username']}, ID: ${endpoint['id']}";
```
