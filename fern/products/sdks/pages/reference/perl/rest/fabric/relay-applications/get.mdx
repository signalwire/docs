Retrieve a single RELAY application resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the RELAY application.
</ParamField>

## **Returns**

`dict` -- The RELAY application resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $item = $client->fabric->relay_applications->{"resource-id"}
print "Relay Application: ${item['name']}, ID: ${item['id']}";
```
