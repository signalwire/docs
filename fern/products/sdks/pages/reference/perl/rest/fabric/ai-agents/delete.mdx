Delete an AI agent resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the AI agent to delete.
</ParamField>

## **Returns**

`dict` -- Empty dict on success (HTTP 204).

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->fabric->ai_agents->delete("agent-id");
print "AI Agent deleted";
```
