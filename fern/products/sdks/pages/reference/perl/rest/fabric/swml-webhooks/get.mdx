Retrieve a single SWML webhook resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SWML webhook.
</ParamField>

## **Returns**

`dict` -- The SWML webhook resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $webhook = $client->fabric->swml_webhooks->{"webhook-id"}
print "Swml Webhook: ${webhook['name']}, ID: ${webhook['id']}";
```
