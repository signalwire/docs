Replace a conference room resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the conference room.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`dict` -- The updated conference room resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->conference_rooms->update("room-id", name => "updated-name");
print "Updated: ${updated['name']}";
```
