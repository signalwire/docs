Partially update a SIP gateway resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the SIP gateway.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`dict` -- The updated SIP gateway resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->fabric->sip_gateways->update("gateway-id", name => "updated-name");
print "Updated: ${updated['name']}";
```
