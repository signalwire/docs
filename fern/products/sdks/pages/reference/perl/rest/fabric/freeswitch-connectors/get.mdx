Retrieve a single FreeSWITCH connector resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the FreeSWITCH connector.
</ParamField>

## **Returns**

`dict` -- The FreeSWITCH connector resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $connector = $client->fabric->freeswitch_connectors->{"connector-id"}
print "Freeswitch Connector: ${connector['name']}, ID: ${connector['id']}";
```
