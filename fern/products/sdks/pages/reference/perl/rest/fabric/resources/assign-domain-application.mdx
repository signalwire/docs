Assign a domain application to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The assigned domain application.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->fabric->resources->assign_domain_application(
    "resource-id", domain => "example.signalwire.com",
)
print "Domain application assigned: ${result['id']}";
```
