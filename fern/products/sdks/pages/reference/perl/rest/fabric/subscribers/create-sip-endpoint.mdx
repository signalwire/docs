Create a new SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="$scalar" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`dict` -- The created SIP endpoint.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->fabric->subscribers->create_sip_endpoint(
    "subscriber-id", username => "alice-sip", password => "s3cret",
)
print "Created SIP endpoint: ${result['id']}";
```
