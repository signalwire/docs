Generate tokens for the SignalWire Chat service. Chat tokens authenticate
client-side connections for real-time chat messaging.

Access via `client.chat` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new();
```

---

### create_token

Create a Chat authentication token.

## **Parameters**

<ParamField path="channels" type="\@arrayref" toc={true}>
  List of channel names the token grants access to.
</ParamField>

<ParamField path="member_id" type="$scalar" toc={true}>
  Identifier for the connecting member.
</ParamField>

<ParamField path="ttl" type="$integer" toc={true}>
  Token time-to-live in seconds.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional token parameters (e.g., `state`, `permissions`).
</ParamField>

## **Returns**

`dict` -- The token object containing the JWT token string.

## **Example**

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $token = $client->chat->create_token(
    channels => ["support-room"],
    member_id => "user-456",
    ttl => 3600,
)
print "Chat token:", token.get("token");
```
