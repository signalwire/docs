Release a phone number from the account.

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The phone number SID to release.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->phone_numbers->delete("PN...");
```
