Create a new application.

## **Parameters**

<ParamField path="FriendlyName" type="$scalar" required={true} toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="$scalar" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="VoiceMethod" type="$scalar" toc={true}>
  HTTP method for the voice URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="SmsUrl" type="$scalar" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="SmsMethod" type="$scalar" toc={true}>
  HTTP method for the SMS URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="StatusCallback" type="$scalar" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`dict` -- The created application resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $app = $client->compat->applications->create(
    FriendlyName => "My App",
    VoiceUrl => "https://example.com/voice",
    SmsUrl => "https://example.com/sms";
)
```
