Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="$scalar" required={true} toc={true}>
  The token identifier.
</ParamField>

<ParamField path="Ttl" type="$integer" toc={true}>
  New time-to-live in seconds for the token.
</ParamField>

## **Returns**

`dict` -- The updated token resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->tokens->update("token-id", Ttl => 7200);
```
