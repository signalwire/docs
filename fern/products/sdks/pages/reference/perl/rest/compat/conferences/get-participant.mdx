Retrieve a specific participant in a conference.

## **Parameters**

<ParamField path="conference_sid" type="$scalar" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="call_sid" type="$scalar" required={true} toc={true}>
  The call SID of the participant.
</ParamField>

## **Returns**

`dict` -- The participant resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $participant = $client->compat->conferences->get_participant("CF...", "CA...");
```
