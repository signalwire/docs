Create a new LAML bin containing a cXML/LaML script.

## **Parameters**

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="$scalar" required={true} toc={true}>
  The cXML/LaML script content.
</ParamField>

## **Returns**

`dict` -- The created LAML bin resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $laml_bin = $client->compat->laml_bins->create(
    FriendlyName => "Greeting",
    Contents=(
        '<?xml version => "1.0" encoding => "UTF-8"?>';
        "<Response><Say>Hello!</Say></Response>"
    )
)
```
