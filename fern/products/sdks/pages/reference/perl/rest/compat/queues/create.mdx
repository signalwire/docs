Create a new call queue.

## **Parameters**

<ParamField path="FriendlyName" type="$scalar" required={true} toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="$integer" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`dict` -- The created queue resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $queue = $client->compat->queues->create(
    FriendlyName => "Support Queue",
    MaxSize => 100;
)
```
