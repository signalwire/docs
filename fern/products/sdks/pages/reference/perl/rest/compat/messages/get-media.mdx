Retrieve a specific media item from a message.

## **Parameters**

<ParamField path="message_sid" type="$scalar" required={true} toc={true}>
  The SID of the message.
</ParamField>

<ParamField path="media_sid" type="$scalar" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`dict` -- The media resource including the content URL.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $item = $client->compat->messages->get_media("SM...", "ME...");
```
