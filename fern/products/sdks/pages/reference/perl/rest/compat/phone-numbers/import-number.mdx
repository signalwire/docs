Import an external phone number into the account.

## **Parameters**

<ParamField path="PhoneNumber" type="$scalar" required={true} toc={true}>
  The phone number to import in E.164 format.
</ParamField>

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  A human-readable label for the number.
</ParamField>

## **Returns**

`dict` -- The imported phone number resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $number = $client->compat->phone_numbers->import_number(
    PhoneNumber => "+15559876543";
)
```
