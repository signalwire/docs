Retrieve a specific member from a queue by call SID.

## **Parameters**

<ParamField path="queue_sid" type="$scalar" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="call_sid" type="$scalar" required={true} toc={true}>
  The call SID of the queue member.
</ParamField>

## **Returns**

`dict` -- The queue member resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $member = $client->compat->queues->get_member("QU...", "CA...");
```
