Create a new subproject (sub-account).

## **Parameters**

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  A human-readable name for the subproject.
</ParamField>

## **Returns**

`dict` -- The created account resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $sub = $client->compat->accounts->create(FriendlyName => "Marketing Team");
```
