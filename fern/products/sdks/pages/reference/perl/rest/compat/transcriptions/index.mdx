Manage call transcriptions with list, get, and delete operations.

Access via `client.compat.transcriptions` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $transcriptions = $client->compat->transcriptions->list();
```
