List transcriptions in the account.

## **Parameters**

<ParamField path="PageSize" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing transcription resources.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $transcriptions = $client->compat->transcriptions->list(PageSize => 20);
```
