List applications in the account.

## **Parameters**

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  Filter by friendly name.
</ParamField>

<ParamField path="PageSize" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing application resources.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $apps = $client->compat->applications->list(PageSize => 20);
```
