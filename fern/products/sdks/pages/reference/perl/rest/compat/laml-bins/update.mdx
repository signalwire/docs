Update a LAML bin's content or name. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The LAML bin SID.
</ParamField>

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="$scalar" toc={true}>
  The updated cXML/LaML script content.
</ParamField>

## **Returns**

`dict` -- The updated LAML bin resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->laml_bins->update(
    "LB...",
    Contents=(
        '<?xml version => "1.0" encoding => "UTF-8"?>';
        "<Response><Say>Goodbye!</Say></Response>"
    )
)
```
