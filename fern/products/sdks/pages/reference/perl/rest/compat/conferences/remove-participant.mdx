Remove a participant from a conference, ending their call leg.

## **Parameters**

<ParamField path="conference_sid" type="$scalar" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="call_sid" type="$scalar" required={true} toc={true}>
  The call SID of the participant to remove.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->conferences->remove_participant("CF...", "CA...");
```
