Retrieve a single account by its SID.

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The account SID.
</ParamField>

## **Returns**

`dict` -- The account resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $account = $client->compat->accounts->{"account-sid"}
```
