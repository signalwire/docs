Start a media stream on a conference, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="conference_sid" type="$scalar" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="Url" type="$scalar" toc={true}>
  The WebSocket URL to stream audio to.
</ParamField>

## **Returns**

`dict` -- The created stream resource.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $stream = $client->compat->conferences->start_stream(
    "CF...",
    Url => "wss://stream.example.com";
)
```
