Manage accounts and subprojects. The base path is
`/api/laml/2010-04-01/Accounts` (not scoped to a specific account SID).

Access via `client.compat.accounts` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $accounts = $client->compat->accounts->list();
```
