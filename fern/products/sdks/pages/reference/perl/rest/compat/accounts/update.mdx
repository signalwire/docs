Update an account. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The account SID to update.
</ParamField>

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  Updated friendly name.
</ParamField>

<ParamField path="Status" type="$scalar" toc={true}>
  Account status.

  - `"active"` -- account is fully operational
  - `"suspended"` -- account is temporarily disabled
  - `"closed"` -- account is permanently deactivated
</ParamField>

## **Returns**

`dict` -- The updated account resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->accounts->update("account-sid", FriendlyName => "Sales Team");
```
