List messages in the account.

## **Parameters**

<ParamField path="To" type="$scalar" toc={true}>
  Filter by destination number.
</ParamField>

<ParamField path="From" type="$scalar" toc={true}>
  Filter by sender number.
</ParamField>

<ParamField path="PageSize" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing message resources.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $messages = $client->compat->messages->list(PageSize => 20);
```
