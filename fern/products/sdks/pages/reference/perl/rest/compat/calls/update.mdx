Update an active call (e.g., redirect to new LAML, end the call).

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The call SID.
</ParamField>

<ParamField path="Url" type="$scalar" toc={true}>
  New URL to fetch LAML instructions from.
</ParamField>

<ParamField path="Status" type="$scalar" toc={true}>
  Set to `"completed"` to end the call.
</ParamField>

## **Returns**

`dict` -- The updated call resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->calls->update("CA...", Status => "completed");
```
