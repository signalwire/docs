List media items (images, files) attached to a message.

## **Parameters**

<ParamField path="message_sid" type="$scalar" required={true} toc={true}>
  The SID of the message.
</ParamField>

## **Returns**

`dict` -- List of media resources attached to the message.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $media = $client->compat->messages->list_media("SM...");
```
