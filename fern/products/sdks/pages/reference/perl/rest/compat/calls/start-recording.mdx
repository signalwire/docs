Start recording an active call.

## **Parameters**

<ParamField path="call_sid" type="$scalar" required={true} toc={true}>
  The SID of the call to record.
</ParamField>

<ParamField path="RecordingStatusCallback" type="$scalar" toc={true}>
  URL to receive recording status events.
</ParamField>

<ParamField path="RecordingChannels" type="$scalar" toc={true}>
  Recording channel mode.

  - `"mono"` -- both legs of the call are mixed into a single audio channel
  - `"dual"` -- each leg of the call is recorded in a separate audio channel
</ParamField>

## **Returns**

`dict` -- The created recording resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $recording = $client->compat->calls->start_recording("CA...");
```
