Update a conference (e.g., end it, or set the announce URL).

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The conference SID.
</ParamField>

<ParamField path="Status" type="$scalar" toc={true}>
  Set to `"completed"` to end the conference.
</ParamField>

<ParamField path="AnnounceUrl" type="$scalar" toc={true}>
  URL returning LAML to play an announcement to all participants.
</ParamField>

## **Returns**

`dict` -- The updated conference resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->conferences->update("CF...", Status => "completed");
```
