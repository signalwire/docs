Search for available local phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="$scalar" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="AreaCode" type="$scalar" toc={true}>
  Filter by area code.
</ParamField>

<ParamField path="Contains" type="$scalar" toc={true}>
  Filter by number pattern (e.g., `"555"`).
</ParamField>

<ParamField path="InRegion" type="$scalar" toc={true}>
  Filter by region or state abbreviation.
</ParamField>

<ParamField path="PageSize" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing available local number resources.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $numbers = $client->compat->phone_numbers->search_local("US", AreaCode => "512");
```
