Retrieve a specific media item from a fax.

## **Parameters**

<ParamField path="fax_sid" type="$scalar" required={true} toc={true}>
  The SID of the fax.
</ParamField>

<ParamField path="media_sid" type="$scalar" required={true} toc={true}>
  The SID of the media item.
</ParamField>

## **Returns**

`dict` -- The media resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $item = $client->compat->faxes->get_media("FX...", "ME...");
```
