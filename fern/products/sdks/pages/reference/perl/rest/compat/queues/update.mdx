Update a queue's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="$scalar" required={true} toc={true}>
  The queue SID.
</ParamField>

<ParamField path="FriendlyName" type="$scalar" toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="$integer" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`dict` -- The updated queue resource.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->compat->queues->update("QU...", MaxSize => 200);
```
