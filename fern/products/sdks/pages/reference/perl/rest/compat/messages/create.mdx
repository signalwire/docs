Send a new SMS or MMS message.

## **Parameters**

<ParamField path="To" type="$scalar" required={true} toc={true}>
  The destination phone number in E.164 format.
</ParamField>

<ParamField path="From" type="$scalar" required={true} toc={true}>
  The sender phone number (must be a number you own).
</ParamField>

<ParamField path="Body" type="$scalar" toc={true}>
  The text body of the message. Required for SMS; optional for MMS if `MediaUrl` is provided.
</ParamField>

<ParamField path="MediaUrl" type="\@arrayref" toc={true}>
  URLs of media to include (for MMS). Up to 10 media URLs.
</ParamField>

<ParamField path="StatusCallback" type="$scalar" toc={true}>
  URL to receive message status webhook events.
</ParamField>

## **Returns**

`dict` -- The created message resource including the message SID.

## **Example**

```perl {10,17}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

# Send an SMS
my $msg = $client->compat->messages->create(
    To => "+15559876543",
    From => "+15551234567",
    Body => "Hello from SignalWire!";
)

# Send an MMS with an image
my $mms = $client->compat->messages->create(
    To => "+15559876543",
    From => "+15551234567",
    Body => "Check this out",
    MediaUrl => ["https://example.com/image.jpg"];
)
```
