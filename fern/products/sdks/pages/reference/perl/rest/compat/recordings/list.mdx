List recordings in the account.

## **Parameters**

<ParamField path="DateCreated" type="$scalar" toc={true}>
  Filter by creation date.
</ParamField>

<ParamField path="CallSid" type="$scalar" toc={true}>
  Filter by the call SID that generated the recording.
</ParamField>

<ParamField path="PageSize" type="$integer" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`dict` -- Paginated response containing recording resources.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $recordings = $client->compat->recordings->list(PageSize => 20);
```
