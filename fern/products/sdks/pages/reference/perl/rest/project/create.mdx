### tokens.create

Create a new API token for the project.

## **Parameters**

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Token configuration fields (e.g., `name`, `scopes`).
</ParamField>

## **Returns**

`dict` -- The newly created token object, including the token value.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $token = $client->project->tokens->create(name => "ci-token");
print "Token:", token.get("token");
print "ID:", token.get("id");
```
