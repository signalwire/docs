Manage project-level API tokens. Tokens control access to your SignalWire project
and can be created, updated, and revoked.

Access via `client.project.tokens` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $token = $client->project->tokens->create(name => "ci-token");
print token["id"];
```
