Create a new regulatory address.

## **Parameters**

<ParamField path="customer_name" type="$scalar" toc={true}>
  The customer or business name for the address.
</ParamField>

<ParamField path="street" type="$scalar" toc={true}>
  Street address.
</ParamField>

<ParamField path="city" type="$scalar" toc={true}>
  City name.
</ParamField>

<ParamField path="state" type="$scalar" toc={true}>
  State or province.
</ParamField>

<ParamField path="postal_code" type="$scalar" toc={true}>
  Postal or ZIP code.
</ParamField>

<ParamField path="country" type="$scalar" toc={true}>
  ISO country code (e.g., `"US"`).
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional address fields as needed by the API.
</ParamField>

## **Returns**

`dict` -- The newly created address object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $addr = $client->addresses->create(
    customer_name => "Acme Corp",
    street => "123 Main St",
    city => "Austin",
    state => "TX",
    postal_code => "78701",
    country => "US",
)
print "Created address:", addr.get("id");
```
