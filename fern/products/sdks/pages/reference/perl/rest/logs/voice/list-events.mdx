List events for a specific voice call. Events include state transitions
(ringing, answered, ended), media events, and other call lifecycle data.

## **Parameters**

<ParamField path="log_id" type="$scalar" required={true} toc={true}>
  The voice log ID.
</ParamField>

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of event objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $events = $client->logs->voice->list_events("voice-log-id");
for my $event (@{events.get("data", [])}) {
    print $event->{"type"}, $event->{"timestamp"};
```
