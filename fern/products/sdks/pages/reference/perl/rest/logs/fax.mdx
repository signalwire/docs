Query fax log entries. Fax logs provide a read-only view of fax activity
in your project.

Access via `client.logs.fax` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new();
```

### list

List fax log entries.

## **Parameters**

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of fax log objects.

---

### get

Retrieve a specific fax log entry.

## **Parameters**

<ParamField path="log_id" type="$scalar" required={true} toc={true}>
  The fax log ID.
</ParamField>

## **Returns**

`dict` -- The fax log object.

## **Examples**

### List fax logs

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $faxes = $client->logs->fax->list();
for my $fax (@{faxes.get("data", [])}) {
    print fax.get("from"), "->", fax.get("to"), fax.get("status");
```

### Get a specific fax log

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $fax = $client->logs->fax->{"fax-log-id"}
print fax.get("from"), "->", fax.get("to");
```
