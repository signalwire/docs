Disable noise reduction on an active call.

## **Parameters**

<ParamField path="call_id" type="$scalar" required={true} toc={true}>
  The ID of the call.
</ParamField>

## **Returns**

`dict` — The API response confirming noise reduction was disabled.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->calling->denoise_stop(call_id => "call-id-xxx");
```
