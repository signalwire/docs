Stop an active audio tap.

## **Parameters**

<ParamField path="call_id" type="$scalar" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="$scalar" required={true} toc={true}>
  The control ID returned from [`tap()`][tap].
</ParamField>

## **Returns**

`dict` — The API response confirming the tap was stopped.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->calling->tap_stop(call_id => "call-id-xxx", control_id => "ctrl-id");
```
