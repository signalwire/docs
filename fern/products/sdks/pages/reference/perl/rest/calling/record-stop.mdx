[record]: /docs/sdks/reference/perl/rest/calling/record

Stop an active recording. The recording file becomes available after stopping.

## **Parameters**

<ParamField path="call_id" type="$scalar" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="$scalar" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`dict` — The API response containing the recording URL and metadata.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->calling->record_stop(call_id => "call-id-xxx", control_id => "ctrl-id");
print(result)  # Contains recording URL;
```
