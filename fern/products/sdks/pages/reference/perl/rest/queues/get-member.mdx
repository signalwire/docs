Retrieve a specific member from a queue.

## **Parameters**

<ParamField path="queue_id" type="$scalar" required={true} toc={true}>
  The queue resource ID.
</ParamField>

<ParamField path="member_id" type="$scalar" required={true} toc={true}>
  The member resource ID.
</ParamField>

## **Returns**

`dict` -- The member object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $member = $client->queues->get_member("queue-id", "member-id");
print member.get("call_id"), member.get("position");
```
