List queues in the project.

## **Parameters**

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of queue objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->queues->list();
for my $q (@{$result->{"data"} // []}) {
    print q["name"], q.get("current_size");
```
