Retrieve the next member in the queue (the member at the front).

## **Parameters**

<ParamField path="queue_id" type="$scalar" required={true} toc={true}>
  The queue resource ID.
</ParamField>

## **Returns**

`dict` -- The next member object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $next_member = $client->queues->get_next_member("queue-id");
print "Next caller:", next_member.get("call_id");
```
