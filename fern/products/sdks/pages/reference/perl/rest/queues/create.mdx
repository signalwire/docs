Create a new queue.

## **Parameters**

<ParamField path="name" type="$scalar" toc={true}>
  Queue name.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional queue configuration fields (e.g., `max_size`, `fifo`).
</ParamField>

## **Returns**

`dict` -- The newly created queue object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $queue = $client->queues->create(name => "support", max_size => 50);
print "Queue ID:", queue["id"];
```
