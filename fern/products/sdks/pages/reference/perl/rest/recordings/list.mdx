List recordings in the project.

## **Parameters**

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters to filter and paginate results.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of recording objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->recordings->list(page_size => 20);
for my $rec (@{$result->{"data"} // []}) {
    print rec.get("id"), rec.get("duration"), rec.get("state");
```
