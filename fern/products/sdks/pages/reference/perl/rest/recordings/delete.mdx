Delete a recording.

## **Parameters**

<ParamField path="recording_id" type="$scalar" required={true} toc={true}>
  The recording resource ID.
</ParamField>

## **Returns**

`dict` -- Empty dict on success.

## **Example**

```perl {9-9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->recordings->delete("recording-id");
```
