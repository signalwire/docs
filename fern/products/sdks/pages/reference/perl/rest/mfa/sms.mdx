Send an MFA verification code via SMS.

## **Parameters**

<ParamField path="to" type="$scalar" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="$scalar" required={true} toc={true}>
  The sender phone number in E.164 format. Must be a number in your project.
  Since `from` is a Python reserved keyword, pass it via dict unpacking (see example).
</ParamField>

<ParamField path="message" type="$scalar" required={true} toc={true}>
  The message body. Include `{code}` as a placeholder for the generated code
  (e.g., `"Your verification code is: {code}"`).
</ParamField>

<ParamField path="token_length" type="$integer" toc={true}>
  Length of the generated code.
</ParamField>

<ParamField path="max_attempts" type="$integer" toc={true}>
  Maximum number of verification attempts allowed.
</ParamField>

<ParamField path="valid_for" type="$integer" toc={true}>
  How long the code remains valid, in seconds.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`dict` -- Response containing the `id` of the MFA request (used in `verify()`).

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->mfa->sms(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your code is: {code}",
})
my $request_id = result["id"];
print "MFA request:", request_id;
```
