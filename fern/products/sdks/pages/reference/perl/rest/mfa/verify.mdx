Verify an MFA code submitted by the user.

## **Parameters**

<ParamField path="request_id" type="$scalar" required={true} toc={true}>
  The MFA request ID returned from `sms()` or `call()`.
</ParamField>

<ParamField path="token" type="$scalar" required={true} toc={true}>
  The verification code entered by the user.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`dict` -- Verification result indicating whether the code is valid.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->mfa->verify("your-mfa-request-id", token => "123456");
if ($result->{"success"}) {
    print "Verification successful";
} else {
    print "Invalid code";
```
