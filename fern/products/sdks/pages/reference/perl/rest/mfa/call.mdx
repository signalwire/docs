Send an MFA verification code via voice call.

## **Parameters**

<ParamField path="to" type="$scalar" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="$scalar" required={true} toc={true}>
  The caller phone number in E.164 format.
  Since `from` is a Python reserved keyword, pass it via dict unpacking (see example).
</ParamField>

<ParamField path="message" type="$scalar" required={true} toc={true}>
  The message to speak. Include `{code}` as a placeholder.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional parameters (e.g., `token_length`, `max_attempts`, `valid_for`).
</ParamField>

## **Returns**

`dict` -- Response containing the `id` of the MFA request.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $result = $client->mfa->call(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your verification code is {code}",
})
my $request_id = result["id"];
```
