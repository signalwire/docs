Send and verify multi-factor authentication (MFA) codes via SMS or voice call.
The typical flow is: request a code with `sms()` or `call()`, then confirm it
with `verify()`.

Access via `client.mfa` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $request = $client->mfa->sms(to => "+15551234567", from_ => "+15559876543");
print request["id"];
```
