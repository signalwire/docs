Update a 10DLC campaign. Uses PUT to replace fields.

## **Parameters**

<ParamField path="campaign_id" type="$scalar" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Fields to update (e.g., `description`, `usecase`).
</ParamField>

## **Returns**

`dict` -- The updated campaign object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $updated = $client->registry->campaigns->update(
    "campaign-id",
    description => "Updated campaign description",
)
print updated.get("description");
```
