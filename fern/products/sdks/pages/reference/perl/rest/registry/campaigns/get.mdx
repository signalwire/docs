Retrieve a specific 10DLC campaign by ID.

## **Parameters**

<ParamField path="campaign_id" type="$scalar" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

## **Returns**

`dict` -- The campaign object.

## **Example**

```perl {9-10}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $campaign = $client->registry->campaigns->{"campaign-id"}
print campaign.get("description"), campaign.get("usecase");
```
