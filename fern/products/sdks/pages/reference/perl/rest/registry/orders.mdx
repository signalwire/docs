Retrieve number assignment orders. Orders track the process of assigning phone
numbers to 10DLC campaigns.

Access via `client.registry.orders` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new();
```

## **get**

Retrieve a specific number assignment order.

### **Parameters**

<ParamField path="order_id" type="$scalar" required={true} toc={true}>
  The order resource ID.
</ParamField>

### **Returns**

`dict` -- The order object.

### **Example**

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $order = $client->registry->orders->{"order-id"}
print order.get("id"), order.get("status");
```
