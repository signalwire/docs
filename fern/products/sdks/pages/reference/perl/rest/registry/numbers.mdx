Manage phone number assignments to 10DLC campaigns. Use this sub-resource to
remove a number from its campaign assignment.

Access via `client.registry.numbers` on a [`SignalWire::REST::RestClient`][restclient] instance.

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new();
```

## **delete**

Remove a phone number from its campaign assignment.

### **Parameters**

<ParamField path="number_id" type="$scalar" required={true} toc={true}>
  The number assignment resource ID.
</ParamField>

### **Returns**

`dict` -- Empty dict on success.

### **Example**

```perl
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

$client->registry->numbers->delete("number-assignment-id");
```
