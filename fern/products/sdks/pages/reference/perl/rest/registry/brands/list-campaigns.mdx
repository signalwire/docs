List messaging campaigns under a brand.

## **Parameters**

<ParamField path="brand_id" type="$scalar" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of campaign objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $campaigns = $client->registry->brands->list_campaigns("brand-id");
for my $c (@{campaigns.get("data", [])}) {
    print c.get("description"), c.get("usecase");
```
