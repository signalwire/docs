List phone numbers assigned to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="$scalar" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**params" type="\%hashref" toc={true}>
  Optional query parameters.
</ParamField>

## **Returns**

`dict` -- JSON response containing a `data` list of number objects.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $numbers = $client->registry->campaigns->list_numbers("campaign-id");
for my $n (@{numbers.get("data", [])}) {
    print n.get("number"), n.get("id");
```
