Create a number assignment order to assign phone numbers to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="$scalar" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="phone_number_ids" type="\@arrayref" toc={true}>
  List of phone number resource IDs to assign.
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional order parameters.
</ParamField>

## **Returns**

`dict` -- The newly created order object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $order = $client->registry->campaigns->create_order(
    "campaign-id",
    phone_number_ids => ["number-id-1", "number-id-2"],
)
print "Order ID:", order.get("id");
```
