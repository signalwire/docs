Retrieve a specific 10DLC brand by ID.

## **Parameters**

<ParamField path="brand_id" type="$scalar" required={true} toc={true}>
  The brand resource ID.
</ParamField>

## **Returns**

`dict` -- The brand object.

## **Example**

```perl {9-10}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $brand = $client->registry->brands->{"brand-id"}
print brand.get("display_name"), brand.get("entity_type");
```
