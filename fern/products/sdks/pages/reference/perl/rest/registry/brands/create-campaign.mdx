Create a messaging campaign under a brand.

## **Parameters**

<ParamField path="brand_id" type="$scalar" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="description" type="$scalar" toc={true}>
  Campaign description.
</ParamField>

<ParamField path="usecase" type="$scalar" toc={true}>
  Campaign use case.

  - `"MIXED"` -- multiple or general-purpose messaging use cases
  - `"MARKETING"` -- promotional or marketing messages
  - `"CUSTOMER_CARE"` -- customer support and service messages
</ParamField>

<ParamField path="**kwargs" type="\%hashref" toc={true}>
  Additional campaign fields.
</ParamField>

## **Returns**

`dict` -- The newly created campaign object.

## **Example**

```perl {9}
use SignalWire::REST::RestClient;

my $client = SignalWire::REST::RestClient->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
)

my $campaign = $client->registry->brands->create_campaign(
    "brand-id",
    description => "Customer notifications",
    usecase => "MIXED",
)
print "Campaign ID:", campaign.get("id");
```
