Join an ad-hoc audio conference. The conference is created automatically if it
does not already exist. Multiple calls can join the same conference by name.

<Info>
Use [`leave_conference()`][leave-conference] to remove the call from the conference.
</Info>

<Info>
This method emits [`calling.conference`][calling-conference] events. See [Call Events][call-events] for payload details.
</Info>

<Info>
This method corresponds to the SWML [`join_conference`][join-conference]
verb. See the [SWML join_conference reference][swml-join-conference-reference] for the
full specification.
</Info>

## **Parameters**

<ParamField path="name" type="$scalar" required={true} toc={true}>
  Conference name. All calls joining the same name are in the same conference.
</ParamField>

<ParamField path="muted" type="$scalar / undef" toc={true}>
  Join the conference muted.
</ParamField>

<ParamField path="beep" type="$scalar / undef" toc={true}>
  Play a beep when joining or leaving.

  - `"true"` -- beep on both enter and exit
  - `"false"` -- no beep
  - `"onEnter"` -- beep only when a participant joins
  - `"onExit"` -- beep only when a participant leaves
</ParamField>

<ParamField path="start_on_enter" type="$scalar / undef" toc={true}>
  Start the conference when this participant enters. If `false`, the
  conference waits until a participant with `start_on_enter=true` joins.
</ParamField>

<ParamField path="end_on_exit" type="$scalar / undef" toc={true}>
  End the conference when this participant leaves.
</ParamField>

<ParamField path="wait_url" type="$scalar / undef" toc={true}>
  URL of audio to play while waiting for the conference to start.
</ParamField>

<ParamField path="max_participants" type="$scalar / undef" toc={true}>
  Maximum number of participants in the conference.
</ParamField>

<ParamField path="record" type="$scalar / undef" toc={true}>
  Recording mode.

  - `"record-from-start"` -- begin recording when the conference starts
  - `"do-not-record"` -- do not record the conference
</ParamField>

<ParamField path="region" type="$scalar / undef" toc={true}>
  Region for the conference media server.
</ParamField>

<ParamField path="trim" type="$scalar / undef" toc={true}>
  Whether to trim silence from the conference recording.
</ParamField>

<ParamField path="coach" type="$scalar / undef" toc={true}>
  Call SID to coach (whisper to one participant).
</ParamField>

<ParamField path="status_callback" type="$scalar / undef" toc={true}>
  URL to receive conference status webhooks.
</ParamField>

<ParamField path="status_callback_event" type="$scalar / undef" toc={true}>
  Events that trigger status callbacks.
</ParamField>

<ParamField path="status_callback_event_type" type="$scalar / undef" toc={true}>
  Content type for status callback requests.
</ParamField>

<ParamField path="status_callback_method" type="$scalar / undef" toc={true}>
  HTTP method for status callbacks (e.g., `"POST"`, `"GET"`).
</ParamField>

<ParamField path="recording_status_callback" type="$scalar / undef" toc={true}>
  URL to receive recording status webhooks.
</ParamField>

<ParamField path="recording_status_callback_event" type="$scalar / undef" toc={true}>
  Events that trigger recording status callbacks.
</ParamField>

<ParamField path="recording_status_callback_event_type" type="$scalar / undef" toc={true}>
  Content type for recording status callback requests.
</ParamField>

<ParamField path="recording_status_callback_method" type="$scalar / undef" toc={true}>
  HTTP method for recording status callbacks.
</ParamField>

<ParamField path="stream_obj" type="$scalar / undef" toc={true}>
  Stream configuration for the conference audio.
</ParamField>

## **Returns**

`dict` -- Server response confirming the join.

## **Example**

```perl {16}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    $call->play([{"type": "tts", "text": "Joining the team conference."}]);

    # Join a conference
    $call->join_conference(
        "team-standup",
        my $beep = "onEnter",
        my $start_on_enter = 1,
    )

    # The call is now in the conference until it leaves or the call ends

});
$client->run();
```
