Send DTMF tones on the call. Use this to navigate IVR menus on outbound calls
or to send tone signals.

<Info>
This method emits [`calling.call.send_digits`][calling-call-send-digits] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="digits" type="$scalar" required={true} toc={true}>
  The DTMF digit string to send. Valid characters: `0-9`, `*`, `#`,
  `A-D`, `W` (0.5s pause), `w` (1s pause).
</ParamField>

<ParamField path="control_id" type="$scalar / undef" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

## **Returns**

`dict` -- Server response confirming the send_digits operation.

## **Example**

```perl {14}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();

    my $result = $call->send_digits("1w2");
    print "Digits sent: ${result}";

});
$client->run();
```
