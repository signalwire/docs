Put the call on hold. The remote party hears hold music or silence while the
call is held.

<Info>
Use [`unhold()`][unhold] to release the call from hold.
</Info>

<Info>
This method emits [`calling.call.hold`][calling-call-hold] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the hold operation.

## **Example**

```perl {15}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    $call->play([{"type": "tts", "text": "Please hold while I look that up."}]);

    my $result = $call->hold();
    print "Hold started: ${result}";

    # Do some processing...
    asyncio.sleep(5);

    # Take the call off hold
    $call->unhold();

});
$client->run();
```
