Start or stop live translation on the call. Translates spoken audio into
another language in real-time.

## **Parameters**

<ParamField path="action" type="\%hashref" required={true} toc={true}>
  Action configuration for the live translation.
</ParamField>

<Indent>
<ParamField path="action.action" type="$scalar" required={true} toc={true}>
  The action to perform.

  - `"start"` -- begin live translation
  - `"stop"` -- end live translation
</ParamField>

<ParamField path="action.source_language" type="$scalar" toc={true}>
  Source language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="action.target_language" type="$scalar" toc={true}>
  Target language code (e.g., `"es-ES"`).
</ParamField>
</Indent>

<ParamField path="status_url" type="$scalar / undef" toc={true}>
  URL to receive translation results via webhook.
</ParamField>

## **Returns**

`dict` -- Server response confirming the operation.

## **Example**

```perl {15}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();

    # Start live translation from English to Spanish
    $call->live_translate({
        "action": "start",
        "source_language": "en-US",
        "target_language": "es-ES",
    })

    # Let the call proceed...
    $call->wait_for_ended();

});
$client->run();
```
