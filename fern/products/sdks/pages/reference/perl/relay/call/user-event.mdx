Send a custom user-defined event on the call. User events allow you to pass
application-specific data through the RELAY event system. Other listeners on
the same call can receive and react to these events.

## **Parameters**

<ParamField path="event" type="$scalar / undef" toc={true}>
  The event name or identifier.
</ParamField>

Additional keyword arguments are forwarded as event parameters.

## **Returns**

`dict` -- Server response confirming the event was sent.

## **Example**

```perl {15}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();

    # Send a custom event with metadata
    my $result = $call->user_event(
        my $event = "customer_identified",
        my $customer_id = "cust-12345",
        my $tier = "premium",
    )
    print "User event sent: ${result}";

});
$client->run();
```
