Stop noise reduction on the call, restoring the original unfiltered audio.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming noise reduction has stopped.

## **Example**

```perl {20}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();

    # Enable noise reduction
    $call->denoise();

    my $action = $call->play([{"type": "tts", "text": "Noise reduction is now active."}]);
    $action->wait();

    my $result = $call->denoise_stop();
    print "Noise reduction stopped: ${result}";

});
$client->run();
```
