Leave the current room.

## **Parameters**

None.

## **Returns**

`dict` -- Server response confirming the leave.

## **Example**

```perl {18}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();

    # Join a room
    $call->join_room("meeting-room-1");

    # Later, leave the room
    $call->leave_room();
    $call->hangup();

});
$client->run();
```
