Stop the AI agent session.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {19}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->ai(
        my $prompt = {"text": "You are a helpful assistant."},
        my $ai_params = {"temperature": 0.7},
    )

    # Stop the AI session
    $action->stop();

});
$client->run();
```
