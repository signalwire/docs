Stop media interception.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {19}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->tap(
        my $tap = {"type": "audio", "params": {"direction": "both"}},
        my $device = {"type": "ws", "params": {"uri": "wss://example.com/tap"}},
    )

    # Stop the tap when done
    $action->stop();

});
$client->run();
```
