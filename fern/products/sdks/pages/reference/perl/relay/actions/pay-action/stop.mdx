Stop payment collection.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {20}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->pay(
        my $payment_connector_url = "https://example.com/payment-connector",
        my $charge_amount = "29.99",
        my $currency = "USD",
    )

    # Stop payment collection
    $action->stop();

});
$client->run();
```
