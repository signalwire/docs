Stop the play-and-collect operation.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {19}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->play_and_collect(
        my $media = [{"type": "tts", "text": "Press 1 for sales, 2 for support."}],
        my $collect = {"digits": {"max": 1, "digit_timeout": 5.0}},
    )

    # Cancel the operation
    $action->stop();

});
$client->run();
```
