Stop the collect operation.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {20}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->collect(
        my $speech = {"end_silence_timeout": 2.0},
        my $initial_timeout = 10.0,
        my $start_input_timers = 0,
    )

    # Stop the collect operation
    $action->stop();

});
$client->run();
```
