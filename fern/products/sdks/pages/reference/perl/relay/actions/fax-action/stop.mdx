Stop the fax operation.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {19}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->send_fax(
        my $document = "https://example.com/invoice.pdf",
        my $identity = "+15559876543",
    )

    # Stop the fax operation
    $action->stop();

});
$client->run();
```
