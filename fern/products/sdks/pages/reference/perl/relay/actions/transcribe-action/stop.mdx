Stop transcription.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {16}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->transcribe();

    # Stop transcription early
    $action->stop();

});
$client->run();
```
