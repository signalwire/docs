Pause audio playback. The playback position is preserved.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {18}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->play(
        [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
        my $loop = 0,
    )

    $action->pause();
    # ... do something ...
    $action->resume();

});
$client->run();
```
