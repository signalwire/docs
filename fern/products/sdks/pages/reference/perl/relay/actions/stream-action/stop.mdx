Stop audio streaming.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {19}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->stream(
        my $url = "wss://example.com/audio-stream",
        my $track = "inbound",
    )

    # Stop the stream when done
    $action->stop();

});
$client->run();
```
