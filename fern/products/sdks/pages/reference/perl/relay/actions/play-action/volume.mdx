Adjust the playback volume.

## **Parameters**

<ParamField path="volume" type="$number" required={true} toc={true}>
  Volume adjustment in dB. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {19,22}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->play(
        [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
        my $loop = 0,
    )

    # Increase volume
    $action->volume(10.0);

    # Decrease volume
    $action->volume(-5.0);

});
$client->run();
```
