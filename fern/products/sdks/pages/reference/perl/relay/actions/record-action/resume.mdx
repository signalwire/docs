Resume a paused recording.

## **Returns**

`dict` -- Server acknowledgment.

## **Example**

```perl {17}
use SignalWire::Relay::Client;

my $client = SignalWire::Relay::Client->new(
    project => "your-project-id",
    token => "your-api-token",
    host => "your-space.signalwire.com",
    contexts => ["default"],
)

$client->on_call(sub {
    my ($call) = @_;
    $call->answer();
    my $action = $call->record(audio => {"direction": "both", "format": "wav"});

    $action->pause();
    # ... sensitive exchange ...
    $action->resume();

    my $event = $action->wait();
    print "Recording URL: ${$event->params->get('record', {})->get('url')}";

});
$client->run();
```
