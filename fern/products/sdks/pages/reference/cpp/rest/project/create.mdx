### tokens.create

Create a new API token for the project.

## **Parameters**

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Token configuration fields (e.g., `name`, `scopes`).
</ParamField>

## **Returns**

`nlohmann::json` -- The newly created token object, including the token value.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

token = client.project.tokens.create("ci-token");
```
