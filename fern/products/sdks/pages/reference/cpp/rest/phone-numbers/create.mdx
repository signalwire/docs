Purchase a phone number. The exact keyword arguments depend on the number being
purchased (typically obtained from a prior `search()` call).

## **Parameters**

<ParamField path="**kwargs" type="nlohmann::json" required={true} toc={true}>
  Number details. Common fields include `number` (the E.164 phone number to
  purchase).
</ParamField>

## **Returns**

`nlohmann::json` -- The newly purchased phone number object.

## **Example**

```cpp {13}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

// Purchase the first available number from a search
available = client.phone_numbers.search("512", 1);
numbers = available.value("data", []);
if (numbers) {
    purchased = client.phone_numbers.create(numbers[0]{"number"});
    std::cout << "Purchased:", purchased{"number"} << std::endl;


```
