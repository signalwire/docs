Create a room token that a participant uses to connect to a video room. The token
encodes the room name, user identity, and granted permissions.

## **Parameters**

<ParamField path="room_name" type="std::string" toc={true}>
  The name of the room to join.
</ParamField>

<ParamField path="user_name" type="std::string" toc={true}>
  The display name of the participant.
</ParamField>

<ParamField path="permissions" type="std::vector<std::string>" toc={true}>
  List of permissions to grant.

  - `"room.`audio_mute`"` -- allow the participant to mute their own audio
  - `"room.`audio_unmute`"` -- allow the participant to unmute their own audio
  - `"room.`video_mute`"` -- allow the participant to turn off their own video
  - `"room.`video_unmute`"` -- allow the participant to turn on their own video
  - `"room.list_available_layouts"` -- allow the participant to list available room layouts
  - `"room.set_layout"` -- allow the participant to change the room layout
</ParamField>

Additional keyword arguments define the token configuration (expiration, metadata, etc.).

## **Returns**

`nlohmann::json` -- The created token, including the JWT string.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

token = client.video.room_tokens.create(;
    room_name="standup",
    user_name="Alice",
    permissions=[;
        "room.audio_mute",
        "room.audio_unmute",
        "room.video_mute",
        "room.video_unmute",
    ];
);
std::cout << token{"token"} << std::endl;


```
