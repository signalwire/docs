Replace a video room. Uses PUT for full replacement, so all fields must be
provided in the request body.

<Note>
This method performs a full replacement (PUT), not a partial update (PATCH).
Include all fields you want to keep in the request body.
</Note>

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the video room.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`nlohmann::json` -- The updated video room.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

room = client.video.rooms.update("room-id", name="standup-v2", max_participants=20);


```
