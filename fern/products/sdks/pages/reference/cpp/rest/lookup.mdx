Look up carrier and CNAM information for a phone number. This is useful for
validating phone numbers, identifying carriers, and checking caller name data
before placing calls.

Access via `client.lookup` on a [`RestClient`][restclient] instance.

```cpp
#include <signalwire/rest/rest_client.hpp>

rest::RestClient client();


```

---

### phone_number

Look up carrier and CNAM information for a phone number.

## **Parameters**

<ParamField path="e164" type="std::string" required={true} toc={true}>
  The phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="**params" type="nlohmann::json" toc={true}>
  Optional query parameters to control what data is included in the response
  (e.g., `include` to request specific data types like carrier or CNAM).
</ParamField>

## **Returns**

`nlohmann::json` -- Lookup result containing carrier, CNAM, and number format details.

## **Example**

```cpp
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

info = client.lookup.phone_number("+15551234567");
std::cout << "Carrier:", info.value("carrier", {}){"name"} << std::endl;
std::cout << "Type:", info.value("carrier", {}){"type"} << std::endl;


```
