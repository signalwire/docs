The REST namespace provides a synchronous HTTP client for the SignalWire platform
APIs. It organizes every HTTP endpoint into namespaced resource objects with
standard CRUD operations, letting you manage phone numbers, fabric resources,
call logs, video rooms, datasphere documents, and more from C++.

## Example

Search for available phone numbers, purchase one, and assign it to a fabric
AI agent resource:

```cpp
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

// Search for available numbers in area code 512
available = client.phone_numbers.search("512", 3);
for (const auto& number : available.value("data", [])) {
    std::cout << std::string(number['number']) + " - " + std::string(number.value("region", "")) << std::endl;

// Purchase the first available number
purchased = client.phone_numbers.create(available{"data"}[0]{"number"});
std::cout << "Purchased: " + std::string(purchased['number']) << std::endl;

// List your AI agent resources
response = client.fabric.ai_agents.list();
for (const auto& agent : response.value("data", [])) {
    std::cout << "Agent: " + std::string(agent['name']) + " (" + std::string(agent['id']) + ")" << std::endl;

// Query recent voice call logs
logs = client.logs.voice.list(5);
for (const auto& log : logs.value("data", [])) {
    std::cout << "Call from " + std::string(log.value("from", "")) + " to " + std::string(log.value("to", "")) << std::endl;


```

<Note>
All three constructor arguments can also be provided via environment variables:
`SIGNALWIRE_PROJECT_ID`, `SIGNALWIRE_API_TOKEN`, and `SIGNALWIRE_SPACE`.
When those are set, you can instantiate with `RestClient()` and no arguments.
</Note>

## Error Handling

REST errors raise `SignalWireRestError`:

```cpp
#include <signalwire/rest/rest_client.hpp>

rest::RestClient client();

try {
    client.phone_numbers.get("nonexistent-id");
} catch (const rest::SignalWireRestError& e) {
    std::cout << "HTTP " + std::string(e.status_code) + ": " + std::string(e.body) << std::endl;
    std::cout << "URL: " + std::string(e.method) + " " + std::string(e.url) << std::endl;


```
