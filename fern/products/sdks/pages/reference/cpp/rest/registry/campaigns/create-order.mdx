Create a number assignment order to assign phone numbers to a campaign.

## **Parameters**

<ParamField path="campaign_id" type="std::string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="phone_number_ids" type="std::vector<std::string>" toc={true}>
  List of phone number resource IDs to assign.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional order parameters.
</ParamField>

## **Returns**

`nlohmann::json` -- The newly created order object.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

order = client.registry.campaigns.create_order(;
    "campaign-id",
    phone_number_ids={"number-id-1", "number-id-2"},
);
std::cout << "Order ID:", order{"id"} << std::endl;


```
