Update a 10DLC campaign. Uses PUT to replace fields.

## **Parameters**

<ParamField path="campaign_id" type="std::string" required={true} toc={true}>
  The campaign resource ID.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Fields to update (e.g., `description`, `usecase`).
</ParamField>

## **Returns**

`nlohmann::json` -- The updated campaign object.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.registry.campaigns.update(;
    "campaign-id",
    "Updated campaign description",
);
std::cout << updated{"description"} << std::endl;


```
