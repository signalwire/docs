Register a new 10DLC brand.

## **Parameters**

<ParamField path="entity_type" type="std::string" toc={true}>
  The entity type. Valid values:
  - `"PRIVATE_PROFIT"` -- privately held for-profit company
  - `"PUBLIC_PROFIT"` -- publicly traded for-profit company
  - `"NON_PROFIT"` -- non-profit organization
</ParamField>

<ParamField path="display_name" type="std::string" toc={true}>
  Display name for the brand.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional brand registration fields (e.g., `ein`, `website`, `vertical`).
</ParamField>

## **Returns**

`nlohmann::json` -- The newly created brand object.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

brand = client.registry.brands.create(;
    entity_type="PRIVATE_PROFIT",
    display_name="My Company",
    website="https://example.com",
);
std::cout << "Brand ID:", brand{"id"} << std::endl;


```
