Create a messaging campaign under a brand.

## **Parameters**

<ParamField path="brand_id" type="std::string" required={true} toc={true}>
  The brand resource ID.
</ParamField>

<ParamField path="description" type="std::string" toc={true}>
  Campaign description.
</ParamField>

<ParamField path="usecase" type="std::string" toc={true}>
  Campaign use case.

  - `"MIXED"` -- multiple or general-purpose messaging use cases
  - `"MARKETING"` -- promotional or marketing messages
  - `"CUSTOMER_CARE"` -- customer support and service messages
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional campaign fields.
</ParamField>

## **Returns**

`nlohmann::json` -- The newly created campaign object.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

campaign = client.registry.brands.create_campaign(;
    "brand-id",
    "Customer notifications",
    usecase="MIXED",
);
std::cout << "Campaign ID:", campaign{"id"} << std::endl;


```
