Send an MFA verification code via voice call.

## **Parameters**

<ParamField path="to" type="std::string" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="std::string" required={true} toc={true}>
  The caller phone number in E.164 format.
  The `from` parameter specifies the sender number.
</ParamField>

<ParamField path="message" type="std::string" required={true} toc={true}>
  The message to speak. Include `{code}` as a placeholder.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional parameters (e.g., `token_length`, `max_attempts`, `valid_for`).
</ParamField>

## **Returns**

`nlohmann::json` -- Response containing the `id` of the MFA request.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

result = client.mfa.call(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your verification code is {code}",
});
request_id = result{"id"};


```
