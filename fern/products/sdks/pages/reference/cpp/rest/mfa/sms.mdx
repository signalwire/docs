Send an MFA verification code via SMS.

## **Parameters**

<ParamField path="to" type="std::string" required={true} toc={true}>
  The recipient phone number in E.164 format.
</ParamField>

<ParamField path="from" type="std::string" required={true} toc={true}>
  The sender phone number in E.164 format. Must be a number in your project.
  The `from` parameter specifies the sender number.
</ParamField>

<ParamField path="message" type="std::string" required={true} toc={true}>
  The message body. Include `{code}` as a placeholder for the generated code
  (e.g., `"Your verification code is: {code}"`).
</ParamField>

<ParamField path="token_length" type="int" toc={true}>
  Length of the generated code.
</ParamField>

<ParamField path="max_attempts" type="int" toc={true}>
  Maximum number of verification attempts allowed.
</ParamField>

<ParamField path="valid_for" type="int" toc={true}>
  How long the code remains valid, in seconds.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional parameters.
</ParamField>

## **Returns**

`nlohmann::json` -- Response containing the `id` of the MFA request (used in `verify()`).

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

result = client.mfa.sms(**{
    "to": "+15559876543",
    "from": "+15551234567",
    "message": "Your code is: {code}",
});
request_id = result{"id"};
std::cout << "MFA request:", request_id << std::endl;


```
