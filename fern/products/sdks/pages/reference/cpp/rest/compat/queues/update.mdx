Update a queue's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="std::string" required={true} toc={true}>
  The queue SID.
</ParamField>

<ParamField path="FriendlyName" type="std::string" toc={true}>
  A human-readable name for the queue.
</ParamField>

<ParamField path="MaxSize" type="int" toc={true}>
  Maximum number of calls allowed in the queue.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated queue resource.
