List conferences.

## **Parameters**

<ParamField path="Status" type="std::string" toc={true}>
  Filter by conference status.

  - `"init"` -- conference has been created but no participants have joined yet
  - `"in-progress"` -- conference is currently active with participants
  - `"completed"` -- conference has ended
</ParamField>

<ParamField path="FriendlyName" type="std::string" toc={true}>
  Filter by the conference's friendly name.
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`nlohmann::json` -- Paginated response containing conference resources.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

conferences = client.compat.conferences.list("in-progress");
```
