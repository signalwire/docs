Search for available toll-free phone numbers in a specific country.

## **Parameters**

<ParamField path="country" type="std::string" required={true} toc={true}>
  The ISO 3166-1 alpha-2 country code (e.g., `"US"`).
</ParamField>

<ParamField path="Contains" type="std::string" toc={true}>
  Filter by number pattern (e.g., `"800"`).
</ParamField>

<ParamField path="PageSize" type="int" toc={true}>
  Number of results per page.
</ParamField>

## **Returns**

`nlohmann::json` -- Paginated response containing available toll-free number resources.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

numbers = client.compat.phone_numbers.search_toll_free("US", Contains="800");


```
