Update an API token. Uses PATCH.

## **Parameters**

<ParamField path="token_id" type="std::string" required={true} toc={true}>
  The token identifier.
</ParamField>

<ParamField path="Ttl" type="int" toc={true}>
  New time-to-live in seconds for the token.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated token resource.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.compat.tokens.update("token-id", Ttl=7200);


```
