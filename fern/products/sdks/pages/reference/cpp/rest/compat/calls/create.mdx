Initiate a new outbound call.

## **Parameters**

<ParamField path="To" type="std::string" required={true} toc={true}>
  The destination phone number in E.164 format.
</ParamField>

<ParamField path="From" type="std::string" required={true} toc={true}>
  The caller ID phone number (must be a number you own).
</ParamField>

<ParamField path="Url" type="std::string" toc={true}>
  The URL that returns LAML instructions when the call connects.
</ParamField>

<ParamField path="StatusCallback" type="std::string" toc={true}>
  URL to receive call status webhook events.
</ParamField>

<ParamField path="StatusCallbackEvent" type="std::vector<std::string>" toc={true}>
  Events to send to the status callback.

  - `"initiated"` -- the call request has been created
  - `"ringing"` -- the destination is ringing
  - `"answered"` -- the call has been answered
  - `"completed"` -- the call has ended
</ParamField>

## **Returns**

`nlohmann::json` -- The created call resource including the call SID.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

call = client.compat.calls.create(;
    To="+15559876543",
    From="+15551234567",
    Url="https://example.com/voice";
);
std::cout << call{"sid"} << std::endl;


```
