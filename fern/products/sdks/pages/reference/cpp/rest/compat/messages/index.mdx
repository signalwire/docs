Manage SMS and MMS messages with CRUD operations and media sub-resources.

Access via `client.compat.messages` on a [`RestClient`][restclient] instance.

```cpp
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

messages = client.compat.messages.list();


```
