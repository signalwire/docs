Update a LAML bin's content or name. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="std::string" required={true} toc={true}>
  The LAML bin SID.
</ParamField>

<ParamField path="FriendlyName" type="std::string" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="std::string" toc={true}>
  The updated cXML/LaML script content.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated LAML bin resource.
