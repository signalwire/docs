Update a message (e.g., redact the body of a sent message).

## **Parameters**

<ParamField path="sid" type="std::string" required={true} toc={true}>
  The message SID.
</ParamField>

<ParamField path="Body" type="std::string" toc={true}>
  Set to an empty string to redact the message body.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated message resource.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.compat.messages.update("SM...", Body="");


```
