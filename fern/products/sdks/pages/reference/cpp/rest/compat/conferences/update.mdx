Update a conference (e.g., end it, or set the announce URL).

## **Parameters**

<ParamField path="sid" type="std::string" required={true} toc={true}>
  The conference SID.
</ParamField>

<ParamField path="Status" type="std::string" toc={true}>
  Set to `"completed"` to end the conference.
</ParamField>

<ParamField path="AnnounceUrl" type="std::string" toc={true}>
  URL returning LAML to play an announcement to all participants.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated conference resource.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.compat.conferences.update("CF...", Status="completed");


```
