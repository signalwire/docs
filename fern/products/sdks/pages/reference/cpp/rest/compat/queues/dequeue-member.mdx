Dequeue a member from a queue, redirecting the call to a new LAML URL.

## **Parameters**

<ParamField path="queue_sid" type="std::string" required={true} toc={true}>
  The SID of the queue.
</ParamField>

<ParamField path="call_sid" type="std::string" required={true} toc={true}>
  The call SID of the queue member to dequeue.
</ParamField>

<ParamField path="Url" type="std::string" toc={true}>
  The URL that returns LAML instructions for the dequeued call.
</ParamField>

<ParamField path="Method" type="std::string" toc={true}>
  HTTP method for the URL. Defaults to `"POST"`.
</ParamField>

## **Returns**

`nlohmann::json` -- The dequeued member resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.compat.queues.dequeue_member(;
    "QU...", "CA...",
    Url="https://example.com/dequeue";
);


```
