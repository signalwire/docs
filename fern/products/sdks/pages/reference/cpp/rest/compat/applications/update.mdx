Update an application's configuration. Uses POST (Twilio convention).

## **Parameters**

<ParamField path="sid" type="std::string" required={true} toc={true}>
  The application SID.
</ParamField>

<ParamField path="FriendlyName" type="std::string" toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="std::string" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="SmsUrl" type="std::string" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="StatusCallback" type="std::string" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated application resource.

    "AP...",
    VoiceUrl="https://example.com/new-voice";
