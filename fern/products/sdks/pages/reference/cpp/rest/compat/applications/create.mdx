Create a new application.

## **Parameters**

<ParamField path="FriendlyName" type="std::string" required={true} toc={true}>
  A human-readable name for the application.
</ParamField>

<ParamField path="VoiceUrl" type="std::string" toc={true}>
  URL to fetch LAML instructions when a call is received.
</ParamField>

<ParamField path="VoiceMethod" type="std::string" toc={true}>
  HTTP method for the voice URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="SmsUrl" type="std::string" toc={true}>
  URL to fetch LAML instructions when an SMS is received.
</ParamField>

<ParamField path="SmsMethod" type="std::string" toc={true}>
  HTTP method for the SMS URL. Defaults to `"POST"`.
</ParamField>

<ParamField path="StatusCallback" type="std::string" toc={true}>
  URL to receive status webhook events.
</ParamField>

## **Returns**

`nlohmann::json` -- The created application resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

app = client.compat.applications.create(;
    FriendlyName="My App",
    VoiceUrl="https://example.com/voice",
    SmsUrl="https://example.com/sms";
);


```
