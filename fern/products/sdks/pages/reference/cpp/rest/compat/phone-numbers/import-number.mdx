Import an external phone number into the account.

## **Parameters**

<ParamField path="PhoneNumber" type="std::string" required={true} toc={true}>
  The phone number to import in E.164 format.
</ParamField>

<ParamField path="FriendlyName" type="std::string" toc={true}>
  A human-readable label for the number.
</ParamField>

## **Returns**

`nlohmann::json` -- The imported phone number resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

number = client.compat.phone_numbers.import_number(;
    PhoneNumber="+15559876543";
);


```
