Send a new SMS or MMS message.

## **Parameters**

<ParamField path="To" type="std::string" required={true} toc={true}>
  The destination phone number in E.164 format.
</ParamField>

<ParamField path="From" type="std::string" required={true} toc={true}>
  The sender phone number (must be a number you own).
</ParamField>

<ParamField path="Body" type="std::string" toc={true}>
  The text body of the message. Required for SMS; optional for MMS if `MediaUrl` is provided.
</ParamField>

<ParamField path="MediaUrl" type="std::vector<std::string>" toc={true}>
  URLs of media to include (for MMS). Up to 10 media URLs.
</ParamField>

<ParamField path="StatusCallback" type="std::string" toc={true}>
  URL to receive message status webhook events.
</ParamField>

## **Returns**

`nlohmann::json` -- The created message resource including the message SID.

## **Example**

```cpp {10,17}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

// Send an SMS
msg = client.compat.messages.create(;
    To="+15559876543",
    From="+15551234567",
    Body="Hello from SignalWire!";
);

// Send an MMS with an image
mms = client.compat.messages.create(;
    To="+15559876543",
    From="+15551234567",
    Body="Check this out",
    MediaUrl={"https://example.com/image.jpg"};
);


```
