Start a media stream on a conference, sending real-time audio to a WebSocket URL.

## **Parameters**

<ParamField path="conference_sid" type="std::string" required={true} toc={true}>
  The SID of the conference.
</ParamField>

<ParamField path="Url" type="std::string" toc={true}>
  The WebSocket URL to stream audio to.
</ParamField>

## **Returns**

`nlohmann::json` -- The created stream resource.

#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

stream = client.compat.conferences.start_stream(;
    "CF...",
    Url="wss://stream.example.com";
