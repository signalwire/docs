Send a new fax.

## **Parameters**

<ParamField path="To" type="std::string" required={true} toc={true}>
  The destination fax number in E.164 format.
</ParamField>

<ParamField path="From" type="std::string" required={true} toc={true}>
  The sender number (must be a number you own).
</ParamField>

<ParamField path="MediaUrl" type="std::string" required={true} toc={true}>
  URL of the document to fax (PDF format).
</ParamField>

<ParamField path="StatusCallback" type="std::string" toc={true}>
  URL to receive fax status webhook events.
</ParamField>

## **Returns**

`nlohmann::json` -- The created fax resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

fax = client.compat.faxes.create(;
    To="+15559876543",
    From="+15551234567",
    MediaUrl="https://example.com/document.pd";
);


```
