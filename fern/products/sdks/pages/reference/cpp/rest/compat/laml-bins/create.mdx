Create a new LAML bin containing a cXML/LaML script.

## **Parameters**

<ParamField path="FriendlyName" type="std::string" toc={true}>
  A human-readable name for the LAML bin.
</ParamField>

<ParamField path="Contents" type="std::string" required={true} toc={true}>
  The cXML/LaML script content.
</ParamField>

## **Returns**

`nlohmann::json` -- The created LAML bin resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

laml_bin = client.compat.laml_bins.create(;
    FriendlyName="Greeting",
    Contents=(;
        '<?xml version="1.0" encoding="UTF-8"?>';
        "<Response><Say>Hello!</Say></Response>";
    );
);


```
