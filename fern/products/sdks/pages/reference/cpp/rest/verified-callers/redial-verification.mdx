Redial the verification call for a caller ID that has not yet been verified.
Use this when the original verification call was missed.

## **Parameters**

<ParamField path="caller_id" type="std::string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

## **Returns**

`nlohmann::json` -- Confirmation that the verification call has been re-initiated.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.verified_callers.redial_verification("caller-id");


```
