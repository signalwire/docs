Submit the verification code received during the verification call or SMS.

## **Parameters**

<ParamField path="caller_id" type="std::string" required={true} toc={true}>
  The caller ID resource ID.
</ParamField>

<ParamField path="verification_code" type="std::string" toc={true}>
  The code received during the verification call or SMS.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional verification parameters.
</ParamField>

## **Returns**

`nlohmann::json` -- The updated caller ID object with verified state.
