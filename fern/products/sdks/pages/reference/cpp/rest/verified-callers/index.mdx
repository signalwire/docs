Manage verified caller IDs for your SignalWire project. Before using a phone
number as a caller ID for outbound calls, it must be verified. This resource
provides full CRUD (with PUT for updates) plus a two-step verification flow.

Access via `client.verified_callers` on a [`RestClient`][restclient] instance.

```cpp
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

callers = client.verified_callers.list();
for (const auto& caller : callers.value("data", [])) {
    std::cout << caller{"id"}, caller{"phone_number"} << std::endl;


```
