Create a new caller ID entry and initiate verification. SignalWire will place
a phone call or send an SMS to verify ownership.

## **Parameters**

<ParamField path="phone_number" type="std::string" toc={true}>
  The phone number to verify in E.164 format.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional parameters (e.g., `friendly_name`).
</ParamField>

## **Returns**

`nlohmann::json` -- The newly created caller ID object with its verification state.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

caller = client.verified_callers.create("+15551234567");
std::cout << "Verification state:", caller{"verification_state"} << std::endl;


```
