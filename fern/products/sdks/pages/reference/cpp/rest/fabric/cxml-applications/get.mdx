Retrieve a single cXML application resource.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the cXML application.
</ParamField>

## **Returns**

`nlohmann::json` -- The cXML application resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

app = client.fabric.cxml_applications.get("app-id");
std::cout << "cXML Application: " + std::string(app['name']) + ", ID: " + std::string(app['id']) << std::endl;


```
