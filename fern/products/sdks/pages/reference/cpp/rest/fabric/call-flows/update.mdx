Replace a call flow resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`nlohmann::json` -- The updated call flow resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.call_flows.update("flow-id", name="updated-name");
std::cout << "Updated: " + std::string(updated['name']) << std::endl;


```
