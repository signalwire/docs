Retrieve a single cXML script resource.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

## **Returns**

`nlohmann::json` -- The cXML script resource.

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

item = client.fabric.cxml_scripts.get("resource-id");
```
