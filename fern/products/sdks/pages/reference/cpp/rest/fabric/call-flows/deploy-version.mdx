Deploy a specific version of a call flow resource.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the call flow.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`nlohmann::json` -- The deployed version details.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

result = client.fabric.call_flows.deploy_version(;
    "call-flow-id", version_id="version-id",
);
std::cout << "Deployed version: " + std::string(result['id']) << std::endl;


```
