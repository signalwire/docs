Assign a domain application to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`nlohmann::json` -- The assigned domain application.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

result = client.fabric.resources.assign_domain_application(;
    "resource-id", domain="example.signalwire.com",
);
std::cout << "Domain application assigned: " + std::string(result['id']) << std::endl;


```
