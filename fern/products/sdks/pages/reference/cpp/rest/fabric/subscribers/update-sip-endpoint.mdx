Partially update a SIP endpoint for a subscriber. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="subscriber_id" type="std::string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="std::string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`nlohmann::json` -- The updated SIP endpoint.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.subscribers.update_sip_endpoint(;
    "subscriber-id", "endpoint-id", username="new-username",
);
std::cout << "Updated: " + std::string(updated['username']) << std::endl;


```
