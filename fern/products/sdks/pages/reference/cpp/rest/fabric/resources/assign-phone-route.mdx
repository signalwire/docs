Assign a phone route to a Fabric resource.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`nlohmann::json` -- The assigned phone route.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

result = client.fabric.resources.assign_phone_route(;
    "resource-id", phone_number="+15551234567",
);
std::cout << "Phone route assigned: " + std::string(result['id']) << std::endl;


```
