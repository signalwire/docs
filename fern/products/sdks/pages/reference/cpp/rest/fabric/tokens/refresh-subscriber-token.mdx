Refresh an existing subscriber authentication token.

## **Parameters**

<ParamField path="token" type="std::string" required={true} toc={true}>
  The existing subscriber token to refresh.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`nlohmann::json` -- The refreshed subscriber token.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

refreshed = client.fabric.tokens.refresh_subscriber_token("existing-token");
std::cout << "Refreshed token: " + std::string(refreshed['token']) << std::endl;


```
