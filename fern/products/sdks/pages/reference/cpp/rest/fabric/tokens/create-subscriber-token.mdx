Create a subscriber authentication token.

## **Parameters**

<ParamField path="subscriber_id" type="std::string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are passed directly to the API as the request body.

## **Returns**

`nlohmann::json` -- The created subscriber token.

#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

token = client.fabric.tokens.create_subscriber_token("subscriber-id");
