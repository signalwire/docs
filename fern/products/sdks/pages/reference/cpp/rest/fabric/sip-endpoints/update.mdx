Replace a SIP endpoint resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`nlohmann::json` -- The updated SIP endpoint resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.sip_endpoints.update("endpoint-id", name="updated-name");
std::cout << "Updated: " + std::string(updated['name']) << std::endl;


```
