Retrieve a single SIP endpoint for a subscriber.

## **Parameters**

<ParamField path="subscriber_id" type="std::string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

<ParamField path="endpoint_id" type="std::string" required={true} toc={true}>
  The unique identifier of the SIP endpoint.
</ParamField>

## **Returns**

`nlohmann::json` -- The SIP endpoint.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

endpoint = client.fabric.subscribers.get_sip_endpoint("subscriber-id", "endpoint-id");
std::cout << "SIP Endpoint: " + std::string(endpoint['username']) + ", ID: " + std::string(endpoint['id']) << std::endl;


```
