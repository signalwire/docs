Retrieve a single Fabric resource by ID.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the resource.
</ParamField>

## **Returns**

`nlohmann::json` -- The Fabric resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

resource = client.fabric.resources.get("resource-id");
std::cout << std::string(resource['type']) + ": " + std::string(resource['name']) + ", ID: " + std::string(resource['id']) << std::endl;


```
