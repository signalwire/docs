Partially update an AI agent resource. Uses PATCH, so only provided fields are changed.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the AI agent.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`nlohmann::json` -- The updated AI agent resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.ai_agents.update("agent-id", name="updated-name");
std::cout << "Updated: " + std::string(updated['name']) << std::endl;


```
