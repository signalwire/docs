Replace a subscriber resource. Uses PUT, so the full resource body must be provided.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the subscriber.
</ParamField>

Additional keyword arguments are the fields for the replacement resource body.

## **Returns**

`nlohmann::json` -- The updated subscriber resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.subscribers.update("subscriber-id", name="bob");
std::cout << "Updated: " + std::string(updated['name']) << std::endl;


```
