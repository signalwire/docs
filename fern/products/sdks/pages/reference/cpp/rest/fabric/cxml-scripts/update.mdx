Replace a cXML script resource. Uses PUT for full replacement.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the cXML script.
</ParamField>

Additional keyword arguments are the full replacement body.

## **Returns**

`nlohmann::json` -- The updated cXML script resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.cxml_scripts.update("resource-id", name="updated-name");
std::cout << "Updated: " + std::string(updated['name']) << std::endl;


```
