Partially update a cXML webhook resource. Uses PATCH for partial.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The unique identifier of the cXML webhook.
</ParamField>

Additional keyword arguments are the fields to update.

## **Returns**

`nlohmann::json` -- The updated cXML webhook resource.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

updated = client.fabric.cxml_webhooks.update("webhook-id", name="updated-name");
std::cout << "Updated: " + std::string(updated['name']) << std::endl;


```
