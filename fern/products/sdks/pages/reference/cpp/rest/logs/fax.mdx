Query fax log entries. Fax logs provide a read-only view of fax activity
in your project.

Access via `client.logs.fax` on a [`RestClient`][restclient] instance.

```cpp
#include <signalwire/rest/rest_client.hpp>

rest::RestClient client();


```

### list

List fax log entries.

## **Parameters**

<ParamField path="**params" type="nlohmann::json" toc={true}>
  Optional query parameters for filtering and pagination.
</ParamField>

## **Returns**

`nlohmann::json` -- JSON response containing a `data` list of fax log objects.

---

### get

Retrieve a specific fax log entry.

## **Parameters**

<ParamField path="log_id" type="std::string" required={true} toc={true}>
  The fax log ID.
</ParamField>

## **Returns**

`nlohmann::json` -- The fax log object.

## **Examples**

### List fax logs

```cpp
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

faxes = client.logs.fax.list();
for (const auto& fax : faxes.value("data", [])) {
    std::cout << fax{"from"}, "->", fax{"to"}, fax{"status"} << std::endl;


```

### Get a specific fax log

```cpp
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

fax = client.logs.fax.get("fax-log-id");
std::cout << fax{"from"}, "->", fax{"to"} << std::endl;


```
