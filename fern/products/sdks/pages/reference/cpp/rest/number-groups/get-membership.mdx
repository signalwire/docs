Retrieve a specific membership.

<Note>
This method uses the top-level membership endpoint and takes a membership ID
directly, not a group ID.
</Note>

## **Parameters**

<ParamField path="membership_id" type="std::string" required={true} toc={true}>
  The membership resource ID.
</ParamField>

## **Returns**

`nlohmann::json` -- The membership object.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

membership = client.number_groups.get_membership("membership-id");
std::cout << membership{"phone_number_id"} << std::endl;


```
