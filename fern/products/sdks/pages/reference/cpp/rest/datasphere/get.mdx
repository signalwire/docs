Retrieve a specific document by ID.

## **Parameters**

<ParamField path="resource_id" type="std::string" required={true} toc={true}>
  The document resource ID.
</ParamField>

## **Returns**

`nlohmann::json` -- The document object.

## **Example**

```cpp {9-10}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

doc = client.datasphere.documents.get("document-id");
std::cout << doc{"name"}, doc{"id"} << std::endl;


```
