List chunks belonging to a document. Chunks are the individual text segments
created when a document is processed for vector search.

## **Parameters**

<ParamField path="document_id" type="std::string" required={true} toc={true}>
  The document resource ID.
</ParamField>

<ParamField path="**params" type="nlohmann::json" toc={true}>
  Optional query parameters for pagination.
</ParamField>

## **Returns**

`nlohmann::json` -- JSON response containing a `data` list of chunk objects.
