Upload a new document to the [Datasphere][ref-datasphere].

## **Parameters**

<ParamField path="name" type="std::string" toc={true}>
  Document name or title.
</ParamField>

<ParamField path="content" type="std::string" toc={true}>
  Document content text.
</ParamField>

<ParamField path="**kwargs" type="nlohmann::json" toc={true}>
  Additional document fields (e.g., `metadata`).
</ParamField>

## **Returns**

`nlohmann::json` -- The newly created document object.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

doc = client.datasphere.documents.create(;
    "FAQ",
    content="Frequently asked questions and answers...",
);
std::cout << "Document ID:", doc{"id"} << std::endl;


```
