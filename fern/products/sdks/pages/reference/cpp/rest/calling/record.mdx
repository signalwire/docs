Start recording an active call. Returns a `control_id` used to
[pause][pause],
[resume][resume],
or [stop][stop] the recording.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call to record.
</ParamField>

<ParamField path="record" type="nlohmann::json" required={true} toc={true}>
  Recording configuration object. Contains:

  - `audio` — Audio recording settings (e.g., `{"format": "mp3", "stereo": true}`)
</ParamField>

<ParamField path="listen_on" type="std::string" toc={true}>
  Which leg of the call to record.

  - `"sel"` -- record only the audio from the call's own leg
  - `"both"` -- record audio from both legs of the call
</ParamField>

## **Returns**

`nlohmann::json` — The API response containing the `control_id` for managing the recording.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

result = client.calling.record(;
    call_id="call-id-xxx",
    record={"audio": {"format": "mp3", "stereo": true}}
);
control_id = result{"control_id"};


```
