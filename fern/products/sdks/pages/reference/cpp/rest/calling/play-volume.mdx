Adjust the volume of an active playback.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="std::string" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

<ParamField path="volume" type="double" required={true} toc={true}>
  Volume level adjustment. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the volume was adjusted.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.play_volume("call-id-xxx", "ctrl-id", 5.0);


```
