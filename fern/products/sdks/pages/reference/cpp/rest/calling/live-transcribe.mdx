Start live transcription on an active call. Unlike standard transcription,
live transcription streams partial results in real time as the caller speaks,
enabling display of in-progress speech.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="language" type="std::string" toc={true}>
  Language code for transcription (e.g., `"en-US"`).
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming live transcription was started.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.live_transcribe(;
    call_id="call-id-xxx",
    language="en-US",
);


```
