Send a custom user-defined event on an active call. User events allow you
to pass arbitrary data between call control applications and event handlers.
Listeners registered for user events on the RELAY side will receive the
payload you send here.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call to send the event on.
</ParamField>

<ParamField path="name" type="std::string" required={true} toc={true}>
  The name of the custom event (e.g., `"my_app.status_update"`).
</ParamField>

<ParamField path="params" type="nlohmann::json" toc={true}>
  Arbitrary key-value data to include in the event payload.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the event was sent.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.user_event(;
    call_id="call-id-xxx",
    "order.confirmed",
    params={"order_id": "ORD-12345", "amount": 49.99},
);


```
