[ai-hold]: /docs/sdks/reference/cpp/rest/calling/ai-hold

Resume an AI session that was previously put on hold with
[`ai_hold()`][ai-hold].

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call with a held AI session.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the AI session was resumed.

## **Example**

```cpp {14}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

// Hold, perform an action, then resume
client.calling.ai_hold("call-id-xxx");

// ... do some background processing ...

client.calling.ai_unhold("call-id-xxx");


```
