Update parameters on an active call, such as metadata or call settings.

<Note>
Unlike most calling methods, `update()` does **not** take a positional `call_id`
parameter. Pass all fields (including call identification) as keyword arguments.
</Note>

## **Parameters**

<ParamField path="**params" type="nlohmann::json" toc={true}>
  Keyword arguments forwarded to the SignalWire API. Common fields include
  `call_id`, `metadata`, and other call settings.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the update.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.update("call-id-xxx", {{"department", "sales"}});


```
