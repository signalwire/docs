Disconnect (unbridge) a connected call. If two call legs are bridged together,
this separates them without ending either leg.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call to disconnect.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the disconnect.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.disconnect("call-id-xxx");


```
