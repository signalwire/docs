Pause an active recording. The recording can be resumed later with
[`record_resume()`][record-resume].

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="std::string" required={true} toc={true}>
  The control ID returned from [`record()`][record].
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the recording was paused.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.record_pause("call-id-xxx", "ctrl-id");


```
