Stop an active AI session on a call. The AI agent is disconnected but
the call itself remains active, allowing further call control operations.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the AI session was stopped.

## **Example**

```cpp {10}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

// Stop AI and then play a goodbye message
client.calling.ai_stop("call-id-xxx");
client.calling.play(;
    call_id="call-id-xxx",
    play=[{"type": "tts", "text": "Thank you for calling. Goodbye!"}];
);
client.calling.end("call-id-xxx", "hangup");


```
