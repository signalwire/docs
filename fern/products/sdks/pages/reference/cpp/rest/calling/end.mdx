End an active call.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call to end.
</ParamField>

<ParamField path="reason" type="std::string" toc={true}>
  The reason for ending the call.

  - `"hangup"` -- normal call termination
  - `"busy"` -- end the call with a busy signal
  - `"cancel"` -- cancel the call before it is answered
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the call was ended.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.end("call-id-xxx", "hangup");


```
