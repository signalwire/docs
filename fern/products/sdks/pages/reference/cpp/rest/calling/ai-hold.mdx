Put an active AI session on hold. The AI agent stops processing speech
while the call remains connected. This is useful for transferring the
caller to a human agent or performing background operations.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call with an active AI session.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the AI session was put on hold.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.ai_hold("call-id-xxx");


```
