Stop an active playback immediately.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="control_id" type="std::string" required={true} toc={true}>
  The control ID returned from [`play()`][play].
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming playback was stopped.

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

);
```
