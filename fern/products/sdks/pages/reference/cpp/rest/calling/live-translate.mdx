Start live translation on an active call. Spoken audio is transcribed and
translated in real time to a target language.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call.
</ParamField>

<ParamField path="source_language" type="std::string" toc={true}>
  The language being spoken on the call (e.g., `"en-US"`).
</ParamField>

<ParamField path="target_language" type="std::string" toc={true}>
  The language to translate into (e.g., `"es-ES"`).
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming live translation was started.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.live_translate(;
    call_id="call-id-xxx",
    source_language="en-US",
    target_language="es-ES",
);


```
