Stop an in-progress fax send operation on an active call.

<Note>
This method stops a fax send that is already in progress. To initiate fax sending,
use the RELAY client's [`send_fax()`][send-fax] method.
</Note>

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call with an active fax send operation.
</ParamField>

<ParamField path="control_id" type="std::string" required={true} toc={true}>
  The control ID of the fax send operation to stop.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the fax send was stopped.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.send_fax_stop("call-id-xxx", "ctrl-id");


```
