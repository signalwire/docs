Stop an in-progress fax receive operation on an active call.

<Note>
This method stops a fax receive that is already in progress. To initiate fax receiving,
use the RELAY client's [`receive_fax()`][receive-fax] method.
</Note>

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call with an active fax receive operation.
</ParamField>

<ParamField path="control_id" type="std::string" required={true} toc={true}>
  The control ID of the fax receive operation to stop.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the fax receive was stopped.

## **Example**

```cpp {9-9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.receive_fax_stop("call-id-xxx", "ctrl-id");


```
