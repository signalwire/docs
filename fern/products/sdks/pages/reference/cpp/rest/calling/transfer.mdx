Transfer an active call to a new destination. The current call leg is replaced
by a new connection to the specified target.

## **Parameters**

<ParamField path="call_id" type="std::string" required={true} toc={true}>
  The ID of the call to transfer.
</ParamField>

<ParamField path="dest" type="nlohmann::json" required={true} toc={true}>
  The transfer destination. Contains `type` (e.g., `"phone"`, `"sip"`) and
  type-specific parameters such as `to_number`.
</ParamField>

## **Returns**

`nlohmann::json` — The API response confirming the transfer was initiated.

## **Example**

```cpp {9}
#include <signalwire/rest/rest_client.hpp>

client = RestClient(;
);

client.calling.transfer(;
    call_id="call-id-xxx",
    dest={"type": "phone", "params": {{"to_number", "+15559876543"}}}
);


```
