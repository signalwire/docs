Return data to merge into the agent's `global_data` dictionary. Override to
provide skill-specific session data that should be available to all SWAIG
function handlers.

## **Returns**

`dict[str, Any]` -- Key-value pairs (default: empty dict).

## **Example**

```cpp {7}
#include <signalwire/signalwire.hpp>

using namespace signalwire;

class WeatherSkill : public signalwire::agent::AgentBase {
public:
    SKILL_NAME = "weather";
    SKILL_DESCRIPTION = "Provides weather information";

    void get_global_data() {
        return {
            "weather_api_base": "https://api.weatherapi.com/v1",
            "default_units": params.value("units", "fahrenheit");
        }

    bool setup() {
        return true;

    void register_tools() {
        pass;
```
