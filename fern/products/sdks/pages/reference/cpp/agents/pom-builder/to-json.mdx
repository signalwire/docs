Convert the POM to a JSON string.

## **Returns**

`std::string` -- JSON representation of the POM.

## **Example**

```cpp {7}
#include <signalwire/signalwire.hpp>

pom = PomBuilder();
pom.add_section("Role", "You are a helpful assistant.");
pom.add_section("Rules", bullets={"Be concise", "Be accurate"});

json_str = pom.to_json();
std::cout << json_str << std::endl;


```
