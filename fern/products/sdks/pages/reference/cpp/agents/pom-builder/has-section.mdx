## **has_section**

Check if a section with the given title exists.

### **Parameters**

<ParamField path="title" type="std::string" required={true} toc={true}>
  Section title to check.
</ParamField>

### **Returns**

`bool` -- `true` if the section exists, `false` otherwise.

---

## **get_section**

Get a section by title for direct manipulation.

### **Parameters**

<ParamField path="title" type="std::string" required={true} toc={true}>
  Section title to look up.
</ParamField>

### **Returns**

`std::optional<Section>` -- The POM Section object, or `std::nullopt` if not found.

## **Example**

```cpp
#include <signalwire/signalwire.hpp>

pom = PomBuilder();
pom.add_section("Role", "You are a helpful assistant.");

if (pom.has_section("Role")) {
    section = pom.get_section("Role");
    std::cout << "Found section: " + std::string(section.title) << std::endl;
```
