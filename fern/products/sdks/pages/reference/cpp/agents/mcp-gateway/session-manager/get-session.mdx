[session]: /docs/sdks/reference/cpp/agents/mcp-gateway/session-manager

Retrieve an active session by ID. Returns `std::nullopt` if the session does not exist,
has expired, or if its underlying process has died. Automatically calls
`touch()` on the session to reset its expiration timer.

## **Parameters**

<ParamField path="session_id" type="std::string" required={true} toc={true}>
  The session ID to look up.
</ParamField>

## **Returns**

[`Session`][session] or `std::nullopt`

## **Example**

```cpp {4}
#include <signalwire/signalwire.hpp>

manager = SessionManager(config);
session = manager.get_session("call-abc-123");
if (session) {
    std::cout << "Session active for service: " + std::string(session.service_name) << std::endl;
} else {
    std::cout << "Session not found or expired" << std::endl;


```
