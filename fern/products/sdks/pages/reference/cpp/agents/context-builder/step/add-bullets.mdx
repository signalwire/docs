[ref-step]: /docs/sdks/reference/cpp/agents/context-builder/step

Add a POM section with bullet points to the step. Cannot be used with `set_text()`.

## **Parameters**

<ParamField path="title" type="std::string" required={true} toc={true}>
  Section heading.
</ParamField>

<ParamField path="bullets" type="std::vector<std::string>" required={true} toc={true}>
  List of bullet point strings.
</ParamField>

## **Returns**

[`Step`][ref-step] -- Self for method chaining.

## **Example**

```cpp {9}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("my-agent", "/agent");

contexts = agent.define_contexts();
ctx = contexts.add_context("default");
step = ctx.add_step("collect_info");
step.add_section("Task", "Collect the caller's account information.");
step.add_bullets("Required Information", [;
    "Full legal name",
    "Account number (10 digits)",
    "Reason for calling";
]);

agent.serve();


```
