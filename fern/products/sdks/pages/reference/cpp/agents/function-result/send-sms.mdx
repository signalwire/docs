Send an SMS or MMS message to a phone number. At least one of `body` or `media`
must be provided. Generates a SWML `send_sms` verb under the hood.

## **Parameters**

<ParamField path="to_number" type="std::string" required={true} toc={true}>
  Destination phone number in E.164 format (e.g., `"+15551234567"`).
</ParamField>

<ParamField path="from_number" type="std::string" required={true} toc={true}>
  Sender phone number in E.164 format. Must be a number in your SignalWire project.
</ParamField>

<ParamField path="body" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Text body of the message.
</ParamField>

<ParamField path="media" type="std::optional<std::vector<std::string>>" default="std::nullopt" toc={true}>
  List of media URLs to include as MMS attachments.
</ParamField>

<ParamField path="tags" type="std::optional<std::vector<std::string>>" default="std::nullopt" toc={true}>
  Tags to associate with the message for searching and filtering in the
  SignalWire dashboard.
</ParamField>

<ParamField path="region" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Region to originate the message from.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Text Message

```cpp {13}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void send_confirmation(const nlohmann::json& args, const nlohmann::json& raw_data) {
    auto phone = args{"phone_number"};
    auto order_id = args{"order_id"};
        return swaig::FunctionResult("I've sent you a confirmation text.")
            .send_sms(phone, "+15559876543", "Your order " + std::string(order_id) + " has been confirmed!");

}
agent.serve();


```

### MMS with Media

```cpp {13}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void send_receipt(const nlohmann::json& args, const nlohmann::json& raw_data) {
    auto phone = args{"phone_number"};
    auto receipt_url = args{"receipt_url"};
        return swaig::FunctionResult("I've sent your receipt.")
            .send_sms(phone, "+15559876543", "Here's your receipt:", {receipt_url});

}
agent.serve();


```
