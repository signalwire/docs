Start recording the call in the background. The conversation continues while
recording is active.

<Note>
For continuous call recording, omit `initial_timeout` and `end_silence_timeout`.
Those parameters are for voicemail-style recordings that automatically stop on
silence. Use [`stop_record_call()`][stop-record-call] to end continuous recordings.
</Note>

## **Parameters**

<ParamField path="control_id" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Identifier for this recording. Pass the same ID to `stop_record_call()` to
  stop this specific recording.
</ParamField>

<ParamField path="stereo" type="bool" default="false" toc={true}>
  Record in stereo (`true`) or mono (`false`).
</ParamField>

<ParamField path="format" type="std::string" default="wav" toc={true}>
  Recording file format.

  - `"wav"` -- uncompressed WAV audio
  - `"mp3"` -- compressed MP3 audio
</ParamField>

<ParamField path="direction" type="std::string" default="both" toc={true}>
  Audio direction to record.

  - `"speak"` -- what the agent says
  - `"listen"` -- what the caller says
  - `"both"` -- both sides of the conversation
</ParamField>

<ParamField path="terminators" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  DTMF digits that stop recording when pressed (e.g., `"#"`).
</ParamField>

<ParamField path="beep" type="bool" default="false" toc={true}>
  Play a beep tone before recording starts.
</ParamField>

<ParamField path="input_sensitivity" type="double" default="44.0" toc={true}>
  Input sensitivity level for the recording.
</ParamField>

<ParamField path="initial_timeout" type="std::optional<double>" default="std::nullopt" toc={true}>
  Seconds to wait for speech to begin before auto-stopping. Used for
  voicemail-style recordings.
</ParamField>

<ParamField path="end_silence_timeout" type="std::optional<double>" default="std::nullopt" toc={true}>
  Seconds of silence after speech to wait before auto-stopping. Used for
  voicemail-style recordings.
</ParamField>

<ParamField path="max_length" type="std::optional<double>" default="std::nullopt" toc={true}>
  Maximum recording duration in seconds.
</ParamField>

<ParamField path="status_url" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  URL to receive recording status webhook events.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Examples**

### Continuous Recording

```cpp {11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void start_recording(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("Recording started.")
            .record_call("main_recording", true, "mp3");

}
agent.serve();


```

### Voicemail Recording

```cpp {11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void record_voicemail(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("Please leave your message after the beep.")
            .record_call("voicemail", true, 120.0, 3.0);

}
agent.serve();


```
