[functionresult]: /docs/sdks/reference/cpp/agents/function-result

Put the call on hold. The caller hears hold music until the hold is released
or the timeout expires.

## **Parameters**

<ParamField path="timeout" type="int" default="300" toc={true}>
  Maximum hold duration in seconds. Clamped to the range 0--900 (15 minutes max).
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {7,11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void hold_for_agent(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("Please hold while I find an available agent.")
            .hold(60);

}
agent.serve();


```
