[functionresult]: /docs/sdks/reference/cpp/agents/function-result

Make the AI agent speak specific text immediately, bypassing the normal LLM
response flow. The text is spoken in the agent's configured voice.

## **Parameters**

<ParamField path="text" type="std::string" required={true} toc={true}>
  Text for the agent to speak.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {12}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void announce_status(const nlohmann::json& args, const nlohmann::json& raw_data) {
    auto status = args{"status"};
        return swaig::FunctionResult()
            .say("Your order status is: " + std::string(status));

}
agent.serve();


```
