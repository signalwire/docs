Execute a generic RPC method on a call. This is the low-level interface for
cross-call communication. For common operations, prefer the specific helpers
[`rpc_dial()`][rpc-dial],
[`rpc_ai_message()`][rpc-ai-message], and
[`rpc_ai_unhold()`][rpc-ai-unhold].

## **Parameters**

<ParamField path="method" type="std::string" required={true} toc={true}>
  RPC method name (e.g., `"dial"`, `"ai_message"`, `"ai_unhold"`).
</ParamField>

<ParamField path="params" type="std::optional<std::map<std::string, nlohmann::json>>" default="std::nullopt" toc={true}>
  Parameters for the RPC method.
</ParamField>

<ParamField path="call_id" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Target call ID for the RPC command.
</ParamField>

<ParamField path="node_id" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Target node ID for the RPC command.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void execute_custom_rpc(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("Executing RPC.")
            .execute_rpc("ai_message", "target-call-id", {"role": "system", "message_text": "Hello from another call."});

}
agent.serve();


```
