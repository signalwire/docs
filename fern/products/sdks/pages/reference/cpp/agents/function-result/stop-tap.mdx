Stop an active media tap stream.

## **Parameters**

<ParamField path="control_id" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Identifier of the tap to stop. If not provided, the most recently started
  tap is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void stop_monitoring(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("Monitoring stopped.")
            .stop_tap("supervisor_tap");

}
agent.serve();


```
