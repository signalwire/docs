Collect and process a credit card payment during the call. Generates a SWML
`pay` verb that walks the caller through entering card details via DTMF or
voice, then submits to your payment connector endpoint.

## **Parameters**

<ParamField path="payment_connector_url" type="std::string" required={true} toc={true}>
  URL of your payment processing endpoint. SignalWire sends the collected card
  data to this URL for processing.
</ParamField>

<ParamField path="input_method" type="std::string" default="dtm" toc={true}>
  How the caller provides card details.

  - `"dtm"` -- caller enters digits on the keypad
  - `"voice"` -- caller speaks the numbers
</ParamField>

<ParamField path="status_url" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  URL to receive payment status change webhook notifications.
</ParamField>

<ParamField path="payment_method" type="std::string" default="credit-card" toc={true}>
  Payment method type. Currently only `"credit-card"` is supported.
</ParamField>

<ParamField path="timeout" type="int" default="5" toc={true}>
  Seconds to wait for the next DTMF digit before timing out.
</ParamField>

<ParamField path="max_attempts" type="int" default="1" toc={true}>
  Number of retry attempts if payment collection fails.
</ParamField>

<ParamField path="security_code" type="bool" default="true" toc={true}>
  Whether to prompt the caller for the card's security code (CVV).
</ParamField>

<ParamField path="postal_code" type="std::variant<bool, std::string>" default="true" toc={true}>
  Whether to prompt for the billing postal code. Pass `true` to prompt, `false`
  to skip, or a string with the actual postal code to use without prompting.
</ParamField>

<ParamField path="min_postal_code_length" type="int" default="0" toc={true}>
  Minimum number of digits required for the postal code.
</ParamField>

<ParamField path="token_type" type="std::string" default="reusable" toc={true}>
  Payment token type.

  - `"one-time"` -- single-use token
  - `"reusable"` -- token can be charged again later
</ParamField>

<ParamField path="charge_amount" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Amount to charge as a decimal string (e.g., `"49.99"`).
</ParamField>

<ParamField path="currency" type="std::string" default="usd" toc={true}>
  ISO 4217 currency code (e.g., `"usd"`, `"eur"`).
</ParamField>

<ParamField path="language" type="std::string" default="en-US" toc={true}>
  Language for TTS payment prompts (e.g., `"en-US"`, `"es-MX"`).
</ParamField>

<ParamField path="voice" type="std::string" default="woman" toc={true}>
  TTS voice for payment prompts (e.g., `"woman"`, `"man"`).
</ParamField>

<ParamField path="description" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Custom description for the payment transaction.
</ParamField>

<ParamField path="valid_card_types" type="std::string" default="visa mastercard amex" toc={true}>
  Space-separated list of accepted card types.
</ParamField>

<ParamField path="parameters" type="std::optional<std::vector<std::map<std::string, std::string>>>" default="std::nullopt" toc={true}>
  Additional name/value pairs to send to the payment connector.
  Use [`create_payment_parameter()`][create-payment-parameter] to build entries.
</ParamField>

<ParamField path="prompts" type="std::optional<std::vector<std::map<std::string, nlohmann::json>>>" default="std::nullopt" toc={true}>
  Custom prompt configurations to override default payment prompts.
  Use [`create_payment_prompt()`][create-payment-prompt] to build entries.
</ParamField>

<ParamField path="ai_response" type="std::optional<std::string>" default="'The payment status is ${pay_result}...'" toc={true}>
  AI response template after payment completes. The `${pay_result}` variable
  is substituted with the payment outcome. Set to `std::nullopt` to disable.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {7,12}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void collect_payment(const nlohmann::json& args, const nlohmann::json& raw_data) {
    auto amount = args{"amount"};
        return swaig::FunctionResult("I'll collect your payment information now.")
            .pay("https://api.example.com/payment", amount, "usd", true, true);

}
agent.serve();


```
