Stop an active background call recording.

## **Parameters**

<ParamField path="control_id" type="std::optional<std::string>" default="std::nullopt" toc={true}>
  Identifier of the recording to stop. If not provided, the most recent
  recording is stopped.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void stop_recording(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("Recording stopped.")
            .stop_record_call("main_recording");

}
agent.serve();


```
