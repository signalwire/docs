[functionresult]: /docs/sdks/reference/cpp/agents/function-result

Set or update key-value pairs in the global session data. Global data persists
for the entire agent session and is accessible in prompt variable expansion
(`${global_data.key}`) and by all SWAIG functions.

## **Parameters**

<ParamField path="data" type="std::map<std::string, nlohmann::json>" required={true} toc={true}>
  Dictionary of key-value pairs to merge into the global data store. Existing
  keys are overwritten; new keys are added.
</ParamField>

## **Returns**

[`FunctionResult`][functionresult] — self, for chaining.

## **Example**

```cpp {11}
#include <signalwire/agent/agent_base.hpp>
#include <signalwire/swaig/function_result.hpp>

agent::AgentBase agent("my-agent", "/agent");
agent.set_prompt_text("You are a helpful assistant.");

void save_customer_info(const nlohmann::json& args, const nlohmann::json& raw_data) {
        return swaig::FunctionResult("I've noted your information, " + std::string(args.value("name", "")) + ".")
            .update_global_data({, "customer_id": args{"customer_id"}, "customer_name": args{"name"}, "verified": true, }));

}
agent.serve();


```
