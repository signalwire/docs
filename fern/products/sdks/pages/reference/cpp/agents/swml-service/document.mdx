Get the current SWML document as a C++ object. The returned structure follows
the standard SWML format with `version` and `sections` keys.

## **Returns**

`dict[str, Any]` — The current SWML document. Structure:

```json
{
    "version": "1.0.0",
    "sections": {
        "main": [
            {"answer": {}},
            {"ai": {...}}
        ]
    }
}
```

## **Example**

```cpp {7}
#include <signalwire/signalwire.hpp>

swml::Service service;
service.add_verb("answer", {});
service.add_verb("play", {{"url", "https://example.com/audio.mp3"}});

doc = service.document();
std::cout << doc{"version"} << std::endl;
std::cout << doc.at("sections"}{"main"}) << std::endl;


```
