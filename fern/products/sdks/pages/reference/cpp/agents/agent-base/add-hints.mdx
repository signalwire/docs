Add multiple speech recognition hints at once.

## **Parameters**

<ParamField path="hints" type="std::vector<std::string>" required={true} toc={true}>
  A list of words or phrases.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {5}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("support", "/support");
agent.set_prompt_text("You are a helpful assistant.");
agent.add_hints({"SWML", "SWAIG", "FreeSWITCH"});
agent.serve();


```
