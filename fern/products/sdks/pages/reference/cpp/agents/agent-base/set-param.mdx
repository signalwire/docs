Set a single AI parameter by key.

## **Parameters**

<ParamField path="key" type="std::string" required={true} toc={true}>
  Parameter name (e.g., `"temperature"`, `"end_of_speech_timeout"`).
</ParamField>

<ParamField path="value" type="nlohmann::json" required={true} toc={true}>
  Parameter value. Type depends on the parameter being set.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {5}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("support", "/support");
agent.set_prompt_text("You are a helpful assistant.");
agent.set_param("temperature", 0.5);
agent.serve();


```
