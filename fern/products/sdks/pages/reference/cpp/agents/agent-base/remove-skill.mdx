[ref-agentbase]: /docs/sdks/reference/cpp/agents/agent-base

Unload a skill from this agent. The skill's tools and configuration are removed.

## **Parameters**

<ParamField path="skill_name" type="std::string" required={true} toc={true}>
  Name of the skill to remove.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {6}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("assistant", "/assistant");
agent.add_skill("datetime");
agent.add_skill("math");
agent.remove_skill("math");
std::cout << agent.list_skills() << std::endl;


```
