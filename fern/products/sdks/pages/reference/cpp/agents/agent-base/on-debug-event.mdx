[enable-debug-events]: /docs/sdks/reference/cpp/agents/agent-base/enable-debug-events

Register a callback for debug events received at the `/debug_events` endpoint.
Register as a callback.

<Note>
[`enable_debug_events()`][enable-debug-events] must be called before events will be delivered.
</Note>

## **Parameters**

<ParamField path="handler" type="std::function<void(std::string, nlohmann::json)>" required={true} toc={true}>
  Callback function with signature `(const std::string& event_type, const nlohmann::json& data)`.

  - `event_type` -- Event label string (e.g., `"barge"`, `"llm_error"`, `"session_start"`, `"step_change"`)
  - `data` -- Full event payload including `call_id`, `label`, and event-specific fields
</ParamField>

## **Returns**

`void` -- Registers the handler callback.

## **Example**

```cpp {7}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("debug-agent", "/debug");
agent.set_prompt_text("You are a helpful assistant.");
agent.enable_debug_events(1);

void handle_debug(const auto& event_type, const auto& data) {
    if (event_type == "llm_error") {
        std::cout << "LLM error: " + std::string(data) << std::endl;
    } else if (event_type == "barge") {
        std::cout << "Barge detected: " + std::string(data.value("barge_elapsed_ms", "")) + "ms" << std::endl;
    } else if (event_type == "step_change") {
        std::cout << "Step changed to: " + std::string(data.value("step", "")) << std::endl;

}
agent.serve();


```
