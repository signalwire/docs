Check whether a specific skill is currently loaded.

## **Parameters**

<ParamField path="skill_name" type="std::string" required={true} toc={true}>
  Skill name to check.
</ParamField>

## **Returns**

`bool` -- `true` if the skill is loaded, `false` otherwise.

## **Example**

```cpp {5-6}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("assistant", "/assistant");
agent.add_skill("datetime");
std::cout << agent.has_skill("datetime") << std::endl;
std::cout << agent.has_skill("web_search") << std::endl;


```
