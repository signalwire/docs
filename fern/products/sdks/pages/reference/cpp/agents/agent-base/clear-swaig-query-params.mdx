[add-swaig-query-params]: /docs/sdks/reference/cpp/agents/agent-base/add-swaig-query-param
[ref-agentbase]: /docs/sdks/reference/cpp/agents/agent-base

Clear all query parameters previously added with
[`add_swaig_query_params()`][add-swaig-query-params].
After calling this method, SWAIG webhook URLs will no longer include any extra
query parameters.

## **Parameters**

None.

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Examples**

### Reset params in dynamic config

```cpp {8}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("assistant", "/assistant");
agent.set_prompt_text("You are a helpful assistant.");

void dynamic_config(const auto& query_params, const auto& body_params, const auto& headers, const auto& agent) {
    // Start fresh each request to avoid stale params
}
    agent.clear_swaig_query_params();
    tier = query_params.value("tier", "free");
    agent.add_swaig_query_params({"tier": tier});

agent.set_dynamic_config_callback(dynamic_config);
agent.serve();


```
