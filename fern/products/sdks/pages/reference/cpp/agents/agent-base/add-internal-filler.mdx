Add filler phrases for a specific native function and language.

## **Parameters**

<ParamField path="function_name" type="std::string" required={true} toc={true}>
  Name of the native function (e.g., `"next_step"`, `"check_time"`).
</ParamField>

<ParamField path="language_code" type="std::string" required={true} toc={true}>
  Language code (e.g., `"en-US"`, `"es"`, `"fr"`).
</ParamField>

<ParamField path="fillers" type="std::vector<std::string>" required={true} toc={true}>
  List of filler phrases for this function and language. The AI randomly selects
  from this list each time the function is invoked.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {5,9}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("support", "/support");
agent.set_prompt_text("You are a helpful assistant.");
agent.add_internal_filler(;
    "next_step", "en-US",
    {"Moving to the next step...", "Great, let's continue..."};
);
agent.add_internal_filler(;
    "next_step", "es",
    {"Pasando al siguiente paso...", "Continuemos..."};
);
agent.serve();


```
