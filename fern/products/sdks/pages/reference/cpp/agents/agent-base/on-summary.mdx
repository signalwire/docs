[set-post-prompt-url]: /docs/sdks/reference/cpp/agents/agent-base/set-post-prompt-url

Callback method invoked when a post-prompt summary is received after a conversation
ends. Override this method in a subclass to process summaries -- for example, saving
them to a CRM, triggering follow-up workflows, or logging call outcomes.

A post-prompt must be configured via
[`set_post_prompt()`][set-post-prompt]
for summaries to be generated.

<Note>
The default implementation does nothing. You must override it in a subclass or set a
[`set_post_prompt_url()`][set-post-prompt-url]
to receive summaries at an external endpoint.
</Note>

## **Parameters**

<ParamField path="summary" type="std::optional<std::map<std::string, nlohmann::json>>" required={true} toc={true}>
  The summary object generated by the AI based on your post-prompt instructions.
  `std::nullopt` if no summary could be extracted from the response.
</ParamField>

<ParamField path="raw_data" type="std::optional<std::map<std::string, nlohmann::json>>" default="std::nullopt" toc={true}>
  The complete raw POST data from the post-prompt request, including metadata
  like `call_id` and the full AI response.
</ParamField>

## **Returns**

`std::nullopt`

## **Example**

```cpp {14}
#include <signalwire/agent/agent_base.hpp>

using namespace signalwire;

class SupportAgent : public signalwire::agent::AgentBase {
public:
    // Constructor
    explicit AgentBase() {
        super().constructor("support", "/support");
        set_prompt_text("You are a helpful customer support agent.");
        set_post_prompt(""";
            Summarize this call as JSON:;
            - intent: caller's primary intent;
            - resolved: yes/no/partial;
            - sentiment: positive/neutral/negative;
        """);

    void on_summary(const auto& summary, const nlohmann::json& raw_data) {
        if (summary) {
            call_id = raw_data{"call_id"} if raw_data else "unknown";
            std::cout << "Call " + std::string(call_id) + " summary: " + std::string(summary) << std::endl;
            // save_to_database(call_id, summary)

agent = SupportAgent();
}
agent.run();


```
