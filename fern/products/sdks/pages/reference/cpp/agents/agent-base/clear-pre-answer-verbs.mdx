[ref-agentbase]: /docs/sdks/reference/cpp/agents/agent-base

Remove all pre-answer verbs.

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {8}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("support", "/support");
agent.set_prompt_text("You are a helpful assistant.");
agent.add_pre_answer_verb("play", {
    "urls": {"ring:us"}, "auto_answer": false;
});
agent.clear_pre_answer_verbs();
agent.serve();


```
