[ref-agentbase]: /docs/sdks/reference/cpp/agents/agent-base

Register a specific SIP username that should be routed to this agent. Usernames are
matched case-insensitively.

## **Parameters**

<ParamField path="sip_username" type="std::string" required={true} toc={true}>
  SIP username to register for this agent.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {6-7}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("support", "/support");
agent.set_prompt_text("You are a helpful assistant.");
agent.enable_sip_routing();
agent.register_sip_username("help-desk");
agent.register_sip_username("customer-service");
agent.serve();


```
