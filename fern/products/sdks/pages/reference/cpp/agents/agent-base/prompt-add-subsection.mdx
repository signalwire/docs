[ref-agentbase]: /docs/sdks/reference/cpp/agents/agent-base

Add a subsection to an existing section. If the parent section does not exist, it is
created automatically.

## **Parameters**

<ParamField path="parent_title" type="std::string" required={true} toc={true}>
  Title of the parent section.
</ParamField>

<ParamField path="title" type="std::string" required={true} toc={true}>
  Subsection heading.
</ParamField>

<ParamField path="body" type="std::string" default="" toc={true}>
  Subsection body text.
</ParamField>

<ParamField path="bullets" type="std::optional<std::vector<std::string>>" toc={true}>
  Subsection bullet points.
</ParamField>

## **Returns**

[`AgentBase`][ref-agentbase] -- Returns `*this` for method chaining.

## **Example**

```cpp {6,8}
#include <signalwire/agent/agent_base.hpp>

agent::AgentBase agent("support", "/support");
agent.prompt_add_section("Capabilities",
    body="You can help with the following:");
agent.prompt_add_subsection("Capabilities", "Orders",
    body="Look up order status, process returns and exchanges.");
agent.prompt_add_subsection("Capabilities", "Billing",
    body="Check account balances and explain recent charges.");
agent.serve();


```
