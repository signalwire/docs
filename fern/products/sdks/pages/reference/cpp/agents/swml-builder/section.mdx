Add a new named section to the SWML document. Delegates to
[`SWMLService.section()`][swmlservice-add-section].

## **Parameters**

<ParamField path="section_name" type="std::string" required={true} toc={true}>
  Name of the section to create.
</ParamField>

## **Returns**

`Self` -- The builder instance for method chaining.

## **Example**

```cpp {10}
#include <signalwire/signalwire.hpp>
#include <signalwire/signalwire.hpp>

swml::Service service;
builder = SWMLBuilder(service);

doc = (;
    builder;
    .answer();
    .section("greeting");
    .play("say:Hello!");
    .build();
);
std::cout << doc << std::endl;


```
