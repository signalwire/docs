Build and render the complete SWML document as a JSON string. Delegates to
[`SWMLService.render_document()`][swmlservice-render-document].

## **Returns**

`std::string` -- The SWML document serialized as JSON.

## **Example**

```cpp {7}
#include <signalwire/signalwire.hpp>
#include <signalwire/signalwire.hpp>

swml::Service service;
builder = SWMLBuilder(service);

json_str = builder.answer().ai("Hello").to_string();
std::cout << json_str << std::endl;


```
