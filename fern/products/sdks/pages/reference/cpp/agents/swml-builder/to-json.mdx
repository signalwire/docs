Build and return the complete SWML document as a C++ object. Delegates to
[`SWMLService.get_document()`][swmlservice-get-document].

## **Returns**

`dict[str, Any]` -- The complete SWML document.

## **Example**

```cpp {7}
#include <signalwire/signalwire.hpp>
#include <signalwire/signalwire.hpp>

swml::Service service;
builder = SWMLBuilder(service);

doc = builder.answer().hangup().to_json();
std::cout << doc << std::endl;
// {"version": "1.0.0", "sections": {"main": [{"answer": {}}, {"hangup": {}}]}}


```
