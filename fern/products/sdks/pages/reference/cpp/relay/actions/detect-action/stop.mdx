Stop detection immediately.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {18}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.detect(;
        {"type": "machine", "params": {"initial_timeout": 5.0}}
    );

    // Stop detection early
    action.stop();

client.run();


```
