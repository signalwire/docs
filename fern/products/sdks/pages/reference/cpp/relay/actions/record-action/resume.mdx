Resume a paused recording.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {17}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.record({"direction": "both", "format": "wav"});

    action.pause();
    // ... sensitive exchange ...
    action.resume();

    event = action.wait();
    std::cout << "Recording URL: " + std::string(event.params.get('record', {) + ").value("url", "")}" << std::endl;

client.run();


```
