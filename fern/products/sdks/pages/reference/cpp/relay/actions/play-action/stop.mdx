Stop playback immediately.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {16}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.play([{"type": "tts", "text": "Hello!"}]);

    // Stop playback early
    action.stop();

client.run();


```
