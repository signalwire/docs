Stop transcription.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {16}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.transcribe();

    // Stop transcription early
    action.stop();

client.run();
```
