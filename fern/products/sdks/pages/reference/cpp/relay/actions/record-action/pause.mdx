Pause the recording.

## **Parameters**

<ParamField path="behavior" type="std::optional<std::string>" toc={true}>
  Optional pause behavior. Controls what happens to the audio stream while paused
  (e.g., silence insertion).
</ParamField>

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {16}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.record({"direction": "both", "format": "wav"});

    // Pause during sensitive information
    action.pause();
    // ... sensitive exchange ...
    action.resume();

client.run();


```
