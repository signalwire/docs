Adjust the prompt playback volume.

## **Parameters**

<ParamField path="volume" type="double" required={true} toc={true}>
  Volume adjustment in dB. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {19}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.play_and_collect(;
        media=[{"type": "tts", "text": "Press 1 for sales, 2 for support."}],
        collect={"digits": {"max": 1, "digit_timeout": 5.0}},
    );

    // Increase prompt volume
    action.volume(10.0);

client.run();


```
