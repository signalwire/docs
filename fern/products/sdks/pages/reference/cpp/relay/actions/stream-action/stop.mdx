Stop audio streaming.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {19}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.stream(;
        url="wss://example.com/audio-stream",
        track="inbound",
    );

    // Stop the stream when done
    action.stop();

client.run();


```
