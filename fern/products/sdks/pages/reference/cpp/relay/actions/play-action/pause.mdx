Pause audio playback. The playback position is preserved.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {18}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.play(;
        [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
        loop=0,
    );

    action.pause();
    // ... do something ...
    action.resume();

client.run();


```
