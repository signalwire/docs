Adjust the playback volume.

## **Parameters**

<ParamField path="volume" type="double" required={true} toc={true}>
  Volume adjustment in dB. Range: `-40.0` to `40.0`.
</ParamField>

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {19,22}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.play(;
        [{"type": "audio", "url": "https://example.com/hold-music.mp3"}],
        loop=0,
    );

    // Increase volume
    action.volume(10.0);

    // Decrease volume
    action.volume(-5.0);

client.run();


```
