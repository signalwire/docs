Stop payment collection.

## **Returns**

`nlohmann::json` -- Server acknowledgment.

## **Example**

```cpp {20}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    action = call.pay(;
        payment_connector_url="https://example.com/payment-connector",
        charge_amount="29.99",
        currency="USD",
    );

    // Stop payment collection
    action.stop();

client.run();


```
