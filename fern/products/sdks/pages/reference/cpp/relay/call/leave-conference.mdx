Leave an audio conference.

## **Parameters**

<ParamField path="conference_id" type="std::string" required={true} toc={true}>
  The conference ID to leave.
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the leave.

## **Example**

```cpp {18}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Join a conference
    result = call.join_conference("team-standup", beep="onEnter");

    // Later, leave the conference
    call.leave_conference(result{"conference_id"});

client.run();


```
