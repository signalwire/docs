Put the call on hold. The remote party hears hold music or silence while the
call is held.

<Info>
Use [`unhold()`][unhold] to release the call from hold.
</Info>

<Info>
This method emits [`calling.call.hold`][calling-call-hold] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`nlohmann::json` -- Server response confirming the hold operation.

## **Example**

```cpp {15}
#include <signalwire/signalwire.hpp>
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    call.play([{"type": "tts", "text": "Please hold while I look that up."}]);

    result = call.hold();
    std::cout << "Hold started: " + std::string(result) << std::endl;

    // Do some processing...
    asyncio.sleep(5);

    // Take the call off hold
    call.unhold();

client.run();


```
