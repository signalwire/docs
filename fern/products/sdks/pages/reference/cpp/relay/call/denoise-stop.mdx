Stop noise reduction on the call, restoring the original unfiltered audio.

## **Parameters**

None.

## **Returns**

`nlohmann::json` -- Server response confirming noise reduction has stopped.

## **Example**

```cpp {20}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Enable noise reduction
    call.denoise();

    action = call.play([{"type": "tts", "text": "Noise reduction is now active."}]);
    action.wait();

    result = call.denoise_stop();
    std::cout << "Noise reduction stopped: " + std::string(result) << std::endl;

client.run();


```
