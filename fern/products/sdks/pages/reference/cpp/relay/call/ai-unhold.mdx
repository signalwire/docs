Resume an AI agent session from hold. The AI resumes processing the
conversation.

## **Parameters**

<ParamField path="prompt" type="std::optional<std::string>" toc={true}>
  A prompt for the AI to speak upon resuming (e.g., "Thank you for holding.
  I have your information now.").
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the AI unhold.

## **Example**

```cpp {27}
#include <signalwire/signalwire.hpp>
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Start an AI agent
    action = call.ai(;
        prompt={{"text", "You are a support agent."}},
    );

    // Put the AI on hold
    call.ai_hold("One moment please.");

    // Do some processing...
    asyncio.sleep(5);

    // Resume the AI
    call.ai_unhold("Thanks for waiting. I found your information.");

client.run();


```
