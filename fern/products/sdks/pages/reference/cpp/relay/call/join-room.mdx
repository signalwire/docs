[leave-room]: /docs/sdks/reference/cpp/relay/call/leave-room

Join a video/audio room. Rooms provide multi-party communication with
additional features beyond conferences, including video support.

<Info>
Use [`leave_room()`][leave-room] to leave the room.
</Info>

## **Parameters**

<ParamField path="name" type="std::string" required={true} toc={true}>
  Room name to join.
</ParamField>

<ParamField path="status_url" type="std::optional<std::string>" toc={true}>
  URL to receive room status webhooks.
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the join.
