Start receiving a fax on the call. Returns a
[`FaxAction`][faxaction] that resolves when the
fax is fully received or an error occurs.

<Info>
This method emits [`calling.call.fax`][calling-call-fax] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="control_id" type="std::optional<std::string>" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="on_completed" type="std::optional<std::function<nlohmann::json(RelayEvent)>>" toc={true}>
  Callback invoked when the fax reception completes. The event contains
  the received document URL and page count.
</ParamField>

## **Returns**

[`FaxAction`][faxaction] -- An action handle with
`stop()` and `wait()` methods.

## **Example**

```cpp {14}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    action = call.receive_fax();
    event = action.wait();

    fax_result = event.params.value("fax", {});
    document_url = fax_result.value("document", "");
    pages = fax_result.value("pages", 0);
    std::cout << "Received fax: " + std::string(pages) + " pages, document: " + std::string(document_url) << std::endl;
    call.hangup();

client.run();


```
