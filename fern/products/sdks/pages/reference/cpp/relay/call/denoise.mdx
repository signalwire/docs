Start noise reduction on the call. Filters background noise from the audio
to improve clarity.

<Info>
Use [`denoise_stop()`][denoise-stop] to disable noise reduction.
</Info>

<Info>
This method emits [`calling.call.denoise`][calling-call-denoise] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`nlohmann::json` -- Server response confirming noise reduction has started.

## **Example**

```cpp {14}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    result = call.denoise();
    std::cout << "Noise reduction started: " + std::string(result) << std::endl;

    action = call.play([{"type": "tts", "text": "Noise reduction is now active."}]);
    action.wait();

client.run();


```
