[relayevent]: /docs/sdks/reference/cpp/relay/events

Wait for a specific event type on this call, optionally filtered by a predicate
function. This is a one-shot listener -- it resolves on the first matching event
and then removes itself.

## **Parameters**

<ParamField path="event_type" type="std::string" required={true} toc={true}>
  The event type string to wait for.
</ParamField>

<ParamField path="predicate" type="std::optional<std::function<bool(RelayEvent)>>" toc={true}>
  Optional filter function. If provided, the wait only resolves when the
  predicate returns `true` for a received event.
</ParamField>

<ParamField path="timeout" type="std::optional<double>" toc={true}>
  Maximum seconds to wait. Raises `asyncio.TimeoutError` if exceeded. `std::nullopt`
  waits indefinitely.
</ParamField>

## **Returns**

[`RelayEvent`][relayevent] -- The first event matching the type and optional predicate.

## **Example**

```cpp {16}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Wait for a specific play event
    action = call.play([{"type": "tts", "text": "Hello!"}]);
    event = call.wait_for(;
        "calling.call.play",
        predicate=[](const auto& e) { e.params{"state"} == "finished"; },
        30,
    );
    std::cout << "Play finished: " + std::string(event.params) << std::endl;

client.run();


```
