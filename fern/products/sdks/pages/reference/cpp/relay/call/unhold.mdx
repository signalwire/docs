Release the call from hold, resuming normal audio between the parties.

<Info>
This method emits [`calling.call.hold`][calling-call-hold] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

None.

## **Returns**

`nlohmann::json` -- Server response confirming the unhold operation.

## **Example**

```cpp {21}
#include <signalwire/signalwire.hpp>
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();
    call.play([{"type": "tts", "text": "Please hold while I look that up."}]);

    // Put the call on hold
    call.hold();

    // Do some processing...
    asyncio.sleep(5);

    result = call.unhold();
    std::cout << "Call resumed from hold: " + std::string(result) << std::endl;
    call.play([{"type": "tts", "text": "Thanks for holding. I have your answer."}]);

client.run();


```
