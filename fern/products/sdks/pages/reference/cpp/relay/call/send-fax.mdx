Send a fax document on the call. The document must be accessible via URL.
Returns a [`FaxAction`][faxaction] that you can
use to stop the fax or wait for it to complete.

<Info>
This method emits [`calling.call.fax`][calling-call-fax] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="document" type="std::string" required={true} toc={true}>
  URL of the document to fax (PDF or TIFF format).
</ParamField>

<ParamField path="identity" type="std::optional<std::string>" toc={true}>
  Caller identity string (TSI) transmitted with the fax.
</ParamField>

<ParamField path="header_info" type="std::optional<std::string>" toc={true}>
  Header text printed at the top of each fax page.
</ParamField>

<ParamField path="control_id" type="std::optional<std::string>" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="on_completed" type="std::optional<std::function<nlohmann::json(RelayEvent)>>" toc={true}>
  Callback invoked when the fax operation completes.
</ParamField>

## **Returns**

[`FaxAction`][faxaction] -- An action handle with
`stop()` and `wait()` methods.

## **Example**

```cpp {14}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    action = call.send_fax(;
        document="https://example.com/invoice.pd",
        identity="+15559876543",
        header_info="Invoice #12345",
    );
    event = action.wait();

    fax_result = event.params.value("fax", {});
    std::cout << "Fax sent: " + std::string(fax_result.get('pages', 0)) + " pages" << std::endl;
    call.hangup();

client.run();


```
