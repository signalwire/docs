[relayevent]: /docs/sdks/reference/cpp/relay/events

Wait for the call to reach the `ended` state. This is a convenience method
equivalent to waiting for a `calling.call.state` event with `call_state == "ended"`.

## **Parameters**

<ParamField path="timeout" type="std::optional<double>" toc={true}>
  Maximum seconds to wait. Raises `asyncio.TimeoutError` if exceeded. `std::nullopt`
  waits indefinitely.
</ParamField>

## **Returns**

[`RelayEvent`][relayevent] -- The state-change event indicating the call has ended.

## **Example**

```cpp {18}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    action = call.play([{"type": "tts", "text": "Goodbye!"}]);
    action.wait();
    call.hangup();

    event = call.wait_for_ended();
    std::cout << "Call " + std::string(call.call_id) + " has ended: " + std::string(event.params) << std::endl;

client.run();


```
