Start or stop live translation on the call. Translates spoken audio into
another language in real-time.

## **Parameters**

<ParamField path="action" type="nlohmann::json" required={true} toc={true}>
  Action configuration for the live translation.
</ParamField>

<Indent>
<ParamField path="action.action" type="std::string" required={true} toc={true}>
  The action to perform.

  - `"start"` -- begin live translation
  - `"stop"` -- end live translation
</ParamField>

<ParamField path="action.source_language" type="std::string" toc={true}>
  Source language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="action.target_language" type="std::string" toc={true}>
  Target language code (e.g., `"es-ES"`).
</ParamField>
</Indent>

<ParamField path="status_url" type="std::optional<std::string>" toc={true}>
  URL to receive translation results via webhook.
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the operation.

## **Example**

```cpp {15}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Start live translation from English to Spanish
    call.live_translate({
        "action": "start",
        "source_language": "en-US",
        "target_language": "es-ES",
    });

    // Let the call proceed...
    call.wait_for_ended();

client.run();


```
