Remove the call from a queue.

<Info>
This method emits [`calling.call.queue`][calling-call-queue] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="queue_name" type="std::string" required={true} toc={true}>
  Name of the queue to leave.
</ParamField>

<ParamField path="control_id" type="std::optional<std::string>" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="queue_id" type="std::optional<std::string>" toc={true}>
  Specific queue ID to leave (if the call is in multiple queues with the
  same name).
</ParamField>

<ParamField path="status_url" type="std::optional<std::string>" toc={true}>
  URL to receive queue status webhooks.
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the call left the queue.

## **Example**

```cpp {20}
#include <signalwire/signalwire.hpp>
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Enter the queue
    call.queue_enter("support");

    // Remove from queue after a timeout
    asyncio.sleep(300) // 5 minute max wait;
    call.queue_leave("support");
    call.play([{"type": "tts", "text": "No agents available. Please try again later."}]);
    call.hangup();

client.run();


```
