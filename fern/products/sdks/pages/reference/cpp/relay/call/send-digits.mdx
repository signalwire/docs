Send DTMF tones on the call. Use this to navigate IVR menus on outbound calls
or to send tone signals.

<Info>
This method emits [`calling.call.send_digits`][calling-call-send-digits] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="digits" type="std::string" required={true} toc={true}>
  The DTMF digit string to send. Valid characters: `0-9`, `*`, `#`,
  `A-D`, `W` (0.5s pause), `w` (1s pause).
</ParamField>

<ParamField path="control_id" type="std::optional<std::string>" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the send_digits operation.

```cpp {14}
#include <signalwire/relay/client.hpp>

);

// @client.on_call

    call.answer();

    std::cout << "Digits sent: " + std::string(result) << std::endl;

client.run();
```
