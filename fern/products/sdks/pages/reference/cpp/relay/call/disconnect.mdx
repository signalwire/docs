[connect]: /docs/sdks/reference/cpp/relay/call/connect

Disconnect a previously bridged call, breaking the connection between the two
call legs. Both parties return to their respective RELAY applications for
further processing. Use this after
[`connect()`][connect] to separate
bridged calls.

## **Parameters**

None.

## **Returns**

`nlohmann::json` -- Server response confirming the disconnect.

## **Example**

```cpp {20}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Bridge to an agent
    call.connect([;
        [{"type": "phone", "params": {"to_number": "+15551234567", "from_number": "+15559876543"}}],
    ]);

    // Disconnect (unbridge) the connected call
    result = call.disconnect();
    std::cout << "Disconnected: " + std::string(result) << std::endl;

    call.play([{"type": "tts", "text": "The other party has disconnected."}]);
    call.hangup();

client.run();


```
