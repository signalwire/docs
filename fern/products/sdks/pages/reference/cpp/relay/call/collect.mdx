Collect DTMF digit or speech input without playing a prompt. Use this when you
want to listen for input silently or after a prompt has already been played
separately. For collecting input with a prompt, use
[`play_and_collect()`][play-and-collect].

<Info>
This method emits [`calling.call.collect`][calling-call-collect] events. See [Call Events][call-events] for payload details.
</Info>

## **Parameters**

<ParamField path="digits" type="std::optional<nlohmann::json>" toc={true}>
  DTMF digit collection settings.
</ParamField>

<Indent>
<ParamField path="digits.max" type="int" toc={true}>
  Maximum number of digits to collect.
</ParamField>

<ParamField path="digits.digit_timeout" type="double" toc={true}>
  Seconds to wait between digits before completing.
</ParamField>

<ParamField path="digits.terminators" type="std::string" toc={true}>
  Characters that terminate digit collection (e.g., `"#"`).
</ParamField>
</Indent>

<ParamField path="speech" type="std::optional<nlohmann::json>" toc={true}>
  Speech recognition settings.
</ParamField>

<Indent>
<ParamField path="speech.end_silence_timeout" type="double" toc={true}>
  Seconds of silence to wait before finalizing speech input.
</ParamField>

<ParamField path="speech.speech_timeout" type="double" toc={true}>
  Maximum seconds to listen for speech.
</ParamField>

<ParamField path="speech.language" type="std::string" toc={true}>
  Speech recognition language code (e.g., `"en-US"`).
</ParamField>

<ParamField path="speech.hints" type="std::vector<std::string>" toc={true}>
  Words or phrases to boost recognition accuracy.
</ParamField>
</Indent>

<ParamField path="initial_timeout" type="std::optional<double>" toc={true}>
  Seconds to wait for the first input before ending with `no_input`.
</ParamField>

<ParamField path="partial_results" type="std::optional<bool>" toc={true}>
  Enable partial speech recognition results.
</ParamField>

<ParamField path="continuous" type="std::optional<bool>" toc={true}>
  Keep collecting after each result instead of stopping.
</ParamField>

<ParamField path="send_start_of_input" type="std::optional<bool>" toc={true}>
  Send an event when input is first detected.
</ParamField>

<ParamField path="start_input_timers" type="std::optional<bool>" toc={true}>
  Start input timers immediately. If `false`, call
  `action.start_input_timers()` to start them manually.
</ParamField>

<ParamField path="control_id" type="std::optional<std::string>" toc={true}>
  Custom control ID. Auto-generated if not provided.
</ParamField>

<ParamField path="on_completed" type="std::optional<std::function<nlohmann::json(RelayEvent)>>" toc={true}>
  Callback invoked when collection completes.
</ParamField>

## **Returns**

[`StandaloneCollectAction`][standalonecollectaction] -- An action
handle with `stop()`, `start_input_timers()`, and `wait()` methods.

## **Example**

```cpp {15}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Collect digits silently (e.g., extension entry)
    action = call.collect(;
        digits={"max": 4, "digit_timeout": 3, "terminators": "#"},
        initial_timeout=10,
    );
    event = action.wait();

    result = event.params.value("result", {});
    extension = result.value("digits", "");
    std::cout << "Extension entered: " + std::string(extension) << std::endl;

client.run();


```
