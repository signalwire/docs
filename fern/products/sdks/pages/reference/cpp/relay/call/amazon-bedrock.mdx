Connect the call to an Amazon Bedrock AI agent. Similar to
[`ai()`][ai] but uses Amazon Bedrock as
the LLM backend.

## **Parameters**

<ParamField path="prompt" type="std::optional<nlohmann::json>" toc={true}>
  The prompt configuration for the Bedrock agent.
</ParamField>

<ParamField path="SWAIG" type="std::optional<nlohmann::json>" toc={true}>
  SWAIG configuration for tool/function definitions.
</ParamField>

<ParamField path="ai_params" type="std::optional<nlohmann::json>" toc={true}>
  AI parameters for the Bedrock session.
</ParamField>

<ParamField path="global_data" type="std::optional<nlohmann::json>" toc={true}>
  Data accessible to the AI and SWAIG functions.
</ParamField>

<ParamField path="post_prompt" type="std::optional<nlohmann::json>" toc={true}>
  Post-prompt configuration.
</ParamField>

<ParamField path="post_prompt_url" type="std::optional<std::string>" toc={true}>
  URL to receive the post-prompt result.
</ParamField>

## **Returns**

`nlohmann::json` -- Server response confirming the Bedrock session.

## **Example**

```cpp {15}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Start an Amazon Bedrock AI agent
    result = call.amazon_bedrock(;
        prompt={{"text", "You are a helpful assistant."}},
        ai_params={"barge_confidence": 0.02},
    );

client.run();


```
