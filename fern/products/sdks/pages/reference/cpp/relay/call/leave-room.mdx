Leave the current room.

## **Parameters**

None.

## **Returns**

`nlohmann::json` -- Server response confirming the leave.

## **Example**

```cpp {18}
#include <signalwire/relay/client.hpp>

client = RelayClient(;
);

// @client.on_call
void handle_call(const auto& call) {
    call.answer();

    // Join a room
    call.join_room("meeting-room-1");

    // Later, leave the room
    call.leave_room();
    call.hangup();

client.run();


```
