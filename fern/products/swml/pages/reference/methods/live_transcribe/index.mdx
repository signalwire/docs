Start live transcription of the call. The transcription will be sent to the specified webhook URL.

<ParamField path="live_transcribe" type="object" required={true}>
  An object that accepts the [`live_transcribe parameters`](#live_transcribe-parameters).
</ParamField>

## **live_transcribe Parameters**

<ParamField path="live_transcribe.action" type="string | object" required={true}>
  The action to perform.
</ParamField>

An object that contains **one** of the [`live_transcribe actions`](./action/index.mdx).
**Possible Values:** [[`stop`](/swml/methods/live_transcribe/action/stop), [`start`](/swml/methods/live_transcribe/action/start), [`summarize`](/swml/methods/live_transcribe/action/summarize)]
