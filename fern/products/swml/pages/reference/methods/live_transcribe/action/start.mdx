Start a live translation session.

<Schema type="TranscribeStartAction" />

## **Example**

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
live_transcribe:
  action:
    start:
      webhook: 'https://example.com/webhook'
      lang: en
      live_events: true
      ai_summary: true
      ai_summary_prompt: Summarize this conversation
      speech_timeout: 60000
      vad_silence_ms: 500
      vad_thresh: 400
      debug_level: 0
      direction:
        - remote-caller
        - local-caller
      speech_engine: deepgram
```
</CodeBlock>
