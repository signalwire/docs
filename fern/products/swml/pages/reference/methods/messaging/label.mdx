Mark any point of the current SWML section with a label so that
[`goto`](/docs/swml/reference/messaging/goto) can jump to it.

## **Properties**

<ParamField path="label" type="string" required={true} toc={true}>
  The label name. Must be unique within the section.
</ParamField>

## **Examples**

### Label paired with goto for retry

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - label: try_lookup
    - request:
        url: "https://api.example.com/lookup"
        save_variables: true
    - switch:
        variable: request_result
        case:
          timeout:
            - goto:
                label: try_lookup
                max: 3
        default:
          - reply:
              body: "Lookup complete."
```
</CodeBlock>
