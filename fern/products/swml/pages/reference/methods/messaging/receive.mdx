Accept the inbound message without sending a reply. `receive` is a no-op step — it acknowledges the
inbound message and does nothing else.

An empty document (`{}`) is treated as an implicit `receive`.

## **Properties**

<ParamField path="receive" type="object" required={true} toc={true}>
  An empty object. `receive` takes no parameters.
</ParamField>

## **Examples**

### Simple-mode receive

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
receive: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "receive": {}
}
```
</CodeBlock>
</CodeBlocks>

### Full-mode receive

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - receive: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "receive": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
