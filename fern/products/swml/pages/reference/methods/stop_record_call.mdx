Stop an active background recording.

## **Properties**

<Schema type="StopRecordCall" />

## **Variables**

Read by the method:

- **record_control_id:** (in) control ID of last recording started.

Set by the method:

- **stop_record_call_result:** (out) `success` | `failed`

## **Examples**

### Stop last call recording

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - stop_record_call: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "stop_record_call": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Stop a specific call recording

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - stop_record_call:
        control_id: my-recording-id
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "stop_record_call": {
          "control_id": "my-recording-id"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
