Join a RELAY room. If the room doesn't exist, it creates a new room.

## **Properties**

<Schema type="JoinRoom" />

## **Variables**

Set by the method:

- **join_room_result:** (out) `joined` | `failed`

## **Examples**

### Joining a room

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - join_room:
        name: my_room
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "join_room": {
          "name": "my_room"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
