The default URL to send status callbacks and reports to for all SWAIG functions. If `defaults` is not set, `web_hook_url` may be set in each [SWAIG function](../functions/index.mdx) object.

<ParamField path="defaults.web_hook_url" type="string">
  The URL to send status callbacks and reports to. Authentication can also be set in the url in the format of `username:password@url`.
</ParamField>
