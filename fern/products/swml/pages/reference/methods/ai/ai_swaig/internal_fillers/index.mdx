An array of objects that define language-specific filler phrases for internal SWAIG functions. These fillers help break silence between responses and are played asynchronously during the function call.

<ParamField path="SWAIG.internal_fillers" type="object">
  An object that contains the [internal_fillers parameters][parameters]
</ParamField>


## `internal_fillers` parameters

The `internal_fillers` object contains parameters for various internal functions that the AI Agent can use during conversations.
Each function can have language-specific filler phrases to enhance the conversation flow.
All functions accept the same [object structure][filler-object] for defining fillers.

<ParamField path="internal_fillers.hangup" type="object">
  Filler phrases played when the AI Agent is hanging up the call.
</ParamField>


<ParamField path="internal_fillers.check_time" type="object">
  Filler phrases played when the AI Agent is checking the time.
</ParamField>


<ParamField path="internal_fillers.wait_for_user" type="object">
  Filler phrases played when the AI Agent is waiting for user input.
</ParamField>


<ParamField path="internal_fillers.wait_seconds" type="object">
  Filler phrases played during deliberate pauses or wait periods.
</ParamField>


<ParamField path="internal_fillers.adjust_response_latency" type="object">
  Filler phrases played when the AI Agent is adjusting response timing.
</ParamField>


<ParamField path="internal_fillers.next_step" type="object">
  Filler phrases played when transitioning between conversation steps when utilizing [`prompt.context`][prompt.contexts].
</ParamField>


<ParamField path="internal_fillers.change_context" type="object">
  Filler phrases played when switching between conversation contexts when utilizing [`prompt.context`][prompt.contexts].
</ParamField>


<ParamField path="internal_fillers.get_visual_input" type="object">
  Filler phrases played when the AI Agent is processing visual input. This function is enabled when the `enable_vision` property is set to `true` in the [`ai.params`](/swml/methods/ai/params) method.
</ParamField>


<ParamField path="internal_fillers.get_ideal_strategy" type="object">
  Filler phrases played when the AI Agent is thinking or considering options. This is utilized when `enable_thinking` is set to `true` in [ai.params].
</ParamField>


## Filler Object Structure

Each [`internal_fillers` parameters][parameters] accepts an [object containing language-specific filler phrases](#language-specific-fillers). All functions use the same structure for defining fillers.

### Language-Specific Fillers

<ParamField path="internal_fillers.{function_name}.{language_code}" type="string[]">
  An array of filler phrases that will be played during the function execution. The key is a valid [language code](#supported-language-codes) and the value is an array of filler phrases that are selected from randomly.
</ParamField>


## Supported Language Codes

Internal filler functions accept an object with the key being one of the supported language codes and the value being an array of filler phrases.

## Examples

### Basic Usage

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
internal_fillers:
  hangup:
    en-US:
      - 'Goodbye!'
      - 'Thank you for calling.'
      - 'Have a great day!'
    es-ES:
      - '¡Adiós!'
      - 'Gracias por llamar.'
      - '¡Que tengas un buen día!'
  check_time:
    default:
      - 'Let me check the time.'
      - 'One moment while I get the time.'
      - 'Just checking the current time.'
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "internal_fillers": {
    "hangup": {
      "en-US": [
        "Goodbye!",
        "Thank you for calling.",
        "Have a great day!"
      ],
      "es-ES": [
        "\u00a1Adi\u00f3s!",
        "Gracias por llamar.",
        "\u00a1Que tengas un buen d\u00eda!"
      ]
    },
    "check_time": {
      "default": [
        "Let me check the time.",
        "One moment while I get the time.",
        "Just checking the current time."
      ]
    }
  }
}
```
</CodeBlock>
</CodeBlocks>
