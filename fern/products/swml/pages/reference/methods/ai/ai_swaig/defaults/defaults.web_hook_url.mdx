The default URL that the user defines to which to send status callbacks and reports. This URL is used for all SWAIG functions unless overridden by a function-specific `web_hook_url`.

<ParamField path="defaults.web_hook_url" type="string">
  The URL to send status callbacks and reports to. Authentication can also be set in the url in the format of `username:password@url`.
</ParamField>
