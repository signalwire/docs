An array of objects that define the steps in the context.
These steps are used to define the flow of the conversation.

<ParamField path="{context_name}.steps" type="object[]" required={true}>
  An array of objects that accept the following properties.
</ParamField>


## **Properties**

The `steps` property accepts one of the following step types:

<Tabs>
<Tab title="Text Step">

<ParamField path="steps[].text" type="string" required={true}>
  The prompt or instructions given to the AI at this step.
</ParamField>


</Tab>
<Tab title="POM Step">

<ParamField path="steps[].pom" type="object[]" required={true}>
  An array of [POM (Prompt Object Model)](/swml/methods/ai/prompt/pom) sections that define structured prompt instructions for the AI at this step.
</ParamField>


</Tab>
</Tabs>
