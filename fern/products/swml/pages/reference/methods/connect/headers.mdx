Custom SIP headers to add to INVITE. Has no effect on calls to phone numbers.

<Schema type="ConnectHeaders" />

## Example

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - connect:
        to: "sip:user-a@<domain>.com"
        headers:
          - name: "x-FROM_NUMBER"
            value: "+2B123456789"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "connect": {
          "to": "sip:user-a@<domain>.com",
          "headers": [
            {
              "name": "x-FROM_NUMBER",
              "value": "+2B123456789"
            }
          ]
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
