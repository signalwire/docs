Custom SIP headers to add to INVITE. Has no effect on calls to phone numbers.

<ParamField path="connect.headers" type="object[]">
  An array of objects that accept the [properties](#properties).
</ParamField>


## **Properties**

<ParamField path="headers[].name" type="string" required={true}>
  The name of the header.
</ParamField>


<ParamField path="headers[].value" type="string" required={true}>
  The value of the header.
</ParamField>


## Example

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - connect:
        to: "sip:user-a@<domain>.com"
        headers:
          - name: "x-FROM_NUMBER"
            value: "+2B123456789"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {

          "to": "sip:user-a@<domain>.com",

              "name": "x-FROM_NUMBER",

            }
          ]
        }
      }
    ]
  }
}
```
