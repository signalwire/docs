Dial a SIP URI or phone number.

## **Properties**

<Schema type="Connect" />


## **Variables**

Set by the method:

- **connect_result:** (out) `connected` | `failed`.
- **connect_failed_reason:** (out) Detailed reason for failure.
- **return_value:** (out) Same value as `connect_result`.

## **Examples**

### Use `connect` with Call Fabric

Use a Call Fabric Resource with the `connect` method by simply including the Resource Address.

Learn more by reading our 
[Introduction to Call Fabric](/platform/call-fabric)
or the guide to
[Managing Resources](/platform/call-fabric/resources).

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - play:
        volume: 10
        urls:
          - 'silence:1.0'
          - 'say:Hello, connecting to a fabric Resource that is a room'
    - connect:
        to: /public/test_room
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "answer": {}
      },
      {
        "play": {
          "volume": 10,
          "urls": [
            "silence:1.0",
            "say:Hello, connecting to a fabric Resource that is a room"
          ]
        }
      },
      {
        "connect": {
          "to": "/public/test_room"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Dial a single phone number

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - connect:
        from: "+15553214321"
        to: "+15551231234"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "connect": {
          "from": "+15553214321",
          "to": "+15551231234"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Dial numbers in parallel

<CodeBlocks>
<CodeBlock title="YAML">
```yaml

sections:
  main:
    - connect:

          - to: "+15551231234"
          - to: "+15553214321"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "connect": {
          "parallel": [
            {
              "to": "+15551231234"
            },
            {
              "to": "+15553214321"
            }
          ]
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Dial SIP serially with a timeout

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - connect:
        timeout: 20
        serial:
          - from: "sip:chris@example.com"
            to: "sip:alice@example.com"
          - to: "sip:bob@example.com"
            codecs: PCMU
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "connect": {
          "timeout": 20,
          "serial": [
            {
              "from": "sip:chris@example.com",
              "to": "sip:alice@example.com"
            },
            {
              "to": "sip:bob@example.com",
              "codecs": "PCMU"
            }
          ]
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

<CodeBlock title="YAML">

  }
