Send SIP REFER to a SIP call.

<ParamField path="sip_refer" type="object" required={true}>
  An object that accepts the [`sip_refer parameters`](#sip_refer-parameters).
</ParamField>


## **sip_refer Parameters**

<ParamField path="sip_refer.to_uri" type="string" required={true}>
  SIP URI to REFER to.
</ParamField>


<ParamField path="sip_refer.status_url" type="string">
  HTTP or HTTPS URL to deliver SIP REFER status events.
</ParamField>


<ParamField path="sip_refer.username" type="string">
  Username to use for SIP authentication.
</ParamField>


<ParamField path="sip_refer.password" type="string">
  Password to use for SIP authentication.
</ParamField>


## **Variables**

Set by the method:

- **sip_refer_to:** (out) The SIP URI the recipient is to INVITE.
- **sip_refer_result:** (out) Overall SIP REFER result.
- **return_value:** (out) Same value as `sip_refer_result`.
- **sip_refer_response_code:** (out) Recipient response to the REFER request.
- **sip_refer_to_response_code:** (out) INVITE response to the recipient.

## **Examples**

### Send SIP REFER and post result

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - sip_refer:
        to_uri: 'sip:alice@example.com'
    - play:
        url: 'say: Connected. The SIP refer result is ${sip_refer_result}'
    - execute:
        dest: 'https://example.com/handle_sip_refer_result'
```
