Play file(s), ringtones, speech or silence.

<ParamField path="play" type="object" required={true}>
  An object that accepts the [`play parameters`](#parameters).
</ParamField>


## **play Parameters**

<Tabs>
  <Tab title="Single URL (url)">
    **play.url** (type: string) *required*
  A single [playable sound](#playable-sounds). Authentication can also be set in the url in the format of `username:password@url`.
    
    </Tab>
  <Tab title="Multiple URLs (urls)">
    **play.urls** (type: string[]) *required*
  An array of [playable sounds](#playable-sounds). Authentication can also be set in the url in the format of `username:password@url`.
    
    </Tab>
</Tabs>

## **Playable sounds**

## **Variables**

Read by the method:

- **`say_voice:`** (in) - Optional voice to use for text to speech.
- **`say_language:`** (in) - Optional language to use for text to speech.
- **`say_gender:`** (in) - Optional gender to use for text to speech.

## **Possible Values for Voice, Language and Ringtone**

## **Examples**

### Playing a single URL

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - play:
        url: 'https://cdn.signalwire.com/swml/audio.mp3'
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "play": {
          "url": "https://cdn.signalwire.com/swml/audio.mp3"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Playing multiple URLs

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - play:
        urls:
          - 'https://cdn.signalwire.com/swml/audio.mp3'
          - 'say: this is something to say'
          - 'silence: 3.0'
          - 'ring:10.0:us'
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "play": {
          "urls": [
            "https://cdn.signalwire.com/swml/audio.mp3",
            "say: this is something to say",
            "silence: 3.0",
            "ring:10.0:us"
          ]
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Specifying a voice to use for speaking

**Globally**

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - set:
        say_voice: gcloud.en-US-Neural2-A
    - play:
        url: 'say:Hi, do I sound different?'
    - play:
        url: 'say:I don''t, do I?'
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "set": {
          "say_voice": "gcloud.en-US-Neural2-A"
        }
      },
      {
        "play": {
          "url": "say:Hi, do I sound different?"
        }
      },
      {
        "play": {
          "url": "say:I don't, do I?"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

**For just one instance**

<CodeBlocks>

sections:

  "sections": {
    "main": [
      {
        "play": {
          "url": "say:Hi, do I sound different?",

</CodeBlock>
