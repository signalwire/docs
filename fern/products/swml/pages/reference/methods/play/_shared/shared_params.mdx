
**play.auto_answer** (type: boolean) Default: `true`
  If `true`, the call will automatically answer as the sound is playing. If `false`, you will start playing the audio during early media, additionally, you can instruct to answer the call with the [`answer method`](/swml/methods/answer).

**play.volume** (type: number) Default: `0`
  Volume gain to apply to played URLs. Allowed values from `-40.0` to `40.0`.

**play.say_voice** (type: string) Default: `Polly.Salli`
  Voice to use with `say:` for text to speech.

**play.say_language** (type: string) Default: `en-US`
  Language to use with `say:` for text to speech.

**play.say_gender** (type: string) Default: `female`
  Gender to use with `say:` for text to speech.

**play.status_url** (type: string)
  HTTP or HTTPS URL to deliver play status events.
