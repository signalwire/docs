The SignalWire AI Gateway Interface. Allows you to create user-defined functions that can be executed during the dialogue.

<ParamField path="amazon_bedrock.SWAIG" type="object">
  An object that contains the [`SWAIG parameters`](#swaig-parameters).
</ParamField>


## **SWAIG Parameters**

<ParamField path="SWAIG.defaults" type="object">
  Default settings for all SWAIG functions. If `defaults` is not set, settings may be set in each function object. Default is not set.
  
  See [`defaults`](./defaults/index.mdx) for additional details.
</ParamField>


<ParamField path="SWAIG.functions" type="object[]">
  An array of JSON objects to define functions that can be executed during the interaction with the AI. Default is not set. The fields of this object are the six following.
  
  See [`functions`](./functions/index.mdx) for additional details.
</ParamField>


<ParamField path="SWAIG.includes" type="object[]">
  An array of objects to include remote function signatures. This allows you to include functions that are defined in a remote location.
  
  See [`includes`](./includes/index.mdx) for additional details.
</ParamField>


<ParamField path="SWAIG.native_functions" type="string[]">
  Prebuilt functions the AI agent is able to call (from [this list of available native functions](./native_functions/index.mdx#available-functions)).
  
  See [`native_functions`](./native_functions/index.mdx) for additional details.
</ParamField>
