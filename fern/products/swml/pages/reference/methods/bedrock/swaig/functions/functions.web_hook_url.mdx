The function-specific URL that the user defines to which to send status callbacks and reports. This URL overrides the `defaults.web_hook_url` for this specific function.

<ParamField path="functions[].web_hook_url" type="string">
  The URL to send status callbacks and reports to. Authentication can also be set in the url in the format of `username:password@url`.
</ParamField>
