**amazon_bedrock.prompt** (type: object) *required*
  An object that accepts the [`prompt parameters`][prompt-parameters].

## **prompt Parameters**

The `prompt` property accepts one of the following objects:

<Tabs>
    <Tab title="Regular Prompt">

**prompt.text** (type: string) *required*
  The instructions to send to the agent.

**prompt.temperature** (type: number) Default: `1.0`
  Randomness setting. Float value between 0.0 and 1.5. Closer to 0 will make the output less random.

**prompt.top_p** (type: number) Default: `1.0`
  Randomness setting. Alternative to temperature. Float value between 0.0 and 1.0.

**prompt.confidence** (type: number) Default: `0.6`
  Threshold to fire a speech-detect event at the end of the utterance.

**voice_id** (type: string) Default: `matthew`
  The voice the Amazon Bedrock agent will use during the interaction.
  Possible Values: `tiffany`, `matthew`, `amy`, `lupe`, `carlos`
        
    </Tab>
    <Tab title="POM Prompts">

**prompt.pom** (type: object) *required*
  The POM (Prompt Object Model) configuration for the agent.

**prompt.temperature** (type: number) Default: `1.0`
  Randomness setting. Float value between 0.0 and 1.5. Closer to 0 will make the output less random.

**prompt.top_p** (type: number) Default: `1.0`
  Randomness setting. Alternative to temperature. Float value between 0.0 and 1.0.

**prompt.confidence** (type: number) Default: `0.6`
  Threshold to fire a speech-detect event at the end of the utterance.

**voice_id** (type: string) Default: `matthew`
  The voice the Amazon Bedrock agent will use during the interaction.
  Possible Values: `tiffany`, `matthew`, `amy`, `lupe`, `carlos`
        
    </Tab>
</Tabs>

## **Variable Expansion**

Use the following syntax to expand variables into your prompt.

**${call_direction}** (type: string)
  Inbound or outbound.

**${caller_id_number}** (type: string)
  The caller ID number.

**${local_date}** (type: string)
  The local date.

**${spoken_date}** (type: string)
  The spoken date.

**${local_time}** (type: string)
  The local time.

**${time_of_day}** (type: string)
  The time of day.

**${supported_languages}** (type: string)
  A list of supported languages.

**${default_language}** (type: string)
  The default language.
