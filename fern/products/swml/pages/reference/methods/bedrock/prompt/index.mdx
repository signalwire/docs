## **Properties**

The `prompt` property accepts one of the following objects:

<Tabs>
    <Tab title="Regular Prompt">

<ParamField toc={true} path="prompt.text" type="string" required={true}>
  The instructions to send to the agent.
</ParamField>


<ParamField toc={true} path="prompt.temperature" type="number" default="1.0">
  Randomness setting. Float value between 0.0 and 1.5. Closer to 0 will make the output less random.
</ParamField>


<ParamField toc={true} path="prompt.top_p" type="number" default="1.0">
  Randomness setting. Alternative to temperature. Float value between 0.0 and 1.0.
</ParamField>


<ParamField toc={true} path="prompt.confidence" type="number" default="0.6">
  Threshold to fire a speech-detect event at the end of the utterance.
</ParamField>


<ParamField toc={true} path="voice_id" type="string" default="matthew">
  The voice the Amazon Bedrock agent will use during the interaction.
  Possible Values: `tiffany`, `matthew`, `amy`, `lupe`, `carlos`
</ParamField>

</Tab>
<Tab title="POM Prompts">

<ParamField toc={true} path="prompt.pom" type="object" required={true}>
  The POM (Prompt Object Model) configuration for the agent.
</ParamField>


<ParamField toc={true} path="prompt.temperature" type="number" default="1.0">
  Randomness setting. Float value between 0.0 and 1.5. Closer to 0 will make the output less random.
</ParamField>


<ParamField toc={true} path="prompt.top_p" type="number" default="1.0">
  Randomness setting. Alternative to temperature. Float value between 0.0 and 1.0.
</ParamField>


<ParamField toc={true} path="prompt.confidence" type="number" default="0.6">
  Threshold to fire a speech-detect event at the end of the utterance.
</ParamField>


<ParamField toc={true} path="voice_id" type="string" default="matthew">
  The voice the Amazon Bedrock agent will use during the interaction.
  Possible Values: `tiffany`, `matthew`, `amy`, `lupe`, `carlos`
</ParamField>

</Tab>
</Tabs>
