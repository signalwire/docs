End the call with an optional reason.

**hangup** (type: object) *required*
  An object that contains the [`hangup parameters`](#hangup-parameters).

## **hangup Parameters**

**hangup.reason**
  The reason for hanging up the call.

## **Examples**

### No parameters

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup: {}

```

### Set a hangup reason

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup:
        reason: "busy"
```
