End the call with an optional reason.

<ParamField path="hangup" type="object" required={true}>
  An object that contains the [`hangup parameters`](#hangup-parameters).
</ParamField>


## **hangup Parameters**

**hangup.reason**
  The reason for hanging up the call.

## **Examples**

### No parameters

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "answer": {}
      },
      {
        "hangup": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Set a hangup reason

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup:
        reason: "busy"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "answer": {}
      },
      {
        "hangup": {
          "reason": "busy"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
