End the call with an optional reason.

## **Properties**

<Schema type="Hangup" />

## **Examples**

### No parameters

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "answer": {}
      },
      {
        "hangup": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Set a hangup reason

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup:
        reason: "busy"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "answer": {}
      },
      {
        "hangup": {
          "reason": "busy"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
