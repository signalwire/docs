End the call with an optional reason.

<APIField
  name="hangup"
  type="object"
  required={true}
>
  An object that contains the [`hangup parameters`](#hangup-parameters).

## **hangup Parameters**

<APIField
  name="hangup.reason"
  type='"hangup" | "busy" | "decline"'
>
  The reason for hanging up the call.

## **Examples**

### No parameters

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup: {}

```

### Set a hangup reason

```yaml andJson
version: 1.0.0
sections:
  main:
    - answer: {}
    - hangup:
        reason: "busy"
```
