Stop an active tap stream.

## **Properties**

<Schema type="StopTap" />

## **Variables**

Read by the method:

- **tap_control_id:** (in) Control ID of last tap stream started.

Set by the method:

- **stop_tap_result:** (out) Success or failed.

## **Examples**

### Stop the last call tap

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - stop_tap: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "stop_tap": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Stop a specific call tap

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - stop_tap:
        control_id: my-tap-id
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "stop_tap": {
          "control_id": "my-tap-id"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
