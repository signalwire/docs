Answer incoming call and set an optional maximum duration.

## **Properties**

<ParamField path="answer" type="object" toc={true}>
  An object that accepts the following properties.
</ParamField>

<Indent>

<ParamField path="answer.max_duration" type="integer" default="14400 seconds (4 hours)" toc={true}>
  Maximum duration in seconds for the call.
</ParamField>


<ParamField path="answer.codecs" type="string" toc={true}>
  Comma-separated string of codecs to offer. Valid codecs are: `PCMU`, `PCMA`, `G722`, `G729`, `AMR-WB`, `OPUS`, `VP8`, `H264`.
</ParamField>


<ParamField path="answer.username" type="string" toc={true}>
  Username to use for SIP authentication.
</ParamField>


<ParamField path="answer.password" type="string" toc={true}>
  Password to use for SIP authentication.
</ParamField>

</Indent>
