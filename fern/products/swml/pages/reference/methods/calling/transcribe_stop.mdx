Stop an active background transcription that was started with [`transcribe`](/docs/swml/reference/calling/transcribe). The transcription tracked for the current call is stopped. This method accepts no parameters.

## **Properties**

<ParamField path="transcribe_stop" type="object" toc={true}>
  An empty object. `transcribe_stop` may also be invoked as a bare string.
</ParamField>

## **Variables**

Read by the method:

- **transcribe_control_id:** (in) Control ID of the active transcription, set by `transcribe`.

Set by the method:

- **transcribe_stop_result:** (out) `success` | `failed`.

## **Examples**

### Stop the active transcription

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - transcribe_stop: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "transcribe_stop": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
