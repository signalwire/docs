Stop the transcription currently running on the call, started with [`transcribe`](/docs/swml/reference/calling/transcribe). This method accepts no parameters.

## **Properties**

<ParamField path="transcribe_stop" type="object" toc={true}>
  An empty object.
</ParamField>

## **Variables**

<ParamField path="transcribe_control_id" type="string" toc={true}>
  Read by the method. Identifier of the active transcription, set by `transcribe`.
</ParamField>


<ParamField path="transcribe_stop_result" type="success | failed" toc={true}>
  Set by the method. Whether the transcription stopped successfully.
</ParamField>
