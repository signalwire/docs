Stop an audio stream started with [`stream`](/docs/swml/reference/calling/stream). By default it stops the most recently started stream; pass a `control_id` to stop a specific one.

## **Properties**

<ParamField path="stop_stream" type="object" toc={true}>
  An object that accepts the following properties.
</ParamField>

<Indent>

<ParamField path="stop_stream.control_id" type="string" default="The most recently started stream is stopped" toc={true}>
  The control ID of the stream to stop, as assigned when you started it with `stream`.
</ParamField>

</Indent>

## **Variables**

<ParamField path="stream_control_id" type="string" toc={true}>
  Read by the method. Control ID of the most recently started stream.
</ParamField>


<ParamField path="stop_stream_result" type="success | failed" toc={true}>
  Set by the method. Whether the stream stopped successfully.
</ParamField>

## **Examples**

### Stop the last stream

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - stop_stream: {}
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "stop_stream": {}
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Stop a specific stream

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - stop_stream:
        control_id: my-stream-id
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "stop_stream": {
          "control_id": "my-stream-id"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
