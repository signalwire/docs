Send digit presses as DTMF tones.

## **Properties**

<Schema type="SendDigits" />

## **Variables**

Set by the method:

- **send_digits_result:** (out) `success` | `failed`

## **Examples**

### Send digits

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - send_digits:
        digits: '012345'
    - play:
        url: 'say: ${send_digits_result}'
```
</CodeBlock>
