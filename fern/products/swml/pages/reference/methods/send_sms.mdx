Send an outbound message to a PSTN phone number.

## **Properties**

<Schema type="SendSMS" />

## **Variables**

Set by the method:

- **send_sms_result:** (out) `success` | `failed`.

## **Examples**

<Tabs>
<Tab title="SMS">

Send a text-only message:

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        body: "Hi, I hope you're well."
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "send_sms": {
          "from_number": "+155512312345",
          "to_number": "+15555554321",
          "body": "Hi, I hope you're well."
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

</Tab>
<Tab title="MMS">

Send a message with media attachment:

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        media: ["https://example.com/image.jpg"]
        body: "Check out this image!"
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "send_sms": {
          "from_number": "+155512312345",
          "to_number": "+15555554321",
          "media": [
            "https://example.com/image.jpg"
          ],
          "body": "Check out this image!"
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
