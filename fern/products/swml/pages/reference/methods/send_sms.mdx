Send an outbound message to a PSTN phone number.

<ParamField toc={true} path="send_sms" type="object" required={true}>
  An object that accepts the following properties.
</ParamField>


## **Properties**

<Tabs>
<Tab title="SMS">

<ParamField toc={true} path="send_sms.to_number" type="string" required={true}>
  Phone number to send SMS message to in e.164 format
</ParamField>


<ParamField toc={true} path="send_sms.from_number" type="string" required={true}>
  Phone number SMS message will be sent from
</ParamField>


<ParamField toc={true} path="send_sms.body" type="string" required={true}>
  Body of the text message
</ParamField>


<ParamField toc={true} path="send_sms.region" type="string" default="Chosen based on account preferences or device location">
  Region of the world to originate the message from
</ParamField>


<ParamField toc={true} path="send_sms.tags" type="string[]">
  Array of tags to associate with the message to facilitate log searches
</ParamField>


</Tab>
<Tab title="MMS">

<ParamField toc={true} path="send_sms.to_number" type="string" required={true}>
  Phone number to send SMS message to in e.164 format
</ParamField>


<ParamField toc={true} path="send_sms.from_number" type="string" required={true}>
  Phone number SMS message will be sent from
</ParamField>


<ParamField toc={true} path="send_sms.media" type="string[]" required={true}>
  Array of media URLs to include in the message
</ParamField>


<ParamField toc={true} path="send_sms.body" type="string">
  Optional text to accompany the media
</ParamField>


<ParamField toc={true} path="send_sms.region" type="string" default="Chosen based on account preferences or device location">
  Region of the world to originate the message from
</ParamField>


<ParamField toc={true} path="send_sms.tags" type="string[]">
  Array of tags to associate with the message to facilitate log searches
</ParamField>


</Tab>
</Tabs>

## **Variables**

Set by the method:

- **send_sms_result:** (out) `success` | `failed`.

## **Examples**

<Tabs>
<Tab title="SMS">

Send a text-only message:

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        body: "Hi, I hope you're well."
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "send_sms": {
          "from_number": "+155512312345",
          "to_number": "+15555554321",
          "body": "Hi, I hope you're well."
        }
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

</Tab>

</CodeBlock>
