Send an outbound message to a PSTN phone number.

**send_sms** (type: object) *required*
  An object that accepts the [`send_sms parameters`](#send_sms-parameters).

## **send_sms Parameters**

<Tabs>
<Tab title="SMS">

**send_sms.to_number** (type: string) *required*
  Phone number to send SMS message to in e.164 format

**send_sms.from_number** (type: string) *required*
  Phone number SMS message will be sent from

**send_sms.body** (type: string) *required*
  Body of the text message

**send_sms.region** (type: string) Default: `Chosen based on account preferences or device location`
  Region of the world to originate the message from

**send_sms.tags** (type: string[])
  Array of tags to associate with the message to facilitate log searches

</Tab>
<Tab title="MMS">

**send_sms.to_number** (type: string) *required*
  Phone number to send SMS message to in e.164 format

**send_sms.from_number** (type: string) *required*
  Phone number SMS message will be sent from

**send_sms.media** (type: string[]) *required*
  Array of media URLs to include in the message

**send_sms.body** (type: string)
  Optional text to accompany the media

**send_sms.region** (type: string) Default: `Chosen based on account preferences or device location`
  Region of the world to originate the message from

**send_sms.tags** (type: string[])
  Array of tags to associate with the message to facilitate log searches

</Tab>
</Tabs>

## **Variables**

Set by the method:

- **send_sms_result:** (out) `success` | `failed`.

## **Examples**

<Tabs>
<Tab title="SMS">

Send a text-only message:

```yaml andJson
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        body: "Hi, I hope you're well."
```

</Tab>
<Tab title="MMS">

Send a message with media attachment:

```yaml andJson
version: 1.0.0
sections:
  main:
    - send_sms:
        from_number: "+155512312345"
        to_number: "+15555554321"
        media: ["https://example.com/image.jpg"]
        body: "Check out this image!"
```

</Tab>
</Tabs>
