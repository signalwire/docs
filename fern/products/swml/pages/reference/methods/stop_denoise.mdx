Stop noise reduction (which was started with [`denoise`](./denoise.mdx)).

**stop_denoise** (type: object) *required*
  An empty object that accepts no parameters.

## **Variables**

Set by the method:

- **denoise_result:** (out) `off`

## **Examples**

### Stop denoise

```yaml andJson
version: 1.0.0
sections:
  main:
    - stop_denoise: {}
    - play:
        url: 'say: Denoising ${denoise_result}'
```
