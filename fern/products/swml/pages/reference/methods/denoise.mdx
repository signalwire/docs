Start noise reduction. You can stop it at any time using [`stop_denoise`](./stop_denoise.mdx).

<ParamField path="denoise" type="object" required={true}>
  An empty object that accepts no parameters.
</ParamField>


## **Variables**

Set by the method:

- **denoise_result:** (out) `on` | `failed`

## **Examples**

### Start denoise

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - answer: {}
    - denoise: {}
    - play:
        url: 'say: Denoising ${denoise_result}'
```
</CodeBlock>
