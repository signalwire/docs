Set the amount of time for the current application to sleep for in milliseconds before
continuing to the next action.

## **Properties**

<Schema type="Sleep" />

## **Examples**

### Timed Sleep Example

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - sleep: 5000
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "sleep": 5000
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>

### Forever Sleep Example

<CodeBlocks>
<CodeBlock title="YAML">
```yaml
version: 1.0.0
sections:
  main:
    - sleep: -1
```
</CodeBlock>
<CodeBlock title="JSON">
```json
{
  "version": "1.0.0",
  "sections": {
    "main": [
      {
        "sleep": -1
      }
    ]
  }
}
```
</CodeBlock>
</CodeBlocks>
