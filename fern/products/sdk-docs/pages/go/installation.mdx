# Go SDK Installation

Install the SignalWire Go SDK using go get:

```bash
go get github.com/signalwire/signalwire-go
```

## Requirements

- Go 1.18 or higher
- Go modules enabled

## Verify Installation

```go
package main

import (
    "fmt"
    signalwire "github.com/signalwire/signalwire-go"
)

func main() {
    fmt.Println("SignalWire Go SDK installed successfully")
}
```
