# Go SDK Quick Start

Get up and running with the SignalWire Go SDK in minutes.

## Initialize the Client

```go
package main

import (
    "context"
    signalwire "github.com/signalwire/signalwire-go"
)

func main() {
    client, err := signalwire.NewClient(
        "your-project-id",
        "your-api-token",
        signalwire.WithSpaceURL("your-space.signalwire.com"),
    )
    if err != nil {
        panic(err)
    }
}
```

## Make Your First Call

```go
call, err := client.Voice.Create(context.Background(), &signalwire.CallParams{
    To:   "+15551234567",
    From: "+15559876543",
    URL:  "https://example.com/voice.xml",
})
if err != nil {
    panic(err)
}

fmt.Printf("Call SID: %s\n", call.SID)
```
