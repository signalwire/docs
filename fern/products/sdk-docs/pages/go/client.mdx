# Go Client Reference

The `Client` struct is the main entry point for interacting with SignalWire APIs.

## Constructor

```go
import signalwire "github.com/signalwire/signalwire-go"

client, err := signalwire.NewClient(
    projectID string,
    apiToken string,
    opts ...ClientOption,
)
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `projectID` | `string` | Your SignalWire project ID |
| `apiToken` | `string` | Your SignalWire API token |
| `opts` | `...ClientOption` | Optional configuration options |

## Client Options

- `WithSpaceURL(url string)` - Set your SignalWire space URL
- `WithHTTPClient(client *http.Client)` - Use a custom HTTP client

## Properties

- `Voice` - Access the Voice API
- `Messaging` - Access the Messaging API
- `Fax` - Access the Fax API
