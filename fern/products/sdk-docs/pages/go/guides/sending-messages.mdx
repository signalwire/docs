# Sending Messages with Go

This guide shows you how to send SMS messages using the SignalWire Go SDK.

## Prerequisites

- SignalWire account with a phone number
- Go SDK installed

## Send an SMS

```go
package main

import (
    "context"
    "fmt"
    signalwire "github.com/signalwire/signalwire-go"
)

func main() {
    client, err := signalwire.NewClient(
        "your-project-id",
        "your-api-token",
        signalwire.WithSpaceURL("your-space.signalwire.com"),
    )
    if err != nil {
        panic(err)
    }

    message, err := client.Messaging.Create(context.Background(), &signalwire.MessageParams{
        To:   "+15551234567",
        From: "+15559876543",
        Body: "Hello from SignalWire!",
    })
    if err != nil {
        panic(err)
    }

    fmt.Printf("Message SID: %s\n", message.SID)
}
```
