# Making Calls with Go

This guide shows you how to make outbound voice calls using the SignalWire Go SDK.

## Prerequisites

- SignalWire account with a phone number
- Go SDK installed

## Basic Outbound Call

```go
package main

import (
    "context"
    "fmt"
    signalwire "github.com/signalwire/signalwire-go"
)

func main() {
    client, err := signalwire.NewClient(
        "your-project-id",
        "your-api-token",
        signalwire.WithSpaceURL("your-space.signalwire.com"),
    )
    if err != nil {
        panic(err)
    }

    call, err := client.Voice.Create(context.Background(), &signalwire.CallParams{
        To:   "+15551234567",
        From: "+15559876543",
        URL:  "https://example.com/voice.xml",
    })
    if err != nil {
        panic(err)
    }

    fmt.Printf("Call initiated: %s\n", call.SID)
}
```
