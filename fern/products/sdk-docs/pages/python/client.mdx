# Python Client Reference

The `Client` class is the main entry point for interacting with SignalWire APIs.

## Constructor

```python
from signalwire.rest import Client

client = Client(
    project_id: str,
    api_token: str,
    space_url: str
)
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `project_id` | `str` | Your SignalWire project ID |
| `api_token` | `str` | Your SignalWire API token |
| `space_url` | `str` | Your SignalWire space URL |

## Properties

- `calls` - Access the Calls API
- `messages` - Access the Messages API
- `fax` - Access the Fax API
