# Making Calls with Python

This guide shows you how to make outbound voice calls using the SignalWire Python SDK.

## Prerequisites

- SignalWire account with a phone number
- Python SDK installed

## Basic Outbound Call

```python
from signalwire.rest import Client

client = Client(
    project_id="your-project-id",
    api_token="your-api-token",
    space_url="your-space.signalwire.com"
)

call = client.calls.create(
    to="+15551234567",
    from_="+15559876543",
    url="https://example.com/voice.xml"
)

print(f"Call initiated: {call.sid}")
```

## Handling Call Events

```python
# Check call status
call = client.calls.get(call_sid)
print(f"Status: {call.status}")
```
