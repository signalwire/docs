# Sending Messages with Python

This guide shows you how to send SMS messages using the SignalWire Python SDK.

## Prerequisites

- SignalWire account with a phone number
- Python SDK installed

## Send an SMS

```python
from signalwire.rest import Client

client = Client(
    project_id="your-project-id",
    api_token="your-api-token",
    space_url="your-space.signalwire.com"
)

message = client.messages.create(
    to="+15551234567",
    from_="+15559876543",
    body="Hello from SignalWire!"
)

print(f"Message SID: {message.sid}")
```

## Check Message Status

```python
message = client.messages.get(message_sid)
print(f"Status: {message.status}")
```
