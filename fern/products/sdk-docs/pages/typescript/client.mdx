# TypeScript Client Reference

The `SignalWire` function creates the main client for interacting with SignalWire APIs.

## Constructor

```typescript
import { SignalWire } from '@signalwire/realtime-api';

const client = await SignalWire({
  project: string,
  token: string,
});
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `project` | `string` | Your SignalWire project ID |
| `token` | `string` | Your SignalWire API token |

## Properties

- `voice` - Access the Voice API
- `messaging` - Access the Messaging API
- `video` - Access the Video API
- `chat` - Access the Chat API
