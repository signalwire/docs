# Making Calls with TypeScript

This guide shows you how to make outbound voice calls using the SignalWire TypeScript SDK.

## Prerequisites

- SignalWire account with a phone number
- TypeScript SDK installed

## Basic Outbound Call

```typescript
import { SignalWire } from '@signalwire/realtime-api';

const client = await SignalWire({
  project: 'your-project-id',
  token: 'your-api-token',
});

const voiceClient = client.voice;

const call = await voiceClient.dialPhone({
  to: '+15551234567',
  from: '+15559876543',
  timeout: 30,
});

console.log(`Call initiated: ${call.id}`);
```

## Handling Call Events

```typescript
call.on('answered', () => {
  console.log('Call was answered');
});

call.on('ended', () => {
  console.log('Call ended');
});
```
