# Sending Messages with TypeScript

This guide shows you how to send SMS messages using the SignalWire TypeScript SDK.

## Prerequisites

- SignalWire account with a phone number
- TypeScript SDK installed

## Send an SMS

```typescript
import { SignalWire } from '@signalwire/realtime-api';

const client = await SignalWire({
  project: 'your-project-id',
  token: 'your-api-token',
});

const messagingClient = client.messaging;

const message = await messagingClient.send({
  to: '+15551234567',
  from: '+15559876543',
  body: 'Hello from SignalWire!',
});

console.log(`Message ID: ${message.id}`);
```

## Handle Incoming Messages

```typescript
messagingClient.on('message.received', (message) => {
  console.log(`Received: ${message.body}`);
});
```
