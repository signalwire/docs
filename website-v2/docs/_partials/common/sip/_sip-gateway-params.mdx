{/* Shared component: SIP Gateway parameters */}

Configure the following settings for your SIP Gateway:

| Parameter | Description |
|-----------|-------------|
| **Name** | A friendly name for this SIP Gateway |
| **URI** | The full SIP URI to forward calls to (e.g., `sip:user@domain.com`) |
| **Encryption** | Whether to require, prefer, or disable encryption |
| **Codecs** | Audio codecs to use for the connection |
| **Ciphers** | Encryption ciphers to use when encryption is enabled |
