Use this endpoint for the Recordings method to initiate a Recording for a [Call](/docs/compatibility-api/sdks/reference/calls).
You can use this endpoint to start recording a Call that is already in progress.
If you want to record the entire Call, you can specify recording params during [creation of the Call](/docs/compatibility-api/sdks/reference/calls/create).

<EndpointSchemaSnippet endpoint="POST /Accounts/{AccountSid}/Calls/{CallSid}/Recordings" />
