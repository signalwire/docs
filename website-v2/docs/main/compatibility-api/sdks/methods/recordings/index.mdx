A Recording represents a recording of a voice or conference call. 
It allows you to see meta information about the recording as well as access the media file itself.

Recordings can be initiated using the [`<Record>`](/docs/compatibility-api/cxml/reference/voice/record) verb, 
by setting the `record` attribute for [Calls](/docs/compatibility-api/sdks/reference/calls/create),
or by using [Conferences](/docs/compatibility-api/cxml/reference/voice/conference).
