## Events

### onTaskReceived

- **client.task.listen**(`{ onTaskReceived: Callback }`)

Emitted whenever a task is received. Your event handler receives the payload.

#### Parameters

<ParamField path="payload" type="Record<string, unknown>" required>
  The message payload.
