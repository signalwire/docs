### once

- **once**(`event`, `fn`)

Attaches an event handler to the specified event. The handler will fire only once.

#### Parameters

| Name    | Type     | Description                                                     |
| :------ | :------- | :-------------------------------------------------------------- |
| `event` | `string` | Name of the event. See the list of [events](/docs/browser-sdk/js/reference/pubsub/client/events).        |
| `fn`    | Function | An event handler.                                               |
