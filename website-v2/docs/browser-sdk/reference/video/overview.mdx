The Video namespace contains the classes and functions that you need to create a video conferencing application.

The [RoomSession](/docs/browser-sdk/js/reference/video/room-session) class is the main interface for managing video room sessions.

### ~createRoomObject~

- `Const` **createRoomObject**(`roomOptions`): `Promise<Room>`

<Warning title="Deprecated">
Use [RoomSession](/docs/browser-sdk/js/reference/video/room-session) instead.
</Warning>

Using Video.createRoomObject() you can create a `RoomObject` to join a room.

#### Parameters

<CreateRoomObjectOptionsPartial />

#### Returns

`Promise<Room>`

#### Example

With an HTMLDivElement with id="root" in the DOM.

```js
// <div id="root"></div>

try {
  const roomObj = await Video.createRoomObject({
    token: "<YourJWT>",
    rootElementId: "root"
  });

  roomObj.join();
} catch (error) {
  console.error("Error", error);
}
```

### ~joinRoom~

- `Const` **joinRoom**(`roomOptions`): `Promise<Room>`

<Warning title="Deprecated">
Use [RoomSession](/docs/browser-sdk/js/reference/video/room-session) instead.
</Warning>

Using Video.joinRoom() you can automatically join a room.

#### Parameters

<CreateRoomObjectOptionsPartial />

#### Returns

`Promise<Room>`

#### Example

With an HTMLDivElement with id="root" in the DOM.

```js
// <div id="root"></div>

try {
  const roomObj = await Video.joinRoom({
    token: "<YourJWT>",
    rootElementId: "root",
  });

  // You have joined the room..
} catch (error) {
  console.error("Error", error);
}
```
