### off

- **off**(`event`, `fn?`)

Remove an event handler.

#### Parameters

| Name    | Type     | Description                                                     |
| :------ | :------- | :-------------------------------------------------------------- |
| `event` | `string` | Name of the event. See the list of [events](/docs/browser-sdk/js/reference/video/room-session/events).        |
| `fn?`   | Function | An event handler which had been previously attached.            |
