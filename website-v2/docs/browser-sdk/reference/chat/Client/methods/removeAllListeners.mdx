- **removeAllListeners**(`event?`)

Detaches all event listeners for the specified event.

#### Parameters

| Name     | Type     | Description                                                                                                          |
| :------- | :------- | :------------------------------------------------------------------------------------------------------------------- |
| `event?` | `string` | Name of the event (leave this undefined to detach listeners for all events). See the list of [events](/docs/browser-sdk/js/reference/chat/client/events).   |
