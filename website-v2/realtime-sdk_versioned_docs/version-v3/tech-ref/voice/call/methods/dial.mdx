### dial

- **dial**(`params`): `Promise<Call>` - See [Call][call-19] for more details.

Create a new outbound call.

This is a generic method that allows you to dial multiple devices in series, parallel, or combinations of both with the use of a [DeviceBuilder][devicebuilder-1]. For simpler use cases, prefer using [dialPhone][voice-client-1] or [dialSip][voice-client-2].

#### Parameters

| Name     | Type                                                                                                   | Description                                                                                                                               |
|:---------|:-------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| `params` | [`VoiceDeviceBuilder`][voicedevicebuilder-1] \| `{ devices: VoiceDeviceBuilder ; region: string }` | Pass only the Dialer specifying the devices to call or an object with the Dialer and a Region of the world to originate the message from. |

#### Returns

`Promise<Call>` - See [Call][call-19] for more details.
