### off

- **off**(`event`, `fn?`)

Remove an event handler.

#### Parameters

| Name    | Type     | Description                                                                                    |
| :------ | :------- | :--------------------------------------------------------------------------------------------- |
| `event` | `string` | Name of the event. See [Events](/docs/server-sdk/v3/node/reference/voice/call/events) for the list of available events. |
| `fn?`   | Function | An event handler which had been previously attached.                                           |
