## Relay.Event

This object represents the last Relay event that completed an operation on the [`Call`][relay-calling-call].

## Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `name` | `string` | The event name. |
| `payload` | `dict` | Raw JSON object of the Relay event. |
