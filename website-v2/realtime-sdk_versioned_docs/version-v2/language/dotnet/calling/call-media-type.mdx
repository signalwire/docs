This is an enumeration that represents the possible types of media.

| `tts` | The media parameters are [`SignalWire.Relay.Calling.CallMedia.TTSParams`][call-media-t-t-s-params]. |

| `ringtone` | The media parameters are [`SignalWire.Relay.Calling.CallMedia.RingtoneParams`][call-media-ringtone-params]. |
