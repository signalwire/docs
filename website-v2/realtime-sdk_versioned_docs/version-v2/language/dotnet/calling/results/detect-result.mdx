# SignalWire.Relay.Calling.DetectResult

This object returned from [`Detect`][call] method or part of the action when [`DetectAsync`][call-1] completes, which represents the final result of a detection operation.

## Properties

| Property | Type | Description |
| :-------- | :---- | :----------- |
| `Successful` | bool | Whether the detect operation has completed successfully. |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
| `Result` | string | Detail related to the event. |
| `Type` | [`SignalWire.Relay.Calling.DetectResultType`][signalwire-relay-calling-detectresulttype] | The type of the detection. |
