# SignalWire.Relay.Calling.SendDigitsResult

This object returned from [`SendDigits`][call] method, which represents the result of a send digits operation.

## Properties

| Property | Type | Description |
| :-------- | :---- | :----------- |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
| `Successful` | bool | Whether the send operation has successfully sent the digits. |
