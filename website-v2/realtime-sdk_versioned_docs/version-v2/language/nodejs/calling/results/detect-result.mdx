This object returned from one of the _synchronous_ detect methods that represents the final result of a detector.

## Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `successful` | `boolean` | Whether the detector has finished successfully |
| `event` | [`Relay.Event`][relay-event] | Last event that completed the operation |
| `type` | `string` | The detector type: `fax`, `machine` or `digit` |
| `result` | `string` | The final detector result |
