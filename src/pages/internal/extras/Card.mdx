# Card Components

import { Card, CardGroup } from "@site/src/components/Extras/Card";
import { MdCottage, MdLink, MdPalette, MdCode } from "react-icons/md";

The `Card` component is a versatile UI element that can display content in a structured format with optional title, icon, and image.

<Card 
    title="Getting Started" 
    image="https://picsum.photos/800/400"
    href="https://www.example.com"
    horizontal
>
    Begin your journey with our platform. Explore the features and capabilities available to you.
</Card>

### Card Props

| Prop | Type | Description |
|------|------|-------------|
| title | string | The card's title displayed at the top |
| icon | ReactNode \| string | Icon displayed next to title. Can be a React component or image URL |
| image | string | URL of the image to display at the top of the card |
| href | string | Makes the card clickable and navigates to this URL |
| horizontal | boolean | Displays the card in a horizontal layout |
| titleIcon | ReactNode | Additional icon displayed next to the title |
| as | ElementType | HTML element type to render the card as |
| className | string | Additional CSS classes |

### CardGroup Props

| Prop | Type | Description |
|------|------|-------------|
| cols | 1 \| 2 \| 3 \| 4 | Number of columns to display (default: 2) |
| className | string | Additional CSS classes |
