# Release Card for blog

Use this component to get a nice looking release card for your release note.

## Example

import ReleaseCard from "@site/src/components/Blog/ReleaseCard";

<ReleaseCard library="@signalwire/realtime" version="1.1.0" releaseType="minor" />

<ReleaseCard library="@signalwire/js" version="1.0.0" releaseType="major" />

<ReleaseCard library="@signalwire/fabric" version="1.0.1" releaseType="patch" />


## Usage

```mdx
import ReleaseCard from "@site/src/components/Blog/ReleaseCard";

<ReleaseCard library="@signalwire/js" version="1.0.0" releaseType="major" />
```
