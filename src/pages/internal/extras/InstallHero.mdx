# InstallHero Component

import InstallHero from "@site/src/components/InstallHero";

The `InstallHero` component provides a beautiful and consistent way to display installation instructions for SDKs and packages. It includes features like copy-to-clipboard functionality, version display, status badges, and prerequisites.

## Basic Usage

<InstallHero
  title="SignalWire Python SDK"
  description="Build real-time communications applications with SignalWire's Python SDK"
  installCommand="pip install signalwire"
  language="bash"
  version="2.0.0"
  githubUrl="https://github.com/signalwire/signalwire-python"
  docsUrl="/docs/sdks/python"
  gettingStartedUrl="/docs/sdks/python/getting-started"
  status="stable"
  prerequisites={[
    "Python 3.7 or higher",
    "A SignalWire account and API credentials",
    "pip package manager"
  ]}
/>

### InstallHero Props

| Prop | Type | Description |
|------|------|-------------|
| title | string | The SDK/package name |
| description | string | Optional description of the SDK/package |
| installCommand | string | The command to install the package |
| language | string | The language of the install command (default: "bash") |
| version | string | Optional version number to display |
| githubUrl | string | Optional link to GitHub repository |
| docsUrl | string | Optional link to documentation |
| gettingStartedUrl | string | Optional link to getting started guide |
| status | "stable" \| "beta" \| "alpha" | Optional status badge (default: "stable") |
| prerequisites | string[] | Optional list of prerequisites |
| className | string | Additional CSS classes |

## Example Usage

```jsx
import InstallHero from '@site/src/components/InstallHero';

<InstallHero
  title="SignalWire Agents SDK"
  installCommand="pip install signalwire-agents"
  githubUrl="https://github.com/signalwire/signalwire-agents-sdk"
  docsUrl="/docs/swml/agents-sdk"
  gettingStartedUrl="/docs/swml/agents-sdk/get-started"
  status="beta"
  pypiPackage="signalwire-agents"
>
  A powerful SDK for building AI agents with SignalWire's platform.

  ## Prerequisites

  - Python 3.8 or higher
  - A SignalWire account
  - pip package manager
</InstallHero>
```

## Props

- `title`: The main title
- `installCommand`: The installation command to display
- `language`: Code block language (default: 'bash')
- `githubUrl`: GitHub repository URL
- `docsUrl`: Documentation URL
- `gettingStartedUrl`: Getting started guide URL
- `status`: 'stable', 'beta', or 'alpha'
- `pypiPackage`: PyPI package name for version fetching
- `children`: Markdown content to display between title and code block
