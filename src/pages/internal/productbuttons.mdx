# ProductButton and ProductButtons components

This component, specifically designed for the homepage, displays the list of products in clickable-card
format with an icon, a name and a description.

## Demo

import ProductButtons from "@site/src/components/HomepageFeatures/ProductButtons"
import { products } from "./_products"
 
<ProductButtons products = {products}/>



## Usage 

```jsx
import ProductButtons from "@site/src/components/HomepageFeatures/ProductButtons"
import { products } from "./products"

<ProductButtons products = {products}/>
```
