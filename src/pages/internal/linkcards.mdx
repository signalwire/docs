# LinkCard and LinkCardContainer component

These components display links in a clickable card with description.

## Demo (full container)


import LinkCardContainer, { LinkCard } from "@site/src/components/HomepageFeatures/LinkCards";

<LinkCardContainer>
    <LinkCard title="Great Link" description="This is a great link! I swear you won't regret clicking on it. There's certainly no malware or other nefarious scripts behind the link."
    link="https://www.youtube.com/watch?v=dQw4w9WgXcQ"/>
    <LinkCard title="Even Better Link" description="This is a great link! I swear you won't regret clicking on it. There's certainly no malware or other nefarious scripts behind the link."
    link="https://www.youtube.com/watch?v=dQw4w9WgXcQ"/>
    <LinkCard title="Small link" description="This is a small link card."
    link="https://www.youtube.com/watch?v=dQw4w9WgXcQ"/>
    <LinkCard title="Great Link" description="This is a great link! I swear you won't regret clicking on it. There's certainly no malware or other nefarious scripts behind the link."
    link="https://www.youtube.com/watch?v=dQw4w9WgXcQ"/>
</LinkCardContainer>
