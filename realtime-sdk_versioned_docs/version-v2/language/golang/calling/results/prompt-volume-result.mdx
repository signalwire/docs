## Relay.Calling.PromptVolumeResult

This object is returned by [`volume`][link] method and represents the final result of a volume control operation.


### Properties

| Name | Type | Description |
| -: | - | - |
| `Successful` | `boolean` | Whether the volume has been changed successfully. |
