# Relay::Calling::PlayVolumeResult

[play]: ../actions/play.mdx#volume

This object is returned by [`volume`][play] method and represents the result of setting the volume.

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether volume has been set successfully. |
