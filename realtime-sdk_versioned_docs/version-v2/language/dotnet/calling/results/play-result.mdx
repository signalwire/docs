# SignalWire.Relay.Calling.PlayResult

This object returned from one the _synchronous_ [`Play`][call] methods or part of the action when an _asynchronous_ equivalent completes, which represent the final result of a play operation.

## Properties

| Property | Type | Description |
| :-------- | :---- | :----------- |
| `Successful` | bool | Whether the play operation has completed successfully. |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
