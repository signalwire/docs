# SignalWire.Relay.Calling.HangupResult

This object returned from [`Hangup`][call] method.

## Properties

| Property | Type | Description |
| :-------- | :---- | :----------- |
| `Successful` | bool | Whether the call has been hung up successfully. |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
| `Reason` | [`SignalWire.Relay.Calling.DisconnectReason`][signalwire-relay-calling-disconnectreason] | The hangup reason. |
