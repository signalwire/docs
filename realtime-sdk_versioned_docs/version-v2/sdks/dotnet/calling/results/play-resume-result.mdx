# SignalWire.Relay.Calling.PlayResumeResult

This object returned from the _synchronous_ [`Resume`][play] method, which represent the final result of a play resume operation.

## Properties

|    |   |   |
| -: | - | - |
| `Successful` | bool | Whether the play resume operation has completed successfully. |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
