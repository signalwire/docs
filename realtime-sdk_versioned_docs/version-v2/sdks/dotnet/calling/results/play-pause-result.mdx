# SignalWire.Relay.Calling.PlayPauseResult

This object returned from the _synchronous_ [`Pause`][play] method, which represent the final result of a play pause operation.

## Properties

|    |   |   |
| -: | - | - |
| `Successful` | bool | Whether the play pause operation has completed successfully. |
| `Event` | [`SignalWire.Relay.Event`][signalwire-relay-event] | The last event that completed the call. |
