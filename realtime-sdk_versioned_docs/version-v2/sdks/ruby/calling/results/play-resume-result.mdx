# Relay::Calling::PlayResumeResult

[play]: ../actions/play.mdx#resume

This object is returned by [`resume`][play] method and represents the final result of a play resume operation.

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether playback has resumed successfully. |
