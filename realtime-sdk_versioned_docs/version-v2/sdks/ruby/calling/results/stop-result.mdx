# Relay::Calling::StopResult

This object is returned from the _synchronous_ `stop` methods on an action when an _asynchronous_ operation is being stopped, which represent the final result of a stop operation.

:::info Available In
v2.3+
:::

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether the stop action has been performed successfully. |
