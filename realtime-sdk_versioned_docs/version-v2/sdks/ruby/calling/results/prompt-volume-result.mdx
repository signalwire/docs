# Relay::Calling::PromptVolumeResult

[prompt]: ../actions/prompt.mdx#volume

This object is returned by [`volume`][prompt] method and represents the result of setting the volume.

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether volume has been set successfully. |
