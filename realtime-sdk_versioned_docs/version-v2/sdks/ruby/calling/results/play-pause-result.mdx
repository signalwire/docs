# Relay::Calling::PlayPauseResult

[play]: ../actions/play.mdx#pause

This object is returned by [`pause`][play] method and represents the result of a play pause operation.

## Properties

| Property | Type | Description |
| :------- | :--- | :---------- |
| `successful` | Boolean | Whether playback has been paused successfully. |
