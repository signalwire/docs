## Relay.Calling.AnswerResult

This object returned from the [`answer`][link] method.

## Methods-submenu

### GetSuccessful

Return `true` if the answer has succeeded, `false` otherwise.

**Parameters**

_None_

```go
 resultAnswer := call.Answer()
  if !resultAnswer.Successful {
   if err := consumer.Stop(); err != nil {
     log.Errorf("Error occurred while trying to stop Consumer")
   }
  return
 }
```
