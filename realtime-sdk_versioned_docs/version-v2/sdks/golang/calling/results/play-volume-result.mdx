## Relay.Calling.PlayVolumeResult

This object is returned by [`volume`][link] method and represents the final result of a volume control operation.


### Properties

|    |   |   |
| -: | - | - |
| `successful` | `boolean` | Whether the playing volume has been changed successfully. |
