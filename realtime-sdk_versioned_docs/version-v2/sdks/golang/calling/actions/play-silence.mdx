## Relay.Calling.PlaySilenceAction

This object represents a `silence action` that is currently active on a call.

### Properties

|    |   |   |
| -: | - | - |
| `call` | [`Relay.Calling.Call`][link] | This is the call the action is occurring on. |


### Methods

<div class="method-definitions"></div>

#### Stop()

Stop the action immediately.

Available In: <img class="method-available-tag" src="https://img.shields.io/badge/-v1.0+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

StopResult

**Examples**

> Play 10 seconds of silence and then stop.

```go
playSilence, err := resultDial.Call.PlaySilenceAsync(10); 
if err != nil {
	signalwire.Log.Error("Error occurred while trying to play silence. Err: %v\n", err)
}
playSilence.Stop()
```
