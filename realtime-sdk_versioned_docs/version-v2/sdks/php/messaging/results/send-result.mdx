## Relay.Messaging.SendResult

This object returned from [`send`][index] method that represents the result of a send operation.

### Properties

|    |   |   |
| -: | - | - |
| `successful` | `boolean` | Whether the send operation has successfully queued the message. |
| `messageId` | `string` | The ID of the message. |

### Methods

#### getMessageId

Returns the ID of the queued message.

Available In: <img className="method-available-tag" src="https://img.shields.io/badge/-v2.1+-brightgreen.svg" />

**Parameters**

_None_

**Returns**

`string` - Message ID.

**Examples**

> Send a message and retrieve the ID.

```php
<?php

$params = [
  'context' => 'office',
  'from' => '+1XXXXXXXXXX',
  'to' => '+1YYYYYYYYYY',
  'body' => 'Welcome at SignalWire!'
];
$client->messaging->send($params)->done(function($sendResult) {
  if ($sendResult->isSuccessful()) {
    $messageId = $sendResult->getMessageId();
  }
});
```
