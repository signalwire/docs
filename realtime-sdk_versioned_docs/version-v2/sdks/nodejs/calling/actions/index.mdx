# Voice Actions

Action objects are returned from asynchronous call methods and represent operations currently active on a call.

## Available Actions

- **ConnectAction** - Represents an active connection attempt
- **DetectAction** - Represents an active detection operation
- **PlayAction** - Represents an active audio playback
- **PromptAction** - Represents an active prompt/collect operation
- **RecordAction** - Represents an active recording session
