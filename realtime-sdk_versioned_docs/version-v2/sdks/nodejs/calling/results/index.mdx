# Voice Results

Result objects are returned from call methods and represent the final outcome of operations.

## Available Results

- **AnswerResult** - Result of answering a call
- **ConnectResult** - Result of connecting calls
- **DialResult** - Result of dialing a call
- **PlayResult** - Result of play operations
- **RecordResult** - Result of recording operations
