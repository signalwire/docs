# Retrieve a Fax

Use this endpoint for the [Fax](./index.mdx) method to retrieve a single fax by its  [SID](/platform/dashboard/guides/what-is-a-sid/).

## Parameters

| Parameter                                           | Description                                                                     |
| :-------------------------------------------------- | :------------------------------------------------------------------------------ |
| `FaxSid` <span className="required-arg">Required</span> | The [SID](/platform/dashboard/guides/what-is-a-sid/) that uniquely identifies the fax to retrieve. |

---

## Request

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Faxes/{FaxSid}.json \
  -X GET \
  -u "YourProjectID:YourAuthToken"
```

</TabItem>

<TabItem value="node" label="Node.js">

```javascript
const { RestClient } = require('@signalwire/compatibility-api')
const client = RestClient('YourProjectID', 'YourAuthToken', { signalwireSpaceUrl: 'example.signalwire.com' })

client.fax.faxes('FaxSid')
          .fetch()
          .then(fax => console.log(fax.to))
          .done();
```
