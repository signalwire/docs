# Delete an Application

Use this endpoint for the [Applications](./index.mdx) method to delete an Application.

## Parameters

|                                        Parameter |                                                                                                   |
| -----------------------------------------------: | ------------------------------------------------------------------------------------------------- |
| `Sid` <span className="required-arg">Required</span> | The Application [SID](/platform/dashboard/guides/what-is-a-sid/) that uniquely identifies the application to delete. |

---

## Request

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Applications/{Sid}.json \
  -X DELETE \
  -u "YourProjectID:YourAuthToken"
```

</TabItem>

const { RestClient } = require('@signalwire/compatibility-api')

      .then(application => console.log(application.sid))

## Responses

<h3><span class="badge badge--success">204</span> No Content</h3>
