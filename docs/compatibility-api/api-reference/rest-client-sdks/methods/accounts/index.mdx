# Accounts

Accounts allow you to list and update your SignalWire **Projects**. 
You can create any number of projects on the SignalWire Dashboard and use the Accounts endpoint to list or update them.

|                                                   Attribute | Type                                               | Description                                                                                                                        |
| ----------------------------------------------------------: | -------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
|       `auth_token` <span class="optional-arg">object</span> | String                                             | The authorization token for this account. This token should be kept a secret and is not returned in a normal request.    |
|     `date_created` <span class="optional-arg">object</span> | Datetime                                           | The date, in RFC 2822 GMT format, this account was created.                                                              |
|     `date_updated` <span class="optional-arg">object</span> | Datetime                                           | The date, in RFC 2822 GMT format, this account was updated.                                                              |
|    `friendly_name` <span class="optional-arg">object</span> | String                                             | The description, up to 64 characters long, of the account. Default is the email address this account is associated with. |
|              `sid` <span class="optional-arg">object</span> | String                                             | The unique identifier for the account.                                                                                   |
|           `status` <span class="optional-arg">object</span> | String                                             | The status of the account. Possible values are: `active`, `suspended`, or `closed`.                                      |
| `subresource_uris` <span class="optional-arg">object</span> | A Map of sub-accounts linked to the given account. |                                                                                                                                    |
|             `type` <span class="optional-arg">object</span> | String                                             | The type of the account. Possible values are: `Trial` and `Full`.                                                        |
|              `uri` <span class="optional-arg">object</span> | String                                             | The URI for the account.                                                                                                 |
