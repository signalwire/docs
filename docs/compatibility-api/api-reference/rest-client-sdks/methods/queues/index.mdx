# Queues

Queue permits you to search and maintain individual call queues.

## Properties

> A sample Queue returned from the API

```json
{
  "account_sid": "b3877c40-da60-4998-90ad-b792e98472ac",
  "average_wait_time": 0,
  "current_size": 0,
  "date_created": "Wed, 26 Sept 2018 18:00:00 +0000",
  "date_updated": "Thur, 27 Sept 2018 19:00:00 +0000",
  "friendly_name": "0.361280134646222",
  "max_size": 123,
  "sid": "b3877c40-da60-4998-90ad-b792e98472qu",
  "uri": "/api/laml/2010-04-01/Accounts/b3877c40-da60-4998-90ad-b792e98472ac/Queues/b3877c40-da60-4998-90ad-b792e98472qu.json"
}
```

|                                                     Attribute |                                                                  |
| ------------------------------------------------------------: | ---------------------------------------------------------------- |
| `average_wait_time` <span class="optional-arg">integer</span> | The average wait time, in seconds, of callers in a queue.        |
|      `current_size` <span class="optional-arg">integer</span> | The number of calls waiting in the queue.                        |
|      `friendly_name` <span class="optional-arg">string</span> | A description that distinguishes a queue.                        |
|          `max_size` <span class="optional-arg">integer</span> | The maximum number of calls that are allowed to wait in a queue. |
|                `sid` <span class="optional-arg">string</span> | The unique identifier for the queue.                             |
