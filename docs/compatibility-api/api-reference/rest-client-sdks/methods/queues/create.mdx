# Create a Queue

Use this endpoint for the [Queues](./index.mdx) method to create a new call queue.

## Parameters

| Parameter                                                 | Description                                                      |
| :-------------------------------------------------------- | :--------------------------------------------------------------- |
| `FriendlyName` <span className="required-arg">Required</span> | A description that distinguishes a queue.                        |
| `MaxSize` <span className="optional-arg">Optional</span>      | The maximum number of calls that are allowed to wait in a queue. |

---

## Request: Create a New Call Queue

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://example.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/Queues.json \
  -X POST \
  --data-urlencode "FriendlyName=Queue1" \
  -u "YourProjectID:YourAuthToken"
```
