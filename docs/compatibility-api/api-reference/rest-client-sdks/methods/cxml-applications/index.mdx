# cXML Applications

cXML Applications are our serverless hosted service that allows you to easily serve static documents without setting up any infrastructure.

<LamlAdmonition />

## The cXML Application Object

> Example response of a cXML Application

```json
{
  "sid": "5184b831-184f-4209-872d-ccdccc80f2f1",
  "date_created": "2019-11-26T20:00:00Z",
  "date_updated": "2020-05-05T20:00:00Z",
  "date_last_accessed": "2020-06-05T20:00:00Z",
  "account_sid": "b3877c40-da60-4998-90ad-b792e98472af",
  "name": "Death Star IVR",
  "contents": "<Response><Say>Hello!</Say></Response>",
  "request_url": "https://{SPACE}.signalwire.com/laml-bins/5184b831-184f-4209-872d-ccdccc80f2f1",
  "num_requests": 1337,
  "api_version": "2010-04-01",
  "uri": "/api/laml/2010-04-01/Accounts/b3877c40-da60-4998-90ad-b792e98472af/LamlBins/5184b831-184f-4209-872d-ccdccc80f2f1",
}
```

|                                                     Attribute |                                                                                                                                                                                                                                       |
| ------------------------------------------------------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|                `sid` <span class="optional-arg">string</span> | The unique identifier of the cXML Application on SignalWire. This can be used to show, update, or delete the Application programmatically.                                                                                                            |
|       `date_created` <span class="optional-arg">string</span> | The date and time, in ISO 8601 format, the Application was created.                                                                                                                                                                           |
|       `date_updated` <span class="optional-arg">string</span> | The date and time, in ISO 8601 format, the Application was updated.                                                                                                                                                                           |
| `date_last_accessed` <span class="optional-arg">string</span> | The date and time, in ISO 8601 format, the Application was last accessed.                                                                                                                                                                     |
|        `account_sid` <span class="optional-arg">string</span> | The unique identifier for the account this Application is associated with.                                                                                                                                                                    |
|               `name` <span class="optional-arg">string</span> | A friendly name given to the cXML Application to help distinguish and search for different Applications within your project.                                                                                                                          |
|           `contents` <span class="optional-arg">string</span> | The contents of the Application, this evaluate to valid XML, with additional support for mustache templating. [You can read more about cXML in our documentation](https://developer.signalwire.com/compatibility-api/reference/xml-overview). |
|        `request_url` <span class="optional-arg">string</span> | The unique URL to the raw contents of the cXML Application. Use this as the URL for configuring webhooks or anything needing the XML returned.                                                                                                |
|          `num_requests` <span class="optional-arg">int</span> | The number of times this Application has been accessed.                                                                                                                                                                                       |
|        `api_version` <span class="optional-arg">string</span> | The version of the SignalWire API.                                                                                                                                                                                                    |
|                `uri` <span class="optional-arg">string</span> | The URL of this resource.                                                                                                                                                                                                             |
