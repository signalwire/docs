# Retrieve a cXML Application

Retrieves the details of a cXML Application that has been previously created. 
Use the unique ID that was returned from your previous request to identify the specific instance.

<LamlAdmonition />

## Parameters

| Parameter                              |
| :------------------------------------- |
| <span class="optional-arg">None</span> |

---

## Request 

> `GET /laml_bins/{ID}`

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://{SPACE}.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/LamlBins/5184b831-184f-4209-872d-ccdccc80f2f1 \
  -u 'YourProjectID:YourAuthToken'
```

</TabItem>

</Tabs>
