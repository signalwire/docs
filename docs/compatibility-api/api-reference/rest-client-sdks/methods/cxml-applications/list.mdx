# List all cXML Applications

Returns a list of your Addresses. 
The addresses are returned sorted by creation date, with the most recent appearing first.

<LamlAdmonition />

## Parameters

| Parameter                                         | Description                                                                                                                                               |
| :------------------------------------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Name` <span className="optional-arg">Optional</span> | The name given to the cXML Application to distinguish different Applications within your project. Will return all cXML Applications containing this value as a substring. |

---

## Request

> `GET /laml_bins`

<Tabs groupId="compatibility-sdks">

<TabItem value="shell" label="cURL">

```shell
curl https://{SPACE}.signalwire.com/api/laml/2010-04-01/Accounts/{AccountSid}/LamlBins?Name=Star \
  -u 'YourProjectID:YourAuthToken'
```

</TabItem>

</Tabs>
