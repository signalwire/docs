# Conferences

The Conference resource permits you to search, modify, and manage conferences in your SignalWire account.

## Properties

> A sample conference returned from the API

```json
{
  "account_sid": "b3877c40-da60-4998-90ad-b792e98472af",
  "api_version": "2010-04-01",
  "date_created": "Mon, 24 Sept 2018 21:00:00 +0000",
  "date_updated": "Tue, 25 Sept 2018 20:00:00 +0000",
  "friendly_name": "Conference1",
  "sid": "b3877c40-da60-4998-90ad-b792e98472ca",
  "region": "us1",
  "status": "completed",
  "subresource_uris": {
    "participants": "/api/laml/2010-04-01/Accounts/b3877c40-da60-4998-90ad-b792e98472af/Conferences/b3877c40-da60-4998-90ad-b792e98472ca/Participants.json",
    "recordings": "/api/laml/2010-04-01/Accounts/b3877c40-da60-4998-90ad-b792e98472af/Conferences/b3877c40-da60-4998-90ad-b792e98472ca/Recordings.json"
  },
  "uri": "/api/laml/2010-04-01/Accounts/b3877c40-da60-4998-90ad-b792e98472af/Conferences/b3877c40-da60-4998-90ad-b792e98472ca.json"
}
```

|                                                 Attribute |                                                                                                                                   |
| --------------------------------------------------------: | --------------------------------------------------------------------------------------------------------------------------------- |
|    `account_sid` <span class="optional-arg">string</span> | The unique identifier for the account that created this conference.                                                               |
| `date_created` <span class="optional-arg">datetime</span> | The date, in RFC 2822 format, this conference was created.                                                                        |
| `date_updated` <span class="optional-arg">datetime</span> | The date, in RFC 2822 format, this conference was updated.                                                                        |
|  `friendly_name` <span class="optional-arg">string</span> | A description, up to 64 characters, of the conference room.                                                                       |
|         `region` <span class="optional-arg">string</span> | The region where this conference audio was mixed. Possible values are `us1`, `us2`, `ie1`, `de1`, `sg1`, `br1`, `au1`, and `jp1`. |
|            `sid` <span class="optional-arg">string</span> | The unique identifier for this conference.                                                                                        |
|         `status` <span class="optional-arg">string</span> | The status of this conference.                                                                                                    |
|            `uri` <span class="optional-arg">string</span> | The URI for this conference.                                                                                                      |
