
# Provider Phonzo(Norway) 

 

## [www.phonzo.com](http://www.phonzo.com)

Last update: 22 April 2010

Username/auth-username are both required for phonzo. Auth-Username will be your DID number.

Incoming and outgoing calls work. In conf/sip\_profiles/external/ create a phonzo.xml file with the following contents:

```xml
<include>
  <gateway name="Phonzo">
    <param name="username" value="10xxxxxxx"/>
    <param name="password" value="xxxxxxx"/>
    <param name="realm" value="sip.phonzo.com"/>
    <param name="proxy" value="sip.phonzo.com"/>
    <param name="auth-username" value="4610xxxxxxx"/>
    <param name="from-domain" value="sip.phonzo.com"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```
