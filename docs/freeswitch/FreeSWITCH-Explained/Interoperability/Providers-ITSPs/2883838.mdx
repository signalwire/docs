
# Provider sipcall.ch (Switzerland) 

 

## [www.sipcall.ch](http://www.sipcall.ch)

Last update: 29 July 2010

Works with incoming and outgoing SIP calls.

your\_username is your phone number starting with 4132

```xml
<include>
  <gateway name="sipcall.ch">
    <param name="proxy" value="free3.voipgateway.org"/>
    <param name="username" value="your_username"/>
    <param name="password" value="your_password"/>
    <param name="extension" value="extension_that _should_be_called_in_your_dialplan"/>
    <param name="register" value="true"/>
    <param name="context" value="default"/>
  </gateway>
</include>
```
