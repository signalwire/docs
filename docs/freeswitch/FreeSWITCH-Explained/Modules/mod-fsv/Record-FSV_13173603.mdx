
# Record FSV 

 

## About

Record an FSV file.

* 1 [Usage](#usage)

### Usage

```xml
<action application="record_fsv" data="file.fsv"/>
```

Before you can actually start recording your video, you need to edit vars.xml to include video codecs

```xml
<X-PRE-PROCESS cmd="set" data="global_codec_prefs=G7221@32000h,G7221@16000h,G722,PCMU,PCMA,GSM,H263,H263-1998"/>
```
