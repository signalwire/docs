
# Py Session Transfer 

 

## About

Transfer inbound caller to a different local extension.

Click here to expand Table of Contents

* 1 [Synopsis](#synopsis)
* 2 [Example](#example)

## Synopsis

```xml
transfer(extension, dialplan, context)
```

* extension - destination extension, eg, "411"
* dialplan - dialplan name, eg, "XML" or use "" and system will use default
* dialplan - context name, eg, "default" or use "" and system will use default

## Example

```xml
self.session.transfer("411", "XML", "default")

self.session.transfer("411", "", "")
```
