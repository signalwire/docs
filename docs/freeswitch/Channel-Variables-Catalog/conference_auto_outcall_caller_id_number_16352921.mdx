
# conference_auto_outcall_caller_id_number 

 Created by  Ryan Harris, last modified on 2018.02.07

## conference_auto_outcall_caller_id_number

string Caller ID number to use when dialing endpoints to join the conference via the [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) application.

**Usage**

```xml
<action application="set" data="conference_auto_outcall_caller_id_number=${effective_caller_id_number}"/>
```
