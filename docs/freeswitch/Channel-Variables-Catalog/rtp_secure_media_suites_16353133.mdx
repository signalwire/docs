
# rtp_secure_media_suites 

 Created by  Ryan Harris, last modified on 2018.02.07

## rtp_secure_media_suites

string An alternative, optional variable that specifies a comma-separated list of available encryption suites. If this is specified, then [rtp\_secure\_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_secure_media) need only specify the permission flag without the list of suites.

Syntax: `rtp_secure_media_suites=<list of encryption suites>`
