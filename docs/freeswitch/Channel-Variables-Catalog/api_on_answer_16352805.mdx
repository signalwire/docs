
# api_on_answer 

 Created by  Ryan Harris, last modified on 2018.02.07

## api_on_answer

string Execute an api (not an application) when the called party answers. To set an application, use [execute\_on\_answer](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#execute_on_answer).

**Usage**

```xml
<action application="export" data="nolocal:api_on_answer=uuid_broadcast ${uuid} beep.wav both"/>
```

```xml
<action application="bridge" data="{api_on_answer='uuid_broadcast ${uuid} beep.wav both'}sofia/gateway/provider/5551231234"/>
```
