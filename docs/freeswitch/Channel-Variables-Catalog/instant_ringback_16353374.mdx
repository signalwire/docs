 Created by  Ryan Harris, last modified on 2018.02.07

## instant_ringback

boolean When set, ringback will not wait for indication before sending ringback tone to calling party.

**Usage**

![(warning)](/images/icons/emoticons/warning.svg) this may cause undesirable effects, like ringing for 3 seconds that turns into a busy signal.

```xml
<action application="set" data="instant_ringback=true"/>
```
