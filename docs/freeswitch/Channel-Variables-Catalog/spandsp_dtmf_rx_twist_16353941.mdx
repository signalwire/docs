
# spandsp_dtmf_rx_twist 

 Created by  Ryan Harris, last modified on 2018.02.08

## spandsp_dtmf_rx_twist

dtmf digit Sets the twist parameter in the spandsp DTMF detector. Twist is set to 8 dB by default. Set this variable prior to executing [spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications).

**Usage**

```xml
<action application="set" data="spandsp_dtmf_rx_twist=8"/>
<action application="spandsp_start_dtmf" />
```
