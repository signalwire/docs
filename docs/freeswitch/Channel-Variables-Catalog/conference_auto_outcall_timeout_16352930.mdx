## conference_auto_outcall_timeout

integer Originate timeout to use when joining a member to a conference via [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference).

This variable controls how long the outbound call will ring. Thus, if you do not ignore early media it will likely never get a chance to increment.

  
**Usage**

```xml
<action application="set" data="conference_auto_outcall_timeout=60"/>
```
