
# sofia_profile_name 

 Created by  Ryan Harris, last modified on 2018.02.08

## sofia_profile_name

string The name of the profile of which the call originated (Leg A).

![(warning)](/images/icons/emoticons/warning.svg) This should NOT be confused with [sip\_profile\_name](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#voicemail_profile_name).

**Usage**

Call from user on external profile

```xml
sofia_profile_name=external
```

Call from user on internal profile

```xml
sofia_profile_name=internal
```
