
# nat_private_addr 

 Created by  Duvid Rottenberg, last modified on 2018.04.24

## nat_private_addr

string

If you use [auto-nat](../FreeSWITCH-Explained/Networking/Auto-Nat_6587487.mdx#about) and a public IP address was detected, this variable will be set with the private IP address to which the detected public IP address is mapped.
