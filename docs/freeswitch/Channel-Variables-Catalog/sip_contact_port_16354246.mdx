
# sip_contact_port

## sip_contact_port

integer Port part of the `Contact` SIP header.

**Usage**

If your request header contains:

```xml
Contact: <sip:gw+test@server.example.com:5060;transport=udp;gw=test>
```

Then the value for this field would be:

```xml
5060
```
