
# fifo_orbit_context 

 Created by  Ryan Harris, last modified on 2018.02.07

## fifo_orbit_context

string Sets the context for [fifo\_orbit\_exten](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#fifo_orbit_exten). Must be set prior to placing the caller in the queue.

**Usage**

```xml
<action application="set" data="fifo_orbit_context=MyContext"/>
```
