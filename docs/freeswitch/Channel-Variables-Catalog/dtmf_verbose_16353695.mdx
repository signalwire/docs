
# dtmf_verbose 

 Created by  Ryan Harris, last modified on 2018.02.08

## dtmf_verbose

boolean Enables verbose logging of Spandsp DTMF detector. Default is `false`. Set this variable prior to executing [spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications).

**Usage**

```xml
<action application="set" data="dtmf_verbose=true"/>
<action application="spandsp_start_dtmf" />
```
