
# monitor_early_media_ring_total 

 Created by  Ryan Harris, last modified on 2018.02.08

## monitor_early_media_ring_total

string Specifies the number of user defined ring tones that can be heard before failing. To be used with [monitor\_early\_media\_ring](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#monitor_early_media_ring).

**Usage**

```xml
<action application="bridge" data="{ignore_early_media=true,monitor_early_media_ring_total=3,monitor_early_media_ring=us_ring:1:440.0+480.0}sofia/dial/string"/>
```
