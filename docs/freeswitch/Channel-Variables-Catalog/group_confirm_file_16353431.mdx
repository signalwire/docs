
# group_confirm_file 

 Created by  Ryan Harris, last modified by  John Boteler on 2022.06.21

## group_confirm_file

string This variable is used together with `group_confirm_key`. In `group_confirm_file`, you specify the wav file you want to play when the called party picks up the call. See [answer confirmation](../FreeSWITCH-Explained/Dialplan/Freeswitch-IVR-Originate_9634753.mdx#answer-confirmation) for more examples.

**Usage**

```xml
<action application="set" data="group_confirm_file=/usr/local/freeswitch/sounds/take_call_question.wav" />
```
