
# rtp_secure_media_outbound 

 Created by  Ryan Harris, last modified on 2018.02.07

## rtp_secure_media_outbound

string

Same syntax as [rtp\_secure\_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_secure_media): `rtp_secure_media_outbound=<permission>[:<list of encryption suites>]`

Examples:

`rtp_secure_media_outbound=true:AEAD_AES_128_GCM_8`

`rtp_secure_media_outbound=optional:AEAD_AES_128_GCM_8`
