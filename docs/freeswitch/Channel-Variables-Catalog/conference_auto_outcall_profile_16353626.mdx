
# conference_auto_outcall_profile 

 Created by  Ryan Harris, last modified on 2018.02.08

## conference_auto_outcall_profile

string Conference profile to use for members joining the conference via the [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) application.

**Usage**

```xml
<action application="set" data="conference_auto_outcall_profile=default"/>
```
