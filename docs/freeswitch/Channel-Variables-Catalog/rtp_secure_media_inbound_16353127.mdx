
# rtp_secure_media_inbound 

 Created by  Ryan Harris, last modified on 2018.02.07

## rtp_secure_media_inbound

string

Same syntax as [rtp\_secure\_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_secure_media): `rtp_secure_media_inbound=<permission>[:<list of encryption suites>]`

Examples:

`rtp_secure_media_inbound=true:AEAD_AES_256_GCM_8,AES_CM_256_HMAC_SHA1_80,AES_CM_256_HMAC_SHA1_32`

`rtp_secure_media_inbound=mandatory:AEAD_AES_256_GCM_8`
