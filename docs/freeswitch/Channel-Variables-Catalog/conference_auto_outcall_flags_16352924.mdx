
# conference_auto_outcall_flags 

 Created by  Ryan Harris, last modified on 2018.02.07

## conference_auto_outcall_flags

string Conference flags to set for members joining conference via the [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) application.

**Usage**

```xml
<action application="set" data="conference_auto_outcall_flags=mute"/>
```
