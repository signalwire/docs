
# presence_id 

 Created by  Ryan Harris, last modified on 2018.02.08

## presence_id

string Will instruct [mod\_sofia](../FreeSWITCH-Explained/Modules/mod_sofia_1048707.mdx#about) to invoke a `PRESENCE_IN` event that will assert the BLF lamp for the subscribed entity. This only lasts while the channel still exists so it should not be used for persistent status (i.e. a user that is DND, or an agent logged into a queue).

**Usage**

```xml
<action application="set" data="presence_id=${dialed_extension}@${domain_name}"/>
```
