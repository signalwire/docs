## session_in_hangup_hook

boolean Allows channel variables to be accessible in the [api\_hangup\_hook](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#api_hangup_hook) that gets executed for the channel. See [Lua env object](../FreeSWITCH-Explained/Client-and-Developer-Interfaces/Lua-API-Reference/index.mdx#special-case-env-object) for an example.

**Usage**

```xml
<action application="set" data="session_in_hangup_hook=true"/>
```
