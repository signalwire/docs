 Created by  Ryan Harris, last modified on 2018.02.07

## playback_ms

integer Contains the number of milliseconds of the length of the audio file just played back. There is also a variable [playback\_seconds](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#playback_ms) which stores the length in whole seconds.
