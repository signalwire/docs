
# sdp_secure_savp_only 

 Created by  Ryan Harris, last modified on 2018.02.07

## sdp_secure_savp_only

string When `rtp_secure_media=true` FreeSWITCH will offer both AVP and SAVP in the SDP. Setting `sdp_secure_savp_only=true` (in addition to [rtp\_secure\_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_secure_media)`=true`) will cause FreeSWITCH to offer only SAVP in the SDP.

**Usage**

```xml
<action application="export" data="sdp_secure_savp_only=true"/>
```
