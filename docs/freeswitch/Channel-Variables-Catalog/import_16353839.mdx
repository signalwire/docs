## import

string The import variable, when used before a bridge, imports the variables of the other channel on the actual channel. Implemented by [mod\_lcr](../FreeSWITCH-Explained/Modules/mod_lcr_6587457.mdx#lrn).

**Usage**

```xml
<action application="set" data="import=this_is_a_variable_name"/>
```
