
# bind_digit_digit_timeout 

 Created by  Attila Gulyas on 2019.12.25

Integer 

Inter-digit timeout, in milliseconds. This sets the time to wait between individual dialed digits.

Default value is 1500 milliseconds.

Mostly only useful in conjunction with [mod\_dptools: play\_and\_get\_digits](../FreeSWITCH-Explained/Modules/mod-dptools/6586933.mdx#about) in the same dialplan extension.

See section **3.1** in [mod\_dptools: bind\_digit\_action](../FreeSWITCH-Explained/Modules/mod-dptools/6586414.mdx#0-about).

Usage

**Example**

```xml
<!-- To wait 10 seconds between digits: -->
<execute application="set" data="bind_digit_digit_timeout=10000" />
```
