
# sip_looped_call 

 Created by  Ryan Harris, last modified on 2018.02.09

## sip_looped_call

boolean `true` if the call has been authenticated via means other than an ACL and the current request IP/port matches the profile IP/port (see [src/mod/endpoints/mod\_sofia/sofia.c](https://freeswitch.org/stash/projects/FS/repos/freeswitch/browse/src/mod/endpoints/mod%5Fsofia/sofia.c)).

**Usage**

In `conf/dialplan/public.xml`:

```xml
<condition field="${sip_looped_call}" expression="^true$">
```

```xml
<extension name="unloop">
  <condition field="${unroll_loops}" expression="^true$"/>
  <condition field="${sip_looped_call}" expression="^true$">
    <action application="deflect" data="${destination_number}"/>
  </condition>
</extension>
```
