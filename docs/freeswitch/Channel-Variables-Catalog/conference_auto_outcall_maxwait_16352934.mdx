
# conference_auto_outcall_maxwait 

 Created by  Ryan Harris, last modified on 2018.02.07

## conference_auto_outcall_maxwait

integer Maximum time in seconds that the channel that initiated the [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) will wait for members to join the conference.

**Usage**

```xml
<action application="set" data="conference_auto_outcall_maxwait=10"/>
```
