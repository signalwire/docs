
# nat_public_addr 

 Created by  Duvid Rottenberg, last modified on 2018.04.24

## nat_public_addr

string

If you use [auto-nat](../FreeSWITCH-Explained/Networking/Auto-Nat_6587487.mdx#about) and FreeSWITCH was able to identify the public IP address pointing to your server, this variable will be set with the auto detected public IP address.
