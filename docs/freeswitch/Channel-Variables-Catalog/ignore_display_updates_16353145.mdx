
# ignore_display_updates 

 Created by  Ryan Harris, last modified on 2018.02.07

## ignore_display_updates

string Tells freeswitch not to send display UPDATEs to the leg of the call. (update\_display)

**Usage**

From `dialplan/default.xml`, to set on A-Leg:

```xml
<action application="set"  data="ignore_display_updates=true"/>
```

To set on B-Leg:

```xml
<action application="bridge" data="{ignore_display_updates=true}sofia/gateway/provider/18005551212"/>
```
