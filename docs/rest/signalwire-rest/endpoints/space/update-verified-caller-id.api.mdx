To update an existing Verified Caller ID, you send a `PUT request to
the Verified Caller ID resource. Any parameters not passed in will
remain unchanged.

The extension is only used when sending a verification call, if you
need to update the extension and initiate a new call - use the Create
a Verified Caller ID endpoint.

#### Permissions
The API token must include the following scopes: _Voice_.
