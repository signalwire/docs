# Overview

:::caution
The [RELAY v4](/sdks/reference/realtime-sdk/relay-v4) is the most up-to-date version of the Realtime Relay SDK.
Consider reading the [Upgrading to Relay v4](/sdks/reference/realtime-sdk/guides/upgrading-to-relay-v4) page to understand the benefits of
[RELAY v4](/sdks/reference/realtime-sdk/relay-v4) and how to upgrade.
:::

The SignalWire Realtime SDK v3 revolutionizes real-time communication by simplifying the creation
and management of various communication services.
Through the straightforward installation process and the initiation of a realtime client,
developers can easily engage with `Video`, `Chat`, `Messaging`, `PubSub`, `Tasks`, and `Voice` functionalities.
The transition to v3 emphasizes ease of use with product-specific constructors, eliminating the need for manual client creation,
and offering an enhanced event-driven model for more intuitive interaction and control over communication services.

## Installation

```bash npm2yarn
npm install @signalwire/realtime-api@~3
```

## Get started

To get started, create a realtime client, for example with [Video.Client](./video/video-roomsession.mdx) and listen for events. For example:

```javascript
import { Video } from "@signalwire/realtime-api";

const video = new Video.Client({
  project: "<project-id>",
  token: "<project-token>",
});

video.on("room.started", async (roomSession) => {
  console.log("Room started");

  roomSession.on("member.joined", async (member) => {
    console.log(member);
  });
});
```
