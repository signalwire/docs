Access the Chat API Consumer. You can instantiate a [Chat.Client](./chat-client.mdx) to subscribe to Chat events. Please check [Chat Events](./chat-client.mdx#events) for the full list of events that a [Chat.Client](./chat-client.mdx) can subscribe to.

#### Example

The following example logs the messages sent to the "welcome" channel.

```javascript
import { Chat } from "@signalwire/realtime-api";

const chatClient = new Chat.Client({
  project: "<project-id>",
  token: "<api-token>",
});

chatClient.on("message", (m) => console.log(m));

await chatClient.subscribe("welcome");
```
