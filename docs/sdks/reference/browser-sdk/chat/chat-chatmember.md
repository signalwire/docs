---
title: "ChatMember"
---

Represents a member in a chat.

## Properties

### channel

Accesses the channel of this member.

**Syntax:** `ChatMember.channel()`

**Returns:** `string`

---

### id

Accesses the ID of this member.

**Syntax:** `ChatMember.id()`

**Returns:** `string`

---

### state

Accesses the state of this member.

**Syntax:** `ChatMember.state()`

**Returns:** `any`
