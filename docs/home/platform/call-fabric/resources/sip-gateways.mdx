# SIP Gateways

<Subtitle>Resources on the SignalWire Call Fabric Platform</Subtitle><br/><br/>

## About

SIP Gateways are a simple but powerful Resource type that forward calls to external SIP addresses.

<Frame caption="When creating a SIP Gateway, give it a name, enter the full external URI, and configure encryption, ciphers, and codec settings.">
    ![Creating a SIP Gateway Resource in the SignalWire Dashboard.](@image/dashboard/resources/sip/sip-gateway.png)
</Frame>

Not to be confused with SIP Domain Apps, which route inbound calls to SignalWire SIP addresses,
or SIP Endpoints, which are for registering customer devices.
