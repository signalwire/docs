The Multi-Factor Authentication setting can be enabled via your SignalWire Space Dashboard at the User level.

First you will need to navigate to your profile. 
Navigate to the drop-down menu in the top-right of your Dashboard.

<figure>
  <img className="img-800" src={require("./6469a15-TopRight.webP").default} alt="A screenshot of the top right hand corner of a SignalWire Space Dashboard, showing a profile picture with a drop-down menu icon." />
</figure>

:::danger Notice

If 2FA/MFA is enabled at the Space level, users will not be able to update individual settings.

:::

To enable Multi-Factor Authentication at the User level return to the drop-down in the top right and select `Profile`.

<figure>
  <img className="img-400" src={require("./b056782-Dropdown.webP").default} alt="A screenshot of the resulting drop-down menu, listing the following options: Profile, Billing, Users, Settings, Switch Spaces, and Log out." />
</figure>

From here you can toggle Multi-factor authentication just for your own user profile.

<figure>
  <img className="img-800" src={require("./f7f0321-Profile_settings.webP").default} alt="A screenshot of the Edit Profile page. A toggle labeled 'Email MFA' is listed beneath the Multi-Factor Authentication header." />
</figure>
