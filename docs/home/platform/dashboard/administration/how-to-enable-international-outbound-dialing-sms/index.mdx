## How to Request International Activation

By default, SignalWire Spaces are only enabled for the US and Canada.
Please [submit a request form](https://support.signalwire.com/hc/en-us/requests/new?ticket_form_id=13502646494107) for International Services to be enabled on your Space.

A new support case will be opened and our technicians will be in contact when approved and enabled.

:::caution

Please Note: Only spaces **NOT** in trial mode will be enabled for international services.

:::

### Enable Geographic Permissions

Once enabled, you will need to add the geographic permission for each country you would like to be able to dial or message.
Follow the steps below to complete:

### Navigation

First, navigate to your signalwire Dashboard, and access the drop-down menu in the top right:

### Selecting Countries

Once in the Space settings menu, click `Geographic Permissions`.

Here you can check the box for the countries for which you would like to enable international Dialing/SMS.

:::caution

Please Note: Messaging can only be sent from US and Canadian (CA) long code numbers.
International SMS is not enabled for toll-free numbers.
