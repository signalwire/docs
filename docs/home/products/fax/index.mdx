Prototype, build, and deploy fax applications quickly and at competitive rates
using our APIs and SDKs as well as the lightweight SignalWire Markup Language ("SWML").

<Admonition type="tip" icon="🔐" title="security first">SignalWire Fax is certified [**SOC 2 Type II**](https://signalwire.com/blogs/industry/soc-2-type-ii-attestation) compliant, and is built from the ground up for seamless **HIPAA** compliance.</Admonition>
