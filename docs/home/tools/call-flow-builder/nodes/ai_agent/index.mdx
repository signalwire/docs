# AI Agent

This node is used to connect to a SignalWire conversational [AI agent](/ai/get-started).
The AI agent can be used to handle natural language processing, sentiment analysis, make and handle API calls, and other conversational tasks.

:::tip
Learn more about AI Agents with our comprehensive [Getting Started](/ai/get-started/) guide!
:::

## **Node Settings**

| Node Setting      | Description                                                                                                                                                                                    |
|:------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **AI Agent Name** | The name of the AI agent to connect to. The dropdown menu will populate with any [AI Agent Resource](/platform/call-fabric/resources#ai-agents) that have been created in your SignalWire portal. |
