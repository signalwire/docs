Fillers that will be played while the AI Agent is running the `get_visual_input` function.

| Name                                                                | Type       | Default | Description                                                                                                                                                                |
|:--------------------------------------------------------------------|:-----------|:---------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `get_visual_input`<span className="optional-arg">Optional</span>    | `object`   | -       | An object containing language-specific filler phrases. Each key must be a [supported language code](/swml/methods/ai/swaig/internal_fillers#supported-language-codes) and its value must be an array of phrases that will be randomly played during visual input processing. |

## Supported Language Codes

The `get_visual_input` function accepts an object with the key being one of the supported language codes and the value being an array of filler phrases.

## Example


```yaml andJSON
get_visual_input:
  en-US:
    - 'Analyzing visual input, please wait.'
    - 'I am scanning my surroundings for data, this won''t take long.'
    - Please wait briefly while I process the data in front of me.
    - 'I am currently digitizing the data so I can proceed, please hold on.'
  es-ES:
    - 'Analizando entrada visual, por favor espera.'
    - 'Estoy escaneando mi entorno para obtener datos, esto no tardará mucho.'
    - Por favor espera un momento mientras proceso los datos frente a mí.
    - Actualmente estoy digitalizando los datos para poder proceder, por favor espera.
```
