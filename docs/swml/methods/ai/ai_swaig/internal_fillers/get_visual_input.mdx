Fillers that will be played while the AI Agent is running the `get_visual_input` function. This function is enabled to be ran when the `enable_vision` property is set to `true` in the [`ai.params`](/swml/methods/ai/params) method.

| Name                                                                | Type       | Default | Description                                                                                                                                                                |
|:--------------------------------------------------------------------|:-----------|:---------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `get_visual_input`<span className="optional-arg">Optional</span>    | `object`   | -       | An [object containing language-specific filler phrases](#language-specific-fillers).  |


## Language-Specific Fillers

| Language Code | Type | Description |
|:--------------|:-----|:------------|
| `[key: string]` | `string[]` | An array of filler phrases that will be played during visual input processing. The key is a valid [language code](#supported-language-codes) and the value is an array of filler phrases that are selected from randomly. |


## Supported Language Codes

The `get_visual_input` function accepts an object with the key being one of the supported language codes and the value being an array of filler phrases.

## Example


```yaml andJSON
get_visual_input:
  en-US:
    - 'Analyzing visual input, please wait.'
    - 'I am scanning my surroundings for data, this won''t take long.'
    - Please wait briefly while I process the data in front of me.
    - 'I am currently digitizing the data so I can proceed, please hold on.'
  es-ES:
    - 'Analizando entrada visual, por favor espera.'
    - 'Estoy escaneando mi entorno para obtener datos, esto no tardará mucho.'
    - Por favor espera un momento mientras proceso los datos frente a mí.
    - Actualmente estoy digitalizando los datos para poder proceder, por favor espera.
```
