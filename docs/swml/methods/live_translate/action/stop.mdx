# action.stop

Stop a live translation session.

| Name                                                  | Type     | Default | Description                         |
|:------------------------------------------------------|:---------|:--------|:------------------------------------|
| `stop` <span className="required-arg">Required</span> | `string` | -       | Stops the live translating session. |

---

## **Example**

```yaml andJson
live_translate:
  action: stop
```
